# freee CFO Learning — エージェント運用マニュアル (AGENTS.md)

freee KB（2,424記事）を活用した **起業家CFO向け16週間学習カリキュラム** のためのAI学習補助システムです。

## コア原則

1. **KB準拠** — すべての解説・問題・シナリオは `kb/` の実際の記事に基づくこと。ハルシネーションは厳禁。
2. **CFO視点** — 知識の暗記ではなく「経営判断に使える理解」を目標とする。
3. **実務直結** — 「自社の場合はどうなるか」まで考えることで学習を完結させる。
4. **引き辞書として使う** — 全記事を通読しようとしない。必要なときに `kb-navigator` で引く。

---

## 学習専用エージェント（`agents/learning/`）

| エージェント | 役割 | いつ使うか |
|---|---|---|
| `kb-navigator` | KB 2,424記事を探索し優先度付きリストで提示 | トピック調査、Week記事の特定 |
| `quiz-master` | 択一・仕訳・計算・シナリオ判断のクイズを生成 | 学習後の理解度確認、弱点特定 |
| `scenario-coach` | リアルなCFOシナリオを対話形式でコーチング | 知識を判断力に変換するトレーニング |

---

## スキル（スラッシュコマンド）

| コマンド | 説明 | 使用例 |
|---|---|---|
| `/cfo-quiz` | Week/トピックの理解度クイズを生成 | `/cfo-quiz Week 5` |
| `/kb-lookup` | KB検索・用語解説（一言定義＋CFO的ポイント＋記事リスト） | `/kb-lookup 定期同額給与` |
| `/week-review` | 週次振り返り・到達チェック・ログ下書き生成 | `/week-review Week 3` |
| `/scenario` | CFOビジネスシナリオの対話実践 | `/scenario 資金繰り 実戦` |

---

## ワークフロー

| ワークフロー | 説明 | 目安時間 |
|---|---|---|
| `weekly-study` | KB選定 → 精読 → クイズ5問 → シナリオ → ログ記録の一気通貫 | 60〜90分 |
| `kb-deep-dive` | 1トピックを体系的に深掘り → 要点シート（`roadmap/resumes/`）に保存 | 40〜70分 |

---

## 典型的な学習フロー

```
# 通常の1週間学習
weekly-study Week {N}
  ├─ kb-navigator が★★★記事を特定
  ├─ 精読サポート（対話形式）
  ├─ quiz-master が5問生成・採点
  ├─ scenario-coach がシナリオ実践
  └─ ログ記録（roadmap/log/YYYY-MM-DD.md）

# 用語が分からないとき
/kb-lookup {用語}
  └─ 一言定義 + CFO的ポイント + KB記事リスト

# 深く掘り下げたいとき
kb-deep-dive {トピック}
  └─ KB全量調査 → 構造化インプット → クイズ → シナリオ → 要点シート保存

# 週末の振り返り
/week-review Week {N}
  └─ 到達チェック確認 → 要復習記事 → ログ下書き → /cfo-quiz に続ける
```
