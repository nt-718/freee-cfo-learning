# IPOとは？上場との違いから審査基準までわかりやすく解説

出典: https://www.freee.co.jp/kb/kb-ipo/about_ipo/

---

IPOとは、会社の株式を一般の投資家に売り出すために、初めて株式市場に株式を公開することで「新規株式公開」とも呼ばれています。
  
  
IPOは非上場企業にとって、企業の社会的信用の向上や、資金調達の選択肢の拡大が望める有効な手段です。特に中小企業や新興企業、ベンチャー企業など、経営の活性化を目的とした企業によるIPOの実施が増えています。
  
  
国内の各証券取引所でもIPOを行うベンチャー企業向けの株式市場が提供されていることから、株式市場でもIPOが積極的に取り入れられているといえるでしょう。
  
  
本記事では、IPOの概要や企業がIPOを行う場合のメリット・デメリット、IPOが積極的に行われている市場などについて詳しく解説します。

## IPOとは

IPOとは、限定された株主にのみ所有を認めていた未上場会社の株式（未公開株）を、初めて証券取引所を通して株式市場に公開し、不特定多数の投資家に向けて株式を売り出すことをいいます。
  
  
IPOで株式を市場に公開する場合は、新たに株式を発行するか、上場前に株主が保有している株式を売り出すことで実施されます。
  
  
IPOの株式の価格は「公募価格」と「売出価格」が存在します。新規で株式を発行し、買い手を募ることを公募と呼び、買い手を募る際の価格を公募価格、発行済みの株式を売り出す価格のことを売出価格といいます。
  
  
これらの価格を設定するのは、上場を希望する企業が依頼する主幹事証券会社です。主幹事証券会社が監査の結果や会社の実績を基に価格を想定した後、投資家などへのヒアリングを通して価格の最終決定を行います。

### IPOと上場の違い

IPOと上場はどちらも「未上場企業が証券取引所に上場し不特定多数の投資家からの株式の売買を可能にすること」という点では同じですが、厳密には少し違いがあります。
  
  
前述したように、IPOは新たに株式を公開することです。IPOに含まれるものは以下のとおりです。

IPOに含まれるもの

* 外部に金融商品を公開して購入を促すこと
* 誰でも株式（有価証券）を購入できる状態にすること
* 増資が目的であるため、新しい株式が発行される
* 既存の株主が保有する株式を売却して、投資家に販売すること
* 保有する株式の売却や株式の新規発行により資金調達を狙いとする
* Initial（新規）であるため、あくまでも初めて株式を公開する際に使う場合のこと

一方で上場とは、株式会社が保有・発行する株式が証券取引所での取引を認められることです。新規の株式発行を伴う必要はありませんが、国内での新規上場の場合ほとんどのケースで新規の株式が発行されています。
  
  
これらの理由から、IPOと上場はほぼ同じ意味で使われています。

### 直接上場（ダイレクトリスティング）との違い

ほかにも、似た言葉として「直接上場（ダイレクトリスティング）」が挙げられます。直接上場とは、新株の発行を行わずに、上場前の株式のみで上場することです。また直接上場には、下記のような特徴があります。

* 上場に際し新規で株を発行（新株）せずに、上場前に発行した既存の株のみを上場させること
* 資金調達を目的としておらず、新株を引き受ける銀行や証券会社などを利用しないため、上場の際に発生する引受手数料等のコストを大幅に削減できる

直接上場はIPOとは異なり、資金調達を目的としないため、幹事証券会社などに支払う新株発行費用などの費用が抑えられます。
  
  
ただし、直接上場では資金調達などのメリットが得られず、株式の価値決定やリスク評価が困難になるというリスクも存在するため、現時点で国内での直接上場実績は1999年以降1社のみです。

### IPOとPOの違い

IPOとPOはとても似ていますが、こちらも意味が異なります。上述したとおり、IPOは新たな株式上場と株式の公募・売り出しを行うことです。
  
  
一方、POはPublic Offeringの略称で「公募」の意味で、すでに上場している企業が資金調達のために新たに株式の発行を行うことを指します。POは、IPOよりも小規模であり、株式取引所に上場していないため、株式の価値決定やリスク評価が困難になることが指摘されています。
  
  
また、企業が所有する自社株や大株主が保有する発行済株式を売却する際、一般の投資家に新たに公開する株式、または既に発行された株式である売出株式の購入を促す「売出」もPOに含まれます。

## IPOするメリットとデメリット

IPOは企業だけでなく投資家や従業員にもメリットとデメリットがあります。
  
  
ここでは、IPOを行うメリットとデメリットを立場別に解説します。

### 企業のメリット・デメリット

企業にとってIPOする主なメリットは以下のとおりです。

IPOを行うメリット

* 金融機関をはじめとした資金調達を受けやすくなる
* 社内体制（経営体制）の強化
* 企業価値、知名度の向上
* 社会的信用度の向上
* 人材採用の向上

IPOを行うことで、一般の投資家からの投資を受けられるようになり、さらに上場に伴う審査に合格できる企業であるという証明になるため、社会的信用の向上も期待できます。
  
  
社会的信用が高まれば、金融機関からの融資が期待できるだけでなく、よりスキルの高い人材が確保しやすくなることもIPOの魅力です。
  
  
一方、デメリットとしては主に以下の3つが挙げられます。

IPOの企業のデメリット

* 上場後の成長維持へのプレッシャーがかかる
* IPO準備や維持のために人的・資金的・時間的コストが掛かる
* 誰でも上場できるわけではない

IPOが積極的に行われているベンチャー企業向けの市場では、本則市場（実績のある企業向けの市場）に比べて投資リスクの高い企業が多いことから、より継続的な企業成長が求められます。
  
  
また、IPOを希望した全ての企業ができるものではありません。上場準備の段階から、人的・資金的・時間的コストが多くかかるため、IPOが可能な企業体制の構築が必要となります。

### 株主のメリット・デメリット

既存の株主と一般投資家にとって主なメリットは以下のとおりです。

IPOをすることによる株主のメリット

* 既存株主の保有株の資産価値が向上する
* 保有する株式の現金化が容易になる
* 期待値が高く利益が出やすい
* 株式の流通性が向上

未公開株式の場合、保有する株式を販売する場所や機会が限られるため、IPOを行い株式を一般に公開することで株式の売買や資産の現金化が可能になります。
  
  
また、IPOは投資家からの期待値が高いため、上場後に株価の上昇が期待できます。そのためIPOは、既存株主や公募で株式を取得した投資家にとっても利益を見込みやすい銘柄です。
  
  
一方、株主にとってのデメリットとして以下の2点が挙げられます。

IPOをすることによる株主のデメリット

* 投資リスクが高い
* 公募への当選が難しい

IPOは、主にベンチャー企業や新興企業が行うことが多く、実績のある企業などの銘柄に比べると株価や企業の業績が安定しているとは言い難いため、株主にとって投資リスクが高い投資先です。
  
  
また、上場時の公募にて一般投資家が株式を購入しますが、人気の銘柄であるほど公募の当選確率が下がるため、IPOを行う企業への投資を考えている投資家にとってはデメリットといえます。

### 従業員のメリット・デメリット

従業員にとって、勤め先がIPOすることで主に以下のメリットがあります。

IPOをすることによる従業員のメリット

* 上場による企業価値の向上
* 上場準備に伴うスキルの向上

IPOは、その審査の厳しさから上場することで得られる企業価値や信用は高く、上場企業に勤めている従業員にも一定のメリットが期待できます。
  
  
また、上場準備のための社内体制の強化や、業務システムの変更などへの対応などをとおしてのスキルアップや、上場を経験したことは将来のキャリア形成にも役立ちます。
  
  
一方、従業員目線での主なデメリットは以下の3点です。

従業員のデメリット

* 社内規則の改定や業務フローへの対応が必要
* 求められる業務のレベルやクオリティが大きく上がる
* IPO準備による業務の増加

IPOは社内体制に大きく影響するため、働き方や社内規定の改定に順次対応しなければなりません。
  
  
また、IPO準備に伴う従業員の業務負荷の増加や、求められる業務の質の水準が高くなることで、モチベーションが低下する要因になってしまうおそれがあります。

## IPO実現までの過程と必要資金をわかりやすく解説

IPOを行うことを社内で決定してから実際に上場の申請をするまでは、最短で2～3年程度の期間が必要です。また、IPO準備や申請には数千万円程度のコストがかかります。
  
  
ここでは、IPOの決定から上場までの流れと必要資金について解説します。

### IPOの準備期間

一般的なIPOの準備から上場までの流れは以下のとおりです。

1. 直前々期の前（申請の3～4年前）
2. 直前々期（申請の2年前）
3. 直前期（申請の1年前）
4. 申請期から上場

上述したように、IPOは実際に申請を行う前に2～3年程度の準備期間が必要です。これは、上場申請に提出する監査証明書のための監査期間が2年程度、監査を受ける体制を整えるための時間が約1年かかるためです。
  
  
最初に受ける簡易の監査（ショートレビュー）の結果次第では、より長い時間が必要です。

#### 1．直前々期の前（申請の3～4年前）

上場先の証券会社や市場によって異なりますが、IPOの申請を行うには2年間の監査証明が必要です。
  
  
直前々期には監査を行う監査法人をはじめとしたIPO準備を手伝ってくれる会社を選定し契約します。本格的な監査を受ける前に、ショートレビューと呼ばれる簡易的な監査を受け、その結果を基に社内体制の整備を行います。
  
  
この期間に本格的な外部監査を受ける準備を整えます。並行して「公開準備室」や「上場準備室」などと呼ばれるIPOに向け、社内整備や申請書類の作成を担う専門部署を社内で立ち上げるとスムーズにIPOの準備が進みます。。

#### 2．直前々期（申請の2年前）

IPOの申請には2年分の監査証明が必要なため、外部監査を受けられる社内体制が整ったら外部監査による監査が始まります。
  
  
直前々期はショートレビューを基に整えた社内体制を運営しながら監査を受けます。この時、外部監査を行う会社からフィードバックを受け、指示体系の明確化や業務フローの改善など、社内体制をさらに強化していきます。

#### 3．直前期（申請の1年前）

直前期には直前々期に受けたフィードバックを基にさらに強化した社内体制で会社を運営し、引き続き外部監査を受けます。
  
  
また、上場申請書類や関連書類の作成を進め、あわせて主幹事証券会社による審査を受け申請準備は完了です。

#### 4．申請期から上場

監査証明書を含めた申請書類を揃えたら申請を行います。申請後は証券会社による審査が進められます。
  
  
具体的には提出した書類を基に形式基準の審査が行われ、その後実質基準の審査では実地調査や面談、説明会などを経て、全ての基準を満たせば上場が認められます。
  
  
上場が確定したら、証券会社と上場契約を取り交わし、上場することを一般公開します。

### IPOの必要資金

IPOは上場申請そのものだけでなく、申請の準備や審査にも費用が発生します。
  
  
実際の費用は、企業規模や上場する市場によって変動しますが、上場準備にはコンサルタントや監査、証券事務代行費用をはじめとしたIPOに発生する必要資金は約2,400万円〜8,000万円程度です。
  
  
実際に必要なコストの種類やその費用目安は以下のとおりです。

  

|  |  |
| --- | --- |
| 項目 | 費用の目安 |
| 監査法人からの短期調査費用 | 150～400万円 |
| 監査費用 | 300～2,000万円 |
| 上場審査料 | 100万円 |
| 上場手数料 | 200～1,500万円 |
| 年間手数料 | 60～150万円 |
| 証券会社など引受手数料 | 公募価格×株式数×手数料率（5～7％） |
| 証券会社のコンサルティング費用 | 500～2,000万円 |
| 有価証券届出書作成 | 100万円 |
| その他上場コンサルタント費用 | 500万円～1,000万円 |
| 株式事務代行費用 | 400万円 |

## IPOを行う主な株式市場

IPOを行うためには、上場を予定する証券取引所および上場を予定している市場の上場基準を満たす必要があります。
  
  
IPOを行うのは新興企業やベンチャー企業が中心です。国内にある4つの証券取引所にはそれぞれベンチャー企業向けの市場があり、各証券取引所にIPOをする企業のほとんどはそれらの市場を利用します。
  
  
東証（東京証券取引所）でのIPOは、プライム市場やスタンダード市場など実績のある企業向けの市場でも一定数の実績があります。しかし、上場基準が厳しいため、ベンチャー企業向けの市場でIPOを行った後に上位市場へ市場区分を変更するのが一般的です。
  
  
**【国内の証券取引所とその市場】**

|  |  |
| --- | --- |
| 東京証券取引所 | ・プライム市場   ・スタンダード市場   ・グロース市場 |
| 名古屋証券取引所 | ・メイン市場   ・プレミア市場   ・ネクスト市場 |
| 福岡証券取引所 | ・本則市場   ・Q-Board |
| 札幌証券取引所 | ・本則市場   ・アンビシャス市場 |

ここでは、各証券会社で提供されている株式市場のうち、IPOを行うのに適した市場であるグロース市場、ネクスト市場、Q-Board、アンビシャス市場について解説します

### 東京証券取引所：グロース市場

東証（東京証券取引所）は、ベンチャー企業向けの市場として「グロース市場」を提供しています。グロース市場は、企業としての成長が高く期待できる国内の企業向けの市場です。
  
  
東証のほかの市場に比べると上場基準は大幅に緩和されていますが、上場維持のためには上場基準の維持だけでなく、時価総額の上昇などが求められます。
  
  
**【グロース市場の上場基準（一部抜粋）】**

|  |  |
| --- | --- |
| 項目 | 概要 |
| 事業計画 | ・成長の可能性が高いと判断できる根拠が主幹事証券会社によって示されている   ・事業内容や市場、企業の特性、リスクなどを踏まえた事業計画や企業情報が適時、適切に開示される見込みがある   ・根拠のある事業計画書が提示されている |
| 流動性 | ・上場企業として最低限の流動性（流通している株式量）がある   ・株主数：150人以上   ・流通株式数：1,000単位以上   ・流通株式時価総額：5億円以上 |
| ガバナンス | ・企業の現状を踏まえたガバナンスが適切に整備、運用されている   ・流通株式比率：25％以上 |

[出典：日本取引所グループ「市場区分見直しの概要」](https://www.jpx.co.jp/equities/improvements/market-structure/01.html)
  

なお、東証（東京証券取引所）は、2022年4月4日（月）に従来の市場区分である東証一部・東証二部・東証マザーズ・JASDAQ（スタンダード・グロース）から、プライム市場・スタンダード市場・グロース市場の3つの市場区分に再編が行われています。

### 名古屋証券取引所：ネクスト市場

名古屋証券取引所は、ベンチャー企業向けの市場として「ネクスト市場」を提供しています。名古屋証券取引所のネクスト市場も東証のグロース市場と同じく、国内のIPOを考えている企業のための市場です。
  
  
ネクスト市場では、メイン市場やプレミア市場に上場する企業に比べると比較的リスクが高い企業でも上場可能です。しかし、ネクスト市場に上場する企業には将来的に上位市場への市場変更を踏まえた事業計画があることや、その進捗や経営状況の適切な開示を行うことが求められています。
  
  
**【ネクスト市場の上場基準（一部抜粋）】**

|  |  |

## IPOを行うための審査基準と各項目とは

IPOを行うにあたって、各証券取引所はそれぞれ審査基準を設けており、上場に際し各審査基準をクリアすることが求められます。どの市場も共通しているのは、上場基準が大きく形式基準と実質基準に分けられることです。
  
  
各証券会社によって求められる項目やその基準に違いはありますが、ここでは、東証のグロース市場の上場基準をベースに解説します。

### グロース市場の形式基準

形式基準とは、主に数値で判断できる基準のことです。グロース市場の形式基準で主に求められることは、「流動性」です。
  
  
流動性とは、株式市場に流通している一般の投資家による売買が可能な株式の数のことです。株式市場に流通している株式の数が多いと流動性が高いとされます。グロース市場の場合、上場する企業として最低限の流動性が求められています。
  
  
公開されているグロース市場の形式基準は以下のとおりです。

  

|  |  |
| --- | --- |
| 項目 | 基準 |
| 株主数 | 150人以上 |
| 流通株式数 | 1,000単位以上 |
| 流通株式時価総額 | 5億円以上 |
| 流通株式比率 | 25％以上 |
| 株式の公募 | 500単位以上の公募の実施 |
| 事業継続年数 | 1年以上前からの取締役会の実施実績がある |
| 虚偽申請や不適正な意見の有無 | 監査報告書などで不適正や虚偽記載がない |
| 単元株式数 | 100株以上 |
| 株式の種類 | 指定された株式の種類である |
| 譲渡制限 | IPOに係る株式に譲渡制限がない |
| 指定振替機関 | 指定振替機関の取り扱い対象であること |

[出典：日本取引所グループ「上場審査基準」](https://www.jpx.co.jp/equities/listing/criteria/listing/02.html)
  

これらの形式基準が設けられているのは、一定以上の流動性を確保することで、株価の乱高下を防ぎ、投資家の利益を守るためです。

### グロース市場の実質基準

ここでは、実質基準で求められる項目について解説します。

#### 適切な情報開示

投資家が、投資の可否を適切に判断するために、上場企業には適時・適切に企業情報の開示を行うことが求められています。
  
  
開示する情報や適切な情報管理が行われているかは、主に以下の4つのポイントで判断します。

1. 企業経営に大きく関わる企業情報の適切な管理と適切なタイミングで適切な内容の情報開示が行われていること。また、インサイダー取引の防止対策が適切に行われていること。
2. 企業情報の開示内容が法令に沿っていること、および、投資判断に大きく関わる経営リスクなどについて分かりやすく提示されていること。
3. 関係者間の取引や株主の株式保有割合などの調整のために情報が故意に操作されていないこと。
4. 親会社などがある場合、IPOを希望する企業が適切に親会社の情報を得ることができ、かつ適切に投資家に情報開示を行えること。

[出典：日本取引所グループ「上場審査の内容（グロース市場）」](https://www.jpx.co.jp/equities/listing-on-tse/new/guide-new/nlsgeu000005p64a-att/nlsgeu000005p6ap.pdf)

#### 健全な企業経営

投資家が安心して株式の売買ができる株式市場の提供と投資家の利益保護のために、IPOを希望する企業は健全な企業経営を行っているかが確認されます。
  
  
健全な企業経営が行われているかどうかは、以下の3つのポイントで判断されます。

1. 取引に際して、特定の人や企業などが不当に利益を得る、または与えるなどの事実がないこと
2. 親族や他社の役員または職員が、IPOを希望する企業の役員だった場合、その職務を適切に遂行でき、また監査などが適切に実施できる状態であること
3. 親会社などがある場合、IPOを希望する企業が親会社から独立して経営判断が行えること

[出典：日本取引所グループ「上場審査の内容（グロース市場）」](https://www.jpx.co.jp/equities/listing-on-tse/new/guide-new/nlsgeu000005p64a-att/nlsgeu000005p6ap.pdf)

#### 適切なコーポレートガバナンスと内部管理体制の整備および運用

上場企業として適切なコーポレートガバナンスと内部管理体制が整っているかを審査されます。IPOの場合、その企業の事業規模や企業としての成熟度などに応じて適当であるかが判断されます。
  
  
適切なコーポレートガバナンスと内部管理体制が整っているかの審査のポイントは、以下の5つです。

1. 企業の役員が適正に業務を遂行するための企業体制が整っていること
2. 内部管理体制が適切に管理運用されていること
3. 内部管理体制維持のために必要な人員が配置されていること
4. 企業の現状にあった会計処理が適切に行われていること
5. 法令遵守のための社内体制が整えられており、法令違反防止対策も適切に運用されていること

[出典：日本取引所グループ「上場審査の内容（グロース市場）」](https://www.jpx.co.jp/equities/listing-on-tse/new/guide-new/nlsgeu000005p64a-att/nlsgeu000005p6ap.pdf)

#### 合理的な事業計画書の作成

上場後も安定して上場を維持し、投資家の利益を守るために、合理的に可能であると判断できる事業計画書を有していることが求められます。
  
  
提示されている事業計画書が適切かを判断するポイントは以下の2つです。

1. 事業内容、環境（内的または外的）、リスクなどを勘案して作成されていること
2. 事業計画書を実行するための経営基盤および事業体制が整っていると判断できること、または、整う見込みがあると判断できること

[出典：日本取引所グループ「上場審査の内容（グロース市場）」](https://www.jpx.co.jp/equities/listing-on-tse/new/guide-new/nlsgeu000005p64a-att/nlsgeu000005p6ap.pdf)

#### その他東証が必要と定める項目

公共の利益や投資家の利益を守るため、東証では、以下の7つのポイントについても審査が行われます。

1. 株主などに付与された権利が、他の株主や公共を保護するという点で適当であると判断できること
2. 事業経営や運営に大きく作用する争いがないこと
3. 企業の主力事業において、その事業の継続に左右する要因がないこと
4. 反社会的勢力による事業経営への干渉がないこと、および、関与を防止するための体制が適切に整備、運用されていること
5. IPO時に公開される株券に議決権がない（当株券以外にIPOを行う銘柄がない）、または少ない場合、別途定められている項目に当てはまること
6. IPO時に公開される株券に議決権がない（当株券以外にIPOを行う銘柄がある）場合、別途定められている項目に当てはまること
7. 公共の利益または投資家の利益に反すると認められる事項がないこと

[出典：日本取引所グループ「上場審査の内容（グロース市場）」](https://www.jpx.co.jp/equities/listing-on-tse/new/guide-new/nlsgeu000005p64a-att/nlsgeu000005p6ap.pdf)

## IPOを目指す企業に求められること

ベンチャー企業や新興企業が、IPOを行う際には「高い成長可能性」があると判断されなければなりません。
  
  
これは、ベンチャー企業向けの市場が上位市場よりも投資リスクが高いためです。投資リスクが高いからこそ、上場審査において企業に成長の見込みがあるのかどうかや、提示された事業計画書が実行可能であるのかが特に重視されます。
  
  
IPOが達成できれば、企業価値の向上により資金調達が容易になる一方で、継続した企業成長が求められることへのプレッシャーが大きいため、IPOが目標になってしまっている場合は注意が必要です。
  
  
また、IPOの後は企業運営や経営判断などに株主の意思を反映させる必要があるため、現状の経営に満足しており、経営の意思決定も自由に行いたい場合はIPOを選択しない方がよい場合もあります。
  
  
IPOを目指す場合、IPOを行う目的やビジョンを明確化することは重要です。IPOで得られる社会的信用、資金調達などのメリットはもちろん、IPOを行うために何をしなければいけないのか、また、どのようなデメリットがあるのかを把握する必要があります。

## まとめ

IPOは非上場企業にとって、社会的信用の獲得や企業を安定して運用していくための資金確保、経営基盤の確立などに有効な手段です。
  
  
IPOはメリットが大きい一方、経営判断に株主の意思を反映させなくてはならないこと、またIPO準備にかかるコストが大きいことは、一部の企業にとってはデメリットになります。
  
  
IPOの検討を始めたら、まずは企業の現状や中長期的な経営計画などを総合的に判断し、IPOのメリットやデメリットを含めて、企業にとっての最善を選択しましょう。

## よくある質問

### IPOとは？

IPOとは『Initial Public Offering』の略で、限定された株主にのみ株式の所有を認めていた未上場会社の株式（未公開株）を、はじめて証券取引所を通して株式市場に公開し、不特定多数の投資家に向けて株式を売り出すことをいいます。詳しくは記事内[IPOとは](https://www.freee.co.jp/kb/kb-ipo/about_ipo/#content1)をご覧ください。

### IPOのメリットとは？

IPOとはInitial Public Offering」の略で、限定された株主にのみ所有を認めていた未上場会社の株式（未公開株）を、初めて証券取引所を通して株式市場に公開し、不特定多数の投資家に向けて株式を売り出すことをいいます。詳しくは記事内[IPOするメリットとデメリット](https://www.freee.co.jp/kb/kb-ipo/about_ipo/#content2)をご覧ください。

## freeeで内部統制の整備をスムーズに

IPOは、スモールビジネスが『世界の主役』になっていくためのスタート地点だと考えています。
  
IPOに向けた準備を進めていくにあたり、必要になってくる内部統制。自社において以下のうち1つでも該当する場合は改善が必要です。

* バックオフィス系の全てのシステムにアクセス権限設定を実施していない
* 承認なく営業が単独で受注・請求処理を行うことができる
* 仕入計上の根拠となる書類が明確になっていない

freee会計のエンタープライズプランは内部統制に対応した機能が揃っており、効率的に内部統制の整備が進められます。

内部統制対応機能

* **不正防止（アクセスコントロール）**のための、特定IPアドレスのみのアクセス制限
* **整合性担保（インプットコントロール）**のための、稟議、見積・請求書発行、支払依頼などのワークフローを用意
* **発見的措置（モニタリング）**のための、仕訳変更・承認履歴、ユーザー情報更新・権限変更履歴などアクセス記録
* 国際保証業務基準3402（ISAE3402）に準拠した**「SOC1 Type2 報告書」を受領**

詳しい情報は、[内部統制機能のページ](https://www.freee.co.jp/solution/ipo/internal-control/)をご確認ください。
