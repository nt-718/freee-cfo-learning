## 高年齢雇用継続給付金とは

**「高年齢雇用継続給付金」とは、60歳到達等時点に比べ、賃金が75％未満に低下した状態で働き続ける60歳以上65歳未満の雇用保険被保険者へ支払われる給付金です。**
  
  
高年齢雇用継続給付金には、以下の2種類があります。

高年齢雇用継続給付金の種類

● 高年齢雇用継続基本給付金：60歳到達後も継続して雇用される人が対象  
● 高年齢再就職給付金：一度離職し基本手当を受給した後、60歳以降に再就職した人が対象

どちらも高齢者の就業意欲を維持・喚起し、雇用継続を支援・促進することが目的の制度です。

### 高年齢雇用継続給付の支給要件

高年齢雇用継続給付金の支給要件は以下です。

高年齢雇用継続給付の支給要件

● 60歳以上65歳未満で雇用保険の一般被保険者である  
● 被保険者だった期間が5年以上ある  
● 原則、60歳到達時点と比較して60歳以降の賃金月額が75%未満になった  
● 再就職の前日時点で基本手当の支給算日数が100日以上ある（高年齢再就職給付金）

## 雇用保険制度改正による高年齢雇用継続給付金の変更点

高年齢雇用継続給付金は、雇用保険制度の改正で一部変更されます。高齢者を雇う事業主は今後の変更点を認識しておきましょう。

### 2025年4月から給付率を10%に縮小

2020年の雇用保険制度改正で高年齢雇用継続給付金の給付率が見直され、2025年4月から、10%へ上限が引き下げられる予定です。
  
  
現在の高年齢雇用継続給付金の支給額は、原則として60歳以降の各月に支払われた賃金の15%が上限なので、5％引き下げられます。

### 70歳まで就業確保措置を雇用安定事業に位置づけ

高年齢者雇用安定法の一部改正で、2021年4月からは70歳までの就業機会確保が、雇用者側の努力義務になりました。
  
  
努力義務ではあるものの、70歳までの定年引き上げや、定年制度の廃止などの対策が求められます。

## 高年齢雇用継続給付金の計算方法

高年齢雇用継続給付金は、賃金の低下率で計算方法が異なります。
  
  
また、賃金が低下した理由次第でみなし賃金が算定されるほか、支給限度額・最低限度額も設定されています。

### 賃金の低下率で計算方法が異なる

高年齢雇用継続給付金は、「賃金の低下率61%以下」の場合と「61%を超えて75%未満」の場合とで、支給額の計算方法が変わります。
  
  
賃金の低下率の計算式は以下の通りです。

【賃金の低下率の計算式】  
賃金の低下率＝支給対象月に支払われた賃金の金額÷賃金月額×100

なお、低下率を求める際の月額賃金は、60歳に到達する前の6ヶ月間の平均賃金が原則で、受給資格認定通知書や支給決定通知書に記載されています。
  
  
月ごとの賃金から低下率に応じた金額が支給されるため、高年齢雇用継続給付金は毎月一律ではありません。支給額の計算式は以下の通りです。

【低下率が61%以下だった場合の支給額の計算式】  
支給額＝支給対象月に支払われた賃金の金額×15%

【低下率が61%を超えて75%未満だったときの支給額の計算式】  
支給額＝183÷280×支給対象月に支払われた賃金の金額+137.25÷280×賃金月額

### 賃金が低下した理由次第ではみなし賃金が算定される

低下した賃金への補填が雇用保険で行われるべきでない場合、低下した賃金は「みなし賃金」に算定されて支給額が計算されます。
  
  
雇用保険で補填されるべきでないと判断される賃金低下の理由は、以下の通りです。
  
  
● 被保険者自身の懲戒や持病・傷病  
● 出産・育児・介護などでの欠勤・遅刻・早退  
● 勤め先の休業　など
  
  
上記の理由で賃金が低下した場合、みなし賃金として算定され、みなし賃金分も賃金が支払われたものとして低下率が計算されます。
  
  
単純に賃金がいくら低下しているかだけでなく、なぜ低下したのかも支給額に影響します。

### 支給限度額と最低限度額がある

高年齢雇用継続給付金には、支給限度額364,595円と最低限度額2,125円が設定されています。
  
  
支給対象月に支払われた賃金が支給限度額の364,595円以上の場合、賃金の低下率に関わらず高年齢雇用継続給付金は支給されません。
  
  
また、支払われた賃金と算定された支給額の合計が支給限度額の364,595円を超える場合は、364,595円から賃金を差し引いた金額が支給されます。
  
  
算定された支給額が最低限度額2,125円以下の場合も、高年齢雇用継続給付金は支給されません。

|  |  |  |
| --- | --- | --- |
| 条件 | 支給されるか | 金額 |
| 賃金が364,595円以上 | 支給されない |  |
| 支給額が2,125円以下 | 支給されない |  |
| 賃金+支給額が364,595円を超える | 支給される | 364,595円 - 賃金 |

## 高年齢雇用継続給付金の申請手順

高年齢雇用継続給付金を申請する際、雇用者側が対応する手続きの手順を解説します。
  
  
高年齢雇用継続基本給付金と高年齢再就職給付金とでは、申請手順に異なる点があるため注意しましょう。

### 高年齢雇用継続基本給付金の申請手順

高年齢雇用継続基本給付金の申請手順は、以下の通りです。

高年齢雇用継続基本給付金の申請手順

1. 被保険者が受験資格確認票と初回申請書を記入して雇用者に提出  
2. 雇用者が受験資格確認票と初回申請書に賃金証明書を添えてハローワークに提出  
3. ハローワークから雇用者に受給資格確認通知書と支給（不支給）決定通知書、次回分の支給申請書が交付される  
4. 雇用者が受給資格確認通知書と支給（不支給）決定通知書、次回分の支給申請書を被保険者に渡す  
5. ハローワークから被保険者に給付金が支給される

上記のように、申請手続きは雇用者を経由して行い、給付金は被保険者に直接支給されます。
  
  
なお、2回目以降の申請には賃金証明書の添付は不要です。被保険者から記入済の支給申請書を受け取り、2ヶ月に一度、指定された月にハローワークに提出します。

### 高年齢再就職給付金の申請手順

高年齢再就職給付金の申請手順は、以下の通りです。

高年齢再就職給付金の申請手順

1. 被保険者が受験資格確認票と初回申請書を記入して雇用者に提出  
2. 雇用者が受験資格確認票と初回申請書をハローワークに提出  
3. ハローワークから受給資格確認通知書と支給申請書が雇用者に交付される  
4. 雇用者は受給資格確認通知書と支給申請書を被保険者に渡す  
5. 被保険者は支給申請書を記入し、雇用者に提出する  
6. 雇用者は支給申請書をハローワークに提出する  
7. ハローワークから雇用者に支給（不支給）決定通知書と次回分の支給申請書が交付される  
8. 雇用者は被保険者に支給（不支給）決定通知書と次回分の支給申請書を渡す  
9. ハローワークから被保険者に給付金が支給される

高年齢再就職給付金も高年齢雇用継続基本給付金と同様に、申請手続きは雇用者を経由して行い、給付金は被保険者に直接支給されます。
  
  
2回目以降は、雇用者が被保険者に記入してもらった支給申請書を指定された月の申請日までにハローワークへ提出します。

## 高年齢雇用継続給付金で雇用者が知っておきたいポイント

**労働者が高年齢雇用継続給付金を受け取る場合、併給できない給付金があることと申請期限があることを、雇用者側は覚えておきましょう。**

### 高年齢再就職給付金と再就職手当は併給できない

高年齢再就職給付金と再就職手当、両方の受給条件を満たす場合、どちらか一方しか受け取れません。
  
  
どちらを受け取るかは雇用者ではなく、労働者本人が選択します。

### 高年齢雇用継続給付金には申請期限がある

高年齢雇用継続給付金には、申請期限は支給対象月の初日から起算して4ヶ月以内です。
  
  
給付金はハローワークから労働者へ直接支払われますが、申請手続きは勤め先を通じて行うため、雇用者は忘れずに手続きしなければなりません。

### 初回申請後は指定された期限までに支給申請書を提出する

高年齢雇用継続給付金は、初回の支給申請をした後も、定期的に支給申請書を提出する必要があります。
  
  
支給申請書はそれぞれ提出期限が定められています。

|  |  |
| --- | --- |
|  | 提出期限 |
| 高年齢雇用継続基本給付金 | 2ヶ月ごとの定められた期間に提出 |
| 高年齢再就職給付金 | 指定された支給申請月の支給申請日までに提出 |
