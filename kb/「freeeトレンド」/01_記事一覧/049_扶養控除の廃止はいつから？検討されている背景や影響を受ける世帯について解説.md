## 扶養控除とは？

**扶養控除は所得控除のひとつであり、納税者に所得税法上の控除対象扶養親族がいる場合に適用されます。**扶養親族の年齢や同居の有無などに応じて、課税対象となる所得から一定の控除額を差し引きます。
  
  
扶養控除で差し引かれる金額や対象となる親族の要件を説明します。

### 扶養控除で控除される金額

扶養控除で控除される金額は、以下のとおりです。

  

|  |  |  |
| --- | --- | --- |
| 扶養親族の区分 | | 控除額 |
| 一般の控除対象扶養親族 | | 380,000円 |
| 特定扶養親族 | | 630,000円 |
| 老人扶養親族 | 同居老親等以外の者 | 480,000円 |
| 同居老親等 | 580,000円 |

[出典：国税庁「No.1180 扶養控除」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1180.htm)

特定扶養親族とは、その年の12月31日時点で19歳以上23歳未満の控除対象扶養親族をいいます。老人扶養親族は、その年の12月31日時点で70歳以上の人が該当します。
  
  
老人扶養親族の「同居老親等」とは、納税者または配偶者の直系尊属（父母、祖父母など）で、納税者や配偶者と同居を常としている人です。病気の治療により入院している場合、その期間が1年以上であっても同居として扱います。ただし老人ホームなどへ入所している場合は、同居として認められません。
  
  
また、2025年の税制改正により、2025年度以後の所得税に対して適用される「特定親族特別控除（仮称）」が新設される見込みです。
  
  
特定親族特別控除（仮称）では、生計をともにする19歳以上23歳未満の扶養親族（青色事業専従者や配偶者は除く）の年間の合計所得金額が58万円以上123万円以下の場合に、以下の金額が控除されます。

| 親族等の合計所得金額 | 控除額 |
| --- | --- |
| 58万円以上85万円以下 | 63万円 |
| 85万円以上90万円以下 | 61万円 |
| 90万円以上95万円以下 | 51万円 |
| 95万円以上100万円以下 | 41万円 |
| 100万円以上105万円以下 | 31万円 |
| 105万円以上110万円以下 | 21万円 |
| 110万円以上115万円以下 | 11万円 |
| 115万円以上120万円以下 | 6万円 |
| 120万円以上123万円以下 | 3万円 |

[出典：財務省「令和７年度税制改正の大綱（１/９）」](https://www.mof.go.jp/tax_policy/tax_reform/outline/fy2025/07taikou_01.htm)

### 扶養親族の範囲

控除の対象となる扶養親族は、次の5つの要件すべてを満たさなければなりません。

**【控除対象となる扶養親族の要件】**

  

* 配偶者以外の親族、または都道府県知事から養育を委託された児童や、市町村長から養護を委託された老人である
* 納税者と生計を一にしている
* 年間の合計所得金額が58万円以下（2024年分以前は48万円以下）である
* 青色申告者の事業専従者としてその年を通じて一度も給与の支払を受けていないこと、または白色申告者の事業専従者でない
* その年の12月31日現在の年齢が16歳以上である

  

[出典：国税庁「No.1180 扶養控除」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1180.htm)

なお、2025年度税制改正により、扶養親族の所得が給与のみの場合、給与収入は「123万円以下」が基準となる見込みです。

## 扶養控除は引き下げが検討されている

少子化対策のひとつとして「児童手当の拡充」が進められている一方、扶養控除の廃止が検討されていました。
  
  
しかし、令和6年度税制改正大綱では、一般の控除対象扶養親族（16歳から18歳）を対象とした扶養控除に関して、控除を廃止するのではなく引き下げる案が示されました。
  
  
具体的には、**所得税の控除額を38万円から25万円に、住民税の控除額を33万円から12万円に引き下げる**案が議論されています。

  

| 項目 | 控除額 |
| --- | --- |
| 所得税の控除額 | 38万円から25万円 |
| 住民税の控除額 | 33万円から12万円 |

  

0歳から18歳までの子どもには児童手当、16歳から22歳の子どもには所得税・住民税の扶養控除による税制優遇を設けているのが現行制度です。
  
  
扶養控除が引き下げられると、納める税金（所得税・住民税）が増えます。そのため、児童手当の受取額が増えたとしても、年収などによっては、実質の手当額は少なくなる可能性があります。
  
  
扶養控除の引き下げに関しては、2025年以降に決定するとされているため、子育て世帯の人は、扶養控除や児童手当に関する議論の動向に注意しましょう。

## 児童手当の拡充

政府は「2030年代に入るまでが、加速する少子化を止めるラストチャンス」と考えています。
  
  
具体的政策である「加速化プラン」のひとつとして実施されているのが「児童手当の拡充」です。
  
  
「次代を担うすべての子どもの育ちを支える基礎的な経済支援」という考えに基づいて、以下のように変更されました。

**【児童手当の拡充】**

  

* 所得制限の撤廃
* 支給期間の延長（18歳になってから最初の3月31日まで）
* 支給回数の増加（偶数月に2ヶ月分を支給）
* 子どもの人数が多い世帯への支援（第3子以降は3万円）

## 扶養控除の引き下げによる影響は？

扶養控除が引き下げられると、16歳から18歳の扶養親族がいる場合の所得税と住民税の控除額が減少し、納税の負担が増えます。年収が高くなるにつれて、扶養控除引き下げの影響が大きくなるでしょう。
  
  
なお、扶養控除の引き下げ対象となる16歳から18歳の年代は、2024年12月から児童手当の対象となりました。控除が引き下げられた一方で、一人当たり年間12万円（第3子以降は年間36万円）が支給されます。

## まとめ

児童手当の拡充に関する議論が進む一方、扶養控除は引き下げが検討されています。
  
  
児童手当における所得制限の撤廃や第3子以降の増額などにより子育て世帯への経済的支援はあるものの、扶養控除が引き下げられると、所得税や住民税の負担が増えます。年収によっては、負担が増加する可能性があります。
