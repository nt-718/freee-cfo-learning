## 電子帳簿保存法とは

**電子帳簿保存法とは、経理のデジタル化に対応するため、決算関係書類や各種帳簿などの税務関係帳簿・書類を電子データで保存することを定めた法律です。**保存方法には「電子帳簿等保存」「スキャナ保存」「電子取引データ保存」の3つがあり、保存にあたってはそれぞれ要件が定められています。

| **保存区分** | **概要** |
| --- | --- |
| **電子帳簿等保存** | 仕訳帳や損益計算書、請求書などの税法上保存が必要な帳簿･書類をパソコンなどで作成した場合、プリントアウトせずにデータのまま保存できる |
| **スキャナ保存** | 紙の領収書･請求書など、その書類自体を保存する代わりにスマホやスキャナで読み取った電子データとして保存できる |
| **電子取引データ保存** | 税法上、帳簿･書類を保存する義務のある人（企業）が、注文書･契約書･送り状･領収書･見積書･請求書などに関する電子データをやり取りした場合、その電子データを保存しなければならない |

## 電子帳簿保存法に対応したシステムの機能

電子帳簿保存法に則った保存をするためには、3つの区分における保存要件の違いを把握し、それぞれに応じた対応が求められます。保存要件には検索要件や保管方法などのさまざまなルールがあり、それら全てに対応した体制をイチから構築するのは多くの労力がかかります。
  
  
**そこで、電子帳簿保存法に対応したシステムを導入することで、効率よく保存要件に則った状態でデータの管理が行えます。**
  
  
電子帳簿保存法に対応しているシステムの主な機能は、以下のとおりです。

| **機能** | **詳細** |
| --- | --- |
| **書類の登録** | ・PDFなどの書類データをアップロードすると、システム上に登録され保存できる |
| **OCRによるデータ化** | ・書類の登録と同時に文字情報を読み取ってデータ化する機能  ・AIによる学習を重ねて、文字の認識結果から自動仕訳を行うものもある |
| **日付や金額による検索** | ・「取引年月日」「取引金額」「取引先」など、後から項目別に検索できる |
| **タイムスタンプの付与** | ・システムの登録時にタイムスタンプの付与が行われる  ・「スキャナ保存」や「電子取引データ保存」の場合、タイムスタンプなどで文書がその時点で存在していたこと（存在性）、改ざんできない文書であること（非改ざん性）を証明する必要がある |

## 電子帳簿保存法に対応するシステムを導入するメリット

企業が電子帳簿保存法に対応するシステムを導入することで、以下のようなメリットが得られます。

電子帳簿保存法に対応するシステムを導入するメリット

* 業務の効率化が図れる
* コスト削減につながる
* 内部統制の強化につながる

それぞれ詳しく解説します。

### 業務の効率化が図れる

電子帳簿保存法対応のシステムを用いずに文書を保存するとなると、改ざん防止のための措置として事務処理規程を定めたり、検索要件を満たした状態で保存するための社内ルールの構築や運用が必要です。
  
  
**システムの導入を導入することで、これらの労力をかけずに電子帳簿保存法の保存要件に則り文書を保存できます。**

  

[出典：国税庁「電子取引データの保存方法をご確認ください」](https://www.nta.go.jp/law/joho-zeikaishaku/sonota/jirei/tokusetsu/pdf/0023006-085_01.pdf)

### コスト削減につながる

電子帳簿保存法に対応しない場合は、すべての文書を紙で管理することとなります。仕訳帳などの帳簿や取引先から紙で受け取った書類などを紙で保存しなくてはならず、紙の保管スペースの確保や、保管場所に出向いて文書を確認することとなりさまざまなコストが発生します。**電子帳簿保存法に対応したシステムを導入することで、紙の保管にかかるコストを削減できます。**
  
  
さらにOCR機能を搭載したシステムを導入であれば、人がデータを打ち込む作業がなくなり、人件費の削減にもつながるためコスト削減に大きく貢献します。

### 内部統制の強化につながる

**タイムスタンプの付与が行えるシステムを用いることで、データの存在性と非改ざん性の証明ができます。**これは電子帳簿保存法に対応するだけでなく、社内での書類の改ざんや差し替えリスクを防ぎ、書類の管理体制を強化することにもつながります。
  
  
電子帳簿保存法に対応した多くのシステムは、クラウド上などで電子データのバックアップが取れます。これにより、災害による書類の紛失や破損のリスクもなくなり、さらなる内部統制の強化が期待できるでしょう。

## おすすめの電子帳簿保存法対応システムのタイプ

**電子帳簿保存法に対応したシステムはさまざまありますが、自社のニーズや課題によって導入すべきシステムのタイプは異なります。**特徴を押さえて自社に適したシステムを導入しましょう。
  
  
以下では、主な電子帳簿保存法に対応したシステムのタイプを解説します。

### 幅広い帳簿類に対応できるタイプ

請求書や領収書、契約書、発注書など幅広い書類を電子保存できるタイプのシステムであれば、社内のペーパーレス化を促進できます。**事業規模の大きな企業や、取引に関わる書類すべてをシステムで一元管理したい企業におすすめです。**

### 請求書に特化したタイプ

**大量の請求書を電子化したい企業は、請求書受領サービスがおすすめです。**紙の請求書をOCRで読み取り、データ入力から保管までしてくれるシステムが多く、経理に時間や人数をかけずに済みます。

### 領収書等の経費精算タイプ

**経費精算業務を効率的に行いつつ電子化も進めたい企業は、経費精算システムがおすすめです。**経理担当者だけではなく、経費申請をする従業員の業務負担も減らすことができます。

## 電子帳簿保存法に対応するシステムの選び方

電子帳簿保存法に対応するシステムを選ぶ際に押さえておきたいポイントを5つ紹介します。システムの導入後に失敗したと後悔しないためにも、それぞれ押さえておきましょう。

### ①JIIMA認証のものであるか

**JIIMA認証とは、公益社団法人日本文書情報マネジメント協会が電子帳簿保存法対応の法的要件を満たしているものに与える認証制度です。**
  
  
電子帳簿保存法に対応したシステムを導入したい場合は、JIIMA認証であるかどうかを確認するとよいでしょう。JIIMA認証を満たした製品には、次のようなロゴが表示されています。

### ②自社の課題を解決できるか

電子帳簿保存法に対応したシステムのなかには、請求書の受領や経費精算など、特定の業務に特化したシステムがあります。**まずは自社のどの業務を効率化したいのかを洗い出し、その業務に対応するシステムを選ぶことが大切です。**
  
  
また、今後の会社の成長や事業の多角化を見据えて、 柔軟に拡張・変更できるシステムを選ぶと良いでしょう。

### ③社内の現行システムと連携できるか

すでに社内で導入しているシステムやツールがある場合、それらと連携できるのかを確認して選ぶのもひとつの方法です。**代表的な連携方法として「Web APIでの直接連携」と「CSVファイル経由」があります。**
  
  
「Web APIでの直接連携」とは、システム同士で直接やり取りをして即座に連携する方法です。一方、「CSVファイル経由」での連携は、データをファイルに書き込んでやり取りをします。自社はどの方法で連携するほうがよいのかを確認してから、システムを選ぶようにしてください。

### ④操作は簡単か

システムの操作性も重要なポイントです。新しいシステムを導入する際は、社内での定着が課題となります。簡単に操作ができるシステムであれば、従業員もすぐに使い慣れて定着していくでしょう。
  
  
また、人事異動や採用によって新たな従業員が経理を担当する場合、操作が簡単なシステムであれば、研修や教育の手間が省けて、 スムーズに対応できるでしょう。

### ⑤予算に見合った料金体系か

電子帳簿保存法に対応したシステムは、サービスごとに料金体系もさまざまです。クラウドシステムを導入する場合は、初期費用だけでも数十万円かかり、月額10万円を超えるランニングコストが発生するものもあります。
  
  
また、基本料金が少額であっても、利用量によって加算される従量課金タイプなどもあります。自社に必要な機能と予算を把握してシステムを利用しましょう。
