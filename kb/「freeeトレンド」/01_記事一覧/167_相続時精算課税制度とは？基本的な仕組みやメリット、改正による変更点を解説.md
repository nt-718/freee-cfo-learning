## 相続時精算課税制度とは

**相続時精算課税制度は、贈与税の制度です。**原則60歳以上の父母または祖父母などから、18歳以上の子または孫に財産を贈与したときに選択できます。
  
  
通常、金銭などを贈与した場合は暦年課税となり、年間110万円の基礎控除額を超えた部分には贈与税が課税されます。
  
  
相続時精算課税制度が導入される前は、子や孫への贈与には年間110万円の基礎控除額内に抑える必要がありました。
  
  
しかし、課税されない金額に制限があると、十分な財産の移転ができません。教育や結婚、出産などで出費の多い子どもや孫世代の負担が増える可能性があります。
  
  
このような課題への問題意識から、相続時精算課税制度が制定されました。
  
  
相続時精算課税制度は、財産の贈与時に制度を選択することで最大2,500万円の特別控除を適用できます。
  
  
しかも、この特別控除額は複数年にわたって活用できる控除額です（前年より前に特別控除額を控除している場合には、その金額を除いた残額が限度額）。
  
  
相続時精算課税制度の選択をすると、暦年での課税とはならず、累積2,500万円までの贈与に対して非課税とすることができます。
  
  
一方、特別控除を適用した相続時精算課税制度の贈与は、贈与税を非課税とする代わりに、相続財産に合計して相続税が課されます。
  
  
相続時精算課税制度は、贈与税と相続税を通算して納税する制度です。
  
  
なお、2023年度の税制改正により、相続時精算課税制度に基礎控除額の新設や土地または建物の価額の特例の創設がなされました。
  
  
この改正は2024年1月1日から施行され、より制度が利用しやすくなっています。
  
  
【関連記事】[贈与税の改正内容は？ 2023年度に生前贈与加算と相続時精算課税制度が変わる！](https://www.freee.co.jp/kb/kb-trend/gift-tax-reform/)

### 相続時精算課税制度の対象者

相続時精算課税制度は誰でも選択できるわけではありません。贈与する人（贈与者）と贈与を受ける人（受贈者）の双方に以下の要件が設けられています。

|  |  |
| --- | --- |
| 贈与する人 | 財産を贈与した年の1月1日時点で60歳以上の父母または祖父母 |
| 贈与を受ける人 | ●　財産の贈与を受けた年の1月1日時点で18歳以上の人  ●　贈与する人の直系卑属（子や孫など）である推定相続人、または孫 |

[出典：国税庁「No.4103 相続時精算課税の選択](https://www.nta.go.jp/taxes/shiraberu/taxanswer/sozoku/4103.htm)
  
  
上記のように、相続時精算課税制度を選択するためには年齢要件を満たす必要があります。父母または祖父母から子または孫への贈与に限られる点に注意しましょう。
  
  
ただし、以下の2つの適用にかかる贈与の場合、贈与する人が贈与した年の1月1日で60歳以上であれば、贈与を受ける人が推定相続人以外であっても相続時精算課税制度を適用できます。

対象者の例外

●　非上場株式等についての贈与税の納税猶予及び免除の特例（措法70の7の5）  
●　個人の事業用資産についての贈与税の納税猶予及び免除（措法70の6の8）

上記は、事業承継で財産を推定相続人以外に贈与する場合などに適用される特例です。中小企業の経営承継が円滑的に進むように、例外的に認められています。

### 相続時精算課税制度の手続き・必要書類

相続時精算課税制度を利用するには、最初の贈与を受けた翌年の2月1日から3月15日までの間に、贈与税の申告書とともに「相続時精算課税選択届出書」を提出します。主な必要書類は以下の通りです。

相続時精算課税制度の手続きに必要な書類

●　贈与税の申告書  
●　相続時精算課税選択届出書  
●　贈与を受ける人の戸籍謄本または抄本、そのほかの書類  
●　申告する人の本人確認書類の提示または写しの添付

「贈与を受ける人の戸籍謄本または抄本、そのほかの書類」では、贈与を受ける人の氏名や生年月日、贈与を受ける人が贈与する人の直系卑属（子や孫など）である推定相続人または孫であることが確認されます。
  
  
また、マイナンバー制度の導入に伴い、書類を提出する際にはマイナンバーカードや運転免許証などの本人確認書類の提示が必要です。

### 相続時精算課税制度と暦年課税の違い

財産の贈与を受けた場合、「相続時精算課税制度」と「暦年課税」のどちらかを選択します。それぞれの違いは以下の通りです。

|  |  |  |
| --- | --- | --- |
|  | 相続時精算課税制度 | 暦年課税 |
| 税の計算 | 贈与者ごとの累積贈与額が特別控除額（2,500万円）を超えた場合に、超えた金額に税率が適用 | 1年間の贈与額から基礎控除額（110万円）を控除した額に税率が適用 |
| 税率 | 一律20％ | 10％～55％の累進税率 |
| 贈与者と受贈者 | 60歳以上の父母または祖父母から18歳以上の子または孫 | だれでも可能 |
| 申告方法 | ・初年度に相続時精算課税選択届出書を提出  ・贈与があった年に申告 | 基礎控除額110万円を超えた年はそのつど申告 |

  
上記のように、相続時精算課税制度は累積贈与額が2,500万円を超えなければ課税されません。
  
  
また、超えた場合の税率は一律20％であり、10～55％の累進税率である暦年課税とは適用される税率が異なります。
  
  
なお、2023年の税制改正により、暦年課税のみ設けられていた基礎控除額が相続時精算課税制度でも適用されます。
  
  
改正の内容は後述しているので、あわせてご確認ください。

## 相続時精算課税制度のメリット

相続時精算課税制度を選択すると、以下のメリットを受けられます。

相続時精算課税制度のメリット

●　累積2,500万円までの控除を受けられる  
●　早い段階で財産を引き継げる  
●　贈与者ごとに制度を選択できる

各メリットの内容を解説します。

### 累積2,500万円までの控除を受けられる

相続時精算課税制度のメリットは、累積2,500万円までの大きな控除が受けられる点です。
  
  
たとえば、祖父から孫へ2,000万円を贈与した場合、暦年課税では110万円の基礎控除額と金額に応じた控除額しか適用されません。
  
  
さらに、贈与額が大きくなるごとに、適用される税率も増えます。
  
  
●暦年課税の税率（特例税率）

|  |  |  |
| --- | --- | --- |
| 基礎控除後の課税価格 | 税率 | 控除額 |
| 200万円以下 | 10％ | ― |
| 400万円以下 | 15％ | 10万円 |
| 600万円以下 | 20％ | 30万円 |
| 1,000万円以下 | 30% | 90万円 |
| 1,500万円以下 | 40％ | 190万円 |
| 3,000万円以下 | 45％ | 265万円 |
| 4,500万円以下 | 50％ | 415万円 |
| 4,500万円超 | 55％ | 640万円 |

[出典：国税庁「No.4408 贈与税の計算と税率（暦年課税）」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/zoyo/4408.htm)
  
  
2,000万円を贈与した場合、基礎控除額110万円を差し引くと1,890万円となり、3,000万円以下の税率45％が適用されます。
  
  
一方、相続時精算課税制度では累積2,500万円までの特別控除を適用できます。
  
  
祖父から孫へ2,000万円を贈与した場合も、それ以外の贈与がなければ特別控除の金額内となり、贈与税が課されない点は大きなメリットです。

### 早い段階で財産を引き継げる

相続時精算課税制度は早い段階で財産を引き継げる点もメリットです。
  
  
財産を引き継ぐ代表的な例には「相続」があり、相続税では「3,000万円＋600万円×法定相続人の数」という比較的大きな基礎控除額が設けられています。
  
  
親世代の財産の一定部分を非課税にし、子ども世代の税負担を軽減するためです。
  
  
しかし、相続税の基礎控除額の恩恵を受けられるのは、相続する人が亡くなったあとです。
  
  
相続する人が存命中に財産を引き継ぐと、「贈与」となって暦年課税に応じた贈与税が課されます。
  
  
一方、相続時精算課税制度を活用すると、生前から累積2,500万円までの控除が受けられます。
  
  
住宅購入や進学費用など、子や孫にまとまったお金が必要となるタイミングで財産を引き継ぐことも可能です。
  
  
相続時精算課税制度は相続発生時に子や孫へ引き継ぐ財産を、必要なときに渡したい場合に役立つ制度です。

### 贈与者ごとに制度を選択できる

相続時精算課税制度は、贈与する人ごとに制度を選択できます。
  
  
たとえば、父からの贈与は相続時精算課税制度を選択し、母からの贈与は暦年課税を選択する方法も法律上可能です。
  
  
そのほか、父からの贈与と母からの贈与の両方で、相続時精算課税制度を選択することもできます。
  
  
この場合、父からの贈与と母からの贈与それぞれで2,500万円の特別控除が適用されるため、合計すると5,000万円の特別控除が受けられます。

## 相続時精算課税制度のデメリット

相続時精算課税制度はメリットがある一方、いくつかのデメリットも存在します。主なデメリットは以下の通りです。

相続時精算課税制度のデメリット

●　選択後は暦年課税へ変更できない  
●　相続税が発生する可能性がある

メリットとデメリットの両方を把握したうえで、制度を利用しましょう。

### 選択後は暦年課税へ変更できない

相続時精算課税制度を選択してしまうと、選択した以降の贈与は贈与する人が亡くなるまで適用されます。
  
  
途中で暦年課税に変えたいと思っても、変更はできないので注意しましょう。
  
  
相続時精算課税制度を利用して生前に贈与するか、相続時に財産を相続するか、どちらにメリットがあるかは個人により異なります。
  
  
ご自身にとってどちらが有利に働くのかをよく考えたうえで、制度を利用しましょう。

### 相続税が発生する可能性がある

贈与を受けた財産は、将来的に相続が発生した場合に、相続財産と足し戻して相続税が課されます。
  
  
贈与財産と相続財産を足し戻した金額が相続税の基礎控除額を超えると、相続税が発生する点に注意が必要です。
  
  
相続時精算課税制度は、言い換えれば「贈与税・相続税を通じた課税」が行われる制度です。
  
  
相続税の控除に相続時精算課税制度による贈与税の控除がプラスされるわけではない点は、覚えておきましょう。

## 相続時精算課税制度の改正による変更点

2023年度の税制改正では、相続時精算課税制度に新たな内容が加わりました。2024年1月1日より施行される改正の内容は以下の通りです。

改正による変更点

●　年110万円の基礎控除が新設された  
●　土地または建物の価額の特例が創設された

改正による変更の詳細を解説します。

### 年110万円の基礎控除が新設された

2023年度の税制改正により、相続時精算課税制度を選択した場合、年110万円の基礎控除が適用されます。
  
  
これまで暦年課税では年110万円の基礎控除額が認められていました。しかし、相続時精算課税制度では、年単位の基礎控除額は設けられていませんでした。
  
  
2024年1月1日以降は、相続時精算課税制度でも暦年課税の基礎控除とは別に、基礎控除額が適用されます。
  
  
年110万円の基礎控除が適用されるのは、2024年1月1日以降の贈与分です。
  
  
2023年12月31日以前に相続時精算課税制度を選択した人も、2024年1月1日以降の贈与には基礎控除額が適用されます。
  
  
また、相続時精算課税では、相続時精算課税選択届出書を提出後に贈与があった場合、金額に関わらず申告が必要でした。
  
  
基礎控除額の創設により、2024年1月1日以降は、受け取る金額の合計が年間あたり110万円を超えない贈与の場合、申告は不要です。

### 土地または建物の価額の特例が創設された

2023年度の税制改正では、土地または建物が被災した場合の特例が創設されました。
  
  
相続時精算課税制度では、相続時に相続税の課税対象となる贈与財産額は、贈与時の金額を基に計算されます。
  
  
2024年1月1日以降は、贈与で取得した土地や建物が災害により一定の被害を受けた場合、災害による被災価額を控除できます。
  
  
上記の災害とは、震災や風水害などの自然現象による災害・火災や公害などの人為災害・害虫や害獣などの生物災害などが対象です。
  
  
土地または建物の価額の特例を受けるには、原則災害発生日から3年が経過する日までに「り災証明書」などを添付して、所轄税務署長に提出します。
  
  
また、災害減免法による贈与税の軽減などを受けた場合、本特例の適用を受けられないので注意しましょう。

## まとめ

相続時精算課税制度を活用すると、生前時の子や孫への贈与に適用できる控除額が大きくなります。
  
  
相続時精算課税制度は、子や孫が結婚・出産・起業・住居建築など、財産を必要なタイミングで引き継ぐのに役立つ制度です。
  
  
ただし、暦年課税と相続時精算課税制度の選択でどちらが有利となるかは、個人の状況で異なります。
  
  
制度のメリットとデメリットを把握したうえでの選択が大切です。
