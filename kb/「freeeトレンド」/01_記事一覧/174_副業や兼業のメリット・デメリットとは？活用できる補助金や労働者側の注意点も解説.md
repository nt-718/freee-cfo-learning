従業員が副業や兼業をすれば、スキルアップや企業の生産性向上などのメリットがある一方で、情報漏えいリスクなどのデメリットがあることも理解しておかなければいけません。

## 副業と兼業の違い

副業と兼業は、仕事を掛け持ちしている点では同じですが、副業の場合は本業ではなく副次的に行う仕事を指します。掛け持ちしている複数の仕事の間に正副の優先順位をつけているのが特徴です。
  
  
一方で兼業の場合は、仕事を掛け持ちしていることを指す見解が一般的であり、掛け持ちしている複数の仕事の間に優先順位はありません。つまり、副業と兼業の違いは「複数の仕事の間に優先順位をつけているかどうか」です。
  
  
なお、収入アップやスキルアップ、キャリア形成など、さまざまな目的で仕事を掛け持ちする点は共通しており、近年は副業や兼業を希望する人が増えています。

## 副業や兼業が推進される理由

近年は副業や兼業を希望する人が増える一方で、副業や兼業を認める企業はいまだに少ない状態です。**働く人の視点に立った働き方改革を進め、柔軟な働き方や人材の流動化などを図るために、国が副業・兼業を推進する姿勢を打ち出しています。**
  
  
2017年に政府主導の会議で策定された「働き方改革実行計画（概要）」には以下の記載があり、国として副業や兼業を推進する姿勢が示されました。
> 副業や兼業は、新たな技術の開発、オープンイノベーションや起業の手段、第２の人生の準備として有効。  
>
> [出典：厚生労働省「働き方改革実行計画 （概要）」](https://www.mhlw.go.jp/file/05-Shingikai-12602000-Seisakutoukatsukan-Sanjikanshitsu_Roudouseisakutantou/0000173130.pdf)

また、2018年には厚生労働省が「[副業・兼業の促進に関するガイドライン](https://www.mhlw.go.jp/content/11200000/000962665.pdf)」を作成するとともに、モデル就業規則の改定が行われました。
  
  
労働者の遵守事項に関する規定「許可なく他の会社等の業務に従事しないこと。」が削除され、副業・兼業に関する規定が新設された点は、働き方の多様化・時代の変化にあわせた改定です。
  
  
内閣府「[新しい資本主義のグランドデザイン及び実行計画](https://www.cas.go.jp/jp/seisaku/atarashii_sihonsyugi/pdf/ap2023.pdf)」でも、副業や兼業を推進する記述が盛り込まれています。

### 対象企業は「副業・兼業支援補助金」を活用できる

国は副業や兼業を推進する取り組みとして、「副業・兼業支援補助金」を利用した企業による副業や兼業の人材送り出しや受け入れの支援を行っています。
  
  
副業・兼業支援補助金は、副業や兼業をする人材を送り出したり、受け入れたりするための費用の一部を助成する補助金制度です。
  
  
副業・兼業支援補助金には「類型A：副業・兼業送り出し型」と「類型B：副業・兼業受け入れ型」の2種類の制度があります。補助上限額や補助の対象となる費用は以下の通りです。

|  |  |  |
| --- | --- | --- |
|  | 類型A：副業・兼業送り出し型 | 類型B：副業・兼業受け入れ型 |
| 補助率 | 2分の1以内 | 2分の1以内 |
| 補助上限額 | １事業者あたり100万円 | 副業や兼業の人材１人あたり50万円、１事業者あたり250万円（５人まで） |
| 補助対象経費 | ①専門家経費  ②研修費  ③クラウドサービス利用費 | ①仲介サービス利用料  ②専門家経費  ③旅費  ④クラウドサービス利用費 |

[出典：経済産業省「副業・兼業支援補助金」](https://www.fukugyo-kengyo-hojo.jp/)
  
  
法人だけでなく個人事業主も補助の対象になる場合があります。ただし、副業・兼業支援補助金を利用するためには、類型別の要件を満たすことが必要です。
  
  
詳しくは、経済産業省ホームページの「[副業・兼業支援補助金](https://www.fukugyo-kengyo-hojo.jp/)」をご覧ください。

## 企業が副業や兼業を推進するメリット

企業が副業や兼業を推進する主なメリットは次の2つです。

副業や兼業を推進する場合のメリット

●　新たな知識・スキルを得て生産性が向上する  
●　優秀な人材の獲得や流出防止につながる

各メリットの内容を紹介します。

### 新たな知識・スキルを得て生産性が向上する

従業員のスキルアップや生産性向上を図る場合、資格取得を奨励して取得費用を補助する方法や、社内で研修を行う方法もありますが、自社で行える取り組みには限界があります。
  
  
その点従業員が副業や兼業をすれば、本業では得られない知識やスキルを身につけられる可能性があります。副業や兼業は、従業員のスキルアップやキャリアアップに役立ち、従業員の成長が企業の生産性向上につながる点がメリットです。

### 優秀な人材の獲得や流出防止につながる

働き方が多様化するなか、副業や兼業を禁止している職場での就労を避ける人もいます。自社が副業や兼業を認めれば、採用活動をしたときに応募してくる人の数が増え、優秀な人材を採用しやすくなる点は企業にとってメリットです。
  
  
また、副業や兼業を多様な働き方として企業が認めることで、従業員の勤め先に対する満足度が向上し、離職防止につながる可能性があります。

## 企業が副業や兼業を推進するデメリット

企業が副業や兼業を推進する主なデメリットは次の3つです。

副業や兼業を推進する場合のデメリット

●　機密情報が漏れるリスクがある  
●　労働時間の管理が複雑になる  
●　優秀な人材の流出や起業による退職につながる可能性がある

各デメリットの内容を紹介します。

### 機密情報が漏れるリスクがある

従業員が副業や兼業によって他社で働くと、自社の情報が他社に漏れるリスクがあります。自社の機密情報が他社に漏れないよう、従業員とは秘密保持契約を交わしておきましょう。
  
  
また、従業員が他社で働くことによって知り得た情報や、身につけた知識を自社に取り込む場合も、取り扱いには注意が必要です。
  
  
他社の機密情報を社員が自社に漏らしてしまうと、自社の従業員を他社に送り込んで情報を盗む意図があったとみなされるリスクがあります。
  
  
企業として副業や兼業を推進する際には、情報管理の重要性を従業員に理解してもらうことが大切です。

### 労働時間の管理が複雑になる

従業員が副業や兼業をしている場合、労働管理を適切に行うためには、副業や兼業での仕事の内容、労働時間などを確認する必要があります。
  
  
従業員が副業や兼業をしていない場合に比べると、労働時間の管理が複雑になり、人事担当者の負担が増えてしまうでしょう。
  
  
また副業や兼業の内容によっては、自社での労働時間と他社での労働時間を足しあわせて労働時間の管理を行わなければいけません。他社での労働時間は、従業員からの報告をもとに集計する必要があるため手間がかかります。
  
  
本業・副業の労働時間の管理については、「[ダブルワーク（副業）の割増賃金は本業・副業どちらが払う？ 残業代の計算方法も解説](https://www.freee.co.jp/kb/kb-trend/double-work-extra-wage/)」で詳しく解説しています。

### 優秀な人材の流出や起業による退職につながる可能性がある

従業員が副業や兼業をして他社の状況を知ると、他社に魅力を感じて自社を辞める場合があります。
  
  
また、副業・兼業によって従業員がさまざまな業種や業態を経験することで、従業員として雇われるのではなく独立を目指すようになり、退職につながる可能性もあります。副業や兼業は、優秀な人材の流出につながる可能性がある点がデメリットです。
  
  
副業や兼業のメリットとして、優秀な人材の獲得や流出防止につながる点を挙げました・しかし、副業や兼業が人材流出や退職につながる可能性がある点も把握しておく必要があります。

## 企業が副業や兼業を禁止または制限するための条件

「労働者が労働時間以外の時間をどのように利用するか」は基本的に労働者の自由です。企業が副業や兼業の禁止規定を設けたからといって、労働者の自由や権利を一方的に制限できるわけではありません。
  
  
企業が副業や兼業を禁止または制限できるケースは、一定の条件を満たす場合に限られます。
  
  
厚生労働省が公開している「副業・兼業の促進に関するガイドライン」によれば、例外的に労働者の副業や兼業を禁止または制限できるのは、以下の１～４のいずれかに該当する場合です。

例外的に副業や兼業を禁止または制限できるケース

1.　労務提供上の支障がある場合  
2.　業務上の秘密が漏えいする場合  
3.　競業により自社の利益が害される場合  
4.　自社の名誉や信用を損なう行為や信頼関係を破壊する行為がある場合

[出典：厚生労働省「副業・兼業の促進に関わるガイドライン」](https://www.mhlw.go.jp/content/11200000/000962665.pdf)
  
  
副業や兼業に関する規定を就業規則に記載する場合は、「原則として、労働者は副業・兼業を行うことができる」と定めたうえで、例外的に副業や兼業を禁止または制限できるケースを記載しておくことが考えられます。

## 企業が副業や兼業を容認・推進するときのポイント

副業や兼業を容認または推進する際には、対応すべき事項がいくつかあります。主なポイントは次の2つです。

副業や兼業を推進する際のポイント

●　就業規則や運用ルールなど社内規定を整備する  
●　副業・兼業の内容や労働時間を従業員から報告してもらう

以下では各ポイントの内容を紹介します。

### 就業規則や運用ルールなど社内規定を整備する

従業員との間でトラブルにならないよう、副業や兼業の取り扱いは就業規則に明記することが大切です。原則として、労働者は副業や兼業を行うことができる旨を記載するとともに、副業や兼業を禁止または制限するケースなどを定めるようにしてください。
  
  
【就業規則の記載例】
> （副業・兼業）  
> 第70条 労働者は、勤務時間外において、他の会社等の業務に従事することができる。  
> ２ 会社は、労働者からの前項の業務に従事する旨の届出に基づき、当該労働者が当該業務に従事することにより次の各号のいずれかに該当する場合には、これを禁止又は制限することができる。  
> ①労務提供上の支障がある場合  
> ②企業秘密が漏洩する場合  
> ③会社の名誉や信用を損なう行為や、信頼関係を破壊する行為がある場合  
> ④競業により、企業の利益を害する場合
>
> [出典：厚生労働省「モデル就業規則」](https://www.mhlw.go.jp/content/001018385.pdf)

厚生労働省の「モデル就業規則」で公開されている記載例は、あくまで一記載例です。就業規則は事業場の実態に即した内容にする必要があるので、自社の状況を踏まえて適切な内容で就業規則を作成するようにしてください。
  
  
また、副業や兼業での業務内容や、労働時間などを労働者から報告してもらう場合の運用ルールも、社内規定で定めるようにしましょう。

### 副業や兼業の内容や労働時間を従業員から報告してもらう

副業や兼業の内容によっては、自社での労働時間と通算して労働時間の管理を行う必要があります。その場合は、従業員から副業・兼業の内容や、労働時間を報告してもらいましょう。従業員に確認すべき主な事項は以下の通りです。

副業や兼業をする従業員への確認事項

●　ほかの使用者の事業場の事業内容  
●　ほかの使用者の事業場で労働者が従事する業務内容  
●　労働時間通算の対象となるか否かの確認

さらに、労働時間通算の対象となる場合には、あわせて次の事項も確認しておくことが望ましいでしょう。

あわせて確認しておきたい事項

●　ほかの使用者との労働契約の締結日、期間  
●　ほかの使用者の事業場での所定労働日、所定労働時間、始業・終業時刻  
●　ほかの使用者の事業場での所定外労働の有無、見込み時間数、最大時間数  
●　ほかの使用者の事業場での実労働時間の報告、手続き  
●　上記の事項に関して確認を行う頻度

## 労働者が副業や兼業をする際の注意点

労働者が本業とは別に副業や兼業をする場合、注意すべき点があります。主な注意点は次の2つです。

副業や兼業をする際の注意点

●　確定申告が必要になる場合がある  
●　体調や労働時間の管理が必要になる

以下では注意点の内容を紹介します。

### 確定申告が必要になる場合がある

副業や兼業で得ている所得が給与所得にあたる場合、年末調整は主な収入を得ている勤務先でしかできません。
  
  
副業先や兼業先では年末調整を受けられないので、税額を正しく計算するためには確定申告が必要です。源泉徴収で所得税を払い過ぎていた場合、確定申告をすれば払い過ぎた分が還付されます。
  
  
また、副業や兼業で得た所得が給与所得や退職所得以外の場合、副業や兼業による所得が20万円を超えると確定申告が必要です。副業や兼業による所得が20万円以下の場合は、所得税の確定申告をする義務はありませんが、住民税の申告書を提出する必要があります。
  
  
会社員が副業や兼業をした場合の確定申告の要否は、「[会社員が副業したら確定申告は必要？副業する前に知っておくべき手続き・注意点について解説](https://www.freee.co.jp/kb/kb-fukugyou/sidejob-income-tax/)」をご覧ください。

### 体調や労働時間の管理が必要になる

労働基準法の規定により、従業員の労働時間の上限は原則として1日8時間、1週40時間です。
  
  
従業員がほかの職場でも勤務していて、副業や兼業の業務内容が労働基準法の適用を受ける場合、本業と副業・兼業の労働時間を通算して1日8時間、1週40時間以内でなければいけません。
  
  
そのため法定労働時間を超えないように、労働時間の管理を行ってください。法定労働時間を超えて残業させる際には、36協定の締結が必要です。
  
  
また、副業や兼業をする場合、労働時間が増えるため、睡眠不足を招き体調不良を起こす可能性があります。副業や兼業をする場合は、体調を崩したり仕事に支障をきたしたりしないよう、体調や労働時間の管理を適切に行うようにしてください。
  
  
体調管理に関連して、「[メンタルヘルス対策とは？企業が行うべき3段階の予防と具体的な取り組み方法](https://www.freee.co.jp/kb/kb-trend/mental-health-countermeasure/)」の記事もぜひご覧ください。

## まとめ

企業が副業や兼業を推進することには、従業員のスキルアップによる生産性の向上や優秀な人材の獲得、流出防止につながるなどのメリットがあります。
  
  
一方で、従業員が副業や兼業をすると情報漏えいのリスクが相対的に高くなり、労働時間管理が複雑になる点がデメリットです。副業や兼業を推進する場合は、メリットとデメリットを踏まえて検討するようにしてください。
  
  
また、企業が従業員の副業や兼業を推進する際には、就業規則を含む社内規定の整備や改定が必要になる場合があります。副業や兼業を容認、推進する場合は、従業員との間でトラブルにならないように配慮しながら社内ルールを整備しましょう。

## よくある質問

### 副業と兼業の違いは？

副業と兼業は、仕事を掛け持ちしている点では同じです。しかし、副業は本業と副次的に行う仕事を指し、掛け持ちする仕事に対して正副の優先順位をつけています。一方、兼業は仕事を掛け持ちしていることを指す見解が一般的であり、複数の仕事の間に優先順位はありません。
  
  
副業と兼業の違いを詳しく知りたい方は「[副業と兼業の違い](#content1)」をご覧ください。

### 副業や兼業は就業規則で禁止できる？

労働者が労働時間以外の時間をどのように利用するかは基本的に労働者の自由であり、副業や兼業を禁止できるケースは限られます。
  
  
副業や兼業を就業規則で禁止できるケースに関して詳しく知りたい方は「[企業が副業や兼業を禁止または制限するための条件](#content5)」をご覧ください。
