## 旧NISA制度と新NISA制度の違い

2024年1月から新NISA制度が始まり、より使いやすい制度に生まれ変わりました。旧制度からの主な変更点は以下の通りです。

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
|  | 旧NISA制度 | | 新NISA制度 | |
| 一般NISA | つみたてNISA | 成長投資枠 | つみたて投資枠 |
| 併用 | 不可 | | 可 | |
|
 年間投資上限額 | 120万円 | 40万円 | 240万円 | 120万円 || 生涯非課税限度額 | ― | | 1,800万円（内つみたて投資枠1,200万円） | |
| 非課税保有期間 | 5年 | 20年 | 無期限 | |

  
  
旧NISA制度では投資額や投資期間が限られ、一般NISAとつみたてNISAの併用ができないなどのデメリットが指摘されてきました。
  
  
改正後の新NISA制度では、旧制度より使い勝手が格段に良くなっています。

## そもそもNISAってどんな制度？

NISAとは、NISA口座（非課税口座）で購入した金融商品から得られる利益に税金がかからない制度です。
  
  
通常、証券口座である一般口座や特定口座で株式投資や投資信託を行い、利益が生じると20.315%の税金がかかります。

|  |  |
| --- | --- |
| 税金の種類 | 税率 |
| 所得税 | 15% |
| 住民税 | 5% |
| 復興特別所得税（2037年12月末まで） | 0.315% |
| 合計 | 20.315% |

  
  
NISA口座では上記の税金がかからないため、手もとに残るお金が減らずに済みます。
  
  
資産形成を税制面からサポートし、個人の投資を促進する目的です。

## 新NISA制度とは？改正のポイントを解説

2024年1月から新NISA制度がスタートしています。旧NISA制度から変わった点は主に5つです。以下ではそれぞれの改正ポイントを解説します。

### 成長投資枠とつみたて投資枠の併用が可能に

旧NISA制度では一般NISAとつみたてNISAの併用はできませんでしたが、新NISA制度では、旧一般NISAにあたる「成長投資枠」と旧つみたてNISAにあたる「つみたて投資枠」の併用が可能です。
  
  
成長投資枠とつみたて投資枠を併用することで、投資できる商品や投資方法の幅が広がり、より柔軟な資産運用ができるでしょう。

### 年間投資上限額が拡大されて最大360万円に

新NISA制度では成長投資枠の年間投資上限額が240万円に、つみたて投資枠の年間投資上限額が120万円になり、2つの枠を併用すれば年間で最大360万円まで投資できます。
  
  
旧NISA制度の年間投資上限額は、一般NISAが120万円、つみたてNISAが40万円でした。併用もできないため、最大でも年間120万円までしか投資できませんでした。

|  |  |  |  |
| --- | --- | --- | --- |
| NISAの種類 | | 年間投資上限額 | 最大投資額 |
| 旧NISA（〜2023年） | 一般NISA | 120万円 | 120万円 （一般NISA利用時） |
| つみたてNISA | 40万円 |
| 新NISA（2024年〜） | 成長投資枠 | 240万円 | 360万円 |
| つみたて投資枠 | 120万円 |

  
  
投資できる金額が増えれば、投資戦略の幅が広がるだけでなく、資産形成のスピードも早められます。

### 1,800万円の生涯非課税限度額の新設

新NISA制度では、1,800万円の生涯非課税限度額が新設されました。生涯非課税限度額は買付額ベースで管理され、新NISA口座内の商品を売却すると、売却した商品の買付額に相当する非課税枠が復活し、翌年以降の買付けに利用できます。
  
  
旧NISA制度には、生涯非課税限度額という考え方はありません。そのため、旧NISA口座で購入した商品を売却すれば、まだ非課税保有期間が残っていても、その枠は使えなくなります。
  
  
新NISA制度では、生涯非課税限度額1,800万円の枠を使い切った場合でも、商品を売却して枠を復活させれば翌年以降に再度投資できます。
  
  
限度額の考え方そのものが変わり、保有商品の売却と新規購入によって商品の入れ替えをしやすくなった点も大きな特徴です。

### 非課税保有期間の無期限化

新NISA制度では、非課税保有期間に制限がありません。
  
  
旧NISA制度には、非課税で商品を保有できる期間に一般NISAで5年、つみたてNISAで20年の制限があります。そのため、一般NISAで買い付けた商品の非課税期間が終了した場合、以下のいずれかを選択しなければなりませんでした。

非課税期間終了後の選択肢

● 翌年の非課税投資枠に移管（ロールオーバー）する  
● 課税口座に移管する  
● 売却する

一般NISAの場合、5年という限られた期間で投資戦略を考えなければならず、長期的な投資や資産形成に活用しにくい点が欠点です。
  
  
ロールオーバーを利用すれば最長で10年間の保有が可能ですが、移管した額だけ翌年の非課税枠を使ってしまい、新たな買い付けができなくなったり、買付可能額が小さくなったりして投資の幅が狭まったりします。
  
  
新NISA制度では非課税保有期間の上限が撤廃され、これら旧NISA制度の欠点が解消されています。

### NISA制度の恒久化

旧NISA制度では、一般NISAの口座開設期間は2028年まで、つみたてNISAの口座開設期間は2042年までとされていました。新NISA制度への移行に伴って制度が恒久化され、口座開設期間の制限はなくなりました。
  
  
時限的な旧NISA制度に比べ、新NISA制度ではより長期的な資産形成が可能です。

## 新NISA制度のメリット

2024年から始まった新NISA制度のメリットとしては、主に以下の点が挙げられます。

新NISA制度の主なメリット

● 年間投資上限額が拡大され、より効率的な資産形成が可能に  
● 成長投資枠とつみたて投資枠の併用により投資戦略の幅が広がる  
● 非課税保有期間の制限がなくなり、非課税投資枠の再利用が可能になったことでより柔軟な運用が可能に

今回の改正により、NISAは運用目的にあわせて、より柔軟で効率的な運用が可能な制度になりました。
  
  
年間投資上限額の拡大は、投資余力に応じて投資額を増やし運用効率を高めることを可能にします。
  
  
成長投資枠とつみたて投資枠の併用は、投資できる商品や投資方法の選択肢を増やし、投資戦略の幅は大きく広がります。
  
  
非課税保有期間に制限のない恒久的な制度となり、非課税投資枠の再利用が可能になったことで、保有期間や制度の終了を気にせず、目的にあわせてより柔軟な運用が可能になりました。

## 新NISA制度のデメリット

2024年から始まる新NISA制度のデメリットとしては、主に以下の点が挙げられます。

新NISA制度の主なデメリット

● 旧NISA制度と同じく、損失が出たときに他口座との損益通算や損失の繰越控除ができない  
● 旧NISA制度とは別の制度であり、旧NISA制度から新NISA制度へロールオーバーはできない

新NISA制度では、NISA口座内で生じた損失を特定口座や一般口座で生じた利益（売却益・配当金）と相殺（＝損益通算）したり、損失を繰り越して翌年以降の利益と相殺したりすること（＝繰越控除）ができません。この点は旧NISA制度と同様です。
  
  
また、旧NISA制度から新NISA制度へのロールオーバーはできません。
  
  
そのため旧NISA制度の一般NISA口座で保有している商品は、非課税保有期間終了前に売却するか、非課税保有期間終了後に課税口座に移管する必要があります。

## 2024年度税制改正でNISA口座の金融機関変更・再開設手続きが簡素化される

2024年度の税制改正によって、NISA口座の金融機関変更手続きや再開設手続きが簡素化されます。簡素化されるのは、廃止通知書の交付手続き（方法）です。
  
  
利用者がNISA口座を廃止する際に、金融機関から利用者に対して交付される廃止通知書は、これまで書面交付のみが認められていました。しかし2024年4月からは、電子的方法（メールなど）での交付（通知）ができるようになります。
  
  
NISA口座の金融機関変更や口座の再開設には、新たにNISA口座を開設する金融機関に廃止通知書の提出が必要であり、この場合も電子的方法による提出が可能になります。
  
  
電子的方法で交付・提出が可能になれば、郵送にかかる時間を短縮でき、書面交付よりも手続きがスムーズになるでしょう。

## まとめ

2024年1月から始まった新NISA制度では、成長投資枠とつみたて投資枠の併用が可能になり、年間投資上限額も大幅に拡大しました。
  
  
非課税保有期間の制限もなくなり、生涯非課税限度額の新設で非課税投資枠の再利用が可能になったことで、より柔軟で効率的な資産形成が可能な制度に生まれ変わっています。
  
  
一方で、損益通算や損失の繰越控除ができない点は旧NISA制度と変わりません。また、旧NISA制度と新NISA制度は別の制度であり、旧制度で一般NISAを利用していた人は、新NISA口座に旧NISA口座で保有している商品をロールオーバーできない点に注意しましょう。
  
  
制度や改正されたポイントをしっかり理解して、NISAを資産形成に活かしていきましょう。
