## e-Gov（イーガブ）とは

**e-Gov（イーガブ）とは、デジタル庁が運営する行政情報のポータルサイトです。**
  
  
法令の検索や電子申請など、行政に関する総合的な検索・案内サービスを提供しており、「電子政府の総合窓口」とも呼ばれます。
  
  
e-Govの目的は、主に以下の2つです。

e-Govの目的

* 公正さ、透明性を高めて開かれた行政を実現する
* 国民の利便性と行政事務の効率化を図る

e-Gov自体は2001年から運用が開始されています。IT化に伴い、「世界一便利で効率的な電子行政」を実現する目的で、2006年から電子申請が始まりました。
  
  
2020年11月に実施されたe-Govのリニューアルや、2020年4月から順次対応が始まった社会保険関係手続きの電子申請義務化を背景に、普及が進んでいます。
  
  
e-Govの利用には「GビズID」が利用可能です。GビズIDは認証システムのことで、e-GovやIT導入補助金の申請など、多くの行政サービスで利用できます。
  
  
GビズIDについては、「[GビズIDとは？ できることやメリット・デメリット、アカウントの取得方法を解説](https://www.freee.co.jp/kb/kb-trend/what-is-gbiz-id/)」で紹介しているので、併せてご覧ください。

## e-Govでできること

e-Govでは、主に以下3つのサービスが利用できます。

e-Govでできること

* 電子申請
* 法令の検索
* 意見の提出

### 行政手続きを電子申請できる

[出典：e-Gov電子申請「e-Gov電子申請で使える主な機能」](https://shinsei.e-gov.go.jp/contents/about-appli/service.html)
  
  
**「e-Gov電子申請」では、各省庁への申請・届出をオンライン上で行えます。**
  
  
e-Gov電子申請に対応している府省庁は、以下の通りです（2024年4月30日時点）

e-Gov電子申請に対応している府省庁

* 国家公安委員会・警察庁
* 総務省
* 消費者庁
* 厚生労働省（※1）
* 経済産業省（※2）
* 国土交通省
* 気象庁
* 環境省

時間や場所にとらわれず申請や届出が完了し、処理状況の確認もできるため、業務効率の大幅な向上が期待できるでしょう。
  
  
（※1）中央労働委員会を含みます。  
（※2）資源エネルギー庁、特許庁および中小企業庁を含みます。

### 法令や行政情報を検索できる

e-Govでは、現在施行されている法令（憲法や法律、規則など）をオンライン上でいつでも検索できます。
  
  
また、各省庁が提供しているオープンデータを横断的に検索できる「データポータル」も利用可能です。人口・社会保障・通信情報などのカテゴリやキーワードで検索でき、業務に活用できます。

### 政策に意見や要望を提出できる

e-Govでは、「パブリック・コメント制度」を利用して、オンライン上で政策に意見や要望を提出できます。
  
  
パブリック・コメント制度とは、行政機関が政令などを定める際、一般の意見を考慮して公正さ・透明性を確保するための制度です。案件のなかには、氏名や連絡先を記載せずに意見を提出できるものもあります。

## 特定の法人は電子申請が義務化されている

行政手続きコスト削減や事業者の負担軽減のため、2020年4月から特定の法人について一部手続きの電子申請が義務化されています。
  
  
以下に該当する法人は、社会保険・労働保険に関する一部の手続きの際、電子申請を利用しなければなりません。

義務化の対象となる特定の法人

* 資本金・出資金・銀行等保有株式取得機構に納付する拠出金の額が1億円を超える法人
* 相互会社
* 投資法人
* 特定目的会社

電子申請が義務付けられたのは、社会保険・労働保険に関する手続きです。
  
  

|  |  |
| --- | --- |
| 健康保険・厚生年金保険 | ●　被保険者報酬月額算定基礎届  ●　被保険者報酬月額変更届  ●　被保険者賞与支払届 |
| 労働保険 | ●　年度更新に関する申告書（概算保険料申告書、確定保険料申告書、一般拠出金申告書）  ●　増加概算保険料申告書 |
| 雇用保険 | ●　被保険者資格取得届  ●　被保険者資格喪失届  ●　被保険者転勤届  ●　高年齢雇用継続給付支給申請  ●　育児休業給付支給申請 |

[出典：厚生労働省「2020年4月から特定の法人について 電子申請が義務化されます。」](https://www.mhlw.go.jp/content/11200000/000867797.pdf)
  
  
なお、e-Gov電子申請は、義務化の対象となっていない法人も利用できます。積極的に活用し、業務効率向上やコスト削減に役立てましょう。
  
  
また、現時点では特定の法人、一部の手続きが対象ですが、今後義務化の対象が拡大されることも考えられます。

## e-Govを利用するメリット

e-Govを利用して行政手続きをオンライン上で行うメリットは、主に以下の3つです。

e-Govを利用するメリット

* 24時間手続きができ利便性が向上する
* 業務効率が向上する
* コストを削減できる

### 24時間手続きができ利便性が向上する

e-Govは、24時間いつでも利用できるため、休日や夜間でも行政手続きが行えます。自宅やお勤め先から手続きできるので、行政機関までの移動時間や窓口での待ち時間も発生しません。
  
  
また、e-Govのマイページ上では、いつでも申請の処理状況や行政機関からの通知を確認できるため便利です。
  
  
さらに利用者設定でメール通知設定を「受信する」にしておけば、進捗状況をメールで受け取ることができ、利便性を上げられます。

### 業務効率が向上する

e-Goveを利用すれば都合のよい時間で手続きできるため、業務効率の大幅な向上が期待できます。移動時間や待ち時間もなくなるため、行政手続きにかかる時間が短縮され、ほかの業務に充てられる時間が増えるでしょう。
  
  
また、e-Govには前年度の情報を取り込み、入力チェックや自動計算する支援機能が備わっている手続きもあります。そのため、書類作成の負担軽減やミスの防止にもつながり、ペーペーレス化によって書類の管理にも手間がかかりません。

### コストを削減できる

e-Govを利用すれば、行政手続きをオンライン化でき書面を減らせるため、印刷費用や管理コストを削減できます。
  
  
また、行政機関の窓口に出向く移動代もかかりません。

## e-Govを利用するデメリット

e-Govを利用すれば業務効率の向上や時間・コストの削減が期待できますが、以下の点には注意が必要です。

e-Govを利用するデメリット

* 健康保険組合はe-Gov電子申請に対応していない
* デジタル環境への適応に時間がかかる可能性がある
* 入力ミスに気付きにくい

### 健康保険組合はe-Gov電子申請に対応していない

e-Gov電子申請は、雇用保険・労働保険など多くの行政手続きが対象ですが、健康保険組合への手続きには対応していません。
  
  
なお、e-Gov電子申請に対応していない場合も、健康保険組合向け電子申請ソフトやマイナポータル経由で電子申請できる場合があります（※）。健康保険組合によってマイナポータルの電子申請に対応している手続きが異なるので、事前に確認しましょう。
  
  
（※）マイナポータルとは、マイナンバーカードを利用してオンライン上で各種手続きができるサイトです。

### デジタル環境への適応に時間がかかる可能性がある

e-Govを利用するためには、デジタル環境への適応が必要です。オンライン手続きやデジタルツールの使用方法を知るには、学習の時間と労力を要します。
  
  
たとえば、オンラインフォームの入力方法や、アカウントの取得方法などを学ぶ必要があります。また、セキュリティに関する意識や対策も重要です。
  
  
e-Govへのアクセスは、特にデジタルに不慣れな人々や高齢者などにとってはハードルとなる可能性があります。学習の時間と労力を割く必要があり、初めての利用や新しい機能の習得には苦労することがあるかもしれません。

### 入力ミスに気付きにくい

e-Gov電子申請を行う際、入力ミスには気を付けましょう。
  
  
e-Gov電子申請では、申請手続き画面で直接必要事項を入力し、必要書類をアップロードして手続きを行います。
  
  
ペーパーレス化によって業務効率向上やコスト削減を図れる一方で、書面で確認できないため不備に気付きにくい傾向があります。
  
  
申請内容に不備があった場合、申請内容一式を再提出しなければならないケースもあるため注意が必要です。
  
  
不備があった場合に備えて、提出前に「申請データを保存」を行っておきましょう。保存しておけば、再提出を求められた際修正が簡単に行えます。

## e-Govを利用するために準備すること

e-Govで電子申請するには、以下4つの事前準備が必要です。

e-govの事前準備

* 電子証明書またはGビズIDの取得
* アカウントの登録（GビズID取得の場合は不要）
* ブラウザの設定確認
* アプリケーションのインストール

電子証明書またはGビズIDの取得には、日数がかかる場合もあるため、余裕をもって準備しましょう。4つの事前準備について、以下で詳しく解説します。

### 電子証明書またはGビズIDの取得

電子証明書が必要な手続きかどうかを事前に確認し、対応している電子証明書を取得しましょう。
  
  
電子証明書とは、本人であること、改ざんされていないことを証明するための電子的な証明書です。
  
  
電子証明書の対象手続きや利用可能な電子証明書の種類、取得方法は、「[e-Gov電子申請](https://shinsei.e-gov.go.jp/contents/preparation)」のサイトにてご確認ください。
  
  
なお、電子証明書の取得に費用がかかったり、ICカードタイプの電子証明書の場合には「ICカードリーダー」が必要だったりします。

### アカウントの登録

次に、電子申請に必要なアカウントを準備しましょう。GビズIDやMicrosoftアカウントがあれば、e-Govのアカウントを登録しなくても利用できます。
  
  

|  |  |
| --- | --- |
| 利用できるアカウントの種類 | 概要 |
| e-Govアカウント | e-Govサービス共通のアカウントで利用できるアカウント |
| GビズID | ひとつのID・パスワードで複数の法人向け行政サービスにログインできるサービス |
| Microsoftアカウント | Microsoftの各種サービスにログインできるアカウント |

  
  
なお、e-Govアカウントは5～10分程度で登録できます。

### ブラウザの設定確認

アカウントの準備が整ったら、ブラウザの設定を行いましょう。
  
  
「e-Gov電子申請」のサイト「利用準備」に記載されている手順に従い、ブラウザの「ポップアップブロックの解除」を行います。

### アプリケーションのインストール

最後に、e-Gov電子申請アプリケーションをインストールすれば準備は完了です。インストール手順も「e-Gov電子申請」のサイトにて確認できます。
  
  
会社によってはセキュリティ対策で、アプリケーションをインストールする際に、管理者権限での操作を求められる場合があるので、その際はシステム担当者や管理者へご相談ください。
  
  
インストールが完了したら、e-Gov電子申請アプリケーションを起動し、電子申請手続きに進みましょう。

## e-Govの課題と取り組み

e-Govには、いくつか課題も存在します。e-Govの課題とそれに対して進められている取り組みを解説します。

e-Govの課題

* 電子申請に対応していない手続きもある
* デジタル格差が公平性に影響する可能性がある
* セキュリティ上のリスクがある

### 電子申請に対応していない手続きもある

すべての行政手続きがe-Govの電子申請に対応しているわけではありません。
  
  
e-Govに対応するには、利用者が提出した申請データを内部処理する府省の「申請受付システム」が必要です。
  
  
また、e-Govでは1回の申請で100MBまでしかファイルを添付できません。
  
  
ほかにもシステムが整備されていないなどの理由で電子申請に対応していない手続きは、書面での対応が必要です。
  
  
「規制改革実施計画」では、2025年までに約12,000種類の手続きを、なるべく前倒しでオンライン化するとしています。具体的には、府省の申請受付システムがなくても受付処理できるようにする、などの取り組みが検討されています。

### デジタル格差が公平性に影響する可能性がある

e-Govの導入にはインターネット接続やデジタルデバイスの利用が必要です。
  
  
しかし、これらのリソースが十分に普及していない場合、e-Govへのアクセスは容易ではないでしょう。
  
  
これによりデジタル格差が拡大し、情報やサービスの不均等性が生じる可能性があります。今以上にデジタルリテラシーやアクセス環境の改善が求められるでしょう。

### セキュリティ上のリスクがある

e-Govでは、2022年9月にサイバー攻撃による可能性が高いとされる障害が発生しています。今後、同じような問題が起きる可能性も考えられるでしょう。
  
  
デジタル庁は、システム障害や情報漏えいを防ぐための点検や、サイバーセキュリティを強化していくとしています。
  
  
また、e-Govでは情報の盗聴や改ざんを防ぐための暗号化など、さまざまなセキュリティ対策が実施されています。

## 社会保険に関する業務を円滑にする方法

社会保険に関する業務は、加入手続きや保険料の計算など多岐にわたります。それらの業務を効率化したいとお考えの方には、[freee人事労務](https://www.freee.co.jp/hr/)がおすすめです。

* 入社時の被保険者資格取得届の作成

従業員・労務担当者双方の対応を簡略化し、効率化とペーパーレス化を同時に実現できるサービスです。
  
  
上記のほかにも年末調整・労働保険の年度更新・算定基礎届の作成・住民税の更新など、人事労務関連のさまざまな業務をサポートします。

## まとめ

e-Govとは、行政手続きの電子申請や行政情報が得られるポータルサイトです。行政機関への申請・届出が24時間365日オンライン上で行えるため、時間・コストを削減できます。
  
  
e-Govの利用には事前準備が必要です。また、まだ対応していない手続きもあるため事前に確認しましょう。
  
  
e-Govを活用すれば、行政手続きをペーパーレス化でき業務効率化を図れます。複雑な労務事務の負担軽減やミス防止のために、電子申請対応のシステム導入も検討しましょう。
