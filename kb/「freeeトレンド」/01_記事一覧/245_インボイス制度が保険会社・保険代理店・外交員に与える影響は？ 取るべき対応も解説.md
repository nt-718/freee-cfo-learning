## インボイス制度とは

2023年10月からインボイス制度が始まります。インボイス制度とは、消費税の仕入税額控除の方式として導入される制度です。
  
  
買い手は、売り手から交付されたインボイス（適格請求書）を保存した取引でのみ、仕入税額控除を受けられます。
  
  
仕入税額控除とは、売上にかかる消費税から仕入れにかかった消費税を差し引いて計算し、消費税の二重課税を解消できる制度です。
  
  
売り手は「適格請求書発行事業者」の登録を受けた場合のみ、インボイスを発行できます。登録を受けない場合は、取引先（買い手）が仕入税額控除を受けられないため、取引継続に影響を与える可能性があります。

[出典：政府広報オンライン「令和5年10月からインボイス制度が開始！事業者が進めておきたい準備とは？ 」](https://www.gov-online.go.jp/useful/article/202210/1.html)
  
  
インボイス制度について詳しく知りたい方は「[インボイス制度（適格請求書等保存方式）で影響を受ける業種とは？](https://www.freee.co.jp/kb/kb-invoice/type_of_industry/) 」を参考にしてください。

## インボイス制度は保険会社に影響がある？

インボイス制度は、保険代理店や保険外交員（免税事業者）の収支に影響を及ぼします。最初に、インボイス制度によって保険会社・保険代理店・保険外交員がどう変わるのかを解説します。主な影響は、以下の2点です。

インボイス制度が保険業界に与える影響

●　保険外交員が免税事業者の場合、保険代理店の税負担が増える  
●　保険外交員（免税事業者）の収入が減少する可能性がある

### 保険会社に影響はほとんどない

**保険会社にとって、代理店に支払う手数料や保険外交員の報酬に消費税がかかるかどうかは重要ではありません。**なぜなら、保険会社の主たる売上となる保険料は非課税取引であり、消費税が発生しないからです。
  
  
つまり、インボイス制度が始まっても、保険会社の収支に大きな影響はありません。ただし、取引のなかには企業年金保険の付加保険料など消費税がかかるものもあるため、保険会社が影響を受ける場合もあります。

### 保険外交員が免税事業者の場合、保険代理店の税負担が増える

**インボイス制度が保険業界に大きな影響を与えるのは、保険代理店が保険外交員（個人事業主）に報酬を支払うときです。**
  
  
保険外交員が免税事業者の場合、インボイスが発行できないため、保険代理店が仕入税額控除を受けられません。

|  |  |
| --- | --- |
| 免税事業者 | 基準期間の課税売上高が1,000万円以下の事業者（納税義務が免除される事業者） |
| 課税事業者 | 基準期間の課税売上高が1,000万円を超える事業者（納税義務者） |
|

  
  
つまり、保険外交員（免税事業者）に報酬を支払っている保険代理店の消費税負担が大きくなります。
  
  
課税事業者の保険外交員は元々消費税を納めているため、適格請求書発行事業者になるデメリットはありません。そのため、基本的には適格請求書発行事業者の登録を受けるべきでしょう。
  
  
実際に、国税庁によると2023年3月末時点で課税事業者全体の約9割が申請を終えています。一方、現在免税事業者の場合は、インボイスに対応することで税負担が増えてしまうデメリットもあります。
[出典：国税庁「インボイス制度の周知広報 の取組方針等について」](https://www.cas.go.jp/jp/seisaku/tekikaku_seikyusyo/dai2/siryou.pdf#page=2)

### 保険外交員（免税事業者）の収入が減少する可能性がある

インボイス制度が始まると、免税事業者のままでいる保険外交員の売上が減る可能性が高いでしょう。
  
  
個人事業主の保険外交員には、課税売上高1,000万円以下の免税事業者も少なくありません。保険外交員が免税事業者だとインボイスが発行できないため、保険代理店は仕入税額控除を受けられず、消費税負担が大きくなります。
  
  
そのため2023年10月1日以降、免税事業者のままでいる場合、保険代理店から契約を終了されるケースが予想されます。保険代理店は、適格請求書発行事業者の登録を受けた保険外交員と契約を結んだほうが、税負担を抑えられるからです。
  
  
現時点で免税事業者の保険外交員は、課税事業者になり適格請求書発行事業者の登録を受けるかどうかを検討しなくてはいけません。

## インボイス制度を導入する際の注意点

適格請求書発行事業者になるとインボイスを発行できますが、注意点もあります。以下2点を踏まえ、登録を受けるかどうかを検討しましょう。

インボイス制度を導入する際の注意点

●　消費税の申告・納付が必要になる  
●　経理事務の負担が増える

### 消費税の申告・納付が必要になる

免税事業者が適格請求書発行事業者の登録を受けるために課税事業者になると、消費税の申告・納付の義務が生じます。
  
  
免税事業者のままでいれば消費税の納税義務は免除されますが、売上減少のリスクがあるため、慎重な検討が必要です。
  
  
なお、インボイス制度開始後6年間は、免税事業者からの仕入であっても一定割合の仕入税額控除ができる経過措置があります（※）。

|  |  |
| --- | --- |
| 経過措置を適用できる期間 | 控除できる割合 |
| 2023年10月1日～2026年9月30日 | 仕入税額相当額の80% |
|
| 2026年10月1日～2029年9月30日 | 仕入税額相当額の50% |
|

  
  
上記措置も踏まえ、適格請求書発行事業者の登録を受けるかどうかを検討しましょう。
  
  
個人事業の方は「[個人事業主のための消費税簡単計算法！確定申告で消費税の処理に悩まないために](https://www.freee.co.jp/kb/kb-blue-return/consumption-tax/)」で消費税について理解を深めておくのがおすすめです。
  
  
（※）一定の要件があります。
  
  
（※）「令和6年度税制改正大綱」にて、同一の免税事業者からの課税仕入れの額がひとつの事業年度内で10億円を超える場合、10億円を超えた部分は経過措置が受けられないことが発表されました。本改正は、2024年10月1日以後に開始する課税期間から適用されます。

### 経理事務の負担が増える

適格請求書発行事業者になると、インボイスの発行が必要となり、経理事務の負担が大きくなります。

インボイス制度を導入する際の注意点

●　インボイスに対応した請求書に様式を変更する  
●　インボイス（写し）を作成・保存する  
●　受け取ったインボイスが要件を満たしているかどうかを確認する  
●　請求書を仕入税額控除の対象・対象外に区別する

## 保険会社がインボイス制度を導入する際に取るべき対応

適格請求書発行事業者になると決めたときに取るべき対応は、主に以下の2点です。

インボイス制度を導入する際に取るべき対応

●　インボイス制度の内容と業務への影響を理解する  
●　適格請求書発行事業者の登録を受ける

### インボイス制度の内容と業務への影響を理解する

インボイス制度が始まるまでに、制度内容と業務への影響を正しく理解しましょう。
  
  
総務省が発表した「消費税の適格請求書等保存方式（インボイス制度）への対応に係る準備状況について」（2022年6月21日発表）によると、インボイス制度・消費税制度への理解が不十分と認められる回答が相当数にのぼっています。
[出典：総務省「消費税の適格請求書等保存方式（インボイス制度）への対応に係る準備状況について」](https://www.soumu.go.jp/main_content/000821071.pdf#page=1)
  
  
しかし、インボイス制度の導入に必要な準備を進めるうえで、制度の概要や業務に与える影響への理解は欠かせません。
  
  
国税庁の[インボイス制度特集ページ](https://www.nta.go.jp/taxes/shiraberu/zeimokubetsu/shohi/keigenzeiritsu/invoice.htm)や、制度に関わる各省庁の相談窓口、免税事業者向けの相談会などを活用しましょう。

### 適格請求書発行事業者の登録を受ける

インボイスを発行するには、適格請求書発行事業者の登録申請が必要です。登録を受ける事業者は、「適格請求書発行事業者の登録申請書」を作成し、納税地を所轄する税務署長にe-Taxや郵送で提出します。
  
  
適格請求書発行事業者の登録期限はありませんが、登録には一定の審査期間がかかるため、余裕を持って申請書を提出しましょう。

[出典：政府広報オンライン「令和5年10月からインボイス制度が開始！事業者が進めておきたい準備とは？ 」](https://www.gov-online.go.jp/useful/article/202210/1.html)

## インボイス制度の導入に必要な準備

インボイス制度をスムーズに導入するには、さまざまな準備が必要です。

インボイス制度の導入に必要な準備

●　業務フローを見直す  
●　インボイス（適格請求書）を準備する  
●　取引先への周知・説明を行う  
●　インボイス対応のシステムを導入する

必要な準備を把握し、計画的に進めましょう。

### 業務フローを見直す

適格請求書発行事業者になると、請求書の書式変更が必要です。消費税の端数処理の方法が変わるなど事務負担が煩雑になるため、業務フローの見直しも求められます。
  
  
また、発行したインボイス（写し）や受け取ったインボイスの保存方法、税額の計算方法も検討しましょう。

### インボイス（適格請求書）を準備する

適格請求書発行事業者になると、記載事項を満たしたインボイス（適格請求書）の発行が必要です。インボイスが必要な取引を整理し、発行できる体制を整えましょう。

インボイスの記載事項

●　適格請求書発行事業者の氏名または名称・登録番号  
●　取引年月日  
●　取引内容（軽減税率の対象品目である旨）  
●　税率ごとに区分して合計した対価の額（税抜き/税込み）・適用税率  
●　税率ごとに区分した消費税額等  
●　書類の交付を受ける事業者の氏名または名称

なお、所定の様式が法律で決まっているわけではありません。現在の書類に不足する項目を追加し、必要事項が記載されていればインボイスと認められます。

### 取引先への周知・説明を行う

適格請求書発行事業者の登録申請が完了したら、取引先が安心して取引を続けられるように登録番号の通知や説明を行います。
  
  
登録番号は、税務署長の登録を受けた際に通知される番号です。以下のローマ字・数字で構成されます。

|  |  |
| --- | --- |
| 法人番号をもつ課税事業者 | 「T」＋法人番号（数字13桁） |
| 上記以外の課税事業者（個人事業者など） | 「T」＋数字13桁 |
|

  
  
必要に応じて、取引先の登録状況も確認しましょう。

### インボイス対応のシステムを導入する

インボイス制度対応のシステムの導入も検討しましょう。システムを導入すれば、インボイス制度に関する下記のような業務を効率化でき、事務負担軽減につながります。

システム導入で効率化できる業務の例

●　登録申請に必要な書類やインボイス対応の請求書の作成  
●　仕入税額控除の対象となる請求書と対象外の請求書の区分  
●　インボイスの保存

また、業務が煩雑になることで起きやすくなるミスも防止できます。

## まとめ

2023年10月から、消費税の仕入税額控除方式としてインボイス制度が始まります。
  
  
保険業界でインボイス制度が大きく影響するのは、保険代理店が保険外交員（個人事業主）に報酬を支払うときです。

適格請求書発行事業者の登録を受ける場合は、2023年10月の制度開始に向け、計画的に準備を進めましょう。

一方、取引先から請求書を交付される側（買い手）は、取引先が適格請求書発行事業者であるかどうかを事前に確認しておく必要があります。

### 売り手：適格請求書発行事業者への登録申請

インボイス制度導入のタイミングで適格請求書発行事業者であるためには、**2023年9月30日まで**に「適格請求書発行事業者の登録申請書」を税務署に提出し、登録を受ける必要があります。

### 買い手：取引先の登録状況を把握

インボイス制度に対応している適格請求書と、従来の請求書では扱いが異なるので、取引先がインボイス制度に対応しているのか把握しておかなければなりません。
  
  
[freee](https://www.freee.co.jp/lp/solution/invoice/01/)では、取引先のインボイス制度への対応状況や登録番号の回収を一括で管理できます。無料で利用できるので、コストをかけずに効率化が可能です。
