## ノマドとは？

ノマド（nomad）とは、もともとはフランス語で遊牧民や放浪者を意味する「nomade」を語源とした言葉です。転じて、時間や場所、組織にとらわれずに働く人を「ノマドワーカー」と呼びます。
  
  
政府の働き方改革の推進を受け、ノマドワーカーのような働き方を選ぶ人が増えています。IT環境の整備により、インターネットを通じてさまざまな場所で働けるようになった点も、ノマドワーカー増加の要因のひとつです。

## ノマドワーカーとフリーランスの違い

ノマドワーカーと似た用語に「フリーランス」があります。フリーランスとは、特定の企業に属さず、個人のスキルで働く個人事業主のことです。一方、ノマドワーカーは時間や場所にとらわれず働くワークスタイルを指しています。
  
  
どちらも個人で仕事を請け負って働いている点では共通していますが、フリーランスは多くの場合、雇用形態や契約形態をあらわす際に使われる表現です。
  
  
また、フリーランスの場合、自宅や事務所など拠点を持って働く人もいるため、フリーランスとノマドワーカーは区別して用いられます。

|  |  |  |
| --- | --- | --- |
| 項目 | フリーランス | ノマドワーカー |
| 雇用形態 | 個人事業主 | 個人事業主 |
| 働き方 | 個人のスキルで働く | 個人のスキルで働く |
| 働く場所 | 自宅や事務所など拠点を持つ場合あり | 時間や場所にとらわれない |
| 違い | 雇用形態・契約形態を指す | ワークスタイルを指す |

## ノマドワーカーのメリット

ノマドワーカーとして働くメリットには、主に下記の3つのメリットがあります。

### 場所にとらわれずに仕事ができる

ノマドワーカーの大きなメリットは、場所にとらわれず、好きな場所で仕事ができる点です。
  
  
インターネットに接続できる環境さえあれば、自宅やカフェ、図書館、コワーキングスペースなど多くの場所で仕事ができます。
  
  
そのため、頻繁に居住地を変える人でも、オンライン上で企業から仕事を受託できます。また、配偶者の転勤で居住地が変わった場合にも柔軟に対応可能です。
  
  
このように場所にとらわれずフレキシブルに働ける点は、ノマドワーカーとして働くメリットでしょう。

### 好きな時間に働ける

ノマドワーカーは、会社員のように就業時間が決められているわけではありません。受注した仕事によっては納期がありますが、働く時間自体はライフスタイルに合わせて自由に設定できます。
  
  
たとえば、日中は家事や趣味などやりたい作業に時間をあて、夜間を中心に業務をこなす働き方も可能です。

### 人間関係のストレスを軽減できる

ノマドワーカーは組織に属さない場合も多いので、上司との付き合いや職場の飲み会など、人間関係が発生する場面が少なくなります。
  
  
会社でオフィスワークをする場合と比較すると、対人関係でのストレスをコントロールしやすい働き方といえます。
  
  
また、会議や打ち合わせもZoomなどを利用しWeb上で行えば、身支度にかける手間も短縮可能です。

## ノマドワーカーのデメリット

ノマドワーカーは場所や時間などにとらわれないなどのメリットがある一方、いくつかデメリットもあります。

### 収入が不安定となる場合がある

日本経営協会がまとめた「ノマドワーカーの働き方実態調査報告書」によると、ノマドワーカーは主に以下のようなときに不安を感じていると報告されています。

ノマドワーカーが不安を感じるとき

● 収入が少ないとき・収入見通しが立たず先細りが懸念されるとき  
● 継続的な顧客がいないとき・顧客が減り続けるとき  
● ひとりで仕事するため仕事や顧客が広がらないとき

組織で仕事をしていく会社員と比べ、ノマドワーカーは仕事の受注先を自分で探さなくてはならず、収入が不安定になるデメリットがあります。
  
  

[出典：日本経営協会「ノマドワーカーの働き方実態調査報告書」](http://www.noma.or.jp/report_home/nomad/tabid/256/Default.aspx)

### 確定申告や社会保険手続きなどを自分でやる必要がある

会社に所属している場合は、月々の源泉徴収・年末調整などの税金や、医療保険・厚生年金などの社会保険は、会社側が手続きしてくれます。
  
  
一方、ノマドワーカーの場合は、仕事を始めたら税務署に開業届を提出し、一定の収入が得られたら確定申告を行わなくてはいけません。また、国民健康保険や国民年金などの手続きも必要です。
  
  
上記のように、自分でやらなければならない手続きがある点は、ノマドワーカーのデメリットです。近年ノマドワーカーの増加にともない、開業届の手続きや確定申告など、重要な手続きをサポートしてくれるツールも充実してきました。すべて自分で行うのではなく、上手くツールを利用することで、ノマドワーカーのデメリットを軽減できます。
