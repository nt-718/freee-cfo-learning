「休眠預金」は、10年以上入金や出金などの取引がない預金などを指します。休眠預金になっても、手続きをすることで預金の引き出しは可能です。しかし、窓口での本人確認などが必要となるため、通常の出金より手間がかかる場合があります。

## 休眠預金とは

**休眠預金とは、2009年1月1日以降の取引から、10年以上取引（入金や出金）がない預金などです。**
  
  
長期間使っていない口座がある場合、休眠預金になっている可能性があります。休眠預金になると、引き出しの際に窓口に出向く必要があるなど、通常の手続きとは異なるため注意が必要です。
  
  
休眠預金になっているかどうかは、口座番号が記載されている書類を用意し、金融機関に問い合わせれば確認できます。
  
  
2018年から、休眠預金が公益目的の活動や社会貢献活動に活用される制度が始まりました。
  
  
以下で詳しく解説します。

### 休眠預金等活用法（休眠預金等に係る資金の活用に関する法律）

「休眠預金等に係る資金の活用に関する法律（民間公益活動を促進するための休眠預金等に係る資金の活用に関する法律）」は、2018年1月に施行されました。
  
  

[出典：政府広報オンライン「放置したままの口座はありませんか？10年たつと「休眠預金」に。」](https://www.gov-online.go.jp/useful/article/201907/1.html)
  
  
金融庁によると、預金者が名乗らないままになっている預金は、毎年1,200億円程度（2014年～2016年）にものぼります。
  
  

[出典：金融庁「民間公益活動を促進するための休眠預金等に係る資金の活用に関する法律 概要」](https://www.fsa.go.jp/policy/kyuminyokin/01gaiyou.pdf)
  
  
長期間放置された預金を預金者にできる限り払い戻し、そのうえで休眠預金を国民に還元すべきとの考えから、休眠預金を民間公益活動の促進に充てる法律が制定されました。

### 休眠預金の対象となるもの

休眠預金の対象になるのは、2009年1月1日以降の取引から10年以上取引のない普通預金や定期預金などです。
  
  
一方、預金保険制度の対象にならない外貨預金・財形年金・財形住宅などは、休眠預金の対象ではありません。以下に休眠預金の対象・対象外の預金などをまとめたので参考にしてください。

|  |  |
| --- | --- |
| 休眠預金の対象となる預金など | ●　普通預金  ●　定期預金  ●　当座預金  ●　別段預金  ●　貯蓄預金  ●　定期積金  ●　金融債（保護預かりのもの）  ●　金融信託（元本補填のもの）  ●　相互掛金 |
| 休眠預金の対象とならない預金など | ●　外貨預金  ●　譲渡性預金  ●　金融債（保護預かりなし）  ●　財形年金・住宅  ●　仕組預金  ●　マル優口座 |

[出典：一般社団法人全国銀行協会「休眠預金ってご存じですか？」](https://www.zenginkyo.or.jp/article/tag-b/10117/)

## 休眠預金になるとどうなる？

長期間使っていなかった預金が休眠預金になっても、預金残高がなくなるわけではありません。
  
  
主に次のような対応が取られたり、手続きが必要になったりします。

休眠預金になるとどうなるのか

●　民間公益活動に活用される  
●　休眠預金の引き出しは窓口での手続きが必要になる  
●　預貯金契約どおりの利子がつく

以下で詳しく解説します。

### 民間公益活動に活用される

最後の取引から9年経過し、休眠預金となりそうな預金については、金融機関のウェブサイトで公告されます。預金残高が1万円以上の場合は、電子メールなどで通知が送られ、受け取れば預金は休眠状態に入りません。
  
  
ただし、金融機関に登録されている住所やメールアドレスが現段階と異なる場合は、通知は届かず、休眠預金となります。住所やメールアドレスを変更した方は注意が必要です。
  
  
もし休眠預金になると、金融機関から預金保険機構にそのお金が移管され、管理が引き継がれます。
  
  
預金保険機構に移管されたお金は、NPO法人などの民間公益活動を行う団体を通じて、活用されます。休眠預金の移管先は以下の通りです。

休眠預金の移管先

●　子ども若者支援  
●　生活困難者支援  
●　地域活性化等支援

また、新型コロナウイルス感染症拡大による社会的課題の解決にも役立てられています。休眠預金を活用して、社会問題を解決する仕組みの構築や、SDGsの達成など社会貢献に繋げられます。
  
  
なお、預金保険機構は当面の間、休眠預金の5割を将来の引き出しに備えて積み立てる予定です。つまり、移管された休眠預金のすべてが活用されるわけではありません。  
したがって、金額は減ってしまいますが、移管後も元々取引を行っていた金融機関で引き出し可能です。

### 休眠預金の引き出しは窓口での手続きが必要になる

休眠預金になっても、取引先の金融機関へ通帳やキャッシュカード、お届印、本人確認書類を持って手続きすれば、いつでも引き出せます。引き出しに期限は設けられていません。
  
  
ただし、ATMでは引き出せず、一般的に窓口での手続きが必要です。もし通帳やキャッシュカードを紛失している場合も、本人確認書類の提示などで引き出せるため、金融機関に必要な書類を問い合わせましょう。
  
  
なお、休眠預金の出金には口座の確認などが必要となるため、通常の出金と比べて時間がかかる場合があります。

### 休眠預金を復活させられない場合や手数料がかかる場合がある

休眠預金になった場合の取り扱いには以下2つのケースがあり、金融機関によって異なります。

休眠預金になった場合の取り扱い

●　手続きをすれば元の口座を引き続き使えるケース  
●　払い戻しを受け、引き続き口座を使いたい場合は新しく口座開設が必要なケース

どのような取り扱いになるかどうかは、口座のある金融機関に問いあわせて確認しましょう。また、金融機関によっては長期間使っていない口座に管理手数料がかかる場合があるため、注意が必要です。

### 預貯金契約どおりの利子がつく

休眠預金期間中の利子は、元本に元の契約に応じて利子相当額を加えた額です。つまり、休眠預金になっている期間も預貯金契約どおりの利子がつき、引き出す際は元本＋利息が受け取れます。

## 休眠預金にしない方法

休眠預金になるのを避けたい場合は、以下の対策をしましょう。

休眠預金にしない方法

●　使わない口座は解約する  
●　入出金などの取引を行う  
●　住所やメールアドレスを変更する  
●　金融機関の公告に対して情報提供を求める

次で詳しく解説します。

### 使わない口座は解約する

使わなくなった口座をそのままにしていると、休眠預金になる可能性があります。
  
  
休眠預金になっても残高がなくなるわけではありませんが、通常より出金に手間がかかることが一般的です。長期間使っていないと口座管理手数料がかかる場合もあるため、今後も使う予定がない口座は解約を検討しましょう。

### 入出金などの取引を行う

休眠預金の対象となるのは、10年以上異動のない預金などです。異動とは、「今後も預貯金などを利用する意思を示したと認められるような取引」を指します。  

[出典：金融庁「長い間、お取引のない預金等はありませんか？」](https://www.fsa.go.jp/policy/kyuminyokin/kyuminyokin.html)
  
  
10年以内に異動となる取引をしていれば、休眠預金にはなりません。具体的には、以下のような取引が異動に該当します。

|  |  |
| --- | --- |
| 全金融機関共通で異動となる取引 | ●　入出金（※1）  ●　振込による残高の異動  ●　口座振替による残高の異動  ●　手形・小切手の提示などによる第三者からの支払請求（※2）  ●　公告された預金などに対する情報提供の求め |
| 各金融機関が許可を受けた場合に異動となる取引 | ●　通帳や証書の発行・記帳・繰越  ●　残高照会  ●　契約内容・顧客情報の変更  ●　口座を借入金の返済に利用する旨の申し出  ●　預金などにかかる情報の受領（※3）  ●　総合口座に含まれるほかの預金の異動 |

（※1）金融機関から支払われる利子は除く  
（※2）金融機関が把握された場合のみ  
（※3）定期預金の満期案内や普通預金の利息計算書など  

[出典：金融庁「休眠預金等活用法Q＆A」](https://www.fsa.go.jp/policy/kyuminyokin/kyuminyokinQA.pdf)
  
  
つまり、入出金や振込、口座振替などを利用すれば、休眠預金にはなりません。また、金融機関によっては、記帳や残高照会、顧客情報の変更も異動に該当します。異動となる取引は、金融機関によって異なるため確認しましょう。

### 住所やメールアドレスを変更する

預金残高が1万円以上で9年間取引がない場合、金融機関から郵送またはメールが届きます。通知を受け取れて確認できた場合、休眠預金にはなりません。
  
  
しかし、以下のように通知が届かない場合は、何もしなければ休眠預金になってしまいます。

通知が届かないために休眠預金になってしまうパターン

●　金融機関に登録している住所が現住所と異なる  
●　メールがあて先不明で届かない

住所やメールアドレスが変わった際は、通知を受け取れるように変更手続きをしておきましょう。なお、預金残高が1万円未満の場合は、通知が送付されないため注意してください。

### 金融機関の公告に対して情報提供を求める

各金融機関は、休眠預金になりそうな預金などがある場合、ホームページで「電子公告」を行います。公告には、休眠預金の対象となりうる預金などの最後の異動日、預金保険機構への移管の期限などが掲載されます。
  
  
公告は移管より前に行われ、預金者が公告開始から移管までの間に公告に対して照会すれば休眠預金にはなりません。
  
  
なお、電子公告が行われるのは、最後の異動日から9年を経過してから10年6ヶ月を経過するまでの間です。また、公告開始日から2ヶ月～1年の間に移管されます。

## まとめ

長い間取引がない口座は、10年以上そのままにしておくと休眠預金として処理されてしまします。休眠預金になってもいつでも出金は可能ですが、窓口での手続きが必要です。また、休眠預金は民間公益活動に活用されます。
  
  
入出金や振込などの取引やメールアドレスや住所の変更手続きなどをすれば、休眠預金を避けることができます。金融機関によっては、長期間使用していない口座に対して手数料が発生する場合があります。そのため、使う予定がない口座は解約も検討しましょう。
