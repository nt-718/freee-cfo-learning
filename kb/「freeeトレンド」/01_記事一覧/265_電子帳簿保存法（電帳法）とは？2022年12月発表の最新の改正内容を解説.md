## 電子帳簿保存法（電帳法）とは？

電子帳簿保存法（電帳法）は、帳簿や国税関係書類を電子データで保存するためのルールを定めた法律です。
  
  
電子帳簿保存法で定められている「電子データ保存」は、以下の3種類に区分されています。

  

|  |  |
| --- | --- |
| **電子データ保存の種類** | **概要** |
| 電子帳簿等保存 | 自己が電子的に作成した仕訳帳などの帳簿や国税関係書類をデータのまま保存する |
| スキャナ保存 | 紙で受領・作成した書類を画像データとして保存する |
| 電子取引データ保存 | 電子的にやりとりした取引情報をデータとして保存する |

2021年度の改正により、上記のうち「電子取引データ保存」が義務化されました。「電子取引データ保存」とは、取引先とデータでやりとりした請求書などをデータで保存することです。
  
  
2023年12月31日までに行った電子取引のデータは引き続き紙で保存しても問題ありません。しかし、2024年1月からは紙での保存が認められず、一定の要件にしたがってデータで保存しなければなりません。
  
  
電子帳簿保存法の概要は別記事「[電子帳簿保存法とは？対象書類や保存要件・改正内容についてわかりやすく解説](https://www.freee.co.jp/kb/kb-deals/deals-records-in-electronic-form/)」で詳しく解説していますので、あわせてご確認ください。

## 電子帳簿保存法の改正が行われた背景

電子帳簿保存法の改正は、社会のデジタル化への動きにあわせて何度か実施されています。
  
  
2021年度の電帳帳簿保存法の改正は、新型コロナウイルス感染症拡大により問題となったデジタル化への遅れなどを背景に、経理の電子化による生産性や記帳水準の向上を目指して行われました。
  
  
改正では、保存要件などのハードルの高さから普及が進まなかったデータ保存を普及させるため、保存要件が大幅に緩和されています。

## 2022年12月発表！電子帳簿保存法の改正内容

2021年度の税制改正により電子取引データ保存が義務化され、2022年度の税制改正では電子取引データ保存に2年間の宥恕（ゆうじょ）措置、つまり猶予期間が設けられました。
  
  
宥恕措置により「やむを得ない事情」がある場合、2022年1月から2年間は紙での保存も認められることが明記されました。
  
  
さらに、2022年12月に発表された2023年度の税制改正大綱では、新たな変更点があります。最新の改正内容を以下で説明します。

### 電子取引データ保存義務化に向けた新たな変更点

電子取引データ保存には2年間の宥恕期間が設けられており、2023年12月31日までは引き続き紙での保存が認められています。
  
  
2022年12月に発表された2023年度の税制改正大綱では、さらに新たな猶予が発表されました。そのため、相当の理由があると認められ、ダウンロードや出力書面の提示・提出に応じられる場合は、2024年1月以降も紙での保存が認められます。

|  |  |  |
| --- | --- | --- |
| **期間** | **電子取引データの保存方法** | **条件** |
| 2023年12月31日まで | 紙での保存が認められる | 全員対象 |
| 2024年1月以降 | 紙での保存が認められる | ・相当の理由がある ・ダウンロードや出力書面の提示・提出に応じられる |
|  | 電子データ保存必須 | 上記以外のケース |

「相当の理由」とは、例えば、資金不足や人手不足などで、電子取引データを保存するためのシステムや社内のワークフローの整備が間に合わないなど、要件に従って電子取引データを保存する環境が整っていない事情がある場合に認められます。
  
  

[出典：国税庁「電子帳簿保存法一問一答【電子取引関係】（令和６年５月）問61」](https://www.nta.go.jp/law/joho-zeikaishaku/sonota/jirei/pdf/00023006-044_03-6.pdf)

### そのほかの改正点

2023年度の税制改正大綱では、電子取引データ保存の猶予措置以外にも改正点があります。
  
  
スキャナ保存・電子取引データ保存では、データを保存する際の要件が緩和されました。
  
  
これによりスキャナ保存では、国税関係書類をスキャナで読み取った際の情報の解像度や階調、大きさの要件などが廃止されています。
  
  
また、電子取引データ保存では、判定期間の売上高が5,000万円以下で出力書面の提示・提出に求められる場合、検索要件のすべてを不要とする緩和などが決められました。

|  |  |
| --- | --- |
| **保存形式** | **改正点** |
| スキャナ保存 | ・画像の解像度・階調・大きさに関する要件廃止  ・入力者情報の確認要件廃止  ・帳簿との関連性の確認要件廃止 |
| 電子取引データ保存 | ・判定期間における売上高が 5,000 万円以下の場合、検索要件のすべてが不要  （電磁的記録のダウンロードの求めに応じられる場合に限る）  ・電磁的記録の保存者に関する情報の確認要件廃止 |

[出典：総務省「令和５年度税制改正の大綱」](https://www.soumu.go.jp/main_content/000853546.pdf#page=81)
  

さらに、改正により過少申告加算税の優遇が受けられる、優良な電子帳簿の範囲が明確に定められたため、優良帳簿への対応がとりやすくなります。

## 電子帳簿保存法改正のポイント・注意点

改正された電子帳簿保存法に関して、以下のポイント・注意点を抑えておきましょう。
  
  
**電子帳簿保存法のポイント・注意点**

* 改ざんなどの不正行為をすると罰則がある
* 電子データも7年間保存する必要がある

### 改ざんなどの不正行為をすると罰則がある

電子帳簿保存法の改正では、データ保存する際の要件が緩和されている一方で、データ改ざんなどの不正を防ぐために罰則も設けられています。
  
  
スキャナ保存したデータや電子取引データに関連して不正行為などをした場合、申告漏れに課される重加算税が10%加算されます。通常の35%～40%の重加算税からさらに10%加算された金額となるため、注意が必要です。

### 電子データも7年間保存する必要がある

データ保存した帳簿や書類も、紙での保存と同様に原則として7年間の保存が義務付けられています。ただし、法人で繰越欠損金が生じている場合は最大10年です。
  
  
ダウンロードを求められた際に速やかに提示・提出できるよう、データの保存要件をしっかり把握して正しくデータ保存の運用をしましょう。

## 電子帳簿保存法の改正で企業にせまられる対応

2023年度税制改正大綱により新たな猶予が設けられましたが、基本的にはデータ保存への対応が必須です。
  
  
そこで、改正電子帳簿保存法に対して企業がとるべき対応を解説します。

### 電子取引データ保存への対応を完了させる

電子取引データ保存への移行が完了していない事業者は、データ保存するための社内整備が必要です。
  
  
最初に、取引先とやりとりしている領収者や請求書を整理し、電子取引の対象になるデータを把握しましょう。紙を介さないすべての取引が電子取引となる点に注意してください。
  
  
次に、データの保存要件を満たす保存方法や保存場所を決めましょう。データ保存するときの要件は、以下の3つです。
  
  
**データ保存の要件**

* 改ざん防止の措置をとる
* 「日付・金額・取引先」で検索できるようにする
* ディスプレイやプリンタなどを備え付ける

改ざん防止の措置には、以下の方法があります。
  
  
**改ざん防止措置**

* タイムスタンプを付与する
* 履歴が残るシステムで授受・保存する
* 改ざん防止のための事務処理規程を定めて守る

保存要件を把握して保存方法や場所を決め、業務フローの見直しや規程の準備、システムの導入などを行いましょう。従業員への説明や取引先への周知も重要です。

### インボイス制度とあわせてペーパーレス化を進める

電子取引データ保存の猶予期間にあたる2023年10月には、「インボイス制度」も始まります。インボイス制度とは、消費税の仕入税額控除の方式として導入される制度です。
  
  
売り手は、登録をすれば適用税率や消費税額などを記載したインボイス（適格請求書）を発行できる「インボイス発行事業者」になれます。また、買い手は、交付されたインボイスを保存した取引について仕入税額の控除が可能です。
  
  
インボイス制度が始まると、請求書への記載事項が増える、仕入の区分が必要になるなど経理の負担は大きくなります。業務効率を上げるためにも、インボイス制度の開始も踏まえてペーパーレス化への移行が重要です。
