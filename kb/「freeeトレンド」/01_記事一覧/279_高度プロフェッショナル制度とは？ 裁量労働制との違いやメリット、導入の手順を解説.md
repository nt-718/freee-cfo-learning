## 高度プロフェッショナル制度とは？

**高度プロフェッショナル制度とは、高度な知識やスキルをもつ労働者を対象に、労働時間や休日などの規定を適用しない制度です。**
  
  
高度な専門知識を持っていて、一定の年収要件（年収1,075万円以上）かつ職務が明確な労働者を対象としています。
  
  
高度プロフェッショナル制度の対象労働者には、労働基準法が定める労働時間・休憩・休日・割増賃金が適用されません。
  
  
代わりに使用者には、対象労働者の心身の安全性に配慮して、健康管理時間の把握や休日の確保、選択的措置が求められます。
  
  
高度プロフェッショナル制度では、労働者は始業時刻や終業時刻が指定されないなど、働く時間に関して広い裁量権が与えられます。
  
  
欧米のホワイトカラー・エグゼプションを参考に、日本の状況にあわせてアレンジされた制度です。

## 高度プロフェッショナル制度と裁量労働制との違い

裁量労働制は、労働時間を労働者の裁量に委ねる雇用形態です。
  
  
労働時間を実際の労働時間で計算せず、事前に設定された「みなし時間」で算出します。
  
  
高度プロフェッショナル制度と裁量労働制、そして一般の労働者に対する労働時間関係規定の適用をまとめると以下の通りです。

|  |  |  |  |
| --- | --- | --- | --- |
|  | 高度プロフェッショナル制度 | 裁量労働制 | 一般の労働者 |
| 労働時間 | ― | △ | 〇 |
| 休憩 | ― | 〇 | 〇 |
| 休日 | ―（※） | 〇 | 〇 |
| 時間外手当 | ― | △ | 〇 |
| 休日手当 | ― | 〇 | 〇 |
| 深夜手当 | ― | 〇 | 〇 |
| 年次有給休暇 | 〇 | 〇 | 〇 |

  
（※）年間104日以上、かつ、4週間を通じ4日以上の休日が必要
  

[出典：厚生労働省「労働時間制度間の比較等」](https://www.mhlw.go.jp/content/11201250/000944727.pdf)
  
  
労働者に働く時間に関する広い裁量を委ねる点で、高度プロフェッショナル制度と裁量労働制は共通しています。
  
  
加えて、裁量労働制には休憩・休日・深夜の割増賃金など、一般の労働者に認められている労働基準法の多くの規定が適用されます。
  
  
しかし高度プロフェッショナル制度の場合は、労働基準法は基本的に適用されません。この点が、高度プロフェッショナル制度と裁量労働制の大きな違いです。
  
  
また、高度プロフェッショナル制度には年収要件がある一方、裁量労働制に年収要件はなく、対象業務も異なります。
  
  
裁量労働制についても「[裁量労働制とは？対象の職種やメリット、今後の見直し方針をわかりやすく解説](https://www.freee.co.jp/kb/kb-trend/discretionary-labor-system/)」であわせて確認しておきましょう。

## 高度プロフェッショナル制度の対象労働者や対象業務

高度プロフェッショナル制度はすべての労働者に適用できるわけではなく、対象となる労働者や業務が限定されています。
  
  
以下では、対象となる労働者と業務の条件を解説します。

### 対象労働者

高度プロフェッショナル制度の対象となる労働者は以下の通りです。

高度プロフェッショナル制度の対象労働者

●　職務が明確に定められている  
●　業務の内容や責任、求められる成果を書面で同意が得られている  
●　見込まれる給与が1,075万円以上である  
●　対象業務以外の業務に常態的に従事していない

高度プロフェッショナル制度を適用するには、労働者との合意のうえで職務が明確に定められていなければなりません。
  
  
業務内容が不明瞭である場合、制度を適用できない点に注意が必要です。
  
  
また、1年間あたりの給与が1,075万円以上である年収要件が設定されています。この金額は確実に支払われる給与の金額であり、業績給や勤務成績など支給額が決まっていない給与は含まれません。

### 対象業務

高度プロフェッショナル制度は、「使用者から具体的な従事時間の指示を受けていない」点が前提条件です。そのうえで、以下の具体的な対象業務に該当する必要があります。

高度プロフェッショナル制度の対象業務

●　金融商品の開発業務  
●　ファンドマネージャーやトレーダー、ディーラーの業務  
●　証券アナリスト業務  
●　コンサルタント業務  
●　新たな技術、商品または研究開発の業務

たとえば、「新興国企業の株式を含む金融商品の開発業務」「金融知識を活用した投資判断に基づく資産運用の業務」などが高度プロフェッショナル制度の対象業務の一例です。
  
  
なお、上記に関連する業務でも、「データ入力や整理業務」「取次業務」など定型的な業務は対象とならないので注意が必要です。

## 高度プロフェッショナル制度のメリット

高度プロフェッショナル制度を導入すると、労働時間に基づいた従来の働き方とは異なる労働環境を構築できます。高度プロフェッショナル制度の主なメリットは以下の通りです。

高度プロフェッショナル制度のメリット

●　生産性の向上が期待できる  
●　従業員の不公平感を軽減できる  
●　ワーク・ライフ・バランスを実現しやすくなる

各メリットの詳細を解説します。

### 生産性の向上が期待できる

高度プロフェッショナル制度は、労働時間ではなく成果に応じて評価されます。対象労働者は効率的な労働を心がけるようになり、結果として生産性の向上が期待できる点は大きなメリットです。

### 従業員の不公平感を軽減できる

従来の労働時間を基準とする制度の場合、仕事が遅く長時間残業をする従業員は、残業代分だけ給与が高くなります。
  
  
一方で、比較的短時間で成果を挙げる従業員は、成果を残しているにも関わらず、残業代があまり加算されません。このような両者の間に生まれる不公平感が、労働時間を基準とする制度の課題でした。
  
  
高度プロフェッショナル制度では、成果を出す従業員ほど評価され、不要な残業による給与が発生しません。そのため、従業員間の不公平感の軽減が可能です。

### ワーク・ライフ・バランスを実現しやすくなる

高度プロフェッショナル制度は働く時間の裁量の幅が広く、時間配分などを自分で調整できます。
  
  
出社や退社の時間、休暇のタイミングも自身の都合にあわせて決定可能です。
  
  
働きながら介護や子育てなどの生活に関する事柄と両立しやすく、ワーク・ライフ・バランスの実現に貢献します。

## 高度プロフェッショナル制度のデメリット

高度プロフェッショナル制度はメリットがある一方で、以下のデメリットが想定されます。

高度プロフェッショナル制度のデメリット

●　長時間労働のリスクがある  
●　評価基準の設定が難しい

各デメリットの内容を紹介します。

### 長時間労働のリスクがある

高度プロフェッショナル制度は、労働基準法の労働時間や休憩、休日などの適用を受けません。結果、対象の従業員が長時間労働をしてしまうリスクがある点に注意しましょう。
  
  
そのためには、使用者は対象労働者に対して以下の措置を実施しなければなりません。

高度プロフェッショナル制度のデメリット

●　従業員の労働時間（健康時間）を客観的に把握する  
●　年間104日以上の休日、かつ4週4日の休日を与える  
●　勤務間で11時間以上のインターバルを確保するなどの選択的措置を実施する  
●　従業員の健康・福祉に配慮した措置を実施する

[出典：厚生労働省「高度プロフェッショナル制度について」](https://jsite.mhlw.go.jp/fukushima-roudoukyoku/content/contents/000415522.pdf)

### 評価基準の設定が難しい

高度プロフェッショナル制度は、成果に応じた評価基準を設定します。従来の勤務時間を基準とした制度と比較すると、明確な評価基準を設定しづらい点が難点です。
  
  
研究開発業務のように成果が出るまで長期間かかる業務もあり、適正な評価基準をどう設定するかが課題に挙げられます。

## 高度プロフェッショナル制度を導入する手順

高度プロフェッショナル制度を導入する一般的な手順は以下の通りです。

高度プロフェッショナル制度の導入手順

1　労使委員会を設置する  
2　労使委員会で決議し、労働基準監督署長へ届け出る  
3　書面で対象労働者の同意を得る  
4　対象労働者を業務に就かせる

各ステップで必要な手続きを解説します。

### ①労使委員会を設置する

高度プロフェッショナル制度の導入には、労使委員会の設置が必要です。
  
  
「委員の半数を労働代表委員が占めている」「委員会の議事録作成・保存・労働者への周知が図られている」点に注意し、労使委員会を設置しましょう。
  
  
労使委員会は、労働者の過半数代表者または、労働組合と事前に話し合いを行って設置します。労使双方の代表委員を選出したあとは、労使委員会を運営するルールを定めましょう。

### ②労使委員会で決議し、労働基準監督署長へ届け出る

労使委員会では、委員の5分の4以上の多数で決議されなければなりません。決議項目は以下の通りです。

労使委員会の決議項目

●　対象業務  
●　対象労働者の範囲  
●　対象労働者の健康管理時間の把握  
●　休日の確保  
●　選択的措置  
●　対象労働者の同意の撤回に関する手続き  
●　健康管理時間の状況に応じた健康・福祉確保措置  
●　同意しなかった際の不利益な取り扱いの禁止  
●　苦情処理の実施と具体的な内容  
●　その他の厚生労働省令で定められた項目

[出典：厚生労働省「⾼度プロフェッショナル制度 わかりやすい解説」](https://www.mhlw.go.jp/content/001140965.pdf)
  
  
決議した内容は所定の様式により、所轄の労働基準監督署長に届け出ます。

### ③書面で対象労働者の同意を得る

労使委員会で決議した内容にしたがい、高度プロフェッショナル制度の対象労働者と書面で同意を交わします。
  
  
同意を交わす際は、概要・労使委員会の決議内容・同意した場合の賃金制度・評価制度などを、対象労働者に書面で明示しなければいけません。
  
  
なお、以下の項目も必ず書面で明示しないといけないので注意してください。

労働者本⼈にあらかじめ書⾯で明示する必須項目

●　高度プロフェッショナル制度の概要  
●　労使委員会の決議の内容  
●　同意した場合の賃金制度、評価制度  
●　同意をしなかった場合の不利益取扱い禁止  
●　同意の撤回と不利益取扱いの禁止が可能なこと

[出典：厚生労働省「⾼度プロフェッショナル制度 わかりやすい解説」](https://www.mhlw.go.jp/content/001140965.pdf)

上記同意のうえ明示する必須項目

●　労働時間や休憩、休日や深夜の割増賃金が適用されない旨の説明  
●　同意の対象となる期間  
●　期間中に支払われる見込み賃金額

[出典：厚生労働省「⾼度プロフェッショナル制度 わかりやすい解説」](https://www.mhlw.go.jp/content/001140965.pdf)

### ④対象労働者を業務に就かせる

対象労働者を業務に就かせる場合には、[以下の対応](https://www.mhlw.go.jp/content/001140965.pdf)をとりましょう。

使用者に求められる対応

1.　健康管理時間を把握する  
2.　休日を付与する  
3.　選択的措置および健康・福祉確保措置を実施する  
4.　苦情に対応する  
5.　6ヶ月以内で定期的に実施状況を労働基準監督署長に報告する  
6.　同意をしなかった場合、不利益な取り扱いをしない

[出典：厚生労働省「⾼度プロフェッショナル制度 わかりやすい解説」](https://www.mhlw.go.jp/content/001140965.pdf)
  
  
高度プロフェッショナル制度では、長時間労働のリスクを伴います。
  
  
使用者には上記の対応が義務付けられており、特に1～3の項目は6ヶ月以内ごとに定期的に労働基準監督署長に報告が必要です。

## まとめ

企業を取り巻く経済環境が変化するなか、組織内での人材の効果的な活用は、経営者や人事労務担当者の大切な課題です。
  
  
高度プロフェッショナル制度は柔軟な働き方に寄与する制度であり、対象業務を実施している企業には重要な選択肢となるでしょう。
  
  
労働時間を基準とし、メンバーシップ型雇用が進められてきた日本では、新たな挑戦となる制度とも捉えられます。
  
  
高度プロフェッショナル制度は労働者の労働時間に対する裁量が広がる反面、長時間労働のリスクが生じます。
  
  
導入の際は労使委員会で健康確保措置を定め、労働者の健康に配慮しましょう。

1980年生まれ、東京都出身。IT大手・ベンチャー人事部での経験を活かし、2021年独立。年間1000件余りの労務コンサルティングを中心に、給与計算、就業規則作成、助成金申請等の通常業務からセミナー、記事監修まで幅広く対応。ITを活用した無駄がない先回りのコミュニケーションと、人事目線でのコーチングが得意。趣味はドライブと温泉。
