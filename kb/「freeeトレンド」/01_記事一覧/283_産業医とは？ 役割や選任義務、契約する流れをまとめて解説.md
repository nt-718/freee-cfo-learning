## 産業医とは

**産業医とは、労働者が健康に仕事ができるよう支援する医師です。**労働安全衛生法に基づいて企業により選任され、医療の専門家としての立場から助言や指導を行います。
  
  
産業医は、医師の資格をもつだけでなく、産業医学に関する一定の要件を備えています。具体的には、厚生労働大臣が定める産業医研修の修了者や、労働衛生コンサルタント試験（保健衛生）に合格した医師などが要件です。
  
  
そのため産業医の導入は、労働者の健康管理や職場の安全衛生・健康意識向上に役立ち、健康で活力ある環境づくりにつながります。

## 産業医の役割

産業医の役割は多岐にわたります。主要な役割は以下の通りです。

産業医の役割

●　健康診断の実施サポートと結果を踏まえた措置  
●　治療と仕事の両立支援  
●　健康状態の悪化を防ぐための職場巡視  
●　健康面でのリスク管理への対応  
●　衛生委員会への出席や衛生講話の実施

[出典：独立行政法人 労働者健康安全機構「中小企業事業者の為に産業医ができること」](https://www.mhlw.go.jp/content/000501079.pdf)
  
  
各項目の詳しい内容を解説します。

### 健康診断の実施サポートと結果を踏まえた措置

企業は労働安全衛生法により、雇用時の健康診断や定期健康診断の実施が義務付けられています。この健康診断実施のサポートと、結果に基づいて意見や必要な措置をとる行為は産業医の大きな役割です。
  
  
たとえば、企業では以下のような健康診断が実施されます。

|  |  |
| --- | --- |
| 健康診断の種類 | |
| 一般健康診断 | ●　雇用時の健康診断  ●　定期健康診断  ●　特定業務従事者の健康診断  ●　海外派遣労働者の健康診断  ●　給食従事者の検便 |
| 特殊健康診断 | ●　法令で義務付けられた健康診断（特に有害業務従事者の健康診断）  ●　VDT作業や腰痛など、指導勧奨による健康診断 |

  

[出典：独立行政法人 労働者健康安全機構「中小企業事業者の為に産業医ができること」](https://www.mhlw.go.jp/content/000501079.pdf)
  
  
健康診断は実施したら終わりではありません。健康診断の診断区分（「A異常なし」「B軽度異常」など）に基づき、従業員が就労できるかの判断が求められます。
  
  
企業は、従業員の健康診断で異常が見られた場合、就労可否や必要な措置について3ヶ月以内に医師の意見を聴く義務があります。
  
  
異常所見が発覚しても、従業員が通常勤務が可能なのか、就業に一定の制限をかけるべきなのか、判断に困る場合もあるでしょう。
  
  
産業医がいれば、診断結果に目を通してもらい、就業上の措置に関して適切な意見がもらえます。もし産業医がいない場合には、地域の産業保険センターなどで意見を聴きましょう。

### 治療と仕事の両立支援

治療が必要な持病をもつ従業員を雇用している場合、「治療」と「仕事」の両立支援も産業医の役割です。
  
  
厚生労働省の管轄である、独立行政法人労働者健康安全機構により発行された「[中小企業事業者の為に産業医ができること](https://www.mhlw.go.jp/content/000501079.pdf)」では、両立支援が必要な病気には以下のような特徴があるとしています。

両立支援が必要な病気

1.　定期的な通院が必要である  
2.　行状や治療薬変更で通院頻度が変わる  
3.　病状の悪化や再発で、入院や自宅療養が必要となる場合がある

[出典：独立行政法人 労働者健康安全機構「中小企業事業者の為に産業医ができること」](https://www.mhlw.go.jp/content/000501079.pdf)
  
  
近年では、上記の病気に罹患していても、就業を希望する労働者が増えています。このような従業員の就業の可否・就業環境・就業条件の整備をする際に、産業医は専門的な見地から助言を行います。
  
  
産業医の存在により、企業は自社内だけでなく、医療的な立場からの適切な意見を加えた判断が可能です。病気の治療のため、休職や離職を選んでいた従業員の就業が期待でき、人材の確保や定着につながります。

### 健康状態の悪化を防ぐための職場巡視

職場巡視は、労働安全衛生規則で定められた産業医の大切な役割です（労働安全衛生規則15条1項）。
  
  
労働安全衛生規則では、産業医に原則毎月1回の職場巡視を義務付けています。(2017年の法改正により、衛生管理者が週1回の巡視結果を産業医に月ごとに提供することで2ヶ月に1回とすることが可能になりました。)
  
  
職場巡視の主な目的は、「作業方法に問題はないか」「衛生状態に有害となる恐れはないか」などを発見し、改善を行う点です。
  
  
また、事務所の職場巡視では、主に下記のような事項をチェックします。

職場巡視のチェック事項

①整理整頓や清掃がなされており清潔な環境となっているか  
②冷暖房等で室温が法令に定める一定の基準となっているか  
③湿度が法令に定める一定の基準となっているか  
④照明が適切に調整され、照度が法令に定める一定の基準となっているか  
⑤VDT作業（PC等の作業）環境が適切か  
⑥休憩室の状況等

[出典：独立行政法人 労働者健康安全機構「中小企業事業者の為に産業医ができること」](https://www.mhlw.go.jp/content/000501079.pdf)

### 健康面でのリスク管理への対応

産業医は以下のようなリスク管理への役割も期待されています。

健康面でのリスク管理への対応

●　ストレスチェック制度への対応  
●　長時間労働への対応

[出典：独立行政法人 労働者健康安全機構「中小企業事業者の為に産業医ができること」](https://www.mhlw.go.jp/content/000501079.pdf)
  
  
ストレスチェック制度とは、2015年12月から労働者数50人以上の事業場で義務付けられた、メンタルヘルスのチェック制度です。産業医は事業場の現状を理解していることから、ストレスチェック実施者の役割を期待されています。
  
  
また、長時間労働は時間が長くなればなるほど、健康への影響を無視できません。そのため、産業医の面接指導などにより、メンタルヘルス不調や脳・心臓疾患の兆候が見られないか、必要な事後措置を行います。

### 衛生委員会への出席や衛生講話の実施

産業医は衛生委員会への出席や、衛生講話の実施も役割のひとつです。
  
  
衛生委員会では、従業員の健康障害防止の基本対策や、リスクアセスメントなどを審議します。産業医は衛生委員会の委員として出席し、必要に応じて勧告を行います。
  
  
そのほか、従業員への健康や衛生管理を目的とした研修（衛生講話）を実施し、従業員の健康に対する教育も役割のひとつでしょう。

## 産業医の選任義務

産業医の選任は法律で定められていて、選任義務の有無や選任する産業医の人数は事業場の労働者数で異なります。労働者数ごとの選任義務は以下の通りです。

|  |  |
| --- | --- |
| 事業場の規模 | 内容 |
| 労働者数50人未満 | 産業医の選任義務はなし  （産業医選任の努力義務はある） |
| 労働者数50人以上3,000人以下 | 1名以上の産業医の選任義務がある |
| 労働者数3,001人以上 | 2名以上の産業医の選任義務がある |

  

[出典：厚生労働省「産業医について ～その役割を知ってもらうために～」](https://www.mhlw.go.jp/file/06-Seisakujouhou-11200000-Roudoukijunkyoku/0000103897.pdf)
  
  
なお事業者は、事業場で常時50人以上の従業員を使用し始めてから、14日以内の選任が必要です。また、産業医を選任したら、所轄の労働基準監督署長への届け出義務がある点に注意しましょう。

### 嘱託産業医と専属産業医の違い

産業医には嘱託（しょくたく）産業医と専属産業医の2種類があり、一定の事業場では専属産業医を選任しなければなりません。嘱託産業医と専属産業医の違いは以下の通りです。

|  |  |
| --- | --- |
| 産業医の種類 | 内容 |
| 嘱託産業医 | ・非常勤で勤務する産業医  ・常時50人以上で999人以下の労働者を使用する事業場は、一部を除き嘱託産業医でよい |
| 専属産業医 | ・ひとつの事業場のみに属する産業医  ・常時1,000人以上の事業場、一定の有害業務に常時500人以上の労働者を使用する事業場では、専属産業医の選任が義務付けられている |

  
たとえば、常時100人が働いている事業場の場合、開業医や医療機関の勤務医に非常勤で産業医を依頼する方法が可能です。
  
  
一方、常時2,000人が働く事業場では、非常勤ではなく、その事業場に専属する産業医の選任が求められます。

## 産業医を選任するメリット

産業医を選任する主なメリットは以下の通りです。

産業医を選任するメリット

●　労働環境の向上  
●　従業員の健康や安全の確保  
●　生産性・パフォーマンスの向上  
●　離職率の低下  
●　コンプライアンスの確保

[出典：独立行政法人 労働者健康安全機構「中小企業事業者の為に産業医ができること」](https://www.mhlw.go.jp/content/000501079.pdf)
  
  
産業医を選任する大きなメリットは、労働環境の向上が期待できる点でしょう。産業医学の見地から助言を受けられ、従業員の健康を確保し、生産性・パフォーマンスの向上、従業員の離職率低下につながります。
  
  
労働人口の減少に伴い、従業員一人ひとりの生産性向上や人材流出の防止は、多くの企業に共通する課題です。産業医の導入は企業に合ったオーダーメイドの健康管理に役立つでしょう。

## 産業医と契約する流れ

産業医と契約する主な流れは以下の通りです。

産業医と契約する流れ

1　希望する産業医を探す  
2　産業医と契約を締結する  
3　労働基準監督署に報告する  
4　助成金を活用する

産業医導入の費用負担軽減につながる助成金も紹介するため、ぜひ参考にしてください。

### ①希望する産業医を探す

初めに、希望する産業医を探しましょう。産業医の探し方には主に以下の方法が挙げられます。

産業医の探し方

●　最寄りの医師会に相談する  
●　近隣の医療機関に相談する  
●　医師紹介会社を活用する  
●　健康診断を依頼している医療機関に相談する

医師会は各都道府県に設置された「都道府県医師会」のほか、市区町村レベルで設置されている「群市区医師会」があります。産業医の選任に迷ったときは相談するとよいでしょう。
  
  
そのほか、近隣の医療機関に相談する方法や、医師紹介会社を活用する方法も有効な手段です。
  
  
産業医の選択では、自社に適した専門性をもつ医師を探すことが重要です。たとえば、IT企業であれば建設業界より業務作業での事故も少ないと推測できます。
  
  
その場合、メンタルヘルスに強い精神科医を検討するなど、自社の業態にあった専門をもつ医師を選ぶといいでしょう。

### ②産業医と契約を締結する

希望する産業医が見つかったら、契約を結びましょう。産業医との契約形態には、業務委託契約・雇用契約・スポット契約が挙げられます。
  
  
業務委託契約を結ぶ際には、委託範囲を明確とする点が重要です。自社が産業医に求める業務を事前に調査し、契約を希望する産業医と十分に相談してください。特に、嘱託で依頼する場合は、産業医の本業とバランスがとれるように注意しましょう。
  
  
そのほか、産業医と雇用契約を結ぶケースや、訪問ごとにスポット契約を結ぶケースも挙げられます。
  
  
なお、契約の際は契約書を作成し、内容を書面で残しておきましょう。[日本医師会作成の産業医契約書例](https://www.med.or.jp/dl-med/doctor/ssi/sangyotool1a.pdf)や各医師会のテンプレートなどもあるので、契約書作成の参考にしてください。

### ③労働基準監督署に報告する

産業医との契約を締結後、所轄の労働基準監督署へ報告します。報告の際は、以下の書類を提出してください。

産業医選任報告の必要書類

●　産業医選任報告書  
●　医師免許の写し  
●　産業医の資格を証明する書類

産業医選任報告書は控えが出ないため、2部提出し、1部は控えとして報告後に受け取りましょう。

### ④助成金を活用する

国は健康で活力ある職場づくりのため、以下の助成金事業を実施しています（2023年度時点）。

|  |  |
| --- | --- |
| 助成金の名称 | 事業内容 |
| 団体経由産業保健活動推進助成金 | ●　事業主団体などをつうじて中小企業などの産業保健活動を支援する助成金  ●　一定の条件のもと産業医と契約した場合、活動費用の80%（上限100万円）が助成される |
| 小規模事業場産業医活動助成金 | ●　労働者数50人未満の事業場を対象とした助成金  ●　産業医と契約して産業医活動などを実施した場合、最大60万円の助成金が受けられる |

  
一定の要件がありますが、産業医の導入に際して助成を受けられる制度です。「産業医を導入したいけれども、費用が負担になる」場合は、積極的に活用しましょう。

## まとめ

従業員が健康で安心して働ける環境づくりは、人事労務管理上の重要なテーマです。産業医は健康面でのリスク管理や、従業員の仕事と治療の両立に役立ち、その重要性は増しています。
  
  
特に、従業員50人以上の事業場の場合、産業医の選任義務が課されています。また、50人未満の小規模企業でも努力義務が課されている状況です。助成金を活用しつつ、産業医の導入を検討しましょう。

## よくある質問

### 産業医とは？

産業医とは、労働者が健康的に働けるよう、アドバイスや指導を行う医師です。
  
  
産業医を詳しく知りたい方は「[産業医とは](#content1)」をご覧ください。

### 産業医の選任義務は？

常時50人以上の労働者を雇用する事業者は、産業医を選任する義務が課されます。
  
  
選任義務を詳しく知りたい方は「[産業医の選任義務](#content3)」をご覧ください。
