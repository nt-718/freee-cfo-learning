## カスハラ（カスタマーハラスメント）とは？

カスハラとはカスタマーハラスメントの略称で、顧客や取引先などから過剰な要求をされたり、商品やサービスに対して不当な言いがかりをつけられたりする迷惑行為を指します。
  
  
業界や企業によって顧客への対応方法や基準は異なるため、明確な定義は難しいものの、厚生労働省では以下のように説明しています。
  
  
「顧客などからのクレーム・言動のうち、当該行為・言動の要求の内容の妥当性に照らして、当該要求を実現するための手段・態様が社会通念上不相当なものであって、当該手段・態様により、労働者の就業環境が害されるもの」
  
  

[出典：厚生労働省「カスタマーハラスメント対策企業マニュアル」](https://www.mhlw.go.jp/content/11900000/000915233.pdf)
  
  
以上の説明から考えると、カスハラに該当する行為は大きく2つのケースに分けられます。

|  |  |
| --- | --- |
| ケース | 主な例 |
| 要求の内容が妥当性を欠いている場合 | ● 提供する商品・サービスに瑕疵（傷や欠点など）・過失が認められない  ● 要求内容が商品やサービスに関係ない |
| 要求実現のための手段・態様が社会通念上不相当な言動の場合 | 【内容の妥当性に関係なく不相当に該当する可能性が高い行為】  ● 暴行や傷害などの身体的な攻撃  ● 脅迫・中傷・名誉毀損などの精神的な攻撃  ● 威圧的な言動  ● 差別的・性的な言動  ● 継続的・執拗な言動  ● 土下座の要求  ● 不退去や居座りなどの拘束的な行動  ● 従業員への個人的な攻撃や要求    【内容の妥当性に照らして不相当に該当する場合がある行為】  ● 商品交換の要求  ● 金銭補償の要求  ● 謝罪の要求（土下座を除く） |

### カスハラの基準とクレームとの違い

クレームは、顧客や取引先がその企業に対して商品・サービスの改善や良品との交換、追加サービスの提供などさまざまな要求を行うことです。
  
  
しかし、正当性のあるクレームは企業にとって顧客満足度を高めるきっかけになります。そのため、クレームとカスハラの判断基準を明確にし、社内で共有しておく必要があるでしょう。
  
  
企業や業界によってカスハラやクレームの判断基準は異なりますが、ひとつの尺度として以下を考えるのが大切です。

カスハラの基準とクレームとの違い

● 要求内容の妥当性はあるか  
● 手段・態様が一般的または常識的な範囲内か

たとえば顧客が購入した商品に破損や欠陥が認められた場合、販売側が交換や返金に応じるのは、要求内容に対する妥当性があると考えられます。
  
  
一方、「使い勝手が悪いから返金してほしい」のような内容のクレームは正当性がないため、カスハラに該当すると考えられるでしょう。
  
  
また、クレームの内容に妥当性があっても、長時間におよぶ叱責や、「バカ」「低能」などの暴言、大声で執拗に責めるなど、言動が一般的または常識的な範囲内でない場合は、カスハラに該当する可能性があります。

## カスハラが増加した背景と現状

厚生労働省が実施した「令和2年度職場のハラスメントに関する実態調査」によると、「過去3年間にカスハラの相談があった」と回答した企業は19.5％あり、パワハラ、セクハラに続いて3番目に高い割合でした。
  
  
また、同調査の労働者に対する調査では「直近3年以内にカスハラを一度以上経験した」と回答した労働者は15.0％と、6~7人に1人はカスハラを経験している結果が出ています。

[出典：厚生労働省「令和2年度職場のハラスメントに関する実態調査」](https://www.mhlw.go.jp/content/11900000/000915233.pdf)

また、同調査の労働者に対する調査では「直近3年以内にカスハラを一度以上経験した」と回答した労働者は15.0％と、6~7人に1人はカスハラを経験している結果が出ています。
  
  
カスハラに悩む企業や労働者は少ないとはいえず、該当件数の推移も増加の傾向があります。
  
  
事実、「カスハラの相談があった」と回答した企業のうち、カスハラの相談が「減っている」と回答したのは12.1％、「増えている」と回答したのは19.4％でした。
  
  
カスハラが増加している原因はいくつか考えられますが、主に以下が挙げられます。

カスハラが増加している原因

● 日本の「おもてなし精神」を美徳とする考え方や企業にも「顧客第一主義」が浸透することにより消費者（顧客など）が権利の主張をしやすくなった  
● 携帯・スマホの普及やフリーダイヤル化など場所を選ばず即座に、無料で、クレームを入れられる環境が整っている  
● SNSにより他人のトラブルを目にしやすくなり、クレームを入れることの規範意識や抵抗感が低下している

時代の変化に伴いクレームを言いやすい環境が整ったことや、クレームを言うことへの規範意識や抵抗感の低下が、カスハラが増加している背景と考えられています。
  
  
今後もカスハラは増加する可能性があるため、企業はカスハラへの対策を考えておく必要があるでしょう。

## 企業によるカスハラ対策の必要性

カスハラは放置してしまうと、企業の経営や従業員、ほかの顧客へ悪影響をおよぼす可能性があります。以下に、カスハラがおよぼす影響をまとめました。

|  |  |
| --- | --- |
| 対象 | 影響 |
| 企業の経営 | ● クレーム対応による時間の浪費と金銭的損失  ● 顧客対応による他業務への支障、人員確保  ● ブランドイメージの低下　など |
| 従業員 | ● 業務パフォーマンスの低下  ● ストレスによる健康不良  ● 恐怖・苦痛による休職・退職　など |
| ほかの顧客 | ● 利用環境や雰囲気の悪化  ● 十分なサービスの提供が受けられない　など |

  
  
自社の大切な従業員を守り、経営に余計な負担をかけないためにも、カスハラは放置せずに事前の対策を行うことが大切です。

### カスハラ被害を受けた従業員から訴えられることも【事例】

労働契約法では企業の労働者に対する安全配慮義務が定められているので、労働者の安全を守るためにはカスハラにも適切な対応が求められます。
  
  
カスハラに対して不適切な対応をしてしまうと、場合によっては従業員から訴えられるケースもあるため、注意が必要です。
  
  
ある市立小学校では教諭が児童の保護者から理不尽な暴言を受けたとき、校長が事実確認を怠り不適切な対応をしたとして訴訟問題になった事例があります。結果として小学校を設置する市および給与を支払う県に対して損害賠償が命じられました。
  
  

[出典：厚生労働省「カスタマーハラスメント対策企業マニュアル」](https://www.mhlw.go.jp/content/11900000/000915233.pdf)
  
  
このようなケーススタディを知ることで、カスハラへの対応の必要性を認識できるでしょう。
  
  
なお、厚生労働省が告示している「[事業主が職場における優越的な関係を背景とした言動に起因する問題に関して雇用管理上講ずべき措置等についての方針](https://www.mhlw.go.jp/content/11900000/000584512.pdf)」によると、以下の取り組みが望ましいとされています。

望ましい取り組み

● 相談に応じ、適切に対応できる体制の整備  
● 被害者への配慮のための取り組み  
● 顧客からの著しい迷惑行為による被害を防止するための取り組み

[出典：厚生労働省「事業主が職場における優越的な関係を背景とした言動に起因する問題に関して雇用管理上講ずべき措置等についての方針」](https://www.mhlw.go.jp/content/11900000/000584512.pdf)

## カスハラに対して企業が取るべき対策

カスハラ対策は従業員や自社を守るための必須事項です。取り組みによって職場環境の改善につながったり、従業員に経験が蓄積されて迷惑行為への対応力が向上したりと、副次的な効果も期待できます。
  
  
以下では、企業がカスハラに対して取るべき具体的な対策を紹介します。
  
  
なお、freeeでは安心して働ける環境を作るために、カスハラへのさまざまな取り組みを行っています。カスハラ対策に取り組む企業の担当者様は「[カスタマーハラスメントに対するfreeeの考え方](https://corp.freee.co.jp/news/20230209freee_customer_harassment.html)」も参考にしてください。

### マニュアルの作成・研修の実施

カスハラ対策では、カスハラに対する方針を従業員に浸透させることが重要です。
  
  
顧客への対応方法やカスハラの判断基準は企業や業界によって異なりますが、社内で明確な基準や対応を定めてマニュアルを作成しましょう。他には定期的に研修を実施するなどして、カスハラに適切に対応できる環境を整えましょう。

### 専門窓口の設置

マニュアルの作成や研修を実施したとしても、従業員が現場でカスハラに直面したときに適切な対応ができるとは限りません。
  
  
カスハラ専門窓口の設置や、担当者を決めておくなど、従業員が気軽に相談できる環境を整えておくのも重要です。

### 被害者のケア

カスハラを未然に防ぐことも重要ですが、カスハラ被害を受けた従業員へのケアも大切です。カスハラで暴言を吐かれたり威圧されたりすると、従業員が精神的なダメージを受けてしまう可能性があります。
  
  
産業医や産業カウンセラーなどの専門家にメンタルケアを依頼したり、定期的なメンタルチェックを実施したりして、従業員への精神的なケアも行いましょう。

### 外部の専門機関との連携

カスハラの程度によっては法律に抵触する場合もあります。そのため、自社での対応が難しい内容に関しては、弁護士に対応を依頼するほうが適切なケースもあります。
  
  
また前述のように、被害者へのケアとして産業医などの専門家に相談できる体制を整えておくのも重要です。
  
  
カスハラ対策では社内での取り組みだけでなく、外部の専門家と連携できる仕組みを整えておくようにしましょう。

## まとめ

カスハラの基準は企業や業界によって異なるため、明確な定義はありません。一般的には過剰な要求をされたり、商品やサービスに対して不当な言いがかりをつけられたりする迷惑な行為がカスハラにあたります。
  
  
カスハラ対策は社内で対応方法や明確な基準を定め、マニュアルの作成や研修を行うなどして従業員に方針を浸透させることが大切です。
  
  
また、カスハラが発生した場合は、被害者へのケアや再発防止措置なども行う必要があるでしょう。十分なカスハラ対策を実施して、従業員が心身ともに安心して働ける環境を整えください。
