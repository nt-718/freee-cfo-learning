## 時間外労働の上限規制とは

法定労働時間を超えて時間外労働や休日労働をさせる場合は、企業と労働者の間で36協定と呼ばれる労使協定を結ばなければなりません。
  
  
時間外労働の上限が規制されると、**36協定や特別条項付きの36協定を結んでいても、一定の時間数を超えた労働は労働基準法違反となります。**違反時の罰則も設定されているため、企業は早急に対策しなければなりません。

### 2024年4月より建設業も改正労働基準法を適用

2024年4月以降は、建設業でも罰則付きの労働時間上限規制が適用されます。
  
  
2019年4月に施行された改正労働基準法でも、罰則付きの労働時間上限規制が導入されていましたが、建設業を含めた一部の業種には猶予期間がありました。
  
  
なお、法定労働時間は1日8時間・週40時間以内とし、毎週少なくとも1回または4週に4回以上の休日を与えなければならないと、労働基準法に定められています。
  
  
法定時間外労働を労働者にさせる場合、事業主は労働者と36協定を締結し、事業所を管轄する労働基準監督署に届け出なければなりません。

### 36協定での時間外労働に上限を設定

**時間外労働にも上限があり、月45時間・年間360時間までが原則です。**
  
  
また、時間外労働と休日労働の合計は月100時間未満かつ、2~6ヶ月平均がすべて、ひと月あたり80時間以内でなければなりません。

|  |  |
| --- | --- |
| 月の時間外労働上限 | 45時間 |
| 年間の時間外労働上限 | 360時間 |
| 時間外労働と休日労働の合計 | 月100時間未満 かつ 2~6ヶ月平均がすべて、ひと月あたり80時間以内 |

  
  
ただし、建設業では災害時の復旧・復興事業に携わる際の時間外労働には、例外的に上限規制の適用が除外されます。

### 特別条項付きの36協定にも上限を設定

改正された労働基準法では、特別条項付き36協定を締結し、届出した場合の時間外労働は年720時間以内、時間外労働と休日労働の合計は、月100時間未満と定めています。
  
  
また、月45時間を超える時間外労働ができるのは、年6回までです。

|  |  |
| --- | --- |
| 項目 | 条件 |
| 時間外労働（年間） | 720時間以内 |
| 時間外労働＋休日労働（月間） | 100時間未満 |
| 月45時間を超える時間外労働 | 年6回まで |

  
  
今までは上限の設定がなく、特別条項付き36協定を結び、届出をすれば何時間でも働かせられ、長時間労働が蔓延する要因でした。そのため、時間外労働の上限が設定されたことで労働環境の改善が期待されています。
  
  
なお、建設業で特別条項付き36協定を締結し、届出をする際は、「時間外労働・休日労働に関する協定届（適用猶予期間中における、適用猶予事業・業務。自動車運転者、建設業、医師等。）様式第9号の4」を使用します。
  
  
フォーマットは厚生労働省のサイトからダウンロードできます。
  
  

[厚生労働省：労働基準法関係主要様式](https://www.mhlw.go.jp/bunya/roudoukijun/roudoujouken01/)

### 違反時の罰則を設定

改正労働基準法では、違反時の罰則も盛り込まれ、6ヶ月以下の懲役または30万円以下の罰金が科されます。
  
  
たとえば、36協定で定める時間数にかかわらず、以下のような場合は6ヶ月以下の懲役または30万円以下の罰金の対象です。

罰金の対象となる場合

● 時間外労働と休日労働の合計時間が月100時間以上になった場合  
● 2～6ヶ月平均のうちいずれかが80時間を超えた場合

## 建設業の労働時間の現状と問題点とは？

建設業の労働時間は、他業種と比較して長い傾向があります。休日数も少なく、長時間労働が常態化しているなど、さまざまな問題も抱えています。

### 建設業は労働時間・出勤日数が多い傾向にある

近年、長時間労働の削減が課題とされ、各業界で労働時間・出勤日数を減らす取り組みがなされています。
  
  
しかし、建設業は他産業と比較して、労働時間の短縮や出勤日数の削減への取り組みがなかなか進んでいないのが現状です。
  
  
特に取引関係において立場の弱い中小企業等では、発注企業からの短納期要請などに応えようとするため、労働時間や出勤日数を減らしにくい問題を抱えています。

### 建設業は4週間あたりの休日数が少ない

建設業の企業は約65％が4週間あたりの休日が4日間で、4週間で休日が8日間ある企業は1割未満です。ほとんどの企業が、週休2日制を導入できていません。
  
  
しっかり休める体制が整っておらず、労働者に大きな負荷がかかりやすい状態にある点が問題視されています。

### 建設業は過労防止の重点業種にもなっている

厚生労働省が作成した、令和元年版の過労死等防止対策白書（※）で、建設業は過労防止の重点業種として挙がりました。
  
  
建設業では、労災認定を受けた精神障害事案の中で現場監督者の自殺事案が多く、長時間労働や仕事内容・量の大きな変化、2週間以上の連続勤務による休日の少なさがストレス要因です。
  
  

[※出典：厚生労働省「令和元年版 過労死等防止対策白書」](https://www.mhlw.go.jp/content/000553598.pdf)

## 時間外労働の上限規制以外に建設業で適用・推進すべき事柄

時間外労働の削減以外にも、割増賃金率の引き上げや有給休暇の取得促進、週休2日制の導入は、建設業でも適用・推進すべき事柄です。

### 割増賃金率の引き上げ

2023年4月からは月60時間を超える時間外労働に対して、50%以上の割増賃金率が適用されました。
  
  
2023年3月末までは中小企業に対する猶予期間があり、月60時間を超える時間外労働への割増賃金率は25%以上で済んでいました。
  
  
しかし、今後は建設業を含め、業種・企業規模を問わず、50%以上の割増賃金率で計算した残業代を支払わなければなりません。

|  |  |  |
| --- | --- | --- |
| 期間 | 対象企業 | 時間外労働（月60時間を超える） |
| 2023年3月末まで | 中小企業 | 25%以上 |
| 大企業 | 50%以上 |
| 2023年4月以降 | すべて | 50%以上 |

### 有給休暇の確実な取得

2019年4月より働き方改革の一環で、年10日以上の有給休暇が付与されている従業員には、年5日の有給休暇の取得が義務づけられました。
  
  
建設業の管理監督者も対象であり、有給休暇を取得させなかった場合は、30万円以下の罰金が科されます。
  
  
なお、年5日の有給休暇の時季指定義務分については、企業側が取得時季を指定できますが、従業員の意見を聴取したうえで、可能な限り従業員の希望に沿った時季に取得できるように意見を尊重するように努めなければなりません。
  
  
また、すでに年5日以上の有給休暇を請求・取得している従業員に対しては、企業が時季の指定はできません。
  
  
ただし、繁忙期であったり、同じ時期に何人もの従業員が休んだりして、業務に支障をきたすと判断されるなら、企業側はほかの時季に変更ができます。

### 週休2日制度の導入を推進

建設業では週休2日制度を導入していない企業が多くありますが、有給休暇を取得させる取り組みに加えて、週休2日制度の導入も検討したい事柄です。
  
  
週休2日制度の導入は休日確保のために推進されている事柄ですが、有給休暇の取得とは異なり、法的規制や罰則はありません。
  
  
しかし、人材を確保するために働きやすい職場環境を整える過程では、長時間労働の是正とともに、しっかり休める体制作りが大切です。
  
  
また、実現のためには公共団体やゼネコンなど民間企業も含めた発注者の理解と協力が不可欠です。

## 時間外労働の上限規制で企業に必要な対策とは

企業では、長時間労働をしなくても完成できる工期・工程の設定や、有給休暇取得の推進、ICTの活用や書類の簡素化による業務の効率化、生産性の向上などに取り組まなければなりません。

### 適切な工期・工程の設定

適正な工期設定や賃金水準の確保に向けて、発注者を含めた関係者と協議の場を設け、時間外労働をしなくても進行できる工程の設定が必要です。
  
  
納期遵守のためには多少時間外労働をしなければ間に合わない場面もあるでしょう。しかし、時間外労働の上限規制が適用となれば、残業ありきで工期や工程を設定していては、労働基準法に違反する恐れがあります。
  
  
工期や工程に無理のある要望を受けた場合は、現場の担当者や監督者だけに対応を任せず、企業として取引先に法や制度に対応するために必要な環境整備への理解と協力を求めなければなりません。

### 有給休暇の取得を促進

建設業に限らず年10日以上の有給休暇が付与されている従業員は、年5日の休暇取得が義務付けられています。
  
  
従業員が有給休暇を取得できているかを確認するため、管理や記録の手順を確立し、法令に基づき確実に推進しましょう。
  
  
有給休暇の取得を推進するには、休みを取りやすい職場の環境作りも重要です。企業風土や体質から、休暇取得を申し出しにくい雰囲気が生まれているなら、環境の改善が必須です。

### 業務の効率化

長時間労働を是正しつつ、今まで通りの仕事を受注して企業経営を継続させるには、業務の効率化、生産性向上への取り組みも欠かせません。
  
  
手間のかかる業務はツールや設備を整えて負担を減らし、生産性を高めましょう。
  
  
建設現場なら、機械を導入して作業の自動化・効率化を図れば、短時間・少人数での作業進行により工期の短縮が可能です。
  
  
設計図や施工図などを電子化すれば、印刷した図面を持ち歩かなくてよくなり、修正があっても書き直した図面をデータですばやく共有できます。
  
  
出退勤報告や作業日報の提出もオンラインで完結する体制にできれば、報告業務も簡略化できるでしょう。

## まとめ

2024年4月から建設業でも改正労働基準法が適用され、労働時間の上限が規制されます。36協定や特別条項付きの36協定を締結しても上限は適用され、違反時には罰則も定められています。
