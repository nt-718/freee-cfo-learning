## 代替休暇とは

**代替休暇とは労働基準法で定められている休暇制度で、残業に関連する制度のひとつです。**
  
まずは代替休暇の概要から解説します。

### 月60時間を超える残業をしたときに使える制度

**1ヶ月の残業時間が60時間を超えた場合、割増賃金を支払う代わりに休暇を与えることが認められています。割増賃金の代わりに与える有給の休暇が代替休暇です。**

[出典：厚生労働省「改正労働基準法のあらまし」](https://www.mhlw.go.jp/content/11200000/000972776.pdf)
  
  
残業をすると、25%以上の割増率で残業代を払わなければならず、残業時間が月60時間を超えるとさらに割増率が50%以上に引き上がりますが、引き上がる分の割増賃金の代わりに代替休暇を付与できます。
  
  
代替休暇制度は、月60時間を超える残業をした労働者が休みを取ってリフレッシュできるよう、労働者の健康を確保するための制度です。
  
  
ただし、代替休暇を取るか取らないかは労働者の自由で、休暇の取得を強制する制度ではありません。労働者が「代替休暇は取らない」と言えば、雇用者は通常どおり月60時間を超える残業時間分に対しては50%以上の割増率で残業代を支払わなければなりません。
  
  
また、残業代の支払いに代えて代替休暇取得に切り替えられるのは、残業が月60時間を超えたときに割増率が50%以上に引き上がる法定時間外労働分のみです。従業員が代替休暇を取得した場合でも、25%以上の割増賃金は支払わなければなりません。
  
  
法定割増賃金率については、別記事「[法定割増賃金率の引上げとは？2023年4月施行の法改正で中小企業が注意すべきポイント](https://www.freee.co.jp/kb/kb-payroll/raising-wage-rate/)」でも詳しく解説しています。

### 2023年4月からは中小企業も対象

従業員が月60時間超の残業をしたとき、残業代の割増率が高くなったり代替休暇制度の対象になったりする企業は、今までは大企業のみでした。しかし、2023年4月からは中小企業も対象です。
  
  
対象となる中小企業とは以下のいずれかに該当する企業です。

|  |  |  |
| --- | --- | --- |
| 業種 | 資本金の額または出資の総額 | 常時使用する労働者数 |
| 小売業 | 5,000万円以下 | 50人以下 |
| サービス業 | 5,000万円以下 | 100人以下 |
| 卸売業 | 1億円以下 | 100人以下 |
| 上記以外のその他の業種 | 3億円以下 | 300人以下 |

  
  
中小企業では2023年3月まで、残業時間が月60時間を超えても割増率は引き上がらず25%以上でしたが、2023年4月からは大企業と同じく50%以上に引き上がりました。
  
  
代替休暇制度は、残業時間が月60時間を超えて割増率が50%以上に引き上がるケースが対象の制度のため、それに伴い2023年4月からは中小企業も代替休暇制度の対象になりました。

## 代替休暇を導入するときに労使協定で定める事項

代替休暇制度を導入するためには、労働者の過半数で組織する労働組合、それがない場合には労働者の過半数を代表する者との間で労使協定を結びます。
  
  
労使協定とは労働者と使用者との間で結ばれる書面による協定です。以下では代替休暇を導入する際に労使協定で定める事項を解説します。

### 代替休暇の時間数の具体的な算定方法

労使協定で定める事項のひとつ目は「代替休暇の算定方法」です。残業代を支払う代わりに休暇を与える際、残業代が何日分の休暇にあたるのか、計算方法を具体的に定めなければなりません。代替休暇の計算式は以下の通りです。

● 代替休暇の時間数 = （1ヶ月の法定時間外労働時間数 - 60） × 換算率  
● 換算率 = 「代替休暇を取得しなかった場合に支払うとされている割増賃金率」 - 「代替休暇を取得した場合に支払うとされている割増賃金率」

法律で定められた最低限の水準で残業代の割増率を設定している企業の場合、残業が月60時間までであれば割増率は25%、月60時間超であれば50%なので、換算率は「50%-25%」で25%と計算できます。

### 代替休暇の単位

労使協定で定める事項の2つ目は「代替休暇の単位」です。代替休暇制度は、まとまった単位で休暇を与えて労働者に休息の機会を与えるための制度なので、1日または半日で与える必要があります。
  
  
1日・半日のいずれかを労働者が選べるルールにもできます。ただし、年次有給休暇のように時間単位では与えられません。
  
  
原則、労働者の1日の所定労働時間の半分を「半日」と指しますが、厳密に所定労働時間の半分とせず、たとえば午前の3時間半、午後の4時間半をそれぞれ半日として扱えます。その場合は労使協定でその旨を定めておきましょう。
  
  
また、代替休暇として与える時間数が1日または半日に満たない場合は、代替休暇以外の他の有給休暇とあわせて1日または半日の休暇を与えても問題ありません。

### 代替休暇を与えられる期間

労使協定で定める事項の3つ目は「代替休暇を与えられる期間」です。代替休暇は法律上、法定時間外労働が1ヶ月60時間を超えた月の末日の翌日から2ヶ月以内に与えなければなりません。
  
  
代替休暇は長時間残業をした労働者に休息の機会を与えることが目的なので、一定の期間内に与えなければいけません。たとえば5月に月60時間超の残業をしたケースであれば代替休暇の付与期限は7月末です。

### 代替休暇の取得日の決定方法、割増賃金の支払日

上述の事項に加えて、「代替休暇の取得日の決定方法」「割増賃金の支払日」も労使協定で定めるべきとされています。不明確なままだと、賃金の支払額が確定できずトラブルになる可能性があるためです。
  
  
取得日の決定方法に関しては、代替休暇を取得するか否かによって企業が支払う残業代の割増率が変わるので、いつまでに決定するのか労使協定に明確に記載しておきましょう。
  
  
たとえば「月末から5日以内に使用者が労働者に代替休暇を取得するか否かを確認し、取得の意向がある場合は取得日を決定する。期日までに意向が確認できなかった場合は、代替休暇を取得せずに、割増賃金を選択したものとみなす。」などと労使協定に記載します。
  
  
また、割増賃金の支払日に関しては、従業員が代替休暇を取る場合と取らない場合に分けたうえで、それぞれのケースでいついくら支払うのか、労使協定で定めます。
  
  
下図は厚生労働省の資料に掲載されている一例です。

* 賃金締切日が月末
* 支払日が翌月20日
* 代替休暇は22か月以内に取得
* 代替休暇を取得しなかった場合の割増賃金率50%
* 代替休暇を取得した場合の割増賃金率25%

[出典：厚生労働省「改正労働基準法のポイント」](https://www.mhlw.go.jp/stf/houdou/2r9852000000uefi-img/2r9852000000ugqj.pdf)

## 代替休暇の計算方法

前述のとおり、代替休暇は1ヶ月の残業時間と換算率を使って計算します。換算率で使う残業代の割増率は企業ごとに異なるので、代替休暇の日数を計算する際は各企業の就業規則等で定めた割増率を確認して計算するようにしてください。

【計算例】  
● 1ヶ月の残業時間：70時間  
● 割増賃金率：60時間までは30%、60時間超は60%  
● 代替休暇の時間数：（70時間 - 60時間） × （60% - 30%） = 3時間

## 代替休暇の導入で必要になる手続き

代替休暇を導入する際、必要になる主な手続きは「労使協定の締結」と「就業規則の変更および変更届の提出」の2つです。
  
  
以下ではそれぞれの手続きについて解説します。

### 労使協定を締結する

代替休暇を導入するためには労使協定を締結して前述の事項を定めます。労使協定を締結する際の流れは以下の通りです。

労使協定を締結する際の流れ

1. 労働組合の代表者、もしくは労働者の過半数を代表する者を管理監督者以外から投票、挙手など民主的な手続きで選出する  
2. 代替休暇の計算方法や取得単位、取得期間などを定めて労使協定を締結する  
3. 労使協定の内容を社内で周知する

なお、労使協定は一定の場合に締結が法律で義務付けられていますが、労働基準監督署への提出義務はありません。
  
  
そのため代替休暇の導入に際して労使協定を締結した場合でも労働基準監督署への届出は不要です。

### 就業規則に記載して変更届を提出する

休暇に関する事項は、就業規則への記載が義務付けられている事項のひとつです。代替休暇を導入するのであれば就業規則に記載しなければなりません。
  
  
常時10人以上の労働者を使用している事業場では労働基準監督署への届出が必要です。
  
  
届出では、以下の書類を労働基準監督署に提出します。

届出に必要な書類

● 従業員から聴取した意見書  
● 就業規則変更届  
● 変更後の就業規則または変更箇所・変更内容が明らかな書類（新旧条文対照表等）

就業規則変更届の提出期限は明確には定められていませんが、変更した場合は遅滞なく変更届の提出が義務付けられています。

## 代替休暇と代休の違いは？

代替休暇と混同しやすい言葉に代休があります。しかし、代替休暇と代休は全く異なる休みなので間違えないように留意するべきです。
  
  
代休とは休日労働を行った後、代わりに他の日を休みにする制度なので、「残業代の代わりに休みを取得する代替休暇」とは異なります。
  
  
代替休暇の場合は、従業員が休暇を取得すると残業代の支払いが一部不要になりますが、代休の場合は残業代の支払い義務はなくなりません。
  
  
代休を取って休んでも休日に労働した事実はなくならず、法定休日労働手当の場合には35%以上の割増賃金を企業が支払います。

|  |  |  |
| --- | --- | --- |
| 項目 | 代替休暇 | 代休 |
| 概要 | 残業代の代わりに休みを取得する制度 | 休日労働後、別の日を休みにする制度 |
| 残業代の支払い | 休暇取得により一部支払いが不要になる | 残業代の支払い義務がなくならない |
| 割増賃金 | ・残業代（25%以上の割増賃金。月60時間を超えると50%以上） | 休日労働手当（法定休日労働：35%以上の割増賃金） |

## まとめ

月60時間を超える残業をすると、割増賃金の支払いに代えて代替休暇を取得できます。2023年4月からは中小企業も代替休暇制度の対象になりました。
  
  
代替休暇を取得するかどうかは従業員が自由に決められますが、企業として代替休暇制度を導入するためには労使協定の締結や就業規則の変更をしなければなりません。
  
  
労使協定では法定の事項を定める必要があるので、代替休暇の計算方法や取得単位、取得期間などを労使で協議したうえで決定し、協定書に記載するようにしてください。
  
  
また、代替休暇制度では導入後、従業員ごとに時間数を計算したり取得日数を管理したりするなど、細かな対応が求められます。
  
  
各従業員の毎月の労働時間を踏まえて、代替休暇制度の対象者が誰なのか、取得できる時間数は何時間分になり、1日単位または半日単位で付与するのか、正しい理解が大切です。
