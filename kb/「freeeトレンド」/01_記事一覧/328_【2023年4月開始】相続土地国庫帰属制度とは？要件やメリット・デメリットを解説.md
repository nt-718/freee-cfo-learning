## 相続土地国庫帰属制度とは

2023年4月27日から、「相続土地国庫帰属制度」が始まりました。**相続土地国庫帰属制度とは、相続または遺贈によって土地を相続した人が、一定の要件のもと土地を国に引き渡せる制度です。**
  
  
土地を相続したものの、遠方で土地を利用できない・管理が負担、などの理由で手放したいと考える人が増えています。活用できないけれど手放せず土地が放置されると、「所有者不明土地」にもなり得ます。
  
  
誰が所有者なのか分からない所有者不明土地は、国土の20%超にものぼり、深刻な社会問題となっていました。そこで、所有者不明土地の発生を未然に防ぐ方策として設けられたのが「相続土地国庫帰属制度」です。

## 相続土地国庫帰属制度の要件

相続土地国庫帰属制度の申請対象者は、相続または相続人に対する遺贈によって土地を取得した人です。
  
  
土地の共有持分を取得した共有者も対象となります。売買など、相続以外の理由で土地を取得した場合は利用できません。
  
  
なお、制度開始前（2023年4月27日より前）に相続や遺贈で取得した場合も申請できます。

### 相続土地国庫帰属制度で引き渡せない土地

管理や処分に大きな費用や労力がかかる土地は、相続土地国庫帰属制度を利用して引き渡すことができません。引き渡せないのは、以下の却下事由・不承認事由に当てはまる土地です。

|  |  |
| --- | --- |
| 却下事由（申請ができない土地） | ●　建物がある土地  ●　担保権や使用収益権が設定されている土地  ●　他人の利用が予定されている土地  ●　特定の有害物質によって土壌汚染されている土地  ●　境界が明らかでない土地・所有権の存否や範囲について争いがある土地 |
| 不承認事由（承認が受けられない土地） | ●　一定の勾配・高さの崖があって、管理に過分な費用・労力がかかる土地  ●　土地の管理・処分を阻害する有体物が地上にある土地  ●　土地の管理・処分のために、除去しなければいけない有体物が地下にある土地  ●　隣接する土地の所有者等との争訟によらなければ管理・処分ができない土地  ●　そのほか、通常の管理・処分にあたって過分な費用・労力がかかる土地 |

  

[出典：法務省「相続土地国庫帰属制度のご案内」](https://www.moj.go.jp/content/001390195.pdf)
  
  
却下事由にあてはまる土地は、申請の時点で却下されます。不承認事由にあてはまる土地は、審査の際に該当とすると認められた場合に不承認となります。

## 相続土地国庫帰属制度を利用するメリット

相続土地国庫帰属制度で国に土地を引き渡す主なメリットは、以下の通りです。

相続土地国庫帰属制度を利用するメリット

●　管理の手間や税金の負担がなくなる  
●　不要な土地だけを手放せる  
●　土地の引き取り手を探さなくてよい

### 管理の手間や税金の負担がなくなる

相続土地国庫帰属制度を利用すれば、相続で取得したものの、活用や管理に困っている土地を国に引き取ってもらえます。
  
  
たとえば、亡くなった親から実家の土地を相続したけれど遠方に住んでいる場合などに、維持管理の負担から解放されます。土地を引き渡せば、維持管理にかかる費用や固定資産税、都市計画税の負担もありません。

### 不要な土地だけを手放せる

相続土地国庫帰属制度では、一筆（独立した一個の土地）単位で土地を手放せます。
  
  
相続した土地を活用できないとき、相続放棄をする方法もありますが、預貯金など不要な土地以外の資産も手放さなくてはなりません（※）。
  
（※）相続放棄とは、相続の開始を知った日から3ヶ月以内に家庭裁判所に申し立てて、被相続人の権利や義務を一切承継しない意思表示をすることです。
  
  
制度開始前は、使わない土地を含めて相続する、または不要な土地以外の資産も含めて相続放棄する方法しかありませんでした。しかし、相続土地国庫帰属制度を利用すれば、不要な土地だけを手放せます。

### 土地の引き取り手を探さなくてよい

相続土地国庫帰属制度を利用すれば、国に土地を引き渡せるため、引き取り手を自ら探す必要がありません。
  
  
活用しにくい土地は、引き取り手を見つけるのに苦労する傾向があります。
  
  
相続土地国庫帰属制度の場合、要件を満たしてさえいれば国が引き取ってくれます。とくに、引き取り手を見つけるのが難しい農地や山林も、申請の対象となるのは大きなメリットでしょう。
  
  
通常、農地を売却する際は農業委員会の許可が必要です。相続土地国庫帰属制度を利用して引き渡す場合、許可を取得する必要はありません。
  
  
また、引き渡し後は国が管理・処分するので安心です。

## 相続土地国庫帰属制度を利用する際のデメリット

相続土地国庫帰属制度の利用は、以下の注意点も踏まえて検討しましょう。

相続土地国庫帰属制度を利用する際のデメリット

●　一定の負担金がかかる  
●　共有者がいる場合は共有者全員で申請しなければならない

### 一定の負担金がかかる

相続土地国庫帰属制度を利用する際、管理費用の一部として国に一定の負担金を納めなくてはなりません。
  
  
負担金は、土地の性質に応じた標準的な管理費用を踏まえて算出した「10年分の土地管理費相当額」です。一筆20万円が基本ですが、土地の区分や面積、所在する地域に応じて負担金が算定される場合もあります。

|  |  |
| --- | --- |
| 土地の種類 | 負担金 |
| 宅地 | 面積にかかわらず20万円  （都市計画法の市街化区域または用途地域が指定されている地域内の土地は面積に応じて算定） |
| 田、畑 | 面積にかかわらず20万円  （都市計画法の市街化区域または用途地域が指定されている地域、農用地区域の田、畑は面積に応じて算定） |
| 森林 | 面積に応じて算定 |
| その他 | 面積にかかわらず20万円 |

  

[出典：政府広報オンライン「相続した土地を手放したいときの「相続土地国庫帰属制度」」](https://www.gov-online.go.jp/useful/article/202303/2.html)
  
  
負担金の額を知りたいときは、法務省ホームページで公開されている「[自動計算シート](https://www.moj.go.jp/MINJI/minji05_00471.html#:~:text=%E8%B2%A0%E6%8B%85%E9%87%91%E9%A1%8D%E3%81%AE%E8%87%AA%E5%8B%95%E8%A8%88%E7%AE%97%E3%82%B7%E3%83%BC%E3%83%88%EF%BC%88%EF%BC%A5%EF%BD%98%EF%BD%83%EF%BD%85%EF%BD%8C%E3%83%95%E3%82%A1%E3%82%A4%E3%83%AB%EF%BC%89%E3%81%AF%E3%81%93%E3%81%A1%E3%82%89)」も活用しましょう。
  
  
なお、相続放棄を選択した場合は、不要な土地以外の資産もすべて手放すことになります。一方で、相続土地国庫帰属制度の負担金のような経済的負担はありません。

### 共有者がいる場合は共有者全員で申請しなければならない

引き渡したい土地が共有地の場合は、共有者全員（相続や遺贈によって持分を取得した相続人を含む）で申請しなければなりません。
  
  
相続以外の理由で土地を取得した場合、本来は申請権を持ちません。しかし共有地である場合、相続した当人以外の共有者は、相続以外の理由で持分を取得した場合でも申請できます。

## 相続土地国庫帰属制度を利用する手順

[出典：政府広報オンライン「相続した土地を手放したいときの「相続土地国庫帰属制度」」](https://www.gov-online.go.jp/useful/article/202303/2.html)
  
  
相続土地国庫帰属制度を利用する際は、法務局への申請手続きが必要です。手続きは、以下の流れで進めます。

相続土地国庫帰属制度を利用する手順

1　事前相談する  
2　法務局に承認申請書を提出する  
3　法務大臣による審査・承認を受ける  
4　一定の負担金を国に納める  
5　土地が国庫に帰属される

### ①事前相談する

申請の前に、事前相談を利用しましょう。法務省の「[法務局手続案内予約サービス](https://www.legal-ab.moj.go.jp/houmu.home-t/top/portal_initDisplay.action)」から土地の所在する法務局（本局）を選択すれば、事前相談の予約が取れます。相談方法は対面または電話、相談時間は1回あたり30分です。
  
  
相談の前に、以下の資料を用意しましょう。

事前相談に必要な資料

●　相続土地国庫帰属相談票  
●　相談したい土地の状況（チェックシート）  
●　国に引き渡したい土地の状況等がわかる資料や写真

「国に引き渡したい土地の状況等がわかる資料や写真」とは、以下のような資料・写真です。

国に引き渡したい土地の状況等がわかる資料や写真の具体例

●　登記事項証明書または登記簿謄本  
●　法務局で取得した地図または公図  
●　法務局で取得した地積測量図  
●　その他土地の測量図面  
●　土地の現況・全体がわかる画像や写真

また、相続土地国庫帰属相談票、相談したい土地の状況（チェックシート）は、[法務省ホームページ](https://www.moj.go.jp/MINJI/minji05_00498.html)にてダウンロードが可能です。
  
  
事前相談では、相続土地国庫帰属制度の対象となるか知りたい、申請に必要な書類に漏れがないか確認してほしいなど、制度の利用に関する具体的な内容を相談できます。

### ②法務局に承認申請書を提出する

次に、申請に必要な書類を作成・用意します。

|  |  |
| --- | --- |
| 作成する書類 | ●　承認申請書  ●　承認申請に係る土地の位置及び範囲を明らかにする図面  ●　承認申請に係る土地と当該土地に隣接する土地との境界点を明らかにする写真  ●　承認申請に係る土地の形状を明らかにする写真 |
| 用意する書類 | ●　申請者の印鑑証明書  ●　固定資産税評価額証明書（任意）  ●　承認申請土地の境界等に関する資料（あれば）  ●　現地案内図（申請土地にたどり着くことが難しい場合）  ●　その他相談時に提出を求められた資料 |

  

[出典：法務省「相続土地国庫帰属制度のご案内」](https://www.moj.go.jp/content/001390195.pdf)
  
  
書類の作成方法や写真の撮影方法は、法務省の「[相続土地国庫帰属制度のご案内](https://www.moj.go.jp/content/001390195.pdf)」で確認しましょう。申請書類の作成は、弁護士・司法書士・行政書士に限り、依頼できます。
  
  
申請先は、引き渡したい土地の所在する法務局（本局）です。支店や出張所には提出できないのでご注意ください。
  
  
提出方法は、窓口と郵送のいずれかです。窓口で提出する場合は、なるべく事前に連絡しましょう。基本は申請者本人が来庁しますが、家族でも問題ありません。
  
  
郵送で提出する場合は、国庫帰属の申請書を書留郵便またはレターパックプラスに入れて、法務局の本局に送付します。
  
  
なお、申請の際、申請書に審査手数料額（土地一筆あたり14,000円）に相当する額の収入印紙を貼って審査手数料を納めます。審査の結果、承認が得られなかった場合も、手数料は返還されません。

### ③法務大臣による審査・承認を受ける

申請後、法務大臣（法務局）が書面調査・実地調査を行います。申請対象の土地にたどり着くのに案内が必要な場合などは、同行を求められるケースもあります。
  
  
申請から帰属の決定までにかかる期間は、半年～1年程度です。

### ④一定の負担金を国に納める

法務大臣による要件審査によって国庫帰属の承認を受けた人は、政令で定められた負担金を納めなくてはなりません。
  
  
申請が承認されると、法務局から申請者に負担金の通知・負担金の納付に関する納入告知書が送付されます。
  
  
通知が到達した翌日から30日以内に、納入告知書を添え日本銀行の本店や代理店、都市銀行・ゆうちょ銀行などの歳入代理店へ納付しましょう。
  
  
期限内に納付できなかった場合、国庫帰属の承認は失効します。再度申請手続きをしなければならなくなるため、必ず期限までに納めましょう。

### ⑤土地が国庫に帰属される

負担金を納付した時点で、土地の所有権が国に移ります。所有権移転登記は国が行ってくれるので、申請者側での実施は不要です。
  
  
国庫に帰属した土地は、国によって管理・処分されます。

## まとめ
