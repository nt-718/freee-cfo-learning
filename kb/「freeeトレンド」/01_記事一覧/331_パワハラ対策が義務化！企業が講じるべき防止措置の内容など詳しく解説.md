## パワハラ（パワーハラスメント）とは

パワハラとは、職務上の立場の優位性を利用し、業務上必要な範囲を超えて身体的または精神的な苦痛を与える言動です。具体的には、以下3つを満たす言動が「パワハラ」と定義されます。

パワハラの定義

1　職務上の立場の優位性を背景に行われるもの  
2　業務の適正範囲を超えて行われるもの  
3　身体的もしくは精神的な苦痛を与えるもの、または就業環境を害するもの

なお、厚生労働省が定めるパワハラの概念には、同僚や部下による上司への当該行為（これを「逆パワハラ」と言います。）も含まれます。
  
  
パワハラに該当し得る言動は、以下6つの類型に分けられます。

パワハラの6つの類型

●　身体的な攻撃  
●　精神的な攻撃  
●　人間関係からの切り離し  
●　過大な要求  
●　過少な要求  
●　個の侵害

以下でパワハラになり得る例、ならない例を紹介しますが、実際にパワハラにあたるかどうかは個別に判断される点にご留意ください。

### 身体的な攻撃

[出典：厚生労働省 あかるい職場応援団「ハラスメントの類型と種類」](https://www.no-harassment.mhlw.go.jp/foundation/pawahara-six-types/)
  
  
「身体的な攻撃」は、相手の体に危害を加える言動です。
  
  

|  |  |
| --- | --- |
| パワハラに該当し得る例 | ・上司が部下に対して殴る、蹴る  ・相手に物を投げつける |
| パワハラに該当しないと考えられる例 | ・同僚間で業務上関係のない喧嘩をした |

  
  
同僚間での業務上関係のない喧嘩は、優位性を利用した言動ではないため、パワハラには該当しないと考えられます。

### 精神的な攻撃

[出典：厚生労働省 あかるい職場応援団「ハラスメントの類型と種類」](https://www.no-harassment.mhlw.go.jp/foundation/pawahara-six-types/)
  
  
「精神的な攻撃」は、人格を否定するような発言や脅迫するような言動など、精神的な攻撃を与えるパワハラです。
  
  

|  |  |
| --- | --- |
| パワハラに該当し得る例 | ・必要以上に長時間にわたる叱責を繰り返す  ・ほかの従業員の前で厳しい叱責を繰り返し行う |
| パワハラに該当しないと考えられる例 | ・何度も遅刻を繰り返し、再三注意しても直らない従業員に上司が強く注意する |

  
  
再三注意しているにもかかわらず遅刻が直らない従業員への注意は、一般的に業務上必要な範囲だとみなされます。

### 人間関係からの切り離し

[出典：厚生労働省 あかるい職場応援団「ハラスメントの類型と種類」](https://www.no-harassment.mhlw.go.jp/foundation/pawahara-six-types/)
  
  
「人間関係からの切り離し」は、仲間外れや無視をするなどの言動をして特定の個人を孤立させようとする言動です。
  
  

|  |  |
| --- | --- |
| パワハラに該当し得る例 | ・特定の従業員に対して1人だけ離れた席で仕事をさせる  ・同僚が集団で無視する  ・業務上必要な連絡事項を伝えない |
| パワハラに該当しないと考えられる例 | ・従業員の教育のため、短期間個室で研修を実施する |

  
  
上記のように、教育を目的に短期間個室で研修を実施する行為は、業務上必要な範囲とみなされ、パワハラには該当しないと考えられます。

### 過大な要求

[出典：厚生労働省 あかるい職場応援団「ハラスメントの類型と種類」](https://www.no-harassment.mhlw.go.jp/foundation/pawahara-six-types/)
  
  
「過大な要求」は、業務上明らかに必要のないことや到底遂行できないようなことを強制する言動です。
  
  

|  |  |
| --- | --- |
| パワハラに該当し得る例 | ・十分な教育をされていないにもかかわらずこれまで経験したことのない業務に就かせる  ・業務上明らかに不要な雑用の対応を強制する  ・休日出勤を強いる |
| パワハラに該当しないと考えられる例 | ・従業員育成のためにややレベルの高い業務を任せる |

  
  
育成を目的に現在の業務よりもややレベルの高い業務を任せる行為は、一般的に業務上必要な範囲だとみなされ、パワハラには該当しません。

### 過少な要求

[出典：厚生労働省 あかるい職場応援団「ハラスメントの類型と種類」](https://www.no-harassment.mhlw.go.jp/foundation/pawahara-six-types/)
  
  
「過少な要求」は、本人の能力や経験に見合わない質の低い業務をさせる、あえて仕事を与えないなどの言動です。
  
  

|  |  |
| --- | --- |
| パワハラに該当し得る例 | ・特定の従業員を退職させるために、営業職の従業員に書類整理だけを命じる  ・気に入らない部下に対して、仕事を一切与えない |
| パワハラに該当しないと考えられる例 | ・業務上の理由で一時的に誰でもできる業務を任せる |

  
  
故意に仕事を与えないのはパワハラに値しますが、上記のように一時的に簡単な業務を任せるのはパワハラには該当しません。

### 個の侵害

[出典：厚生労働省 あかるい職場応援団「ハラスメントの類型と種類」](https://www.no-harassment.mhlw.go.jp/foundation/pawahara-six-types/)
  
  
「個の侵害」は、プライベートに過度に立ち入るパワハラです。
  
  

|  |  |
| --- | --- |
| パワハラに該当し得る例 | ・勤務時間外にSNSで休日をどのように過ごすのか聞き出す  ・私物を撮影する  ・スマホを勝手に見る  ・機微な個人情報を本人の同意を得ずほかの従業員に勝手に話す  ・しつこく飲み会に誘う |
| パワハラに該当しないと考えられる例 | ・従業員の家庭状況に配慮するためにヒアリングを行う |

  
  
従業員の家庭状況への配慮は、業務上必要であるため、パワハラに該当しないとの考えが一般的です。

## パワハラ防止法の改正でパワハラ対策が義務化

**2020年6月1日に改正パワハラ防止法（労働施策総合推進法）が施行され、2022年4月から全企業にパワハラ防止措置が義務化されました。**パワハラ防止法とは、パワハラを防止し、職場環境をよりよくするための法律です。
  
  
職場でのパワハラは、深刻な社会問題といえます。厚生労働省の「職場のハラスメントに関する実態調査」（2020年）によると、過去3年以内にパワハラを受けたと回答した人は、31.4％と約3割にのぼる結果でした。
  
  

[出典：厚生労働省「職場のハラスメントに関する実態調査」](https://www.mhlw.go.jp/content/11200000/000775817.pdf)
  
  
深刻な背景を踏まえ、より具体的なパワハラ対策措置を定めて、ハラスメントのない職場を実現するために実施されたのが本改正です。

## 企業に義務付けられたパワハラ防止対策の内容

パワハラ防止法の改正で企業に義務付けられた具体的なパワハラ防止措置は、大きく4つに区分されます。

4つのパワハラ防止措置

●　事業主の方針等の明確化および周知・啓発  
●　相談に応じ、適切に対応するために必要な体制の整備  
●　職場におけるパワハラに関する事後の迅速かつ適切な対応  
●　併せて講ずべき措置

### 事業主の方針等の明確化および周知・啓発

事業主は、パワハラに関する方針等を明確化して就業規則などで定め、従業員に周知・啓発しなければなりません。

事業主が定める方針等

●　職場でのパワハラの内容  
●　パワハラを行ってはいけない旨の方針  
●　パワハラを行った者に対して厳正に対処する旨の方針  
●　パワハラを行った者に対する対処の内容

社内報やパンフレットで周知する場合は、従業員一人ひとりに周知されるよう、配布方法を工夫する必要があります。

### 相談に応じ、適切に対応するために必要な体制の整備

事業主は、職場でのパワハラの相談に応じ、適切に対応するために必要な体制を整えなくてはなりません。講ずべき具体的な措置は、以下の2つです。

|  |  |
| --- | --- |
| 必要な措置 | 具体例 |
| 相談窓口をあらかじめ定めて労働者に周知する | ・相談に対応する担当者を決めておく  ・相談に対応するための制度を設ける  ・相談への対応を外部機関に委託する |
| 従業員の相談内容や状況に応じて相談窓口担当者が適切に対応できるようにする | ・留意点をまとめたマニュアルに基づいて対応できるようにしておく  ・相談への対応に関する研修を行う |

### 職場におけるパワハラに関する事後の迅速かつ適切な対応

職場でパワハラが起きてしまったときは、以下4つの措置をとらなければなりません。

パワハラに関する事後措置

●　事実関係を迅速かつ正確に確認する  
●　速やかに被害者に対する配慮のための措置を適正に行う  
●　事実関係の確認後、行為者に対する措置を適正に行う  
●　再発防止に向けた措置を講ずる

事実関係は、相談者・行為者の双方から確認します。相談者の心身の状況に配慮しながら実施しましょう。
  
  
相談者と行為者に主張の不一致があり事実確認が十分でない場合は、第三者からも聞き取りをするなどの対応が求められます。

### 併せて講ずべき措置

事業主は、二次的な被害を防止するために、以下2つの措置を講じなければなりません。

併せて講ずべき2つの措置

●　相談者やパワハラ行為者などのプライバシーを保護するために必要な措置を講じ、労働者に周知する  
●　相談したことなどを理由に、解雇その他不利益な取り扱いをされない旨を定め、労働者に周知・啓発する

なお、パワハラの相談をしたことを理由に不利益な取り扱いをする行為は、パワハラ防止法で禁止されています。

## パワハラ以外のハラスメントと防止対策

職場で起こりやすい、パワハラ以外の代表的なハラスメントと防止するための対策を紹介します。

職場で起こりやすい代表的なハラスメント

●　セクハラ（セクシュアルハラスメント）  
●　マタハラ（マタニティハラスメント）  
●　ジェンハラ（ジェンダーハラスメント）

### セクハラ（セクシュアルハラスメント）

セクハラとは、職場内のほかの者を不快にさせる性的な言動です。
  
  
異性だけでなく、同性に対する言動もセクハラの対象です。また、上司だけでなく同僚・取引先・顧客もセクハラ行為者となり得ます。
  
  
セクハラの類型は、大きく以下2つに区分されます。

|  |  |
| --- | --- |
| 対価型セクハラ | 性的な言動に対する拒否や抵抗が原因で、従業員が不利益を受けること |
| 環境型セクハラ | 性的な言動によって、労働者が就業するうえで見過ごせない程度の支障が生じること |

  
  
たとえば、解雇・降格・減給する、昇進・昇格の対象から外すなどの行為は、対価型セクハラに該当します。また、上司が腰や肩を繰り返し触る行為は、環境型セクハラです。
  
  
男女雇用機会均等法では、事業主にセクハラ防止措置の実施を義務付けています。企業は、パワハラと同様に以下のような防止措置を講じなくてはなりません。

企業が講ずべきセクハラ防止措置

●　事業主の方針の明確化および周知・啓発  
●　相談に応じ、適切に対応するために必要な体制の整備  
●　職場でのセクシュアルハラスメントにかかる事後の迅速かつ適切な対応  
●　併せて講ずべき措置

パワハラ対策と併せて、適切な対策を実施しましょう。

### マタハラ（マタニティハラスメント）

マタハラとは、妊娠や出産、育児休業などを理由に精神的・肉体的な苦痛を与える言動です。大きく以下2つの類型に区分されます。

|  |  |
| --- | --- |
| 制度等の利用への嫌がらせ型 | 出産・育児・介護に関する制度の利用を阻害する行為 |
| 状態への嫌がらせ型 | 出産・育児による就労状況の変化に対して、嫌がらせや不利益な扱いを行う、もしくはその示唆をする行為 |

  
  
マタハラとなり得るのは、以下のような言動です。

マタハラとなり得る言動の例

●　男性従業員が育児休業の取得を相談すると、「育児休業は女が取得するものだ」と言われ諦めた  
●　「こんな忙しいときに妊娠するなんて」と繰り返し言われた

育児休業などの相談を理由に不利益な取り扱いをする行為は、男女雇用機会均等法、育児・介護休業法の改正によって2019年から禁止されています。
  
  
また、一般的にパワハラ・セクハラ・マタハラをあわせて職場の3大ハラスメントと言い、企業はセクハラ同様マタハラに関しても防止措置を講じなければなりません。

### ジェンハラ（ジェンダーハラスメント）

ジェンハラとは、性に関する差別意識などから起こる不当な嫌がらせです。たとえば、以下のような発言が該当します。

ジェンハラとなり得る発言の例

●　「この仕事は女にはできない」  
●　「女なのに気が利かない」  
●　「男なのに頼りがいがない」  
●　「男ならもっとお酒を飲め」

性的な言動を行うセクハラに対し、ジェンハラは「性別に対する固定概念」にもとづく言動を指し、価値観の違いなどが原因で起きます。
  
  
未然に防ぐために、以下のような対策を行いましょう。

ジェンハラ対策の例

●　知識を深める社内研修を実施する  
●　ジェンハラの内容を明確にし、周知・啓発する  
●　社内アンケートを実施し、ジェンハラの問題を把握する

## パワハラが起きないような職場環境を整えることが大切

職場でパワハラが起きてしまったときの事後対処は重要ですが、そもそもパワハラが起きないような職場環境を整えるのが理想です。
  
  
パワハラを未然に防ぐため、パワハラの原因や背景となる要因を解消する取り組みを行いましょう。

パワハラ予防の例

●　コミュニケーションを活性化させるための研修を実施する  
●　パワハラの正しい知識を身に付けるための研修を実施する  
●　パワハラに関する明確な社内ルールを決めておく  
●　パワハラに対するアンケートを実施する

自覚なく行為者になってしまう可能性もあるため、社内でハラスメントに対する理解を深めましょう。
  
  
また、職場で円滑な人間関係を築けるよう積極的にコミュニケーションの時間を作ることも大切です。親睦会を開く、上司と部下で一緒にランチをとるなど、できる取り組みから始めていきましょう。

パワハラは、職場における立場の優位性を利用した不当な嫌がらせです。職場での能力の発揮を妨げるだけでなく心身を傷つける言動であり、社会通念上許されるものではありません。
  
  
**パワハラ対策が喫緊の課題となるなかで、2022年4月からすべての企業にパワハラ防止措置が義務付けられました。**企業は、パワハラに対する方針を周知・啓発する、相談窓口を設けるなどの措置を講じ、パワハラが起きないような職場環境づくりを行いましょう。

### パワハラ防止対策として企業は何を行えばよい？

2022年4月からすべての企業に対して義務化されたパワハラ防止措置は、大きく4つです。
●　事業主の方針等の明確化および周知・啓発  
●　相談に応じ、適切に対応するために必要な体制の整備  
●　職場におけるパワハラに関する事後の迅速かつ適切な対応  
●　併せて講ずべき措置
