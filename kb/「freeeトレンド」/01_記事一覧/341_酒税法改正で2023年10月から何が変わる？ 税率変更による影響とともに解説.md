## 酒税法改正に伴う2023年10月の酒税変更とは

酒税法とは、お酒の分類・製造・販売の免許・酒税の税率・徴収などを定めた法律です。明治以降、大きな財源となり、古くから国の重要な財源のひとつでした。
  
  
酒税法の対象は、原則としてアルコール分1度以上の「酒類」です。酒類は製法などの違いから4種類に分類され、下記の通り品目ごとに異なる税率が適用されています。

[出典：財務省「酒税に関する資料」](https://www.mof.go.jp/tax_policy/summary/consumption/d08.htm)
  
  
現在注目されている2023年10月の酒税変更は、2017年の酒税法改正に基づき実施されます。

### ビールの税率は2026年に一本化へ

この法改正は、2026年10月までに、複雑に分類されてきた酒税を段階的に簡素化する観点から行われました。2023年10月に実施される改正内容は、2020年10月に続く2回目のビール系飲料の税率変更です。
  
  
2020年の酒税法改正には、似た特性をもつお酒に対して公平な税率を適用する狙いがありました。
  
  
たとえばビール系飲料は、原料や製法の違いによって、ビール、発泡酒、新ジャンルの3種類に分かれています。

|  |  |  |  |
| --- | --- | --- | --- |
| 種類 | | 酒税（350mlあたり） | 原料 |
| ビール | | 70円 | 麦芽や定められた副原料を使用 |
| 発泡酒 | 麦芽比率50％以上 | 70円 | ● 麦芽やビールの製造に認められない副原料を使用  ● ビールの副原料を規定量を超えて使用 |
| 麦芽比率25％以上50％未満 | 58円 |
| 麦芽比率25％未満 | 47円 |
| 新ジャンル | | 38円 | ● 麦芽を使用せず大豆やえんどうを使用  ● 麦芽比率50％未満の発泡酒にスピリッツを加えたものなど |

[出典：財務省「酒税に関する資料」より一部抜粋（小数点以下切り上げ）](https://www.mof.go.jp/tax_policy/summary/consumption/d08.htm)
  
  
ビールと似た見た目や味わいをもつビール系飲料は、税率の分類が複雑です。
  
  
麦芽比率50％以上の発泡酒・麦芽比率25～50％未満の発泡酒・麦芽比率25％未満の発泡酒と、税率は3種類にも分かれています。
  
  
しかし、2020年、2023年の税率変更を経て、**2026年にはビール系飲料は税額54.25円（350mlあたり）に一本化される予定**です。
  
  
税額を350mlあたりで換算すると、ビール系飲料の酒税はビールが70円から63.35円へ引き下げ、麦芽比率25％未満の発泡酒が46.99円のまま据え置き、新ジャンル（第3のビール）が37.8円から46.99円へ引き上げになります。
  
  
同時期に、清酒や果実酒などのワインや、日本酒に当たる醸造酒類の税率も一本化されます。チューハイなどその他の発泡性酒類においても、2026年10月に引き上げを予定しています。
  
  
これにより税制の簡素化が向上し、消費者や業界にとってもよりわかりやすくなることが期待されています。
  
  

[出典：財務省「酒税に関する資料」](https://www.mof.go.jp/tax_policy/summary/consumption/d08.htm)

## 酒税法改正の歴史と背景

日本の酒税には、長い歴史があります。室町時代から現在まで、酒税は国を動かす財源として重要視されてきました。しかし、酒税法を制定する目的は、過去と現在では異なります。
  
  
室町時代や鎌倉時代、政府はお酒を飲みすぎて民衆が堕落するのではないかと不安視していました。そのため、当初は販売禁止令を出していましたが、税収への期待から壺銭などの酒税が開始された背景があります。
  
  
しかし明治時代に入ると、軍事費増強へと酒税法の目的が変わり、1878年には現在と同様の「酒類に応じて税率が変動する」仕組みへ改正されました。  
戦争が続いて酒蔵の作り手が減り、生産量が減ってしまうと、税収減を補うための酒税の税率アップも行われました。
  
  
その後、酒税法が全文改正され、現在の酒税法が制定されたのは1953年です。その後、1989年の法改正では、ウィスキーなど高価格な物品に対する従価税と級別制度が廃止されました。
  
  
この廃止により貴金属など多くの高級品は減税に向かいましたが、お酒は高級品とみなされたまま高い税率を維持されます。2006年に10種類に分かれていた酒類が現行の4種類になるなど、近年は酒税法の簡素化が進んでいます。

## 酒税法改正の目的

2023年の酒税法改正は、酒税制度の公平性の確保を目指す重要な取り組みです。日本の酒類市場の発展と、消費者の利益を考慮した税制の構築を目的としています。
  
  
そこで、わかりやすく公平な酒税を観点とした法改正により、国は酒税の一本化を進めている状況です。
  
  
酒税は、酒類の製造や販売に課せられる税金であり、国や地方自治体の財源の一部です。酒類の消費を調節する役割も担っており、健康や社会的な問題への配慮も考慮されています。
  
  
日本の酒類販売数は1996年をピークに減少傾向です。理由としては、若い世代の酒離れのほか、複雑な税率による商品開発などの影響だと指摘されています。
  
  
背景には、ビールと似た飲料が多様化し、税率の分類が複雑になったことも挙げられます。税率の差異が商品の選択や市場の健全な競争に影響を与える可能性があるため、より明確な税制の確立が必要です。

## 酒税法改正がもたらす影響

酒税法改正により、2020年、2023年、2026年と段階的に税率の変更が実施されています。こうした税率変更により、市場や消費者にもたらされる影響を具体的に解説します。

### 発泡酒と新ジャンルの市場が縮小

2023年10月の税率変更によってビールが減税される一方、安さが魅力の新ジャンルが増税されて発泡酒と同じ税額になります。
  
  
改正により、同じビール系飲料のなかで税率の違いにより分けられてきた3つのジャンルの価格差が接近しました。そのため、低価格で新ジャンルを選んできた消費者の買い控えにより、ビール系飲料の市場が縮小すると予測されています。
  
  
そうした予測も踏まえ、発泡酒や新ジャンルは安さ以外に機能性や味わいなどが強化され、ビールとの差別化が進むと思われます。またビール会社各社は、減税により追い風を受けるビールの販売に力を入れるでしょう。

### チューハイの需要が上昇・ワインへの影響は

税率の変更はビール系飲料だけではなく、チューハイを含む「その他の発泡性酒類」でも実施されます。現在の税額は350mlあたり28円ですが、2026年10月には35円へとアップする予定です。
  
  
2026年にビール系飲料が一本化されたあとと比べても、ビール系飲料54.25円とチューハイ35円とのあいだには20円近い税額の開きがあります。そのため、ビール系飲料よりもお手頃なチューハイやサワーへ消費者の需要が流れると予想されます。
  
  
前回ビールと新ジャンルの税額が変更された2020年10月以降、新ジャンルの販売シェアは減少し、チューハイは増加しました。新ジャンルとチューハイは価格帯が近く、2023年10月の税率変更でも、代替品としてチューハイへ移行する消費者は多いと考えられます。
  
  
また、ワインや日本酒などの醸造酒類の税率も一本化される予定です。これによって、ほかの酒類と同様に税率の差異が解消され、より統一的な税制が導入される見通しです。

### 飲食店でも税率変更への対応が必要な可能性

酒税の税率変更の影響は、メーカーや消費者だけではなく、お酒を扱う飲食店にも及ぶ可能性があります。
  
  
酒税はメーカーの販売時あるいは輸入時に課税されます。飲食店は通常、酒税を申告する義務はありませんが、税率が変更されるタイミングだけは例外的に「酒類の手持品課税（戻し税）」が実施されるため注意が必要です。
  
  
2023年の税率変更の場合、改正時点で在庫のあるお酒に対して、税率を引き上げられた酒類には課税が生じ、引き下げられた酒税には還付が行われます。
  
  
ただし、1,800ℓ以上のお酒を保有する飲食店が対象なので、個人店の多くは申告しなくても済むでしょう。申告する場合は10月31日が期限ですので、早めに確認しておくと安心です。

## 酒税法改正による課題

酒税法改正では、税率の高さが常に指摘されてきました。とくにビールに対する税金は、かつては高級品として扱われていたために、富裕層をターゲットとした税金として高い税率が維持されてきた経緯があります。
  
  
消費税が導入された1989年、自動車や貴金属などの高級品に課される個別間接税は、段階的に廃止されました。しかし、たばこやガソリンなどとともに、お酒への課税は今も残されています。
  
  
ビールは現在では大衆の飲み物としての地位を獲得しており、その課税に関しては疑問視される声があがってきました。特に酒税と消費税の二重課税の問題が指摘されています。
  
  
国は、メーカーが酒税を支払ったあとに販売価格を設定し、その中に税金が含まれるため、消費者への課税に問題はないとしています。
  
  
しかし、法改正後も変わらず高い税率が維持されているため、今後もこの課税に関する議論は続くでしょう。

## まとめ

酒税法改正により、2020年に続き、2023年10月にもビールや発泡酒、ワインなどの税率が改正されます。
  
  
とくにビール系飲料は、2023年のビールの減税と新ジャンルの増税を経て、2026年10月には発泡酒とあわせて税率が一本化されます。
  
  
ビールがお手頃になる一方、安さが魅力の新ジャンルの値段が上がるため、消費者への影響は大きいとの予測です。
  
  
メーカー各社はビールや新ジャンルの新たな魅力の提案、チューハイの新商品開発など、すでに準備を始めています。
  
  
消費者としては値段の動向が気になりますが、これを機に家計を見直すなどして、今後もお酒と上手に付き合っていきましょう。

酒税法改正により2023年10月から、ビールの税率が下がって新ジャンルの税率が上がり、ビール系飲料の価格差が接近する予定です。
