ステルスマーケティングとは、消費者に広告だと明示せず商品やサービスを宣伝する行為を指します。
  
  
広告主に関与しているにもかかわらず、一般消費者のふりをして商品やサービスを薦めるのは、消費者を裏切る行為であり悪質です。これまで法律では規制できませんでしたが、2023年10月からはステマ規制が始まります。
  
  
本記事では、ステルスマーケティングの問題点や法規制をまとめました。ステルスマーケティングにならないための対策も解説するので、参考にしてください。

## ステルスマーケティング（ステマ）とは

**ステルスマーケティングとは、宣伝・広告である旨を隠して商品やサービスを宣伝することです。**略して「ステマ」とも呼ばれます。
  
  
「ステルス」には、「内密」「こっそり行う」などの意味があり、企業が関与しているにもかかわらず一般消費者を装って宣伝するなどの行為を指します。代表的な手法は、以下の2つです。

ステルスマーケティングの代表的な手法

●　広告主が有名な芸能人やブロガー、インフルエンサーに依頼し、広告だと分からないように商品やサービスを紹介させる  
●　広告主がブロガーや口コミ代行業者に依頼して、または自ら、高評価な口コミを投稿する

### ステルスマーケティングの法規制

デジタル化が著しく進むなかで、SNS上でのステルスマーケティングは一層深刻な問題となっています。
  
  
EUやアメリカなどの諸外国では、ステルスマーケティングが違法な行為として法規制されている一方で、日本は法規制が遅れていました。
  
  
景品表示法（不当景品類及び不当表示防止法）の「優良誤認」「有利誤認」に該当すれば措置命令がとられましたが、該当しない場合は法律で規制できないのが実情です。

|  |  |
| --- | --- |
| 優良誤認 | ・商品やサービスを実際よりも優れていると宣伝する  ・競合他社と比べて特に優れているわけではないのに、優れているかのように宣伝する |
| 有利誤認 | ・商品やサービスの取引条件が実際よりも有利だと偽って宣伝する  ・競合他社と比べて特に安いわけではないのに、著しく安いかのように宣伝する |
|

  
  
しかし、昨今一層顕在化しているステルスマーケティングから消費者を保護するため、2022年から複数回にわたり「ステルスマーケティングに関する検討会」が実施されています。
  
  
その結果、2023年3月28日、ステルスマーケティングが景品表示法上の「不当表示」に追加されました。2023年10月1日からは、日本でも違法な行為として法規制の対象となります。

### ステマ規制の対象となるもの

2023年10月から、「一般消費者が事業者の表示であることを判別することが困難である表示」がステマ規制で禁止されます。つまり、消費者が広告だと見分けられない表示が法規制の対象です。
  
  
規制の対象は広告主であり、インフルエンサーなどの宣伝者は対象になりません。
  
  
法規制の対象となるかは個別に判断されますが、たとえば以下のようなケースが該当します。

ステマ規制の対象となるケースの例

●　広告主がインフルエンサーなどの第三者に依頼した、SNSや口コミサイト上での紹介  
●　ECサイトに出店する広告主が商品・サービスの購入者などに依頼した、レビューを通じた紹介  
●　広告主がアフィリエイターに依頼したアフィリエイトでの商品・サービスの紹介  
●　広告主がほかの事業者に依頼し、口コミ投稿を通じて競合他社の商品・サービスを自社と比較して低く評価させるもの

ただし、事業者が表示内容の決定に関与したと認められない場合は、規制の対象にはなりません。

## ステマは何が悪い？ ステルスマーケティングの違法性と問題点

ステルスマーケティングの問題点を理解していないと、意図せず違法行為に関与してしまう可能性もあります。ステルスマーケティングがもたらす主なリスクを正しく理解しましょう。

ステルスマーケティングの違法性・問題点

●　消費者が商品やサービスを正しく選択できなくなる  
●　企業や業界、インフルエンサーの信用がなくなる  
●　景品表示法に違反する  
●　意図せず加害者にもなり得る

それぞれについて、以下で解説します。

### 消費者が商品やサービスを正しく選択できなくなる

**ステルスマーケティングは、消費者の商品やサービスに対する正しい選択を妨げるおそれがあります。**
  
  
消費者は通常、広告にはある程度の誇張が含まれていると理解して商品やサービスを選択します。
  
  
しかし、ステルスマーケティングでは広告であることが明示されません。そのため、実際は企業が関与しているにもかかわらず、消費者は「誇張されていない、信頼できる情報」と捉えてしまいます。
  
  
このように、ステルスマーケティングは消費者を裏切る行為であり悪質です。

### 企業や業界、インフルエンサーの信用がなくなる

ステルスマーケティングが発覚すると、その商品やサービス、企業も信頼を失うでしょう。ステルスマーケティングを行った企業だけでなく、競合他社や広告業界全体の信用も損ないかねません。

インフルエンサーがステルスマーケティングをする主なリスク

●　ファンが減る  
●　企業からの依頼がなくなる  
●　ネット炎上する

さらに、企業から依頼を受けたタレントやインフルエンサーにもネガティブなイメージがついてしまいます。その結果、上記のような状態になる可能性もあります。

### 景品表示法に違反する

ステルスマーケティングをすると、景品表示法に違反する可能性があります。
  
  
2023年10月1日からステルスマーケティングは、景品表示法の「不当表示」として法規制の対象です。
  
  
景品表示法に違反すると消費者庁から措置命令が出され、従わない場合は罰則（2年以下の懲役または300万円以下の罰金）が科せられます。場合によっては、企業名が公表される可能性もあります。

### 意図せず加害者にもなり得る

消費者は、ステルスマーケティングの被害者であるのと同時に加害者にもなり得ます。
  
  
インフルエンサーなどの宣伝者は、ステマ規制の対象ではありません。しかし消費者でもあるインフルエンサーが、ステマ規制に意図せず抵触してしまう可能性があります。
  
  
消費者庁のアンケート調査によると、広告主からステルスマーケティングを依頼された経験があるインフルエンサーのうち、その依頼を「全部受けた」または「一部受けた」と回答した人は約45%にのぼります。
[出典：消費者庁「ステルスマーケティングに関する検討会 報告書」](https://www.caa.go.jp/policies/policy/representation/meeting_materials/review_meeting_005/assets/representation_cms216_221228_03.pdf#page=12)
  
  
企業からPRである旨を掲載しないでほしいと言われ、「次の依頼につながるのでは」と安易にステルスマーケティングを行ってしまうケースもあります。
  
  
意図せず加害者にならないために、消費者もステルスマーケティングの違法性や問題点を正しく理解しなくてはいけません。

## ステルスマーケティング（ステマ）を防ぐための対策

ステルスマーケティングには、信頼を失う、法律に違反するなどさまざまなリスクが伴います。ステルスマーケティングの加害者にならないためにも、以下の対策を理解しておきましょう。

ステルスマーケティングを防ぐ対策

●　広告であることを明示する  
●　正しい情報のみを発信する

それぞれについて、以下で解説します。

### 広告であることを明示する

消費者庁が公表したステマ規制の運用基準によると、消費者が「事業者の表示」だと見分けられなければいけません。以下のような文言を記載して広告である旨を明示しましょう。

広告だと明示する文言の例

●　広告  
●　宣伝  
●　プロモーション  
●　PR

また、「○社から商品の提供を受けて紹介している」などの文章を表示する、ハッシュタグを利用するなどの方法で、企業名やブランド名、広告主との関係がわかりやすいように投稿してください。

### 正しい情報のみを発信する

虚偽の口コミを投稿する、実際には利用していない商品やサービスの良い評価を作り上げる行為は絶対に避けましょう。消費者が正確な情報を得られなくなるおそれがあります。
  
  
「いいね」や「フォロワー」を購入して水増しする行為も同様です。宣伝者には、誤認を与えないよう、表現方法に注意した発信が求められます。

## ステルスマーケティング（ステマ）に対する法的な対処法

ステルスマーケティングだと知らずに購入してしまった場合や、問題のある行為を見つけたときは、以下の対処法を検討しましょう。

ステルスマーケティングの法的な対処法

●　返品する  
●　消費者庁に相談する

それぞれについて、以下で解説します。

### 返品する

ステルスマーケティングだと気付かず購入した商品やサービスを返品したい場合、購入したサイトで返品の取り扱いを確認しましょう。
  
  
法律上、販売会社が返品を受ける義務はありませんが、返品のルールを掲載しなければならないと決まっています。
  
  
返品のルールが掲載されていない場合、商品の到着後8日間以内ならクーリング・オフにて送料自己負担で返品が可能です。

### 消費者庁に相談する

消費生活のなかで起きたトラブルを相談したいときは、「[消費者ホットライン](https://www.caa.go.jp/policies/policy/local_cooperation/local_consumer_administration/hotline/)」（188番）に問い合わせましょう。
  
  
全国に約800ヶ所の消費生活センターが設置され、すべての市区町村に消費生活相談窓口が設けられています。
  
  
また、消費者庁の「[景品表示法違反被疑情報提供フォーム](https://www.caa.go.jp/policies/policy/representation/contact/disobey_form/)」を利用すれば、オンライン上で景品表示法に関する情報提供が可能です。電話や郵送からも情報を提供できます。

## まとめ

ステルスマーケティングは、広告だと明示せず宣伝する行為です。消費者の正しい商品選択を妨害する行為であり、広告主・宣伝者ともに信用を大きく損なうおそれがあります。
  
  
これまでステルスマーケティングに対する法律の規制はありませんでしたが、2023年10月からステマ規制が始まります。
  
  
加害者にならないためにもステルスマーケティングの違法性や問題点を正しく理解し、対策しましょう。
