## 仮設工事とは

**仮設工事とは、建設工事を安全かつ円滑に進めるために、一時的な施設や設備を設ける工事のことを指します。**完成後は撤去され、目に見える形では残りませんが、現場の安全・効率・品質を支える重要な工程です。
  
  
仮設工事を行う主な目的は、次のとおりです。

| 主な目的 | 具体的な内容 |
| --- | --- |
| 安全性の確保 | 墜落・転落・侵入を防止するため |
| 作業効率の向上 | 作業環境を整備するため |
| 法令の遵守・近隣への配慮 | 騒音・粉塵を防止するため |

## 仮設工事の種類

仮設工事には「直接仮設工事」と「共通仮設工事」の2種類があります。違いを理解しておくことで、見積書の内容確認や工期の把握がしやすくなるでしょう。

### 直接仮設工事

直接仮設工事は、建物本体の施工に直接関わるもので、主に次のような作業が含まれます。

| 作業 | 内容 |
| --- | --- |
| 水盛・遣り方 | 建物の位置や高さの基準を設定する |
| 養生 | 既存部分を保護し、粉塵や傷を防止する |
| 墨出し | 柱・壁・開口部の位置を線で示す |

いずれも施工精度や品質を左右する工程です。位置のズレや仕上げ面の損傷、粉塵被害を防ぐためにも、熟練した技術者による正確な施工が不可欠です。

### 共通仮設工事

共通仮設工事は、現場全体の安全性と作業環境を整えるためのものです。建物そのものには直接関係しませんが、作業の効率化と事故防止に直結します。
  
  
主な作業内容は、次のようなものを設置することです。

* 仮設トイレ
* 仮設電気・仮設水道
* 現場事務所
* 足場
* 仮設ネット・仮囲い
* 養生

新築工事ではこれらの多くが必要になります。一方、リフォームでは養生を中心に行うなど、内容は工事の種類によって変化します。

### 水盛・遣り方

水盛・遣り方（みずもり・やりかた）は、建物の位置と高さを正確に地面上に示す作業です。この作業で設ける基準線がずれると、基礎や構造体の精度に影響するため、最初の段階で入念に行う必要があります。
  
  
現場では、木杭と水糸で水平を出す方法が広く用いられています。この方法のメリットは、コストがかからないことです。一方で、天候の影響を受けやすいため、雨天時は防水シートなどで保護する必要があります。
  
  
レーザーレベルによる測定を併用すると、より高精度な位置出しが可能になります。

### 仮設トイレ

仮設トイレは、作業員の衛生と安全を守り、快適に作業できる環境を整えるための必須設備です。労働安全衛生法でも設置が義務付けられており、法令遵守の観点からも欠かせません。
  
  
上下水道が使用できる現場では水洗式、使用できない現場では汲み取り式を採用します。
  
  
悪臭の発生や配管の凍結を防ぐ管理も重要です。仮設トイレをレンタルする際は、設置業者や清掃業者と定期清掃契約を結んで衛生環境を保ちましょう。

### 仮設電気

仮設電気は、電動工具や照明に電力を供給するインフラです。電力が確保できなければ作業が進まないため、早めに電力会社に申請しておきましょう。
  
  
電力会社への申請から使用開始までに時間がかかることもあるため、早めの準備が必要です。
  
  
使用予定の機器に応じて電力量を設定しなければ、ブレーカーが頻繁に落ちるなどのトラブルを招きます。分電盤の設置場所を決める際は安全性と動線を考慮し、漏電防止ブレーカーも必ず設置しましょう。

### 仮設水道

仮設水道は、コンクリートの養生や左官の練り、清掃など、さまざまな工程で必要になります。水の確保が遅れると作業全体が止まるため、早期の手配が欠かせません。仮設電気と同様、水道局への申請は着工前から早めに行いましょう。
  
  
水道を引き込む際は、本設を前倒しで使用するか、仮設配管を敷設するかを、工期やコストに応じて判断します。
  
  
なお、広い現場では散水栓を複数設け、作業導線を短くすることで効率が上がります。冬季には、凍結防止や漏水対策も必要です。

### 現場事務所

現場事務所は、工程・安全・品質を管理する現場の中心拠点です。図面確認や打ち合わせ、書類整理といった、日常的な管理業務の多くがここで行われるため、エアコンや机のほか、通信環境を整える必要があります。
  
  
形態は長期工事向けのプレハブ型と、短期・狭小地向けのコンテナ型が主流です。車両や人の動線を妨げないよう、出入口付近の見通しがよい場所に設置しましょう。

### 仮設足場・ゴンドラ

仮設足場やゴンドラは、高所作業における安全性と作業効率を確保するための設備です。[労働安全衛生規則](https://www.jaish.gr.jp/anzen/hor/hombun/hor1-2/hor1-2-1-m-0.htm)には、高さ2m以上の作業では足場の設置が義務付けられており、転落防止や落下物対策の基本となります。
  
  
足場には次のような種類があり、現場の条件に応じて使い分けます。

| 足場の種類 | 内容 |
| --- | --- |
| 枠組足場 | 一般的な建築現場で多く使われる標準タイプ |
| 単管足場 | 自由度が高く、狭小地や複雑な形状に対応 |
| システム足場 | 組立が早く安全性が高いが、コストはやや高め |

組立や解体は有資格者が担当します。設置したら完了ということではなく、定期的な点検が必要です。
  
  
狭い場所や高層建物の外壁工事では、足場とゴンドラを併用するほうが安全で効率的な場合もあります。

### 仮設ネット・仮囲い

仮設ネットと仮囲いは、現場の安全と近隣への配慮を両立するために欠かせません。
  
  
仮設ネットは足場の外側に張り、工具や資材の落下、塗料や粉塵の飛散を防止します。通気性の高いメッシュシートを使用すれば、風の影響を受けにくく安全性も向上します。
  
  
一方、仮囲いは現場の周囲をパネルやフェンスで囲う設備です。立入防止や防犯、景観保全の役割を果たします。
  
  
仮設ネット・仮囲いを設置しなければならない高さは、[建築基準法施行令](https://laws.e-gov.go.jp/law/325CO0000000338)で1.8m以上と定められていますが、これは法的な最低ラインであり、現場の条件によっては3m以上の仮囲いを設ける場合もあります。
  
  
仮設ネット・仮囲いの外側に企業名や緊急連絡先を掲示すると、信頼性を高める効果も期待できるでしょう。
  
  
なお、台風などの強風時には、ネットが風を受けて倒壊する恐れがあります。補強や一時撤去をするなど、適切な判断が必要です。

### 養生

養生は、建物や設備を傷や汚れから守るための保護作業です。資材搬入や作業の際に、床や壁を損傷しないように事前に保護しておくことが重要です。
  
  
床を保護するには、ベニヤ板やプラスチック養生板を使用します。ベニヤ板は安価で扱いやすい一方、水に弱いため、短期工事向けです。プラスチック養生板は、耐久性・防水性に優れており、長期工事に適しています。
  
  
また、壁や柱などの立面には、クッションシートや段ボールシートを使用し、衝撃を和らげます。
  
  
さらに、マンションなどのリフォーム現場では、共用スペースや既存設備の養生も必要です。階段や手すりなどを丁寧に保護することで、施工後のトラブルを防ぎます。
  
  
ただし、長期間の養生は湿気やカビの原因となるため、定期的な換気と点検を行うことが大切です。

### 墨出し

墨出しは、設計図面上の寸法や位置を、実際の現場に正確に示す作業です。この線を基準にすべての作業が進むため、わずかでもズレが生じていると建物全体の品質に直結します。
  
  
墨出しの種類は、次のとおりです。

| 墨出しの種類 | 内容 |
| --- | --- |
| 基準墨 | 建物の中心線を示す |
| 陸墨 | 高さの基準を示す |
| 返り墨 | 壁や柱などの仕上げ位置を示す |

近年はレーザー墨出し器の普及によって、水平・垂直の精度が高まり、作業効率も向上しています。

### 廃材処分

廃材処分は、建設現場で発生する廃棄物を法令にしたがって適正に処理する工程です。
  
  
現場で発生する主な廃材には次のようなものがあります。

| 廃材の種類 | 内容 |
| --- | --- |
| 木くず | 型枠材・合板など |
| 金属くず | 鉄筋・配管・金物など |
| コンクリートがら | 解体・打設時の残材など |
| 石膏ボード | 内装工事での端材など |
| 廃プラスチック類 | 梱包材・養生シートなど |

これらはすべて産業廃棄物に分類され、[廃棄物処理法](https://laws.e-gov.go.jp/law/345AC0000000137)にもとづいた分別・運搬・処理が義務付けられています。
  
  
不法投棄や不適切な処理は、罰則や企業の信用失墜につながります。処理を委託する際は、マニフェスト（管理票）を用いて処理経路を記録・追跡し、適正処理を証明しましょう。
  
  
分別を徹底すればリサイクル資源の再利用が進み、処分コストの削減にもつながります。

### 整理整頓・清掃

整理整頓と清掃は、現場の安全性と作業効率を高める基本業務です。建設業では、次の「5S」の考え方が重視されています。

| 5S | 内容 |
| --- | --- |
| 整理 | 不要なものを取り除く |
| 整頓 | 必要なものを使いやすく配置する |
| 清掃 | 常にきれいな状態を保つ |
| 清潔 | 清掃を習慣化し、衛生的な環境を維持する/td> |
| しつけ | 決めたルールを守り続ける |

資材や工具を整理することで、資材の破損や事故を防げます。必要な道具を探す時間も短縮できるため、作業効率を向上させることも可能です。
  
  
朝礼後や終業前に10分程度の清掃時間を設けて、全員で整理整頓を習慣化することが効果的です。

### 資材運搬

資材運搬は、必要な資材を安全かつ効率的に現場へ届けるための物流管理です。搬入が遅れると工程全体に影響するため、作業スケジュールと連動した搬入計画が不可欠です。
  
  
重量物の運搬や高所への搬入にはクレーン車、敷地内での短距離運搬にはフォークリフトや台車を使用します。都市部の狭い現場では、大型車が入れない場合もあるため、荷揚げ機（揚重機）を設置して搬入するケースもあります。
  
  
搬入ルートは他作業と動線が交差しないよう設計し、資材の仮置きスペースを確保することがポイントです。人力運搬は負担とリスクが大きいため、なるべく機械化して安全性と生産性を両立させましょう。

### 警備

警備は、工事現場の安全と防犯を維持するための業務です。
  
  
特に公道に面した現場では、車両が出入りする際の歩行者・車両の安全を確保するために交通誘導員の配置が求められます。また、道路使用許可を取得する際は、警備員の配置が条件となる場合もあります。
  
  
安全性の確保に加え、資材・工具の盗難防止も警備の重要な役割です。高価な機材や銅線は狙われやすいため、防犯カメラやセンサーライトの設置、夜間警備員の配置でリスクを最小化します。
  
  
工事開始前に警備計画を作成し、警察署への提出と近隣への説明を行うことで、信頼性の高い現場運営が可能になるでしょう。

## 仮設工事の流れ

仮設工事は、建設現場を安全に整え、本体工事を円滑に進めるための準備工程です。
  
  
主な流れは、次のとおりです。

| 工程 | | 主な作業内容 |
| --- | --- | --- |
| STEP1 | 現地調査 | 敷地や周辺環境を確認する |
| STEP2 | 仮設計画の立案 | 必要な設備・動線を設計する |
| STEP3 | 整地 | 地面を平らに整える |
| STEP4 | 仮設インフラ設置 | トイレ・電気・水道を整備する |
| STEP5 | 地縄張り・水盛遣り方 | 建物の位置と高さを確定する |
| STEP6 | 足場・仮囲い設置 | 落下・侵入防止ネットを設置する |
| STEP7 | 現場事務所・警備配置 | 管理・防犯体制を整備する |
| STEP8 | 仮設撤去・清掃 | 足場や仮設物を撤去し清掃する |

これらの工程を計画的に進めることで、安全性・効率・品質が大きく向上します。スケジュールには余裕をもたせ、各工程での点検・確認体制を確実に整えておきましょう。

## 仮設工事にかかる期間の目安

仮設工事は、工事の種類によって設置期間が大きく異なります。工事の種類ごとの期間の目安は、次のとおりです。

| 工事の種類 | 仮設工事の期間目安 |
| --- | --- |
| 一般住宅の新築 | 約1〜2週間 |
| リフォーム工事 | 数日〜1週間 |
| マンション修繕 | 数週間〜1ヶ月 |
| 商業施設・高層建築 | 1ヶ月以上 |

なお、つり足場や張出し足場のほか、高さ10m以上の足場などを設置する際は、労働基準監督署への届け出が必要です。
  
  
このような届け出が必要なケースでは、書類準備や確認に時間を要することもあります。全体のスケジュールを立てる際は、通常より余裕をもって期間を見積もりましょう。

## 仮設工事にかかる費用の目安

仮設工事は、安全確保と作業効率を支える重要な初期工程であり、費用も全体計画に大きく影響します。

### 仮設工事の一般的な費用相場

仮設工事の費用は、建物の規模・構造・工期によって変動します。

| 建物の規模 | 費用の目安 | 工事費に占める割合 |
| --- | --- | --- |
| 戸建住宅・小規模リフォーム | 約15〜30万円 | 約3～5％ |
| 大型商業施設・マンション | 約4,000万〜15,000万円以上 | 約20〜25％ |

都市部では、狭小地対応や防音・防塵設備などに追加の費用が必要な場合が多く、基準価格の1.2〜1.5倍に上昇する傾向があります。
  
  

[出典：国土交通省「令和３年度マンション大規模修繕工事に関する実態調査」](https://www.mlit.go.jp/jutakukentiku/house/content/001619430.pdf)
  
[出典：国土交通省「より適正な価格等の設定」](https://www.nilim.go.jp/lab/peg/img/file539.pdf)

### 仮設工事費の内訳

仮設工事費の主な内訳は、次のとおりです。

| 区分 | 主な内容 | 費用目安 |
| --- | --- | --- |
| 初期費用 | 足場・仮囲い・電気・水道の引き込みなど | 全体の30〜40％ |
| ランニングコスト | トイレ・事務所・警備などの月額費 | 全体の40〜50％ |
| 撤去費用 | 足場解体・仮設撤去 | 全体の10〜15％ |
| 付帯費用 | 廃材処分・清掃・運搬・近隣対策 | 全体の10〜20％ |

これらの内訳を理解しておくことで、見積書の妥当性を判断しやすくなるでしょう。
  
  

[出典：国土交通省「公共建築工事共通費積算基準」](https://www.mlit.go.jp/gobuild/content/001733141.pdf)

### 仮設工事の費用を抑えるコツ

仮設工事費は、事前の準備と現場管理の工夫で抑えられます。安全を犠牲にせず、無駄を減らす意識が大切です。
  
  
特に、次の4つのポイントを意識しましょう。

* 早めに計画を立てる
* 複数社に見積もりを依頼する
* 仮設トイレや仮設事務所の台数・サイズを最適化する
* 工事完了後は足場や仮設設備を早めに撤去するなどして、工期を短縮する

これらはいずれも、現場担当者や仮設業者の工夫で実現できる節約策です。
  
  
ただし、足場や仮囲いなど、安全に関わる設備は削減してはいけません。これらは作業員の命を守り、事故や停電・転落などの重大トラブルを防ぐために必須の設備です。短期的なコスト削減よりも、安全性と信頼性を優先しましょう。

## 仮設工事を行うときの注意点

仮設工事を行う際の主な注意点は、次のとおりです。

仮設工事を行うときの注意点

* [傾斜地や高層建物では足場費用が割高になる](#content7-1)
* [見積書の工事内容を必ず確認する](#content7-2)
* [美装工事や撤去費用は別途計上される](#content7-3)
* [廃棄物の保管・処理ルールを守る](#content7-4)

誤解や追加費用の発生を防ぐために、見積もり前の現地調査と契約内容の確認を徹底しましょう。

### 傾斜地や高層建物では足場費用が割高になる

傾斜地や高層建物、狭小地などの特殊な現場では、標準的な足場を使用できないケースがあります。
  
  
傾斜地では、地盤補強や高さ調整が必要になりますし、高層建物では、風圧対策や落下防止ネットの強化が必要です。また狭小地では、資材搬入や隣地との協議に追加コストがかかる可能性もあります。
  
  
このような理由から、通常の現場に比べて足場費用が割高になる傾向があります。あらかじめ追加費用が発生する可能性を把握しておきましょう。

### 見積書の工事内容を必ず確認する

仮設工事の見積書は、「一式」とまとめて記載されることがあります。内訳が不明確なまま契約すると追加請求の原因になります。
  
  
次のような項目は、標準見積もりに含まれていないことがあるため、注意が必要です。

* 仮設電気・仮設水道の申請費
* 道路使用許可申請費
* 廃材処分・清掃費
* 近隣挨拶や警備の延長費

契約前に、作業範囲や単価の根拠を必ず確認しましょう。項目別・数量別に明細を提示してもらえるよう依頼することで、トラブルを未然に防げます。

### 美装工事や撤去費用は別途計上される

仮設工事の見積もりには、引渡し前の清掃や原状回復といった美装工事が含まれないことが一般的です。
  
  
工事中の簡易清掃とは異なり、次のような最終清掃や撤去作業には、別途費用がかかることがあります。

* ハウスクリーニング（床・窓・壁など）
* 外構や駐車場の高圧洗浄
* 配管・杭・基礎の撤去

契約時に、どこまでが仮設工事の範囲か確認しておきましょう。

### 廃棄物の保管・処理ルールを守る

建設現場で発生する廃材は産業廃棄物に分類されており、[廃棄物処理法](https://laws.e-gov.go.jp/law/345AC0000000137)にもとづく適正管理が義務付けられています。
  
  
不法投棄や不適正処理は、5年以下の懲役または1,000万円以下の罰金（法人は3億円以下）という厳しい罰則の対象です。
  
  
現場では次のルールを守りましょう。

* 廃棄物を種類ごとに分別して保管する
* 保管場所に表示板（種類・管理者名・数量など）を掲示する
* 飛散・流出・悪臭を防止する措置を取る

さらに、マニフェスト制度によって、処理経路を追跡・記録し、5年間保管する義務もあります。処理を委託する際は、許可業者と書面契約を交わし、適正処理が行われているか確認しましょう。
  
  

[出典：環境省「産業廃棄物のマニフェスト制度の概要」](https://www.env.go.jp/content/900494971.pdf)

## よくある質問

### 仮設工事に含まれるものは何ですか？

仮設工事には、本体工事を安全かつ効率的に進めるために必要な、一時的な設備を設置する作業が含まれます。代表的な項目は、次のとおりです。

* 水盛・遣り方
* 仮設トイレ
* 仮設電気
* 仮設水道
* 現場事務所
* 仮設足場・ゴンドラ
* 仮設ネット・仮囲い
* 養生
* 墨出し
* 廃材処分
* 整理整頓・清掃
* 資材運搬
* 警備

いずれも完成後には撤去される一時的な設備ですが、作業の安全性と品質を支えるために欠かせません。
  
  
詳しくは記事内「[仮設工事の主な項目](#content3)」をご覧ください。

### 仮設工事にはどのような種類がありますか？

仮設工事は、建物の施工に直接関わる「直接仮設工事」と、現場全体の環境を整える「共通仮設工事」に大きく分けられます。
  
  
前者は墨出しや養生など品質を左右する工程、後者は足場や仮囲い、事務所設置など安全や効率を支える工程です。
  
  
詳しくは記事内「[仮設工事の種類](#content2)」をご覧ください。

### 仮設工事の目的は何ですか？

仮設工事の主な目的は、次のとおりです。

* 作業員の安全確保
* 工事の効率化
* 品質維持
* 近隣への配慮
* 法令遵守

足場や仮囲いは墜落や転落事故を防ぎ、仮設電気や水道は作業環境を整えることで、効率的な施工を支えます。また、防音・防塵シートや警備員の配置によって、近隣の安全や快適さにも配慮します。
  
  
これらの仮設工事があってこそ、本体工事を安全かつ円滑に進められるのです。
