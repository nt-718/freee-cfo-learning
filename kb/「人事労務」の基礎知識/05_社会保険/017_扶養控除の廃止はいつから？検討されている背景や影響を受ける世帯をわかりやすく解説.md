## 扶養控除とは？

**扶養控除は所得控除のひとつであり、納税者に所得税法上の控除対象扶養親族がいる場合に適用されます。**扶養親族の年齢や同居の有無などに応じて、課税対象となる所得から一定の控除額を差し引きます。

### 扶養控除で控除される金額

扶養控除で控除される金額は、以下の通りです。

|  |  |  |
| --- | --- | --- |
| 扶養親族の区分 | | 控除額 |
| 一般の控除対象扶養親族 | | 380,000円 |
| 特定扶養親族 | | 630,000円 |
| 老人扶養親族 | 同居老親等以外の者 | 480,000円 |
| 同居老親等 | 580,000円 |

[出典：国税庁「No.1180 扶養控除」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1180.htm)
  
  
特定扶養親族とは、その年の12月31日時点で19歳以上23歳未満の控除対象扶養親族をいいます。老人扶養親族は、その年の12月31日時点で70歳以上の人が該当します。
  
  
老人扶養親族の「同居老親等」とは、納税者または配偶者の直系尊属（父母、祖父母など）で、納税者や配偶者と同居を常としている人です。
  
  
病気の治療により入院している場合、その期間が1年以上であっても同居として扱います。ただし老人ホームなどへ入所している場合は、同居として認められません。

### 扶養親族の範囲

控除の対象となる扶養親族は、次の5つの要件すべてを満たさなければなりません。

控除対象となる扶養親族の要件

●　配偶者以外の親族、または都道府県知事から養育を委託された児童や、市町村長から養護を委託された老人である  
●　納税者と生計を一にしている  
●　年間の合計所得金額が48万円以下（令和元年分以前は38万円以下）である  
●　青色申告者の事業専従者としてその年を通じて一度も給与の支払を受けていないこと、または白色申告者の事業専従者でない  
●　その年の12月31日現在の年齢が16歳以上である

[出典：国税庁「No.1180 扶養控除」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1180.htm)
  
  
扶養親族の所得が給与のみの場合、給与収入は「103万円以下」が基準です。

## 扶養控除は廃止が検討されている

**少子化対策のひとつとして「児童手当の拡充」が進められている一方、扶養控除の廃止が検討されていました。**
  
  
しかし、2024年度税制改正大綱では、一般の控除対象扶養親族（16歳〜18歳）を対象とした扶養控除に関して、控除を廃止するのではなく引き下げる案が示されました。
  
  
具体的には、所得税の控除額を38万円から25万円に、住民税の控除額を33万円から12万円に引き下げる案が議論されています。

  

|  |  |
| --- | --- |
| 所得税の控除額 | 38万円から25万円（2026年分から適用） |
| 住民税の控除額 | 33万円から12万円（2027年度分から適用） |

  

0歳から15歳までの子どもには児童手当、16歳から22歳の子どもには所得税・住民税の扶養控除による税制優遇を設けているのが現行制度です。
  
  
**扶養控除がなくなると、納める税金（所得税・住民税）が増えます。**そのため、児童手当の受取額が増えたとしても、年収などによっては、実質の手当額は少なくなる可能性があります。
  
  
扶養控除の廃止あるいは引き下げに関しては、2024年に結論を得ることとされているため、子育て世帯の人は、扶養控除や児童手当に関する議論の動向に注意しましょう。

## 児童手当の拡充案

政府は、2030年代に入るまでの6〜7年が、加速する少子化を止めるラストチャンスだと考えています。
  
  
具体的政策である「加速化プラン」のひとつとして検討されているのが、「児童手当の拡充」です。2024年度中の実施を目指して計画が進んでおり、次の3つの案が挙げられています。

児童手当の拡充で挙げられる案

●　所得制限の撤廃  
●　支給期間の延長  
●　子どもの人数が多い世帯への支援（第3子以降は3万円）

「次代を担うすべての子どもの育ちを支える基礎的な経済支援」という考えに基づいて検討されているのが、親の収入による所得制限の撤廃と支給期間の延長です。
  
  
現行制度では「中学卒業まで」が支給対象ですが、「高校卒業まで」に拡大すれば、3歳から高校生までの子どもに対して月1万円が支給されます。
  
  
また子どもが3人以上の世帯が減少している点などを考慮し、第3子以降の支給額を増額することで、多子世帯の経済的な負担をサポートする狙いです。

## 扶養控除廃止で影響を受けるのは？

児童手当が拡充され、扶養控除が廃止された場合、年収が高くなるにつれて扶養控除廃止の影響が大きくなります。
  
  
高校生の子ども1人を扶養している方の場合、年収1,000万円を超えたあたりから受け取れる児童手当（年間12万円）に対して、税金の負担額が上回ります。

|  |  |  |  |
| --- | --- | --- | --- |
| 年収  （給与収入のみ） | 扶養控除廃止による  所得税・住民税の増加分 | 児童手当拡充分 | 差引金額 |
| 500万円 | 7万6,000円 | 12万円 | -4万4000円 |
| 800万円 | 11万4,000円 | 12万円 | -6000円 |
| 1,000万円 | 12万5,400円 | 12万円 | +5,400円 |
| 1,200万円 | 16万3,400円 | 12万円 | ＋4万3,400円 |

  
※給与所得控除を考慮し、所得控除としては扶養控除・基礎控除のみを考慮して算出
  
  
高校卒業までの児童手当総額と税金負担額を比較した場合、年収が変わらないままと仮定すると、年収1,000万円以上の世帯での負担増加分が大きくなります。
