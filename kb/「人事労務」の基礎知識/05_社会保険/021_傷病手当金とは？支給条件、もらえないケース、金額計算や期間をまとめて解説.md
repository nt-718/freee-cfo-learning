## 傷病手当金とは

**傷病手当金は、被保険者が療養中に事業主から十分な給金が支払われない場合に、被保険者とその家族の生活を保障するために設けられた制度です。**
  
  
被保険者が業務外のケガや病気で仕事に就けない場合に、支給条件を満たすことで傷病手当金が支給されます。金額・振込時期・給付期間などは以下のとおりです。

  

| 項目 | 概要 |
| --- | --- |
| 支給対象 | ・業務外のケガや病気であること  ・勤務ができない状態であること  ・連続する3日間の休業を含めて4日間以上仕事に就けなかった状態であること  ・仕事を休んだ期間の給与支払いがないこと |
| 支給金額 | 支給開始日以前の継続した12ヶ月間の平均給与（平均標準報酬月額） ÷ 30日 × 3分の2 ※被保険者期間が12ヶ月未満の場合は異なる計算式が適用される |
| 振込時期 | 申請から10営業日（2週間）程度 |
| 支給期間 | 支給開始日から通算して1年6ヶ月 |
| 必要な手続き | ・申請書および証明書の用意 ・全国健康保険協会または健康保険組合へ申請 |

[出典：厚生労働省「傷病手当金について」](https://www.mhlw.go.jp/content/12401000/000619554.pdf)

## 傷病手当金の支給条件

傷病手当金の支給には、以下の条件をすべて満たす必要があります。それぞれの注意点を確認しましょう。

傷病手当金の支給条件

1. 業務外のケガや病気であること
2. 勤務ができない状態であること
3. 連続する3日間の休業を含めて4日以上仕事を休まなければならないこと
4. 仕事を休んだ期間の分の給与支払いがないこと

### 業務外のケガや病気であること

**傷病手当金は業務に関わらない病気やケガが対象です。**そのため、業務内で発生した事故などでケガをした場合や、仕事との因果関係が認められる病気は傷病手当金ではなく、労災保険の給付対象です。
  
  
また業務に関わらない療養であっても、美容整形など病気とみなされないものについては給付対象ではありません。

### 勤務ができない状態であること

**病気やケガでの療養により、勤務ができない状態になった被保険者に傷病手当金が支給されます。**
  
  
「勤務ができない状態」とは、今まで担当していた業務ができない状態のことです。勤務ができない状態であるかは、医師の意見や業務内容をもとに会社が加入している健康保険が判断します。
  
  
たとえば、部署を異動したり業務内容を変更したりすることで、勤務が継続できている場合でも、リハビリ出勤で給与の支払いがされていない場合は「勤務ができない状態」して判断され、傷病手当金が支払われる可能性があります。

### 連続する3日間の休業を含めて4日以上仕事を休まなければならないこと

**傷病手当金は、病気やケガが理由で3日間以上連続して仕事を休んだときに4日目の休みから支給されます。**
  
  
この4日間のうち最初の3日間は待期期間とされるため、給付金の算出対象からは除外されます。
  
  
**なお、最初の3日間（待期）は有給休暇や休日、祝日に該当する日であっても対象としてカウントされ、4日目からは連続した休みである必要はありません。**

### 仕事を休んだ期間の分の給与支払いがないこと

病気やケガで仕事を休んでいる期間に給与の支払いがないことも、傷病手当金が支給される条件です。
  
  
しかし、給与が支払われている場合でも傷病手当金よりも給与の支払額が少ない場合は、差額が支払われます。

## 傷病手当金の支給額

傷病手当金の1日あたりの金額は、以下の計算式で求められます。

支給開始日以前の継続した12ヶ月間の平均給与（平均標準報酬月額） ÷ 30（日） × 3分の2

支給開始日が4月1日かつ、3月から遡った12ヶ月間の平均給与（平均標準報酬月額）が30万円の人が1日あたりに受け取る傷病手当金は以下のとおりです。

支給額の計算方法

1日あたりの支給額  
（26（万円） × 2（ヶ月） + 30（万円） × 10（ヶ月）） ÷ 12（ヶ月） ÷ 30（日） × 2/3

また、健康保険の加入期間が支給開始日以前までで12ヶ月に満たない人は、以下のうち低い額となる方を採用し、上記の計算を行います。

加入期間が12ヶ月に満たない場合の支給額の計算方法

1. 支給開始月以前の直近の継続した各月の標準報酬月額の平均
2. 標準報酬月額の平均値  
   30万円（※）：支給開始日が平成31年4月1日以降の方  
   ※当該年度の前年度9月30日における全被保険者の同月の標準報酬月額を平均した額

  

[出典：全国健康保険協会「病気やケガで会社を休んだとき（傷病手当金）」](https://www.kyoukaikenpo.or.jp/g3/sb3040/r139/)

## 傷病手当金がもらえないケース

**傷病手当金は以下の条件に該当すると、支給金額が調整されます。**金額が調整される条件と概要は以下のとおりです。

### 障害厚生年金・障害手当金を受けられる場合

仕事を休んでいる原因となっているケガや病気で厚生年金保険の障害厚生年金または障害手当金を受給している場合は、原則として傷病手当金は支給されません。
  
  
ただし、障害厚生年金を受給している場合で、日額（障害厚生年金を360分の1で割った額）が傷病手当金の日額より少ないときは、その差額が支給されます。
  
  
なお、障害厚生年金に加えて障害基礎年金も受給している場合は、その合算額で日額を計算します。**一方、障害手当金を受給している場合は、傷病手当金の支給総額が障害手当金の額に達するまでの期間、傷病手当金は支給されません。**支給総額が障害手当金の額を超えた後は、通常通り傷病手当金が支給されます。

### 老齢退職年金を受けられる場合

**資格喪失後に傷病手当金の継続給付を受けている人が同時に老齢退職年金を受給している場合は、原則として傷病手当金は支給されません。**
  
  
ただし、老齢退職年金の日額（老齢退職年金を360分の1で割った額）が傷病手当金の日額を下回る場合は、その差額が傷病手当金として支給されます。

### 労災保険から休業補償給付を受けられる場合

**過去に労災保険から休業補償給付を受けた病気やケガと同一の理由で再び働けなくなった場合は、傷病手当金は支給されません。**また、業務外の理由で働けなくなった場合であっても、別の原因で労災保険の休業補償給付を受けている期間中は、傷病手当金の支給は行われません。
  
  
ただし、休業補償給付の日額が傷病手当金の日額を下回る場合に限り、その差額が傷病手当金として支給されます。

### 雇用保険から失業給付を受けられる場合

**雇用保険の失業給付は、働く意思や能力があるにもかかわらず仕事が見つからない場合に支給される給付金です。**一方、傷病手当金は働くことができないことを要件とする給付金であるため、要件が相反します。そのため、**傷病手当金と失業給付金の併給はできません。**

### 出産手当金が同時に受けられる場合

出産手当金と傷病手当金の受給資格が重複する場合は、原則として出産手当金が優先して支給されます。ただし、傷病手当金の額が出産手当金の額を上回る場合には、その差額が支給されます。

## 傷病手当金が支給される（振り込まれる）時期

傷病手当金が支給される（振り込まれる）時期は、申請書類を健康保険協会もしくは健康保険組合が受け取ってから10営業日とされています。
  
  
ただし、**申請した書類に不備があると、再申請が必要な分だけ傷病手当金の振り込みも遅れます。**

## 傷病手当金が支給される期間

傷病手当金が支給される期間は、以前は、支給開始日から最長で1年6ヶ月でしたが、**法改正により支給開始日が2020年7月2日以降の傷病手当金から「通算して1年6ヶ月」に変更されました。**

  
[出典：厚生労働省「健康保険の傷病手当金の支給期間が通算化されます」](https://www.mhlw.go.jp/content/12400000/000857062.pdf)
  

そのため、途中で給与が発生するなどの理由で傷病手当金が支給されない期間があれば、その期間は1年6ヶ月のカウントに含まれず、支給日が通算して1年6ヶ月になるまで継続して支給されます。

## 傷病手当金をもらうために必要な手続き

**傷病手当金をもらうためには、支給申請書を記入し加入している健康保険期間へ申請する必要があります。**
  
  
支給申請書は本人が記載する部分以外にも、医師等が記入する欄と事業主が記入する欄があります。
  
  
必要な書類は、加入している健康保険機関により詳細が異なります。ここからは、全国健康保険協会（協会けんぽ）のWebサイトに記載されている内容にもとづいて、手続きに必要な書類や注意点を解説します。

### 必要書類

**まずはじめに傷病手当金の支給申請書を記入します。**支給申請書は被保険者記入欄用・事業主記入欄用・療養担当者記入用の3種類があります。
  
  
また、支給申請書に添付する書類は、傷病手当金を受け取る人や所属している事業者の状況により異なります。詳しくは以下の表のとおりです。

  

| 条件 | 必要な添付書類 |
| --- | --- |
| 支給開始日以前の12ヶ月以内に事業所を変更した場合や定年再雇用等で被保険者証の番号が変更された場合 | 変更前の事業所の名称・所在地・事業所に使用されていた期間がわかる書類 |
| 障害厚生年金の給付を受けているがマイナンバーでの情報照会を希望しない場合 | ・障害厚生年金給付の年金証書もしくは、これに準ずる書類のコピー  ・障害厚生年金の直近の年金額を証明する書類（年金額改定通知書等）のコピー |
| 障害手当金の給付を受けているがマイナンバーでの情報照会を希望しない場合 | ・障害手当金の支給を証明する書類のコピー |
| 老齢退職年金の給付を受けているがマイナンバーでの情報照会を希望しない場合 | ・老齢退職年金給付の年金証書またはこれに準ずる書類のコピー  ・老齢退職年金の直近の額を証明する書類（年金額改定通知書等）のコピー |
| 労災保険から休業補償が支給されている場合 | ・休業補償給付支給決定通知書のコピー |
| 傷病の原因が第三者によるものの場合 ※交通事故やけんかなどが該当 | ・第三者行為による傷病届 |
| 被保険者が亡くなり、相続人が請求する場合 | ・被保険者との続柄がわかる「戸籍謄本」等 |
| 被保険者のマイナンバーを記載した場合 | 以下のいずれかの書類を添付台紙に添付する。   ①マイナンバーカード（個人番号カード）の表面・裏面両方のコピー  ②マイナンバーカードがない場合は、マイナンバーの記載がある住民票もしくは住民票記載事項証明書と、身元確認書類の2つを添付 |

[出典：全国健康保険協会「健康保険傷病手当金支給申請書」](https://www.kyoukaikenpo.or.jp/g2/cat230/r124/)
  

申請時に提出した添付書類は返却されないため、原本が手元からなくなると困る書類はコピーしたものを提出しましょう。

### 勤務先が行う手続き

**支給申請書の事業主記入用紙では、被保険者者が申請期間中に出勤した日や、出勤していない日に対して支払った賃金を記入します。**事業主記入用紙は、必ず事業主が記入したうえで内容に相違がないことを証明する必要があります。
  
  
また、被保険者の勤務状況や給与支給状況が1枚の事業主用紙に収まらない場合は、申請書の事業主用紙をコピーして複数枚申請することが可能です。

### 本人が行う手続き

**被保険者は、支給申請書の被保険者用紙の記入と、受診している医療機関に療養担当者記入用紙への記入を依頼する必要があります。**
  
  
療養担当者記入用紙には、具体的な傷病名や仕事を休まなければいけない客観的な判断、療養が必要な期間などを記載してもらいましょう。

## 退職後も傷病手当金を受けられるか

退職後であっても以下の条件を満たしていれば引き続き傷病手当金を受けることができます。

* 退職日までに健康保険の被保険者であった期間が1年以上ある
* 退職日の前日までに連続して3日以上出勤せず退職日も出勤していない
* 退職日に傷病手当金を受給していたケガや病気によって引き続き働けない状態にある

なお、退職日に挨拶のために出勤してその日分の給与が支給された場合は条件を満たせないため、退職後に傷病手当金は支給されません。

  

[出典：全国健康保険協会「傷病手当金について」](https://www.kyoukaikenpo.or.jp/g6/cat620/r307/#q6)

## まとめ

**傷病手当金は、業務に関わらない理由による病気やケガで仕事を長期間休まなければならない人に手当が支給される制度です。**傷病手当金の支給が認められるためには、定められている条件を満たしたうえで、支給申請書を加入している健康保険機関へ提出する必要があります。
  
  
健康保険への加入期間や給与額によって支給される金額が異なりますので、自身が加入している健康保険機関の内容をよく理解して利用するようにしましょう。

詳しくは「[傷病手当金が支給される期間](#content6)」をご確認ください。

* 勤務ができない状態であること
* 連続する3日間の休業を含めて4日以上仕事を休まなければならないこと

詳しくは「[傷病手当金の支給条件](#content2)」で解説しています。
