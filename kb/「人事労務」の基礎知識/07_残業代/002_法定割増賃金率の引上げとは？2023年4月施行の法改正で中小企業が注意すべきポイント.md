## 2023年4月から始まる法定割増賃金率引上げの概要

2023年4月1日（土）より、中小企業を対象に、企業が従業員に支払わなくてはならない時間外労働の法定割増賃金率が一部変更され、**月60時間を超える時間外労働についての法定割増賃金率が50％以上になります。**
  
  
法定割増賃金率の引上げは、2008年12月に公布、2010年4月に施行された改正労働基準法のひとつです。
  
  
月60時間を超える時間外労働についての法定割増賃金率が25％以上から50％以上に引上げられましたが、中小企業についてはその影響の大きさや費用負担の大きさが考慮され、適用が猶予されていました。
  
  
今回の適用は、この猶予期間が終了したことによるものです。

  

[出典：厚生労働省「月６０時間を超える時間外労働の 割増賃金率が引き上げられます」](https://www.mhlw.go.jp/content/000930914.pdf)
  
[出典：厚生労働省「働き方改革を推進するための関係法律の整備に関する法律（平成30年法律第71号）の概要」](https://www.mhlw.go.jp/content/12602000/000349688.pdf)

### 法定割増賃金率引上げの変更点

大企業は2010年4月に改正労働基準法が施行された際に法定割増賃金率の引上げが適用されており、今回対象となるのは中小企業のみです。
  
  
中小企業が従業員に支払わなくてはならない法定割増賃金率の割増率は、以下のとおり改正されます。

|  |  |  |  |
| --- | --- | --- | --- |
| 項目 | 対象 | ～2023年3月31日（金） | 2023年4月1日（土）～ |
| **1ヶ月60時間を超える** 法定時間外労働時間 | 大企業 | 50％ | 50％ |
| 中小企業 | 25％ | 50％ |

### 対象となる中小企業の範囲

今回の法定割増賃金率引上げの対象となる中小企業の判断基準は、以下のとおりです。

|  |  |  |  |
| --- | --- | --- | --- |
| 業種 | 資本金 |  | 従業員数 |
| 小売 | 5,000万円以下 | もしくは | 50人以下 |
| サービス | 5,000万円以下 | 100人以下 |
| 卸売り | 1億円以下 | 100人以下 |
| その他 | 3億円以下 | 300人以下 |

[出典：厚生労働省「改正労働基準法のポイント」](https://www.mhlw.go.jp/content/11200000/000972775.pdf)

従業員数は、繁忙期などで一時的に増える有期雇用などは含めず、常時雇用する従業員数で判断します。

## 法定割増賃金率とは

法定割増賃金率とは、法定労働時間を超える労働時間について支払わなくてはならない法定割増賃金の割合のことをいいます。
  
  
従業員への支払いが義務付けられている法定割増賃金に含まれるのは、時間外手当・残業手当・休日手当・深夜手当です。
  
  
法定割増賃金率の割増率は以下のとおりです。

|  |  |  |
| --- | --- | --- |
| 法定割増賃金の種類 | | 割増率 |
| ・時間外手当 ・残業手当 | 法定労働時間(\*)を超える分 | 25％以上 |
| 限度時間（1ヶ月45時間または年間360時間等）を超える分 | 25％以上 |
| 1ヶ月60時間を超える分 | 50％以上 |
| 休日手当（週1回出勤の分） | | 35％以上 |
| 深夜手当（22時～5時に勤務した分） | | 25％以上 |

(\*)1日8時間および週40時間を超える時間

[出典：東京労働局「労働基準法 割増賃金編」](https://jsite.mhlw.go.jp/tokyo-roudoukyoku/content/contents/000501860.pdf)

休日出勤や深夜勤務の労働時間が時間外労働時間であった場合、それぞれの割増率を足した割合を1時間あたりの賃金にかけて計算します。具体的な計算方法については後述します。

## 具体的な法定割増賃金の計算方法

法定割増賃金率は、1時間あたりの賃金が計算の基準となります。月給などの固定賃金の場合には、1時間あたりの賃金を算出します。
  
  
1時間あたりの賃金は以下のように計算します。

1ヶ月あたりの賃金の計算方法

**1時間あたりの賃金 = 月給 ÷ 1ヶ月の平均所定労働時間**
  
  
1ヶ月の平均所定労働時間 =｛（365日 - 年間所定休日数）× 1日の所定労働時間数 } ÷ 12ヶ月

**この場合の月給には基本給のほか、役職手当や資格手当など、労働時間内の勤務に対する手当も含まれます。**ただし、通勤手当や住宅手当など、勤務の内容に対する報酬でない手当や臨時の手当は含めません。
  
  
上記で算出した1時間あたりの賃金をもとに、従業員に支払う法定割増賃金を求めます。
  
法定割増賃金の計算方法は以下のとおりです。

法定割増賃金の計算方法

法定割増賃金 = 時間外労働時間数 × 1時間あたりの賃金 × 割増率

法定割増賃金は、時間外労働・休日労働・深夜労働それぞれに適用されます。たとえば60時間を超える時間外労働と深夜労働が重複している場合は、50％ + 25％ = 75％であり、法定割増賃金率は75％となります。
  
  
60時間を超える時間外労働と深夜労働をした場合の計算例は以下のとおりです。

法定割増賃金の計算例

1時間あたりの賃金が1,500円  
60時間の時間外労働時間のうち4時間が深夜労働  
60時間を超える時間外労働時間が12時間、うち2時間が深夜労働の場合
  
  
・56時間分の時間外労働分（60時間以下）  
 **56時間 ×（1,500円×1.25）=105,000円**
  
  
・4時間分の深夜労働および時間外労働（60時間以下）  
**4時間×（1,500円×1.5）=9,000円**
  
  
・10時間分の時間外労働分（60時間超）  
**10時間×（1,500円×1.5）=22,500円**
  
  
・2時間分の深夜労働および時間外労働（60時間超）  
**2時間×（1,500円×1.75）=5,250円**

[出典：厚生労働省「割増賃金を計算する際の基礎となる賃金は何か」](https://www.mhlw.go.jp/bunya/roudoukijun/faq_kijyunhou_24.html)
  
[出典：厚生労働省「改正労働基準法」](https://www.mhlw.go.jp/topics/2008/12/dl/tp1216-1l-02.pdf)

## 法定割増賃金率引上げに伴う中小企業の課題と注意点

法定割増賃金率引上げに伴う中小企業の課題と法改正への対応時の注意点は以下の通りです。

  

法定割増賃金率引上げに伴う課題と注意点

* 適切な労働時間の管理
* 代替休暇制度導入の検討
* 就業規則の改定
* 業務効率の見直し

### 適切な労働時間の管理

法定割増賃金を正しく計算するためには、適切な労働時間の把握と管理が必須です。
  
  
勤怠管理システムや給与計算ソフトを導入することで適正な労働時間の管理が可能になるだけでなく、労働時間や給与計算にかかる業務の効率化も期待できます。
  
  
特に残業時間が多い会社の場合、法定割増賃金率の引上げは金銭的な負担が大きくなるため、業務効率の見直しは残業時間削減のためにも有効な手段です。
  
  
法定割増賃金を正しく支払わなかった場合、労働基準法37条1項、119条1項により6ヶ月以下の懲役または30万円以下の罰金を課される可能性があります。
  
  
勤怠管理システムの導入やシステムの更新などを通して労働時間の管理を適切に行い、2023年4月以降の法定割増賃金の計算方法を間違えないように注意しましょう。

  

[出典：e-GOV法令検索「労働基準法」](https://elaws.e-gov.go.jp/document?lawid=322AC0000000049)

### 代替休暇制度導入の検討

**代替休暇制度とは、1ヶ月60時間を超える時間外労働に対し、法定割増賃金を支払う代わりに有給休暇を付与できる制度**です。会社側は一定の費用負担軽減ができ、従業員も休暇を取得できます。
  
  
ただし、導入には過半数の組合員または代表者との労使協定の締結が必要です。また、代替休暇の取得は労働者個人の判断に委ねられ、会社から取得を強制することはできません。

  

[出典：厚生労働省「改正労働基準法のポイント」](https://www.mhlw.go.jp/content/11200000/000972775.pdf)

### 就業規則の改訂

法定割増賃金率の引上げに伴い、就業規則や給与規則の改訂を行わなくてはなりません。
  
  
就業規則上で60時間を超える時間外労働の割増賃金率が50％未満と記載されている場合には、2023年4月からの法改正に対応した就業規則の変更を行いましょう。また、上述した代替休暇制度を導入する場合、それらに関する規則の改訂も必要です。

  

[出典：厚生労働省「月60時間を超える時間外労働の 割増賃金率が引き上げられます」](https://www.mhlw.go.jp/content/000930914.pdf)

### 業務効率の見直し

月60時間を超える時間外労働に対して法定割増賃金率が引上げられた背景として、労働時間が週60時間を超える労働者（2008年の厚生労働省が行った労働力調査中に、収入をともなう仕事を1時間以上行った人）が全体の10％にのぼることがあげられます。
  
  
労働者の健康への配慮や労働ではない時間の保全は、国が働き方改革を推進する上での課題のひとつでした。
  
  
今回の法定割増賃金率の引上げに際し業務効率を見直すことで生産性の向上につなげ、結果として時間外労働の削減を進めることが企業に求められています。

  

[出典：総務省「平成20年　労働力調査年報」](https://www.stat.go.jp/data/roudou/report/2008/ft/index.html)
  
[出典：厚生労働省「改正労働基準法のポイント」](https://www.mhlw.go.jp/content/11200000/000972775.pdf)

## 活用できる制度とそのポイント

法定割増賃金率の引上げ対応を行うことで、会社の業務負担や金銭的負担が大きくなります。
  
  
しかし、時間外労働削減への取り組みを行う場合は助成金を受けられる可能性があります。また、勤務時間や業務改善についての相談窓口も充実しているため、必要に応じて積極的に活用しましょう。

### 業務改善助成金

業務改善助成金は中小企業の生産性向上や会社内の最低賃金の引上げの取り組みを支援する制度です。
  
  
人材育成や設備の導入などを通して会社の最低賃金を30円以上引上げた場合に、投資した金額の一部に対して助成が受けられるものです。
  
  
ただし、助成の対象になるのは、会社、もしくは事業所の最低賃金と会社の所在地の最低賃金の差が30円以内の会社もしくは事業所のみです。

  

[出典：厚生労働省「[２]業務改善助成金：中小企業・小規模事業者の生産性向上のための取組を支援」](https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/koyou_roudou/roudoukijun/zigyonushi/shienjigyou/03.html)

### 公的な相談窓口

労働基準監督署には労働時間に関する相談および支援コーナーが設置されています。ここでは、労働時間に関する制度や法令についてや、業務改善や労働時間の改善のために利用できる助成金についての提案が受けられます。
  
  
その他、厚生労働省が主導する働き方改革推進支援センターでは、労務管理や生産性向上、賃金引上げに関する支援が受けられます。社会保険労務士をはじめとした経営の専門家に直接相談をすることも可能です。

  

[出典：厚生労働省「中小企業事業主のみなさまへ」](https://www.mhlw.go.jp/file/04-Houdouhappyou-11202000-Roudoukijunkyoku-Kantokuka/0000199552.pdf)
  
[出典：厚生労働省「働き方改革推進支援センターのご案内」](https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/0000198331.html)

## まとめ

法定割増賃金率の引上げは、2023年4月から中小企業にも適用されます。
  
  
今回の割増賃金率の引上げ対象は月に60時間を超える分のみですが、従業員の雇用環境改善や企業のコスト削減、生産性の向上のためにも、労働時間の管理や業務効率の見直しは重要なポイントです。法改正に向けた社内体制の整備や規則の改訂も忘れずに行いましょう。

## よくある質問

### 法定割増賃金率とは？2023年4月に何が変わるの？

法定割増賃金率とは、法定労働時間を超える時間外労働や休日出勤、深夜労働に対して支払う割増賃金の割合のことです。中小企業には適用が猶予されていましたが、2023年4月より適用が開始されます。詳しくは[こちら](#content1)で解説しています。

### 法定割増賃金率の引上げで企業が対応すべき内容は？

法定割増賃金率の引上げに対応するために、法改正対応に則った社内規定の改正など、適切な労働時間管理が必須です。具体的な対応すべき内容に関しては[こちら](#content4)で一覧にしているのでご覧ください。
