## 産休（産前産後休業）制度とは

**産休（産前産後休業）制度とは、「産前休業」および「産後休業」から構成される休業制度です。**労働基準法第65条が定めており、女性従業員のみが取得可能です。
  
  
産休は母子の保護を目的に制定された制度で、仕事を休むことで出産準備や産後の体調回復に専念できるようになりました。
  
  

[出典：e-Gov法令検索「労働基準法 第65条」](https://elaws.e-gov.go.jp/document?lawid=322AC0000000049)

### 産休の期間

産休の期間は以下のとおりです。

  

|  |  |
| --- | --- |
| 産前休業 | 出産予定日から逆算し、最大6週間前（42日前）から出産当日まで  ただし、多胎妊娠の場合は14週間前（98日前）から出産当日まで |
| 産後休業 | 出産翌日から起算して8週間後（56日後）まで |

  

産前休業と産後休業における最大の違いは、「産後休業の取得は原則として義務である」という点にあります。産後は体調の急変が起きやすい時期でもあり、育児指導やメンタルケアが必要な観点から、産婦本人が就業を希望した場合でも会社は休業させる義務があります。
  
  
ただし、産後6週間経過後に産婦本人が就業を請求し、医師が支障はないと認めた場合に限り、配慮しながらの就業が可能です。
  
  

[出典：e-Gov法令検索「労働基準法 第65条第2項」](https://elaws.e-gov.go.jp/document?lawid=322AC0000000049)
  
**一方、産前休業は本人の意思で請求するものであり、会社に取得させる義務はありません。**
  
  
産前休業は出産予定日が基準となっているため、実際の出産日次第では取得日数が変動する点にも注意が必要です。
  
  
出産日が予定日より早まった場合、産前休業は短くなり、出産日の翌日から産後休業に切り替わります。出産日が遅くなった場合は出産日当日まで産前休業を延長し、翌日から産後休業に移ります。（昭和26年4月2日婦発113号）

### 産前産後休業の対象者

産前産後休業は「労働基準法」に規定されており、パート・アルバイト・派遣社員・契約社員・正社員の雇用形態に関係なく、**すべての女性従業員が取得できます。**
  
  
会社は女性従業員が産前産後休業を取得したことを理由に、解雇やその他の不当な扱いをしてはいけません。また、産休期間中や産休終了後30日の間は解雇することが禁止されています。
  
  

[出典：e-Gov法令検索「雇用の分野における男女の均等な機会及び待遇の確保等に関する法律 第9条第2項」](https://elaws.e-gov.go.jp/document?lawid=347AC0000000113)
  
[出典：「労働基準法 第19条」](https://elaws.e-gov.go.jp/document?lawid=322AC0000000049)

## 育休（育児休業）制度とは

産休と似ている制度に**育休（育児休業）制度**がありますが、両者は異なる法律に基づく別制度です。
  
  
育休（育児休業）とは、子どもの養育と就業の両立を目的に制定された制度で「育児・介護休業法」で定められています。一方、産休は「労働基準法」で定められています。
  
  
育休は、母親だけでなく父親や里親（特別養子縁組したケースのみ適用）が取得することも可能です。
  
  
本人から申し出があったときに職場が断ることはできませんが、取得義務はないため育休を取らずに復職することもできます。請求期間は育休を開始したい日の原則2週間前と定められているため、対象者がいる企業では早めに取得意思を確認しておきましょう。

### 育休の期間

育休の期間は以下のとおりです。

  

|  |  |
| --- | --- |
| 育休の取得開始日 | 母親：産前休業終了の翌日以後  父親：出生日の翌日以後  里親：特別養子縁組の正式委託日以後 |
| 育休取得終了日 | 原則、子が1歳になる前日まで |

  

育休の終了日は「子が1歳になる前日まで」と定められていますが、状況により延長できる場合があります。保育園の空きがなく預け先が確保できないなど特別な理由がある場合には「子が1歳6ヶ月になる日」まで取得を延長できます。
  
  
それでも保育園の空きが見つからなかった場合、さらに「子が2歳になる日」まで取得を延長できるため、その期間中に保活をすることが可能です。
  
  
育児・介護休業法における育休の取得は、子どもが1歳になるまでの期間は原則2回まで、1歳6ヶ月および2歳になるまでの期間は各1回までと定められています。「1週間の育休を5回」といった形式での取得はできません。
  
  

[出典：厚生労働省「育児休業制度について」](https://www8.cao.go.jp/kisei-kaikaku/kisei/meeting/wg/koyou/20191226/191226koyou03.pdf)
  
  
なお、2010年6月30日に施行された「パパ・ママ育休プラス制度」では、父母ともに育休を取得した場合に限り、「子が1歳2ヶ月になる日まで」取得を延長できるようになりました。父母はそれぞれ、最大1年間育児休業を取得することが可能です。
  
  

[出典：厚生労働省「育児休業制度 p28」](https://www.mhlw.go.jp/content/11909000/000355360.pdf)

### 育児休業の対象者

育休の対象者は、**「1歳未満の子どもを育てている養育者（男女従業員）」**に限られます。保育園の空きが見つからないなど特別な事情があれば、1歳6ヶ月または2歳になるまで育休を延長することもできますが、1歳未満の時期から継続して育休を取得している必要があります。
  
  
ただし、日々雇用される形態の従業員は育休取得対象外です。また、下記に該当する場合は労使協定により育休取得対象外となります。

* 入社から1年未満である場合
* 育児休業の申し出の日から1年（延長の場合は申し出から6ヶ月）以内に雇用関係が終了することが明らかな場合
* 1週間の所定労働日数が2日以下である場合

[出典：厚生労働省 「育児・介護休業等に関する労使協定の例」](https://www.mhlw.go.jp/content/11909000/06.pdf)
  
[出典：「育児・介護休業法のあらまし」](https://www.mhlw.go.jp/content/11909000/000355354.pdf)
  
  
なお、パート・アルバイト・派遣社員・契約社員であっても、子どもが1歳になる日まで労働契約が維持・更新される場合は、育休を取得できます。

## 育児休暇と育児休業の違い

広義の「育休」のなかには、「育児休業」のほかに「育児休暇」が含まれる場合があります。
  
  
しかし、以下のように、育児休暇は育児休業と違いがあります。

  

|  | 育児休暇 | 育児休業 |
| --- | --- | --- |
| 法的保証 | 育児を目的として取得する単なる休暇。 | 「育児・介護休業法」によって定められた休業制度。 |
| 育児休業給付制度の有無 | 給付金制度なし。 | 「育児休業給付金」（休業開始時の賃金の67%。休業開始から6ヶ月経過後は50%）の支給あり。 |
| 育休中の社会保険料免除の有無 | 免除なし。 | 社会保険料（健康保険料・厚生年金保険料）の免除あり。 |

  

育児休業ではなく育児休暇を取得する場合、給付金や社会保険料免除などの金銭的なメリットがないため、休暇中の生活費は自身で用意する必要があります。

## 産後パパ育休（出生時育児休業）とは

産後パパ育休（出生時育児休業）とは、男性従業員の育児参加および産後の負担分散を目的に制定された制度です。育児介護休業法が改正され、2022年10月1日に施行されました。
  
  
「育休」と名がついていますが、育児休業とは別の制度です。
  
  
産後パパ育休（出生時育児休業）を取得できる期間は、「子どもの出生後8週間以内に4週間まで」と定められています。最初にまとめて申し出た場合のみ、分割して2回まで取得できます。
  
  
原則として休業の2週間前までに請求する必要がありますが、急に出産日が早まったなど事情がある場合は、この限りではありません。出産日確定前に請求することもできるため、柔軟な制度として活用されています。
  
  

[出典：e-Gov法令検索「育児休業、介護休業等育児又は家族介護を行う労働者の福祉に関する法律 第9条の3第3項」](https://elaws.e-gov.go.jp/document?lawid=403AC0000000076_20230401_503AC0000000063)
  
  
なお、育児休業とは異なり、労使協定を締結している場合に限っては従業員が合意した範囲で休業中も就業できます。
  
  
「休業期間中の所定労働日・所定労働時間の半分」「休業開始・終了予定日を就業日とする場合は当該日の所定労働時間数未満」など条件がありますが、就業と育児を両立したい人には最適です。
  
  

[出典：厚生労働省「育児・介護休業法 改正ポイントのご案内 令和4年4月1日から3段階で施行」](https://www.mhlw.go.jp/content/11900000/000789715.pdf)

### 産後パパ育休（出生時育児休業）の期間

産後パパ育休（出生時育児休業）の期間は以下のとおりです。

  

|  |  |
| --- | --- |
| 産後パパ育休（出生時育児休業）の取得開始日 | 出産予定日から |
| 産後パパ育休（出生時育児休業）の取得終了日 | 出生日または出産予定日のいずれか遅い方から起算して８週間後まで |

  

産後パパ育休（出生時育児休業）は「出産予定日」が基準となっているため、予定日と実際の出産日とずれた場合は、産後パパ育休の取得対象期間が変更します。

  

|  |
| --- |
| **実際の出産日が予定日より早い場合**  出産日から、出産予定日の8週間後まで  ※この場合は、出産日の1週間前には請求する必要がある |
| **実際の出産日が予定日より遅い場合**  出産予定日から、出産日の8週間後まで |

  

[出典：厚生労働省「 産後パパ育休制度（出生時育児休業制度）」](https://www.mhlw.go.jp/content/11909000/000907662.pdf)

### 産後パパ育休（出生時育児休業）の対象者

産後パパ育休（出生時育児休業）の対象者は、「産後休業を取得していない従業員」と定められています。
  
  
制度名のとおり基本的には父親を対象とした制度ですが、養子縁組などをした女性なども取得可能です。なお、事実婚の場合は申出時点で認知している必要があります。
  
  
ただし、日々雇用される形態の従業員は育休取得対象外です。また、下記に該当する場合は労使協定により産後パパ育休（出生時育児休業）の取得対象外となります。

* 入社から1年未満である場合
* 産後パパ育休（出生時育児休業）の申し出の日から8週間以内に雇用関係が終了することが明らかな場合
* 1週間の所定労働日数が2日以下である場合

なお、パート・アルバイト・派遣社員・契約社員であっても、出生日から8週間以降まで労働契約が維持・更新される場合は、産後パパ育休（出生時育児休業）を取得できます。

## 産休・育休中にもらえるお金

産休・育休中は、一定の条件を満たすことで受給できる手当金や給付金があります。

  

| 産前産後期間中にもらえるお金 | | | |
| --- | --- | --- | --- |
| もらえるお金 | 給付時期 | 給付額 | 対象者 |
| 出産育児一時金 | 出産時 | 1児につき50万円  （双子の場合は100万円） | 健康保険の被保険者、または被扶養者 |
| 出産手当金 | 産前・産後休業中 | 直近継続12ヶ月の報酬月額平均÷30×2/3相当額×日数 | 健康保険に加入している母親、正社員、契約社員、パート、アルバイト、派遣社員 |
| 出生時育児休業給付金 | 産後パパ育休取得中 | 原則、休業開始前の賃金の2/3 | 雇用保険被保険者を対象に産後パパ育休取得者 |

  

| 育休期間中にもらえるお金 | | | |
| --- | --- | --- | --- |
| もらえるお金 | 給付時期 | 給付額 | 対象者 |
| 育児休業給付金 | 育児休業中 | 育休開始時の賃金日額×67%×日数  （180日まで）  それ以降は50% | 雇用保険被保険者で育児休業取得者 |

  

それぞれ、下記で詳しく解説します。

### ①出産育児一時金

|  |  |
| --- | --- |
| 支給対象者 | 健康保険や国民健康保険の被保険者など |
| 金額 | 上限50万円 |
| 請求方法 | 病院経由で請求、または加入中の協会けんぽや健康保険組合などに請求 |

  

**出産育児一時金とは、出産に要する費用を補助するための保険給付です。**健康保険の被保険者を対象とした制度であり、50万円を上限として支給されます。
  
  
協会けんぽや健康保険組合などから直接病院に給付金が届く「直接支払制度」を活用すれば、一時的に出産費用を建て替える必要がなくなるため便利です。その他、被保険者の口座に給付金を振り込む方法や、家族など受取代理人の口座宛に振り込む方法もあります。
  
  
なお、出産費用が50万円未満だった場合、差額は被保険者（または受取代理人）の口座に振り込まれます。「直接支払制度」であれば出産翌日以降の入院中に病院経由で手続き可能ですが、それ以外の方法を取る場合は加入中の協会けんぽや健康保険組合などに別途手続きが必要です。
  
  

[出典：全国健康保険協会「子どもが生まれたとき」](https://www.kyoukaikenpo.or.jp/g3/sb3280/r145/)

### ②出産手当金

|  |  |
| --- | --- |
| 支給対象者 | 健康保険組合に加入中の被保険者、かつ産休中に給与が減額された者 |
| 金額 | 直近継続12ヶ月の報酬月額平均÷30×2/3相当額 |
| 請求方法 | 職場を通して加入中の協会けんぽや健康保険組合などに請求 |

  

出産手当金は、産前産後休業に伴う収入減を補うための制度で、妊産婦自身が被保険者になっていることが条件です。しかし、下記に該当する場合は対象外となります。

* 配偶者など親族の扶養になっている場合
* 国民健康保険に加入している場合（職域国保など一部の国民健康保険組合加入者は除く）
* 日々雇用される形態の従業員で、出産月の前4ヶ月間において26日分以上保険料を未納付している場合

会社が毎年提出する社会保険料の算定手続きに基づき、直近継続12ヶ月の報酬月額平均÷30×2/3相当額が支給されます。ただし、あくまでも減給分を補うための制度であり、休業中に出産手当金の支給額より多い給与を受け取っていた場合は支給されません。
  
  
手続きは、職場を通じて加入中の協会けんぽや健康保険組合などに提出します。産前・産後分を分けて請求することも、出産日以降にまとめて提出することも可能です。
  
  

[出典：全国健康保険協会「出産手当金について」](https://www.kyoukaikenpo.or.jp/g6/cat620/r311/#q3)

### ③育児休業給付

|  |  |
| --- | --- |
| 支給対象者 | 下記3つの条件を全て満たす雇用保険被保険者   ・育児休業開始前の2年間に、賃金支払基礎日数が11日以上ある月が12ヶ月以上ある  ・支給単位期間（育児休業開始日から1ヶ月ごとに区分した期間）ごとに、事業所から支払われた賃金が休業開始前の1ヶ月当たり賃金の8割未満であること  ・各支給単位期間に仕事をしたと認められる日が10日以下であること |
| 金額 | 育児休業開始から6ヶ月までは、育児休業開始時の賃金日額×支給日数の67%を支給  それ以降は育児休業開始時の賃金日額×支給日数の50％を支給 |
| 請求方法 | 職場を通して所轄のハローワークに請求 |

  

育児休業給付金は、育児休業におよび産前産後休業に伴う収入減を補うための制度です。雇用保険の加入者を対象としているため、雇用保険に加入してない従業員は対象外となります。
  
  
なお、賃金日額は育休給付金請求時に提出する、過去の給与額に基づいて計算されます。出産手当金や出産育児一時金と異なり、請求先は協会けんぽや健康保険組合などではなくハローワークです。
  
  
育児休業給付受給資格者は、会社経由で請求することにより、雇用保険から育児休業給付金の支給を受けられます。給付期間は子どもの1歳（延長の場合は1歳6ヶ月または2歳）の誕生日の前々日までです。
  
  

[出典：厚生労働省「育児休業給付の内容と支給申請手続」](https://www.mhlw.go.jp/content/11600000/001126859.pdf)

## 産休・育休中は社会保険料が免除される

**産休・育休中は、毎月支払いが必要であった社会保険料が免除されます。**従業員負担分だけでなく会社負担分も免除されるため、忘れずに手続きしましょう。

### 産前産後休業中の手続き

産前産後休業中は、会社の申し出により、健康保険料と厚生年金保険料の支払いが従業員負担分・会社負担分ともに免除されます。免除される期間は、産前産後休業開始日が属する月から、産前産後休業終了日の翌日が属する月の前月までです。

  

|  | 提出書類 | 添付書類 | 提出先 |
| --- | --- | --- | --- |
| 健康保険 | 健康保険・厚生年金保険産前産後休業取得者申出書 | - | 健康保険組合または事業所管轄の年金事務所 |
| 厚生年金保険 | 同上 | - | 事業所管轄の年金事務所 |

### 育児休業中の手続き

育児休業中は、会社の申し出により、健康保険料と厚生年金保険料の支払いが従業員負担分・会社負担分ともに免除されます。免除される期間は、育児休業開始日が属する月から、育児休業終了日の翌日が属する月の前月までです。

  

|  | 提出書類 | 添付書類 | 提出先 |
| --- | --- | --- | --- |
| 健康保険 | 健康保険・厚生年金保険育児休業等取得者申出書 | - | 健康保険組合または事業所管轄の年金事務所 |
| 厚生年金保険 | 同上 | - | 事業所管轄の年金事務所 |

### 育児休業を終了する際の社会保険の手続き

育児休業などを取得した従業員が、当初の育児休業等終了予定日よりも前に育児休業を終了する場合は、会社が「健康保険・厚生年金保険育児休業等取得者終了届」を提出する必要があります（当初申し出たとおりに育児休業などを終了する場合には不要）。
  
  
また、育児休業および育児休業に準ずる休業終了日に3歳未満の子どもを養育している従業員は、一定の条件を満たせば、標準報酬月額を改定することができます。
  
  
その際、従業員の申し出を受けた会社は、「健康保険・厚生年金保険育児休業等終了時報酬月額変更届」を提出しなくてはなりません。
  
  
さらに、標準報酬月額の改定により等級が下がる場合、従業員は会社に申し出て「厚生年金保険養育期間標準報酬月額特例申出書」を提出すれば、子どもが3歳になるまでの間、休業前の高い標準報酬月額で将来の年金額を計算してもらえます。

  

|  | 提出書類 | 添付書類 | 提出先 |
| --- | --- | --- | --- |
| 健康保険 | 健康保険・厚生年金保険育児休業等取得者終了届  健康保険・厚生年金保険育児休業等終了時報酬月額変更届 | - | 健康保険組合または事業所管轄の年金事務所 |
| 厚生年金保険 | 厚生年金保険養育期間標準報酬月額特例申出書 | 戸籍謄（抄）本（被保険者と子の続柄、および子の生年月日の証明）被保険者と子の住民票の写し（マイナンバー記載なしの原本。被保険者と子の同居を証明） | 事業所管轄の年金事務所 |

## まとめ

産休や育休の制度は、妊産婦やその家族が安心して出産・育児ができるように制定された制度です。取得できる期間を出産予定日（または出生日）から計算し、無理なく仕事と両立させる手続きを整える必要があります。
  
  
会社側は、従業員が出産前後に慌てたり、育児中の金銭的な負担を感じたりしなくて済むよう、スムーズな手続きをするのが理想です。給付金・手当金や社会保険料免除など金銭的な手続きも会社経由での請求となることが多いため、事前に要件をチェックしておきましょう。

## よくある質問

### 産休はいつから取得できる？

産休は、出産予定日6週間前（双子以上妊娠の場合は14週前）から、出産8週間後まで取得できます。
  
  
産休の取得期間について詳しく知りたい人は、「[産休の期間](#content1-1)」をご覧ください。

### 産休中にもらえるお金とは？

産休中には、出産育児一時金と出産手当金をもらえます。
  
  
育休中にもらえる育児休業給付金も併せて、「[産休・育休中にもらえるお金](#content6)」をご覧ください。
