## 介護に必要なフェイスシートとは

**介護現場におけるフェイスシートとは、利用者の基本情報をまとめたプロフィール書類です。**具体的には以下のような情報が記載されます。

* 利用者の氏名
* 生年月日
* 家族構成
* 生活歴
* 病歴

フェイスシートは、ケアマネージャー・看護師・介護職員など、利用者のケアに関わるチーム全員が最初に目を通す書類です。**主に、介護サービスの利用開始や施設へ入所するタイミングで作成されます。**
  
  
フェイスシートで利用者の全体像を把握すれば、チーム内の情報共有がスムーズになります。結果的に、利用者一人ひとりに合ったケアプランを作成できるでしょう。

## フェイスシートを作成する目的や役割

フェイスシートの作成には、情報共有の円滑化や適切なケアプランの立案など重要な目的と役割があります。それぞれを正しく理解し、日々の業務に活かしましょう。

### 多職種間でスムーズに情報共有するため

フェイスシートは、多職種間で利用者の基本情報を共有し、一貫したケアを提供するために作成します。ケアマネージャーや看護師など関係者間で情報にばらつきがあると「緊急時にキーパーソンへの連絡が遅れる」といった問題が生じかねません。
  
  
たとえば、担当者が不在でも別の職員がフェイスシートでアレルギー情報を確認できれば、食事の提供ミスを防げます。
  
  
フェイスシートはチーム全体で正確な情報を共有し、誰でも安全に対応するための情報源です。

### 利用者にあわせたケアプランを作成するため

利用者一人ひとりのケアプランを作成する際も、フェイスシートが役立ちます。
  
  
生活歴や家族背景といった情報がなければ、本人の意向や生活の課題を把握できず、具体的な目標を立てられません。
  
  
具体的には、「家族構成」の項目から支援可能な人数を読み取り、訪問介護を週何回利用するかなどを検討します。
  
  
過不足のないサービスを提供するためにも、フェイスシートの作成が重要です。

### 利用者と信頼関係を築くため

フェイスシートは利用者の負担を減らし、職員との良好なコミュニケーションを促すために活用します。複数の職員が何度も同じ質問をすると、利用者にストレスを与えかねません。
  
  
職員がフェイスシートで趣味を把握し、「〇〇がお好きなのですね」と話しかければ、利用者は「自分を理解してくれている」と安心感を抱くでしょう。
  
  
利用者と関わるときはフェイスシートに目を通して、一人ひとりに寄り添った声かけが大切です。

## フェイスシートとアセスメントシートの違い

上述したように、フェイスシートは施設利用者の基本情報をまとめたプロフィール書類です。一方、アセスメントシートとは、フェイスシートをもとに具体的なケアプランをまとめた書類です。
  
  
フェイスシートとアセスメントシートの違いは、以下のとおりです。

  

|  | フェイスシート | アセスメントシート |
| --- | --- | --- |
| 目的 | 利用者の基本情報を知ること | 基本情報をもとにケアプランを作成すること |
| 内容 | プロフィール | ・健康状態  ・問題行動  ・コミュニケーション能力 など |
| 主な作成者 | ・ケアマネージャー  ・生活相談員 など | ケアマネージャー |
| 作成タイミング | 初回面談時 | ケアプランの作成・更新時 |

  

ケアプランの作成では、まずフェイスシートで利用者の全体像を把握し、次にアセスメントシートで専門的な課題を抽出します。
  
  
フェイスシートには項目の指定がありませんが、アセスメントシートは厚生労働省が指定する「[課題分析標準項目](https://www.fukushi.metro.tokyo.lg.jp/documents/d/fukushi/jiritsushitsu_guideline_chapter2_202511ver)」に従って記載するのが義務付けられています。

## フェイスシートの基本項目

上述したように、フェイスシートには決まったフォーマットはありません。そのため、記入項目にも決まりはありませんが、以下のような基本情報を記載するのが一般的です。

フェイスシートの主な記載項目

* 氏名
* 年齢
* 住所
* 連絡先
* 家族構成
* 生活歴
* 職歴
* 既往歴
* 趣味
* 要介護認定の有無
* 相談内容・困っていること
* アレルギーの有無
* 現在利用している介護・医療サービス　など

たとえば「若い頃は教師だった」「編み物が趣味だった」といった生活歴や趣味の情報もよりよいケアを提供するヒントになる可能性があるため、積極的に記述しておきましょう。

## フェイスシートの書き方

フェイスシートは、関係者の誰がみても利用者の状況が分かるようにまとめることが重要です。
  
  
ここでは、フェイスシートを作成するときのポイントを解説します。

### 記入欄はすべて埋める

フェイスシートの記入欄は、空欄を残さずにすべて埋めましょう。記入欄が空欄だと、後から見た職員が記載漏れなのか、該当する情報がないのかを判断できません。
  
  
項目に空欄が生じる場合は、「特記事項なし」や「本人より聴取拒否」など、理由を記載しておきましょう。
  
  
とくにアレルギーや既往歴といった安全に関わる項目が空欄の場合、ケアの提供に影響します。利用者本人への聞き取りで情報が不十分な場合は、家族にも確認して内容を補完しましょう。

### 誰が読んでもわかる共通言語で書く

フェイスシートは、ケアマネージャーや看護師、介護職員などさまざまな専門職が閲覧します。
  
  
特定の職種にしか伝わらない言葉では、情報の解釈にずれが生じるおそれがあるため、専門用語や略語を避けて記述しましょう。

【専門用語から書き換えの例】

  

* 臥床（がしょう） → ベッドで横になっている
* 拘縮（こうしゅく） → 寝たきりなどが原因で、関節が固まって動きにくくなっている

フェイスシートを作成する職員は、「初めて利用者に関わる職員が読むかもしれない」という視点での記述が大切です。

### 客観的な事実を簡潔に記入する

フェイスシートが曖昧な表現で書かれていると、職員によって解釈が異なるおそれがあるため、客観的かつ具体的に記述することが大切です。

【食欲がない場合】

  

* NG例：「食欲がない」
* OK例：「食事摂取量：主食1/3、副食半分」

  

【歩行が不安定な場合】

  

* NG例：「歩行が不安定」
* OK例：「室内5m歩行で2回のよろめきあり」

ヒアリングした内容だけでは情報が不十分な場合には、実際に利用者の様子を観察させてもらいましょう。食事量や歩行のスピード、注意点などを明確にできます。その際は、実施理由・方法・所要時間・観察された内容などを明確に記載してください。

### 最新の状態に更新する

フェイスシートは一度作成したら終わりではなく、利用者の状態にあわせて情報を更新しなければなりません。
  
  
たとえば、緊急連絡先が古いままになっていると、家族への連絡が遅れてしまい、重要な意思決定に支障が出る可能性があります。
  
  
また、主治医や服薬状況の変更が共有されていない場合、急な体調変化の際に誤った対応をしかねません。
  
  
要介護度の変更やキーパーソンの交代があった際は情報を更新し、チーム全員への共有を徹底しましょう。

### ジェノグラムで家族関係を可視化する

ジェノグラムとは、家族構成や3世代以上にわたる家族の関係性を、図を用いて視覚的に表した家系図のことです。文章だと複雑になりがちな場合は、ジェノグラムで可視化しましょう。
  
  
ジェノグラムで使われる記号は性別や関係性で異なります。

  

| 記号・線 | 意味 |
| --- | --- |
| ⬜︎ | 男性 |
| ◯ | 女性 |
| 二重線の⬜︎もしくは◯ | 本人（利用者） |
| ☆ | キーパーソン |
| 横線 (—) | 婚姻関係 |
| ⚫︎ | 死亡 |

  

また、図形の中にそれぞれの年齢となる数字を記載するのがルールの1つです。
  
  
**たとえば「利用者は70歳の女性、夫は死亡、45歳の娘がいる」場合の描き方は、下図のよになります。**

  
  

ジェノグラムによって、誰が主な介護者で、誰に協力を求められるかといった支援体制を、チーム全体で共有できます。

## フェイスシート作成後の正しい管理方法

フェイスシートには利用者の個人情報が書かれています。作成したら適切に保管することが大切です。保管・管理方法については、施設側で明確なルールを設けるようにしましょう。
  
  
ここからは、フェイスシートの保管や取り扱いに関するポイントを解説します。

### 施設外に持ち出さない

施設外に持ち出すと、紛失や盗難による情報漏えいのリスクが高まります。
  
  
業務上、やむを得ず施設外へ持ち出さなければならない場合には、施設側で持ち出す場合のルールや手続きを定めておきましょう。

### 鍵のかかる場所で保管する

フェイスシートを誰でも閲覧できる状態に置くことは、情報漏えいの直接的な原因となります。
  
  
紙・電子データの媒体問わず、フェイスシートを閲覧できる職員を制限しましょう。

**【媒体別フェイスシートの保管先】**

  

* 紙媒体の場合：施錠可能なキャビネットや書庫
* 電子データの場合：パスワード\*やアクセス権を設定したdrive\*パスワードは定期的に変更する

### メール送信時はパスワードを設定する

フェイスシートの電子データをメールで送信する場合は、必ずパスワードを設定しましょう。第三者が不正にファイルを開くことを防ぐため、情報漏えいのリスクを低減できます。
  
  
以下が具体的な手順です。

1. Microsoft Officeの機能などを使い、ファイルにパスワードを設定する
2. ファイルをメールに添付して送信する
3. 設定したパスワードを、別のメールで送信する

第三者に推測されにくいよう、英数字や記号を組み合わせた複雑なパスワードを設定するのがおすすめです。

### 利用者へのサービス終了後も保管する

利用者へのサービス提供が終了した後も、フェイスシートを含む介護記録は一定期間保管しなければなりません。地域によって異なりますが、多くの自治体ではサービス終了から5年間と定めています。
  
  
たとえば、東京都渋谷区の公式サイトには下記の記載がありました。

> 利用者に対するサービス・活動事業の提供に関する記録は、国基準訪問サービス、訪問型サービスA、国基準通所サービス、通所型サービスAのいずれのサービスにおいても、その完結の日から5年間保存していただく必要があります。
>
> [出典：渋谷マイポータル「介護予防・日常生活支援総合事業のサービス・活動事業の記録の保存期間を教えてください。」](https://dcp.city.shibuya.tokyo.jp/biz/s/faq/faq-000012273)

安全に事業所を運営するため、事前に自治体の公式サイトで確認しておきましょう。

## まとめ

フェイスシートは、利用者の基本情報をチームで共有し、ケアプランの作成や信頼関係の構築を支える重要な書類です。正しい書き方や管理方法を実践することで、一貫したケアの提供につながります。
  
  
緊急時にも役立つものなので、正確かつ最新の情報が常に記載されているように心がけましょう。

## よくある質問

### フェイスシートって何？

介護施設におけるフェイスシートとは、利用者の氏名・生年月日・家族構成・生活歴といった基本情報をまとめた書類です。
  
  
ケアマネージャーや介護職員などのケアに関わるチーム全員が、利用者の情報を同じレベルで把握し、一貫したケアを提供するために作成されます。
  
  
詳しくは、記事内「[介護に必要なフェイスシートとは](#content1)」をご確認ください。

### アセスメントシートとフェイスシートの違いは？

アセスメントシートとフェイスシートの違いは、その書類を使用する目的にあります。
  
  
フェイスシートは、氏名や病歴といった客観的な事実を記録する書類です。一方、アセスメントシートは、事実をもとに「なぜそうなっているのか」「どのような支援が必要か」を明確にする書類といえます。
  
  
詳しくは、記事内「[フェイスシートとアセスメントシートの違い](#content1-2)」をご覧ください。

### フェイスシートにはどんなことを書けばいい？

フェイスシートに決まった様式はないので、施設によって項目は異なります。
  
  
一般的に記載される項目としては、以下のようなものがあります。

* 氏名
* 住所
* 生活歴
* 既往歴
* 服薬状況
* アレルギー
* 家族構成
* 介護保険情報

これらの情報によって、利用者のケアに必要な医療情報や家族との関わりを把握します。
  
  
詳しくは、記事内「[フェイスシートの基本項目](#content2)」で解説しています。
