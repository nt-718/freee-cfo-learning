## 社会保険とは

**社会保険とは、病気やケガなどのリスクに備えるための公的な保険で、健康保険・厚生年金保険・介護保険・雇用保険・労災保険の総称です。**毎月保険料を支払うことで万が一の場合に必要な保障を受けられます。
  
  
**社会保険には「広義」と「狭義」があり、下図の通り、健康保険・厚生年金保険・介護保険・雇用保険・労災保険の5つ全てをまとめて「広義の社会保険」と呼びます。**そのうち健康保険・厚生年金保険・介護保険の3つを「狭義の社会保険」、雇用保険・労災保険をまとめて「労働保険」といいます。（下図）

  
  
各保険の概要は以下の通りです。

| 保険名 | 概要 |
| --- | --- |
| 健康保険 | 病気やケガをしたときなどに必要な給付を受けられる医療保険制度 |
| 厚生年金保険 | 企業に勤める労働者を対象とした公的年金制度 |
| 介護保険 | 要介護認定または要支援認定を受けたときに介護費用の補助などを受けられる制度 |
| 雇用保険 | 従業員が失業したときや病気で働けなくなったときなどに必要な給付を受けられる公的保険制度 |
| 労災保険 | 労働者が業務中や通勤中にケガをしたときなどに必要な給付を受けられる公的保険制度 |

[出典：日本年金機構「事業所が健康保険・厚生年金保険の適用を受けようとするとき」](https://www.nenkin.go.jp/service/kounen/tekiyo/jigyosho/20130502.html)

**以下に該当する事業所は、社会保険のうち健康保険と厚生年金保険の加入が法律で義務づけられています。**

健康保険・厚生年金保険の加入義務がある事業所

* 法人である
* 従業員を常時5名以上雇っている個人事業主（※）  
  ※サービス業の一部（クリーニング業・飲食店・ビル清掃業等）や農業、漁業等は除く

**合同会社も法人のため、設立する場合は必ず社会保険（健康保険・厚生年金保険）に加入しなければなりません。**
  
  
社会保険の区分について、詳しくは別記事「[社会保険とはこんな仕組み！国民健康保険との違いや、切替方法を解説](https://www.freee.co.jp/kb/kb-payroll/the-defference-on-social-insurance-and-national-health-insurance/)」をご覧ください。

## 合同会社の設立時に社会保険の加入義務はある？

**全ての法人には社会保険のうち健康保険と厚生年金保険への加入義務があり、これは、****合同会社であっても同様です。**
  
  
ただし、**「従業員を雇う場合」と「従業員を雇わず一人社長の場合」では、加入義務が生じる社会保険の種類が異なります。**以下では、それぞれのケースで加入が必要な社会保険について解説します。
  
  
なお、広義の社会保険のうち「介護保険」は、従業員の有無による加入義務の違いはないため、本記事では省いて解説をしています。介護保険は、健康保険の加入対象者のうち40歳以上の人が加入対象です。

### 従業員がいない一人社長の場合

従業員がいない一人社長の合同会社の場合、社会保険の加入義務は以下の通りです。

|  | 加入義務 |
| --- | --- |
| 健康保険 | 対象 |
| 厚生年金保険 |
| 雇用保険 | 対象外 |
| 労災保険 |

**健康保険と厚生年金保険は、従業員がいるかどうかに関わらず加入が必要です。**一方で労働保険（雇用保険、労災保険）は、雇用される従業員を対象とした制度のため、従業員がいない一人社長の合同会社であれば加入対象外です。

### 従業員を雇う場合

**合同会社を設立して従業員を雇うのであれば、健康保険・厚生年金保険のほか、労働保険への加入も必要です。**

|  | 加入義務 |
| --- | --- |
| 健康保険※1 | 対象 |
| 厚生年金保険※1 |
| 労災保険 |
| 雇用保険※2 |

※1. 一定の短時間労働者は加入対象外
  
  
※2. 1週間あたりの所定労働時間が20時間以上で、31日以上の雇用見込みがある場合に該当しない従業員は対象外
  
  
合同会社を含む全ての法人は、従業員を雇用する際に健康保険・厚生年金保険への加入が義務付けられています。ただし、**パートやアルバイトの短時間労働者に関しては、以下の条件のいずれかに当てはまる場合には加入対象外です。**

健康保険・厚生年金保険の加入対象外となる短時間労働者の条件

* 週の所定労働時間が20時間未満
* 所定内賃金が月額8.8万円未満
* 雇用の見込みが2ヶ月以内であること
* 学生である

**労災保険・雇用保険は、雇用された従業員を対象とした制度のため、全ての従業員に対して加入義務が生じます。**
  
  

[出典：厚生労働省「労働保険制度」](https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/koyou_roudou/roudoukijun/hoken/980916_1.html)
  
[出典：日本年金機構「適用事業所と被保険者」](https://www.nenkin.go.jp/service/kounen/tekiyo/jigyosho/20150518.html)

### 社会保険に加入できないケース

合同会社であれば原則として社会保険（健康保険・厚生年金保険）への加入が必要ですが、**従業員がいない一人社長の会社では、「役員報酬がゼロまたは低額な場合」は社会保険に加入できないケースがあります。**
  
  
健康保険・厚生年金保険では、役員であっても法人から労務の対償として報酬を受けている者が加入対象とされるので、逆にいえば、報酬がない者は加入対象になりません。
  
  
また、役員報酬が低額だと、社会保険料の給与天引きができないおそれがあることから、加入対象外になることがあります。
  
  
**健康保険・厚生年金保険の加入対象外となる役員報酬額の明確な基準はなく、事業の実態などを踏まえて総合的に判断されます。**役員報酬がゼロまたは低額な場合は、社会保険の加入の要否を年金事務所に確認しましょう。
  
  

[出典：日本年金機構「疑義照会回答（厚生年金保険 適用）」](https://www.nenkin.go.jp/service/seidozenpan/gigishokai.files/kounen_tekiyou.pdf)

### 健康保険と厚生年金保険の加入手続き方法

健康保険・厚生年金保険は、年金事務所へ書類を提出し手続きを行います。合同会社設立時に提出する主な書類は、以下の通りです。

| 書類 | 提出が必要な場合 | 提出期限 | 提出先 |
| --- | --- | --- | --- |
| 健康保険･厚生年金保険新規適用届 | 法人事業の場合は強制加入、個人事業の場合は従業員5人以上で強制加入（一部の業種は任意加入） | 当該事実の発生から5日以内 | 年金事務所 |
| 健康保険･厚生年金保険被保険者資格取得届 | 健康保険･厚生年金保険の被保険者となる従業員を雇用する場合 |
| 健康保険被扶養者（異動）届 | 被保険者となった者に被扶養者がいる場合や被扶養者の追加等があった場合 |

中小企業が多く加入する「[協会けんぽ（全国健康保健協会）](https://www.kyoukaikenpo.or.jp/)」であれば、3つの届出の用紙は[日本年金機構](https://www.nenkin.go.jp/)のサイトからダウンロードできます。
  
  
**合同会社を設立して健康保険・厚生年金保険の適用対象になる場合、まずは「健康保険･厚生年金保険新規適用届」を提出する必要があり、手続きの際には添付書類として登記簿謄本も提出してください。**
  
  
「健康保険･厚生年金保険被保険者資格取得届」は、会社設立時や設立後に加入対象となる従業員を雇用した際に提出します。
  
  
「健康保険被扶養者（異動）届」の提出にあたっては、被扶養者となる家族の収入を確認できる書類や、住民票の写しなど続柄を確認できる書類もあわせて提出します。
  
  

[出典：日本年金機構「新規適用の手続き」](https://www.nenkin.go.jp/service/kounen/tekiyo/jigyosho/20150311.html)
  
[出典：日本年金機構「就職したとき（健康保険・厚生年金保険の資格取得）の手続き」](https://www.nenkin.go.jp/service/kounen/tekiyo/hihokensha1/20150422.html)
  
[出典：日本年金機構「従業員（健康保険・厚生年金保険の被保険者）が家族を被扶養者にするとき、被扶養者に異動があったときの手続き」](https://www.nenkin.go.jp/service/kounen/tekiyo/hihokensha1/20141202.html)

### 労働保険（労災保険・雇用保険）の加入手続き方法

従業員を雇って労働保険の加入手続きを行う場合、主な提出書類は以下の通りです。

| 書類 | 提出が必要な場合 | 提出期限 | 提出先 |
| --- | --- | --- | --- |
| 労働保険関係成立届 | 労働者を雇用して労働保険の適用対象となる場合 | 保険関係が成立した日の翌日から**10日以内** | 労働基準監督署 |
| 労働保険概算保険料申告書 | 保険関係が成立した日の翌日から**50日以内** |
| 雇用保険適用事業所設置届 | **設置の日の翌日から10日以内** | 公共職業安定所 |
| 雇用保険被保険者資格取得届 | **当該事実があった日の翌月10日** |

従業員を雇って労働保険の適用事業となったときは、まずは保険関係成立届を労働基準監督署に提出した後に、その年度分の労働保険料を概算保険料として申告・納付します。保険関係成立届や概算保険料申告書の用紙は、労働基準監督署などで取得できます。
  
  
**雇用保険に関する届出は、保険関係成立届の後でないと手続きができないので、労災保険の届出から先に行いましょう。**
  
  

[出典：厚生労働省「労働保険の成立手続」](https://www.mhlw.go.jp/www2/topics/seido/daijin/hoken/980916_2.htm)

## 合同会社が負担する社会保険料はいくら？

社会保険料の負担割合は、保険の種類によって異なり、会社と従業員の双方で負担します。**従業員の月々の給与からは従業員負担分が控除され、会社は会社負担分を加えて社会保険料として納付する仕組みです。**賞与からも健康保険料・介護保険料・厚生年金保険料・雇用保険料が控除されます。
  
  
負担割合と計算式は以下の通りです。

| 保険 | 負担割合 | 計算式 |
| --- | --- | --- |
| 健康保険 | 会社と従業員で折半 | 標準報酬月額  （標準賞与額）  ×  健康保険料率  （都道府県により異なる） |
| 厚生年金保険 | 会社と従業員で折半 | 標準報酬月額  （標準賞与額）  ×  厚生年金保険料率（18.30%） |
| 介護保険 | 会社と従業員で折半 | 標準報酬月額  （標準賞与額）  ×  介護保険料率（1.60％）※1 |
| 労災保険 | 会社側の全額負担 | 従業員の賃金総額  （給与支給額）  ×  労災保険料率  （業種により異なる） |
| 雇用保険 | 会社と従業員双方の負担  （会社と従業員の負担率は異なる） | 従業員の賃金総額  （給与支給額）  ×  雇用保険料率（15.5%）※2 |

※1. 一般被保険者の場合、2024年3月分（4月30日納付期限分）からの料率
  
※2. 一般事業の場合の料率
  
  
たとえば、協会けんぽの場合、2024年3月分からの東京都の保険料率は、健康保険が9.98％（介護保険の加入対象者は11.58％）、厚生年金保険が18.30％です。月給が20万円であれば、以下の計算式で算出された健康保険料・厚生年金保険料を会社と従業員で負担します。

**【健康保険料】**
  
200,000円 × 9.98％ = **19,960円**
  
  
**【厚生年金保険料】**
  
200,000円 × 18.30％ = **36,600円**

[出典：全国健康保険協会「協会けんぽの介護保険料について」](https://www.kyoukaikenpo.or.jp/g3/cat330/1995-298/)
  
[出典：日本年金機構「厚生年金保険料額表」](https://www.nenkin.go.jp/service/kounen/hokenryo/ryogaku/ryogakuhyo/20200825.files/R06ryogaku.pdf)
  
[出典：厚生労働省「労災保険率表」](https://www.mhlw.go.jp/content/rousaihokenritu_r05.pdf)
  
[出典：厚生労働省「雇用保険料率」](https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/0000108634.html)

## 副業で合同会社を設立すると社会保険はどうなる？

「本業の勤務先」と「副業で設立した合同会社」の両方で社会保険の加入要件に該当したら、日本年金機構が公開している「[健康保険・厚生年金保険被保険者所属選択・二以上事業所勤務届](https://www.nenkin.go.jp/shinsei/kounen/tekiyo/hihokensha/20140820-03.files/28-3.xlsx)」を年金事務所に提出します。**勤務先のうち、どちらの管轄の年金事務所で社会保険の手続きを行うかを選んでください。**
  
  
保険料を計算する際の標準報酬月額は、それぞれの事業所で受ける報酬月額を合算した月額によって決まります。**保険料は、各事業所で受ける報酬月額に基づいて按分した金額をそれぞれの事業所で支払います。**

|  |  |
| --- | --- |
| 提出期限 | 事実発生から10日以内 |
| 提出先 | 選択する事業所の所在地を管轄する年金事務所 |

## 社会保険に加入しなかった場合の罰則（ペナルティ）

社会保険の加入義務がある事業所であるにもかかわらず社会保険に加入していなかった場合、立入調査や罰金などの罰則（ペナルティ）を科される可能性があります。**社会保険に未加入のままでは、雇用調整助成金や産業雇用安定助成金などの助成金の受給もできないので、速やかに加入の手続きを行いましょう。**
  
  
以下では、合同会社の設立時に社会保険に加入しなかった場合に、どのような流れで立入調査などが行われるのか解説します。

### 1. 年金事務所から加入要請が届く

まず、管轄の年金事務所から電話や文書で加入要請があります。この時点での加入では、ペナルティは発生しません。

### 2. 立入検査の警告文書が届く

加入要請に応じないと、立入検査前の警告として、年金事務所で加入手続きをするよう文書で通知が届きます。この時点の加入でも、ペナルティは発生しません。

### 3. 最終的に立入検査・最大過去2年分の保険料を納付

警告文書が届いても放置し続けていると、最終的には立入検査が実施され、強制加入となります。**立入検査まで実施されると、最大で2年間さかのぼって保険料の納付を求められます。**

#### 罰則の適用

**以下のいずれかに該当した事業者は、6ヶ月以下の懲役または50万円以下の罰金が科される可能性があります。**

罰則の対象となる事業者条件

* 被保険者の資格の取得・喪失、報酬月額や賞与額について、適切に届出をしない、または虚偽の届出を行った場合
* 任意適用事業所の取消認可、被保険者資格の取得・喪失の確認、標準報酬（標準報酬月額および標準賞与額）の決定・改定を、被保険者または元被保険者に通知しなかった場合
* 保険料の納付義務に違反し、督促状で指定された期限までに保険料を支払わなかった場合
* 保険料の納付義務に違反し保険料を納付しなかったうえに、健康保険印紙の受払いや現金納付に関する帳簿を備え付ていなかったり、受払状況を保険者に報告しなかったり、虚偽の報告を行っていたりした場合
* 厚生労働大臣または社会保険庁長官による被保険者資格、標準報酬、保険料、保険給付に関する立入検査などで、以下に該当する行為をした場合
  + 書類や物品の提出・提示を拒否した
  + 担当職員の質問に答えなかった
  + 虚偽の答弁をした
* 立入検査を妨害・拒否・回避した

  

[出典：e-Gov法令検索「健康保険法｜第二百八条」](https://laws.e-gov.go.jp/law/211AC0000000070#Mp-Ch_11-At_208)

## まとめ

合同会社を設立すると、原則として社会保険の加入義務が生じます。従業員のいない一人社長の会社でも健康保険・厚生年金保険に加入する必要があり、従業員が一人でもいれば労災保険・雇用保険に加入しなければいけません。
  
  
会社の設立時には、社会保険の加入手続き以外にも数多くの事前準備が必要になり、さまざまな手続きをすることになります。設立準備が忙しくて届出の提出を忘れたり手続きミスをしたりしないように、社会保険の手続きは期間に余裕をもって早めに進めるようにしましょう。

## よくある質問

### 合同会社は社会保険に加入しなければならない？

法人であれば社会保険（健康保険・厚生年金保険）への加入が必須です。合同会社も法人であり、加入義務が生じます。
  
  
詳しくは記事内、「[合同会社の設立時に社会保険の加入義務はある？](#content2)」をご覧ください。

### 合同会社が社会保険に加入しない場合どうなる？

立入検査まで実施されると、最大で2年間さかのぼって保険料の納付を求められます。また、健康保険法第208条に抵触してしまった場合は、6ヶ月以下の懲役または50万円以下の罰金が課される可能性があります。
  
  
詳しくは記事内、「[社会保険に加入しなかった場合の罰則（ペナルティ）](#content6)」をご覧ください。
