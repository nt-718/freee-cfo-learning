## 横浜市で会社を設立する2つの方法

会社を設立する方法は、大きく以下の2つがあります。

横浜市で会社を設立する2つの方法

* 自分で手続きする
* 専門家に依頼する

創業に向けてコストをかけたくない場合は、自分で書類作成して登記申請すると費用を抑えて会社設立が可能です。一方で、手間を省きたい、ミスなく進めたいといった場合は、専門家への依頼が選択肢になります。

### 自分で手続きする

**会社設立を自分で行う場合は、自分で登記書類を作成して法務局に提出します。**申請にあたっては、事前に会社の基本事項を決定し、印鑑・定款の作成が必要です。
  
  
自分で手続きする方法は手間と時間がかかりますが、費用を抑えて申請手続きができ、会社設立・会社法・税金などの知識も身に付きます。
  
  
なお、会社設立の際は「[freee会社設立](https://www.freee.co.jp/launch/)」のような無料サービスを利用すると、自分一人でも手続きがしやすいです。freee会社設立では、各種書類が一括作成できるほか、常駐の「起業ダンドリコーディネーター」にビデオ面談・LINEで相談も可能です。

### 専門家に依頼する

**司法書士・行政書士・税理士・社会保険労務士といった専門家に依頼して会社設立を進める方法もあります。**たとえば税理士には、会社設立前に以下のような相談が可能です。

会社設立前に税理士に相談できること

* 会社設立手続きの代行※
* 必要書類の作成
* 税務に関するアドバイス
* 法務・許認可に関するアドバイス　など

※登記申請は司法書士のみが行える独占業務のため、税理士は対応不可

専門家に依頼する際は費用がかかりますが、申請の手間が省けて本業に集中できます。

## 横浜市で会社設立するときに利用できる創業支援の融資

**会社設立の初期費用や運転資金を調達する際には、横浜市や神奈川県、あるいは日本政策金融公庫の融資制度を利用する方法があります。**創業支援の融資は、条件面で優遇されることもあるため、資金調達が必要な際はぜひ利用を検討しましょう。

横浜市で会社設立するときに利用できる創業支援の融資

* 横浜市中小企業融資制度「創業おうえん資金」
* 横浜市中小企業融資制度「スタートアップおうえん資金」
* 日本政策金融公庫「新規開業資金」
* 神奈川県中小企業制度融資の創業者向けの融資

※各融資制度の情報は、2024年12月時点のものです。最新の情報は横浜市、神奈川県、日本政策金融公庫のウェブサイトでご確認ください。

### 横浜市中小企業融資制度「創業おうえん資金」

**「創業おうえん資金」は、これから創業する人や創業されて間もない人が対象の融資制度です。**

  

| 項目 | 内容 |
| --- | --- |
| 融資対象者（概要） | 次のいずれかに該当する方  ・これから創業する方で、次のいずれかの方  （1）1ヶ月以内に市内で個人事業を開始する方  （2）2ヶ月以内に市内で会社を設立し事業を開始する方   ・すでに創業されている方で、次のいずれかに該当する方  （1）個人事業の開始または会社設立から5年未満の方  （2）個人事業を開始後に会社設立し、事業を譲渡して当該会社に承継させる場合で、個人事業の開始から5年未満の方   ・事業を継続している会社により新たに設立（分社化）された会社で、設立日から5年未満の方 |
| 融資額 | 3,500万円以内 |
| 資金使途 | 運転資金・設備資金 |
| 利率（年利） | 固定金利：1.9％以内 |
| 融資期間 | ・運転資金10年以内 ・設備資金10年以内（据置12ヶ月以内を含む） |
| 保証人 | 必要となる場合がある。 ただし、法人代表者以外の連帯保証人は原則不要 |
| 担保 | 原則不要 |
| 保証料率 | 0.3％（0.1％助成）  ※保証料率は横浜市信用保証協会による保証料率割引（0.4％）適用後の料率 |

  

保証利率は横浜市による補助を適用すると0.3％になります。**また、要件に合致する場合は、保証料率を0.2％上乗せして、経営者保証不要とすることも可能です。**
  
  
申し込みから融資を受けるまでの流れは、以下の通りです。

1. 金融機関に融資の相談・申し込み
2. 金融機関が融資の審査を行い、横浜市信用保証協会に保証を依頼
3. 横浜市信用保証協会が保証の審査を行い、保証を決定
4. 金融機関が融資を実行

[出典：横浜市経済局 横浜市信用保証協会「創業おうえん資金」](https://www.sinpo-yokohama.or.jp/media/2024sougyououentokubetsu.pdf)
  
[出典：横浜市「創業おうえん資金」](https://www.city.yokohama.lg.jp/business/kigyoshien/yushiseido/yushiseido/venture-sougyo.html)

### 横浜市中小企業融資制度「スタートアップおうえん資金」

**「スタートアップおうえん資金」は、対象のスタートアップが利用できる融資制度です。**

  

| 項目 | 内容 |
| --- | --- |
| 融資対象者（概要） | 次のいずれかに該当し、  ・創業を予定されている方で、2ヶ月以内に市内で会社を設立し事業を開始する方   ・すでに創業されている方で、次のいずれかに該当する方  （1）会社設立後5年未満の方  （2）個人事業を開始後に会社設立し、事業を譲渡して当該会社に承継させる場合で、個人事業の開始から5年未満の方   ・事業を継続している会社により新たに設立（分社化）された会社で、設立日から5年未満の方   かつ、以下の条件のいずれかに該当する方  ・横浜市認定特定創業支援等事業を修了した方   ・「YOXOアクセラレータープログラム」そのほか横浜市指定のスタートアップ支援プログラムのいずれかを修了した方   ・東工大横浜ベンチャープラザに現在入居している方   ・「横浜市スタートアップビザ」における確認証明書の発行を受けた方   ・日本政策金融公庫の「挑戦支援資本強化特別貸付（資本性ローン）」を利用している方または本資金と協調融資を受ける方   ・市保証協会が実施する「創業スクール」を修了した方 |
| 融資額 | 3,500万円以内 |
| 資金使途 | 運転資金・設備資金 |
| 利率（年利） | 固定金利：1.5％以内 |
| 融資期間 | 運転資金：10年以内 設備資金：10年以内 （据置1年または要件を満たす場合に3年以内） |
| 担保 | 不要 |
| 連帯保証人 | 不要 |
| 保証料率 | 事業者負担ゼロ（全額助成） |

  

**たとえば、特定創業支援等事業のセミナーを受講して証明書発行を受けることで、スタートアップおうえん資金の要件のひとつを満たすことができます。**そのほか、YOXOアクセラレータープログラム・横浜市指定のスタートアップ支援プログラムを修了した人も対象です。
  
  
流れとしては、金融機関への相談後に、金融機関での審査が実施されます。その後、金融機関から横浜市信用保証協会への保証依頼、保証協会での審査を経て融資が実行されます。

  

[出典：横浜市信用保証協会「スタートアップおうえん資金」](https://www.sinpo-yokohama.or.jp/hosyouseido/2024siseido/2024startupouen/)
  
[出典：横浜市「スタートアップおうえん資金」](https://www.city.yokohama.lg.jp/business/kigyoshien/yushiseido/yushiseido/startup.html)

### 日本政策金融公庫「新規開業資金」

日本政策金融公庫は、中小企業や小規模事業者への融資を行う、財務省管轄の政府系金融機関です。**新規開業資金は、幅広い創業・スタートアップを支援する融資制度です。**

  

| 項目 | 内容 |
| --- | --- |
| 融資対象者 | 新たに事業を始める方または事業開始後おおむね7年以内の方 |
| 融資限度額 | 7,200万円（うち運転資金4,800万円） |
| 資金使途 | 事業開始／事業開始後の設備資金・運転資金 |
| 利率（年利） | 基準利率（2.50～3.60％） ただし、要件を満たす場合は特別利率 |
| 返済期間 | 設備資金：20年以内（うち据置期間5年以内） 運転資金：10年以内（うち据置期間5年以内） |
| 担保・保証人 | 申し込み者の希望を聞きながら相談して決定 |

  

**新たに事業を始める人、事業開始後税務申告を2期終えていない人は、原則として無担保・無保証人で融資が受けられます。**
  
  
女性・若者・シニアの人、廃業歴があり再チャレンジする人、中小会計を適用する人は、それぞれ通常よりも有利な条件で融資制度の利用が可能です。
  
  
手続きは、一般的にインターネット申し込み後に、面談で事業計画などの話をして、融資の決定後に契約手続きをして融資が実行される流れです。

  

[出典：日本政策金融公庫「新規開業資金」](https://www.jfc.go.jp/n/finance/search/01_sinkikaigyou_m.html)

### 神奈川県中小企業制度融資の創業者向けの融資

**神奈川県中小企業制度融資は、神奈川県信用保証協会の保証を付けたうえで、金融機関からの融資が受けられる制度です。**こちらの制度で、創業者向けの融資があります。

  

| 項目 | 内容 |
| --- | --- |
| 融資対象者（概要） | ・これから創業する方で、次のいずれかの方  （1）1ヶ月以内に個人事業を開始する方  （2）2ヶ月以内に法人事業を創業予定の方（NPO法人、医療法人を除く）   ・事業を行っていない個人が事業を開始し、創業後5年を経過していない中小企業者（NPO法人、医療法人を除く）   ・分社化を予定している、または分社化後5年を経過していない会社（中小企業に限る） |
| 融資限度額 | 3,500万円 |
| 資金使途 | 運転資金・設備資金 |
| 利率（年利） | 固定金利：年1.8％以内（創業特例の場合は年1.6％以内） |
| 融資期間 | 1年超10年以内 |
| 担保 | 不要 |
| 保証人 | 原則として法人代表者以外の連帯保証人は不要 |
| 信用保証料率 | 0.40％（経営者保証不要の場合0.60％）  創業特例 0.00％（経営者保証不要の場合0.20％） |

  

申し込みから融資を受けるまでの流れは、以下の通りです。

1. 金融機関に融資の相談・申し込み
2. 金融機関が融資の審査を行い、神奈川県信用保証協会に保証を依頼
3. 神奈川県信用保証協会が保証の審査を行い、保証を決定
4. 信用保証書が発行され、金融機関が融資を実行

[出典：神奈川県「神奈川県中小企業制度融資のしおり」](https://www.pref.kanagawa.jp/documents/10460/r6_seidoyusisiori.pdf)

## 横浜市で会社設立するときの相談先

**会社設立の際には、横浜企業経営支援財団や横浜商工会議所など、地元に密着した相談先で疑問や不安を質問・相談できます。**また、日本政策金融公庫では横浜の支店での相談のほか、オンライン相談も可能です。

横浜市で会社設立するときの相談先

* ワンストップ経営相談（横浜企業経営支援財団）
* 横浜商工会議所
* 日本政策金融公庫

### ワンストップ経営相談（横浜企業経営支援財団）

**ワンストップ経営相談（横浜企業経営支援財団）は、創業・新規事業に関する疑問などの総合的な相談窓口です。**創業やビジネスプラン作成も、中小企業診断士、税理士、社会保険労務士、弁護士、弁理士などの専門家と財団の職員が応えてくれます。
  
  
事前にWeb・電話で予約のうえ、対面またはWeb上（Zoom）で相談を行います。

  

[出典：横浜企業経営支援財団「ワンストップ経営相談（事前予約制）」](https://www.idec.or.jp/business/soudan/onestop.html)

### 横浜商工会議所

**横浜商工会議所では、事業計画書の作成・会社の設立方法・開業資金の借入方法などを、専門家に無料で相談できます。**横浜商工会議所のウェブサイト内から「オンライン予約システム」に進むと、相談の予約が可能です。

  

[出典：横浜商工会議所「創業・経営革新に関わる専門相談」](https://www.yokohama-cci.or.jp/executive/improvement_consultation/)

### 日本政策金融公庫

**日本政策金融公庫では、電話・来店・オンラインで、創業の情報提供やアドバイスが受けられます。**

  

| 項目 | 内容 |
| --- | --- |
| 電話相談 | フリーダイヤルで専門の相談員が対応 |
| 来店・オンライン相談 | ・ビジネスサポートプラザ  （1）全国3ヶ所（新宿、名古屋、大阪）に設置  （2）来店またはオンラインで相談が可能   ・創業サポートデスク  （1）平成24年4月に全国152支店を設置  （2）店舗一覧または創業相談予約のページから予約が可能   ・全国の商工会議所・支援機関での出張相談 |
