## 正常営業循環基準とは

**正常営業循環基準とは、貸借対照表を作成する際に資産・負債を流動か固定かに分類する会計上のルールのひとつです。**英語ではNormal operating cycle ruleと表記されます。

  

**正常営業循環基準は、仕入→製造→在庫→販売→代金回収を基本とした「正常な営業サイクル」により得た資産や負債を流動資産、流動負債とします。**
  
  
営業サイクルは業種によって異なります。業種ごとの営業サイクルの例は以下のとおりです。

業種ごとの営業サイクルの例

* **製造業**  
  仕入→製造→在庫→販売→代金回収
* **小売業**  
  仕入→棚卸資産→販売→代金回収
* **不動産賃貸業**  
  物件の取得・建築→テナント契約→入金→契約満了→保証金・敷金の返却

また、資産の分類方法には、正常営業循環基準のほかに1年基準があります。
  
  
貸借対照表について詳しく知りたい方は別記事「[貸借対照表とは？読み方や作り方から損益計算書との違いまで解説](https://www.freee.co.jp/kb/kb-accounting/balance-sheet/)」をご確認ください。

### 正常営業循環基準と1年基準の違い

**正常営業循環基準は通常「仕入→製造→在庫→販売」の営業サイクルを基準にしており、この期間の長短に制限はありません。**
  
  
一方、1年基準はこの営業サイクルに含まれない資産や負債のうち、回収すべき資産や返済が必要な負債の期日が決算の翌日から1年以内かどうかが基準となっています。
  
  
企業会計原則上、正常営業循環基準を先に適用し、ここに当てはまらない借入金や未払金などについては一年基準を適用します。**正常営業循環基準と一年基準のどちらかに該当するものは流動資産・流動負債となり、当てはまらないものは固定資産・固定負債となります。**

## 流動資産・負債に分類される勘定科目

流動資産・流動負債に分類される勘定科目の例とポイントについてそれぞれ解説します。

### 流動資産

**流動資産に分類される勘定科目は主に以下のとおりです。**

  

| 勘定科目 | ポイント |
| --- | --- |
| 現金 | 小切手などの現金と同様とされているものも含む |
| 普通預金 | 1年以内に満期を迎える定期預金も流動資産として処理 |
| 受取手形 | 支払期日の上限は120日とされており、満期日に支払いが受けられるため流動資産に該当 |
| 有価証券 | 売買や保有目的のものや、1年以内に満期を迎えるものが対象 |
| 売掛金 | 商品を販売した代金が後日支払われる権利であり、取引先が期日までに支払いを行うため、比較的近い将来に現金化できるため、流動資産に該当 |

  

流動資産について詳しくは別記事「[流動資産とは？固定資産との違いから主な種類や流動比率の求め方まで解説](https://www.freee.co.jp/kb/kb-accounting/current-assets/)」をご覧ください。

### 流動負債

流動負債に分類される勘定科目は主に以下のとおりです。

  

| 勘定科目 | ポイント |
| --- | --- |
| 買掛金 | 一般的に買掛金の支払期日は1年以上とならないため、流動負債に該当 |
| 支払手形 | 取引先への支払い手段として振り出しを行った際に用いる |
| 短期借入金 | 返済期日が1年以内の日付に設定されているもの |
| 未払金 | 消耗品など仕入に関わらないものを後払いで購入した際に発生 仕入に関わる後払いは買掛金に該当 |
| 前受金 | 商品の受け渡し完了までは流動負債 |

  

長期借入金の場合も通常は固定負債となりますが、返済期日が1年以内となっているものは流動負債に該当します。
  
  
**営業サイクルの外にある未払金・借入金などの債務は、正常営業循環基準は適用されないため、一年基準が適用されます。**
  
  
流動負債について詳しくは別記事「[流動負債とは？勘定科目や流動負債と固定負債の分類基準をわかりやすく解説](https://www.freee.co.jp/kb/kb-accounting/current-liabilities/)」をご覧ください。

## 固定資産・固定負債に分類される勘定科目

固定資産・固定負債に分類される勘定科目の例とポイントについてそれぞれ解説します。

### 固定資産

固定資産に分類される勘定科目は主に以下のとおりです。

  

| 勘定科目 | ポイント |
| --- | --- |
| 有形固定資産 | 建物や車などの形がある資産。 |
| 無形固定資産 | 特許権やソフトウェアなどの形がない資産。 |
| 投資その他の資産 | 関連会社の株式や長期貸付金などの資産。満期になるまで1年以上あるものが対象 |

  

**建物や車など、時間の経過によって価値が減少していく場合は減価償却資産、土地などの価値が減少しない場合は非減価償却資産となります。**
  
  
固定資産について詳しくは別記事「[固定資産とは？金額基準や流動資産との違い、課税対象となる資産を解説](https://www.freee.co.jp/kb/kb-accounting/non-current-assets/)」をご覧ください。

### 固定負債

固定負債に分類される勘定科目は主に以下のとおりです。

  

| 勘定科目 | ポイント |
| --- | --- |
| 社債 | 一般的には返済期限が1年以上先であるため、固定負債に該当。 \*返済期限が1年未満の場合は流動負債となる |
| 長期借入金 | 返済日まで1年以上あるものは固定負債に該当。 \*返済日が1年未満の分は流動負債に分けて仕訳を行う |
| 繰延税金負債 | 将来的に支払う税金に相当する金額を計上する際は固定負債に該当 |

  

固定負債について詳しくは別記事「[固定負債とは？流動負債との違いや勘定科目をわかりやすく解説](https://www.freee.co.jp/kb/kb-accounting/fixed-debta/)」をご覧ください。

### 正常営業循環基準と1年基準の違いは？

**どちらも貸借対照表の資産と負債を分類するためのルールです。**
  
  
正常営業循環基準を優先して適用し、当てはまらない物については一年基準を適用します。正常営業循環基準と一年基準のどちらかに該当するものは流動資産・流動負債、当てはまらないものは固定資産・固定負債となります。
  
  
詳しくは記事内「[正常営業循環基準と1年基準の違い](#content1-1)」をご覧ください。
