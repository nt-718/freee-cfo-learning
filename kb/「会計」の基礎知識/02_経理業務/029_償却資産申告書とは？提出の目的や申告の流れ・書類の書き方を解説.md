償却資産申告書は、土地・建物・車両（大型特殊車両以外）を除く事業用の機械や器具などの所有を自治体に申告するための書類で、固定資産税の算定に用いられます。
  
  
所有状況が把握しづらい償却資産を適正に管理するために提出が義務付けられており、期限までに申告しないと過料などのペナルティが科される可能性があります。
  
  
なお、申告対象は有形固定資産ですが、少額資産は申告不要となるケースがあるため、どの資産が対象になるのか事前に確認しましょう。
  
  
本記事では、償却資産申告書の概要や対象となる資産、書き方・記入例、提出方法などを詳しく解説します。

## 償却資産申告書とは

**償却資産申告書とは、各自治体が固定資産税を正しく計算するために、法人や個人に対して所有する償却資産に関する自己申告を求める書類のことです。**
  
  
償却資産とは、**建物・土地・車両（大型特殊車両以外）を除く事業用の機械や器具などの資産**を指し、これに該当する償却資産には固定資産税が課せられます。
  
  
償却資産は、資産ごとに決められた使用可能年数（耐用年数）にわたって取得価額を配分し、費用計上（減価償却）していきます。
  
  
建物や土地も固定資産税の対象ですが、償却資産には該当しません。建物や土地は登記によって所有状況が明らかになっているため、申告する必要がありません。
  
  
また、車両（大型特殊車両以外）は自動車税の対象となるため、償却資産に対する固定資産税の対象とはなりません。

## 償却資産の対象となるもの

**償却資産の申告対象は、毎年1月1日時点で所有している償却資産です。**
  
  
前年12月31日に取得し、翌1月1日時点で所有している資産については当年に申告を行う必要があります。取得時点が1月2日である場合、その資産については当年は申告せず、翌年の申告対象になります。
  
  
業種に関係なく、地方税である償却資産の主な対象は下表の通りです。

  

| 償却資産の種類 | 内容 |
| --- | --- |
| 構築物 | 舗装路面、庭園、門・塀・緑化施設等の外構工事、看板（広告塔等）、ゴルフ練習場設備、受変電設備、予備電源設備、その他建築設備、内装・内部造作等 |
| 機械および装置 | 各種製造設備等の機械および装置、大型特殊自動車のうち建設機械に該当するもの（分類番号が「0、00～09および000～099」）、機械式駐車設備（ターンテーブルを含む）等 |
| 船舶 | ボート、釣船、漁船、遊覧船等 |
| 航空機 | 飛行機、ヘリコプター、グライダー等 |
| 車両および運搬具 | 大型特殊自動車（分類記号が「9、90から99、900から999」の車両）等 |
| 工具、器具および備品 | パソコン、陳列ケース、看板（ネオンサイン等）、医療機器、測定工具、金型、理容および美容機器、衝立等 |

[出典：東京都主税局「固定資産税（償却資産）」](https://www.tax.metro.tokyo.lg.jp/kazei/work/shokyak_sis#gaiyo_03)

なお、申告対象資産の課税標準額が免税点（150万円）未満となる場合、固定資産税は課税されませんが、申告は必要です。また、本来減価償却が可能であるものの現実に減価償却を行っていない資産や、耐用年数が経過して償却が済んでいる資産なども申告の対象です。
  
  
どのような種類の資産が申告対象になるかは、事前に自治体のホームページや送付される「償却資産（固定資産税）申告の手引き」を参考にしましょう。

### 無形固定資産は償却資産申告の対象外

**無形固定資産は、形を持たない、かつ1年を超えて利用される資産を指し、償却資産申告の対象外です。**
  
  
主に下表にあるものが無形固定資産です。

  

| 無形固定資産の 対象となるもの | 内容 |
| --- | --- |
| 特許権 | 発明を保護し独占するための権利 |
| 商標権 | 商品やサービスを区別するために使用する「マーク」や「ネーミング」を保護し独占する権利 |
| 実用新案権 | 物品の形状、構造または組み合わせにかかる考案を保護し独占するための権利 |
| 意匠権 | 工業デザインを守り独占する権利 |
| 借地権 | 地代を支払い、借りた土地（借地）の上に建物を建てる権利 |
| 鉱業権 | 鉱業法に基づき、一定の区域で鉱物のある地層から鉱物を採掘し、取得できる権利 |
| ソフトウェア | コンピューターに一定の仕事を行わせるためのプログラム |
| のれん | 目に見えない資産価値をいい、ブランドやノウハウが対象 |
| 電話加入権 | NTT東日本またはNTT西日本の電話回線を契約する際に必要となる権利 |

### 少額資産が申告不要となるケース

取得価額が少額である資産の場合、法人税法などで各種特例が設けられています。以下に当てはまる場合は、償却資産としての申告が不要です。

少額資産について、償却資産申告が不要のケース

1. 使用可能期間が1年未満または取得価額が10万円未満の資産で、使用開始時に全額を損金算入した資産
2. 取得価額が20万円未満で、使用した年以後3年間にわたって均等に損金算入する資産（いわゆる一括償却資産）

他方、以下に当てはまる場合は償却資産の申告を行わなければなりません。

少額資産について、償却資産申告が必要なケース

1. 租税特別措置法の規定により、中小企業特例（少額減価償却資産の特例）を適用して損金算入した資産
2. 少額であっても個別に減価償却することを選択した資産

中小企業特例とは、取得価額が30万円未満であり、2006年4月1日から2026年3月31日までの間に取得した減価償却資産について、中小企業の場合はその取得価額全額を損金に算入できるというものです。
  
  
2.について、法人が個別に10万円未満の資産を減価償却資産として計上した場合には、償却資産の申告対象となりますが、10万円未満を損金として処理した場合は申告する必要はありません。
  
  
なお個人の場合は、10万円未満は全て必要経費として計上するため、償却資産の申告対象外となります。

  

[出典：東京都主税局「固定資産税（償却資産）」＞「Q7 少額資産は申告の対象になりますか。」](https://www.tax.metro.tokyo.lg.jp/kazei/work/shokyak_sis#gaiyo_03)
  
[出典：国税庁「No.5408 中小企業者等の少額減価償却資産の取得価額の損金算入の特例」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/hojin/5408.htm)

## 償却資産申告書の提出が必要な理由

償却資産には、建物や土地のように登記制度が設けられておらず、自治体で各法人や個人の償却資産の所有状況を確認・把握することは困難です。
  
  
そのため、償却資産の申告が地方税法で義務付けられ、**自治体は法人や個人が提出した償却資産申告書を利用して償却資産の保有状況を確認し、その内容をもとに税額を計算します。**
中には、申告書の内容が正しいことを確認するために、資料提供が求められたり、自治体の担当者による現地調査が実施されたりするケースもあります。

### 償却資産申告書を提出しない場合どうなるか

償却資産申告書を提出しなかった場合、虚偽の申告をした場合の罰則は以下の通りです。

  

| 該当の条文 | 罰則 |
| --- | --- |
| 正当な理由なく償却資産申告書を提出しなかった場合 | 地方税法第386条と 各自治体の条例 10万円以下の過料 |
| 虚偽の申告をした場合 | 地方税法第385条 1年以下の懲役または 50万円以下の罰金 |

[出典：e-Gov法令検索「地方税法（昭和二十五年法律第二百二十六号）」](https://laws.e-gov.go.jp/law/325AC0000000226/)

正当な理由なく償却資産を申告しない場合、地方税法第386条と自治体の条例により**10万円以下の過料**が科されることがあります。
  
  
また、虚偽の申告については地方税法第385条の規定により罰則が設けられ、**1年以下の懲役または50万円以下の罰金**が科される可能性があります。
  
  
対象の償却資産については正しく認識し、正確な処理を行いましょう。

### 償却資産がない場合は償却資産申告書が提出不要になる？

**償却資産を所有していない場合でも、多くの自治体で償却資産がない旨の申告が求められます。**
  
  
たとえば東京都の場合、償却資産を所有していない人は「該当資産なし」として申告を行います。しかし、すでに「該当資産なし」で申告している人で増加資産がない場合は、翌年度以降の申告は不要です。
  
  
なお、リース資産など例外的に報告が必要なケースもあるため、各自治体のウェブサイトなどで申告の要否や方法を確認し、必要な際には忘れずに申告を行いましょう。

  

[出典：東京都主税局「固定資産税（償却資産）申告の手引き」](https://www.tax.metro.tokyo.lg.jp/documents/d/tax/R7_shinkokutebiki_mihiraki)

## 償却資産申告から納税までの流れ

償却資産の申告から納税までの一連の流れを、東京都の場合を例に紹介します。

  

| No | 流れ | 内容 |
| --- | --- | --- |
| 1 | 償却資産申告書の提出 | 1月1日時点で所有している償却資産について申告書を作成し、同年1月31日までに提出する |
| 2 | 償却資産の評価 | 申告内容や調査をもとに、自治体が資産ごとの評価額を決定する |
| 3 | 課税台帳への登録・公示 | 自治体が資産の評価額などを償却資産課税台帳に登録し、その旨が公示される |
| 4 | 課税台帳の閲覧 | 償却資産課税台帳に登録された価格などが閲覧可能になる |
| 5 | 審査の申出 | 償却資産課税台帳に登録された価格について不服がある場合は、審査の申し立てができる |
| 6 | 税額の算出・納税通知書の交付 | 各資産の評価額を足し合わせた「課税標準額」をもとに税額が算出され、6月上旬に納税通知書が発行される |
| 7 | 審査の請求 | 課税内容に不服がある場合は、審査の請求ができる |
| 8 | 納税 | 通常4回の納期で納付する（東京23区では6月・9月・12月・翌2月） |

[出典：東京都主税局「固定資産税（償却資産）」](https://www.tax.metro.tokyo.lg.jp/kazei/work/shokyak_sis#gaiyo_04)

なお、不服申し立てをした場合は、いったん決定されている金額で納税します。審査の結果、申し立ての内容が認められて納税額に変更が生じた場合には、後日還付もしくは納付の案内が届きます。

## 償却資産申告書の書き方・記入例

償却資産申告書の書き方には、以下の2つがあります。

償却資産申告書の書き方の種類

* 電算処理方式
* 一般方式

電算処理方式は、申告者がソフトウェアやシステムを使用して所有資産の評価額を算出し、提出する方法です。
  
  
償却資産の取得が頻繁に行われる法人などには適していますが、基本的には申告者が資産に関する必要な情報を提出し、自治体側で評価額を算出する一般方式のほうが簡単です。
  
  
以下では、一般方式による申告書別の記載項目と書き方を詳しく説明します。なお書き方については、年度により変更が生じる可能性があるため、毎年自治体で公表される手引書もご参照ください。

### 種類別明細書（増加資産・全資産）

[出典：東京都主税局「種類別明細書（増加資産・全資産用）」](https://www.tax.metro.tokyo.lg.jp/documents/d/tax/shinkokusho_03_a)
  

**増加資産・全資産用の種類別明細書には、前年の1月2日から当年1月1日までに取得した資産について記入します。**
  
  
所有者名や明細書の枚数、年度の記載のほか、種類別明細書で記載が必要な項目は下記の通りです。

  

| 項目名 | 記載内容 |
| --- | --- |
| 資産の種類 | 1.構築物 2.機械および装置 3.船舶 4.航空機 5.車両および運搬具 6.工具、器具および備品 からひとつ選択 |
| 資産の名称 | 20文字以内で記載（例：冷暖房用機器） |
| 数量 | 全く同じものを同時に設置した場合は購入数を記載（多くの場合は1を記載） |
| 取得年月 | 年号は、1.明治 2.大正 3.昭和 4.平成 5.令和（令和の場合は印刷されているため記入不要）  年月は、資産を取得した年月を記載 ※1月1日に取得した場合は、前年の12月を取得年月とする |
| 取得価額 （手数料や据付料など付随費用を含む） | 資産を取得するために支出した、または支出すべき金額を記載（税抜経理方式を採用している場合は税抜き価格、税込経理方式を採用している場合は税込み価格） |
| 耐用年数 | 減価償却資産の耐用年数等に関する省令により該当するものを記載 |
| 増加事由 | 1.新品取得 2.中古品取得 3.移動による受け入れ 4.その他 から選択 |
| 摘要 | 特記事項がある場合、また増加事由で「4. その他」を選択した場合に記入 |

[出典：東京都主税局「（2） 種類別明細書（増加資産・全資産用）の記入方法」](https://www.tax.metro.tokyo.lg.jp/documents/d/tax/R7_shinkokusho_kinyu2)

### 種類別明細書（減少資産）

[出典：東京都主税局「種類別明細書（減少資産用）」](https://www.tax.metro.tokyo.lg.jp/documents/d/tax/shinkokusho_05_a)
  

**減少資産用の種類別明細書には、前年1月2日から当年1月1日までに異動、つまり資産の全部または一部の減少や資産の一部の修正があった資産について記入します。**
  
  
増加資産・全資産用の種類別明細書と、記載事項はおおむね同じです。異なる部分は、下記の通りです。

  

| 項目名 | 記載内容 |
| --- | --- |
| 異動区分 | 1.減少 3.修正 からひとつ選択 |
| 耐年改正 | 耐用年数省令の改正によって耐用年数を変更する場合、〇印を記載 |
| 減少等の事由 | 1.売却 2.除却 3.移動 4.その他 からひとつ選択 |
| 摘要 | 売却先・移動先や具体的な減少内容、減少した取得価額、修正が発生した理由など、「減少等の事由」に対応する内容を記載 |

  

資産の名称や取得年月を修正する場合、資産の一部が減少した場合などは異動区分で修正を選択し、資産名称や取得年月、取得価額、数量など変更のある項目に抹消線を引いて欄内の下段に変更後の内容を記載します。
  
  
減少等の事由で「4. その他」を選択した場合には、摘要欄への事由の記載が必須です。

  

[出典：東京都主税局「（3）種類別明細書（減少資産用）の記入方法」](https://www.tax.metro.tokyo.lg.jp/documents/d/tax/R7_shinkokusho_kinyu3)

### 償却資産申告書（償却資産課税台帳）

## 償却資産申告書の提出先と提出期限

**償却資産申告書の提出期限は、1月31日です。**1月31日が土曜日・日曜日・祝日などの場合は、翌開庁日が提出期限となります。
  
  
なお、償却資産申告書の提出先は、償却資産の所在する市区町村の役所（東京23区は償却資産の所在する区にある都税事務所）です。

## 償却資産申告書の提出方法

償却資産申告書は、以下の方法で提出できます。

償却資産申告書の提出方法

* 市区町村の役所（東京23区は都税事務所）の窓口での提出
* 郵送での提出
* eLTAX（エルタックス）での電子申告

償却資産申告書の提出は、各市区町村の役所窓口への持ち込みや郵送のほか、eLTAXからも手続きできます。eLTAXとは、地方税の手続きをインターネットで実施できる地方税ポータルシステムです。
  
  
窓口の混雑緩和のためにeLTAXや郵送での提出を推奨している自治体もあります。窓口での提出以外の方法もぜひ積極的に利用していきましょう。

  

[出典：地方税共同機構「地方税ポータルシステム」](https://www.eltax.lta.go.jp/eltax/gaiyou/)

## まとめ

償却資産申告書は、自治体が固定資産税を算出するにあたって、法人・個人の償却資産の所有状況を把握するために提出を求める書類です。
  
  
償却資産申告書の書き方については自治体ごとに毎年手引書が用意されており、計算式はシンプルです。1月31日までに市区町村の役所（東京23区は都税事務所）に、窓口・郵送・eLTAXのいずれかの方法で提出します。
  
  
正当な理由なく償却資産を申告しない場合、申告内容に虚偽がある場合は罰則が科されるおそれもあるため、償却資産の正しい認識と適切な処理が重要です。
  
  
また、償却資産を所有していない場合も、多くの自治体で償却資産がない旨の申告が求められるため、忘れずに申告書を提出しましょう。

## よくある質問

### 償却資産申告書はなぜ提出する？

土地や建物と異なり、償却資産には登記制度がないため、自治体にとっては各法人や個人の償却資産の所有状況を把握するのが困難です。自治体が償却資産の所有状況をもとに正しく税額を計算するために、償却資産申告書の提出が求められます。
  
  
詳しくは、記事内「[償却資産申告書の提出が必要な理由](#content3)」をご覧ください。

### 償却資産申告書を提出しないとどうなる？

正当な事由がなく申告をしなかった場合は、地方税法第386条などにより10万円以下の過料が科されます。虚偽の申告をした場合も、罰則規定が設けられています。
  
  
詳しくは、記事内「[償却資産申告書を提出しない場合どうなるか](#content3-1)」で解説しています。

### 償却資産申告書の提出先は？

自治体によって異なりますが、一般的に償却資産の所在する市区町村の役所に提出します。
  
  
詳しくは、記事内「[償却資産申告書の提出先と提出期限](#content6)」をご覧ください。
