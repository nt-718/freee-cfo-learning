## 経理とは

経理とは、会社経営において重要なお金の管理をする部門です。会社が利益や資産を生み出すためには、お金の流れを数値化し正確に管理することが大切で、経理はそんなお金に関わる業務を行います。
  
  
経理の具体的な業務としては、主に以下のとおりです。

経理業務の具体例

* 日々の売上管理
* 日々の仕入れ管理
* 給与や保険の管理・計算
* 税金の計算
* 決算書作成

また、これらのデータを資料にまとめるのも経理の仕事の1つで、経営陣に対して財務状況に関する改善提案を行う場合もあります。
  
  
経営陣は、経理がまとめた資料や提言をもとに経営状態を把握し、今後の意思決定に役立てる仕組みです。経理の仕事は一つひとつの業務内容に派手さはなく、コツコツと地道に積み重ねていくものばかりですが、「縁の下の力持ち」として重要な役割を担っています。

### 経理と会計の違い

会計とは、本来は「お金や物のやりとり、それらの出入りを記録すること」を意味します。つまり、**会社のお金の流れを全体的に把握するために必要な業務が「会計」です。**そのため、経理と会計はまったくの別物というわけではありません。
  
  
また、会社の会計は財務会計と管理会計の2つに分けられます。
  
  
財務会計は、株主や金融機関などの利害関係者に企業の財政状態と経営成績を開示することが、主な目的です。一方の管理会計は、経営者が会社の経営状況を正確に把握するために行われます。
  
  
経理は、上記の会計業務を行うために、日々の売上や仕入れなど、会社の日常的な出納管理を行う仕事です。また、日々のお金の流れの管理に加え、月次の仕事である給与・保険の管理や年次の仕事である決算書作成なども経理は行います。
  
  
そのため、会社によっては会計と経理を明確に区別していないことも多く、一人がすべての業務を兼任するケースも多いです。

### 経理と財務の違い

会計と財務は、業務内容や目的が明確に異なります。
  
  
財務の主な業務は、金融機関との融資交渉などの資金調達や投資、M&Aといった資産運用です。財務担当者は、経理が作成した貸借対照表（B/S）や損益計算書（P/L）などの決算書・将来の事業計画等をもとに、資金調達や資産運用を企画します。
  
  
経理・会計業務は現状を把握するために有効なのに対し、財務は現状を把握したのちにこれから動かす予定のお金を管理することが特徴です。
  
  
会社によっては、財務専門の部門を設けることもありますが、経営者が財務業務を行うこともあります。なお、貸借対照表や損益計算書について詳しく知りたい方は、別記事「[貸借対照表とは？ 会社の財務状況を簡単に把握！](https://www.freee.co.jp/kb/kb-accounting/balance-sheet/)」と「[損益計算書とは？ 項目別の見方やチェックポイント、活用法を解説](https://www.freee.co.jp/kb/kb-accounting/pl/)」をあわせてご確認ください。

### 経理と簿記の違い

上述したように、**経理**とは職種や会社における部門のことを指すものです。一方の**簿記**は、経理業務に用いられるスキルのことであり、具体的な仕事内容や職種ではありません。
  
  
ただし、経理が行う業務には簿記に関する知識が含まれているため、経理業務を行うにあたって簿記のスキルは欠かせません。経理担当者になるために簿記資格を取得する人も多くいるため、これから経理を目指す場合は簿記の勉強に取り組むのもおすすめです。
  
  
なお、簿記について詳しく知りたい方は、別記事「[簿記とは？仕組みや学ぶメリットを初心者でもわかりやすく解説](https://www.freee.co.jp/kb/kb-accounting/book-keeping/)」をご確認ください。

## 経理の仕事内容

経理の年間業務は、「日次業務」「月次業務」「年次業務」の3つに分類できます。それぞれの業務内容やスケジュールなどについて、詳しく確認してみましょう。
  
  
また、経理業務の年間スケジュールについて詳しく知りたい方は、別記事「[法人の経理業務の年間スケジュールとは？各決算月ごとの業務内容について解説](https://www.freee.co.jp/kb/kb-accounting/accounting-annual-schedule/)」をご確認ください。

### 日次業務（毎日行う仕事）

日次業務では、日々の会社の取引をすべて記録する業務などを中心に行います。具体的には以下のとおりです。

日次業務の例

* 現金出納管理
* 経費精算
* 伝票記帳・整理
* 売掛金や買掛金の管理

これらの業務は毎日コツコツと行わなければならない細かい作業ですが、決算書を作成する際に欠かせないデータになるので、抜け漏れや間違いのないように取り組みましょう。

### 月次業務（毎月行う仕事）

月次業務とは、毎月行う経理業務のことで、主な業務は以下のとおりです。

月次業務の例

* 給与計算と社会保険料の計算
* 月次決算書の作成
* 予算実績管理　など

また、これらの業務は、一般的に以下のサイクルで前半と後半に分かれて実施されます。

  

| 経理の月次業務 | |
| --- | --- |
| 前半 | 取引先の入金確認 |
| 月次決算書作成 |
| 予算実績管理 |
| 住民税・源泉所得税の納付（毎月10日まで） |
| 後半 | 給与計算 |
| 取引先への支払い・請求書発行 |
| 社会保険料の納付 |

  

経理担当者は、1ヶ月単位で会社の予算計画と現状の実績を照らし合わせ、売上の増減や削減できる経費などについて確認します。また、経営陣は月次決算書をもとにその後の経営方針を決定するケースが多いので、月次決算書は月初にスピード感をもって作成することが大切です。
  
  
さらに、経理の月次業務には売上代金の請求・支払い、税金の納付などの業務があります。源泉所得税は原則として翌月10日までに納付する義務がありますので、忘れずに納付しましょう。

> 源泉徴収した所得税及び復興特別所得税は、原則として、給与などを実際に支払った月の翌月10日までに国に納めなければなりません。
>   
>   
>
> [出典：国税庁「No.2505 源泉所得税及び復興特別所得税の納付期限と納期の特例」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/gensen/2505.htm)

### 年次業務（毎年行う仕事）

年次業務とは、経理担当者が事業年度内に1度行う業務のことです。3月決算の会社の場合、経理の年間の仕事サイクルは以下のようになります。

  

| 経理の年次業務 | |
| --- | --- |
| 4月 | 決算整理 |
| 5月 | 年次決算書作成 |
| 税務申告 |
| 6月 | 賞与計算・振込 |
| 社会保険の算定基礎届提出 |
| 7月 | 労働保険の更新 |
| 8月 | ー |
| 9月 | ー |
| 10月 | ー |
| 11月 | 中間税務申告 |
| 12月 | 賞与計算・振込 |
| 年末調整 |
| 1月 | ー |
| 給与支払報告書・法定調書の提出 |
| 償却資産税申告書提出 |
| 2月 | ー |
| 3月 | 実地棚卸 |

  

このように、経理には1年を通してさまざまな業務がありますが、なかでも事業年度の取りまとめとなる「決算業務」は特に重要です。年度末の決算期では、会計帳簿をもとに損益計算書や貸借対照表などの決算書を作成します。
  
  
決算書とは、一会計年度の会社の利益や財務状況を示すもので、法人税などの申告や株主への報告にも使われる重要な書類です。そのほか、年末調整業務は従業員のお金に関わることで連携が必要になるため、普段からコミュニケーションを取るなどして事前に準備しておきましょう。
  
  

[出典：国税庁「申告と納税」](https://www.nta.go.jp/publication/pamph/koho/kurashi/html/06_1.htm)
  
[出典：厚生労働省「労働保険の年度更新とは」](https://www.mhlw.go.jp/bunya/roudoukijun/roudouhoken01/kousin.html)
  
  
決算書の作成方法について詳しく知りたい方は、別記事「[決算書の作り方を解説！作成手順や必要書類とは？](https://www.freee.co.jp/kb/kb-accounting/dos-for-financial-statements/)」をご確認ください。

## 【状況別】経理業務の違い

上述した経理業務は基本的にどの会社でも同じですが、会社規模や業種によって違いが生じることがあります。経理担当者の負担にも関わることなので、以下で詳しく確認しましょう。

### 会社規模による経理業務の違い

会社の規模が大きいほど、経理業務の量は多くなります。これは、従業員や発生する取引が多くなることで、仕訳作業や経費処理などにかかる工数が増えるからです。
  
  
しかし、大企業では経理に携わる人員も複数配置しているケースも多く、数十人単位の部署として設けていることもあります。そのため、業務も担当者ごとに細かく分担されていて連携が必要になりますが、1人当たりの負担はそれほど大きくないでしょう。
  
  
一方で、中小企業などの小規模組織では、1人や少人数で経理を行っているケースが多いです。そのため、各業務量自体は少なくてもすべての業務を担当しなければならないため、負担が大きくなるかもしれません。

### 業種による経理業務の違い

経理の業務内容は、業種によっても変わることがあります。これは、ビジネスモデルやそれに伴うお金の管理の仕方が業種によって異なるためです。
  
  
たとえば、小売業では経理担当者が仕入管理や実地棚卸を担当したり、製造業では製造コストを計算したりといった各業種ならではの対応が必要になります。

## 経理の仕事に向いている人の特徴

比較的忙しいとされがちな経理の仕事ですが、以下の特徴に当てはまる場合は、経理の仕事に向いているでしょう。

  

* ・几帳面な性格で丁寧に仕事ができる人
* ・数字を取り扱うのが好きな人
* ・簿記や会計などの専門知識がある人
* ・年齢性別問わずコミュニケーションを取れる人

### 几帳面な性格で丁寧に仕事ができる人

経理は会社のお金を扱う仕事であり、少しの金額のズレや漏れも許されません。そのため、几帳面な性格で確認を怠らず、常に丁寧な仕事ができる人に向いている仕事だといえます。
  
  
その分責任も大きくなってしまいますが、丁寧な仕事をする人は信頼されやすく、経理としてのやりがいも感じられるでしょう。

### 簿記や会計などの専門知識がある人

経理は会社のお金を管理する仕事であるため、毎日のように数字やデータを取り扱います。仕訳や書類作成などの業務では、Excelや会計ソフトなどを使用することが一般的です。
  
  
そのため、日頃から計算やデータ処理に慣れており、かつ好きであるという人は経理に向いているといえます。

### 簿記や会計などの専門知識がある人

経理が行う業務は会社法や税法などに関わってくることから、簿記や会計の知識が求められます。未経験だと用語から仕事内容まで身につけるには時間がかかるため、できるだけ事前にスキルや知識を身につけておくのがおすすめです。
  
  
逆にいえば、すでに経理に必要な簿記・会計の知識があるという人は、経理が欲しい会社にとって重宝されるでしょう。また、実務経験がある場合はなおさらです。

### 年齢性別問わずコミュニケーションを取れる人

経理担当者には、社内外問わず多くのコミュニケーションが要求されます。見積書や請求書の手配では社外の取引先と、年末調整や決算書作成では社内の従業員や経営陣というように、さまざまな方面での連携が必要です。
  
  
そのため、年齢性別問わずスムーズなコミュニケーションが取れる人は、経理業務に向いているといえます。
  
  
経理初心者向け勉強方法について詳しく知りたい方は、別記事「[【経理初心者】おすすめの勉強法は？求められる知識やスキルを確認！](https://www.freee.co.jp/kb/kb-accounting/accountant_study_skills/)」をご確認ください。また、経理の仕事に関する記事は、別記事「[経理の仕事は難しい？業務内容や業務を効率化する方法について解説](https://www.freee.co.jp/kb/kb-accounting/accounting-difficult/)」「[経理の仕事はきつい？向いている人の特徴や解決策をわかりやすく解説](https://www.freee.co.jp/kb/kb-accounting/accounting-hard/)」「[経理は忙しすぎる！？忙しさを解消する方法や経理業務の繁忙期をわかりやすく解説](https://www.freee.co.jp/kb/kb-accounting/accounting-busy/)」もご確認ください。

## 経理の仕事におけるやりがいとメリット

経理の仕事には、以下のようなやりがい・メリットがあります。

経理の仕事におけるやりがいとメリット

* 会計や財務などの専門性が高い知識と経験が身につく
* 会社の経営層と近い距離で仕事ができる
* 会社全体の役に立つことができるとができる

### 会計や財務などの専門性が高い知識と経験が身につく

経理が行う仕事は、法人の決算や税金にかかわるものであるため、専門性の高い知識と経験を積むことができます。また、会計・財務の知識は転職時だけでなく、独立時や経営者の立場になった際にも非常に役に立つでしょう。

### 会社の経営層と近い距離で仕事ができる

経理は、会社のお金に関わる経済的な業務を行うため、会社の経営陣との関わりも自然と増えます。たとえば、毎月の業績を貸借対照表や損益計算書などにまとめて報告するなど、会社にもよりますが経営層の動きがよく見えるポジションです。会社の経営層と近い距離での仕事が多く、同時に会社のお金を扱うという責任からも、やりがいを感じる場面は多くあるでしょう。

### 会社全体の役に立つことができる

経理は、社員の給料を支払ったり会社の税金を支払ったりと、会社を運営する上で欠かせない仕事です。毎日コツコツと業務を進めるため、営業などとは異なり大きく売り上げに貢献できるわけではありませんが、会社の事業を円滑に進めるために全ての部署に日々貢献できる仕事です。

## まとめ

経理とは、会社のお金や取引の流れを管理する役割を担う、会社経営において必要不可欠な部門です。主な業務には、日々の取引の仕訳や毎月行う給与支払い、年次決算における決算書作成などがあります。
  
  
日頃から社内外の多くの人と関わって、毎日コツコツと業務を進めていくことが経理の特徴です。そのため、コミュニケーションが好きで几帳面な性格をしている人に向いている仕事といえるため、ぜひ挑戦してみてください。

## よくある質問

### 経理とはどんな仕事ですか？

経理とは、会社で発生した取引の仕訳や決算書の作成などを行う仕事です。会社経営に必要不可欠な部門で、お金を取り扱う責任のある仕事だといえます。また、会計や簿記などの専門知識が必要な仕事です。詳しくは記事内「[経理とは](#content1)」をご覧ください。

### 経理はどういう人に向いていますか？

経理はお金を取り扱うミスの許されない仕事であるため、几帳面で丁寧に仕事ができる人に向いています。また、コミュニケーションを多く取るポジションでもあるため、人を選ばずにスムーズなやり取りができる人もよいでしょう。詳しくは「[経理の仕事に向いている人の特徴](#content4)」をご覧ください。

### 経理と簿記の違いはなんですか？

経理と簿記の違いは、仕事を指すかスキルを指すかです。経理は会社の部門や仕事内容を指すのに対し、簿記は会計業務を行うのに必要なスキルを指しています。しかし、経理を行うにあたって簿記は必要不可欠なスキルです。詳しくは記事内「[経理と簿記の違い](#content1-3)」をご覧ください。
