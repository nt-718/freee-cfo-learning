## 粉飾決算とは

**粉飾決算とは、企業が自社の経営成績や財政状態を良く見せるため、故意におこなう虚偽の決算報告です。**たとえば売上の水増し計上、本来であれば来期計上しなければいけない売上の前倒し、子会社を利用した売上操作といった不正な会計処理は、すべて粉飾決算です。
  
  
上場企業であれば監査法人により監査が入るため、粉飾行為は簡単にできない一方、監査が入らない中小企業は会計操作が比較的容易です。**しかし、一度粉飾決算をしてしまうと、つじつまを合わせるために毎期続けなければいけない状態になり、数値の異常からいつかは発覚してしまいます。**

### 粉飾決算の種類

粉飾決算は以下の2種類に分けられます。

粉飾決算の種類

* 粉飾決算：自社の財務状況を良く見せるためにおこなう虚偽の会計報告
* 逆粉飾決算：自社の財務状況を悪く見せるためにおこなう虚偽の会計報告

**粉飾決算の手法には、一般的に収支の水増しや原価を低く操作する方法、複数の企業が共謀しておこなう循環取引があります。**循環取引とは、特定の商品の売買をグループ内で繰り返し、売上や利益を不正計上する方法です。
  
  
逆粉飾決算には、在庫隠しや原価を高く操作する方法、子会社からの架空仕入れ計上、売上の翌期計上といった手法で行われます。

## 粉飾決算はどうして起きるのか

本来であれば絶対におこなってはいけない粉飾決算ですが、起きてしまう2つの理由を解説します。

### 業績を良く見せるため

**粉飾決算が起きる理由のひとつは、業績を良く見せるためです。**
  
  
業績が高いと見せることで会社の信用度を上げ、株価を維持しやすくなるだけでなく、株主に対して配当金も多く出せ、業績悪化についての責任追及も逃れられます。また、業績が良ければ融資も受けやすくなるため、資金繰りも良好になるだけでなく、取引の維持や拡大にもつながるでしょう。

### 脱税のため（逆粉飾）

**粉飾決算が起きるもう一つの理由は、脱税です。**
  
  
**税金は、収入（売上 - 経費）に対してはもちろん、会社が保有する資産に対しても課税されます。**そのため、これらの納税額を抑えるために、売上額や仕入額等を操作したり、保有する資産を少なく計上するのです。

## 粉飾決算がバレたときのペナルティとは

**もしも粉飾決算が発覚すれば、事業の継続が困難になるだけではありません。**
  
  
**粉飾決算は、法律違反に対して国が罰する「刑事罰」や、行政機関が規則違反に対して課す「行政処分」などの対象になります。**また、粉飾決算により損害を受けた株主や取引先などの第三者が、粉飾決算を行った企業や個人に対して求める「損害賠償」への対応も求められます。

### 損害賠償

**粉飾決算が発覚した際に、粉飾決算に起因して損害を受けた第三者から、その保障などを求められるのが損害賠償です。**
  
  
粉飾決算は前述したとおり、自社の財務状況を良く見せるためにおこなう虚偽の会計報告を指します。その粉飾決算による情報を信じて取引をし、損害を受けた個人や法人などの第三者から損害賠償を請求される可能性があります。

|  |  |  |
| --- | --- | --- |
| **損害賠償の請求者** | **損害賠償の請求理由** | **該当する法律** |
| **融資先・取引先** | 債務不履行により | 民法415条 |
| **株主** | 不法行為により | 民法709条 |

**ただし民法709条による損害賠償請求については、被害者側が以下を立証する必要があり、請求が困難な場合もあります。**

被害者が立証しなくてはならないもの

* 違法行為
* 故意・過失の有無
* 損害額
* 違法行為と損害の因果関係

上記の立証が難しく損害賠償請求ができなくなってしまうため、金融商品取引法では、会社役員などに損害賠償請求できるように別途規定が設けられています。
  
  
例として、粉飾決算を知らずに株式を取得した株主は、会社に対し生じた損害賠償を請求できる規定があります。
  
  
損害賠償を請求できる規定は、以下のとおりです。

* 重要な事項について虚偽の記載がある
* 記載すべき重要な事項について記載が欠けている
* 誤解を生じさせないために必要な、重要な事実の記載が欠けている

損害賠償は民法だけでなく、金融商品取引法においても請求される可能性がある点を覚えておきましょう。

  

[出典：e-GOV法令検索「民法415条」](https://elaws.e-gov.go.jp/document?lawid=129AC0000000089#:~:text=%E4%B8%8D%E5%B1%A5%E8%A1%8C%E3%81%AB%E3%82%88%E3%82%8B%E6%90%8D%E5%AE%B3%E8%B3%A0%E5%84%9F%EF%BC%89-,%E7%AC%AC%E5%9B%9B%E7%99%BE%E5%8D%81%E4%BA%94%E6%9D%A1,-%E5%82%B5%E5%8B%99%E8%80%85%E3%81%8C)
  
[出典：e-GOV法令検索「民法709条」](https://elaws.e-gov.go.jp/document?lawid=129AC0000000089#:~:text=%E4%B8%8D%E6%B3%95%E8%A1%8C%E7%82%BA%E3%81%AB%E3%82%88%E3%82%8B%E6%90%8D%E5%AE%B3%E8%B3%A0%E5%84%9F%EF%BC%89-,%E7%AC%AC%E4%B8%83%E7%99%BE%E4%B9%9D%E6%9D%A1,-%E6%95%85%E6%84%8F%E5%8F%88%E3%81%AF%E9%81%8E%E5%A4%B1)

### 刑事罰

**刑法は、粉飾決算自体を罰する条文を有していません。しかし以下のような事由で刑事罰を受ける可能性もあります。**

|  |  |
| --- | --- |
| **粉飾決算により、本来の財務内容であれば受けられないような融資を受けた（詐欺罪）** | 刑法246条 |
| **粉飾決算により、本来であればできなかったはずの剰余金配当をおこなった（違法配当罪）** | 会社法第963条 |
| **届出書類に虚偽の記載をおこなった** | 金融商品取引法197条 |

  

粉飾決算により銀行から融資を受けたことが悪質だと判断されれば、刑事事件とみなされて逮捕される可能性も否定できません。2019年には東証一部上場企業が粉飾決算を疑われ、**逮捕者が出た事例**も存在します。

  

[出典：e-GOV法令検索「刑法246条」](https://elaws.e-gov.go.jp/document?lawid=140AC0000000045#:~:text=%EF%BC%88%E8%A9%90%E6%AC%BA%EF%BC%89-,%E7%AC%AC%E4%BA%8C%E7%99%BE%E5%9B%9B%E5%8D%81%E5%85%AD%E6%9D%A1,-%E4%BA%BA%E3%82%92%E6%AC%BA)

### 行政処分

**刑事罰について前述しましたが、刑事罰を課すためには「故意であった」証拠が必要です。**故意の立証が困難であるとき、金融法では、課徴金を課す行政処分が下されるケースもあります。
  
  
**課徴金制度とは、刑事罰の要件とされている「故意」が要求されず、虚偽記載の要件を満たせば対象となる制度です。**
  
  
課徴金の金額は、金融商品取引法172条4項において、以下のとおり定められています。

|  |  |
| --- | --- |
| **対象** | **課徴金金額** |
| **有価証券報告書・訂正報告書** | 600万円または時価総額の10万分の6のうち、高い方 |
| **四半期・半期・臨時報告書等** | 300万円または時価総額の10万分の3のうち、高い方 |

**なお、過去5年以内に課徴金の対象となった者が再度違反すれば、課徴金は1.5倍です。**また発覚前に自発的に申告すれば、課徴金は2分の1になります。

  

[出典：e-GOV法令検索「金融商品取引法172条」](https://elaws.e-gov.go.jp/document?lawid=323AC0000000025_20240401_505AC0000000079#:~:text=%E9%87%91%E7%B4%8D%E4%BB%98%E5%91%BD%E4%BB%A4%EF%BC%89-,%E7%AC%AC%E7%99%BE%E4%B8%83%E5%8D%81%E4%BA%8C%E6%9D%A1,-%E7%AC%AC%E5%9B%9B%E6%9D%A1%E7%AC%AC%E4%B8%80)

### 加算税

**粉飾決算が発覚すれば、加算税を課せられることがあります。**加算税とは国税の過少申告や無申告、不納付などの問題が起きた際に、本来収めるべき税額に加算して課せられる税金で、区分と税率は以下のとおりです。

|  |  |  |
| --- | --- | --- |
| **どのような場合か** | **区分** | **税率** |
| **過失の場合** | 過少申告加算税 | 最大15％ |
| **故意や悪意がある場合** | 重加算税 | 35％ |

  

**上記のほか、申告していなかった分の法人税に、最大14.6％の延滞税も課せられます。**法人税とは法人の所得に対して課せられる国税で、法人は決算日の翌日から2ヶ月以内に申告し、納付しなければなりません。
  
  
法人税については、別記事「[法人税とは？計算方法から仕組みまでわかりやすく解説](https://www.freee.co.jp/kb/kb-accounting/corporate-tax/)」をご確認ください。

  

[出典：国税庁「延滞税について」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/osirase/9205.htm)
  
[出典：国税庁「加算税の概要」](https://www.mof.go.jp/tax_policy/summary/tins/n04_3.pdf)

### その他のペナルティ

**損害賠償や刑事罰、行政処分、加算税の他、株式市場から以下のような処分が課せられる可能性があります。**

証券取引所からのペナルティ

* 上場廃止処分
* 特設注意市場銘柄への指定
* 改善報告書の提出

上場している企業では上場廃止処分や、「内部管理体制について改善の必要性が高い」と認められる特設注意市場への指定、違反の経過および改善措置を記載した改善報告書の提出が求められることもあります。
  
  
証券取引所が必要と認めれば、違反行為の公表や、上場契約違約金の支払いもあるでしょう。

## 粉飾決算の見抜き方

**粉飾決算を見抜くためには、決算書や各種帳簿の確認が必要です。**決算書を含めた各種帳簿は数年連続して見比べて分析し、違和感があればその箇所から重点的に確認します。

### 売掛金の確認

**粉飾決算を見抜く方法は、売掛金の確認です。**長期間積算されていない売掛金はないか、売掛金や受取手形などの売上債権を現金化できるまでの期間である「売上債権回転期間」が不自然に増加していないか、確認しましょう。
  
  
売掛金を確認し、不自然な点があれば、架空の売上がある可能性も考えられます。また売掛金の取引先が事業内容と合わないときも、注意が必要です。

### 買掛金の確認

**買掛金の確認も、粉飾決算を見抜く方法です。**買掛金や未払金が不自然に減っていれば、経費や仕入れの金額を少なく計上し、利益を多く見せている可能性があります。

### 在庫（原価率・粗利率）の確認

**原価率や粗利率といった在庫の確認も、粉飾決算を見抜く方法です。**棚卸し在庫を実態よりも多くすれば、売上高に対する原価の割合である原価率が下がり、サービスや商品を販売した際の利益である粗利が上がるため、実際の売上よりも利益を多く見せられます。
  
  
不自然に粗利率が上昇していないか、または原価率が急激に低下していないか確認し、粉飾決算をしていないか見抜きましょう。

### 次期繰越利益の確認

**最後に挙げられるのは、次期繰越利益の確認です。**次期繰越利益とは企業の利益のうち、前期に利益として処分されず、来期に繰り越された金額を指します。
  
  
次期繰越利益を確認する際は、正しく引き継がれているか、前期の期末棚卸高と翌年の期首棚卸高が一致しているか確認しましょう。

## 粉飾決算の防止対策

**粉飾決算は、たとえ従業員1人が少額で行ったものでも、一度はじめてしまうとそのしわ寄せとして毎年でる異常値の調整をする必要が出てきます。**最終的に大きな問題に繋がる可能性もあるため、粉飾決算の防止は、従業員など個人レベルのものを含め、社を挙げての対策が必要です。
  
  
ここからは粉飾決算の防止対策として、3つの方法を紹介します。

### 社内教育

**粉飾決算の防止対策として挙げられる1つ目の防止対策は、社内教育です。**「少しくらいなら粉飾決算をしても問題ないだろう」といった考えは周りに伝染し、いずれ大きな問題を起こす可能性があります。
  
  
前述したとおり、粉飾決算は損害賠償だけでなく、刑事罰や行政処分の対象となる行為です。粉飾が発覚すれば企業は社会的信用を失い、倒産のリスクも考えられます。
  
  
粉飾決算の罰則はもちろん、関わった個人や企業に与える影響などを社員に理解させ、「粉飾決算は絶対にしてはいけない」という社内認識を構築しましょう。

### 監視体制の強化

**2つ目の防止対策は、監視体制の強化です。**粉飾された決算書が提出される前に発見できるよう、日々の監視体制を整えましょう。
  
  
業務に伴う経理処理を複数の部署や従業員でクロスチェックを行うことで、不適切な会計処理だけでなく人為的なミスを発見できる可能性が高くなります。また、クロスチェックをはじめとした監視体制を強化するために、属人化している業務がないかどうか確認しましょう。特定の人物に権限が集中しないような仕組みづくりも大切です。

### 資金の流れの可視化

**最後に紹介する粉飾決算の防止対策は、資金の流れの可視化です。**会計ソフトなどを利用して、日々の資金繰りや取引を可視化しましょう。
  
  
会計ソフトは、自動入力や自動仕分けといった多くの機能により、ミスのない効率的な会計処理が可能です。事前に資金不足になるタイミングも把握できるため、会計士や税理士と共有し、対策を練ることもできます。

## まとめ

**粉飾決算とは、会社の財務状況を実際よりも良く見せようとする虚偽の会計報告です。****しかし粉飾決算が発覚した際のペナルティには損害賠償や刑事罰、行政処分などがあります。公表されれば企業の社会的信用も失墜し、最悪の場合は倒産にもなりかねません。**
  
  
取引先が粉飾決算をしていた場合でも、自社に影響を及ぼすことがあるため、定期的に決算書や各種帳簿をよく確認しましょう。
