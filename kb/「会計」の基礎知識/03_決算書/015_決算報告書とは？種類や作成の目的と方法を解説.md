## 決算報告書とは

**決算報告書とは、企業の財務状況・経営成績・キャッシュフローの状況を示す公文書で、法人が事業年度末に作成する書類です。**決算報告書には複数の書類が含まれ、貸借対照表や損益計算書などが該当します。「決算書」ともいいます。
  
  
**決算報告書（または決算書）は、複数の法律で作成が義務付けられており、適用される法律によって「財務諸表」や「計算書類」と呼称が異なります。**
  
  
ここでは、決算報告書の役割や関連する法律、決算報告書の開示義務について解説します。

### 決算報告書がもつ役割

**決算報告書は、事業年度の財政状態と経営成績を期ごとにまとめて、税務署、株主、取引先、金融機関に資産状況を説明する役割を果たします。**
  
  
税務署、株主、取引先、金融機関が決算報告書を見る目的は、それぞれ以下のとおりです。

|  |  |
| --- | --- |
| **対象** | **目的** |
| **税務署** | 決算内容の不備がないか |
| **株主** | 会社が健全に運営されているか |
| **取引先** | 引き続き取引をしても問題はないか |
| **金融機関** | 返済能力のある会社か、融資はどれくらいできるか |

  

決算報告書は、税申告に必要な書類や預貯金、資産や負債の状況をまとめた書類など、複数の書類から構成されます。
  
  
決算報告書類の種類についての詳細は後述する「[決算書の種類](https://www.freee.co.jp/kb/kb-accounting/financial-report/#content3)」をご覧ください。

### 決算報告書に関連する法律

**決算報告書に関連する法律には、法人税法・会社法・金融商品取引法の3つが挙げられます。**それぞれの法律の基本的な概要は以下のとおりです。

|  |  |
| --- | --- |
| **法律** | **概要** |
| **法人税法** | 法人税の納税義務者、課税所得の範囲、税額の算出方法、申告について定めた法律 |
| **会社法** | 会社の設立・運営・清算に関する手続きやルール定めた法律。決算報告書を会社法では「計算書類」という。 |
| **金融商品取引法** | 金融商品の取引における透明性を高め、公正な取引が可能な環境構築が目的の法律。決算報告書を金融商品取引法では「財務諸表」という。 |

### 決算報告書の開示義務

企業は特定の相手に対し、決算報告書を開示する義務があります。決算報告書の開示が必要となる相手は、税務署や上場企業・大企業、株主や債権者（請求があった場合のみ）です。

#### 税務署

すべての企業は事業規模を問わず、税務署への決算報告書の提出が義務とされています。税務署に決算報告書を提出することで、法人税の申告における根拠を税務署に示すことが可能です。
  
  
また、税務調査が入り、調査時に税務調査官から決算報告書の提示を求められた場合には、それに応じなければなりません。

  

[出典：e-Gov法令検索「国税通則法｜第四十六条の二」](https://elaws.e-gov.go.jp/document?lawid=337AC0000000066)

#### 上場企業・大企業

**上場企業の場合、金融商品取引法による「有価証券報告書」の開示義務があります。**また、最終事業年度における負債合計額が200億円以上か、資本金が5億円以上の大企業（会社法上）は、年次決算報告書の提出・公告が必要です。

  

[出典：e-Gov法令検索「金融商品取引法｜第二十四条」](https://elaws.e-gov.go.jp/document?lawid=323AC0000000025_20231129_505AC0000000079)

#### 株主や債権者

**議決権の3%を有する株主から請求があった場合は、企業は株主に対して決算報告書を開示しなければなりません。**株主には企業の財政状況や経営状況など、株式における価値の評価や投資における判断に必要な情報を求める権利があることが理由です。
  
  
債権者の場合も同様で、請求があった場合は企業は決算報告書を開示する必要があります。企業の財務状況・健全性を把握し、債権における価値の評価、返済能力を判断するためです。

  

[出典：e-Gov法令検索「会社法｜第四百三十三条」](https://elaws.e-gov.go.jp/document?lawid=417AC0000000086)

## 決算報告書はいつ必要になるのか

決算報告書は、法人税の確定申告や株主総会の際に必要となります。法人の確定申告では、決算報告書を確定申告書とあわせて提出しなければなりません。そのため、**法人税の申告・納付期限である事業年度終了日の翌日より2ヶ月以内までに、決算報告書の作成および、税務署への提出を行います。**
  
  
また、株主総会の際も、株主に対して決算報告を行う場合は決算報告書を用意しなければなりません。株主総会は会社法に基づき、事業年度が終了した翌日より3ヶ月以内に召集する必要があります。そのため、株主総会開始前までに、決算報告書を用意しておきましょう。

  

[出典：国税庁「C1-1 法人税及び地方法人税の申告（法人税申告書別表等）」](https://www.nta.go.jp/taxes/tetsuzuki/shinsei/annai/hojin/shinkoku/01.htm)
  
[出典：e-Gov法令検索「会社法｜第二百九十六条」](https://elaws.e-gov.go.jp/document?lawid=417AC0000000086)

## 決算報告書の種類

決算報告書は、法人税法、会社法、金融商品取引法それぞれで必要書類が異なります。

|  |  |
| --- | --- |
| **法律** | **必要書類** |
| **法人税法** | ・貸借対照表  ・損益計算書  ・株主資本等変動計算書  ・勘定科目内訳書 |
| **会社法** | ・貸借対照表  ・損益計算書  ・株主資本等変動計算書  ・個別注記表 |
| **金融商品取引法** | ・貸借対照表  ・損益計算書  ・株主資本等変動計算書  ・キャッシュ・フロー計算書  ・附属明細書 |

### 貸借対照表

**「B/S（Balance sheet）」とも呼ばれる貸借対照表とは、事業年度末の資産・負債・純資産が記載された書類のことです。**企業における資金調達や運用状況を把握するために活用されます。また、開業後の累積的な企業活動を示した書類として活用されることもあります。
  
  
貸借対照表についての詳細は別記事「[貸借対照表とは？ 会社の財務状況を簡単に把握！](https://www.freee.co.jp/kb/kb-accounting/balance-sheet/)」をあわせてご確認ください。

### 損益計算書

**「P/L（Profit and Loss statement）」とも呼ばれる損益計算書とは、企業における一定期間の収益・損益計算をまとめた書類で、企業の経営状況を把握できます。**損益計算書は、収益・費用・利益の3つから構成されます。
  
  
貸借対照表が事業年度末という「特定の時点」における経営成績を示すのに対し、損益計算書は「ある一定期間」の経営期間を示すのが特徴です。
  
  
損益計算書についての詳細は別記事「[損益計算書とは？ 項目別の見方やチェックポイント、活用法を解説](https://www.freee.co.jp/kb/kb-accounting/pl/)」をあわせてご確認ください。

### キャッシュ・フロー計算書

**キャッシュ・フロー計算書とは、企業が所有する現金の動きを示す書類です。**貸借対照表・損益計算書とキャッシュ・フロー計算書をまとめて「財務三表」といいます。
  
  
営業活動・投資活動・債務活動の3つの活動でのお金の動きを記し、正確に把握するための書類として活用されます。
  
  
キャッシュフロー計算書についての詳細は別記事「[キャッシュ・フロー計算書とは？計算方法や見方、作り方のポイントを解説](https://www.freee.co.jp/kb/kb-accounting/cash_flow_statement/)」をあわせてご確認ください。

### 勘定科目内訳書

**勘定科目内訳書とは、貸借対照表・損益計算書における勘定科目の内訳を示す書類のことです。**法人税法施行規則第35条により、決算日の翌日から2ヶ月以内に、税務署への提出が義務づけられています。
  
  
税務署は勘定科目内訳書から、申告書類が正しく作成されているか、不自然な内容の取引がないかを確認します。
  
  
勘定科目内訳書についての詳細は別記事「[勘定科目内訳明細書を作成する際の注意点](https://www.freee.co.jp/kb/kb-accounting/important-point-of-statement/#content1)」をあわせてご確認ください。

  

[出典：国税庁「C1-1 法人税及び地方法人税の申告（法人税申告書別表等）」](https://www.nta.go.jp/taxes/tetsuzuki/shinsei/annai/hojin/shinkoku/01.htm)

### 株主資本等変動計算書

**株主資本等変動計算書とは、貸借対照表における純資産のうち、株主に帰属する項目が変動した際にその理由を記載する書類です。**
  
  
株主資本が変動した理由を開示するために、重要な書類として扱われます。株主資本等変動計算書に記載される純資産は、株主資本、評価・換算差額、新株予約権、少数株主持分の4つに分類されます。

### 個別注記表

**個別注記表とは、貸借対照表・損益計算書・株主資本等変動計算書の各計算書類に記載した注記を、1枚にまとめた計算書類のことです。**企業の財産状況や、損益の状態を正確に把握するために必要です。

### 計算書類の附属明細書

**附属明細書とは、決算書の内容を補足する重要な事項などが記載された書類のことです。**固定資産・引当金・販売費・管理費の明細などを記載します。
  
  
附属明細書は、事業年度ごとに、貸借対照表・損益計算書・株主資本等変動計算書・個別注記表とあわせて作成・提出しなければなりません。

## 決算報告書の作成手順

決算報告書は決算後すぐに作成できるわけではなく、事前準備を含めた以下の手順を踏むことが必要です。

決算報告書の作成手順

1. 決算の対象となる記帳を完了させる
2. 総勘定元帳への転記および試算表の作成
3. 決算書の作成

決算報告書の作成手順についての詳細は別記事「[決算書の作り方を解説！作成手順や必要書類とは？](https://www.freee.co.jp/kb/kb-accounting/dos-for-financial-statements/#content3-1)」をあわせてご確認ください。

### 1．決算の対象となる記帳を完了させる

まずは、今期の決算報告書のベースになる仕訳帳へ記帳を完了させます。記帳が完了したら、事業年度の取引がわかる領収書や請求書、通帳のコピーなどを整理し、実際の残高と照合して抜け漏れがないかチェックしてください。
  
  
残高との照合までを終えたら、**決算整理事項を確認のうえ決算整理仕訳を行います。**決算整理事項には、事業年度をまたぐ取引の仕訳や、在庫の点検による棚卸資産の残高、固定資産の減価償却が該当します。
  
  
これらの確認が完了したら、記帳内容の修正・追加を行ってください。その後、実際の残高と一致するか確認します。

### 2．総勘定元帳への転記および試算表の作成

続いて、仕訳帳に記帳した各勘定項目の内容を総勘定元帳に転記します。
  
  
**総勘定元帳への転記を終えたら、その内容をベースに記帳の整合性を確認できる決算整理前試算表を作成しましょう。**なお、このときの作業を手作業で進めてしまうとミスが発生しやすいため、会計ソフトなどの使用がおすすめです。

### 3．決算書の作成

総勘定元帳への転記、試算表の作成までを終えたら、損益計算書、貸借対照表、キャッシュ・フロー計算書、個別注記表、勘定科目内訳書などの決算報告書を作成します。
  
  
なお、決算報告書の作成は専門知識が必要になるため、会計ソフトを利用したり、税理士などに依頼したりするのが一般的です。
  
  
決算報告書を1人で作成する場合についての詳細は別記事「[法人決算を自分1人で完結させるには？流れや必要書類について解説](https://www.freee.co.jp/kb/kb-accounting/how-to-prepare-financial-statements-alone/)」をあわせてご確認ください。

## 決算報告書の作成期限

法人の確定申告は、申告書類に決算報告書を添付して申告を行います。そのため、前述のとおり、**確定申告の期限である事業年度終了後の2ヶ月以内に、あわせて決算報告書も作成を間に合わせる必要があります。**
  
  
なお例外もあり「申告期限の延長の特例」の手続きを行い、条件を満たしていれば、法人税の申告期限を延長できます。

  

[出典：国税庁「確定申告書の提出期限」](https://www.nta.go.jp/law/joho-zeikaishaku/hojin/group_faq/19.htm)
  
[出典：国税庁「C1-17　定款の定め等による申告期限の延長の特例の申請」](https://www.nta.go.jp/taxes/tetsuzuki/shinsei/annai/hojin/annai/1554_12.htm)

決算に向けて、日々の記帳時間を少しでも短縮して、経理業務を楽にしたいですよね。
  
  
[シェアNo.1のクラウド会計ソフト\*1「freee会計」](https://www.freee.co.jp/accounting/)では、面倒な入力作業や仕訳を自動化し、経理業務にかかる時間を半分以下\*2に削減。ボタンクリックひとつで貸借対照表・損益計算書などの決算書が作成可能です。

※2 自社調べ。回答数1097法人。業務時間が1/2以上削減された法人数
  
  
また、「[freee申告](https://www.freee.co.jp/tax-filing/)」を併用すれば、法人で必要な申告書の作成から電子申請までを一気通貫で完結できます。
  
  
数ある会計ソフトの中でも、[freee会計](https://www.freee.co.jp/accounting/small-medium/)が選ばれる理由は大きく3つ。

  

* 一度の入力で複数の業務が完了。重複作業や転記作業はほぼ発生なし！
* 決算業務は正しく、確実に対応できる！
* 国内で唯一、法人税申告書の作成まで一気通貫で行える！

  

それぞれの特徴についてご紹介していきます。

### 一度の入力で複数の業務が完了。重複作業や転記作業はほぼ発生なし！

見積書・請求書を[freee会計](https://www.freee.co.jp/accounting/)で発行すると、書類へ入力した金額をもとに、自動で入金管理・売上仕訳まで完了。銀行口座やクレジットカード、POSレジなどと同期すれば、自動で利用明細を取り込み、勘定科目の登録はもちろん、売掛金や買掛金の消し込み、入金仕訳などの記帳も簡単に行えます。
  
  
さらに、領収書・受取請求書などをスマホのカメラで撮影し[freee会計](https://www.freee.co.jp/accounting/small-medium/)に取り込むだけで、取引先名や金額などをAI解析し、自動で入力。支払管理・仕訳も自動で作成できます。
  
  
[freee会計](https://www.freee.co.jp/accounting/)は一度の入力で複数の業務が完了するうえ、自動入力・自動仕訳によって手作業の少ない経理を実現します。

### 決算業務は正しく、確実に対応できる！

[freee会計](https://www.freee.co.jp/accounting/)には、正しい決算書を作るためのチェック機能も充実。預金残高との一致や会計ルールとの整合性をfreeeが自動判定し、修正が必要そうなリストを自動作成します。修正後は、ボタンクリックひとつで貸借対照表・損益計算書などの決算書が作成可能です。
  
  
＜作成可能な書類例＞  

* 貸借対照表・損益計算書
* 仕訳帳・総勘定元帳
* 固定資産台帳
* 試算表
* 現金出納帳　など

  

PDFやCSVファイルへの出力も可能なため、士業の方への共有や、社内での資料作成にも活用できます。また、領収書1枚・仕訳1件単位でコメント機能を使ってやりとりできるため、士業の方ともスムーズにコミュニケーションがとれます。

### 国内で唯一、法人税申告書の作成まで一気通貫で行える！

[freee申告](https://www.freee.co.jp/tax-filing/)を併用すれば、[freee会計](https://www.freee.co.jp/accounting/)のデータと自動連携して、法人税の申告書の書類選択や税額計算、入力作業のほとんどを自動化。申告書類作成の時間削減や転記ミスを防ぐことができます。さらに、e-Taxなどで事前準備を済ませておけば、freee申告上から電子申告まで一気に完結させることが可能です。

## まとめ

事業年度後に行った決算の結果をまとめた書類を、決算報告書といいます。決算報告書は利害関係者に資産状況を説明する役割があり、確定申告や株主総会の際に必要です。法人税法、会社法、金融商品取引法と、3つの法律ごとに必要書類が異なり、かつ提出先が異なることも覚えておかなければなりません。
  
  
基本的な作成手順・作成期限とあわせて、決算報告書に関する理解を深めましょう。

## よくある質問

### 決算報告書にはどのようなものがある？

会社法・法人税法・金融商品取引法それぞれで、貸借対照表・損益計算書・株主資本等変動計算書・キャッシュフロー計算書・附属明細書が必要になります。
  
  
詳しくは記事内「[決算報告書の種類](#content3)」をご覧ください。

### 決算報告書はいつまでに提出すべき？

確定申告書の提出期限である「事業年度終了後の2ヶ月以内」にあわせて作成・提出しなければなりません。
  
  
詳しくは記事内「[決算報告書の作成期限](#content6)」をご覧ください。
