## 損益分岐点とは

損益分岐点とは、**売上高と費用が等しくなり損益がプラスマイナスゼロとなる売上高**のことです。
  
  
つまり、売上高が損益分岐点を超えてくれば利益がでることになり、損益分岐点の満たないと利益がでるどころか損失をだすことになります。
  
  
売上高が損益分岐点未満であれば損失が生じることになり、損益分岐点を超えることで利益が生じることから「採算点」ともいわれています。

## 損益分岐点の計算式

損益分岐点の売上高を算出するには以下の計算式を使います。

**損益分岐点＝固定費÷｛1－（変動費÷売上高）｝**

費用は固定費と変動費に分けることができます。

#### 固定費

売上高の変動に関わらずかかる費用となります。例えば事業所の家賃や水道光熱費、また人件費や広告宣伝費、機械設備の費用、また減価償却費などがあたります。

#### 変動費

売上の増減によって変動する費用のことです。商品の仕入や、外注費、また歩合給などがあります。一般的には売上原価として、売上高が増加するごとに変動費は増加することになります。
  
  
売上高から変動費を差し引いた金額は**「限界利益」**といいます。限界利益の額で、全ての固定費を回収できるか測ることができます。
  
  
また売上高の占める限界利益の割合を**「限界利益率」**といいます。
  
  
・**限界利益＝売上高ー変動費**
  
・**限界利益率＝限界利益÷売上高**
  
  
この限界利益率を用いて、損益分岐点を求めることもできます。

**損益分岐点＝固定費÷限界利益率**

損益分岐点を算出する前に、限界利益率をしっかりと確認する必要があります。そのうえで、損益分岐点を下げていくために、変動費を下げたり、固定費も見直していく必要があります。
  
  
業種別で固定費と変動費を細かく分解する方法については、[中小企業庁の費用分解基準](https://www.chusho.meti.go.jp/bcp/contents/level_c/bcpgl_05c_4_3.html)をご覧ください。

## 損益分岐点を計算する際の注意点

損益分岐点の計算をする場合、ある時点での計算を行うことが基本となります。常に変わらないビジネスであればその後に大きな費用変動はなく、ある時点の計算結果を参考にしていれば問題ありません。
  
  
しかし例えば、為替変動によって変動費である仕入れ値の単価が変わったり、生産量が大幅に増えたことで製造設備を買い足すなど固定費と設定していた費用が増える場合もあるでしょう。
  
  
最初に設定した変動費と固定費に変化があった場合は、そのたびに損益分岐点が変わってしまうため、赤字ビジネスにならないよう、変化がある際は数字変動をきちんと確認し、都度計算しましょう。

## エクセルで損益分岐点を計算する

会社の損益分岐点をエクセルで計算してみます。ますは会社における1年間の業績予想を元に算出してみます。以下の金額は暫定的にしています。

【収益】

|  |  |
| --- | --- |
| 売上高 | 36,000,000 |

【費用】

|  |  |
| --- | --- |
| 仕入 | 19,800,00 |
| 役員報酬 | 6,000,000 |
| 給料 | 9,969,000 |
| 交通費 | 348,000 |
| 通信費 | 268,000 |
| 交際費 | 185,000 |
| 減価償却費 | 548,000 |
| 賃借料 | 800,000 |
| 保険料 | 148000 |
| 修繕費 | 228,000 |
| 光熱費 | 235,000 |
| 消耗品費 | 228,000 |
| 雑費 | 98,000 |
| 支払利息 | 225,000 |
| **費用合計** | **39,080,000** |
| 変動費 | 19,800,000 |
| 固定費 | 19,280,000 |

便宜的に費用は仕入が変動費用として、その他は全て固定費とします。
  
  
上記の業績では、
  
**・費用合計39,080,000円  
・変動費19,800,000円  
・変動費19,280,000円**  
となります。
  
  
費用合計に対して売上高が少ないため、前期の1年間の業績は赤字となっています。これを黒字転換するために、損益分岐点を把握しておかなければなりません。

**損益分岐点＝固定費÷限界利益率**

となるため、まずは限界利益率を求めていきます。

|  |  |
| --- | --- |
| 売上高 | 36,000,000 |
| 変動費 | 19,800,000 |
| 限界利益 | 16,200,000 |
| 限界利益率 | 0.45 |

**損益分岐点＝固定費19,280,000÷限界利益率0.45**

  

|  |  |
| --- | --- |
| 固定費 | 19,280,000 |
| 限界利益率 | 0.45 |
| 損益分岐点 | 42,844,444 |

上記の計算では**損益分岐点は42,844,444円**となります。
  
  
前期の業績で固定費が変わらず、変動費率、つまり限界利益率が変わらないという条件がつきますが、損益分岐点が確認することができます。

## エクセルで損益分岐点のグラフを作る

上記のグラフは右上に向かう点線が変動費となり、そのうえの青色の直線が変動費に固定費を加算したものです。
  
  
左下から損益分岐点を超え一番右上まで伸びている赤色の直線が売上高となります。一定の売上高になるまでは、利益ではなく損失が発生している状態となります。
  
  
損益分岐点で利益損失はプラスマイナスゼロとなり、損益分岐点を超えることで利益が発生します。

限界利益は売上から変動費を差し引いた金額となります。
  
  
限界利益は売上があればプラスとなりますが、損益分岐点まで売上がいかないと利益はでずに赤字となってしまいます。また損益分岐点を超えてくると、限界利益の増加分は全て利益となります。
  
  
企業の営業活動で必要な費用は変動費と固定費に分けることができるとされていますが、人件費であっても必ず固定費になるというわけではありません。
  
  
基本的に従業員の給与は固定費となりますが、売上が増加することで既存の従業員では業務がまわらず、臨時のパートやアルバイトを雇う場合もあります。そのため、変動費として仕入や一部の販売管理費だけでなく、固定費とも変動費ともどちらとも判断しがたい費用が発生します。
  
  
また営業社員で歩合給をつけている場合には、従業員の給与であっても固定費ではなく変動費扱いとなるのが一般的です。

## まとめ

エクセルで損益分岐点を計算することはそれほど難しいことではありません。固定費と変動費の金額が確認できれば、損益分岐点をだすことができます。ただし、損益分岐点を算出することは目的ではありません。  
会社が赤字であれば、売上高を増やし変動費と固定費を下げて黒字転換にする、黒字であればどの程度の余裕があるのか見極めることが大切です。
