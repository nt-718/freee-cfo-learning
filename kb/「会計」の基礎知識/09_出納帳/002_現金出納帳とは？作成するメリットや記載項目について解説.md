# 現金出納帳とは？作成するメリットや記載項目について解説

出典: https://www.freee.co.jp/kb/kb-accounting/basic-knowledge-of-cashbook/

---

事業を運営するうえで、現金の管理は欠かせません。日々の現金の出入りを正確に把握できていないと、資金繰りの悪化や会計処理のミスにつながる可能性があります。
  
  
そこで活用すべきなのが、現金出納帳（げんきんすいとうちょう）です。現金出納帳とは、現金の入出金を記録し、管理するための帳簿であり、正しく運用することで経営の透明性を高め、確定申告や決算時の手間を減らすことができます。
  
  
本記事では、現金出納帳の基本的な役割や記入方法、管理のポイントを詳しく解説します。経理業務を税理士など外部に委託している場合であっても、現金出納帳は自身で作成をするよう求められるケースが多いため、書き方や注意点を理解しておきましょう。

## 現金出納帳とは

**現金出納帳（げんきんすいとうちょう）とは、事業での現金の入出金を記録し、管理するための帳簿です。**「いつ・どこで・何のために」現金が動いたのかを明確に把握でき、経理のミスを防いだり、確定申告や決算の準備をスムーズに進めたりするのに役立ちます。
  
  
**会計帳簿は、大きく「主要簿」と主要簿を補うための「補助簿」に分けられ、現金出納帳は補助簿にあたります。**

  
  
補助簿の作成は必須ではなく、事業モデルや商習慣などに応じて作成すればよいとされています。しかし、**現金取引があるにもかかわらず現金出納帳を作成していないと、現金の流れを正確に把握できません。**キャッシュ・フローの悪化やミス、不正などにつながる可能性があるため、現金の流れを把握するためにも現金出納帳を作成しましょう。
  
  

[出典：国税庁「帳簿の記帳のしかた-事業所得者用-」](https://www.nta.go.jp/taxes/shiraberu/shinkoku/kojin_jigyo/kichou03.pdf)

### 現金出納帳と小口現金出納帳の違い

現金出納帳と似た帳簿に、小口現金出納帳（こぐちげんきんすいとうちょう）があります。
  
  
どちらも現金の入出金を記録する点では同じです。ただし、**現金出納帳が事業全体の現金の入出金を記録するのに対し、小口現金出納帳は事務所や店舗に置いてある少額の現金を記録します。**
  
  
一般的に、企業では通常の現金出納帳とは別に、小口経費（たとえば消耗品の購入費、交通費・出張費など）を管理するために「小口現金」と呼ばれる一定額の現金を用意します。小口現金出納帳は、こうした小口現金の入出金を管理するものです。
  
  
事業全体の現金の動きを管理するためには現金出納帳を活用しますが、**小口経費（少額の支出）を管理するための帳簿として小口現金出納帳を活用することで、細かい経費の管理や現金出納帳との照合などに役立ちます。**

## 現金出納帳を作成するメリット

**現金の入出金を把握することは、安定的な事業運営に欠かせません。**お金の流れを正確に把握できれば、経理業務の円滑化やキャッシュ・フローの改善などにもつながります。
  
  
現金出納帳を作成する主なメリットは以下の2つです。

現金出納帳を作成するメリット

* お金の流れを可視化できる
* 不正や誤記入を防止できる

### お金の流れを可視化できる

**現金出納帳を作成することで、どのような現金の入出金があったのかを可視化できます。**収入と支出の具体的な内容や金額を時系列で記載するため、過去の取引履歴を確認する際にも役立ちます。
  
  
お金の流れの可視化により、現金の過不足を防ぎ適正な管理を行うことにつながります。すると、**どの項目で現金を多く使っているかなども把握でき、経費削減のヒントにできるなどのメリットも生まれます。**

### 不正を防止できる

**現金の管理が曖昧な状態だと、不正などのトラブルが起きるリスクが高まります。**
  
  
現金出納帳を作成し、日常的に現金残高との一致を確認する習慣をつけることで、**経理の透明性が向上し、現金の持ち出しなどの不正に気付きやすくなります。**

## 現金出納帳の記載項目

現金出納帳に記載すべき基本項目は、以下の5つです。

現金出納帳に記載する6項目

* 日付
* 勘定科目
* 摘要
* 収入金額／支出金額
* 差引残高

  

[出典：国税庁「帳簿の記帳のしかた-事業所得者用-」](https://www.nta.go.jp/taxes/shiraberu/shinkoku/kojin_jigyo/kichou03.pdf)

上記のほか、「合計金額」「繰越金額」を記載する企業もあります。**記載項目は、企業のルールや商慣習に従い決定しましょう。**

  
  
それぞれの項目を以下で解説します。

### ①日付

お金の入金もしくは支払いがあった日付を記載する項目です。
  
  
同じ日に収入や支出があった場合は、それぞれ別の欄に記載しましょう。**領収書の立替精算などは領収書の日付を記載するのではなく、実際に支払われた精算日を記載します。**

### ②勘定科目

勘定科目とは、取引の内容をわかりやすく分類する簿記の科目のことで、以下はその一例です。

  

| 勘定科目 | 概要 |
| --- | --- |
| 仕入 | 販売した商品の仕入れ高や製造原価 |
| 減価償却費 | 法定耐用年数に応じて費用として計上する償却資産額 |
| 地代家賃 | 事務所や店舗の賃借料やリース代 |
| 水道光熱費 | 電気代、ガス代、水道代 |
| 通信費 | 郵便代、電話代、インターネット回線料、運送料 |
| 福利厚生費 | 備品、社員旅行などの費用 |
| 旅費交通費 | 電車、バス、タクシーなどの交通費 |
| 接待交際費 | 取引先などとの飲食代、贈答品代、慶弔費 |
| 広告宣伝費 | 新聞や雑誌などへの出稿費用 |
| 損害保険料 | 火災保険料、自動車保険料など |
| 修繕費 | 建物や機械、備品などの管理修繕費用 |
| 消耗品費 | 事務用品など耐用年数が1年未満で、価格が10万円未満のもの |
| 荷造運賃費 | 荷造りに必要な費用 |
| 租税公課 | 事業関係で支払う税金 |
| 給料手当 | 従業員へ支給される給与や賞与、各種手当 |
| 専従者給与 | 経営者と生計を同じくする家族に対する給与 |
| 外注工賃 | 外部業者へ支払う発注費用 |
| 利子割引料 | 借入金の利子や手形の割引料など |
| 雑費 | いずれの費用にも当てはまらない費用 |
| 仮払金 | 内容や金額が未確定で一時的に支払われたお金。確定した時点で該当する科目への振り替えを行う |
| 立替金 | 取引先や従業員などに対して一時的に支払いを立て替えたお金 |

### ③ 摘要（てきよう）

**摘要は、取引内容の詳細を記載する項目です。**具体的に「何をいくつ購入したか」「どこからどこまで移動したか」「どの銀行に入金したか」などを記載します。

### ④収入・支出の金額（入・出金額）

収入・支出の金額は、取引の金額を記載する項目です。**現金が増えた場合は収入欄に、現金を支払った場合は支出欄にそれぞれ税込の金額を記載します。**

### ⑤差引残高（現金残高）

差引残高は、取引後の残高を記載する項目です。日次の残高と月次・年次の残高はそれぞれ以下の計算式で算出可能です。

日次と月次・年次の残高計算方法

**【日次の残高】**
  
前日の差引残高 ＋ 当日の収入金額 － 当日の支出金額
  
  
**【月次・年次の残高】**
  
前月（前年）の繰越金 ＋（月または年度）収入合計 －（月または年度の）支出合計

**月次や年次の残高計算時には、月（年）の収入・支出額と、月（年）の繰越金額をそれぞれ計算し、月間（年間）の記載内容に誤りがないかを確認します。**

## 現金出納帳の書き方・手順

現金出納帳は、現金の入出金を取引順に記載して作成します。記載する内容は現金取引に限られるため、基本的な書き方を把握していればそれほど複雑ではありません。
  
  
以下の2つのケースを例に、現金出納帳の具体的な書き方・手順を解説します。

【ケース①】
  
1月9日に、品川から目黒までのタクシー代1,000円を現金で支払った
  
  
【ケース②】
  
1月20日に、A社から売掛金5万円を現金で回収した

経費を現金で支払った際は、現金出納帳と同時に経費帳への記載が必要です。また、**売掛金を回収して現金を受け取った場合は売掛帳、買掛金を現金で支払った場合は買掛帳にも記載します。**
  
  
現金出納帳は取引が発生した際に必ず記載を行い、当日中に実際の現金残高と帳簿残高が一致しているかどうかを確認しましょう。

### 【書き方の例／ケース①】タクシー代を支払った場合

タクシー代（品川から目黒まで）1,000円を現金で支払った際は、日付の欄に出金があった日付を記載します。今回のケースでは「1月9日」と記載しますが、上段に1月の取引が記載されている場合、「1月」は省略して問題ありません。

  

| 令和7年 | | 科目 | 摘要 | 収入（円） | 支出（円） | 残高（円） |
| --- | --- | --- | --- | --- | --- | --- |
| 1 | ✕ | ✕✕ | ✕✕ |  | ✕✕ | 32,400 |
|  | 9 | 旅費交通費 | 品川→目黒　タクシー代 |  | 1,000 | 31,400 |

  

勘定科目の欄には、「旅費交通費」と記載します。なお、仕訳は以下の通りです。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 勘定科目 | 金額（円） | 勘定科目 | 金額（円） |
| 旅費交通費 | 1,000 | 現金 | 1,000 |

  

摘要欄には、「品川→目黒 タクシー代」などと記載します。摘要欄の書き方にルールはありませんが、見返した際に取引内容がわかるよう記載するのがポイントです。
  
  
タクシー代の支払いは出金取引であるため、出金欄に「1,000」と記載します。最後に、直前の残高に入出金額を加算または減算した金額を残高の欄に記載しましょう。
  
  
上記のケースでは、3万2,400円から1,000円を差し引いた3万1,400円が残高です。

### 【書き方の例／ケース②】売掛金を回収した場合

A社から売掛金5万円を現金で回収した日を日付の欄に記載します。今回のケースでは、「1月20日（または20日）」です。

  

| 令和7年 | | 科目 | 摘要 | 収入（円） | 支出（円） | 残高（円） |
| --- | --- | --- | --- | --- | --- | --- |
| 1 | ✕ | ✕✕ | ✕✕ |  | ✕✕ | 31,400 |
|  | 20 | 売掛金 | A社 | 50,000 |  | 81,400 |

  

勘定科目の欄には「売掛金」と記載します。仕訳は以下の通りです。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 勘定科目 | 金額（円） | 勘定科目 | 金額（円） |
| 現金 | 50,000 | 売掛金 | 50,000 |

  

摘要欄には、取引内容がわかるように「A社」などと記載しましょう。売掛金の回収は入金取引であるため、入金欄に「5,000円」と記載します。
  
  
**最後に、直前の残高に入出金額を加算または減算した金額を残高欄に記載します。**今回のケースでは、3万1,400円に5万円を加算した8万1,400円が残高です。

## 現金出納帳の書き方のルール

現金出納帳には、定められた様式はありません。しかし、一般的な書き方のルールがいくつかあります。

### 月末や期末で締める場合

**月や期が変わるタイミングでは、ページに余っている行があっても必ず新しいページに移動します。**
  
  
月末になったら最後の行の下に二重線を引き、その下の行の摘要欄に「〇月合計」と記載して、収入と支出それぞれの合計金額を書きましょう。
  
  
また、合計金額を記載した下の行の摘要欄に「次月繰越」と記載して、繰り越す残高を支出欄に書きます。

### 次のページに移動する場合

1ページを使い切ってページを移動する場合は、最初の行の摘要欄に「前頁より繰越」と記載し、次の行から入出金の記録をしていきます。
  
  
新しい月や期に変わるタイミングでページを移動する場合は同様に「期首繰越」「前月繰越」と記載します。

## 現金出納帳の残高が合わないときの処理方法

現金出納帳の残高と実際の現金残高が合わない、または現金出納帳の残高がマイナスになる場合は、以下を確認し原因を明らかにしましょう。

現金出納帳の残高が合わないときに確認するポイント

* 期首残高の記載漏れがないか
* 入出金の金額に誤りがないか
* 入出金の記載漏れがないか

**どうしても過不足金の原因がわからない場合は、「現金過不足」の勘定項目で実際の現金残高にあわせて調整し、後日原因が判明したら取り消しを行います。**期末の決算までに原因が判明しない場合は、「雑損失」もしくは「雑収入」として処理します。

  

| 勘定科目 | 過不足の状況 |
| --- | --- |
| 雑損失 | 現金が不足している場合  （実際の現金 ＜ 現金出納帳の残高） |
| 雑収入 | 現金が余っている場合  （実際の現金 ＞ 現金出納帳の残高） |

  

**1円でも過不足金があったならば、決して経理担当者自身でその金額を抜いたり、補填したりしてはいけません。**現金出納帳の現金過不足は、業務の過程で何らかのミスがあったことを示します。次に同じミスが起きないよう、業務改善をしていくことが大切です。

## 現金出納帳を作成する方法

現金出納帳は、基本的な記載項目やルールさえ押さえておけば簡単に作成できます。ご自身に合った方法で、効率的かつ正確に現金出納帳を作成しましょう。
  
  
現金出納帳を作成する主な方法は、以下の通りです。

現金出納帳を作成する主な方法

* ノートに手書きする
* 会計ソフトで作成する
* 表計算ソフト（エクセル、Googleスプレッドシートなど）で作成する

ノートに手書きする方法は、記載・計算・訂正を手作業で行うため、手間がかかるだけでなく、ミスにもつながりやすくなります。パソコンなどのデジタルデバイスを使い慣れていなくとも作成できるのが特徴ですが、**現金取引が多い場合は会計ソフトや表計算ソフトで作成する方法を検討しましょう。**
  
  
会計ソフトの中には、日々の取引を入力すれば現金出納帳などの帳簿を自動的に作成してくれるものもあります。また、エクセルの無料テンプレートを使用するのも手段のひとつです。
  
  
現金出納帳をエクセルで作りたい方は、別記事「[現金出納帳をエクセルで作成する方法（無料テンプレート付き）](https://www.freee.co.jp/kb/kb-accounting/cashbook_excel/)」をご覧ください。

## まとめ

現金出納帳（げんきんすいとうちょう）とは、事業で「いつ・どこで・何のために」現金が動いたのかを明確に把握でき、経理のミスを防いだり、確定申告や決算の準備をスムーズに進めたりするのに役立ちます。
  
  
会計帳簿の「補助簿」に該当するため、作成が義務付けられているわけではありません。ただし、現金取引がほとんど発生しない場合も、お金の流れを把握しておくことは重要です。お金の流れを正確に把握していないと、キャッシュ・フローが悪化したり、不正が起きたりするリスクが高まります。
  
  
クラウド会計ソフトを用いて銀行口座の入出金データを同期しておくと、企業全体のお金の流れをリアルタイムに把握できます。日々の会計業務を効率化するためにも、クラウド会計ソフトの活用を検討しましょう。

## よくある質問

### 現金出納帳を作成するメリットは？

現金出納帳を作成するメリットは、「お金の流れを可視化できる」「不正を防止できる」の2点が挙げられます。お金の流れを正確に把握できれば、経理業務の円滑化やキャッシュ・フローの改善などにもつながります。
  
  
詳しくは、記事内「[現金出納帳を作成するメリット](#content2)」をご覧ください。

### 現金出納帳を手書きする際の書き方は？

現金取引が発生した順に、「日付」「勘定科目」「摘要」「収入金額／支出金額」「差引残高」などの項目を記載します。現金出納帳を手書きする場合、記載・計算・訂正を手作業で行わなければなりません。
  
  
詳しくは、記事内「[現金出納帳の記載項目](#content3)」をご覧ください。
