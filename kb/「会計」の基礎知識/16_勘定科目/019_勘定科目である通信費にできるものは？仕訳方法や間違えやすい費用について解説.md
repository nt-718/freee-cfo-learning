通信費とは、業務上使用した電話やインターネット、郵便などを経費計上するために使用される勘定科目です。通信費には、個人事業主などが私用と併用して使っているスマートフォンや自宅のインターネット利用料なども、按分して経費計上できます。
  
  
ただし、電話機やPCの購入費用や機器のレンタル費用などは通信費に該当せず、別の勘定科目を使用する必要があるため、間違えないように注意が必要です。
  
  
本記事では、通信費に該当する各費用や仕訳方法、間違えやすい費用などについて詳しく解説します。

## 通信費とは

**通信費とは、業務上使用した電話やインターネット、郵便などを経費計上するために使用される勘定科目です。**通信費は業務上必要不可欠なものであるため、仕訳で使用する頻度の高い勘定科目といえます。
  
  
通信費には、インターネット料金やプロバイダ料金などさまざまな費用が該当し、個人事業主が業務用と私用で併用している費用も、按分することで経費計上可能です。

## 通信費として仕訳できる主な経費の種類

通信費として仕訳できる主な費用は、以下のようなものがあります。

通信費として仕訳できる主な経費

* インターネット料金
* プロバイダ料金
* 固定電話・携帯電話の通話料や通信料
* テレビ・有線放送料金
* 郵送料金、切手代　など

### インターネット料金

**業務上使用したインターネットに関する費用は、すべて通信費に該当します。**回線使用料だけではなく、契約時の入会金や取り付け工事費に関しても通信費として経費計上可能です。さらに、経理処理のための会計システムや電子メールなどのクラウドサービスの利用料や、レンタルサーバの利用料も通信費として扱えます。

### プロバイダ料金

ネットを繋げて仕事を行う場合、インターネット回線に加えて別途プロバイダの契約が必要です。つまり、インターネット接続サービス料として、プロバイダ会社へ固定費を支払うことになるでしょう。このような**プロバイダ利用料金も、通信費に含まれます。**

### 電話料金

**業務で使用する固定電話やスマートフォンの利用料金は、通信費として計上できます。**近年では使用頻度が減少しましたが、FAX送信代などの電話関連費用も同じく通信費として計上可能です。
  
  
また、個人事業主等が1台のスマートフォンで仕事と私用を併用している場合は、業務上の使用分と按分すれば経費計上できます。按分する際には、仕事で使用する頻度や時間などに応じて通信費として計上する割合を定めます。

### テレビ・有線放送料金

**オフィスや店舗でテレビや有線放送を利用している場合、これらにかかる受信料も通信費に該当します。**また、導入のために工事を行なった場合にかかった費用も、同じように通信費として計上可能です。

### 郵便・切手等にかかる料金

取引先や社内間での書類送付・郵便小包などにかかった郵送料金は、通信費として経費計上します。業務で使用する切手や官製ハガキの購入料金も、同様に通信として計上可能です。ただし、**宣伝用のダイレクトメールや印紙代などは通信費に該当しないため、注意しましょう。**
  
  
また、切手をまとめて購入した場合、最初は「貯蔵品」として計上し、切手を使用した際に通信費として処理します。詳しくは、後述する「[郵便・切手等にかかる料金の仕訳方法](#content4-5)」をご覧ください。
  
  
また、商品や製品などを配送する際の費用は荷造運賃と呼ばれ、通信費での経費計上はできません。荷造運賃については、後述する「[荷造運賃との違いに気をつける](#content5-1)」をご覧ください。

## 通信費と間違えやすい経費例

通信費と間違えやすい主な経費は、以下のとおりです。通信費とそのほかの経費を適切に区分けし、正しい勘定科目で計上するように理解しておきましょう。

  

| 費用 | 正しい勘定科目 |
| --- | --- |
| 電話機・コピー機・PCの購入費用 | 消耗品費 |
| 機器のレンタル・リース料金 | ファイナンシャルリースの場合：リース料  レンタルリースの場合：賃借料 |
| FAX用紙・コピー用紙の購入費用 | 消耗品費 |
| 私製ハガキや便箋等（未使用）の購入費用 | 消耗品費 |
| ダイレクトメールの送料 | 広告宣伝費 |
| 印紙代 | 租税公課 |
| 祝電・電報にかかった費用 | 取引先に対しての場合：交際費  社員に対しての場合：福利厚生費 |
| 電話加入権にかかった費用 | 無形固定資産であるため経費としての扱いは不可 |

  

上記のうち、**リース料はリース資産定額法により減価償却する必要があります。**契約時にリース資産、支払い時にリース債務を計上し、資産総額をリース期間で分割して事業年度の分配を減価償却費として計上しなければなりません。
  
  
なお、**賃借料は、中小企業が1年未満のレンタル期間かつ1件あたりのリース総額が300万円以下である場合、減価償却をしない賃貸借処理が可能です。**これを「オペレーティング・リース取引」といい、賃借料を支払いリース料金や消耗品として経費計上できます。
  
  

[出典：国税庁「5 リース資産の償却等」](https://www.nta.go.jp/law/joho-zeikaishaku/hojin/7081/05.htm)
  
[出典：リース事業協会「新リース会計・税制のご案内」](https://www.leasing.or.jp/book/docs/book_kaikeizeisei.pdf)

## 通信費の仕訳方法と仕訳例

通信費の仕訳方法は、経費の種類によって異なることがあります。そのため、以下で紹介する仕訳例を参考にして正しい会計処理を行いましょう。

### インターネット料金の仕訳方法

オフィスなどで利用した場合のインターネット料金は、全額通信費として経費計上できますが、自宅で仕事をした場合は私用と事業用を按分する必要があります。
  
  
たとえば、オフィスで使用したインターネット料金1万円を、事業用口座から支払った場合の仕訳は以下のとおりです。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 通信費 | 10,000円 | 普通預金 | 10,000円 |

  

一方で、自宅で使用したインターネット料金1万円を、事業用口座から支払った場合の仕訳は以下のとおりです。なお、そのうち事業用としては70％を使用し、私用では30％を使用したとします。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 通信費 | 7,000円 | 普通預金 | 10,000円 |
| 事業主貸 | 3,000円 |

  

上記のように、事業用口座から私用の支払いを行った場合は、借方に「事業主貸」を入れて仕訳します。
  
  
なお按分について詳しい解説は、後述する「[仕事とプライベートで同じ電話・インターネットを使用している場合](#content5-2)」をご覧ください。

### プロバイダ料金の仕訳方法

インターネットを利用するために契約したプロバイダ料金の仕訳は、以下のとおりです。
  
  
以下は、オフィスで契約しているプロバイダ契約料1,000円が、事業用口座より引き落とされた場合の仕訳例です。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 通信費 | 1,000円 | 普通預金 | 1,000円 |

### 電話料金の仕訳方法

業務で使用したスマートフォンや固定電話の使用料は以下のように仕訳をします。以下は、オフィスの固定電話の使用料1万円が、事業用口座より引き落とされた場合の仕訳例です。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 通信費 | 10,000円 | 普通預金 | 10,000円 |

  

**スマートフォンや固定電話を私用と併用した場合は、インターネット料金と同様に按分が必要です。**
  
  
以下は、プライベートと併用しているスマートフォンの使用料1万円が、個人用口座より引き落とされた場合の仕訳例です。なお、事業用としては70％を使用し、私用では30％を使用したとします。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 通信費 | 7,000円 | 事業主借 | 7,000円 |

上記の場合、個人用口座から事業用の支払いを行ったため、貸方に事業主借を入れます。

### テレビ・有線放送料金の仕訳方法

オフィスなどに設置しているテレビのNHK受信料3,000円を、事業用口座より支払った場合、以下のように仕訳します。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 通信費 | 3,000円 | 普通預金 | 3,000円 |

  

このほか、初期の設置にかかった工事費用なども、通信費として上記の例と同じ形式で計上できます。

### 郵便・切手等にかかる料金の仕訳方法

書類送付や郵便小包などにかかった郵送料金の仕訳を紹介します。
  
  
先述のように**あらかじめ購入した切手や官製ハガキは、購入時には貯蔵品として計上し、使用するタイミングで通信費を使い会計処理します。**なお、切手の購入は非課税対象の取引で、切手使用時に消費税が課税される仕組みです。そのため、消費税の取り扱いがわかりやすいよう、切手使用時の仕訳例については税抜経理方式を採用して解説します。
  
  
たとえば、1,200円分の切手を10枚、現金で購入し、その後120円分を使用した場合、以下のように仕訳します。
  
  
【購入時】

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 貯蔵品 | 1,200円 | 現金 | 1,200円 |

  

【使用時】

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 通信費 | 110円 | 貯蔵品 | 120円 |
| 仮払消費税 | 10円 |

  

切手料金に含まれている消費税は切手を使った時に課税対象となるため、税抜経理方式を採用している場合は消費税と分けて処理する必要があります。
  
  
上記が一般的な例ですが、**頻繁に切手を使用していて経理処理が煩雑な場合には、例外として切手購入時に通信費として計上することが認められています。**これは、購入した切手をまだ使っていない場合でも使ったこととして処理する考え方で、会計処理の負担軽減が可能です。
  
  
同じく1,200円分の切手を10枚、現金で購入したとしても、切手購入時に通信費に計上する場合は以下のように仕訳します。
  
  
【購入時】

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 通信費 | 1,091円 | 現金 | 1,200円 |
| 仮払消費税 | 109円 |

  

**上記の方法で経理処理をした場合は、期末に未使用分の切手を確認して貯蔵品として計上する必要があります。**たとえば、期末に600円分の切手5枚が残っていた場合、以下のように仕訳をします。
  
  
【期末】

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 貯蔵品 | 600円 | 通信費 | 540円 |
| 仮払消費税 | 60円 |

[出典：国税庁「物品切手等の購入費」](https://www.nta.go.jp/law/shitsugi/shohi/16/16.htm)
  
[出典：国税庁「第2款　販売費及び一般管理費等」](https://www.nta.go.jp/law/tsutatsu/kihon/hojin/02/02_02_02.htm)

## 勘定科目で通信費を用いる際の注意点

通信費を勘定科目として選択する際は、以下の点に注意が必要です。

勘定科目で通信費を用いる際の注意点

* 荷造運賃との違いに気をつける
* 仕事とプライベートで同じ電話・インターネットを使用している場合
* 記帳時には補助科目や摘要欄を適切に活用する

### 荷造運賃との違いに気をつける

業務に関わる書類などの郵送料金は通信費になりますが、同じ手段として郵送をした場合でも、送る目的によって通信費にはならないケースもあります。この場合は、荷造運賃という勘定科目を使用します。
  
  
荷造運賃とは、販売した商品や製品などを発送するために荷造りをした費用や郵送料金などのことです。つまり、**売上を目的とした配送費用などは、荷造運賃に分類されます。**通信費とよく間違われやすい勘定科目なので、注意しましょう。
  
  
たとえば、購入された商品を小包郵便物で発送し、配送料の500円を現金で支払った場合、以下のように仕訳します。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 荷造運賃 | 500円 | 現金 | 500円 |

[出典：国税庁「荷造運賃」](https://www.keisan.nta.go.jp/r3yokuaru/aoiroshinkoku/hitsuyokeihi/nizukuriunchin/nizukuriunchin.html)

### 仕事とプライベートで同じ電話・インターネットを使用している場合

個人事業主や自宅を事務所としている法人の場合、1台のスマートフォンや自宅のインターネット環境をプライベートと併用していることがあります。このような場合は、**原則として事業用に使用した割合のみを通信費とする按分が必要です。**
  
  
按分する際は、使用時間を基準として計算を行います。また、税務調査に備えて、なぜその比率になるかを客観的に説明できるような、正当な根拠を用意しておくと良いでしょう。
  
  
按分を使用した場合の仕訳例については、記事内の「[インターネット料金の仕訳方法](#content4-1)」「[電話料金の仕訳方法](#content4-3)」をご覧ください。

### 記帳時には補助科目や摘要欄を適切に活用する

通信費は、事業を行う上で頻繁に使用される勘定科目です。一方で、正しく理解していないと勘定科目を誤ってしまう恐れが高いものでもあります。そのため、**明瞭に管理していくためにも、帳簿の適要欄や補助科目を活用しましょう。**
  
  
特に、毎月引き落としされるリース料や電話料金などは、摘要欄に何月分の請求に対しての支払いなのかを記載しておくと良いでしょう。
  
  
たとえば、5月分の支払いとして複合機リース料金5,000円が、事業用口座より引き落とされた場合、摘要欄には以下のような記述をします。

  

| 借方 | | 貸方 | | 摘要 |
| --- | --- | --- | --- | --- |
| 通信費 | 5,000円 | 普通預金 | 5,000円 | **5月分  複合機リース料** |

  

また、**補助科目は、勘定科目の中身を管理するために設定する項目をいいます。**法律で定められているものではないので、自身が管理しやすいように自由に設定でき、**通信費という勘定科目の補助科目に、インターネット代・電話代・通話代などと設定することもできます。**
  
  
たとえば、5月分の支払いとしてNTT電話料金5,000円が、事業用口座より引き落とされたとします。補助科目と摘要欄には、それぞれ以下のように記入できます。

  

| 借方 | | 補助科目 | 貸方 | | 摘要 |
| --- | --- | --- | --- | --- | --- |
| 通信費 | 5,000円 | **電話代** | 普通預金 | 5,000円 | **5月分  NTT固定電話料** |

  

このように、勘定科目の内訳として補助科目を使用したり詳細な内容を摘要に入れたりすることで、管理がしやすく明瞭な経理処理ができます。

## まとめ

通信費とは、インターネット料金やプロバイダ料金などの経費を会計処理する際に用いる勘定科目です。しかし、PCや電話の購入費用は通信費とはならず消耗品費となったり、本来ならば通信費として計上できる費用が使用目的によって通信費とならなかったりする場合もあり、注意しなければなりません。
  
  
また、仕事用と私用を混在して利用するケースでは、按分を行って処理する必要があります。通信費の取り扱いについて正しく理解し、補助科目や摘要を使用して日頃からわかりやすい経理処理を行いましょう。

## よくある質問

### 通信費を勘定科目にできるものは？

通信費を勘定科目にできる費用には、インターネット料金やプロバイダ料金などがあります。また、テレビ・有線放送料金や郵便・切手等にかかる料金も通信費としての計上が可能です。
  
  
詳しくは記事内「[通信費として仕訳できる主な経費の種類](#content2)」をご覧ください。

### 通信費とは具体的になんですか？

通信費とは、業務上使用する通信関連や配送などに関する経費を計上するための勘定科目です。インターネット料金・プロバイダ料金・固定電話・携帯電話の通話料や通信料・テレビや有線放送料金・郵送料金や切手代などが、通信費に含まれます。
  
  
詳しくは記事内「[通信費とは](#content1)」をご覧ください。

### 電話代は通信費にできますか？

業務上使用した電話代はFAX送信代、さらに初期に行った工事費などは通信費として計上可能です。ただし、私用の携帯電話と併用をしている場合は、按分が必要となるため注意してください。
  
  
詳しくは記事内「[電話料金](#content2-3)」をご覧ください。
