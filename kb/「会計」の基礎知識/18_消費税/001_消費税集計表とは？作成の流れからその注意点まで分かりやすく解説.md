## 消費税集計表とは

**消費税集計表は、納める消費税額を事前に計算するための書類です。**消費税の申告時に提出する義務はなく、特に決められたフォーマットもないため、納税者が自由に作成してかまいません。
  
  
**消費税集計表では、売上や仕入ごとに消費税を集計し、その結果をもとに納税額を試算します。**詳しくは後述しますが、消費税の税率には標準税率（10%）と軽減税率（8%）があり、内訳として国税・地方税に分けられます。各税率ごとに集計する必要があり、さらに非課税取引・不課税取引・免税取引が混在していれば、これらも分けて集計しなければなりません。
  
  
消費税の集計はこのように区分が複雑なため、消費税集計表を作成してあらかじめ試算しておくと、申告の際に役立つでしょう。
  
  
**ただし消費税集計表は、あくまでも目安の金額を把握し、納税に備えるための表です。実際に確定した納税額と差異が生じる可能性がある点には留意しましょう。**

  

[出典：国税庁「消費税のしくみ」](https://www.nta.go.jp/publication/pamph/koho/kurashi/html/01_3.htm)

## 消費税集計表の作成手順

消費税集計表を作成するための明確なルールはないものの、以下の手順を踏んで作成するとスムーズに進められます。

消費税集計表の作成手順

1. 自社の経理方式が税抜・税込なのかを確認する
2. 日々の帳簿付けにおいて税区分が正確かを確認する
3. 税率ごとに課税取引額を算出する
4. 課税取引の税抜金額に対し、必要な税率をかけて消費税額を算出する

**まずは各取引における経理方式が、税抜・税込のどちらかを確認します。**最終的には税抜金額に税率をかけて消費税額を算出するため、自社の経理方式が税込で統一されているのであれば、税込金額に税率をかけないよう注意しましょう。
  
  
また、日々の仕訳が正しい税区分になっているかをチェックします。消費税集計表のもととなる仕訳帳や元帳は、正確な税区分で入力されている必要があります。軽減税率に対応した会計ソフトを使用している場合も、取引ごとに税区分を入力しておくと安心です。
  
  
**税区分を確認できたら、各税率・区分ごとに課税売上や課税仕入の金額を集計します。****この際、非課税取引・不課税取引・免税取引が課税取引に混在しないよう注意が必要です。**課税取引額が算出できたら、それぞれに対し必要な税率を乗じて消費税額を求めてください。

  

[出典：国税庁「年間取引の集計（課税取引金額計算表等の作成）」](https://www.nta.go.jp/taxes/shiraberu/zeimokubetsu/shohi/keigenzeiritsu/pdf/sakusei_nagare/10.pdf)

### 消費税集計表を作成するための書類

**消費税集計表を作成するにあたって必要な書類は、総勘定元帳や消費税区分表です。**
  
  
総勘定元帳には、各仕入・売上に応じた勘定科目や税率ごとの集計が記載されています。消費税区分表は、各税率や非課税取引・不課税取引を取引内容ごとに記載した書類です。

  
  
なお、消費税は以下の税率区分があります。

消費税の税率区分

標準税率（10%）：うち国税7.8%、地方税2.2%  
軽減税率（8%）：うち国税6.24%、地方税1.76%

**2019年9月30日までは8%（うち国税6.3%、地方税1.7%）でしたが、10月1日から現在の税率へと引き上げられました。**

  

[出典：国税庁「軽減税率制度の概要」](https://www.nta.go.jp/taxes/shiraberu/zeimokubetsu/shohi/keigenzeiritsu/01.htm)

また、課税取引・非課税取引・不課税取引・免税取引にはそれぞれ要件があり、取引に応じて適切に区分しなくてはなりません。

  

|  |  |  |
| --- | --- | --- |
| **区分** | **要件** | **例** |
| **課税取引** | ・国内取引  ・事業取引  ・対価を得る取引  ・資産の譲渡や貸付、役務の提供 | ・商品やサービスの購入・提供 |
| **非課税取引** | 課税取引の要件には当てはまるものの、社会政策的に配慮されている取引や課税対象としてなじまない取引 | ・土地や有価証券、商品券などの譲渡  ・預貯金や貸付金の利子  ・社会保険料  ・商品券や金券の譲渡  ・公的書類の発行手数料 |
| **不課税取引** | 課税取引の要件に当てはまらない取引 | ・贈与や寄附 ・出資に対する配当  ・海外取引  ・給与 |
| **免税取引** | 課税取引の要件には当てはまるものの、海外で消費される取引 | ・商品の輸出や国際輸送  ・外国事業者へのサービス提供 |

[出典：国税庁「No.6205 非課税と免税の違い」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shohi/6205.htm)
  
[出典：国税庁「No.6209 非課税と不課税の違い」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shohi/6209.htm)

## 消費税集計表の記載項目

**消費税集計表に記載する項目は、主に5つです。**ここでは標準税率（10%）だと想定して記載します。

|  |  |  |
| --- | --- | --- |
| 項目 | 概要 | 申告書への記載有無 |
| **1.課税売上高 （税込）** | 税込の課税売上金額 （税率10%なら税抜額の110%） | 記載なし |
| **2.課税標準額** | 課税対象額 （税率10%なら「1」の金額に100/110をかけた額） | 記載あり ※1,000円未満は切り捨て |
| **3.消費税額** | 売上に対する国税の額 （消費税率10%なら「2」に7.8%をかけた金額） | 記載あり |
| **4.控除税額小計** | 仕入・控除分の消費税の合計額 | 記載あり |
| **5.差引税額** | 「3」の消費税額から「4」の仕入・控除税額を除いた額 | 記載あり ※100円未満は切り捨て |

## 消費税集計表の作成時の注意点

**消費税集計表を作成するときに誤った額を算出しないよう、帳簿への仕訳や記載内容について日頃からしっかり確認しましょう。**また、各勘定科目に応じて適切な税区分・税率区分になっているかを取引ごとによく見直します。
  
  
簡易課税を選択しているのであれば、課税売上分のみの集計でかまいません。一方、本則課税であれば売上・仕入ともに税区分ごとの集計が求められます。
  
  
**税額を算出する際は、非課税・不課税・免税取引が混在しないよう注意しましょう。**
