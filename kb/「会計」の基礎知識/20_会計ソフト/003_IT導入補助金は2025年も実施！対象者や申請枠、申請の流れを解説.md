## IT導入補助金とは

**IT導入補助金とは、中小企業や小規模事業者などを対象に、ITツールやクラウドシステムなどを導入する際の経費を一部補助する制度のことを指します。**この制度の目的は、自社の課題やニーズに合ったITツールの導入による、業務効率化や売上向上の支援です。

  
  
IT導入補助金の申請者は中小企業や小規模事業者などといった補助事業者となり、申請先はIT導入補助金事務局になります。補助事業者は、IT導入支援事業者（ITベンダー・サービス事業者）に対してITツールの導入について相談します。
  
申請スケジュールや利用可能な製品については、[IT導入補助金ガイド2025のページ](https://www.freee.co.jp/solution/it-hojokin/)でも詳しく解説しています。

## IT導入補助金のメリット

IT導入補助金を活用することで得られる主なメリットを解説します。

### ITツール購入のリスクを軽減できる

**IT導入補助金は、ITツールを導入するにあたっての初期投資リスクを大幅に軽減できることが一番のメリットといえます。**
  
  
この補助金制度は**ITツールの購入費用を事後的に補助するものです。**まずは補助金を申請し、補助金の交付決定後に購入手続きを行う仕組みになっているため「購入したのに補助金が下りない」というリスクを回避できます。
  
  
なお、補助金が還付されるのは原則として導入実績の報告から1～2ヶ月後です。ITツール購入後に還付される仕組みなので、補助金を期待した無理な資金繰りは避けましょう。

### 原則として返済の必要がない

**IT導入補助金は、原則として返済不要の補助金です。**事業計画に沿ってITツールを適切に導入し、導入後に事業実績を適切に報告すれば、返済を求められることはありません。
  
  
しかし、**後から還付した補助金を返済するよう求められるケースもあります。**返済を要求される事例としては、事業期間中や補助金交付後に不正行為があった場合や、情報漏洩などの問題が発覚した場合です。

### 複数回申請できる

**IT導入補助金は募集が複数回あり、一度不採択になっても同年度内であれば何度でも再申請できます。**
  
  
過去に採択された実績がある場合でも、交付決定日から12ヶ月以上経過していれば再度申請できるので、必要に応じたITツールの導入に役立ちます。

### 少額でも補助される

**IT導入補助金には、補助金の下限設定がない枠や少額からでも補助される枠があります。**
  
  
そのため少額のITツール導入でも申請が可能で、資金に余裕がない企業でも利用しやすいのもメリットといえるでしょう。

## IT導入補助金2025の概要

**2025年3月31日から、IT導入補助金2025の受付が開始されます。**
  
  
IT導入補助金2025では、以下の4つの申請枠が設定されています。

  

* 通常枠
* インボイス枠（インボイス対応類型・電子取引類型）
* セキュリティ対策推進枠
* 複数社連携IT導入枠

  

それぞれの枠について解説します。

### 通常枠

**通常枠は、中小企業や小規模事業者などの事業をデジタル化することを目的として、ソフトウェアやシステムの導入を支援するものです。**
  
  
次に挙げる業務プロセスの中から1種類以上のソフトウェアを申請することが、条件となります。なお、汎用プロセスのみを選ぶことはできません。

業務プロセス（共通プロセス）

* 顧客対応・販売支援
* 決済・債権債務・資金回収
* 供給・在庫・物流
* 会計・財務・経営
* 総務・人事・給与・労務・教育訓練・法務・情報システム
* その他業種固有のプロセス

業務プロセス（業務特化型プロセス）

* 汎用・自動化・分析ツール

補助率と補助金額は、次のとおりです。プロセス数に応じて補助額が設定されており、**最大450万円の補助金を受けられます。**

  

|  |  |
| --- | --- |
| 補助率 | 1/2以内 |
| 補助額 | 1プロセス以上：5万円以上150万円未満  4プロセス以上：150万円以上450万円以下 |

[出典：IT導入補助金2025事務局ホームページ](https://it-shien.smrj.go.jp/applicant/subsidy/normal/)

### インボイス枠

インボイス枠は、インボイス対応類型と電子取引類型の2つに分かれます。それぞれの内容は以下のとおりです。

#### インボイス対応類型

**インボイス枠（インボイス対応類型）は、インボイス制度に対応した会計ソフト、受発注ソフト、決済ソフト、PC・ハードウェアなどの導入を支援する枠です。**インボイス制度への対応を強力に推進することを目的としています。
  
  
**このPCには、ノートパソコンも含まれます。**すでにノートパソコンがある場合で、よりスペックの高いものへ買い替えたいときにも使用可能です。
  
  
**会計・受発注・決済ソフト**

  

|  |  |  |
| --- | --- | --- |
| 補助率 | 3/4以内、4/5以内※1 | 2/3以内 |
| 補助額 | 50万円以下※2 | 50万円超〜350万円以下 ※3※4 |

  

**PC・ハードウェアなど**

  

| 補助対象 | PC・タブレットなど | レジ・券売機など |
| --- | --- | --- |
| 補助率 | 1/2以内 | 1/2以内 |
| 補助額 | 10万円以下 | 20万円以下 |

  

※1 中小企業は3/4、小規模事業者は4/5
  
※2 会計・受発注・決済のうち1機能以上を有することが機能要件
  
※3 補助額50万円超の際の補助率は、補助額のうち50万円以下については3/4（小規模事業者は4/5）、50万円超については2/3
  
※4 会計・受発注・決済のうち2機能以上を有することが機能要件
  
  

[出典：IT導入補助金2025事務局ホームページ](https://it-shien.smrj.go.jp/applicant/subsidy/digitalbase/)

#### 電子取引類型

**電子取引類型は、インボイス制度に対応した「受発注」の機能を有するソフトウェアを導入し、労働生産性の向上およびインボイス制度への対応をサポートする枠です。**
  
  
以下の流れでITツールを導入し、導入経費の一部を補助することで、中小企業・小規模事業者などの生産性向上とインボイス制度への対応を促進することを目的としています。

ITツール導入の流れ

1. 発注者である事業者がインボイス制度対応のITツール（受発注ソフト）を導入
2. 受注者である中小企業・小規模事業者等に対してITツールを供与

|  |  |
| --- | --- |
| 補助率 | 中小企業・小規模事業者等：2/3 以内  その他事業者等：1/2 以内 |
| 補助額 | （下限なし）～350万円以下 |

[出典：IT導入補助金2025事務局ホームページ](https://it-shien.smrj.go.jp/applicant/subsidy/digitalbased_invoice/)

### セキュリティ対策推進枠

**セキュリティ対策促進枠は、サイバー攻撃の増加に伴う潜在的なリスクに対処するため、サイバーインシデントに関するさまざまなリスク低減策をサポートする枠です。**
  
  
「サイバーセキュリティお助け隊サービスリスト」の中で、IT導入支援事業者が提供し、事務局に事前登録されているサービスを導入するときに、最大2年分のサービス利用料が補助金として支援されます。

  

|  |  |
| --- | --- |
| 補助率 | 小規模事業者：2/3以内  中小企業：1/2以内 |
| 補助額 | 5万円以上150万円以下 |

[出典：IT導入補助金2025事務局ホームページ](https://it-shien.smrj.go.jp/applicant/subsidy/security/)

### 複数社連携IT導入枠

**複数社連携IT導入枠は、業務において関連するサプライチェーンや、特定の商圏で事業を展開する複数の中小企業・小規模事業者が、連携してITツールを導入し、生産性向上を図る取り組みを支援するものです。**
  
  
通常枠よりも補助率が高く、複数社が連携して効果的な取り組みを行えるよう、コーディネート費や外部専門家への謝金なども対象としています。

  

| 補助対象経費 | | | 補助率 | 補助額 | |
| --- | --- | --- | --- | --- | --- |
| 基盤導入経費 | ソフトウェア | | 3/4以内、  4/5以内  ※1 | 50万円以下×グループ構成員数 | 3,000万円以下※2 |
| 2/3以内  ※1 | 50万円超〜350万円以下×グループ構成員数 |
| ハードウェア | PC・タブレット等 | 1/2以内 | 10万円×グループ構成員数 |
| レジ・券売機等 | 20万円×グループ構成員数 |
| 消費動向等分析経費 | | | 2/3以内 | 50万円以下×グループ構成員数 |
| その他経費 | | | 2/3以内 | 200万円以下※3 | |

  

※1 補助額のうち50万円以下については補助率は3/4以内（ただし、小規模事業者は4/5以内）、50万円超については補助率は2/3以内
  
  
※2 基盤導入経費と消費動向分析経費の合計額は3,000万円が上限
  
  
※3 補助額上限は「【基盤導入経費と消費動向等分析費の合計額】 × 10% × 2/3（補助率）」もしくは200万円のいずれか小さい額

### 小規模事業者

小規模事業者に該当するのは、以下のとおりです。

  

| 業種・組織形態 | 常勤従業員 |
| --- | --- |
| 商業・サービス業（宿泊業・娯楽業除く） | 5人以下 |
| サービス業のうち宿泊業・娯楽業 | 20人以下 |
| 製造業その他 | 20人以下 |

### 対象外となる事業者

**次のいずれかに該当する「みなし大企業」は、申請の対象外です。**

1. 発行済株式の総数または出資価格の総額の2分の1以上を同一の大企業が所有している中小企業・小規模事業者等
2. 発行済株式の総数または出資価格の総額の3分の2以上を大企業が所有している中小企業・小規模事業者等
3. 大企業の役員または職員を兼ねている者が、役員総数の2分の1以上を占めている中小企業・小規模事業者等
4. 発行済株式の総数または出資価格の総額を、上記1～3に該当する中小企業・小規模事業者等が所有している中小企業・小規模事業者等
5. 上記1～3に該当する中小企業・小規模事業者等の役員または職員を兼ねている者が、役員の全てを占めている中小企業・小規模事業者等
6. 確定している（申告済みの）直近過去3年分の各年又は各事業年度の課税所得の年平均額が15億円を超える中小企業・小規模事業者等

[出典：IT導入補助金2025 交付規程](https://it-shien.smrj.go.jp/download)
  
  
そのほか、以下の事業者も対象外となります。

* 申請する中小企業・小規模事業者等またはその法人の役員が、暴力団等の反社会的勢力であるもの。反社会的勢力との関係を有するもの。また、反社会的勢力から出資等資金提供を受ける場合も対象外とする。
* 事務局が、本補助金の交付対象者として不適当であると判断する者

  

[出典：IT導入補助金2025 交付規程](https://it-shien.smrj.go.jp/download)

## IT導入補助金2025の4つのポイント

IT導入補助金2025で特に押さえておきたい4つのポイントを解説します。

### 1. インボイス枠｜小規模事業者の補助率は最大80％

**インボイス枠のうちインボイス対応類型は、小規模事業者であれば補助率が最大4/5（80％）となります。**同類型における中小企業の補助率は最大3/4（75％）なので、小規模事業者のほうがより補助の割合が高くなるのが特徴です。

  

| 事業者の種類 | 補助率 |
| --- | --- |
| 中小企業 | 3/4 |
| 小規模事業者 | 4/5 |

  

なお、上記の補助率が適用される補助額の上限は50万円となります。50万円を超える場合には、50万円まではそれぞれ上記の補助率が適用され、50万円を超える部分については中小企業・小規模事業者いずれも2/3が適用されます。

### 2. セキュリティ対策推進枠｜補助額・補助率拡大

**以下に該当する事業者は、セキュリティ対策推進枠における補助額上限が150万円となり、補助率についても最大2/3に拡大しています。**

  

| 業種分類 | 定義 |
| --- | --- |
| 商業・サービス業（宿泊業・娯楽業以外） | 従業員の数が5人以下の会社／個人事業主 |
| 商業・サービス業（宿泊業・娯楽業） | 従業員の数が20人以下の会社／個人事業主 |
| その他の業種 | 従業員の数が20人以下の会社／個人事業主 |

### 3. 通常枠｜最低賃金近傍の事業者の補助率拡大

## IT導入補助金2025の申請の流れ

IT導入補助金の申請の流れを解説します。

### 1. 公募要領の理解

**まずは、IT導入補助金2025の概要と枠ごとの公募要領を理解しましょう。**詳しくは、[IT導入補助金2025事務局ホームページ](https://it-shien.smrj.go.jp/download/#basic)から確認できます。

### 2. GビズID取得・SECURITY ACTION宣言実施

**交付申請の要件には、「GビズIDプライム」（ID・パスワード等）が必要になります。**GビズIDホームページから取得可能で、発行までの期間は2週間ほどが目安です。
  
  
また、**申請事業者自らが情報セキュリティ対策に取り組むことを自己宣言する制度「SECURITY ACTION」の対応も行います。**
  
  
IT導入補助金2025の交付申請作成時に、SECURITY ACTIONの宣言済みアカウントIDの入力が必要です。[SECURITY ACTION](https://www.ipa.go.jp/security/security-action/it-hojo.html)ページから登録と宣言が行えます。

### 3. IT事業者・ITツールの選定

**補助金の交付申請を行う準備として、まずは自社の業種や事業規模、経営課題に沿って、IT導入支援事業者と導入したいITツールを選定しておきましょう。**

### 4. 交付申請

**IT導入支援事業者と商談を進めていき、交付申請の事業計画を策定します。**
  
  
その後、以下の流れで交付申請を行ってください。

交付申請の流れ

1. IT導入支援事業者から『申請マイページ』の招待を受け、代表者氏名等の申請者基本情報を入力する。
2. 交付申請に必要となる情報入力・書類添付を行う。
3. IT導入支援事業者にて、導入するITツール情報、事業計画値を入力する。
4. 『申請マイページ』上で入力内容の最終確認後、申請に対する宣誓を行い事務局へ提出する。

### 5. 交付決定

**交付申請内容の審査が終わると、交付決定の通知がなされます。**通知を受けた申請者は補助事業者となり、補助事業を始められます。

### 6. ITツールの発注・契約・支払い

**交付申請が完了して事務局から「交付決定」を受けたら、ITツールの発注・契約・支払いなどを行います。**
  
  
交付決定前に発注・契約・支払い等を行った場合は、補助金の交付を受けられないので注意してください。

### 7. 事業実績報告

**補助事業の完了後は、実際にITツールの発注・契約、納品、支払いなどをしたことがわかる証憑を提出します。**
  
  
証憑の提出は、以下の流れで行います。

証憑提出の流れ

1. 中小企業・小規模事業者等の担当者が『申請マイページ』から事業実績報告に必要な情報の入力及び証憑の添付を行い、 事業実績報告を作成する。
2. 事業実績報告が作成された後、IT導入支援事業者が内容の確認及び必要情報の入力を行う。
3. 最終確認後、中小企業・小規模事業者等の担当者が事務局に事業実績報告を提出する。

なお、事業が適正に行われなかった場合や、実績報告期間中に正しく実績報告が行われなかった場合は、補助金の交付を受けられません。
  
  
**実績報告が提出されるまでにすべてのITツールにおいて「事業」が完了していること、ITツールの利用・運用が開始されている必要があります。**

### 8. 補助額の確認・承認

**申請マイページで確定検査の結果と補助金交付決定額を確認できるので、内容に間違いがなければ承認します。**
  
  
承認にはSMS認証が必要です。承認しない場合、補助金は交付されません。

### 9. 事業実施効果報告

**事業実施効果報告は、定められた期限内に補助事業者が「申請マイページ」より必要な情報を入力し、IT導入支援事業者の確認を経て提出することができます。**

### 通常枠

この場合は、システムの年間利用料に加えて、初期設定費用も補助対象経費として採択を受けました。

## IT導入補助金の不正受給に注意

**IT導入補助金を活用する際は、不正行為を行っていないか注意しましょう。**
  
  
具体的にどのような行為が不正とみなされるのか、不正が発覚した場合の罰則について解説します。

### IT導入補助金の不正受給行為とは

[IT導入補助金2025事務局ホームページ](https://it-shien.smrj.go.jp/news/1)では、次の行為を不正にあたる行為として明示しています。

1. 本補助事業と同一の内容で国（独立行政法人を含む）から他の補助金、助成金等の交付を重複して受けていた場合。
2. 事業期間中及び補助金交付後において、不正行為、情報の漏洩等の疑いがあり、補助事業者として不適切な行為を行っていた場合。
3. ITツールが導入されていない、役務の提供がなされていない等、補助事業が遂行されていない場合。
4. 補助事業者自身が行うべき行為（申請マイページの開設及びその後の交付申請における手続き等）を当該補助事業者以外が行っていた場合（なりすまし行為）。
5. ITツールの販売金額に占める補助事業者の自己負担額を減額又は無償とするような販売方法（形式・時期の如何を問わず、補助事業者に実質的に還元を行うもの）あるいは、一部の利害関係者に不当な利益が配賦されるような行為を行っていた場合。

### 不正が発覚した場合の罰則

**補助金不正が発覚した場合は、交付決定取消、補助金の返還請求、IT導入支援事業者登録取消がなされます。**
  
  
また、補助金の不正受給によって罰則を課せられるだけでなく、社会的な信用を失うことにもなりかねません。

### 不正を防ぐために意識すべきこと

**IT導入補助金に関する金銭の額が一致しないケースは、不正と疑われやすくなります。**
  
  
たとえば、**申請者がITツールを購入した際にポイント・クーポンなどを利用し、購入額を証明する証憑に記載された金額と実質的に支払われた金額が一致しないケースです。**
  
  
IT導入補助金事務局では不正行為等の調査を行っていますので、疑わしい行為はしないように注意しましょう。
