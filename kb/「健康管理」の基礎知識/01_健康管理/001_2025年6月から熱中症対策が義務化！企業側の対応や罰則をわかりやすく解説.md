## 2025年6月から熱中症対策が義務化に

2025年6月から、厚生労働省が所管する改正労働安全衛生規則が施行され、屋内外を問わず暑熱環境で作業を行う労働者に対して、熱中症予防措置の実施が企業に義務付けられました。
  
  
これまでも指針レベルで予防策が求められていましたが、今回の改正により義務として明文化された点が大きなポイントです。
  
  
本改正により、暑熱環境での労働において、WBGT値を把握し、一定基準を超える場合に具体的な予防措置を講じることが必要となります。WBGT値とは、気温・湿度・輻射熱（太陽光や地面などによって伝わる熱）・風速などを総合的に評価する暑さの指数です。
  
  
対応を怠った場合には、労働基準監督署による是正指導や罰則の可能性もあるため、企業としては制度理解と実行が急務です。
  
  

[出典：厚生労働省「労働安全衛生規則の一部を改正する省令の施行等について」](https://www.mhlw.go.jp/content/001490909.pdf)

### 熱中症対策の対象となる環境条件

熱中症対策が義務化されるのは、**WBGT28度以上または気温31度以上の環境下で、連続1時間以上または1日4時間を超えて実施する業務環境です。**
  
  
具体的には以下のような状況が該当します。

| 区分 | 作業例 |
| --- | --- |
| 屋外 作業 | ・建設現場（とび職・解体業・舗装業など）  ・道路・橋梁工事  ・屋外でのイベント設営・警備業務  ・運送業（荷積み・荷下ろし） |
| 屋内 作業 | ・高温多湿の工場内作業（鋳造・鉄鋼・クリーニング・食品加工など）  ・換気が不十分な倉庫や作業場  ・空調が不十分な厨房や物流センター |

たとえば、十分に冷房の効いたオフィスや空調完備の屋内での作業においては、これの対象とはなりません。
  
  
なお、短時間であっても熱中症のリスクが高い作業に該当する場合は、対策が必要となる可能性があります。

## 現場で企業が熱中症予防に求められる対策

2025年6月の施行により、企業にはWBGT値の計測や暑熱環境に応じた具体的な対策の実施が義務化されました。具体的には以下の4つに分られます。

### ① 作業環境管理

暑熱環境での作業時には、作業場所の温度や湿度を適切に管理することが熱中症予防の基本となります。
  
  
作業環境管理では以下の実施が推奨されます。

作業環境管理の具体的な対策

1. **WBGT値の低減等**
     
   ・発熱体と労働者の間に遮へい物を設置
     
   ・屋外作業では簡易な屋根を設け、直射日光や照り返しを遮る
     
   ・屋内外問わず、扇風機・スポットクーラー・除湿機などによる通風・冷房設備を整備
2. **休憩場所の整備等**
     
   ・冷房などを備えた涼しい休憩場所の設置
     
   ・冷水や冷たいおしぼりなどの支給

労働者が気軽に身体を冷やし、熱を逃がせる環境づくりが重要です。

### ② 作業管理

作業の計画や進行管理においても、熱中症リスクを軽減するための配慮が必須です。
  
  
作業管理では以下が推奨されています。

作業管理の具体的な対策

1. **作業時間の短縮等**
2. **暑熱順化**
3. **水分及び塩分の摂取**  
   ・自覚症状の有無にかかわらず、水分や塩分をこまめに補給させる
4. **服装等**  
   ・透湿性・通気性の良い服装を着用させる
5. **作業中の巡視**

暑熱順化とは、身体が暑さに慣れて発汗機能や循環機能が向上し、熱中症にかかりにくくなる状態を指します。
  
  
例としては、以下のような方法が有効です。

* 初夏や配属直後などは、軽作業から開始して段階的に慣らす
* 発汗を促す有酸素運動や軽い作業を1日30分程度実施
* 3日以上の休暇明けには、再度順化を意識した作業計画を立てる

暑熱順化は数日〜1週間程度で得られますが、効果を保つには継続が重要です。
  
  
作業中においては、水分や塩分をこまめに補給するように促しましょう。また、作業中の服装も、ファン付きのウェアやクールタオル、UVカット素材の作業服などを導入することで暑熱リスクの低減につながります。
  
  
高齢者や体力に不安のある労働者には、より冷却効果の高い装備の優先提供も検討されます。

### ③ 健康管理

作業者の健康状態を把握し、体調不良の早期発見・対応を行う体制整備が求められます。健康管理では、以下が推奨されています。

健康管理の具体的な対策

1. 健康診断結果に基づく対応等
2. 日常の健康管理
3. 労働者の健康状態の確認
4. 身体の状況の確認

体調不良者を作業に従事させないために、作業前の健康チェックは必ず実施するようにしましょう。主な実施内容には以下が含まれます。

* 毎朝の検温や血圧測定、体調アンケートの実施
* 飲酒・睡眠不足・風邪気味などの自覚症状を申告させる
* 既往歴（心疾患・糖尿病・高血圧など）を考慮した配置

体調不良が疑われた場合は、躊躇なく作業を中断し、管理者への報告を徹底させることが重要です。
  
  
現場で働く1人ひとりが安全に業務を続けられるよう、日々の健康チェックと迅速な対応ルールを整えていきましょう。

### ④ 労働衛生教育

労働衛生教育は、熱中症の危険性と対策方法について、現場作業者に正しく理解してもらうための教育を指します。
  
  
労働者に対して熱中症の対処法やリスク、事例について、共有しておくことで、適切な作業環境の構築につながります。労働衛生教育で推奨されている内容としては以下が挙げられます。

労働衛生教育の具体的な対策

1. 熱中症の症状
2. 熱中症の予防方法
3. 緊急時の救急処置
4. 熱中症の事例

教育内容は毎年見直し、最新の事故事例や法令改正を反映させることで、現場に即した実践的な内容を維持できるでしょう。
  
  
万が一、熱中症が疑われる従業員が発生した際、迅速な対応ができるよう救急体制の整備も義務とされています。

* 現場ごとに救急対応責任者を設定
* 倒れた際の初期対応フロー（涼しい場所へ移動・衣服を緩める・体を冷やすなど）を全員に周知
* 最寄りの医療機関・救急連絡先を掲示
* 作業現場から救急搬送までの経路と連絡網を事前に確認し、シミュレーションを実施

連絡手段としてはスマホや無線・メッセージアプリなどを併用し、現場に応じた最適な手段を確保する必要があります。

## 熱中症対策義務違反による罰則

熱中症対策義務に違反した場合、罰則や行政指導を受ける可能性があります。

### 6ヶ月以下の拘禁刑または50万円以下の罰金

本改正で新たに事業者に義務付けられた措置を怠った場合、同条に基づき刑事罰が科される可能性があります。

* 6ヶ月以下の拘禁刑または50万円以下の罰金
* 法人に対しては両罰規定により、事業者だけでなく法人自体にも罰金が科されることがある

  

[出典：e-Gov法令検索「労働安全衛生法第119条1項、第122条」](https://laws.e-gov.go.jp/law/347AC0000000057#Mp-Ch_12-At_119)

複数の事業者が関与する共同作業現場などにおいて、違反とみなされるケースがあります。
  
  
改正労働安全衛生規則にもとづく熱中症対策が不十分であった場合、元請だけでなく関係するすべての事業者も罰則対象となります。

### 作業の全部または一部の停止

[労働安全衛生法第98条](https://laws.e-gov.go.jp/law/347AC0000000057#Mp-Ch_10-At_98)に基づき、厚生労働省や労働基準監督署は、熱中症による重大事故が発生した場合や、指導に従わない事業所に対しては行政処分として作業停止命令を出すことが可能です。
  
  
命令に従わない場合は、さらなる刑事罰や公表措置に発展するケースもあります。
  
  
熱中症による重篤な労災（死亡や後遺障害）が発生した場合は、早期の現場調査や是正命令が出される可能性が高くなるので注意が必要です。

### 安全配慮義務違反

民事上の責任として、労働契約法第5条に基づく「安全配慮義務違反」が問われる可能性もあります。
  
  
労働契約法第5条の「安全配慮義務」とは、使用者（企業・雇用主）が労働者の生命や健康を損なわないように配慮する義務のことです。

> 労働者の安全への配慮
>   
>   
> 使用者は、労働契約に伴い、労働者がその生命、身体等の安全を確保しつつ労働することができるよう、必要な配慮をするものとする。
>   
>   
>
> [出典：労働契約法第5条](https://www.mhlw.go.jp/bunya/roudoukijun/roudoukeiyaku01/dl/13.pdf)

WBGT値の測定や冷却・休憩設備の設置が義務化され、その対策も、安全配慮義務の一環であり、怠れば企業が責任を問われるリスクが高まります。

## 熱中症対策義務化で企業がまずすべきこと

熱中症対策義務化で企業は計画的に対策を講じていくことが求められます。以下のような段階を踏んで、熱中症対策の具体的な予防策を決定していきましょう。

1. 現場の作業について対象となるかどうか確認
2. 早期発見のための報告体制を構築
3. 症状悪化を防ぐための手順決定

### 1. 現場の作業について対象となるかどうか確認

まず、自社の事業所や業務内容が熱中症対策義務化の対象であるかを確認する必要があります。以下に該当する場合には特定の措置が企業側に義務付けられます。

* WBGT（湿球黒球温度）28度又は気温31度以上の作業場において行われる作業
* 継続して1時間以上または1日あたり4時間を超えて行われることが見込まれる作業

対象となる場合、WBGT値の測定や記録を日常的に行い、高温環境下での作業制限や休憩ルールの整備が必要です。

### 2. 早期発見のための報告体制を構築

熱中症による従業員の体調不良を早期に把握するための仕組みづくりを検討しましょう。たとえば、判断基準や対応方法のマニュアル化や体調チェックの報告フローの明確化などが挙げられます。
  
  
また、体調が悪化したときに報告がしやすい職場環境に整えることも重要です。

### 3. 症状悪化を防ぐための手順を決める

熱中症の初期症状がみられた場合の応急対応マニュアルを整備し、全従業員に周知します。
  
  
マニュアルの具体的な内容としては、以下のようなものが挙げられます。

* 水分・塩分の補給や冷却（首・脇・足の付け根など）
* 意識障害がある場合は、直ちに救急搬送を手配
* 対応責任者の明確化（現場で誰が対応・判断するか）とエスカレーションルートの整備

また、過去の事例を用いたシミュレーショントレーニングの実施も、現場対応力向上に有効です。
  
  
現場ごとに「異変に気づいたら何をするか」を明確にし、全員が迷わず行動できる体制を整えておきましょう。

## 熱中症対策年間スケジュール

熱中症対策は夏季だけでなく、年間を通じて段階的な準備・実施・振り返りが必要です。具体的な年間スケジュールは以下のとおりです。

| 時期 | 主な対応内容 |
| --- | --- |
| 1月〜3月 | ・前年の事故/対応を振り返り  ・WBGT測定器の整備・点検  ・予算計画・物品（飲料・空調・冷却グッズ）準備 |
| 4月〜5月 | ・WBGT測定の運用開始  ・熱中症対策マニュアルの更新  ・教育訓練（衛生講習）  ・初期対応訓練 |
| 6月〜9月 | ・本格的なWBGT測定・記録運用  ・体調不良者への即時対応  ・現場巡視とリスク評価の強化 |
| 10月〜12月 | ・対応内容の振り返り・再評価  ・教育・啓発コンテンツの更新  ・行政報告の準備・提出（必要な場合） |

## 【業種別】熱中症対策の取り組み事例

すでに多くの企業が熱中症リスクに対応するための具体的な対策を導入しています。ここからは実際に導入された熱中症対策を業種別に紹介します。

### 建設業

建設現場では、屋外作業や閉所での作業が多く、日射や輻射熱、風通しの悪さが熱中症リスクが高いです。
  
  
こうした環境に対しては、WBGT値の測定と掲示を行い、数値が高い場合には水分・塩分の摂取を30分または1時間ごとに促し、職長が摂取状況を記録しました。
  
  
また、炎天下での作業を中止とするWBGT値の基準を事前に定め、作業計画に反映しています。
  
  
さらに、ミストファンやスポットクーラー、パラソルの設置など物理的な暑熱環境の改善策を積極的に導入しました。
  
  
休憩所が確保できない現場では、作業車を一時的な休憩所として活用する工夫も見られます。
  
  

[出典：厚生労働省「熱中症対策事例紹介 －企業別取組事例(令和３年度)」](https://neccyusho.mhlw.go.jp/case/r3/#sec01)

### 運輸業

船内やコンテナ内などでの作業は、高温多湿かつ屋内外を行き来することが多く、身体への負担が大きくなります。
  
  
そのため、水分及び塩分の摂取やファン付き作業服での作業といった労働環境の整備が進んでいます。
  
  
また、社内教育を通じて熱中症リスクに関する情報共有を行い、全社員に対策意識を浸透させる企業もあります。
  
  
作業中は、職員及び作業会社の責任者が巡視を行い、作業者の顔色・行動の確認を実施し、安全掲示板を設置して熱中症に関する注意喚起をしています。
  
  

[出典：厚生労働省「熱中症対策事例紹介 －企業別取組事例(令和３年度)」](https://neccyusho.mhlw.go.jp/case/r3/#sec11)

### 警備業

警備業では、ひとり現場や顧客先のルールにしたがう必要があるため、企業独自の熱中症対策が取りづらいという課題があります。
  
  
そこで、原則60歳以上でひとり現場になる場合に空調服の支給を実施し、作業車を休憩スペースとして活用する工夫が導入されました。
  
  
作業車内では、サンシェードを取り付けて断熱効果を高めつつ、車内をクーラーで冷やすといった工夫もされています。
  
  
また、社内に冷凍のスポーツドリンクや熱中症対策の塩分補給飴などで休息ができる環境を整えています。
  
  

[出典：厚生労働省「企業別取組内容(令和２年度)」](https://neccyusho.mhlw.go.jp/case/)

## まとめ

2025年の労働安全衛生規則の改正により、職場における熱中症対策が義務化されました。
  
  
これまで努力義務とされてきた対策が、今後は不履行により罰則を受けるリスクも生じるため、企業は本格的な対策が求められます。

## よくある質問

### WBGT値とは？

WBGT（湿球黒球温度）値とは、気温・湿度・輻射熱（太陽光や地面などによって伝わる熱）・風速などを総合的に評価する暑さの指数です。
  
  
企業はWBGT値を常に把握し、その値に応じた作業制限・休憩時間・冷却対策等を行うことが求められます。
  
  
WBGT値の測定方法については、以下のとおりです。

| 測定方法 | 内容 |
| --- | --- |
| 測定機器 | 市販のWBGT指数計を使用（屋外用・屋内用の仕様を確認） |
| 測定場所 | 作業員が実際に作業する位置で計測 |
| 測定タイミング | 作業開始前・休憩後・日差しが強まる時間帯（特に11〜14時）など複数回実施 |
| 記録方法 | 日付・時刻・場所・数値を記録し、管理者が確認 |

WBGT値を基準とした対策判断の基準は、以下のとおりです。

| 対策判断の基準/th> | 内容 |
| --- | --- |
| 屋外作業の基準 | WBGT値 28度以上 → 強い警戒（こまめな休憩・作業時間短縮） |
| 屋内作業の基準 | WBGT値 25度以上 28度未満 → 警戒（作業環境改善・冷却対策） |
| 基準値超え時の対策 | 作業強度の軽減、時間帯の変更、冷房設備や日陰の設置など即時対策を講じる |

実測された値が基準値を超えるおそれがある場合は、作業の強度・時間・場所を変更するなどの対策が必要とされています。

### 熱中症対策が義務化になった理由は？

今回の労働安全衛生規則改正の背景には、熱中症による労働災害の増加と深刻化があります。
  
  
特に建設業・運送業・製造業など、屋外や高温多湿の環境下での作業が多い業種では、熱中症が原因による死亡災害が発生しています。
  
  
令和6年における職場での熱中症による死傷者（死亡・休業4日以上）は、1,257人（前年比151人・約14％増）であり、全体の約4割が建設業と製造業です。
  
  
そのような背景を受け、2025年6月1日から熱中症対策が義務化となりました。
  
  

[出典：厚生労働省「令和6年「職場における熱中症による死傷災害の発生状況」(確定値)を公表します」](https://www.mhlw.go.jp/stf/newpage_58389.html)
