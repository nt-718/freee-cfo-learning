## リサイクル預託金とは？

**リサイクル預託金とは、車両を購入する際に支払う、「自動車リサイクル料金」を指します。**
  
  
「自動車リサイクル料金」は、リサイクル費用と自動車リサイクルシステムの運営費用で構成されています。具体的な内訳は以下の通りです。

  

|  |  |
| --- | --- |
| リサイクル費用 | ・シュレッダーダスト料金  ・エアバッグ類料金  ・フロン類料金 |
| 自動車リサイクルシステムの運営費用 | ・情報管理料金  ・資金管理料金 |

  

リサイクル預託金は、廃車後の解体によって出る廃棄物の処理に必要な費用を、車両の購入時に所有者が負担します。2005年から施行された自動車リサイクル法で始まった制度です。
  
  
以前は解体業者や破砕業者が負担する廃棄費用が重くなり、不法投棄・不適正処理の問題も起きていました。リサイクルや処理の適正化を図るために制度が整備され、現在は車の所有者も費用を負担する必要があります。
  
  
リサイクル預託金は廃棄費用に使うため、基本的に返金されません。ただし、売却した場合は、買取価格に含めて返金されるケースが多く存在します。なお、売却でも買取業者が廃車処分する廃車買取の場合は、返金されません。

## リサイクル預託金は経費計上できる？

**事業用の車両に支払ったリサイクル預託金は経費にできますが、計上は廃車時です。**
  
  
リサイクル預託金を支払った時点では、将来使う費用に該当するため資産計上し、廃車時に資産から経費に振り替えて計上します。

## リサイクル預託金の仕訳で使う勘定科目

リサイクル預託金の仕訳で使う勘定科目は、次の通りです。

リサイクル預託金の仕訳で使う勘定科目

* 資金管理料以外の費用は【長期前払費用】
* 個別に勘定科目を作る場合は【リサイクル預託金】
* リサイクル料金のうち資金管理料の部分は【支払手数料】

各勘定科目の詳細を、以下で解説します。

### 【長期前払費用】

リサイクル預託金は、将来の廃車時に必要な費用を先に支払う性質から「長期前払費用」での仕訳が適しています。
  
  
長期前払費用とは、前払費用のうち、決算日から1年を超えて資産から費用に振り替える費用です。
  
  
ただし、自動車リサイクル料金のうち「資金管理料」の部分は、「支払手数料」で仕訳します。

### 【リサイクル預託金】

長期前払費用に代わり、「リサイクル預託金」の勘定科目を作って仕訳しても構いません。
  
  
会計システムのなかには、自分で勘定科目を設定できる機能をもったソフトウェアもあります。わかりやすいように勘定科目を設定したい場合は、使用しているソフトウェアの機能を確認しましょう。

### 【支払手数料】

自動車リサイクル料金のうち「資金管理料」の部分は将来使う費用ではないため、「支払手数料」で仕訳します。支払手数料は、銀行の振込手数料や、弁護士・税理士などに対して支払う報酬などを計上する際に使用する勘定科目です。
  
  
料金の内訳は、自動車リサイクル料金を支払ったときに発行される「リサイクル券（預託証明書）」に記載されています。適切に仕訳して計上するため、金額を確認しましょう。
  
  
また、廃車時には長期前払費用（またはリサイクル預託金）で資産計上していた分を、支払手数料に振り替えて経費計上します。

## 【事例で解説】リサイクル預託金の仕訳例

事例を挙げならリサイクル預託金の仕訳例を解説します。リサイクル預託金に関する仕訳で迷う場合は参考にしてください。

### 社用車を購入してリサイクル預託金を支払った場合

新車購入した際に支払うリサイクル預託金は、長期前払費用と支払手数料に仕訳します。
  
  
たとえば社用車の新車購入にともない、自動車リサイクル料金15,000円（うち、資金管理料金は290円）を現金で支払った場合の仕訳は以下の通りです。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 長期前払費用 | 14,710円 | 現金 | 15,000円 |
| 支払手数料 | 290円 |

  

長期前払費用の代わりにリサイクル預託金の勘定科目を設けて仕訳しても構いません。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| リサイクル預託金 | 14,710円 | 現金 | 15,000円 |
| 支払手数料 | 290円 |

  

料金の内訳は、自動車リサイクル料金の支払い時に発行されるリサイクル券に記載されています。各種金額を確認して、適切に仕訳しましょう。

### 社用車を売却してリサイクル預託金が返金された場合

車両を売却し、リサイクル預託金に相当する金額を受け取った場合も仕訳が必要です。
  
  
たとえば社用車を売却したため、買取業者からリサイクル預託金分の金額14,710円が現金で支払われた場合の仕訳は以下の通りです。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 現金 | 14,710円 | 長期前払費用 | 14,710円 |

  

買取業者が提示する見積や料金内訳を確認して、仕訳しましょう。

### 社用車を廃車したためリサイクル預託金を経費計上する場合

社用車を売却ではなく廃車処分した場合、リサイクル預託金はリサイクル費用に使われるため返金されません。資産計上していた長期前払費用（またはリサイクル預託金）を支払手数料に振り替え、経費計上します。
  
  
振り替える際の仕訳は以下の通りです。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 支払手数料 | 14,710円 | 長期前払費用 | 14,710円 |

  

資金管理料金の部分を除いたリサイクル預託金は、廃車して、リサイクルに使われる時点で経費になります。

## リサイクル預託金の仕訳に関する注意点

リサイクル預託金の仕訳では、以下の点に注意しましょう。

リサイクル預託金の仕訳に関する注意点

* 課税区分に注意する
* 仕入税額控除を受けるにはインボイスが必要
* リサイクル券は保管する

以下で詳しく解説します。

### 課税区分に注意する

自動車リサイクル料金は、課税対象と課税対象外に該当する部分があるため、課税区分に注意しましょう。購入時に支払う自動車リサイクル料金のうち、「資金管理料金」の部分は課税仕入です。
  
  
しかし、資金管理料金以外は廃車時の費用を前払いするため課税対象になりません。
  
  
売却時にリサイクル券を譲渡してリサイクル預託金を受け取る場合、受け取った金額は非課税売上に該当します。
  
  
廃車時には車両を廃棄するサービスを受けるため、課税仕入で経費計上しましょう。税率は支払った時点ではなく、廃車した時点の税率を適用します。

### 仕入税額控除を受けるにはインボイスが必要

リサイクル預託金の支払いで仕入税額控除を受けるには、インボイス（適格請求書）が必要です。新車購入時のインボイスは、自動車リサイクルシステムからダウンロードできます。
  
  
インボイス制度の詳細は「[2023年10月から始まったインボイス制度とは？図解でわかりやすく解説！](https://www.freee.co.jp/kb/kb-invoice/about_invoice/)」をご覧ください。

### 仕入税額控除を受けるにはインボイスが必要

自動車リサイクル料金を支払った際に発行されるリサイクル券は預託の証明です。支払った金額や料金の内訳も記載されるため、仕訳にも必要です。
  
  
また、リサイクル券は車両を手放す際にも求められる書類であるため、自動車関連の書類とまとめて保管するとよいでしょう。
  
  
なお、リサイクル券を紛失した場合、原則再発行できません。紛失時は「自転車リサイクル料金の預託状況」を印刷して代用しましょう。

## まとめ

リサイクル預託金は廃車するときに必要な費用で、車両の購入時に支払います。
  
  
事業用の車両に対して支払ったリサイクル預託金は、経費計上が可能です。ただし、支払い時点では、資金管理料金以外の部分を資産に計上します。将来、廃車処分する際、資産から経費に振り替え、売却時には返金を受けるケースが一般的です。
  
  
また、仕訳する際は購入時、売却時、廃車時で課税区分が変わる点にも注意しましょう。
