## 前払金に用いる勘定科目

前払金に用いる勘定科目は、支払う側（買い手）と受け取る側（売り手）で異なります。

前払金に用いる勘定科目

* 前払金を支払ったときは【前払金】【前渡金】
* 前払金を受け取ったときは【前受金】

### 【前払金】【前渡金】

**手付金・内金などの先払いしたお金は、「将来、商品やサービスを受け取る権利」であるため、「前払金（前渡金）」を用いて資産（流動資産）に計上します。**
  
  
「前払金（前渡金）」とは、商品やサービスを受ける前に代金の一部または全部を先払いしたときに用いる勘定科目です。
  
  
商品やサービスの引渡し時に「前払金」を取り崩し、「仕入高」などに振り替える処理を行います。
  
  
「前払金」と「前渡金」は同じ意味をもつため、どちらを使っても問題ありません。仕入れにかかる代金の前払いを「前渡金」、それ以外の代金の前払いを「前払金」と区別して処理することもできます。
  
  
なお、内金は代金を前払いするものです。
  
  
手付金も最終的に代金の一部になりますが、買主はその手付金を放棄し、売主は手付金の倍額を払い返すことで契約を解除できる点が異なります。

### 【前受金】

商品やサービスを販売する際、代金の一部または全部を前もって受け取った場合は、「前受金」の勘定科目を用いて負債（流動負債）に計上します。
  
  
後日、商品やサービスの引渡し時に「前受金」を取り崩し、「売上高」などに振り替える処理が必要です。

## 【事例で解説】前払金の仕訳例

前払金を支払ったとき、受け取ったときの仕訳をいくつか紹介します。

前払金の仕訳例

* 商品を仕入れ、前払金を現金で支払った場合
* ホテルの予約料金を支払った場合
* 事業用の車を購入し、頭金を支払った場合
* 商品を販売し、前払金を受け取った場合

### 商品を仕入れ、前払金を現金で支払った場合

商品20万円を仕入れ、手付金としてA社に5万円を現金で支払った場合の仕訳を紹介します。
  
  
前払金を支払ったときは、資産の増加として次のように処理を行います。
  
  
商品の引渡しやサービスの提供を受けるまでは費用に計上しません。

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 前払金 | 5万円 | 現金 | 5万円 |

  

商品を受け取った際、「仕入高」などに振り替えます。なお、残りの代金は後日支払うこととします。

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 仕入高 | 20万円 | 前払金 | 5万円 |
|  |  | 買掛金 | 15万円 |

  

支払時（買掛金）

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 買掛金 | 15万円 | 現金 | 15万円 |

## 前払金を会計処理する際のポイント・注意点

「前払金」には、混同しやすい勘定科目がいくつか存在します。
  
  
以下のポイント・注意点をおさえて前払金の会計処理を行いましょう。

前払金を会計処理する際のポイント・注意点

* 継続的なサービスへの支払いは「前払費用」に計上する
* 金額や使いみちが決まっていないときは「仮払金」に計上する
* 有形固定資産の手付金は「建設仮勘定」で処理する
* 消費税は商品・サービスを受けたときに課税される

### 継続的なサービスへの支払いは「前払費用」に計上する

継続的なサービスに対する支払いは、「前払金」ではなく「前払費用」で処理しましょう。
  
  
「前払費用」は、まだ提供されていないサービスに対して先払いした費用に用いる勘定科目です。
  
  
「前払費用」は、一時的な商品・サービスに対して使う「前払金」とは違い、継続的に受けるサービスに対する支払いに用います。

前払費用で処理する費用の例

* 火災保険料
* 自動車保険料
* 家賃
* 年間契約している倉庫や機器のレンタル料
* アプリの年間利用料

たとえば、2年契約の火災保険に加入し、火災保険料10万円を一括で支払った場合、次のように仕訳できます。
  
  
支払時

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 保険料 | 5万円 | 普通預金 | 10万円 |
| 前払費用 | 5万円 |  |  |

  

2年目分の保険料は、翌期に前払費用を取り崩して経費に計上します。
  
  
翌期

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 保険料 | 5万円 | 前払費用 | 5万円 |

  

保険期間が2年以上の火災保険料を一括で支払った場合は、上記のように期間に応じて期間按分を行いましょう。

### 金額や使いみちが決まっていないときは「仮払金」に計上する

「仮払金」とは、金額や使いみちが決まっていない場合に概算で支払った金銭に用いる勘定科目です。
  
  
たとえば、交通費や出張旅費の仮払いの際に使用します。
  
  
先払いしたときに使用する点では「前払金」と共通していますが、「仮払金」は金額や使いみちが確定するまでの間、一時的に用いる勘定科目です。
  
  
一方、「前払金」は支払った時点で金額や使いみちが確定しています。混同しないよう気を付けましょう。

### 有形固定資産の手付金は「建設仮勘定」で処理する

有形固定資産の手付金や内金を支払ったときは、「前払金」ではなく「建設仮勘定」の勘定科目を用いて処理します。
  
  
「建設仮勘定」は、建設または製作中の固定資産を処理するための勘定科目です。
  
  
「前払金」は流動資産であるため、固定資産の手付金や内金には使用しません。
  
  
有形固定資産が完成するまでの間、一時的に「建設仮勘定」で処理し、引渡しを受けたときに固定資産勘定への振替処理を行います。
  
  
たとえば、事業用のビルを建設する際、手付金として普通預金から100万円を支払った場合の仕訳は以下の通りです。
  
  
支払時

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 建設仮勘定 | 100万円 | 普通預金 | 100万円 |

  

ビルが完成して引き渡しを受けた際、「建物」に振り替えます。
  
  
引渡し時

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 建物 | 100万円 | 建設仮勘定 | 100万円 |

### 消費税は商品・サービスを受けたときに課税される

消費税は、商品の引渡しやサービスの提供を受けたときに課税されます。
  
  
前払金を支払った時点や、前受金を受け取った時点では消費税がかかりません。
  
  
したがって、支払時・受取時は全額を「前払金」や「前受金」で処理し、実際に商品の引渡しやサービスの提供がなされたときに消費税の処理を行いましょう。

## まとめ

商品やサービスを受ける前に手付金や内金を支払ったときは、「前払金」または「前渡金」の勘定科目で資産に計上しましょう。
  
  
商品の引渡しやサービスの提供を受けた際に「前払金」を取り崩し、「仕入高」などに振り替えます。
  
  
また、前払金を受け取ったときはいったん「前受金」に計上し、引渡し時に「売上高」などで処理します。
  
  
「前払費用」や「仮払金」など、前払金と混同しやすい勘定科目が多いため、違いを理解して正確に仕訳しましょう。
