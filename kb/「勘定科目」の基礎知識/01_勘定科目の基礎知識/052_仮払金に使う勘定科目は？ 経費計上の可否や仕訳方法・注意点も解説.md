## 勘定科目の仮払金とは

仮払金とは、用途や金額が確定していない段階で、会社が仮の金額で一時的に支払うお金のことです。貸借対照表の勘定科目のうち、仮払金は流動資産の中の「その他の流動資産」に分類されます。
  
  
仮払金を支払うケースとしては、従業員が高額なお金を立て替えずに済むよう、事前に企業が従業員に現金を渡すケースが挙げられます。企業が仮払金を支出するケースとは、たとえば以下のようなケースです。

仮払金を支出するケース

* 旅費、出張費を事前に概算払いする場合
* 従業員が取引先の接待をする際に事前にお金を渡す場合
* 従業員が備品の買い出しに行く際に事前にお金を渡す場合

### 前払金や立替金との違い

前払金と立替金は、仮払金と同じく貸借対照表の流動資産に分類される勘定科目です。
  
  
しかし、前払金・立替金と仮払金では拠出したお金の内容が異なるため、仕訳する際には仮払金・前払金・立替金の3つの勘定科目を使い分ける必要があります。

  

| 勘定科目 | 定義 |
| --- | --- |
| 前払金 | ・商品・サービスの受け取りより先に支払った代金の一部または全部  ・支払った時点で目的や金額が明確になっている |
| 立替金 | ・取引先や従業員などが本来負担すべき金銭を会社が一時的に立て替えたもの  ・後日、従業員や取引先から回収する |
| 仮払金 | ・自社が負担すべき費用を一時的に支払ったもの  ・支払時には目的や金額が明確になっていない |

  

前払金とは、代金の一部または全部を商品・サービスの受け取りより先に支払って前払いしたときのお金です。仮払金と違い、前払金の場合は支払った時点で目的や金額が明確になっています。
  
  
立替金とは、取引先や従業員などが本来負担すべき金銭を、会社が一時的に立て替えたときのお金です。仮払金は自社が負担すべき費用であるのに対して、立替金は本来取引先や従業員が負担すべき費用です。立て替えた企業は取引先や従業員から後日回収します。

## 仮払金は経費になる？

仮払金は貸借対照表上の資産に分類される勘定科目です。経費ではありません。仮払金を払った時点ではあくまで仮払いに過ぎず、実際に経費として使われるかどうかは不明です。
  
  
仮払いをしても後日全額が戻ってきて費用がかからない可能性もあるので、仮払いをした時点では支出済の経費にはあたらず、経費計上はできません。
  
  
仮払いで一時的に支払ったお金を管理する勘定科目「仮払金」を使って仕訳します。

## 【事例で解説】仮払金の仕訳例

企業や個人事業主が仮払金を支出した場合、「仮払金を支出した時点」と「実際に費用がかかって用途や金額が確定した時点」の2回、仕訳を行う必要があります。
  
  
消費税に関する仕訳をするのは、仮払金を支出したときではなく、支出したお金の用途や金額が確定して経費として計上するタイミングです。
  
  
仮払金はどのような勘定科目を使って仕訳するのか、以下では具体的な事例を用いて仮払金の仕訳例を紹介します。

### 従業員が出張する際に10万円を渡して7万円かかった場合

勘定科目「仮払金」を使って仕訳するケースとして、「あらかじめ支出した仮払金の金額」よりも「実際にかかった費用の金額」が少ないケースの仕訳を見てみましょう。
  
  
従業員に出張費用として10万円の仮払金を渡して、実際には70,000円（交通費20,000円、宿泊費50,000円）、消費税10%がかかった場合、仕訳では借方・貸方にそれぞれ以下のように記入します。
  
  
仮払金を支出したときの仕訳は以下の通りです。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 仮払金 | 100,000円 | 現金 | 100,000円 |

  

仮払金の用途・金額が確定して精算するときの仕訳は以下の通りです。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 交通費 | 20,000円 | 仮払金 | 100,000円 |
| 宿泊費 | 50,000円 |
| 仮払消費税 | 7,000円 |
| 現金 | 23,000円 |

  

仮払金の精算をする際、仮払金として支出したお金が余って従業員から現金が戻ってきた場合は、借方に勘定科目「現金」を使って返金額を記入します。

### 従業員が出張する際に5万円を渡して6万円かかった場合

勘定科目「仮払金」を使って仕訳するケースとして、「あらかじめ支出した仮払金の金額」よりも「実際にかかった費用の金額」が多いケースの仕訳を見てみましょう。
  
  
従業員に出張費用として5万円の仮払金を渡して、実際には60,000円（交通費15,000円、宿泊費45,000円）、消費税10%がかかった場合、仕訳では借方・貸方にそれぞれ以下のように記入します。
  
  
仮払金を支出したときの仕訳は以下の通りです。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 仮払金 | 50,000円 | 現金 | 50,000円 |

  

仮払金の用途・金額が確定して精算する場合は以下の仕訳をしましょう。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 交通費 | 15,000円 | 仮払金 | 50,000円 |
| 宿泊費 | 45,000円 | 現金 | 16,000円 |
| 仮払消費税 | 6,000円 |  |  |

  

仮払金の精算をする際、当初支出した金額では足りず追加で現金を従業員に支出した場合は、貸方に勘定科目「現金」を使って支出額を記入します。

## 仮払金の仕訳に関する注意点

仮払金の仕訳に関する主な注意点は次の2つです。

仮払金の仕訳に関する注意点

* 支出した仮払金の内容はできるだけ早く確定させる
* 領収書・レシートの提出を義務付けて不正防止に努める

### 支出した仮払金の内容はできるだけ早く確定させる

仮払金はあくまで仮の支出です。仮払金として支出したお金の精算をし忘れることがないよう、支出したお金を使う予定・精算手続きの期日を把握しておきましょう。
  
  
仮払金は、時間が経つほど使用用途や内容の確認が難しくなり、適切な会計処理ができなくなる可能性が高くなります。仮払金を支出した後、支出したお金の内容をできるだけ早く確定させることが大切です。
  
  
交通費や出張費として従業員に仮払金を渡す場合は、実際にお金を使う予定の日がいつなのか確認しましょう。また、実際に費用がかかった日以降にできるだけ早く精算手続きをするよう、従業員にあらかじめ伝えておきましょう。

### 領収書・レシートの提出を義務付けて不正防止に努める

仮払金は、具体的な用途や金額が確定しない中で支出するお金であり、不正が起きることがあります。企業や個人事業主は、仮払金を適切に管理して、不正が起きないように取り組みを行うことが大切です。
  
  
たとえば、従業員の出張にあわせて仮払金を渡すケースでは、交通費や宿泊費の具体的な内容を経理担当者が確認できるよう、領収書・レシートの提出を社内規定で従業員に義務付けることが考えられます。
  
  
また、経理担当者を複数人配置して仮払金の拠出時や精算時に複数人でチェックすれば、不正が疑われるケースの発見がしやすくなります。

## まとめ

仮払金は経費ではなく資産として計上します。経費として計上できるのは、支出したお金の用途や金額など具体的な内容が確定したときです。仮払金を支払った時点では、仮払いに過ぎず経費の金額が確定しているわけではないので経費計上できません。
  
  
**仮払金を支払った場合、勘定科目「仮払金」を使って仕訳します。**仮払金は、流動資産の中の「その他の流動資産」に分類される勘定科目です。
  
  
仮払金を拠出したときには、できるだけ早く内容を確定させるとともに、不正が起きないように領収書やレシートを確認して本当にお金がかかったのか確認してください。
  
  
仮払金に関する仕訳の考え方を正しく理解して、目的やケースに応じて適切に仕訳しましょう。
