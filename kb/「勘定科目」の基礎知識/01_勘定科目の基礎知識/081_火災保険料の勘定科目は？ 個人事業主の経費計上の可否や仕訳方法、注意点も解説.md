## 個人事業主が支払う火災保険料は経費計上できる？

経費にできる費用とは事業に関係する支出です。**個人事業主が事業に関係する火災保険料を支払った場合は、一般的に経費にできます。**逆に事業とは関係ない火災保険料は経費にできないので、自宅の火災保険料は経費計上できません。
  
  
自宅兼事務所の火災保険料を支払った場合は、事務所の保険料に相当する金額のみ経費にできます。自宅兼事務所の火災保険料を経費計上する場合、自宅の火災保険料に相当する金額と事務所の火災保険料に相当する金額を分ける家事按分が必要です。

## 火災保険料の仕訳に使う勘定科目

火災保険料の仕訳に使う勘定科目はケースによって異なります。個人事業主が火災保険料を支払った場合、仕訳で使う主な勘定科目は以下の通りです。

火災保険料の仕訳に使う勘定科目

* 契約期間が1年以内の保険契約に基づく保険料の場合は【保険料】または【損害保険料】
* 保険料を前払いした場合は【前払費用】や【長期前払費用】
* 支払った保険料のうち貯蓄に相当する部分がある場合は【保険積立金】

以下では、火災保険料の仕訳に使う各勘定科目の概要を紹介します。

### 【保険料】【損害保険料】

契約期間が1年以内の保険契約に基づいて個人事業主が火災保険料を支払った場合、「保険料」または「損害保険料」の勘定科目を使って仕訳します。
  
  
火災保険の契約期間が年度をまたぐ場合、各年度に対応した火災保険料を年度ごとに計上することが原則ですが、保険契約期間が1年以内の場合は、継続処理を前提に支払時の費用として一括計上が可能です。
  
  
保険料（損害保険料）とは、企業や個人事業主が支払う保険料のうち、主に損害保険の保険料を支払う際に使う勘定科目です。火災保険・運送保険・盗難保険などに加入して保険料を支払った場合、保険料（損害保険料）で仕訳します。

### 【前払費用】【長期前払費用】

火災保険料を前払いした場合は「前払費用」や「長期前払費用」の勘定科目で仕訳します。
  
  
契約期間が1年を超える場合、支払時に一括して経費計上することはできません。保険料を年度ごとに期間按分して、各年度に対応する保険料を年度ごとに計上する必要があります。
  
  
支払った保険料のうち、翌年度に対応する分は前払費用として計上し、翌々年度以降に対応する分は長期前払費用として計上します。
  
  
長期前払費用とは、前払費用のうち1年を超えて費用となる部分の金額を計上する際に使う勘定科目です。

### 【保険積立金】

火災保険料のうち、貯蓄に相当する部分は勘定科目「保険積立金」を使って仕訳します。
  
  
保険期間が3年以上で、かつ当該保険期間満了後に満期返戻金を支払う旨の定めのある損害保険契約の場合、支払った保険料のうち積立保険料に相当する部分の金額は保険期間の満了まで資産に計上します。また、保険契約の解除もしくは失効のときも同様です。
  
  
保険積立金とは、保険料を支払った際に会計上資産にあたる金額を計上するための勘定科目です。保険料を支払った場合、保険に相当する部分と貯蓄に相当する部分があるときは、以下のように仕訳しましょう。

  

|  | 仕訳する際の取り扱い |
| --- | --- |
| 保険部分の保険料 | 勘定科目「保険料」などを使って費用として計上 |
| 貯蓄部分の保険料 | 勘定科目「保険積立金」を使って資産として計上 |

## 個人事業主が支払う火災保険料の仕訳に関する注意点

個人事業主が火災保険料を支払って経費計上する際の主な注意点は次の2つです。

個人事業主が火災保険料を仕訳する際の注意点

* 自宅兼事務所では家事按分の根拠を明確にする
* 契約期間が事業年度をまたぐ場合は年度ごとに期間按分する

火災保険料に関連する支出をする場合、経費にできる支出なのかや使う勘定科目が何か、間違えることなく適切に経理処理を行う必要があります。

### 自宅兼事務所では家事按分の根拠を明確にする

火災保険料のうち経費にできるのは事業に関係する金額だけです。個人事業主の事務所が自宅も兼ねている、いわゆる自宅兼事務所の場合、自宅の火災保険料に相当する金額は経費にできません。
  
  
火災保険料のうち事業に関係する金額を計算する家事按分が必要です。
  
  
家事按分の計算方法には、事業用として使用している面積で按分する方法や、使用した時間で按分する方法があります。税務調査が入った際などに家事按分の根拠を示せないと経費として認められない可能性があるので、家事按分の根拠は明確にすることが大切です。
  
  
建物に占める仕事部屋の面積割合で家事按分する場合には自宅兼事務所の図面を保管しておくなど、家事按分の根拠となる資料を保管しておきましょう。

### 契約期間が事業年度をまたぐ場合は年度ごとに期間按分する

保険契約期間が事業年度をまたぐ場合、期間案分の考え方に基づいて各事業年度に案分することが原則です。
  
  
契約期間が1年以内の場合は、継続処理を前提に支払時の費用とすることも認められていますが、保険契約期間が1年超であれば、当年度・翌年度・翌々年度以降など、年度ごとに保険料を期間按分して各年度に対応する保険料を計上します。

## 【事例で解説】火災保険料の仕訳例

個人事業主が火災保険料を支払った場合、どのような勘定科目を使って仕訳するのか、具体的な事例を用いて仕訳例を紹介します。

### 1年契約の火災保険料を一括で支払った場合

契約期間1年の火災保険契約を結び、保険料15,000円を口座振替で一括で支払って全額を当年度の費用とする場合、仕訳例は以下の通りです。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 保険料 | 150,000 | 普通預金 | 150,000 |

  

契約期間1年（2024年9月～2025年8月）の火災保険に加入して保険料150,000円を支払い、年度末の12月までの保険料を当年度の費用、1月以降の保険料を翌年度の費用として計上する場合は、以下のように仕訳します。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 保険料 | 37,500 | 普通預金 | 150,000 |
| 前払費用 | 112,500 |

### 自宅兼事務所で1年契約の火災保険料を一括で支払った場合

自宅兼事務所で活動する個人事業主が火災保険料を支払った場合は、家事按分が必要です。
  
  
自宅兼事務所で事務所割合を30%とする場合、1年契約の火災保険料（月額保険料15,000円）を一括で支払ったときの仕訳は以下の通りです。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 保険料 | 105,000 | 普通預金 | 150,000 |
| 事業主貸 | 45,000 |

### 3年契約の火災保険料を一括で支払った場合

火災保険の契約期間が3年の場合、当年度分は「保険料」または「損害保険料」で仕訳し、翌年度以降の部分は「前払費用」と「長期前払費用」で仕訳します。
  
  
契約期間が2024年1月から2026年12月までの3年間で、3年分の保険料360,000円を一括で振り込むと仕訳例は以下の通りです。
  
  
保険料を当年度分120,000円・翌年度分120,000円・翌々年度分120,000円に分けて、それぞれに勘定科目を設定して仕訳します。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 保険料 | 120,000 | 普通預金 | 360,000 |
| 前払費用 | 120,000 |
| 長期前払費用 | 120,000 |

### 積立型の火災保険を契約して保険料を支払った場合

個人事業主が積立型の火災保険を契約した場合、貯蓄部分があれば「保険積立金」の勘定科目で処理する必要があります。
  
  
積立型の火災保険で1年分の保険料300,000円（うち積立部分100,000円）を支払い、経費にあたる金額は支払時の費用として一括計上する場合、以下のように仕訳しましょう。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 保険料 | 200,000 | 普通預金 | 300,000 |
| 保険積立金 | 100,000 |

## まとめ

火災保険料は、事業に関係がある場合に経費にできます。**仕訳をする際に用いる勘定科目は、保険契約期間が1年以内であれば一般的に保険料や損害保険料です。**
  
  
保険契約期間が1年超の場合は前払費用や長期前払費用などの勘定科目を使って仕訳し、貯蓄に相当する部分がある場合は勘定科目として保険積立金を使います。
  
  
火災保険料の仕訳の考え方や使う勘定科目を正しく理解して、適切に仕訳を行うことが大切です。

父も祖父も税理士という家系に長女として生まれる。実家は60年続く税理士事務所。学生結婚し、子供を授かるも、母、妻、娘の役割以外に、自分の人生も生きていきたいと２人の子供を育てながら税理士試験に合格する。自身も経営者の立場を経験しない事には、お客様の気持ちに真に寄り添うことはできないと感じ、実家の税理士事務所とは別に2021年に西村税理士事務所を開業。現在は、女性起業支援を中心に活動している。
