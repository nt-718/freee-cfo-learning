## 送料に用いる勘定科目

**送料は、目的に応じて勘定科目が変わる支出です。**同じ郵便や宅急便で送っても、送料が発生した状況によって用いる勘定科目・仕訳方法が異なります。
  
  
状況別に、送料に用いる勘定科目を解説します。

送料に用いる勘定科目

* 商品や製品を発送する際にかかった費用を処理する場合は【荷造運賃】
* 商品を販売し、顧客が負担する送料を立て替えた場合は【立替金】
* 商品を仕入れた際に送料を負担する場合は【仕入高】
* 10万円以上の固定資産を購入した際に負担した場合は【固定資産】
* 取引先や支店、従業員に書類や備品を発送する場合は【通信費】
* ECサイトで紙や文房具などの消耗品を購入した場合は【消耗品費】
* お歳暮やお中元を贈る場合は【交際費】

### 荷造運賃

商品や製品を発送する際にかかった費用を処理するときは、「荷造運賃」の勘定科目を用います。荷造運賃は、仕入れる側ではなく「販売する側」が使用する勘定科目です。
  
  
郵送代だけでなく、包装紙やガムテープなどの梱包にかかった費用も荷造運賃に含めて処理します。

### 立替金

商品を販売し、顧客が負担する送料を立て替えたときは、「立替金」を用いて資産に計上します。立替金とは、取引先や従業員が支払うべきお金を一時的に立て替えたときに使用する勘定科目です。
  
  
商品を販売した際に発生する送料は、自社が負担するのか、顧客が負担するのかで勘定科目が異なるため、正しく理解しておきましょう。

### 仕入高

商品を仕入れた際に送料を負担するときは、「仕入高」の勘定科目を使用し、仕入れた商品の本体価格に含めて処理します。
  
  
商品販売時に負担した送料は、本体価格とは別に「荷造運賃」で処理しますが、仕入れ時は取得原価に含める点に注意が必要です。

### 固定資産

10万円以上の固定資産を購入した際に負担した送料は、「固定資産」の勘定科目を使用し、取得原価に含めて処理します。
  
  
固定資産とは、事業で長期的に保有・使用する資産のことで、以下のような資産があります。

|  |  |
| --- | --- |
| **固定資産** | **具体例** |
| 工具器具備品 | パソコン、カメラ、事務机、コピー機、エアコン |
| 機械装置 | ブルドーザー、パワーショベル、コンベア |
| 付属設備 | ボイラー設備、店舗内装、ガス設備、電気設備、照明設備 |
| 医療用器械備品 | 血圧計、内視鏡、電子温度計 |
| 農機具 | トラクター、田植機、精米機 |

### 通信費

取引先や支店、従業員に書類や備品を発送する際の発送費用は、「通信費」を用いて処理します。
  
  
通信費とは、電話・郵便・テレビ・インターネットなどの通信にかかった費用を処理するための勘定科目です。
  
  
宅配便などを利用して物品を送る場合、商品を発送する際は「荷造運賃」、書類や備品を送る際は「通信費」を用います。
  
  
荷造運賃と通信費の違いを理解し、正しく使い分けましょう。

### 消耗品費

ECサイトを利用し、紙や文房具などの消耗品を購入した際にかかった送料に用いる勘定科目は、「消耗品費」です。
  
  
消耗品を取得するために要した費用として、購入した消耗品の費用に含めて処理します。
  
  
消耗品費とは、消耗品や、耐用年数が1年未満のもの・取得価額が10万円または30万円未満の少額減価償却資産を購入した際に使用する勘定科目です。

### 交際費

お歳暮やお中元を贈るときの送料に用いる勘定科目は、「交際費」です。お歳暮やお中元を贈るために要した費用として、購入した贈答品の費用に含めて処理します。
  
  
なお、交際費とは、得意先や仕入先に対する接待・供応・慰安・贈答・その他に類する行為にかかった支出を処理するための勘定科目です。

## 【事例で解説】送料の仕訳例

送料の仕訳方法は物品の内容などによって異なります。状況に応じて使い分けましょう。
  
  
送料を仕訳する際の具体例をいくつかのケースに分けて紹介します。なお、消費税については、この記述では考慮しておりません。

送料の仕訳例

* 商品を販売した際にかかった送料を自社で負担した場合
* 商品を販売した際に顧客負担の送料を立て替えた場合
* 商品を仕入れた際に送料を負担した場合
* 15万円のパソコンを購入し、送料を負担した場合
* 取引先に簡易書留で書類を送付した場合
* 社内で使用するボールペンを購入した場合

### 商品を販売した際にかかった送料を自社で負担した場合の仕訳例

10,000円の商品を販売し、宅急便の送料940円を自社で負担した場合は、「荷造運賃」を用いて以下のように仕訳します。

  

|  |  |  |  |
| --- | --- | --- | --- |
| **借方** | | **貸方** | |
| **売掛金** | 10,000円 | **売上** | 10,000円 |
| **荷造運賃** | 940円 | **現金** | 940円 |

### 商品を販売した際に顧客負担の送料を立て替えた場合の仕訳例

10,000円の商品を販売し、顧客負担の送料940円を立て替えたときの仕訳は、以下の通りです。

  

|  |  |  |  |
| --- | --- | --- | --- |
| **借方** | | **貸方** | |
| **売掛金** | 10,000円 | **売上** | 10,000円 |
| **立替金** | 940円 | **現金** | 940円 |

  

顧客から販売代金と送料を回収した際に、以下のような仕訳を行います。

  

|  |  |  |  |
| --- | --- | --- | --- |
| **借方** | | **貸方** | |
| **普通預金** | 10,940円 | **売掛金** | 10,000円 |
|  |  | **立替金** | 940円 |

### 商品を仕入れた際に送料を負担した場合の仕訳例

5,000円の商品を仕入れた際に送料500円を負担したときは、仕入れた商品の本体価格に送料を含め、以下のように仕訳します。

  

|  |  |  |  |
| --- | --- | --- | --- |
| **借方** | | **貸方** | |
| **仕入高** | 5,500円 | **現金** | 5,500円 |

### 15万円のパソコンを購入し、送料を負担した場合の仕訳例

次に、10万円以上の固定資産を購入した際に送料を負担したケースです。15万円のパソコンを購入し、送料2,500円を支払った場合、以下のように仕訳します。

  

|  |  |  |  |
| --- | --- | --- | --- |
| **借方** | | **貸方** | |
| **工具器具備品** | 152,500円 | **現金** | 152,500円 |

### 取引先に簡易書留で書類を送付した場合の仕訳例

取引先に書類を送付し、郵便代434円を支払ったときは、「荷造運賃」ではなく「通信費」で仕訳します。

  

|  |  |  |  |
| --- | --- | --- | --- |
| **借方** | | **貸方** | |
| **通信費** | 434円 | **現金** | 434円 |

### 社内で使用するボールペンを購入した場合の仕訳例

社内で使用するボールペンを2,000円分購入し、370円の送料を支払ったときは、ボールペンの本体価格に含めて以下のように仕訳します。

  

|  |  |  |  |
| --- | --- | --- | --- |
| **借方** | | **貸方** | |
| **消耗品費** | 2,370円 | **現金** | 2,370円 |

## 送料を経費に計上する際のポイント・注意点

送料を払ったときの仕訳・会計処理に関して、以下のポイント・注意点をおさえておきましょう。

送料を経費に計上する際のポイント・注意点

* 購入時にかかる送料は取得原価に含める
* 一度決めた勘定科目は変更しない
* 荷造運賃の未使用分は貯蔵品に振り替える

### 購入時にかかる送料は取得原価に含める

基本的に、物品を購入した際に発生した送料は、購入するために要した付随費用として取得原価に含めます。つまり、購入した物品の勘定科目にあわせて処理をします。
  
  
送料単体で「荷造運賃」を用いて処理するのは、商品を販売する際に送料を負担したときです。

### 一度決めた勘定科目は変更しない

勘定科目に法律上の明確なルールはないため、自由に設定できます。ただし、基本的に変更はできません。
  
  
一度決めた勘定科目は、「継続性の原則」にもとづき、毎期継続する必要があります。正当な理由なく変更することはできないため注意しましょう。

### 荷造運賃の未使用分は貯蔵品に振り替える

商品を発送するための段ボールやガムテープをまとめ買いしている場合は、原則として決算時に未使用分を「荷造運賃」から「貯蔵品」に振り替える処理が必要です。
  
  
たとえば、商品発送用の段ボールを10,000円分まとめて購入し、決算時に1,000円分が未使用のまま残った場合、購入時・決算時にそれぞれ以下のような仕訳を行います。
  
  
＜購入時＞

|  |  |  |  |
| --- | --- | --- | --- |
| **借方** | | **貸方** | |
| **荷造運賃** | 10,000円 | **現金** | 10,000円 |

  

＜決算時＞

|  |  |  |  |
| --- | --- | --- | --- |
| **借方** | | **貸方** | |
| **貯蔵品** | 1,000円 | **荷造運賃** | 1,000円 |

## まとめ

事業を営んでいると、さまざまな場面で送料が発生します。
  
  
送料に用いる勘定科目には「荷造運賃」や「通信費」などがあり、勘定科目・仕訳方法は状況によって異なります。
  
  
商品販売時にかかった送料は「荷造運賃」を用いて処理します。一方、購入時にかかった送料は原則として取得原価に含めるため、注意が必要です。
  
  
たとえば、商品を仕入れた際は、商品の本体価格に含めて「仕入高」で仕訳します。
  
  
送料に用いる勘定科目の違いを正しく理解したうえで、明確なルールを決めて適切に仕訳しましょう。
