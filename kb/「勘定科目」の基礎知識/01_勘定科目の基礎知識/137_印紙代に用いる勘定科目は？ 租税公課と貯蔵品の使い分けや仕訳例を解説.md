## 収入印紙代とは？

印紙税などを国に納める際に使用する証票を「収入印紙」といいます。
  
  
印紙税とは、契約書や領収書などの経済取引に伴う書類に課される税金です。印紙税が課せられる課税文書に収入印紙を貼付し、消印をすることで納付します。
  
  
印紙税が課税されるのは、印紙税法で定められている20種類の文書です。また、印紙税額は、課税文書に記載された金額に応じて決まります。
  
  
収入印紙について、詳しくは「[収入印紙とは？購入方法や正しい貼り方、注意点などを解説](https://www.freee.co.jp/kb/kb-kakuteishinkoku/about_revenue_stamp/)」の記事をご覧ください。

## 収入印紙代に用いる勘定科目

収入印紙代を払ったときに使用する勘定科目には、「租税公課」と「貯蔵品」の2つがあります。

収入印紙代の勘定科目・仕訳例

* 収入印紙を使ったときの勘定科目は【租税公課】
* 収入印紙を買い置きするときの勘定科目は【貯蔵品】

どのようなときに用いるのかを理解し、正しく使い分けましょう。

### 租税公課

収入印紙代（印紙税の納付額）は、経費として計上できます。収入印紙を購入してすぐに使用した場合は、「租税公課」の勘定科目を用いるのが一般的です。
  
  
租税公課とは、国や地方自治体、公共団体に納める費用を計上する勘定科目です。国や地方税などの「租税」、公共団体に納める会費や交付金である「公課」を合わせたものを指します。

### 貯蔵品

収入印紙をまとめて購入し、未使用のまま保管しておく場合は、「貯蔵品」の勘定科目を使用します。
  
  
貯蔵品とは、決算日の時点で未使用の事務用品や消耗品を資産として計上するための勘定科目です。収入印紙や切手・文房具・パンフレット・包装材料などが該当します。
  
  
収入印紙を買い置きする場合の仕訳方法は、以下の2種類です。

収入印紙を買い置きする場合の仕訳方法は2種類

* 購入時に「貯蔵品」で仕訳
* 決算時に「貯蔵品」で仕訳

本来は、購入時に「貯蔵品」で処理し、使用時に貯蔵品から振り替えます。
  
  
しかし、処理が複雑になるため、実際には購入時にまとめて経費として計上します。計上後、決算の際に未使用分を「貯蔵品」に振り替える方法が一般的です。

## 【事例で解説】収入印紙代の仕訳例

購入後すぐに使用する場合と買い置きする場合に分けて、収入印紙の購入時・決算時の仕訳例を紹介します。

* 収入印紙を使った場合の仕訳例
* 収入印紙を買い置きする場合の仕訳例

### 収入印紙を使った場合の仕訳例

郵便局で1万円の収入印紙を現金で購入し、すぐに使ったときは、「租税公課」を用いて以下のように経費に計上します。

  

|  |  |  |  |
| --- | --- | --- | --- |
| **借方** | | **貸方** | |
| **租税公課** | 10,000円 | **現金** | 10,000円 |

### 収入印紙を買い置きする場合の仕訳例

収入印紙をまとめて購入し、未使用のまま置いておくときは、「貯蔵品」を用いて資産に計上します。
  
  
4万円の収入印紙を現金でまとめて購入し、200円の収入印紙を使用した場合、購入時・使用時の仕訳は以下の通りです。
  
  
＜購入時＞

|  |  |  |  |
| --- | --- | --- | --- |
| **借方** | | **貸方** | |

＜使用時＞

|  |  |  |  |
| --- | --- | --- | --- |
| **借方** | | **貸方** | |
| **租税公課** | 200円 | **貯蔵品** | 200円 |

  

また、購入時に「租税公課」で費用処理をし、決算時に「貯蔵品」に振り替える方法もあります。
  
  
たとえば、4万円の収入印紙を現金でまとめて購入し、決算時に400円分が未使用で残っていた場合の仕訳は、以下の通りです。

＜決算時＞

|  |  |  |  |
| --- | --- | --- | --- |
| **借方** | | **貸方** | |

## 収入印紙代の仕訳・会計処理に関するポイント・注意点

収入印紙代の仕訳・会計処理をする際、以下のポイント・注意点をおさえましょう。

収入印紙代の会計処理に関するポイント・注意点

* 金券ショップで購入した場合は課税取引として消費税の処理が必要となる
* 補助科目を設定する
* 電子契約なら印紙税は課税されない

### 金券ショップで購入した場合は課税取引として消費税の処理が必要となる

収入印紙の購入は「非課税取引」であり、原則として消費税はかかりません。
  
  
ただし、非課税となるのは、郵便局やコンビニなどの便切手類販売所および印紙売りさばき所（〒マークの看板があるお店）で購入した場合に限ります。
  
  
金券ショップで購入した場合は課税取引となるため、消費税（8%）を考慮した仕訳が必要です。
  
  
たとえば、金券ショップで1万円分の収入印紙を9,800円で購入し、現金で支払った場合は、以下のように仕訳を行います（税抜方式を採用している場合）。

  

|  |  |  |  |
| --- | --- | --- | --- |
| **借方** | | **貸方** | |
| **租税公課** | 9,074円 | **現金** | 9,800円 |
| **仮払消費税** | 726円 |  |  |

### 補助科目を設定する

補助科目の設定は必須ではありませんが、設定しておくと管理や集計が容易になります。補助科目は、勘定科目の内訳をより詳細に把握するために使用します。
  
  
印紙、消費税、固定資産税、会費などの補助科目を設定し、経理の負担を軽減しましょう。

### 電子契約なら印紙税は課税されない

印紙税が課されるのは、書面で作成された課税文書であり、電子データで作成された契約書や領収書には課税されません。
  
  
つまり、電子データで作成した場合、収入印紙に関する会計処理は不要です。
  
  
電子契約は、経理事務の負担軽減や収入印紙代の削減につながります。業務効率化やコスト削減のために、電子契約の導入も検討しましょう。

## まとめ

印紙税を納めるための収入印紙代は、「租税公課」の勘定科目で経費に計上できる支出です。
  
  
収入印紙を購入してすぐに使用する場合は、「租税公課」の勘定科目を使用します。
  
  
一方、購入した収入印紙を保管する場合は、購入時に「租税公課」で計上し、未使用分は決算時に「貯蔵品」で処理するのが通常です。
  
  
郵便局などで収入印紙を購入すれば消費税はかかりません。しかし、金券ショップで購入した場合は、消費税も考慮して処理する必要があります。
  
  
「租税公課」と「貯蔵品」の使い分けを理解し、収入印紙代の会計処理を正しく行いましょう。
