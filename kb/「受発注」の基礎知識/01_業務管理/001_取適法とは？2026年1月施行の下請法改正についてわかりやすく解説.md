取適法（とりてきほう）とは、2026年1月1日に施行される、「中小受託取引適正化法」の略称のことです。現行の「下請法」を改正して作られており、事業者間の業務委託取引における中小事業者の利益を保護し、取引の適正化を図るための新しい法律になります。

## 取適法（とりてきほう）とは？

**取適法（とりてきほう）とは、2026年1月1日に施行される「中小受託取引適正化法」の略称のこと**です。正式名称を「中小受託取引適正化法」といい、事業者間の業務委託取引における中小事業者の利益を保護し、取引の適正化を図るための新しい法律として、現行の下請法を改正して作られました。
  
  
下請法では、発注者（親事業者）と受注者（下請事業者）の資本金規模によって適用対象が限定されています。そのため、発注者の資本金が小さい場合や、受注者が個人事業主（フリーランス）である場合など、法律の保護を受けられないケースが多く存在していました。
  
  
取適法ではこの「保護の穴」を埋めるため、資本金にかかわらず、すべての発注者が規制対象となります。**フリーランスを含む幅広い中小事業者を、受注者として保護する画期的な内容となっている**のです。
  
  
取適法の施行日は、2026年1月1日です。この施行日に向けて、今後は具体的な運用ルールを定めた政令や規則、詳細な解釈を示すガイドラインなどが公表される予定となっています。企業はこれらの情報にも注意を払い、施行日までに万全の準備を整えることが必要です。

## 取適法の目的

取適法の最大の目的は、**立場の弱い中小事業者が発注者から不当な不利益を受けることを防ぎ、公正な取引環境を整備すること**です。
  
  
具体的には、以下のような課題の解決を目指しています。

| 解決を目指す課題 | 具体的な目的 |
| --- | --- |
| 支払遅延の防止 | 成果物を受け取ったにもかかわらず、代金の支払いが遅れる問題をなくす |
| 不当な減額・買いたたきの禁止 | 一方的な理由で報酬を減額したり、不当に低い価格での発注を強いたりすることを禁じる |
| 契約内容の明確化 | 「言った・言わない」のトラブルを防ぐため、発注内容や報酬、納期などを書面で明確にすることを義務付ける |
| フリーランスなど多様な働き手の保護 | 組織に属さず個人で働くフリーランスなども、安心して取引できる環境を整える |

  

これらの目的の達成によって、中小事業者が安心して事業に集中できる環境を創出し、日本経済全体の活性化につなげることが期待されています。

## 取適法で何が変わる？下請法からの主要な改正点

取適法は、現代の取引実態に合わせて保護の範囲を大幅に広げ、新たなルールを追加する実質的な下請法のアップグレードといえます。
  
  
ここでは、主要な3つの改正点を解説します。

### 法律名・用語の変更

下請法から取適法に改正するにあたって、具体的には以下のように**法律名や用語が変更となっています。**

| 改正前 | 改正後 |
| --- | --- |
| 下請代金支払遅延等防止法 | 製造委託等に係る中小事業者に対する代金の支払の遅延等の防止に関する法律 |
| 下請代金 | 製造委託等代金 |
| 親事業者 | 委託事業者 |
| 下請事業者 | 中小受託事業者 |

### 適用対象の拡大

今回の改正により、対象となる取引および事業者の対象が拡大しました。

#### 対象となる取引範囲の拡大

下請法で対象となる取引は、以下の4類型の取引に限定されていました。

* 製造委託
* 修理委託
* 情報成果物作成委託
* 役務提供委託

取適法ではこの垣根が取り払われ、これら4類型に加えて「**特定運送委託」と「金型以外の型・治具等の製造委託」も対象**です。
  
  
特定運送委託においては、荷待ちや荷役強要といった物流業界の不適切な取引慣行も適用されます。金型以外の型・治具等の製造委託については、木型や樹脂型なども含まれます。特定製品専用の型を無償で使用させたり、保管したりする行為に対して、適正な対価の支払いを促すのが目的です。

  

[出典：公正取引委員会「取適法リーフレット」](https://www.jftc.go.jp/file/toriteki_leaflet.pdf)

#### 対象となる事業者の拡大

これまで下請法では、発注側（親事業者）と受注側（下請事業者）双方の資本金規模によって適用対象が限定されていました。そこで**取適法では、取引適用の基準として従業員（常時使用する）数についても加わります。**資本金基準または従業員基準のいずれかに該当すれば、適用対象となります。
  
  
具体的には、以下のとおりです。
  
  
**製造委託・修理委託・特定運送委託・情報成果物作成委託・役務提供委託※**

| 発注側（委託事業者） | 受託側（中小受託事業者） |
| --- | --- |
| ・資本金3億円超  ・資本金1,000万円超3億円以下  ・常時使用する従業員300人超 | ・資本金3億円以下の法人  ・資本金1,000万円以下  ・常時使用する従業員数が300人以下 ※個人含む |

※情報成果物作成委託・役務提供委託については、プログラム作成、運送、物品の倉庫における保管、情報処理に限る
  
  
**情報成果物作成委託・役務提供委託※**

| 発注側（委託事業者） | 受託側（中小受託事業者） |
| --- | --- |
| ・資本金5,000万円超  ・資本金1,000万円超5,000万円以下  ・常時使用する従業員100人超 | ・資本金5,000万円以下  ・資本金1,000万円以下  ・常時使用する従業員100人以下 ※個人含む |

※情報成果物作成委託・役務提供委託については、プログラム作成、運送、物品の倉庫における保管、情報処理を除く

  

[出典：公正取引委員会「中小受託取引適正化法ガイドブック」](https://www.jftc.go.jp/file/toriteki002.pdf)
  
[出典：公正取引委員会「取適法リーフレット」](https://www.jftc.go.jp/file/toriteki_leaflet.pdf)
  
[出典：公正取引委員会「改正法条文」](https://www.jftc.go.jp/partnership_package/toritekihou-jyobun.pdf)
  

これにより、これまで保護が手薄だったスタートアップ企業やフリーランスも、法律によって守られるようになります。

### 新たな禁止行為の追加

従来の11の禁止行為に加え、今日の経済情勢を反映した、より実効性の高い保護規定が新たに追加されます。

#### 協議に応じない一方的な代金決定の禁止

労務費や原材料価格、エネルギーコストなどが高騰したことを理由に、**受注側が取引価格の引き上げを求めて協議を申し入れた場合、発注側は正当な理由なくこの協議を拒否することができなくなります。**  
  
これは「値上げに必ず応じる義務」ではありませんが、話し合いのテーブルに着くことを義務付けるものです。これにより、コストが上がっているにもかかわらず、発注側の優越的な地位によって価格交渉の機会さえ与えられない、といった事態を防ぐ狙いがあります。

#### 手形払等の禁止

政府が進める「2026年度末の手形廃止」の方針を後押しする形で、**手形による支払いが禁止されることになります。**
  
  
また、そのほかの支払手段（電子記録債権や一括決済方式など）についても、支払期日までに代金相当額満額を得ることが困難であるといった観点から、禁止されます。これにより、受注側の資金繰りの安定化を図ります。

## 取適法で発注者に課される4つの義務

取適法では、発注者である「委託事業者」に対して、下請法と同様に義務を定めています。発注側になるすべての事業者は、これらのルールを正確に理解し、遵守しなければなりません。

### 書類などの作成・保存義務

委託した業務内容や支払い状況など、**取引に関する一連の記録を作成し、2年間保存する義務があります。**

### 発注内容などの明示義務

発注する際は、**委託内容、報酬額、納期、支払期日といった契約の重要事項を記載した書面（または政令で定める電磁的方法による記録）を直ちに交付しなければなりません。**  
  
口頭での発注や、曖昧な内容での依頼は認められなくなります。

### 支払期日を定める義務

**報酬の支払期日は、成果物を受領した日（役務提供の場合は役務の提供を受けた日）から起算して60日以内のできる限り短い期間内で定めなければなりません。**

### 遅延利息の支払義務

定めた支払期日までに報酬を支払わなかった場合、**受領日から60日を経過した日から実際に支払う日までの日数に応じ、年率14.6%の遅延利息を支払わなければなりません。**

## 取適法で発注者（委託事業者）に禁止される11の行為

下請法と同様に、取適法では発注者の優越的な地位の濫用を防ぐため、以下の11項目が禁止されます。

  

| 禁止行為 | 内容 |
| --- | --- |
| 受領拒否 | 発注した成果物の受領を、受注者側の責任なく拒むこと |
| 代金の支払遅延 | 定めた支払期日までに代金を支払わないこと |
| 代金の減額 | 発注時に決めた代金を、受注者側の責任なく減額すること |
| 受領後の返品 | 受領した成果物を、受注者側の責任なく返品すること |
| 買いたたき | 市場価格等に比べて、著しく低い代金を不当に定めること |
| 購入・利用強制 | 発注者が指定する商品やサービスを、正当な理由なく強制して購入・利用させること |
| 報復措置 | 受注者が公正取引委員会などに違反行為を知らせたことを理由に、取引を減らしたり停止したりすること |
| 有償支給原材料等の対価の早期決済 | 有償で支給した原材料の代金を、製品の代金支払期日より早く相殺したり支払わせたりすること |
| 不当な経済上の利益の提供要請 | 受注者に対し、協賛金や作業の手伝いなど、発注内容とは別の不当な経済上の利益を提供させること |
| 不当な給付内容の変更・やり直し | 受注者に責任がないにもかかわらず、発注の取り消しや内容の変更、無償でやり直しを強要すること |
| 協議に応じない一方的な代金決定 | 協議に応じなかったり、必要な説明を行わなかったりと、一方的に代金を決定すること |

[出典：公正取引委員会「取適法リーフレット」](https://www.jftc.go.jp/file/toriteki_leaflet.pdf)

## 取適法施行で事業者が今から準備すべきこと

取適法の施行は取引のあり方そのものに関わる大きな変更であり、準備には時間がかかります。すべての事業者が、発注者・受注者の両方の視点から、今すぐ準備を始めるべきといえます。
  
  
具体的には、以下のステップで準備を進めましょう。

### 1. 自社の取引関係の総点検

まずは、**自社が行っているすべての業務委託取引を洗い出し、発注者（委託事業者）に該当する取引と、受注者（中小受託事業者）」に該当する取引を整理**します。
  
  
とくに、これまで下請法の対象外だった取引先（資本金1,000万円以下の法人や個人事業主への発注）を正確に把握することが重要です。

### 2. 契約書・発注書フォーマットの見直し

取適法が定める「書面の作成・保存義務」に対応するため、**現在の契約書や発注書のフォーマットを見直します。**

以下の項目が漏れなく記載されているか確認し、必要であればテンプレートを改訂します。

* 委託事業者および受託事業者の名称
* 委託した日
* 委託する業務の具体的な内容
* 成果物の納期（または役務提供の期日）
* 成果物の受領場所
* 報酬（代金）の額と算定方法
* 支払期日
* （検査を行う場合）その基準や実施時期

弁護士や行政書士などの専門家にリーガルチェックを依頼することも有効です。

### 3. 社内規程の整備と担当者への周知徹底

発注から検収、支払いまでの業務フローを見直し、取適法を遵守できる体制を構築します。とくに**支払期日（60日以内）が守られるよう、経理部門との連携を強化する必要があります。**  
  
また、**購買担当者や業務を委託する可能性のあるすべての従業員に対して、取適法に関する研修会を実施することも有効です。**法律の内容や自社の新しいルール、違反した場合のリスクなどを周知徹底させることが重要となります。

### 4. 下請法からの移行準備

現在、下請法を遵守して運用している事業者も、取適法へのスムーズな移行準備が必要となります。
  
  
保護対象が広がることで、管理すべき取引が増加する可能性があります。**管理体制やシステムが対応可能か、今のうちから検討することが肝心**です。

## 取適法に違反した場合のペナルティ

取適法に違反した場合、発注者には厳しい措置が取られます。
  
  
**公正取引委員会または中小企業庁は、違反の疑いがある事業者に対して調査を行い、指導や助言を行います。**それでも改善が見られない場合は勧告が行われ、その事実が公表される可能性があることに注意が必要です。
  
  
**勧告に従わない場合や検査を拒んだ場合などには、罰金50万円以下が科されることもあります。**
  
  
金額以上に大きいのが、違反企業として社名が公表されることによるレピュテーション（評判）リスクです。事業者の社会的信用が失墜し、取引先や消費者からの信頼を失うことにつながりかねません。

## まとめ

取適法（とりてきほう）は、単なる下請法の改正法ではなく、日本の取引慣行をより公正で透明性の高いものへと変革する大きな一歩です。発注者は資本金に関わらず規制対象となり、受注者の保護範囲が大幅に拡大されます。発注者には、書面の交付義務や60日以内の支払いなどが厳格に義務付けられることが特徴です。
  
  
すべての事業者がこの新しいルールを正しく理解し、誠実に対応していくことが、自社の持続的な成長と社会全体の発展につながるでしょう。
