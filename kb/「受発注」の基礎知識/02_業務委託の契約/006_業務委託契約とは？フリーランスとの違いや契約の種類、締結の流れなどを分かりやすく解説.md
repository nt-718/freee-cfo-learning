## 業務委託契約とは

**業務委託契約とは、企業や組織が行っている業務の一部を外部の企業や個人に委託するときに結ぶ契約のことを指します。**業務委託契約の特徴は、受託者側は単に労働力を提供するのではなく、「業務に対する成果」を提供する点にあります。
  
  
受託者側は一般的な雇用契約とは異なり、依頼者と対等な立場で委託された業務を行います。したがって、業務の進め方や時間的な制約に縛られず、自身の裁量で業務を遂行していきます。
  
  
依頼者は、業務の完遂もしくは成果物の完成をもって対価となる報酬を受託者側へ支払い、契約関係は終了します。契約内容によっては、中長期にわたり継続して業務を委託することも可能です。

### 雇用契約との違い

**雇用契約とは、企業などの雇用主と個人などの労働者が結ぶ労働契約のことです。**労働者は正社員や契約社員、パート・アルバイトといった雇用形態にかかわらず、指揮命令権者である雇用主の指示に従って業務を遂行します。
  
  
雇用契約では、あらかじめ定められた労働条件（勤務時間・賃金・勤務地など）にもとづいて、労働者が雇用主に対して労働力を提供します。雇用主側はその提供された労働の対価として、給与を支払う義務が生じます。
  
  
業務の完遂や成果物の完成に対して報酬を支払う業務委託契約とは異なり、労働の対価として給与を支払うのが雇用契約の特徴です。業務委託契約では受託者側が業務の進め方について制約に縛られないのに対し、雇用契約では労働者が「従業員」となり、従業員は雇用主の指示に従う必要があります。

### 派遣契約との違い

**派遣契約とは、派遣会社と派遣先の企業が交わす契約形態です。**派遣会社と雇用契約を結んだ派遣社員が、派遣先の企業の指示に従って業務にあたります。
  
  
派遣契約では、労働者は派遣先と直接雇用関係を結ぶわけではなく、あくまでも派遣会社と契約を結びます。労働者に対する指揮命令権が派遣先の企業にあるため、勤務時間や作業のやり方を指示したり、社内ルールの遵守を求めたりすることが可能です。
  
  
対して業務委託契約は、先述のとおり依頼者と受託者は基本的に対等の立場であり、依頼者が仕事の進め方を指示することはできません。契約上、指揮命令権が行使されるかどうかという点で、業務委託契約と派遣契約は異なります。

### フリーランスとの違い

業務委託と混同されやすい言葉のひとつに、「フリーランス」があります。
  
  
業務委託とは、企業や組織が行っている業務の一部を外部へ委託するときに結ぶ契約形態のこと。一方のフリーランスは、特定の企業に雇用されずに独立して働く個人やそういった働き方を示す言葉です。エンジニア、Webデザイナー、イラストレーター、ライター、カメラマンなどの専門職・クリエイター職に多く、複数のクライアントから同時に仕事を引き受けるケースも多くあります。
  
  
フリーランスとして働く人の多くは、委託元と業務委託契約を結んで働くのが一般的です。そのため業務委託とフリーランスは混同されがちですが、業務委託は契約形態、フリーランスは働き方という点で違いがあります。
  
  
なお、業務委託は性質によって、次で解説する民法上の「請負契約」と「委任契約（準委任契約）」に分かれます。

## 業務委託契約の種類

**民法上で業務委託契約の定義は存在せず、一般的に「請負契約」「委任契約」「準委任契約」を総称する言葉として使われています。**これらの契約は、それぞれ対価が生じる業務内容や成果物に対する責任の有無などの点で違いがあります。
  
  
業務委託の契約について詳しく知りたい方は、別記事「[請負契約とは？委託契約や準委任契約との違いや印紙・郵送不要で契約書の作成方法も解説](https://www.freee.co.jp/kb/kb-contract/contract_documents/)」をご覧ください。

### 請負契約とは

**請負契約とは、成果物の完成によって依頼をした企業が報酬を支払う契約形態です。**成果物を仕上げるまでの過程や業務時間などに両者間での取り決めはなく、一定基準を満たした成果物が完成したかどうかが問われます。
  
  
請負契約に適している主な職種は、プログラマー・デザイナー・ライター・営業・警備員などです。報酬額や支払期日、成果物の納品場所などはあらかじめ契約書で決められます。
  
  

[出典：e-Gov法令検索「民法　第六百三十二条」](https://laws.e-gov.go.jp/law/129AC0000000089)

### 委任契約・準委任契約とは

**委任・準委任契約では、請負契約のように成果物の完成に対して対価を支払うのではなく、業務を行った行為そのものに報酬を支払います。**委任契約は弁護士などの法律行為を伴う業務を依頼する場合に、準委任契約はコンサルタントや事務職など法律行為以外の業務を依頼する場合に結ばれます。
  
  
相談業務や事務作業は請負契約のように必ずしも成果物があるわけではないため、委任契約・準委任契約によって業務内容の明確化が必要です。これらの契約に適している職種として委任契約は弁護士、準委任契約は医師・コンサルタント・エステティシャン・事務員などが挙げられます。
  
  

[出典：e-Gov法令検索「民法　第六百四十三条、第六百五十六条」](https://laws.e-gov.go.jp/law/129AC0000000089)

## 業務委託契約のメリット

業務委託契約で業務を依頼するメリット

* 自社に足りないスキルやリソースを補える
* 人材の採用や育成にかかるコストを削減できる
* 従業員がコア業務に集中し、生産性が向上する

業務委託契約は、専門的なスキルを持つ外部の即戦力人材に仕事を依頼できるため、自社に適した人材や、従業員の稼働時間が足りない場合に重宝します。自社で即戦力人材を雇用するとなると、採用や育成に時間とお金がかかります。しかし業務委託契約なら、人材の採用や育成にかかるコストを削減しながら求める成果を得ることが可能です。
  
  
また、業務委託契約によって外部の助けを得ることで、従業員が行うべきコア業務に集中できるようになり、社内の生産性の向上も期待できるでしょう。

## 業務委託契約のデメリット

業務委託契約で業務を依頼するデメリット

* 自社に業務ノウハウが蓄積されづらい
* 業務委託先の管理に手間がかかる
* 高い報酬を支払わなくてはならないケースがある

前述のとおり、業務委託契約は外部に仕事を依頼できる一方で、自社に実務的なノウハウが蓄積されづらい傾向にあります。また、委託先への発注や検収、支払いなどの管理に手間がかかるほか、委託先に対して指揮命令権を行使できないため、進捗や品質の管理にも苦労するケースがあるでしょう。
  
  
さらに業務内容の専門性によっては、業務委託先へ高い報酬を支払わなければならない可能性もあります。このような場合は一概に「業務委託契約の活用がコスト削減につながる」とはいい切れないため、業務委託契約によるメリットが得られるかどうかよく検討しましょう。

## 業務委託契約を締結する流れ

業務委託契約を締結するには、次のような流れで契約を進めていきます。それぞれの手順を正しく理解し、後からトラブルを招かないように契約を締結しましょう。

業務委託契約の締結までの手順

1. 委託先を選定する
2. 契約条件の交渉を行う
3. 業務委託契約書を締結する

### ①委託先を選定する

まずは依頼したい業務の委託先を選定します。主な委託先の探し方はWebサイトでの公募やSNSでの募集、紹介などです。
  
  
**Webサイトで公募する際は、クラウドソーシングや求人サイトを用いるほか、自社で募集をかけるなどの手段があります。**それぞれ費用や手間、契約締結までの期間の目安が異なるため、自社の状況に合わせた手段を選択しましょう。
  
  
手数料がかかっても委託先を早く見つけたい場合は、あらかじめ求人フォーマットが整っており、受注を希望する人材が集まっているクラウドソーシングや求人サイトの活用が有効です。一方、時間がかかっても費用をかけたくない場合は、自社サイトでの募集が適しています。
  
  
委託先の探し方について詳しく知りたい方は、別記事「[業務委託の求人方法、フリーランスの見つけ方と受発注管理の基礎知識](https://www.freee.co.jp/kb/kb-deals/outsourcing-recruitment-method/)」をご覧ください。

### ②契約条件の交渉を行う

委託先を選定したら、契約条件の交渉を行います。決めておくとよい条件は次のとおりです。

  

* 業務内容
* 契約期間
* 報酬と支払い方法
* 禁止事項

  

**この時点で細かい内容まで決めておくと、後から「この業務は依頼されていない」「報酬が思っていた金額と違う」のようなトラブルを防げます。**契約条件が固まったら、業務範囲や報酬に関する認識のズレを避けるため、受託者から見積書を出してもらいましょう。

### ③業務委託契約書を締結する

互いに合意に達したら、業務委託契約書を作成して業務委託契約を締結します。**業務委託契約書は紙と電子契約のどちらでも問題ありません。**
  
  
業務委託契約書は「事業に関する重要な資料」に該当するため、会社法432条によって作成後10年間の保存が義務付けられています。電子契約書の場合は、電子帳簿保存法に則って保存します。
  
  
電子帳簿保存法に則った電子契約書の保存について詳しく知りたい方は、別記事「[電子帳簿保存法とは？対象書類や保存要件・改正内容についてわかりやすく解説](https://www.freee.co.jp/kb/kb-deals/deals-records-in-electronic-form/)」をご覧ください。
  
  

[出典：e-Gov法令検索「会社法　第四百三十二条」](https://laws.e-gov.go.jp/law/417AC0000000086)
  
[出典：国税庁「電子帳簿保存法の概要」](https://www.nta.go.jp/law/joho-zeikaishaku/sonota/jirei/02.htm)

## 業務委託契約書に必要な項目

業務委託契約書の作成は義務付けられているものではなく、口約束でも契約を結ぶことは可能です。しかし、**「いった・いわない」などのトラブルを避けるためにも、業務委託契約書を作成しておくと安心です。**
  
  
ここでは、業務委託契約書に記載しておくべき項目を紹介します。

  

| 項目 | 内容 |
| --- | --- |
| 委託する業務内容 | ・受託者が行う業務を明確にする  ・業務内容、提供方法などを明記 |
| 発注・受注の手段 | ・すべての案件に共通の手段があれば記載  ・案件ごとに異なる場合は、その旨と伝達の手段を記載 |
| 納品・検収の手段 | ・「納品」は委託者に対して成果物を交付すること、「検収」は成果物を確認すること  ・納品方法や検収期間、結果の通知、修正回数なども記載 |
| 報酬 | ・報酬の金額や計算方法、支払い時期、支払い手段を記載 |
| 成果物の帰属 | ・成果物の知的財産権が委託者と受託者のどちらに帰属するのかを記載  ・翻訳権・翻案権（著作権法27条）および二次的著作物の利用に関する原著作者の権利（著作権法28条）については、委託者に移転させる場合はその旨を記載する |
| 禁止事項 | ・受託者の禁止事項がある場合は記載する  ・受託者に服務規定の遵守を課した場合は偽装請負（※）となる可能性があるので注意 |
| 再委託の可否 | ・委託者が第三者に業務を再委託できるかどうかを記載  ・再委託によって生じた損害の責任の所在も定めておくとよい |
| 契約期間・更新・解除 | ・契約期間や更新、解約の手続きを記載  ・契約が終了した後も続く条項があれば記載 |
| 秘密保持 | ・業務の過程で入手した情報の流出を防ぐ  ・個人情報や機密情報が流出しないために記載 |
| その他 | ・所轄裁判所や瑕疵担保期間、反社会勢力の排除のための条項も記載しておく |

  

※「労働者派遣」または「労働者供給」であるにもかかわらず「業務委託契約」に偽装すること
  
  

[出典：e-Gov法令検索「著作権法　第二十七条、第二十八条」](https://laws.e-gov.go.jp/law/345AC0000000048)
  
  
業務委託契約書を作成する際にチェックしておきたいポイントについて詳しくは、別記事「[その契約書、実はリスクだらけ？はじめての業務委託契約書で失敗しないためのポイント11選](https://www.freee.co.jp/kb/kb-launch/contract-9points/)」をご覧ください。

## 業務委託契約書の作成時に気を付けたいポイント

業務委託契約書には、業務委託契約の内容を明示して正式に業務を依頼する目的のほか、契約者間のトラブルを事前に防ぐ働きがあります。業務委託契約書を作成する際は、以下の3点に注意しましょう。

### 収入印紙が必要なケースもある

収入印紙とは、租税や行政に対する手数料の支払いのために政府が発行する証票のことです。課税文書を作成した際に印紙税の納税義務が発生し、収入印紙が必要となります。
  
  
**業務委託契約書のうち、請負契約の第2号文書と継続的取引の第7号文書にあたる場合は、収入印紙が必要です。**第2号文書の場合は、契約金額に応じて印紙税額が非課税から60万円まで定められています。第7号文書の場合は一律4,000円です。
  
  
なお、委任契約である場合や電子契約である場合は、収入印紙を貼付する必要はありません。
  
  

[出典：国税庁「No.7102 請負に関する契約書」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/inshi/7102.htm?_fsi=RtQXd3wC)
  
[出典：国税庁「No.7104 継続的取引の基本となる契約書」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/inshi/7104.htm)

### 偽装請負とならないようにする

**偽装請負とは、「業務委託契約」を結んでいながら「労働者派遣」または「労働者供給」のような実態で働かせることです。**
  
  
業務委託契約では、発注者と受託者の間に指揮命令権がありません。それにもかかわらず発注者が受託者に指示を出すことで責任の所在が曖昧になり、労働者の雇用や安全衛生面などの労働条件が十分に確保されない恐れがあります。
  
  
偽装請負かどうかは業務の実態で判断され、以下のようなケースが挙げられます。

  

| 偽装請負の種類 | 内容 |
| --- | --- |
| 代表型 | 請負といいながら、発注者が業務の細かい指示を労働者に出したり、出退勤・勤務時間の管理を行ったりする。 |
| 形式だけ責任者型 | 現場には形式的に責任者を置くが、責任者は発注者の指示を個々に伝えるだけであり、発注者が指示をしているのとなんら変わらない。 |
| 使用者不明型 | 受注した業者がさらに他の業者に発注することで、労働者が誰に雇われているのか不明になる状態。 |

  

偽装請負は違法となるため、業務委託契約を締結する際は指示の出し方や管理に注意しましょう。

### 細かい数字のミスがないように気を付ける

業務委託契約の受発注には、作成、確認、押印など委託者と受託者それぞれに手間が発生します。特に発注数や報酬、納期などはトラブルを防ぐために細かくチェックしなくてはなりません。
  
  
紙で業務委託契約書を作成している場合は、書類の回収や管理も手間となるでしょう。必要に応じて**電子契約が行える業務委託管理ツールを利用することで、業務の効率化が図れます。**

## フリーランスとの契約にまつわる疑問を解消

業務委託先として多い選択肢のひとつに、前述した「フリーランスとして活躍する個人事業主」が挙げられます。
  
  
**近年はフリーランスとの取引に関する法整備が強化されており、業務の委託者が注意すべき点も増えています。**フリーランスとの契約において、具体的にどのような点に注意が必要なのか、よくある疑問をもとに解説します。

### フリーランスとの一般的な契約期間

フリーランスとの契約期間は、業務内容やプロジェクトの規模、クライアントのニーズなどによって異なります。一般的に、以下のような傾向があります。

  

|  |  |
| --- | --- |
| 1ヶ月 | 短期的なスポット案件や、試用期間的な意味合いで設定されることが多い。比較的単価の高い専門家や、特定のタスクのみを依頼する場合などによく見られる。 |
| 3ヶ月 | プロジェクトの区切りが良く、双方が状況を見ながら更新するかどうか判断しやすい期間。エンジニアやWebデザイナーの案件でよく見られる。初回契約として設定されることも多い。 |
| 6ヶ月 | 比較的大規模なプロジェクトや、ある程度の期間を要する開発・制作案件などで見られる。3ヶ月契約を更新して6ヶ月契約となるケースも多い。 |

  

フリーランスとの取引は契約更新を前提としているケースも多く、実際には上記よりも長期間にわたって同じクライアントと取引を続けるフリーランスが多いといえるでしょう。

### フリーランスの職種による契約期間の違い

一口にフリーランスといっても、職種によって依頼内容や契約期間が異なります。例として、次の3つの職種の傾向を紹介します。

  

|  |  |
| --- | --- |
| エンジニア | プロジェクト単位での契約が多く、3ヶ月～6ヶ月が中心。保守運用フェーズに入ると、長期化する傾向がある。 |
| Webライター | 記事単位や文字単価での単発契約から、メディア専属ライターとしての月額契約（長期）まで契約期間は多様。 |
| Webディレクター | プロジェクト期間に連動することが多く、3ヶ月から1年程度と幅があるケースが多い。 |

### フリーランスとの契約更新に必要な作業とその時期

フリーランスとの契約を更新をする場合は、次の点を確認しましょう。
  
  
まず、**現在締結している業務委託契約書に契約更新に関する条項があるか確認します。**契約書によっては、「契約期間満了の〇ヶ月前までに、当事者のいずれかから別段の意思表示がない限り、同一条件で自動的に更新される」といった自動更新に関する条項や、「契約期間満了の〇ヶ月前までに更新の意思を書面で通知する」といった手続きに関する規定が設けられている場合があります。その場合は、契約書の定めに従わなければなりません。
  
  
契約書に特に定めがない場合や、自動更新でない場合の一般的な考え方は以下のとおりです。

  

* 契約満了の1ヶ月～1.5ヶ月前：更新意向の確認・打診
* 契約満了の遅くとも1ヶ月前まで：正式な意思表示と条件提示
* 契約満了日まで：契約書の締結

  

契約書の内容確認や修正には時間がかかることも考えられるため、余裕をもって準備を進めてください。

### フリーランスへの委託期間によって変わる事業者側の義務項目

**「フリーランス・事業者間取引適正化等法」（フリーランス法）は、2024年11月1日に施行されたフリーランスとの取引に関する新しい法律です。**
  
  
フリーランス法では、業務委託の期間（1ヶ月未満／1ヶ月以上／6ヶ月以上）に応じて変化する義務項目があるため注意しましょう。

#### すべての期間で義務とされる項目

以下は、すべての期間で義務とされる項目です。

  

|  |  |
| --- | --- |
| 書面等の交付 | 発注の際は、契約条件や業務内容、報酬額、納期、支払い方法などの重要な取引条件について、書面または電子メールなどにより明示する義務がある。 |
| 支払期日の設定 | 業務の遂行に対してフリーランスが給付（納品または業務提供）を行った後、その受領日から起算して60日以内に報酬の支払期日を設定しなければならない。 |
| 募集情報の最新かつ正確な表示 | 募集・契約先選定の過程において、フリーランスに提供される情報（業務内容、報酬、納期、業務範囲、契約条件など）は正確かつ最新の状態に保つ必要がある。 |
| ハラスメント行為に関する体制整備 | ハラスメントについて、相談窓口の設置や対応手順の整備など、必要な体制を整えておく義務がある。 |

#### 1ヶ月以上／6ヶ月以上の場合に義務とされる項目

以下は、1ヶ月以上／6ヶ月以上の場合に義務とされる項目です。

  

|  |  |
| --- | --- |
| 受領拒否 | 正当な理由なく、納品物の受領を拒否してはならない。不備がある場合も、内容を説明し、誠実に協議のうえ対応する必要がある。 |
| 報酬の減額 | 合意済みの報酬を、納品後などに一方的に減額することは認められない。 |
| 買いたたき | 通常の相場より著しく低い報酬を設定・強要することはできず、適正な報酬水準を尊重する必要がある。 |
| 購入・利用強制 | 契約業務とは関係のない物品やサービスの購入や利用を、フリーランスに強制してはならない。 |
| 不当な経済上の利益の提供要請 | 金銭、労務、サービスなどを、契約の範囲を超えて無償または不当に提供させることは禁止されている。 |
| 不当な給付内容の変更・やり直し | 費用を負担せず、一方的な仕様変更や納品後のやり直しを求めることはできない。変更が必要な場合は協議を行い、必要に応じて追加報酬を設定する。 |

#### 6ヶ月以上の場合のみ義務とされる項目

以下は、6ヶ月以上の場合のみ義務とされる項目です。

  

|  |  |
| --- | --- |
| 妊娠・出産・育児・介護に対する配慮 | フリーランスから妊娠、出産、育児、または家族の介護などと業務を両立する必要がある旨などの申し出があった場合、業務量の調整や納期延長などの柔軟な対応を行い、就業への配慮に努めなければならない。 |
| 中途解除等の予告 | 1ヶ月を超える継続的な業務委託契約を中途で解除する場合、原則として解除の30日前までに予告する必要がある。 |

[出典：「中小企業庁発行のフリーランス法パンフレット」](https://www.chusho.meti.go.jp/keiei/torihiki/download/freelance/law_03.pdf)

業務委託契約とは、企業や組織が内部で行っている業務・作業を外部の企業や個人に委託するときに結ぶ契約形態のことを指します。

### 業務委託契約とフリーランスの違いは？

業務委託とは、企業や組織が行っている業務の一部を外部の企業や個人へ委託するときに結ぶ契約形態を指します。一方のフリーランスは、特定の企業に雇用されずに独立して働く個人やそういった働き方を示す言葉です。両者には、契約形態を示すのか、働き方を示すのか、という違いがあります。
  
  
詳しくは記事内の「[フリーランスとの違い](#content1-3)」をご覧ください。
