## 同意書とは

**同意書とは、特定の行為や条件に対し、相手方が内容を理解し納得したうえで同意したことを書面化したものです。**口頭でのやり取りだけでは、「いった／いわない」の争いが発生する恐れがあります。「同意していた」という事実を書面に残すことで、当事者間で取り決めた内容が明確になるため、誤解やトラブルの防止につながります。
  
  
特定の行為や条件などの約束事を取り決める際には、まず同意書が必要かどうかを検討します。必要だと判断した場合は、適切な内容の同意書を作成しましょう。
  
  
次に、作成した同意書を相手方に提示して十分な説明を行います。意見を取り入れるなどして、内容に修正が発生する場合はこの時点で対応しましょう。相手方が記載の内容に納得したら、署名または記名押印をしてもらって完了です。

### 同意書と承諾書の違い

同意書と承諾書は「何らかの事柄や行為について異存（異なる意見や考え）がない旨を表明する書面」という点で共通しています。**同意書ではなく承諾書の作成を求められる場面があるかもしれませんが、内容は基本的に同じと考えて問題ありません。**
  
  
同意書と承諾書を使い分けるポイントは、法令や契約中で要求される用語に合わせることです。具体的には、同意を求める（求められる）場合は「同意書」、「承諾」を求める（求められる）場合には「承諾書」となります。

### 同意書と契約書の違い

同意書と契約書には、作成者と法的性質において明確な違いがあります。

| 種類 | 特徴 |
| --- | --- |
| 同意書 | ・相手方の行為や条件に対する同意を示す書面  ・同意を表明する側が一方的に作成する  ・一方が同意する者となり、もう一方が同意を確認する者となる |
| 契約書 | ・当事者間の権利・義務を詳細に定める書面  ・当事者全員が作成者となる  ・双方の合意に基づいて締結される |

このように、**同意書は一方向の意思表示を記録する書面であるのに対し、契約書は双方の合意内容を定めるという点で大きく異なります。**

## 同意書の法的効力が発揮される主な場面

同意書が持つ効果は、対象事項によってさまざまです。適切に作成された同意書は、以下のような場面で重要な法的効力を発揮します。

### 契約内容やサービス提供に関するトラブル

**顧客から「こんな条件だとは思わなかった」といったクレームが入った場合や、契約解除・返金などを求められた場合**、契約内容についての同意書があれば証拠となります。サービス内容や条件について顧客が理解・同意していたことを客観的に証明できるため、企業側のリスク軽減につながります。

### 金銭の支払い遅延や未払いのトラブル

**商品代金やサービス利用料の支払いが滞り、督促しても支払いに応じてもらえない場合**、支払い条件が記載された同意書があれば、債権回収の際の根拠資料として活用できます。

### 個人情報の取り扱いやプライバシーに関するトラブル

**顧客の個人情報を収集したり利用したりする際**、その目的や範囲について同意書を作成しておくと、後の疑義が生じた場合や情報漏洩時の対応において、個人情報保護法に基づき適切な同意を取得していたことの証明として機能します。

### 権利の利用許諾や範囲に関するトラブル

**撮影した写真や映像などを商用利用する際の肖像権、従業員の著作物を会社が利用する際の著作権、またインタビュー記事の掲載許可など**、知的財産権に関するトラブルの防止に役立ちます。

### 免責事項やリスク受諾の確認が必要な場面

**イベント参加時に発生した事故、特定のサービス利用に伴う潜在的なリスク、またソフトウェアの利用規約における責任制限など**、事業者側の責任範囲を明確にし、利用者がリスクを理解・受諾していたことを証明したい場合に有効です。

### 契約の有効性そのものが争点となるトラブル

**未成年者が親権者の同意を得ないまま契約した場合の取り消しや、意思能力が十分でない状態での契約など**、契約の有効性自体が問題となった際、適切な同意書の存在が契約の正当性を裏付ける証拠となります。

## ビジネスで使われる主な同意書の種類と具体例

ビジネスシーンでは、業種や取引内容に応じてさまざまな種類の同意書が使われています。**これらの同意書は、それぞれ異なる法的要件や業界特有のルールに基づいて作成しなければなりません。**

| 同意書の種類 | 同意を得たい主な場面・内容 |
| --- | --- |
| 個人情報の取り扱いに関する同意書  （プライバシーポリシー同意） | ・会員登録  ・アンケート調査  ・メルマガ配信  ・第三者提供 |
| サービス利用規約・約款への同意書 | ・Webサービス利用開始時  ・アプリダウンロード時  ・プラン変更時 |
| 秘密保持に関する同意書 | ・商談前の情報開示  ・協業検討時  ・従業員採用時 |
| 保護者同意書 | ・未成年者の契約締結  ・学習塾などの申し込み  ・アルバイトの雇用契約  ・携帯電話契約 |
| インフォームドコンセント | ・手術や治療の実施  ・臨床試験参加  ・医療研究協力  ・薬事承認試験 |
| セミナー・イベント参加同意書 | ・研修参加  ・展示会出展  ・オンラインセミナー配信 |
| アンケート・インタビュー協力同意書 | ・市場調査  ・顧客インタビュー  ・事例取材 |
| 従業員向け同意書 | ・入社時  ・就業規則変更時  ・秘密情報保持  ・転勤・異動 |
| 取引先向け同意書 | ・価格改定  ・支払条件変更  ・取引条件見直し |

自社の業種や取引形態に応じて適切な同意書を選択し、法令要件を踏まえた内容で作成することが、リスク管理につながるでしょう。

## 同意書の書き方と作成時のポイント

法的効力のある同意書を作成するには、明確で理解しやすい内容にすることが重要です。同意書作成時の必須要素と基本的な書き方について押さえておきましょう。

### 同意書の基本的な構成要素（必須記載事項）

同意書を作成するポイントは、法的要件を満たしつつ、相手方が理解しやすい構成にすることです。特に同意書に記載すべき事項は、表のとおりです。

| 記載事項 | 詳細 |
| --- | --- |
| 表題・タイトル | ・何を対象とする同意書なのかが一目でわかるように記載  （たとえば「〇〇〇に関する同意書」など） |
| 同意書作成日 | ・同意を取得した日付を明確に記載 |
| 同意を求める者（甲） | ・企業名  ・代表者名  ・所在地  ・連絡先  など |
| 同意をする者（乙） | ・氏名  ・住所  ・連絡先  ・本人確認に必要な情報  など |
| 同意の対象となる具体的な事項 | ・何に対する同意なのかを明確かつ具体的に記載 |
| 同意内容の詳細 | ・条件  ・範囲  ・期間  ・メリット  ・デメリット  など |
| 同意の意思表示 | ・内容に同意することを明記  （「上記内容を理解し、これに同意します。」など） |
| 署名・押印欄 | ・同意者の署名（または記名押印）欄  ・日付記入欄 |

これら記載事項を盛り込むことで、法的に有効で実用性の高い同意書を作成できます。とくに**「同意の対象となる具体的な事項」や「同意内容の詳細」については、後のトラブルを防ぐためにも正確に記載しておきましょう。**

### 同意書の書き方に関するポイント

同意書の法的な有効性を確保するためには、文章の書き方にも配慮が必要です。効果的な同意書を作成するポイントはいくつかあります。

#### 平易な言葉を使用し、専門用語は解説を加える

同意書は法的な文書ですが、相手方が内容を正確に理解できることが重要です。誰でも理解しやすい文書にするために、**専門用語や法律用語の使用は極力避けましょう。**やむを得ず使用する場合は、わかりやすい解説を併記します。

#### 曖昧な表現を避け、一義的に解釈できる記述にする

「適切に」「合理的な範囲で」「必要に応じて」といった抽象的な表現は、トラブルの原因となる可能性があります。期間や範囲、方法などは数値を用いるなどして、**具体的な基準や条件を明示するよう心がけましょう。**

#### 箇条書きなどを活用し、視覚的に整理する

長文の同意書は読み手にとって負担が大きく、重要な点を見落とすリスクにもなります。読みやすい文書を作成するには、**箇条書きや番号付きリスト、表などを活用して視覚的に整理しましょう。**また、重要事項は太字やアンダーラインで強調するなどの工夫も有効です。

#### 同意者に不利益となる可能性がある条項はわかりやすく説明する

免責事項や制限条項など、同意者にとって不利益となる可能性がある内容については、より丁寧でわかりやすい説明を心がけましょう。リスクの内容や発生する可能性、回避方法などを具体的に記載し、**同意者が十分に理解したうえで判断できるよう配慮することが大切です。**

#### 自由意思に基づく同意であることを前提とする

同意書への署名は、相手方の自由意思に基づいて行われる必要があります。強制的な印象を与える表現や、不当に不利益を強いるような条項は避け、**相手方が自主的に判断できる環境を整えることが重要です。**

#### 関連法規を遵守した内容にする

個人情報保護法や消費者契約法、労働基準法など、**同意書の内容に関連する法令の要件を満たしているか確認しましょう。**法的要件を満たしていない同意書は、その効力が認められない恐れがあります。

## 同意書テンプレートの賢い活用法と注意点

同意書の作成には、テンプレートの活用がおすすめです。あらかじめポイントを押さえたテンプレートを活用すれば、作成時間の短縮や品質の向上を実現できます。ただし、テンプレートをそのまま使うのではなく、自社の状況に合わせた適切なカスタマイズが重要です。

### テンプレートを利用するメリット

同意書テンプレートの活用は、効率的な書類作成を実現するうえでメリットが大きいといえます。

テンプレートを利用するメリット

* 作成時間の短縮と効率化が図れる
* 基本的な記載項目の漏れを防止できる
* 誰が作成しても一定の品質を担保しやすい

たとえば一から同意書を作成する場合、構成を考えてから文章を作成するなど多くの時間が必要です。しかし、テンプレートなら基本的な構成や定型的な文言は既に用意されており、**書類作成の時間を短縮できます。**
  
  
また、同意書に必須の記載事項は多岐にわたるため、重要な項目を見落とすリスクも考えられます。テンプレートには記載項目があらかじめ網羅されているため、**抜け漏れを防ぎ、法的要件を満たした同意書を作成しやすくなります。**
  
  
作成者の経験やスキルにかかわらず、一定水準以上の同意書の作成が可能な点もテンプレートを活用するメリットです。これにより、**組織全体での書類品質の標準化が実現されます。**

### どこが提供している同意書テンプレートを使うのか

同意書のテンプレートはさまざまな機関や組織から提供されており、それぞれ異なる特徴とメリット・デメリットがあります。それぞれの違いを理解しておきましょう。

#### 政府や官公庁が提供するテンプレート

政府や官公庁が提供するテンプレートは**法的要件を満たしており、信頼性が高い**点が特徴です。特定の業界や用途に特化したものも多く、関連法令の最新動向が反映されているため安心して利用できます。

#### 弁護士事務所や法務系Webサイトのテンプレート

法律の専門家である弁護士が作成・監修したテンプレートは、法的な精度が高く、実務での使用に適しています。解説や作成時のポイントが併記されているケースも多く、**同意書作成の学習にも役立ちます。**

#### ビジネス書式集サイトなど

書式集サイトには、幅広い業種や用途に使えるよう豊富なテンプレートがそろっています。用途やデザインなどの希望に合わせて、**自社に適したものを見つけやすい**のが特徴です。ただし、法的な監修が不十分であるケースもないとはいえないため、重要な取引や法的リスクが高い案件では専門家によるチェックが必要です。

#### 自社で保管・運用されている過去の同意書

手元に保管している過去の同意書を活用する方法もあります。自社の業務や取引実態に最も適合しており、**過去の運用実績があることから実用性が高い**のも魅力です。社内での承認プロセスもすでに完了しているため、迅速に活用できるでしょう。

### 同意書テンプレート利用時の注意点

テンプレートは便利なツールですが、適切に活用するためにはいくつかの注意点があります。
  
  
第一に、**テンプレートをそのまま使用することは避けましょう。**汎用性の高いテンプレートは、あらゆる業種や取引に対応できるよう一般的な内容となっています。そのため、自社固有のリスクに対する保護が不十分であったり、不要な項目が含まれていたりする可能性があります。
  
  
また、**テンプレートが最新の法令に対応しているかどうかの確認も欠かせません。**法律は社会情勢の変化に応じて頻繁に改正されるため、古いテンプレートでは現行法に適合していない恐れがあります。法的要件を満たしていない同意書は、その効力が認められないだけでなく、場合によっては違反となる可能性があります。活用する際は作成日を必ず確認し、最新の法令に対応しているかをチェックしてください。
  
  
法的な有効性に不安がある場合は、迷わず弁護士や専門家にサポートしてもらいましょう。同意書の内容が複雑な場合や、重要な取引、高額な案件に関わる場合は特に重要です。

## 業務効率化を図りたい場合の選択肢「電子同意書」

デジタル化が進む現代のビジネス環境において、業務効率化とコスト削減を同時に実現できる選択肢として注目されているのが電子同意書です。

### 電子同意書とは

**電子同意書とは、電子署名や電子契約システムを利用して作成・締結する同意書のことです。**インターネット上で同意書の送付から署名、保管まですべてのプロセスを完結できます。
  
  
基本的な仕組みとしては、同意を求める側がシステム上で同意書を作成し、電子メールで相手方に送付します。受信者はブラウザ上で内容を確認し、電子署名で同意を表明します。なお、署名済みの同意書は自動的にシステム内に保管され、必要に応じていつでも確認・ダウンロードすることが可能です。

### 電子同意書を利用するメリット

**電子同意書を導入すると、業務効率化やコスト削減に加え、セキュリティ面での安全性も期待できるようになります。**近年、多くの企業で導入が進んでいます。

| メリット | 詳細 |
| --- | --- |
| 効率化・迅速化 | ・作成から署名、保管まですべてオンラインで完結  ・従来の紙ベースと比較して署名までの時間短縮を実現 |
| コスト削減 | ・紙代、印刷代、郵送費、保管スペースの確保費用などを削減  ・印紙税も電子文書は課税対象外となるケースが多い |
| 管理負担の軽減 | ・同意書の検索性の向上  ・瞬時に該当の同意書を検索でき、業務効率が向上  ・紙の劣化や紛失のリスクがなく、管理担当者の負担が軽減 |
| セキュリティ向上 | ・改ざん防止機能が強化  ・アクセス権限の設定で非権限者による閲覧や修正を防止  ・なりすまし防止のための本人認証機能も充実 |

### 電子同意書の法的有効性

**電子同意書の法的有効性は、電子署名法（電子署名及び認証業務に関する法律）によって保障されています。**同法第3条において「電磁的記録であって情報を表すために作成されたものについて、当該電磁的記録に記録された情報について本人による電子署名が行われているときは、真正に成立したものと推定する」と規定されているためです。
  
  
これにより、適切な電子署名が施された電子同意書は、手書き署名や押印のある紙の同意書と同等の法的効力が認められます。ただし、いくつかの要件を満たす必要がある点に注意しましょう。

法的有効性を保つ要件

* 電子署名は本人のみが行うことができるものであること
* 電子署名を行った後に改変されていないことを確認できること

なお、適切な電子契約サービスを利用すればこれらの要件を満たせるため、法的に有効な電子同意書を作成できるようになります。
