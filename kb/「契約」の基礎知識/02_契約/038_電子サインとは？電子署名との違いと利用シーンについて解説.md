## 電子サインとは？

電子契約とは、書面で契約書を作成する代わりに、契約内容を電子データによって保存・管理するものです。
  
  
従来の契約は、紙の契約書に当事者双方が署名・押印することで、当人同士が合意の上で契約を締結した事実が担保されました。しかし、電子データによるやりとりを行う電子契約では、従来のような署名や押印はできません。そこで、電子契約において、従来の署名・押印の代わりに本人性を担保し、内容に改ざんがないことを保証するものが「電子サイン」です。
  
  
電子サインは、紙の契約書における署名・押印と同じ役割を持ち、電子データ化（PDF化）された契約書上に施されることで効力を発揮します。
  
  
現在、電子契約には電子証明書を発行し、契約の当事者同士が電子署名を行う「当事者型電子署名」と、電子契約サービスを介して行う「立会人型電子署名」という方法があります。特に近年、広く普及している立会人型電子署名は、さまざまな契約で利用されています。
  
  
これは、契約を結ぶ当事者に代わり、第三者である電子契約サービスの提供者（事業者）が署名するもので、依頼を受ける際にメール認証やパスワードなどを使い、本人からの依頼手続きであると確認することで、本人性を担保する仕組みです。

## 電子サインと電子署名の違い

電子サインとしばしば混同されるものに、電子署名があります。実際は、「電子サインの一種として電子署名がある」という関係で、同じものではありません。
  
  
電子サインは、従来の署名・押印の代わりに本人性を担保し、内容に改ざんがないことを保証する仕組みです。紙の契約書に行う署名・押印も、利用シーンによって認印を使う場合と実印・印鑑証明書のセットを使う場合があるように、電子サインにも通常の方法と、より厳格に本人性・非改ざん性を担保する方法があります。
  
  
電子署名は後者にあたり、「公開鍵暗号」「公開鍵基盤（PKI）」「ハッシュ関数」という3つの技術を使うことで、より厳格に本人性・非改ざん性を担保した電子サインの方法なのです。
  
  
なお、電子署名以外の方法による電子サインには、固有の呼び名がありません。ここでは便宜的に「一般的な電子サイン」とし、両者の違いについて下記にまとめました。

#### 一般的な電子サインと電子署名の比較

|  |  |  |
| --- | --- | --- |
|  | 一般的な電子サイン | 電子署名 |
| 紙の契約書における役割 | 認印 | 実印＋印鑑証明 |
| 本人性の担保 | メール認証など | 第三者機関である認定局が発行する電子証明書 |
| 非改ざん性の担保 | タイムスタンプなど | 電子証明書、タイムスタンプなど |
| 民亊訴訟における証拠力 | 高いとはいえない場合もある | 高い |
| 契約相手への負担 | 特になし | あり（電子証明書の取得など） |

紙の契約書に例えると、一般的な電子サインは認印を使って契約を結ぶ方法、電子署名は実印と印鑑証明を使って契約を結ぶ方法にあたります。
  
  
一般的な電子サインでは、メール認証などの方法で本人性を確認しますが、電子署名では、自治体が発行する印鑑登録証明書にあたる「認定局が発行する電子証明書」を利用するというより厳格な方法で、本人性の確認を行います。
  
  
その分、一般的な電子サインより手間はかかるものの、電子署名形式で契約を結ぶほうが、万が一の契約トラブルから民事訴訟に発展した場合でも、証拠力は高くなります。

### 一般的な電子サインと電子署名の使い分け方

紙の契約において、その内容によって認印と実印・印鑑証明のどちらかを選択するように、電子サインでも2つのサイン方式を使い分けるのが一般的です。通常の契約は契約相手にも負担の少ない一般的な電子サインで行い、特に重要な契約に対して電子署名を利用します。
  
  
たとえば、雇用契約や業務委託契約など、従来認印を使うことの多かった契約は一般的な電子サインで行い、大口の売買契約や業務提携契約など、重要度の高いものについては電子署名で行うケースが多いといえます。

## 電子サインを作るにはどうすればいい？

電子サインを作成するには、大きく分けて「手書き署名」と「電子印鑑（印影の電子化）」の2つの方法があります。

### 手書き署名を作る

手書き署名とは、電子文書（PDFデータなど）に直接署名を行うものです。PDFの電子署名ツールを利用するか、手書き署名に対応した電子契約サービスを活用します。

### 電子印鑑を作る

電子印鑑とは、印影をデジタルデータ化して、PDFなどのデータ文書に貼りつけられるようにしたものです。印影の画像データに使用者や押印時間など、識別情報が保存されており、それによって本人性や非改ざん性を担保しています。電子印鑑は、電子印鑑に対応した電子契約サービスを活用することで利用できます。
  
  
なお、実際の印影をスキャナなどでパソコンに取り込み、背景を透過するなどの処理を行っただけの画像データでも電子印鑑と呼ばれますが、こちらは電子サインには該当しません。その理由として、本人性や非改ざん性を担保する仕組みがなく、一般の画像データと変わらないからです。
  
  
日本では押印文化が根づいていることもあり、電子サインの方法としては電子印鑑が選ばれることが多いようです。

## 電子サインが使われるシーンとは？

電子サインは、紙の書類における署名・押印の代わりを果たすものです。「電子契約の締結の際に使われるもの」というイメージが強いですが、活用できるのは契約の場面だけにとどまりません。
  
  
電子サインを導入するメリットは、契約に限らずあらゆる署名・押印を電子サインに置き換えることで、紙書類を電子化して保存できるところにあります。
  
  
電子サインを導入することで、下記のような手続きの電子化が可能になります。

### 電子サインを導入することで電子化が可能となりうる書類の種類

#### 契約書類

* 雇用契約書などの人事・労務関係書類
* 業務委託契約書や秘密保持契約書などの契約書
* 売買契約書や取引基本契約書などの商取引に関する契約書
* 土地売買契約書や土地賃貸契約書などの不動産に関する契約書
* 金銭借用書などの金銭消費賃借契約に関する契約書
* 合併契約書や株式譲渡契約書などの会社運営に関わる契約書

#### そのほかの書類

* 請求書や見積書
* 社内稟議などの承認

  
特に注目したいのは、社内稟議の電子化が可能になる点です。書類の印刷や手渡しによる回覧の必要がなくなり、在宅ワークや出張中といった遠隔からでも承認が可能になるため、決裁までスムーズに進むようになるほか、内部統制の強化やペーパーレス化によるコスト削減にもつながります。
  
  
社外と関わる契約書などの電子化には相手方の同意を必要とするため、なかなか思いどおりに進まないことも考えられますが、社内の手続きで使われていた署名・押印に電子サインを導入するだけでも、業務効率化につながることが期待できるでしょう。
