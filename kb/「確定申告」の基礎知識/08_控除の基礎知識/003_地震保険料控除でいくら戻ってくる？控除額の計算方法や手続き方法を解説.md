## 地震保険料控除とは

**地震保険料控除とは、納税者が居住用家屋や生活用動産を対象とする地震保険の保険料を支払った場合に、一定額を控除できる制度です。**この控除を受けることで課税対象となる所得額が減少し、結果として所得税と住民税の負担を軽減できます。
  
  
**控除できる金額には上限があり、支払った保険料の全額が控除されるわけではありません。控除上限額は、所得税が最大5万円、住民税が最大2.5万円です。**
  
  
また、地震保険料控除の適用を受けるためには、手続きが必要です。会社員などの給与所得者は年末調整で、個人事業主などの給与所得者以外の人は確定申告で申請します。
  
  
手続きの際には、保険会社から送付される「地震保険料控除証明書」が必要です。証明書が届いたら、なくさないように保管しておきましょう。

  

[出典：国税庁「地震保険料控除」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1145.htm)
  
[出典：東京都主税局「地震保険料控除」](https://www.tax.metro.tokyo.lg.jp/kazei/life/kojin_ju#:~:text=70%2C000%E5%86%86%E3%81%A7%E3%81%99%E3%80%82-,%E5%9C%B0%E9%9C%87%E4%BF%9D%E9%99%BA%E6%96%99%E6%8E%A7%E9%99%A4,-%EF%BC%881%EF%BC%89%E5%9C%B0%E9%9C%87%E4%BF%9D%E9%99%BA)

## 地震保険料控除の対象となる保険契約

地震保険料控除の対象となる保険契約には、「地震保険契約」と特例として扱われる「旧長期損害保険契約」の2種類があります。
  
  
それぞれ詳しく解説します。

### 地震保険契約

地震保険料控除の主な対象は、火災保険に付帯して契約する地震保険です。主に、地震・噴火・津波による損害の補償が地震保険に含まれます。
  
  
控除の対象となるのは、納税者本人または生計を一にする配偶者・親族が所有する居住用家屋と生活用動産（家財）です。**地震保険を火災保険とセットで契約していても、火災保険の保険料は対象外です。**
  
  
また、支払った損害保険料が控除の対象となるかどうかは、保険会社から届く「地震保険料控除証明書」で確認できます。この証明書は、年末調整や確定申告の際に必要となるため、保管しましょう。

  

[出典：国税庁「地震保険料控除」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1145.htm)
  
[出典：国税庁「地震保険料控除の対象となる保険や共済の契約」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1146.htm)

### 旧長期損害保険契約（経過措置）

旧長期損害保険契約とは、現在の地震保険料控除制度が始まる前（2007年以前）に契約された一部の長期損害保険を指します。従来の「損害保険料控除」が廃止された際に、経過措置として地震保険料控除の対象に含められました。
  
  
控除対象となる旧長期損害保険契約は、以下の全ての要件を満たすものです。

控除対象となる旧長期損害保険契約の要件

* 2006年12月31日以前に締結された契約であること
* 保険期間が10年以上で、満期返戻金等がある契約であること
* 2007年1月1日以後に、その契約の内容変更（保険料の変更を伴うものなど）が行われていないこと

旧長期損害保険契約の控除額には、現在の地震保険とは別に控除上限が設けられています。
  
  
所得税の控除額は、以下のとおりです。

| 支払保険料 | 所得税の控除額 |
| --- | --- |
| 10,000円以下 | 支払保険料の全額 |
| 10,000円超20,000円以下 | 支払保険料の1/2 + 5,000円 |
| 20,000円超 | 一律15,000円 |

[出典：国税庁「地震保険料控除」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1145.htm)
  
  

住民税の控除額は、以下のとおりです。

| 支払保険料 | 住民税の控除額 |
| --- | --- |
| 5,000円以下 | 支払保険料の全額 |
| 5,000円超15,000円以下 | 支払保険料の1/2 + 2,500円 |
| 15,000円超 | 一律10,000円 |

[出典：東京都主税局「地震保険料控除」](https://www.tax.metro.tokyo.lg.jp/kazei/life/kojin_ju#:~:text=70%2C000%E5%86%86%E3%81%A7%E3%81%99%E3%80%82-,%E5%9C%B0%E9%9C%87%E4%BF%9D%E9%99%BA%E6%96%99%E6%8E%A7%E9%99%A4,-%EF%BC%881%EF%BC%89%E5%9C%B0%E9%9C%87%E4%BF%9D%E9%99%BA)
  
  

地震保険料控除と旧長期損害保険契約控除の両方を受けられる場合でも、控除額の合計上限は所得税5万円、住民税2万5,000円となります。

## 地震保険料控除でいくら戻る？控除額の計算方法

地震保険料控除によっていくら税金が戻るのかは、控除される金額と適用される税率によって決まります。以下では、所得税と住民税の控除額の上限、戻る税金の計算方法を解説します。

### 地震保険料控除額

地震保険料控除では、支払った地震保険料の合計額に応じて、所得税と住民税それぞれに控除上限が設けられています。

  

| 年間の地震保険料 | 所得税の控除額 | 住民税の控除額 |
| --- | --- | --- |
| 50,000円以下 | 支払った保険料の全額 | 支払った保険料の1／2 |
| 50,000円超 | 一律50,000円 | 一律25,000円 |

  

**地震保険料を複数年分まとめて一括で支払った場合でも、一度に全額を控除することはできません。**支払った総額を1年あたりの保険料に換算し、契約年数に応じて毎年控除を受ける形です。

  

[出典：国税庁「地震保険料控除」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1145.htm)
  
[出典：松江市「地震保険料控除」](https://www.city.matsue.lg.jp/soshikikarasagasu/zaiseibu_shiminzeika/zeikin/3/11919.html#:~:text=%E4%B8%80%E5%BE%8B35%2C000%E5%86%86-,%E5%9C%B0%E9%9C%87%E4%BF%9D%E9%99%BA%E6%96%99%E6%8E%A7%E9%99%A4,-%E7%B4%8D%E7%A8%8E%E7%BE%A9%E5%8B%99%E8%80%85)

### 地震保険料控除で戻る税金の計算方法

控除によって戻る（軽減される）税金は、以下の計算式を用いて算出します。

地震保険料控除で戻る税金の計算式

**戻る税金 = (所得税の控除額 × 所得税率) + (住民税の控除額 × 住民税率)**

所得税率は課税所得の金額に応じて変わり、以下のように段階的に設定されています。

#### 所得税率の速算表

|  |  |  |
| --- | --- | --- |
| 課税対象の所得金額 | 税率 | 控除額 |
| 1,000円〜1,949,000円 | 5％ | 0円 |
| 1,950,000円〜3,299,000円 | 10％ | 97,500円 |
| 3,300,000円〜6,949,000円 | 20％ | 427,500円 |
| 6,950,000円〜8,999,000円 | 23％ | 636,000円 |
| 9,000,000円〜17,999,000円 | 33％ | 1,536,000円 |
| 18,000,000円〜39,999,000円 | 40％ | 2,796,000円 |
| 40,000,000円以上 | 45％ | 4,796,000円 |

[出典：国税庁「No.2260　所得税の税率」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/2260.htm)

住民税の所得割の税率は、原則として全国一律で10%が適用されます。

  

[出典：総務省「個人住民税」](https://www.soumu.go.jp/main_sosiki/jichi_zeisei/czaisei/czaisei_seido/150790_06.html)
  

たとえば、年間の地震保険料が5万円を超え、所得税率が20％の場合、控除額の計算は以下のとおりです。

所得税の軽減額：控除額50,000円 × 所得税率20％ = 10,000円
  
住民税の軽減額：控除額25,000円 × 住民税率10％ = 2,500円
  
合計の軽減額：10,000円 + 2,500円 = 12,500円

このケースでは、年末調整や確定申告を行うことで、年間12,500円の税負担を軽減できます。

## 年末調整による地震保険料控除の手続き方法

会社員など給与所得者が地震保険料控除を受ける際は、年末調整による手続きが必要です。手続きは、以下の手順で行います。

年末調整による地震保険料控除の手続き方法

1. [会社から書類を受け取る](#content4-1)
2. [保険会社から控除証明書を受け取る](#content4-2)
3. [申告書に記入する](#content4-3)
4. [控除証明書を添付・提出する](#content4-4)
5. [税金の還付を受けるまたは不足分を納税する](#content4-5)

それぞれ詳しく解説します。

  

[出典：国税庁「給与所得者の保険料控除の申告」](https://www.nta.go.jp/taxes/tetsuzuki/shinsei/annai/gensen/annai/1648_05.htm)

### ①会社から書類を受け取る

まず勤務先から「給与所得者の保険料控除申告書」を受け取ります。給与所得者の保険料控除申告書は、年末調整で生命保険料控除や地震保険料控除などの各種保険料控除を受けるために使用する書類です。
  
  
一般的に**10月下旬～11月上旬にかけて、勤務先から年末調整に必要な書類とあわせて従業員に配布されます。**国税庁のホームページからも様式をダウンロードして入手することも可能です。

  

[出典：国税庁「給与所得者の保険料控除の申告」](https://www.nta.go.jp/taxes/tetsuzuki/shinsei/annai/gensen/annai/1648_05.htm)

### ②保険会社から控除証明書を受け取る

加入している保険会社から郵送される「地震保険料控除証明書」を準備します。この証明書には、その年に支払った保険料のうち、控除の対象となる金額が明記されています。
  
  
**地震保険料控除証明書は、毎年10月中旬以降に保険会社から契約者あてに郵送されるのが一般的です。**控除を受けるためには控除証明書の原本が必要となるため、紛失しないようにしましょう。

### ③申告書に記入する

**受け取った「給与所得者の保険料控除申告書」の「地震保険料控除」欄に、必要事項を記入します。**記入にあたっては、控除証明書に記載されている保険会社名や保険期間、実際に支払った保険料の金額などを正確に転記する必要があります。

  
  

さらに、控除対象となる建物の居住者や家財所有者の氏名、続柄などの記入も必要です。

  

[出典：国税庁「令和7年分 給与所得者の保険料控除申告書」](https://www.nta.go.jp/publication/pamph/gensen/nencho2025/pdf/307.pdf)

### ④控除証明書を添付・提出する

記入が完了した申告書に地震保険料控除証明書の原本を添付し、勤務先に提出します。**書類の提出期限は会社によって異なりますが、11月上旬〜中旬頃が一般的です。**
  
  
提出期限を過ぎてしまうと、年末調整で控除を受けることができなくなり、自ら確定申告をする必要が生じる場合があるため、期限を厳守しましょう。

### ⑤税金の還付を受けるまたは不足分を納税する

会社は、提出された書類に基づいて年末調整の計算を実施し、税金の過不足を精算します。払い過ぎていた税金があった場合は、差額が還付金として戻ってきます。**還付金は、12月の給与または翌年1月の給与と一緒に従業員に支払われるのが一般的です。**
  
  
なお、住民税の軽減分は年末調整で還付されるのではなく、翌年度の住民税額に反映され、毎月の納税額が軽減されます。

[出典：国税庁「地震保険料控除」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1145.htm)

### ①保険料控除証明書を用意する

まずは、加入している保険会社から送付される「地震保険料控除証明書」を用意します。
  
  
控除証明書は、原則として原本が必要となるため、紛失しないように保管しておきましょう。ただし、e-Tax（電子申告）を利用して確定申告をする場合、証明書の内容を正確に入力すれば、証明書の提出を省略することができます。
  
  
なお、**原則として法定申告期限から5年間、税務署等からこれらの書類の提示または提出を求められることがあるので、ご自宅などで保存しておきましょう。**

  

[出典：国税庁「所得税及び復興特別所得税についてよくある質問」](https://www.e-tax.nta.go.jp/toiawase/qa/kakutei/tempu01.htm)

### ②控除額を計算する

次に、年間に支払った地震保険料の合計額に基づいて、所得税の控除額を計算します。たとえば、地震保険料が5万円以下であれば支払額の全額が、5万円超であれば一律5万円が控除されます。

### ③確定申告書に記入する

計算した所得税の控除額を確定申告書の該当欄に記入します。**確定申告書の第一表および第二表にある「所得から差し引かれる金額」の「地震保険料控除」に控除額を記入します。**

  
  

国税庁の「[確定申告書等作成コーナー](https://www.keisan.nta.go.jp/kyoutu/ky/sm/top#bsctrl)」を利用する場合は、画面の指示にしたがって証明書の内容（保険会社名、支払保険料など）の入力が必要です。必要事項を入力することで、控除額が自動で計算され、申告書に反映されます。

  

[出典：国税庁「令和7年所得税及び復興特別所得税の確定申告の手引き」](https://www.nta.go.jp/taxes/shiraberu/shinkoku/tebiki/2025/pdf/001.pdf#page=18)

### ④申告書を提出する

記入が完了した確定申告書に地震保険料控除証明書を添えて、税務署に提出します。
  
  
申告書の提出方法には、主に以下の3つがあります。

確定申告書の提出方法

* e-Tax（電子申告）
* 郵送
* 税務署窓口への持ち込み

**提出期限は通常、翌年の2月16日～3月15日です。**

  

[出典：国税庁「確定申告を忘れたとき」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/2024.htm)

### ⑤税金の還付または納税を行う

控除を適用した結果、源泉徴収などにより過納していた税金がある場合は、後日指定した銀行口座に還付されます。
  
  
なお、住民税は確定申告の内容に基づいて翌年度の税額が計算され、軽減された住民税が課税されます。**確定申告書を提出すれば、市区町村にも情報が共有されるため、改めて住民税の申告をする必要はありません。**

  

[出典：国税庁「手順6 住民税 事業税に関する事項を記入する」](https://www.nta.go.jp/taxes/shiraberu/shinkoku/tebiki/2024/03/order6/3-6_01.htm)

## まとめ

地震保険料控除は、支払った地震保険の保険料に応じて所得税・住民税の負担を軽減できる制度です。所得税で最大5万円、住民税で最大2.5万円の控除が受けられます。
  
  
控除される金額は「控除額 × 税率」で求められます。会社員は年末調整、個人事業主は確定申告で控除証明書を提出して申告することで手続きが完了します。
  
  
制度を正しく理解して有効活用することで、災害に備えながら税負担を軽減しましょう。
