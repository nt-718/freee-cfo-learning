## 住民税とは

住民税とは、一定の所得基準を超える人が支払う地方税です。住民税には、都道府県民税と市町村民税の2種類があり、いずれも住民票のある市区町村にまとめて納付します。
  
  
住民税は所得に応じた「所得割」と一定額の「均等割」で構成され、納税者ごとに異なる金額が課されます。

## 住民税の徴収方法

住民税の徴収方法には、「特別徴収」と「普通徴収」の2種類があります。会社員は基本的に特別徴収、個人事業主は普通徴収で住民税を支払います。

### 特別徴収

特別徴収は、企業などが納税義務者の給与から住民税を天引きし、本人の代わりに納税する方法です。主に会社員や公務員が特別徴収の対象です。
  
  
企業には、毎年5月から6月頃に住民税決定通知書（事業所用・従業員用）が届きます。住民税決定通知書は、前年1年間の所得から計算された住民税の金額が記載されているものです。
  
  
住民税決定通知書の金額をもとに、6月から翌年5月にかけて1/12ずつ給与から天引きされ、企業がまとめて納付します。

### 普通徴収

普通徴収は、納税義務者本人が住民税を直接納付する方法です。主に個人事業主などが普通徴収の対象です。
  
  
確定申告や住民税申告をもとに毎年6月頃に納税通知書が届き、年4回（6月・8月・10月・翌年1月）に分けて納付します。住民税の納付は、金融機関・コンビニ・口座振替・スマホ決済などから可能です。

## 住民税はいつからいつまでの収入が対象？

住民税は前年1月1日から12月31日までの所得をもとに計算され、6月から翌年5月の1年間に課税・徴収されます。
  
  
たとえば、2024年1月1日から12月31日までの所得に対しては、2025年6月から2026年5月まで課税・徴収されます。

## 会社員の住民税はいつから（何月から）引かれる？

会社員の住民税がいつから天引きされるかは、入社1年目と2年目、そして転職のタイミングで異なります。住民税は前年の所得に応じて課税されるため、新卒と転職の場合に分けて、天引きが始まる時期を解説します。

### 新卒会社員の住民税はいつから引かれる？

**新卒で入社すると2年目の6月に支給される給与から住民税が差し引かれます。**たとえば、給料日が6月25日であれば、その日の給与から住民税の天引きが始まります。
  
  
多くの企業では、6月に支給される給与は前月（5月）の勤務分ですが、住民税は支給月である6月の給与から引かれる仕組みです。つまり、勤務月ではなく支給月を基準に天引きが始まります。

#### 社会人1年目も住民税が徴収されるケースがある

社会人1年目は、原則として前年の所得がないため、住民税の支払いはありません。ただし、アルバイトなどで110万円を目安とする金額を超える収入が前年にあった場合は、翌年に住民税が課される場合があります。
  
  
この場合、住民税は就職先での天引き（特別徴収）ではなく、自治体から本人宛に届く納付書で支払う普通徴収となるのが一般的です。
  
  
多くの自治体では扶養がない場合、所得が45万円以下であれば住民税は非課税です。給与所得控除の最低金額65万円を考慮すると、給与収入が110万円を超えるあたりが、住民税が発生する目安となります。
  
  
扶養親族の有無などにより、非課税となる収入基準が変わる場合もあります。

### 転職後の住民税の天引きはいつから？

転職先で住民税の天引きが始まるのは通常、**次の6月から**です。勤務開始月を基準にすると、以下のタイミングで住民税の天引きが始まります。

| 勤務を開始した月 | 住民税の天引きが始まる時期 |
| --- | --- |
| 1月～5月 | 当年6月の給与から |
| 6月～12月 | 翌年6月の給与から |

[出典：東京都主税局「個人住民税と特別徴収について」](https://www.tax.metro.tokyo.lg.jp/kazei/life/kojin_ju/tokubetsu/about)
  
[出典：神奈川県「個人住民税の特別徴収について」](https://www.pref.kanagawa.jp/zei/kenzei/a001/b003/002)

ただし、**退職前に異動届を受け取り自治体へ提出すれば、住民税の特別徴収を転職先でも継続可能**です。届出書は転職先に手渡し、転職先が記入・提出して手続きを行います。
  
  
この場合、手続きが処理されるタイミングによって、住民税の天引きが開始される時期は異なります。届出をしない場合と比べると、早い時期から天引きが開始されます。

#### 転職前の職場での住民税はどうなる？

転職前の職場での住民税は、一括天引きまたは普通徴収として処理されます。
  
  
**【退職が1月から5月のケース】**

|  |  |
| --- | --- |
| 退職日 | 2025年3月1日の場合 |
| 退職日を含む住民税の課税期間 | 2024年6月から2025年5月 |
| 退職時点で天引きされていない住民税 | 2025年3月から5月分 |
| 住民税の徴収方法 | 最終給与や退職金から一括で天引き |

前々年の所得に基づいて課税された住民税の残りは、退職前の職場での最終給与や退職金から一括で天引きされます。
  
  
2023年の所得に基づき2024年6月から2025年5月に課税された住民税のうち、2025年3月から5月分が天引き前の住民税として残っている形です。
  
  
これらが最終給与や退職金から一括で天引きされます。
  
  
なお、給与や退職金から控除しきれない場合は普通徴収に変更され、後日届く納付書によって納付する必要があります。
  
  
**【退職が6月から12月のケース】**

|  |  |
| --- | --- |
| 退職日 | 2025年7月1日の場合 |
| 退職日を含む住民税の課税期間 | 2025年6月から2026年5月 |
| 退職時点で天引きされていない住民税 | 2025年7月から2026年5月分 |
| 住民税の徴収方法 | 普通徴収に切り替え（または退職前の職場で一括天引き） |

一方、退職が6月から12月の場合は、前年の所得に基づいて課税された住民税の残りは、普通徴収に切り替わるか、希望すれば退職前の職場で一括天引きされます。
  
  
2024年の所得に基づき2025年6月から2026年5月に課税される住民税のうち、2025年7月分から2026年5月分が天引き前の住民税として残っています。
  
  
これらは普通徴収として支払うか、希望すれば退職前の職場で一括天引きされます。

## 個人事業主はいつから住民税を支払う？

個人事業主も前年の所得をもとに住民税を支払います。そのため、新たに事業を開始し、開業前の年に所得がない場合は2年目以降から住民税の支払いが必要です。
  
  
個人事業主の場合、確定申告や住民税申告をもとに6月中頃に納税通知書が市区町村から届きます。その後、年4回に分けて住民税を納付します。

## 住民税はいくら納付する？

住民税は「所得割」と「均等割」を合計した金額を納付します。所得割と均等割の概要は以下のとおりです。納付する金額は納税者ごとに異なります。

|  |  |
| --- | --- |
| 所得割 | 税率10%で所得金額に応じて負担 |
| 均等割 | 所得金額に関わらず定額で負担（市町村民税3,000円+道府県民税1,000円+森林環境税1,000円） |

[出典：総務省「個人住民税」](https://www.soumu.go.jp/main_sosiki/jichi_zeisei/czaisei/czaisei_seido/150790_06.html)

## 住民税が非課税になる条件

生活保護を受けている人や所得水準などの一定の条件を満たす人は、住民税が非課税です。たとえば東京23区の場合、以下に該当する人は所得割・均等割ともに非課税となります。

住民税が非課税になる条件（東京23区の場合）

* 生活保護法による生活扶助を受けている方
* 障害者・未成年者・寡婦またはひとり親で、前年中の合計所得金額が135万円以下  
  （給与所得者の場合は、年収204万4千円未満）の方
* 前年中の合計所得金額が区市町村の条例で定める額以下の方

住民税が非課税になる所得の基準額は以下のとおりです。

|  |  |
| --- | --- |
| 同一生計配偶者または扶養親族がいる場合 | 35万円 ×（本人・同一生計配偶者・扶養親族の合計人数）+ 31万円 以下 |
| 同一生計配偶者または扶養親族がいない場合 | 45万円以下 |

[出典：東京都主税局「個人住民税」](https://www.tax.metro.tokyo.lg.jp/kazei/life/kojin_ju)

なお、非課税となる基準額は自治体ごとに異なります。

## 期限までに納付が難しいときの対応方法

期限までに納付が困難なときは、早めに市区町村の税務課へ相談しましょう。収入や家計の状況を説明すれば、分割納付や納付の猶予などの提案を受けられることがあります。
  
  
**住民税を支払わないまま放置すると督促状が届き、延滞金が加算されます。**さらに、場合によっては預貯金・給与・不動産などの財産が差し押さえられることもあります。期限までに納付が難しいと判断した時点で、すみやかに対応しましょう。

一方、個人事業主の場合は、住民税決定通知書とともに届く納付書を使い、自身で普通徴収により納付します。住民税の納付を開始する時期を理解し、適切に住民税を支払いましょう。

### 住民税の特別徴収はいつから天引きされる？

新卒で入社すると、2年目の6月から住民税が差し引かれます。
