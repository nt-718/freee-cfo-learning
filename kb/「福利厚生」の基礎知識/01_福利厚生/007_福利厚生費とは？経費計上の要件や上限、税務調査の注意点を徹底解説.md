## 福利厚生とは？節税につながる従業員のための費用

**福利厚生とは、法人が従業員やその家族のために支払う給与や賞与以外のサービス・制度です。**従業員の生活やモチベーションを向上させたり、より働きやすい環境を整えたりすることを目的としています。
  
  
この**福利厚生には、社会保険など法律で企業に義務付けられている「法定福利」と、社宅や食事補助など、企業が独自に設ける「法定外福利」という2種類があります。**これらにかかる費用は、経費（損金）として計上できるため、法人税の節税につながるのが通常です。
  
  
ただし会計実務上、法律で義務付けられた費用は「法定福利費」、企業が任意で設定するものは「福利厚生費」と、勘定科目を分けて処理します。
  
  
一般的に「節税」や「経費計上の要件」として議論されるのは、企業の裁量で決められる「法定外福利（=福利厚生費）」のことです。本記事では、主にこの「福利厚生費」について、経費として認められる要件などを詳しく解説します。

### 法定福利費と法定外福利費の違い

福利厚生費は、法律で義務付けられている「法定福利費」と、企業が独自に設ける「法定外福利費」の2種類に大別されます。
  
  
**法定福利費とは、法律で企業に義務付けられている福利厚生にかかる費用**です。具体的には以下のものが該当します。

法定福利費の例

* 健康保険料
* 厚生年金保険料
* 介護保険料
* 雇用保険料
* 労災保険料

法定外福利費とは、企業が任意で提供する福利厚生にかかる費用です。法律上の義務はありませんが、従業員満足度やエンゲージメントの向上などを目的に多くの企業が導入しています。

法定外福利費の例

* 社宅
* 食事補助
* 通勤手当
* 健康診断
* 社員旅行
* 慶弔見舞金

### 交際費・消耗品費との違い

福利厚生費と似た経費に、「交際費」と「消耗品費」があります。**これらの費用との主な違いは、「対象者」と「業務との関連性」**です。

| 勘定科目 | 対象者 | 業務との関連性 |
| --- | --- | --- |
| 福利厚生費 | 自社の従業員 | 業務に直接は関係しない（例：社宅の家賃補助など） |
| 交際費 | 取引先など社外の人物 | 業務に関連する（例：クライアントとの会食費など） |
| 消耗品費 | （問わない） | 業務に直接必要になる物品（例：PC、文房具など） |

たとえば、**従業員との食事会は福利厚生費ですが、クライアントとの会食は交際費として計上します。**
  
  
また、消耗品費は原則として使用期間が1年未満、かつ取得価額が10万円未満のものが該当します。

## 福利厚生費として認められる4つの大原則

福利厚生費として費用に計上するには、以下4つの要件を満たしている必要があります。

福利厚生費として計上する要件

1. 給与ではないこと
2. 現金もしくは、換金性の高いものではないこと
3. 全従業員が利用できること
4. 社会通念上、金額が妥当であること

まず**福利厚生費は給与や現金、もしくは換金性の高いものではないことが条件**です。たとえば、勤続年数などの表彰で渡す現金や商品券は、福利厚生費とは認められません。現金や商品券を支給する場合は「給与」として扱います。
  
  
同様に、**一見すると福利厚生費のように思える住宅手当も福利厚生費に該当しません。**こちらは給与にプラスして支払われるため、給与として計上します。
  
  
また福利厚生費として計上するには、全従業員が平等に利用でき、かつ社会通念上妥当とされる金額である必要があります。一部の従業員しか参加できない飲み会や、高額な海外旅行の費用は福利厚生費として計上することはできません。

## 福利厚生費として認められる費用と上限の目安

ここからは、多くの法人で採用されている代表的な福利厚生の上限や福利厚生費として計上するための要件を紹介します。

### 通勤手当

**通勤にかかる費用を補助する通勤手当には上限が設定されており、上限を超える分は課税の対象となります。**
  
  
電車やバスなどの公共交通機関を利用している場合、1ヶ月あたり15万円までは非課税です。マイカーで通勤している場合は、走行距離で上限が異なります。片道の通勤距離が2km未満までは全額課税となりますが、2km以上になると上限金額が発生します。

#### マイカー通勤の上限金額

| マイカー通勤の場合に非課税となる1ヶ月当たりの限度額 | |
| --- | --- |
| 片道の通勤距離 | 1ヶ月あたりの上限額 |
| 2km未満 | 全額課税 |
| 2km以上10km未満 | 4,200円 |
| 10km以上15km未満 | 7,100円 |
| 15km以上25km未満 | 12,900円 |
| 25km以上35km未満 | 18,700円 |
| 35km以上45km未満 | 24,400円 |
| 45km以上55km未満 | 28,000円 |
| 55km以上 | 31,600円 |

[出典：国税庁「No.2585 マイカー・自転車通勤者の通勤手当」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/gensen/2585.htm)

なお、福利厚生費の要件として「現金もしくは、換金性の高いものではない」とありますが、通勤手当は例外的に現金での支給が認められています。

### 社宅

給与にプラスして支払う住宅手当は福利厚生費に計上できませんが、**法人が所有もしくは契約を結んだ賃貸物件に従業員を住まわせる場合は、家賃を福利厚生費として計上できます。**
  
  
税法上、上限は定められていません。ただし、福利厚生費として計上するのであれば、一般的な範囲に収めたほうが無難です。
  
  
2020年12月に日本経済団体連合会が発表した「[第64回福利厚生費調査結果報告](https://www.keidanren.or.jp/policy/2020/129_honbun.pdf#page=8)」によると、従業員1人1ヶ月あたりの住宅関連法定外福利費は住宅及び持家援助として11,639円となっています。構成としては法定外福利内の48.2％です。
  
  
また、家賃を福利厚生費として非課税にするには「賃貸料相当額の50％以上」を、従業員から家賃として受け取っている必要があります。
  
  
賃貸料相当額の計算方法は、以下のとおりです。

賃貸料相当額の計算方法

① （その年度の建物の固定資産税課税標準額）× 0.2％
  
② 12円 ×（その建物の総床面積（平方メートル）／ 3.3（平方メートル））
  
③ （その年度の敷地の固定資産税の課税標準額）× 0.22％
  
④ ① + ② + ③ ＝ **賃貸料相当額**
  
  

[出典：国税庁「No.2597 使用人に社宅や寮などを貸したとき」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/gensen/2597.htm)

注意したいのは「賃貸料相当額＝家賃」ではないということです。賃貸料相当額を導き出すには固定資産税の課税標準額を調べ、その上で複雑な計算をする必要があります。

### 健康診断

**一般的な健康診断の費用も福利厚生費として計上できます。**地域によって違いますが、一人あたり7,500～1万2,000円が一般的です。とくに上限はありませんが、PET検診や高額なオプションなど、あまりにも費用が高いものは経費として認められません。2万円程度までなら、問題なく計上できるでしょう。
  
  
**福利厚生費として計上するためには、従業員全員が受診できることが必須**です。ただし「オプション検査などは、健康保険組合指定の予防対象者に限る｣など、合理的な範囲内で制限を設けることは可能です。
  
  
なお、支払いは必ず法人名義で行いましょう。健康診断にかかった費用を従業員に支払ってもらい、その分を給与に上乗せして支給すると給与と見なされ、福利厚生費にできません。

### 食事補助

**社食で食事を提供したり、お弁当を頼んだりする食事補助は、福利厚生費の上限が1人あたり月3,500円以下と定められています。**食事補助を福利厚生費と認めてもらう要件は、以下の2つです。

食事補助が福利厚生費として認められる要件

1. 従業員が食事の半分以上の価格を負担していること
2. 現物で支給していること

たとえば月に5,000円の食事をして、そのうちの2,500円を従業員が負担していれば、その差額は福利厚生費として計上可能です。仮に従業員が2,000円しか負担していなかった場合は、差額の3,000円は給与となり課税の対象となります。
  
  
先述した「第64回福利厚生費調査結果報告」では、食事に関連する法定福利費は給食として、1人1ヶ月あたり1,729円です。基本的に上限を超えなければ問題はありませんが、飲酒が含まれている場合には業務内の食事と認められず、福利厚生費に計上できない場合があるので注意しましょう。
  
  
なお、深夜勤務者向けに現物での支給が難しい場合に限り、現金での支給が認められています。この場合は税抜300円までの支給が可能です。

### 社員旅行

社員旅行の費用も、以下の要件を満たせば福利厚生費として計上可能です。

社員旅行が福利厚生費として認められる要件

1. 従業員全員が参加可能で、50％以上が参加している
2. 4泊5日以内（海外の場合は現地での滞在日数）の旅行
3. 不参加の従業員に対して金銭を支給しない

**旅行費用に関しては、とくに上限も設けられていません。**
  
  
ただし、あまり高額な費用の計上は難しいため、1人あたり10万円程度の計上に留め、残りは自己負担してもらうのがよいでしょう。

### 慶弔見舞金

**従業員本人や親族の結婚祝い金、災害見舞金といった慶弔費は、例外的に現金での支給が認められている福利厚生費**です。ただし、経費（損金）として計上するためには規程を作成しなければなりません。
  
  
とくに上限はありませんが、結婚祝い金や出産祝いなどの慶事は、1万～3万円程度が一般的です。死亡慶弔金や災害見舞金などは相手の状況によって変動しますが、大体1万～10万円と考えておきましょう。

## 税務調査で福利厚生費として認められないケース

福利厚生費は節税効果が高い反面、税務調査で要件を満たしていないと判断され、「給与」として認定（「福利厚生費」としては否認）されるケースが少なくありません。給与と認定されると追徴課税の対象となります。
  
  
ここでは、福利厚生費で間違いやすいケースと注意点を解説します。

### 一部の役員・従業員しか利用できない

**福利厚生費として認められない最も多い理由が、「機会の均等」の原則を満たしていないケース**です。
  
  
具体的には以下のようなものが挙げられます。

「機会の均等」の原則を満たさない例

* 役員だけで行く海外旅行やゴルフ
* 特定の部署や役職者のみが利用できる保養所
* 希望者のみに支給される食事手当（全従業員が利用できる制度になっていない）

福利厚生は、あくまで全従業員が平等に利用できる機会が提供されている必要があります。結果的に一部の従業員しか利用しなかったとしても、「希望すれば誰でも利用できる」状態にしておく点がポイントです。

### 社会通念上、金額が高すぎる

**福利厚生の内容が常識の範囲を逸脱しており、過剰に優遇されていると判断された場合も否認の対象**となります。
  
  
具体的には以下のようなケースです。

社会通念上、金額が高すぎる例

* 1名あたり数十万円もかかる豪華な海外旅行
* 高額なブランド品などの記念品
* 役員社宅の家賃を会社が全額負担している

「社会通念上」の明確な金額の目安はありませんが、一般的な企業の福利厚生レベルを大きく超えないことがひとつの目安となります。

### 現金や換金性の高いものを支給している

通勤手当や一部の慶弔見舞金などの例外を除き、**現金や商品券、ギフトカードなどの換金性の高いものを支給すると、給与とみなされます。**

現金や換金性の高いものの例

* 食事補助として毎月5,000円の現金を支給する「食事手当」
* 創業記念として全従業員に1万円分の商品券を配布する
* 忘年会の景品として高額なギフトカードを用意する

これらはすべて給与課税の対象となるため、注意が必要です。

## 福利厚生費として認められるための「社内規程」の重要性

**税務調査で福利厚生費を正しく主張し、否認されるリスクを避けるためには「福利厚生規程」を整備しておくことが極めて重要**です。
  
  
社内規程は、以下の点で効力を発揮します。

* 公平性の証明：全従業員を対象とした制度であることを客観的に証明できる
* 妥当性の根拠：支給基準や金額が社内で統一されたルールに基づいていることを示している
* 税務調査への備え：調査官に対して、福利厚生費の正当性を明確に説明できる

**規程には、「目的」「対象者」「利用条件」「支給内容・金額の上限」「申請手続き」などを具体的に明記しましょう。**規程を作成し、全従業員に周知することで、福利厚生費としての正当性が格段に高まります。

## 福利厚生費がもたらす節税対策

**福利厚生費は、法人の節税対策に有効な手段**です。福利厚生費を適切に活用することは、企業と従業員の双方にメリットをもたらします。

企業側のメリット（法人税の負担軽減）

福利厚生費は経費（損金）に算入できるため、課税対象となる所得を圧縮でき、法人税の負担が軽減されます。

従業員側のメリット（所得税・社会保険料の負担軽減）

たとえば給与として現金（住宅手当など）を受け取ると、その分所得が増え、所得税や社会保険料の負担も増加します。一方で、福利厚生（社宅制度など）として非課税のサービスを受ける場合は、従業員の課税所得は増えません。可処分所得を維持したまま生活の質を向上でき、従業員満足度の向上につながります。

ただし、社宅制度などで給与から天引きされる金額が増えると、社会保険料報酬額の差により、将来受け取れる年金額に影響が出る可能性もあります。制度設計の際は、従業員の不利益にならないようにバランスを考慮しましょう。
  
  
また、要件に該当しないものを福利厚生費として計上してしまうと、あとから所得税として課税され、追徴金を支払うことになります。事前に税理士や税務署に相談するなど、慎重な対応が必要です。

## 節税効果の高い福利厚生の例

節税効果の高い福利厚生として、「社宅制度」「法人保険」があります。

### 社宅制度

社宅制度は、福利厚生の中でもとくに節税効果の高い福利厚生です。先述したように、住宅関連費用は法定外福利厚生費用の約半分を占めており、**社宅制度を導入することで、福利厚生費に計上できる金額が大きく増えます。**
  
  
社宅制度には、自社の所有する物件を従業員に貸し出す「所有社宅」と、自社で契約した賃貸物件を貸し出す「借り上げ社宅」の2つがあります。
  
  
所有社宅は従業員との賃貸契約が容易ですが、管理や修繕といったランニングコストがかかります。一方、借り上げ社宅はランニングコストが必要ありませんが、賃貸契約が煩雑であることがデメリットです。どちらも一長一短あるため、自社に合った方を選びましょう。

### 法人保険

**法人保険は保険料を福利厚生費として計上できるため、節税につながります。**解約返戻金がある定期型の法人保険では、解約時に返戻金ももらえるため、一石二鳥の節税効果があります。
  
  
ただし、**保険料の全額を福利厚生費に計上できるのは、最高解約返戻率（ピーク時の返戻金）が50％までの場合**です。もし50％を超える場合には、一定の期間、保険料の一部を資産として計上しなければなりません。
  
  
たとえば最高解約返礼率が50％超70％以下の場合、保険期間の開始日から保険期間の4割が経過するまで、支払っている保険料の4割を資産に計上します。ただし、上記の期間を過ぎれば全額損金が認められるため、解約までの期間や解約返戻金の割合によって、大きな節税効果が期待できます。

## まとめ

福利厚生費をうまく利用することで、法人にも従業員にも大きなメリットとなります。有効的に利用するには、認められる要件を守り、計上できる範囲を理解しておくことが重要です。
  
  
もし福利厚生費に計上できるか迷った場合には、税理士や税務署に問い合わせましょう。

## よくある質問

### 福利厚生費に上限額はある？

福利厚生費そのものに法律上の上限額はありません。ただし、通勤手当や食事補助など、個別の項目ごとに非課税となる上限額が定められています。また、社会通念上、金額が妥当である必要があり、高額すぎる場合は否認されるリスクがあります。
  
  
詳しくは、記事内の「[福利厚生費として認められる費用と上限の目安](#content3)」をご覧ください。

### 役員だけの食事会は福利厚生費になる？

役員だけの食事会は福利厚生費として認められません。「給与」または「交際費」として扱われます。福利厚生費は、全従業員が平等に利用できる機会が提供されている必要があるためです。
  
  
詳しくは、記事内の「[税務調査で福利厚生費として認められないケース](#content4)」で解説しています。

### 福利厚生費として経費計上するために最も重要なことは？

「全従業員が対象である」かつ「社会通念上、妥当な金額である」という要件を満たす必要があります。また、税務調査で否認されるリスクを避けるため、支給基準や上限額を定めた「福利厚生規程」を作成し、全従業員に周知しておくことが求められます。
  
  
詳しくは、記事内の「[福利厚生費として認められるための「社内規程」の重要性](#content5)」をご覧ください。
