## クリニック承継とは？新規開業と承継開業の違い

**クリニック継承とは、既存クリニックの経営権や有形・無形の資産（医療機器・スタッフ・患者など）を、後継者が引き継ぐことです。**
  
  
クリニック開業には大きく分けて「新規開業」と「承継開業」の2つの選択肢があります。
  
  
それぞれのメリットとデメリットは以下のとおりです。

  

| 項目 | 新規開業 |
| --- | --- |
| メリット | ・院長の理念を100％反映できる |
| ・立地や診療科目を選択できる |
| ・最新機器を導入できる |
| デメリット | ・内装や機器への多額の初期投資がかかる |
| ・ゼロからの集患が必要になる |
| ・ゼロからのスタッフ採用・教育が必要になる |

  

| 項目 | 承継開業 |
| --- | --- |
| メリット | ・初期投資を抑えられる |
| ・短期間で運営開始できる |
| ・スタッフや取引先との関係を継続できる |
| デメリット | ・既存問題を引き継ぐリスクがある |
| ・前院長の方針や慣習に拘束されることがある |
| ・設備老朽化への対応が必要になることがある |

  

新規開業はクリニックのコンセプトや内装・設備などをゼロから自由に設計できますが、初期費用が高額になりやすいです。一方、承継開業は新規開業に比べて初期のリスクを抑え、スムーズなスタートが期待できる開業方法です。
  
  
両者のメリット・デメリットを正しく理解し、自身にあった選択をすることが重要です。

## クリニック承継にかかる費用

クリニックの承継費用は、資産価値と収益力によって大きく変動し、さらに各種税金も発生します。承継にかかる費用の相場と、見落としがちな税金について解説します。

### クリニック承継の費用相場

クリニック承継にかかる費用の相場は、以下の「時価純資産」と「営業権（のれん代）」を合計して算出されます。

#### 時価純資産

時価純資産とは、土地・建物や医療機器といったプラスの資産から、負債を差し引いたものを指します。なお、個人クリニックでは負債は譲渡されない可能性があります。

#### 営業権（のれん代）

営業権（のれん代）は、地域の評判や患者数といった目に見えない価値を換算したもので、一般的には利益の3年～5年分が目安といわれています。
  
  
時価純資産が影響することから、医療機器が高額になりやすい眼科や整形外科や、自由診療の割合が高い美容皮膚科などは、承継価額が高くなる傾向があります。

### クリニック承継にかかる税金

クリニック承継の際には、承継の形態に応じてさまざまな税金が発生するため、事前の納税資金計画が不可欠です。
  
  
親子承継と第三者承継で必要になりうる税金は以下のとおりです。

  

|  | 譲渡側（売り手） | 譲受側（買い手） |
| --- | --- | --- |
| 親子承継 | ー | ・贈与税 ・相続税 |
| 第三者承継 | 譲渡所得税 | ・消費税 （医療機器、営業権の譲渡に対するもの） ・登録免許税 ・不動産取得税 |

  

これらの税金は承継時の費用に大きく影響することから、事前に税理士のような専門家に相談し、税務シミュレーションを行うことが重要です。

## クリニックの承継に使える補助金・融資制度

クリニック承継には多額の資金が必要になります。国や金融機関が用意する補助金や融資制度を賢く活用し、資金調達の負担を抑えることが大切です。
  
  
クリニックの承継に使える補助金・融資制度には、以下のようなものが存在します。

* [日本政策金融公庫：事業承継・集約・活性化支援資金](https://www.jfc.go.jp/n/finance/search/jigyoukeisyou_t.html)
* [中小企業庁：事業承継・M&A補助金](https://jsh.go.jp/r6h/?utm_source=chatgpt.com)

日本政策金融公庫の「事業承継・集約・活性化支援資金」は比較的低い金利の融資であるため、長期の返済計画を立てやすいのが特徴です。
  
  
一方、中小企業庁の「事業承継・M&A補助金」は、承継を機に新たな取り組みを行う際の経費や、専門家への依頼費用の一部が補助される制度です。
  
  
公募期間や要件が定められているため、常に最新の情報をチェックしましょう。

## クリニック承継の手続き【失敗しない4つのステップ】

クリニック承継の手続きを円滑にするには、案件探しから承継後の運営まで、計画的に進める必要があります。具体的な流れとしては以下のとおりです。

クリニック承継の流れ

* [STEP1. 承継案件をリサーチ（第三者承継の場合のみ）](#content4-1)
* [STEP2. 契約書作成](#content4-2)
* [STEP3. 行政手続き](#content4-3)
* [STEP4. 承継後の運営](#content4-4)

なお、承継相手が決まっている親子承継であればSTEP11は不要のため、STEP2から読み進めてください。

### STEP1. 承継案件をリサーチ（第三者承継の場合のみ）

第三者承継を成功させる第一歩は、自身の理念や希望条件にあった承継案件を見つけ出すことです。
  
  
承継案件を探す場所として、以下のような場が挙げられます。

* 地域の医師会
* M&A仲介会社
* 医薬品卸業者や医療機器メーカー
* 取引のある金融機関
* M&Aマッチングプラットフォーム

承継案件を検討する際は、立地条件・患者数・収益性など、多角的な視点から評価することが大切です。
  
  
興味のある案件が見つかったら、双方が安心して交渉を進められるよう、機密情報を開示してもらう前に秘密保持契約（NDA）を締結するのが一般的です。
  
  
複数の案件を比較検討し自身の理念や将来ビジョンに合致するクリニックを慎重に選定しましょう。

### STEP2. 契約書作成

第三者承継はもちろん、親子承継であっても、思わぬトラブルを防ぐため、弁護士などの専門家を交えて法的に有効な契約書を作成することが重要です。
  
  
第三者承継では、建物や営業権などの資産評価を行い、その結果を基に譲渡価格や条件を交渉します。
  
  
大規模のクリニックを承継する際は公認会計士による買収監査（デューデリジェンス）を行いましょう。
  
  
交渉がまとまったら、弁護士に依頼して、引き継ぐ資産の範囲や万が一のリスクを明記した事業譲渡契約書を作成・締結します。
  
  
とくに、承継後に発覚した債務や問題の取り扱いは、しっかりと規定しておくようにしましょう。

### STEP3. 行政手続き

契約締結後は、各種行政手続きを行います。承継するのが個人クリニックか医療法人かによって手続きが大きく異なります。
  
  
それぞれ提出が必要な書類は以下のとおりです。

  

| 区分 | 提出が必要な書類 |
| --- | --- |
| 個人クリニック | 【譲渡側（売り手）】  保健所：診療所廃止届  厚生局：保険医療機関廃止届    【譲受側（買い手）】  保健所：診療所開設届  厚生局：保険医療機関指定申請 |
| 医療法人 | 【手続きの義務は医療法人に帰属】  保健所：診療所開設許可事項一部変更届  所轄の都道府県：登記事項変更完了届、役員変更届  法務局：医療法人変更登記申請書  厚生局：保険医療機関届出事項変更届 |

  

行政手続きにはそれぞれ提出期限が設けられているため、事前に管轄の機関に相談しながら計画的に準備を進めましょう。

### STEP4. 承継後の運営

第三者承継はもちろん、親子承継であっても院長の交代を患者に告知し、前院長と新院長が一緒に診療する引き継ぎ期間を設けることが理想です。
  
  
スタッフとの関係を構築するには、新院長の理念の説明や雇用条件の確認を行い、不安を解消することが大切です。
  
  
また、承継直後は患者数が一時的に減少することもあるため、以下のように新体制を周知する取り組みも積極的に行いましょう。

* HPのリニューアル
* 地域への挨拶回り
* 内覧会の開催　など

前院長からの推薦や紹介があれば、患者の安心感も高まりスムーズな移行が期待できます。

## クリニック承継でよくあるトラブル・失敗例

クリニック承継はメリットが大きい一方で、特有の失敗も存在します。承継を成功させるためには、起こりうるトラブルと対策を事前に知っておくことが大切です。
  
  
ここでは、クリニック継承でよくあるトラブルや失敗例について解説します。

### 親子承継：相続・財産分与に関するトラブル

親子承継でよくあるのが、後継者以外の兄弟姉妹から「クリニックをひとりで相続するのは不公平」として、その資産価値に見合った遺産を要求されトラブルに発展するケースです。
  
  
相続トラブルを回避するには、親（現院長）が元気なうちに専門家を交えた相続対策を行うことが不可欠です。
  
  
後継者にクリニックを承継させる旨を明記した遺言書を作成したり、計画的に生前贈与を進めたりすることで、他の相続人との間で事前に合意形成を図っておきましょう。

### 第三者承継：契約条件に関するトラブル

第三者承継でよくあるのは、契約書に記載されていなかった偶発債務（未払いの残業代など）が、承継後に発覚して想定外の負担を強いられるケースです。
  
  
契約条件に関するトラブルを回避するには、承継先の徹底的な事前調査が欠かせません。
  
大規模のクリニックを承継する際は、専門家による徹底した買収監査（デューデリジェンス）を行うことが不可欠です。
  
  
財務状況や法務リスクを洗い出し、契約書には「開示した情報が真実である」ことを保証させる「表明保証」などの条項を必ず盛り込みましょう。

### 親子承継・第三者承継：スタッフや患者の引き継ぎに関するトラブル

どんな承継でも起こりうるのは、院長の交代を機に、前院長を信頼していた古参スタッフと患者が離れてしまい、売上が大幅に減少するケースです。
  
  
引き継ぎに関するトラブルを回避するには、承継前からスタッフや患者とコミュニケーションを取る機会を設けることが重要です。
  
  
承継後も前院長に非常勤として一定期間勤務してもらい、患者やスタッフの不安を和らげることもひとつの手段です。

## よくある質問

### クリニック継承は誰でもできる？

医療法人化しているクリニックの「経営権のみ」を継承するのであれば誰でも可能です。
  
  
一方、個人クリニックや医療法人化しているクリニックの「院長」としてクリニックを継承する際は、必ず医師免許が必要になるため、誰でもできるわけではありません。
  
  
この原則は、親子間・第三者承継のどちらの方法であっても適用されます。クリニック継承を検討する上での重要な前提として、正しく理解しておきましょう。
  
  
なお、クリニック継承をする際は、思わぬトラブルを防ぐため、法的に有効な契約書を作成することが重要です。

### クリニック継承とは何ですか？

そもそも「継承」は、業務や財産といった具体的なものを受け継ぐ際に使われる言葉です。
  
クリニック継承というと、既存クリニックの経営権や有形・無形の資産（医療機器・スタッフ・患者など）を、後継者が引き継ぐことを指します。
  
  
なお、事業を引き継ぐ際は、先代の事業に対する想いや信念を含めて引き継ぐという意味で「承継」という言葉が用いられるのが一般的です。
  
  
クリニック承継に関しては「[クリニック承継とは？新規開業と承継開業の違い](#content1)」をチェックしてください。

### クリニックの承継手続きは？

クリニックの承継の際は、以下の4つのステップに沿った流れが一般的です。

1. 承継案件をリサーチ（第三者承継の場合のみ）
2. 契約書作成
3. 行政手続き
4. 承継後の運営

承継するのが個人クリニックか医療法人か、また承継相手が親か第三者かによって、手続きは大きく異なります。手続きは非常に複雑なため、弁護士や行政書士、開業支援コンサルタントなどの専門家のサポートを受けながら進めるのが一般的です。

### クリニックの承継にかかる費用はいくらですか？

クリニックの承継にかかる費用は、規模や地域によって数千万円～数億円と幅広いため、明確な金額を示すことは困難です。
  
  
費用は、主に「時価純資産（土地・建物・医療機器など）」「営業権（のれん代）」を基に算出されます。
  
  
営業権（のれん代）は、クリニックの収益力や将来性を評価した無形資産であり、交渉の大きなポイントとなります。
  
  
費用の詳細に関しては「[クリニック承継にかかる費用](#content2)」をチェックしてください。

### クリニックが移転する理由は何ですか？

クリニックが移転する主な理由には、以下のようなものが考えられます。

* 施設の老朽化：患者が満足できる対応が困難になった
* 事業の拡張：より広い物件に移転して診療科や設備を増強する
* 承継に伴う移転：後継者の意向により地域を変える必要があった
* 立地条件の悪化： 周辺地域の再開発や人口動態の変化により、集患が難しくなった

いずれの理由でも、移転は大きな決断であり、スタッフや患者への影響を最小限に抑える配慮が必要です。
