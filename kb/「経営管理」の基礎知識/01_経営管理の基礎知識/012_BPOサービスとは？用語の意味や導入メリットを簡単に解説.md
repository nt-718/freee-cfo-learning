## BPOとは

**BPOとはビジネス・プロセス・アウトソーシングの略称で、企業が行う業務プロセスの一部について、企画から実行までを一括して外部の専門事業者に委託することです。**
  
  
BPOは人事・総務・経理など間接部門の業務などを筆頭に、特定の専門性やノウハウが必要な場合、社内のリソースが不足している場合に活用されます。**単一のタスクではなく、業務プロセスごと外部の専門事業者に委託できることから、業務改善や経営資源の最適化を目的に導入され、企業の経営課題の解決にも役立ちます。**
  
  
BPOを活用して間接業務を外部に任せることで、企業の従業員は本来行うべき業務に集中しやすくなり、業務品質や成果の向上、コスト削減が期待できます。

### アウトソーシングとの違い

BPOと似た言葉に、「アウトソーシング」があります。BPOもアウトソーシングの一種といえますが、アウトソースする範囲や目的などに違いがあります。
  
  
**一般的に「アウトソーシング」と呼ばれるものは、業務プロセスのうちの一部の業務だけを委託します。**対して、BPOは業務プロセス全体（企画から実行まで）を専門事業者にすべて外部委託するものです。
  
  
また、アウトソーシングは人手が足りない既存業務や臨時業務などを一時的に外部委託する目的で導入されるケースが多い点でもBPOとは異なります。

### シェアードサービスとの違い

同様にBPOと似た仕組みとして、「シェアードサービス」があります。
  
  
**シェアードサービスとは、複数のグループ企業が経理・総務・人事などの間接業務を集約して共有することで、グループ企業全体のコストの削減や効率化を図るものです。**業務をグループ企業内の組織に委託するという点で、BPOとは異なります。

  
  

シェアードサービスについて詳しく知りたい方は、別記事「[シェアードサービスとは？BPOとの違いや成功事例を紹介](https://www.freee.co.jp/kb/kb-business-administration/shared-service/)」をご覧ください。

## BPOの対象業務

BPOの対象となる業務は、バックオフィス業務からフロント業務まで多岐にわたります。具体的には以下のような業務が挙げられます。

BPOの対象となる業務例

* 経理
* 総務
* 人事・労務
* 法務
* 一般事務（データ入力・マニュアル作成など）
* 人材採用
* 受付業務
* コールセンター
* OA・IT機器やWeb関連の運用保守
* 営業代行　など

## BPOの契約形態

BPOを導入する場合、委託先の企業とは業務委託契約を締結することになります。
  
  
ただし、**民法上は業務委託契約という概念が定められていないため、実際には「請負契約」または「準委任契約」のいずれかを選択するのが一般的です。**
  
  
請負契約と準委任契約は、契約の目的がそれぞれ異なります。**請負契約は、業務の結果としての成果物を目的としているのに対して、準委任契約は業務の提供を目的としており成果については求められません。**
  
  
たとえば、ソフトウェア開発や給与計算など最終的に成果物が発生する業務を発注する場合は、請負契約が用いられます。一方、受付業務やコールセンターなど成果を求められず業務そのものの提供を目的としている場合には、準委任契約を用いることになります。

|  |  |  |
| --- | --- | --- |
|  | **請負契約** | **準委任契約** |
| **契約の目的** | 業務の結果（成果物）が目的 | 業務の提供自体が目的 |
| **該当業務** | ソフトウェア開発や給与計算など、成果物が明確な業務 | 受付業務やコールセンター業務など、成果物があいまいな、または求められない業務 |

  

業務委託契約の締結の流れや注意点について詳しくは、別記事「[業務委託契約とは？契約の種類や締結の流れを分かりやすく解説](https://www.freee.co.jp/kb/kb-deals/outsourcing-agreement/)」をご覧ください。

## BPO導入のメリット

BPOの導入は、単純な業務委託にとどまらず、経営を合理化するうえでのメリットがあります。ここでは、BPO導入の主なメリットについて3つご紹介します。

### 企業競争力の向上

**BPOの導入によって、自社の経営資源である従業員の人的リソースをコア業務に投入することが可能です。**従業員が手間のかかっていた事務作業などから解放され、本来注力すべき戦略策定や意思決定に集中できれば、企業競争力の向上や事業成長も期待できるでしょう。

### 業務プロセスの改善

**BPOは業務プロセス全体を外部の専門事業者に委託するため、業務効率化や標準化の観点からプロセスの再設計・改善・運用を行うことができます。**これによって、従来どおりに社内で業務を遂行するよりも、処理スピードや精度が上がる可能性があります。

### コスト削減

**BPOを活用すれば、社内の従業員が業務を習得するための教育コストが不要になります。**結果的に今までかかっていた人件費よりも安い費用で外部に業務委託できる可能性があり、経営コストの削減が見込めます。

## BPOの委託先の選定基準

BPOを上手く利用することで、企業にとってさまざまなメリットがあります。ここでは、BPOの委託先を選ぶ際に考慮すべき基準について説明します。

### セキュリティ体制

外部に業務委託するにあたっては、企業の保有する個人情報や機密性の高い社内情報を取り扱うこともあるでしょう。そのため**BPOの委託先を選定する際には、必ずセキュリティ体制を確認し、一定の水準を満たしているかどうかをチェックしましょう。**
  
  
たとえば個人情報を取り扱ううえで、委託先が「プライバシーマーク」の付与事業者であれば、一定の信頼がおけると判断できます。

### 業務の専門性

専門性の高い業務を委託する場合は、どのような実績があるか、自社が求める規模や水準の業務をこなせる体制があるかが重要なポイントになります。**BPOサービスを提供する企業によって得意領域や対応範囲に違いがあるため、直接問い合わせるなどして確認するとよいでしょう。**

### コスト

コストについては、単純に金額の安さだけで判断してはいけません。**BPOの導入目的に対して期待する効果が得られそうかを基準に、金額の妥当性を確認しましょう。**
  
  
サービスが安価であっても、自社の抱える課題を解決できそうであれば妥当なサービスであると判断できるでしょう。反対に金額が高くても、経営面で大きなメリットがあると判断できれば投資する価値はあります。委託する業務によっては長期的な運用が前提となるため、目先のコストにとらわれずに判断することが大切です。

### 対応範囲の広さ

BPOの場合、部分的な業務のアウトソーシングとは異なり、業務プロセスの全体を一括して委託できる点にメリットがあるといえます。**自社のニーズと照らし合わせながら、できるだけ対応範囲の広いサービスを導入するほうがよいでしょう。**

## まとめ

BPOの導入は、自社の従業員がコア業務に集中できることにより、企業経営にさまざまなメリットをもたらします。近年はBPOサービスを提供する事業者が増えているため、自社の求める業務領域や対応品質を満たす事業者をよく検討し、うまく活用しましょう。
