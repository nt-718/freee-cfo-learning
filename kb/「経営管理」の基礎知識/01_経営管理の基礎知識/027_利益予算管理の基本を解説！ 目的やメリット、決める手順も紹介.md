## 予算管理に欠かせない4つの予算

「予算」とは、目標の利益を達成するために必要な売上や、経費の目安をあらかじめ決めておいたものです。
  
  
これまでの経営実績や過去のデータをもとに、年間や月単位で予想される売上と、予想される経費の線を引いたものを「予算」として設定します。
  
  
予算は会社の経営の目標ともなり、経営や発展を左右する重要な役割を担っています。予算を策定し、管理することは会社の経営戦略に欠かせません。
  
  
予算管理とは、予算と実績値を比較し、分析や改善を行う管理業務です。予算管理には以下4つの予算が用いられます。

|  |  |
| --- | --- |
| **予算の種類** | **概要** |
| **売上予算** | 期中に達成すべき売上目標 |
| **原価予算** | 商品・サービスの提供のために必要となる原価の目標 |
| **経費予算** | 人件費・交通費などの企業活動に必要な費用の見積もり |
| **利益予算** | 期中に達成すべき利益目標 |

  

売上予算は過去の売上実績をもとに、数値をより高くすることが一般的です。原価予算は販売業では商品仕入、製造業では原材料費の見積もりを指します。
  
  
また経費予算は、人件費や交通費、事務所の家賃・光熱費など、事業活動にかかわる経費の見積もりです。
  
  
本記事では、上記の予算のうち期中に達成すべき利益目標である「利益予算」に関して詳しく解説します。

## 利益予算とは？

**「利益予算」とは、会社が目指す利益を生み出すために設定する予算です。**
  
  
企業活動を継続するには、単に売上を上げるだけではなく、「利益」を出さなければなりません。
  
  
利益予算は、売上から原価や経費を差し引いて算出します。利益を確保するには、売上を伸ばすことが重要です。しかし売上予算を達成していなくても、原価や経費の削減によって利益予算を達成できる場合もあります。

## 利益予算を決める目的

「利益予算」の目的は、利益を生み出すことです。予算は会社の数値目標であり、経営の指針でもあります。予算（目標）を決めておかないと経営の軸がぶれやすく、状況の悪化へと繋がりやすくなります。
  
  
また「売上予算」だけを決めても、利益を確保できるとは限りません。どれだけ売上実績が出ても、その分コストが膨大にかかっていれば売上高がマイナスになるケースもあるでしょう。
  
  
事業を継続させるには、売上高から原価や経費を差し引いた「利益」をどれだけ生み出せるかが重要です。利益が確保できれば、次の事業展開にもつながります。
  
  
利益予算は、会社の最大の目的である「利益」を確保するための重要な役割を担っています。

## 利益予算を決めるメリット

あらかじめ利益予算を決めておくと、経営戦略や方針が明確になります。販売数を増やして売上高を増やすべきか、今よりもさらにコストを抑えるべきか、などの見通しが立ちます。
  
  
見通しが立つことで、機械を新調する・優秀な人員を雇う・新しく投資するなど今後の方針も考えやすくなるでしょう。
  
  
また利益予算を決めておけば、目標達成できなかった場合も課題の分析や改善策の検討がしやすくなります。

## 利益予算を決める流れ

利益予算を決める流れは、以下の通りです。

利益予算を決める流れ

1. 大まかな利益目標と予算を決める
2. 各部門で分析し、予算を算出する
3. 各部門で調整後、予算を決定する

それぞれ詳しく解説します。

### 1.大まかな利益目標と予算を決める

取締役会で、会社全体の大まかな利益目標と予算額を決定します。
  
  
この時点では、大まかに達成したい利益を目標額として決定してもよいでしょう。「今年はこのくらいの利益は絶対に出したい」と意気込みを予算に考慮しても構いません。
  
  
少し大きめに予算を設定してもよいですが、どう頑張っても達成できない無謀な予算を計画しないようにしましょう。

### 2.各部門で分析し、予算を算出する

予算設定に関する方針が固まると、次は経理部・予算部・経営企画部など、経営に関わる部署で方針内容を検討しましょう。
  
  
損益計算書・過去の実績・データをもとに部門ごとの予算を算出します。部門が多ければ多いほど、利益額にばらつきが見えるでしょう。
  
  
当然、販売や提供するサービスの違いによって、利益が大きく出る部門もあれば比較的小さな部門もあります。時期によって販売数や売り上げが伸び悩む部門もあるため、ひとつひとつの部門傾向を考慮し、各部門に適した予算を配分しましょう。

### 3.各部門で調整後、予算を決定する

各部門に配分した予算を部門内で十分に検討してもらい、要望や意見があれば部門調整が必要です。
  
  
明らかに少なすぎる・多すぎる場合や、時期の関係で販売数や売上高が予算に届きそうにない場合は、要望や意見として予算部や経営企画部などに相談します。
  
  
相談があった場合は、内容を検討して再度予算を見直しましょう。特に要望・意見がなければ、当期の利益予算を決定します。

## 利益予算決定の具体的な手順

次に、利益予算を決める具体的な手順を解説します。

利益予算を決める具体的な手順

1. 前期分の変動損益計算書を作成する
2. 当期の変動損益計算書を作成する
3. 当期の利益予算を決める
4. 月ごとの利益予算を算出して実績管理する

それぞれ詳しく解説します。

### 1.前期分の変動損益計算書を作成する

前期の損益計算書をもとに固定費と変動費に分け、前期分の変動損益計算書を作成します。変動損益計算書とは、通常の損益計算書とは違い、原価と費用を固定費・変動費に分けて表示した資料です。
  
  
変動損益計算書を作成すれば、限界利益率（式：限界利益率 ＝ 限界利益 ÷ 売上高）や損益分岐点（利益と損失の境目）を把握できます。
  
  
限界利益（式：限界利益 ＝ 売上高 － 変動費）とは、商品やサービスを販売して得た売上高から販売に関する変動費を引いた利益です。
  
  
この限界利益がプラスとなれば、その分、固定費を回収できるため利益が出ます。つまり、限界利益率が高いほど損益分岐点が下がり、利益を出しやすい状態にあると言えるでしょう。
  
  
限界利益は利益予算を決める際にも役立ちます。適正な目標を立てるためにも、限界利益率をよく理解しておきましょう。

### 2.当期の変動損益計算書を作成する

前期の変動損益計算書から予算を検討する際には、経営上必要な利益を出すためにどうすべきか考えなければなりません。
  
  
必要利益を得るためにも、変動損益計算書を用いて予算シミュレーション（基本は売上・変動費・固定費の増減）をする必要があります。

### 3.当期の利益予算を決める

必要利益を生み出す予算計画が立てられた時点で、当期の変動損益計算書から利益予算を決定します。ここまでが利益予算の作成手順です。
  
  
ただし、この時点で終了すると単に予算を立てただけの状態であるため、次の段階で一歩進んだ予算計画を立てます。

### 4.月ごとの利益予算を算出して実績管理する

利益予算が決定したら達成のために毎月必要な利益を予測し、月ごとに実績管理をすることがポイントです。予算を決めればそれでいいわけではありません。
  
  
必ず利益を獲得するためにも、月ごとの予測利益を出して毎月の実績と比較しましょう。実績管理をすると月ごとのペースが読み取れ、課題にも早く気づけます。

## 会計ソフトを活用して利益予算を立てる方法も！

会計ソフトを活用し、利益予算を立てる方法もあります。
  
  
利益予算を立てることは、重要な経営管理のひとつです。しかし、利益予算の決定や実績管理には時間と労力がかかります。
  
  
会計ソフトを活用すれば、経営の意思決定の際に必要なデータを簡単に出力できるため、利益予算の作成にかかる時間や労力を削減できます。
  
  
また、予算と実績のズレを迅速に把握でき、意思決定のスピードを高められるでしょう。

## まとめ

利益予算とは、利益達成のためにあらかじめ決めておく売上や経費に関する予算のうち、利益を確保するために決める予算のことです。利益予算を決めておけば、「売上がいくらあればよいのか」「経費をいくらにするか」などが明確になります。
  
  
予算は設定だけで満足してはならず、予算達成もしくは予算を上回れるように努めていかなければなりません。利益予算を立てると、目標に届かなかった場合も迅速に課題の分析や改善ができます。
  
  
予算設定の流れを理解し、今後の発展のために役立てていきましょう。
