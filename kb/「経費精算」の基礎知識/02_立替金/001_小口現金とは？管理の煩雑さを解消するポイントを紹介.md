## 小口現金とは

**小口現金とは、交通費や消耗品代の精算など日常的に発生する少額の経費に対応するための現金を指します。**従業員が経費を立て替えてもすぐに精算できるメリットがあり、とくに営業職など経費の発生しやすい部署における負担の軽減を期待できます。
  
  
一方で、小口現金は精算用の小銭を用意する必要があるなど、経理担当者の負担が増えてしまうデメリットもあります。また、現金を先に手渡して仮払いした場合、少額であっても領収書などで使用額を確認し、紛失や横領などが発生しないよう注意しなければなりません。
  
  
小口現金の管理には、毎日の現金残高と帳簿残高の照合を複数人でチェックするといった慎重な取り扱いが求められます。管理ミスにより残高が合わない場合は、原因を特定するための調査も必要です。
  
  
なお、小口現金の導入は企業ごとの判断によるため、導入していないところもあります。いくらまで使用できるのかといった金額の上限に関するルールも、企業によってさまざまです。

## 小口現金管理の流れ

小口現金は適切な管理が必要です。具体的には、小口現金担当者が以下の流れで毎日管理します。

小口現金管理の流れ

* 始業後、金庫から小口現金を取り出し前日の残高と合っているか確認する
* 領収書を持つ従業員に対して、経費精算対応を行う
* 小口現金から支払った金額と内容を小口現金出納帳へ記載する
* 終業時、小口現金出納帳と残高が一致しているか確認する
* 小口現金を金庫へしまう

このように、**毎日の残高の確認と、経費精算を行うごとに小口現金出納帳への記載を行う必要があります。**小口現金出納帳についての詳細は、以下の記事をあわせてご確認ください。
  
  
【関連記事】
  
[小口現金出納帳とは？現金出納帳との違いや書き方をわかりやすく解説](https://www.freee.co.jp/kb/kb-accounting/difference-of-cash-journal-and-cashbook/)
  
  
また、**スムーズな経費精算を行うには定期的な現金補充も忘れてはいけません。**小口現金の補充方法は、主に「随時補給制度」と「定額資金前渡制度」の2つがあります。
  
  
随時補給制度とは、支出が発生した際に現金を補充する制度です。こまめな補充ができる一方、手間や管理の負担が増えてしまうため、この方法を採用している企業は少ないといえます。
  
  
定額資金前渡制度とは、あらかじめ小口現金として置く金額を決めておき、一定期間ごとに減った分を補充する制度です。たとえば小口現金を10万円とした場合、期間内に3万円と1万円を精算すると、後日4万円をまとめて補充することになります。随時補給制度と異なり、支出のたびに補充する必要がないため担当者の負担軽減が期待できます。

## 小口現金の仕訳方法

小口現金は、勘定科目「小口現金」を使用して仕訳を行います。
  
  
なお、**小口現金は貸借対照表の資産にあたるため、借方項目となります。**増加する場合は、借方・減少する場合は貸方に記入すると覚えておきましょう。

### 小口現金から費用を払い出したとき

**消耗品や交通費など小口現金から払い出した費用は、何にいくら使ったのかを把握できるよう仕訳する必要があります。**
  
  
たとえば下表では消耗品費4,000円、旅費交通費1万円、通信費5,000円、光熱費2万円、雑費3,000円を支払っているとわかります。この場合、払い出した費用と項目をそれぞれ借方に記載します。また、貸方には小口現金4万2,000円を記載します。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 消耗品費 | 4,000 | 小口現金 | 42,000 |
| 旅費交通費 | 10,000 |  |  |
| 通信費 | 5,000 |  |  |
| 光熱費 | 20,000 |  |  |
| 雑費 | 3,000 |  |  |

### 預金から小口現金を引き出したとき

預金から小口現金に充てるためのお金を引き出した場合、上表を参考にすると、借方に小口現金15万円と記載し、貸方に預金15万円と記載して仕訳できます。

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 小口現金 | 150,000 | 預金 | 150,000 |

  

また、**小口現金を預金から引き出し補充したときは、借方に小口現金と金額を記載します。**上表では、預金から補充した例を表しています。この場合は、貸方に預金5万円と記載し、借方に小口現金5万円と記載しましょう。

  

| 借方 | | 貸方 | |
| --- | --- | --- | --- |
| 小口現金 | 50,000 | 預金 | 50,000 |

  

なお、定額資金前渡制度を採用している場合は、月や週など決められた期間ごとに小口現金を補充します。

## 小口現金の管理の煩雑さを改善するポイント

小口現金の管理は、残高や出納帳の確認を毎日行うなど、担当者への業務負担が大きくなりがちです。経費精算する従業員が多ければ、精算したり記帳したりする頻度も増えるため、手間やミスが増える可能性も考えられます。
  
  
小口現金の管理の煩雑さを改善するには、以下3つのポイントが挙げられます。

* 立替払いは月1回決められた日に精算する
* 小口現金専用のクレジットカードを導入する
* 業務で良く使う備品はまとめて購入する
* 経費精算システムを活用する

**業務効率の向上や担当者の負担軽減の方法を考えることで、小口現金を管理する負担も減らせます。**また、給与合算などによって小口現金自体を廃止することも対策として有効です。

### 立替払いは月1回決められた日に精算する

旅費や交通費、雑費など従業員が立替えた費用については、月に1回決められた日に精算してください。
  
  
小口現金の管理が煩雑になる理由のひとつは、精算回数の多さです。担当者は精算のたびにほかの業務の手を止めて対応しなければならず、残高の確認や記帳にも時間を取られてしまいます。
  
  
**あらかじめ月に1回と決めておけば、その都度お金のやりとりをする手間を削減できるほか、精算したい従業員も時間を節約できます。**また、まとめて精算する際に小口現金ではなく、給与と一緒に振り込む方法も有効です。領収書を添えて申請してもらうような仕組みを構築しておけば、効率化も図れます。

### 小口現金専用のクレジットカードを導入する

煩雑な小口現金の管理を削減するためには、社内で小口現金専用のクレジットカードを導入するのも選択肢のひとつです。
  
  
**小口現金専用のクレジットカードを従業員に配布すれば、小口現金の受け渡しや立替払いといった従業員の負担も削減できます。**

### 業務で良く使う備品はまとめて購入する

業務で使用する頻度が高い備品は、まとめて購入するのがおすすめです。とくに切手や文房具は1個あたりの費用が安価であるほか、購入数も多くなりやすいため、その都度小口現金で精算していては負担が生じてしまいます。また細かい金額を記帳する回数が増えると、手間だけでなくミスも増える恐れがあります。
  
  
**企業向けのECサイトやクレジットカードを活用すれば、そもそも現金を使用する頻度を減らせるため、残高計算や記帳の手間も削減することも可能です。**

### 経費精算システムを活用する

小口現金の管理の手間を軽減するには、経費精算システムの導入も一案です。小口現金出納帳は手書きでもExcelデータでも問題ありませんが、記載ミスの発生や業務効率の低下につながる恐れがあります。
  
  
**経費精算システムはそのようなリスクを減らせるほか、お金の出入りを可視化しやすくなるのもメリットです。**たとえば単なる数字の入力ミスだけでなく、領収書を重複して登録してしまうといったミスも防ぎやすくなります。
  
  
とくにクラウド型のシステムを採用すれば入力場所を選ばないため、外出先やテレワーク時の経理作業の効率化も期待できるでしょう。
  
  
先に紹介した小口現金専用のクレジットカードを経費精算システムに連携すれば、記帳なしでも経理担当者の負担を軽減できます。

## まとめ

小口現金は、従業員が経費を立て替えた際にスムーズに精算できるところがメリットですが、毎日の残高確認やこまめな記帳など、現金を管理する担当者には一定の負担が生じます。
  
  
小口現金の管理が煩雑だと感じる場合は、月1回の精算日を決めたり・よく使う備品は会社でまとめて購入したりなどの方法で改善を試みましょう。
  
  
小口現金の管理を効率化するには、経費精算システムの導入もおすすめです。管理の手間や記帳ミスを減らすことができ、担当者の負担軽減につながります。
