## 請求書の書き方にルールはある？

請求書は大きく「適格請求書（インボイス）」と「適格請求書以外の請求書」に分けられます。いずれも**特定のフォーマットはなく、請求書の形式に関しては手書き（紙）、Excel、専用の作成ソフトなど、どのような形式でも問題ありません。**
  
  
**適格請求書とは、2023年10月以降のインボイス制度開始に伴い発行される、一定の記載要件を満たした請求書などのことです。**記載要件が満たされていれば、領収書なども適格請求書として認められます。インボイス制度では、基本的に買い手（発注者）が仕入税額控除を受けるためには、売り手（受注者）が発行した適格請求書と法定事項が記載された帳簿の保存が必須です。
  
  
対して、**適格請求書以外の請求書とは、適格請求書の記載要件を満たしていない請求書が該当します。**請求書としては機能するものの、適格請求書以外の請求書で行われた取引については仕入税額控除の対象になりません。

### 請求書の発行や送付に関する決まりは？

請求書の発行は、法律によって義務化されているわけではありません。しかし、自社の商品やサービスの対価を請求するうえでなくてはならない書類です。**取引先とのトラブルを避けるためにも、取引が発生した際は必ず請求書を発行・受領するようにしましょう。**
  
  
また、**請求書の送付方法についても決まりはなく、郵送・デジタルデータのどちらでも構いません。**どちらの方法も広く採用されており、取引の状況や相手方の要望に応じて選択することができます。詳しくは記事内の「[請求書の送り方に関するルール](#content4)」をご覧ください。

## 請求書に記載する事項とは

一般的に「請求書」と呼ばれる書類に記載する事項は、法律では定められていません。しかし、国税庁のホームページでは、次の5つが請求書の記載項目として挙げられています。

請求書の記載項目

1. 書類作成者の氏名または名称
2. 取引年月日
3. 取引内容
4. 税込みの取引金額
5. 書類の交付を受ける事業者の氏名または名称

  

[出典：国税庁「No.6625 適格請求書等の記載事項」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shohi/6625.htm)

上記を満たしていれば、請求書として機能する書類と見なすことができるでしょう。それぞれの項目の書き方について、以下で詳しく見ていきましょう。
  
  
なお、請求書の書き方について詳しく知りたい方は、別記事「[請求書の書き方を徹底解説｜作成方法や注意点についてもまとめました](https://www.freee.co.jp/kb/kb-invoice/writing-guide/)」をご覧ください。

### 1.書類作成者の氏名または名称

書類作成者の氏名または名称は、請求書の題目の下や日付の下付近に記載するケースが一般的です。具体的には、以下の事項を記載します。

  

| 項目 | 概要 |
| --- | --- |
| 社名 | ・「株式会社〇〇」や「〇〇合同会社」など、請求書の発行者である自社の名称。  ・「（株）〇〇」ではなく「株式会社〇〇」と正しく記載する。 |
| 部署名と担当者名 | ・「営業部　山田太郎」など、担当者の部署名と氏名を記載。 |
| 社判や担当者印 | 社判や担当者印を押さなければならないという法的な決まりはないものの、押印するケースが多い。相手方の希望に応じて対応するのが望ましい。 |
| 連絡先 | 電話番号やメールアドレスなどの連絡先を記載。電話番号やメールアドレスの誤記は、コミュニケーションの遅延やトラブルにつながりかねないため注意が必要。 |

### 2.取引年月日

取引年月日の記載により、取引が実際に行われた日を明確にします。「2024年1月24日」などのように、具体的な年月日を正確に記載します。
  
  
記載の際には、以下の点に注意しましょう。

* **西暦と和暦の混在**：両者を混在させないことが一般的。社内の請求書発行のルールとして、どちらを使用するかあらかじめ定めておくことが望ましい。
* **発行日と取引日の混同**：請求書の発行日と取引日は同じではない。取引年月日とは「実際に取引が行われた日付」である。
* **掛売取引の場合の日付**：掛売取引では実際の取引日から請求書発行までに期間が空くため、取引日を間違えないようにする。

### 3.取引内容

取引内容には、対価の請求が必要な商品・サービスの品名や単価などを記載します。
  
  
具体的には、以下の事項を記載します。

  

| 項目 | 概要 |
| --- | --- |
| 品名、単価、数量、金額 | ・「商品A、単価1,000円、数量2、金額2,000円」といったように、提供した商品・サービスの品名、単価、数量、金額を分けて記載  ・品名が正しくないと、取引内容について誤解が生じるおそれがある。省略したり、一部にのみ伝わるような通称で書いたりすることは避ける |
| 小計、消費税、合計額 | ・取引額の小計を算出し、そこに消費税を加算し、小計に消費税を加えた合計額を明記する  ・計算ミスがあると支払い後にトラブルが起きたりするリスクが高まる |

### 4.税込みの取引金額

請求書の取引金額には、消費税を加えた記載もしなければなりません。具体的には、以下の項目を記載します。

  

| 項目 | 概要 |
| --- | --- |
| 内税・外税 | ・金額が内税（消費税込）か外税（消費税別）かを明確に記載する |
| 請求金額の詳細な分類 | ・前月請求金額、入金額、繰越金額、当月請求金額、合計請求金額など、分けて記載する |

  

取引金額の記載時には、相手の読みやすさを考慮し「¥1,000,000」のように金額は3桁ごとにカンマを入れて記載するようにしましょう。また、単価は税抜きで記載し、小計に消費税を加算して記載するのが一般的です。

### 5.書類の交付を受ける事業者の氏名または名称

書類の交付を受ける事業者の氏名・名称は、「誰に請求しているのか」という点を明らかにするために記載します。いずれも、誤記は失礼にあたるため十分に注意しましょう。

  

| 項目 | 概要 |
| --- | --- |
| 事業者の氏名や名称 | ・相手方（買い手側）の会社名や事業主の名前などを記載する  ・氏名などは、一般的に請求書用紙の左上に記載する  ・「（株）〇〇」などの短縮も避けて「株式会社〇〇」と正しく記載する |
| 住所、部署名、担当者名 | ・相手方（買い手側）の状況に応じて書く項目は異なるが、取引先の住所、部署名、担当者名も記載することが望ましい |

### 【その他】記載が望ましい事項

国税庁が挙げている上記5項目以外にも、状況に応じて請求書に記載しておくとよい項目として、以下が挙げられます。

  

| 項目 | 内容 |
| --- | --- |
| 請求書番号 | ・請求書に記載する任意の管理番号  ・見積書や納品書との照合を容易にし、特定の請求書を探しやすくするなど管理しやすくなる |
| 振込先 | ・代金の振込先となる金融機関の口座情報  ・銀行名、支店名、預金種別、口座番号、口座名義（カタカナ記載）、銀行コード、支店コードを含めると親切  ・振込手数料の負担者を明記するとよりよい |
| 支払期限 | ・請求金額の支払期日  ・通常は契約時に取り決めた内容に沿って記載  ・明記しておくと「いつまでに支払いがなかった場合催促を行うか（されるか）」という相互認識が一致するため、トラブルが起きても対処しやすい |

  

これら追加項目を請求書に含めることで取引を管理しやすくなり、相手方（買い手側）とのコミュニケーションもスムーズになります。なかでも、**「振込先」の情報は金銭のやり取りに直接関わることから、できれば請求書に記載したほうがよいでしょう。**

### 請求書への押印は必要？

請求書には印鑑を押すのが社会通念上一般的とされていますが、法的な取り決めは存在しないため、印鑑は押しても押さなくても構いません。
  
  
しかし、請求書に印鑑を押すことで「会社として発行している請求書」であることを証明でき、偽造されてしまったときのリスク回避にもなります。そのため、どうしても対応できない理由がなければ、押印はしておくほうがベターといえるでしょう。

## 適格請求書に記載すべき事項

2023年10月から始まったインボイス制度は、消費税の適正な徴収と透明性の向上を目的としています。2023年10月以降、買い手側が消費税の仕入税額控除を受けるためには、売り手側が発行した適格請求書の保存が必要です。
  
  
ただし、適格請求書は、適格請求書発行事業者の登録を受けた課税事業者のみが発行できます。発行にあたっては、以下の項目の記載が必要です。

適格請求書の記載要件

* 発行事業者の氏名または名称および登録番号
* 取引年月日
* 取引内容（軽減税率の対象品目である旨）
* 税率ごとに区分して合計した対価の額（税抜または税込）および適用税率
* 税率ごとに区分した消費税額など
* 書類の交付を受ける事業者の氏名または名称

  

[出典：国税庁「4 適格請求書の記載事項」](https://www.nta.go.jp/taxes/shiraberu/zeimokubetsu/shohi/keigenzeiritsu/pdf/qa/01-09.pdf)

**適格請求書以外の請求書と大きく異なるのは、「登録番号」の記載が必要という点です。**登録番号とは適格請求書発行事業者として登録した事業者に発行される番号のことで、この番号が明記されていないと適格請求書として認められません。
  
  
適格請求書について詳しく知りたい方は、別記事「[適格請求書とは？書き方や保存期間、簡単に作成する方法について解説](https://www.freee.co.jp/kb/kb-invoice/invoice_document/)」をご覧ください。

## 請求書の送り方に関するルール

**請求書は、主に郵送、メール、電子契約システムで送られることが一般的ですが、送り方に法的なルールはありません。**ここでは、郵送、メール、電子契約システムを用いた送り方について、それぞれ注意すべきポイントを解説します。

### 郵送する場合

紙で発行した請求書を送る際は、郵送を用います。**請求書は「信書」に分類されるため、メール便や宅配便では送付できません。郵送する場合は書留や速達を利用して、安全・迅速に到着するよう手配しましょう。**

#### 請求書は「原本」と「控え」、どちらを渡すべき？

**請求書は請求先（買い手側）に「原本」を渡し、請求する側（売り手側）は「控え」を保持します。**
  
  
請求書の控えとは、請求書の原本をコピー（取引の証拠）したものなどが該当し、法的な作成義務はありません。ただし、**請求書の控えを作成した場合は「取引証憑書類」に該当するため、7年間（欠損金額がある年度は10年間）にわたり保存する義務が生じます。**保存期間の起算日は、課税期間の末日の翌日から2ヶ月が経過した日です。
  
  

[出典：国税庁「5 適格請求書等の写しの保存 」](https://www.nta.go.jp/taxes/shiraberu/zeimokubetsu/shohi/keigenzeiritsu/pdf/qa/01-10.pdf)

#### 郵送時には送付状を添えるのがビジネスマナー

**請求書の原本を郵送する際には、「送付状」を添えるのが一般的なビジネスマナーとされています。**送付状には以下の情報を含めるようにしましょう。

一般的な送付状の記載事項

* 日付：実際に請求書を送付した日付
* 宛先と差出人の情報：受取人（宛先）と差出人の名前・住所
* 挨拶文と送付の旨：簡潔な挨拶文と、請求書を送付する目的を伝える本文
* 同封書類の内容：送付状と一緒に同封する書類のリスト

送付状を添えることは受取人に対して丁寧な印象を与えるだけでなく、送付書類に過不足はないかなどの確認に役立ちます。

### メールを使う場合

**メールを使用する場合、パソコンで作成した請求書データをPDF形式で保存し、メールに添付して送ります。**この方法は郵送と比べて印刷代や切手代が不要、発送の手間もかからないため迅速かつ効率的です。
  
  
企業によっては紙の請求書を希望する場合や、メールでデータとして請求書を送付したのちに「原本も郵送してほしい」と希望する場合もあります。**紙の請求書も郵送する必要があるか、取引先に確認しておきましょう。**

### 電子契約システムを使う場合

**電子契約システムを利用する場合は、インターネット上でPDFファイルに押印や署名を施し、請求書のやり取りを行います。**電子契約システムを利用すれば、システム上でPDF化された請求書をそのままシステム上で送付でき、メールなどの外部ツールを用いないためセキュリティ面も安心です。
  
  
郵送する場合に比べて、印刷代や切手代、郵送代などを減らせるため、コスト削減やペーパーレスの推進などに有効です。また、発送にかかる手間がなくなるだけでなく、請求管理の一元化も実現できます。

## 請求書の保存に関するルール

請求書の保存方法には「紙」と「電子データ」の2種類があり、紙で受け取った請求書は紙のままでの保存が認められています。
  
  
**電子帳簿保存法改正により、2024年1月1日以降は電子データで受け取った請求書は必ず電子データのまま保存しなければならなくなり、印刷して紙で保存することはできません。**ただし、紙の請求書を受け取った場合は紙のまま、または保存要件を満たしていればスキャンしてデータ保存することも可能です。
  
  
電子帳簿保存法について詳しく知りたい方は、別記事「[電子帳簿保存法とは？対象書類や保存要件・改正内容についてわかりやすく解説](https://www.freee.co.jp/kb/kb-deals/deals-records-in-electronic-form/)」をご覧ください。
  
  
なお、請求書の保存期間に関するルールは、法人と個人事業主で異なります。以下ではそれぞれの保存期間を解説します。

### 法人の請求書の保存期間

**法人の場合、請求書は最低7年間保存する義務があり、赤字が出た年度分（欠損金を繰り越す場合）は10年間の保存が求められます。**
  
  
保存期間は、確定申告の期日の翌日を起点とし、「7年後もしくは10年後の事業年度終了日翌日から2ヶ月後まで」です。
  
  
請求書の保管に関して詳しく知りたい方は、別記事「[請求書の保管期間とは？保管方法や注意すべきポイントを解説](https://www.freee.co.jp/kb/kb-invoice/keeping-invoice/)」をご覧ください。

### 個人事業主の請求書の保存期間

**個人事業主の場合は、適格請求書以外の請求書の保管期間は最低5年間です。**しかし、**消費税の課税事業者となっている場合や、インボイス制度対応の適格請求書を発行している場合、適格請求書については7年間の保存が必要です。**
  
  
保存期間の起点は、法人と同じく確定申告した期日の翌日です。そこから請求書の種類に応じて「5年後もしくは7年後の確定申告期限日」まで保存しなければなりません。迷いそうな場合は、請求書を7年間保存しておくと安心でしょう。
  
  
適格請求書の保存期間について詳しくは、別記事「[適格請求書とは？書き方や保存期間、簡単に作成する方法について解説](https://www.freee.co.jp/kb/kb-invoice/invoice_document/#content6)」をご覧ください。

## まとめ

請求書の作成、送付、保存に関するルールやマナーは、取引先と確実な金銭のやり取りを行うため、そして「安心して取引できる信頼関係」を構築・維持するために重視すべきものです。
  
  
ルールに沿って請求書を発行・管理することで、発行する側も受け取る側も不安やミスの少ない取引を実現できます。

## よくある質問

### そもそも請求書を作成するのにルールはある？

請求書の作成や発行に、法的に定められたルールはありません。ただし、請求書は自社の商品・サービスの対価を請求するうえで極めて重要な書類です。取引先との不要なトラブルを避ける意味でも、取引が発生した際は必ず請求書を発行・受領するようにしましょう。
  
  
なお、適格請求書の記載事項や請求書の保存に関しては一定のルールがあります。くわしくは記事内の「[請求書の書き方にルールはある？](#content1)」をご覧ください。

### 請求書を送るときのルールは？

請求書の送付には、郵送、メール、電子契約システムなど複数の方法があり、それぞれにルールやビジネスマナーがあります。一例として、請求書は信書にあたるため郵送する場合はメール便などが使えません。
  
  
くわしくは記事内の「[請求書の送り方に関するルール](#content4)」をご覧ください。

### 適格請求書（インボイス）に記載すべき事項は？

適格請求書には、インボイス制度に基づいて通常の請求書とは異なる特定の記載要件があります。適格請求書の記載事項には、「適格請求書発行事業者の登録番号」などが挙げられます。
  
  
くわしくは記事内の「[適格請求書に記載すべき事項](#content3)」をご覧ください。
