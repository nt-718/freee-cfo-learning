# インボイス残業はどのくらい発生する？制度開始に伴う業務内容の変化を解説

出典: https://www.freee.co.jp/kb/kb-invoice/invoice-overtime/

---

インボイス制度（適格請求書等保存方式）は、複数の税率に沿った消費税の仕入税額控除を受けられる制度で、2023年10月1日より開始されました。事業者が消費税の仕入税額控除を受けるには、適格請求書発行事業者に登録済みの事業者が発行した「適格請求書」の受領・保管が必要になります。
  
  
インボイス制度の導入により適格請求書の確認や保管に関する業務が増えることで、経理担当者の業務負担が増加し、いわゆる「インボイス残業」が発生する可能性があります。
  
  
本記事では、インボイス制度により経理担当者の負担や残業にどの程度影響が出るのかについて解説します。

## インボイス制度で残業は増える？

インボイス制度が導入されたことで、請求書や領収書などの処理を行う経理担当者は、インボイス制度への対応業務が追加されます。
  
  
特に一般課税（本則課税）で消費税を納税している企業にとっては、取引先の売り手事業者が適格請求書発行事業者であるかどうかによって請求書や領収書の処理手順が異なります。**従来の請求書や領収書への対応に加え、適格請求書（インボイス）への対応が発生することで、業務量が大きく増える見込みです。**
  
  
freeeの調査では、インボイス制度開始後、各種領収書の確認や仕分け、新しいルールに沿った帳簿の記載などをすべて手入力で行うと、業務時間はこれまでの4倍になると想定されます（下図参照）。

  
  

インボイス制度について詳しく知りたい方は、別記事「[2023年10月から始まったインボイス制度とは？図解でわかりやすく解説！](https://www.freee.co.jp/kb/kb-invoice/about_invoice/)」をご覧ください。

## インボイス制度により増える経理業務とは

インボイス制度開始後、経理担当者の負担となる業務のひとつに請求書を受け取ったあとの経費精算業務が挙げられます。
  
  
前述したように、インボイス対応を含めた業務時間は手作業だと従来の4倍に増えると予測できます。従来と制度開始後では、経理処理の行程に以下のような違いがあります。

  

| 手順 | インボイス制度開始後の業務 | 従来の業務 |
| --- | --- | --- |
| 1 | 書類の仕分け・整理  （PDFなら印刷も） | 書類の仕分け・整理  （PDFなら印刷も） |
| 2 | **国税庁HPで登録番号を確認** | 仕訳・記帳 |
| 3 | **記載項目の確認** | 仕訳の確認 |
| 4 | **税額・税区分の確認** | 支払確認 |
| 5 | **区分ごとに記帳** | 仕訳修正チェック |
| 6 | 仕訳の確認 | 書類の保管 |
| 7 | 支払確認 |  |
| 8 | **（再）国税庁HPで番号確認** |  |
| 9 | **（再）記載項目の確認** |  |
| 10 | 仕訳修正チェック |  |
| 11 | 書類の保管 |  |

  

ここでは、インボイス制度によって増える業務の具体例を解説します。

### 請求書などの書類を整理する

従来より対応していた書類の整理は、制度が開始されてから複雑になります。
  
  
**これまでどおり従業員や取引先からの請求書・領収書のチェックに加えて、適格請求書かそうでない請求書か仕分けしなくてはなりません。**これは、請求書や領収書の種類、取引の種類によって仕入税額控除の可否やその税率が異なるためです。
  
  
取引先から領収書などを含めた書類を受け取った場合、まず、以下の種類ごとに仕分けする必要があります。

書類の分類

* 適格請求書または適格簡易請求書
* 適格請求書または適格簡易請求書に該当しない請求書や領収書など
* 帳簿への記載のみで仕入税額控除ができる取引
* 領収書やレシートがなく、仕入税額控除の対象ではない取引

[出典：国税庁「適格請求書等保存方式の概要」](https://www.nta.go.jp/taxes/shiraberu/zeimokubetsu/shohi/keigenzeiritsu/pdf/0020006-027.pdf)
  
  
**登録番号にミスがあったり、実際には登録されておらず虚偽の番号だったりした場合は、適格請求書だと認められず仕入税額控除も受けられません。**そのため「T＋13桁」の登録番号が記載されているか、その番号に間違いがないかを必ず確認しましょう。なお、法人であれば「T」以降は法人番号です。
  
  
登録番号が実在するかどうかは、国税庁の[適格請求書発行事業者公表サイト](https://www.invoice-kohyo.nta.go.jp/index.html)もしくは[法人番号公表サイト](https://www.houjin-bangou.nta.go.jp/)で確認してください。
  
  
また、税区分についても品目ごとに税率が適切か、軽減税率の対象品目はあるかをチェックし、税率ごとの合計税額に誤りがないかも確認します。
  
  
登録番号についてより詳しく知りたい方は、別記事「[インボイス制度における登録番号とは？登録番号の取得方法と確認方法について解説](https://www.freee.co.jp/kb/kb-invoice/invoice_registration_number/)」をご覧ください。

### 区分ごとに帳簿に記入する

適格請求書とそうでない請求書の内容を確認したら、各書類の税率に沿って、記帳します。税率は以下のように区分されるため、該当の書類がどのケースに当てはまるのかをチェックしながら進めましょう。

  

| 請求書の種類 | 区分 |
| --- | --- |
| 適格請求書、適格簡易請求書 | ・消費税10％  ・消費税8％  ・消費税10％および8％の混合 |
| 適格請求書以外の請求書等 | 経過措置の割合  （支払った消費税の80～50％） |
| 帳簿への記帳のみで仕入税額控除が可能な取引 | ・消費税10％  ・消費税8％ |

  

適格請求書以外の請求書は原則、仕入税額控除ができません。ただし、**制度開始後6年間（2029年9月30日まで）は、経過措置として80〜50%の仕入税額控除が可能です。**
  
  
インボイス制度の経過措置について詳しく知りたい方は、別記事「[インボイス制度の経過措置とは？ 利用するための要件についてわかりやすく解説](https://www.freee.co.jp/kb/kb-invoice/invoice_transitional_measures/)」をご覧ください。
  
  
なお、**簡易課税制度を選択している事業者は、消費税の納税額を計算する際に仕入税額控除ではなく業種に応じたみなし仕入率をもとに計算します。**これは、売上に対する消費税額のみをもとに計算するため、売り手との取引に際してこれらの書き分けは不要です。
  
  
簡易課税制度やみなし仕入率について詳しく知りたい方は、別記事「[インボイス制度で簡易課税制度はどうなる？新たに課税事業者になる場合の軽減措置についても解説](https://www.freee.co.jp/kb/kb-invoice/invoice_simplified_tax_system/)」をご覧ください。
  
  

[出典：国税庁「適格請求書等保存方式の概要」](https://www.nta.go.jp/taxes/shiraberu/zeimokubetsu/shohi/keigenzeiritsu/pdf/0020006-027.pdf)

### ダブルチェックをする

インボイスに関わる業務では確認すべき事項が多く、不備があれば仕入税額控除が受けられなくなってしまうため、より丁寧な確認作業が求められます。仕訳が完了した適格請求書に誤りがないか、帳簿への記載内容が正しいかを改めて見直しましょう。
  
  
確認すべき主な項目として、以下が挙げられます。

請求書処理にあたって確認すべき主な項目

1. 自社の消費税課税区分に応じた仕訳ができているか
2. 取引先がインボイス登録しているか
3. 受け取った領収書や請求書にT＋13桁のインボイス登録番号があるか
4. T＋13桁のインボイス登録番号が国税庁に登録されている番号か
5. 受け取った証憑を正しく記帳できているか
6. クレジットカード利用の経費を正しく記帳できているか
7. クレジットカード明細と領収書を二重で計上していないか
8. 自社が少額特例に対応しているか
9. 少額特例に該当する取引かどうか

あわせて、請求書や領収書に記載どおりの金額が取引先や従業員へ適切に支払われているかもチェックしてください。万が一ミスや記入漏れがあった際は、仕訳や記帳・支払の修正をすぐに実施します。

### 各請求書を正しく保管する

仕訳の確認が完了したら、これまで整理した請求書や領収書を法律に沿って保存します。
  
  
紙で発行された適格請求書であれば、原本のまま、もしくはスキャンしてPDF化した状態で電子保存してもかまいません。ただし、**電子的に受け取った請求書は、電子データのまま保存するよう義務付けられています。**電子的な請求書とは、メールやインターネットを介したデータや、記録用の媒体に入れて受け取ったデータなどが当てはまります。
  
  
保存期間は、法人であれば紙も電子データも、適格請求書を交付または提供した日の属する課税期間の末日の翌日から2ヶ月を経過した日から7年間です。

  
  
一定の要件を満たした少額取引や適格請求書を受け取るのが難しい取引に関しては、帳簿のみの保存が認められます。
  
  
インボイス制度開始や電子帳簿保存法の改正により保存方法も複雑化したため、取引や受領した書類の様式に合わせて適切に管理しましょう。
  
  

[出典：国税庁「適格請求書等保存方式の概要」](https://www.nta.go.jp/taxes/shiraberu/zeimokubetsu/shohi/keigenzeiritsu/pdf/0020006-027.pdf)
  
[出典：国税庁「５ 適格請求書等の写しの保存」](https://www.nta.go.jp/taxes/shiraberu/zeimokubetsu/shohi/keigenzeiritsu/pdf/qa/01-10.pdf)

## 月末月初のインボイス残業の増加に注意

前述のとおり、インボイス制度の導入によって経理担当者の日常業務は増加すると考えられます。
  
  
特に、月末や月初は多くの企業で締め作業が重なるため、請求書や領収書の受領・発行が集中します。さらにインボイスへの対応が発生すると、月末月初は経理担当者のインボイス残業が生じやすいといえるでしょう。
  
  
そこで、インボイス制度対応による残業を防ぐための対策を立てることも大切です。具体的には、以下のような社内ルールの見直しや運用改善が効果的です。

インボイス残業への対策例

* インボイス制度開始による各種請求書、領収書の要件、社内ルールについて社内に周知する
* 請求書の種類ごとに経費精算書の記載方法などのルールを設定する
* 月末月初に不要な事務作業が集中しないよう、書類の提出期限や期間についてのルールを設定する
* 請求書の受領・仕訳・記帳・支払・保存の流れを一本化できるよう会計ソフトを導入する

なかでも会計ソフトの利用は、経理業務全体の効率化ができ、経理業務の負担を大きく削減できます。

## まとめ

インボイス制度が開始することにより、経理担当者の業務量は増加します。インボイスの対応で請求書関連の業務が複雑化し、細かい確認作業が従来よりも多く発生するでしょう。他の日常業務と並行してインボイス対応を進めると、月末や月初に対応が集中し、インボイス残業が発生する可能性があります。
  
  
経理担当者の業務負担軽減のためにも、これまでの請求書受領ルールを見直したり、会計ソフトを導入したりするなどして、この機会に業務効率化を図りましょう。

## よくある質問

### インボイス制度で残業は増えるの？

インボイス制度の導入で、関連業務にかかる時間は、すべて手作業の場合、従来の4倍に増えると予測できます。業務時間の増加に伴い、インボイス残業が発生すると考えられます。
  
  
詳しくは「[インボイス制度で残業は増える？](#content1)」をご覧ください。

### インボイス制度の開始に伴ってどんな業務が増えるの？

インボイス制度が開始されると、以下の業務が追加されます。

  

* ・適格請求書とそれ以外の書類の仕分け
* ・適格請求書に記載の事項が適切かどうかの確認
* ・取引先がインボイス登録事業者かどうかの確認
* ・複数の消費税率や税額控除の割合ごとの記帳
* ・受領した適格請求書をルールに沿って保存

  

制度開始前より業務工程が増えるため、経理担当者が抱える業務の種類も複雑化するでしょう。
  
  
詳しくは「[インボイス制度により増える経理業務とは](#content2)」をご覧ください。
