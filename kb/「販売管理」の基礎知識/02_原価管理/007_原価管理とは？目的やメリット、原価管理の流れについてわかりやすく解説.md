## 原価管理とは

**原価管理とは、特定の製品製造やプロジェクトが完了するまでにかかるコストを管理することで赤字化を防ぎ、確実に利益を出すための取り組みを指します。**
  
  
おおまかな流れは次のとおりです。

1. 目標となる原価を見積もり（標準原価の設定）を行う
2. 実際にかかっている原価を把握する
3. 標準原価と実際の原価の差異を確認・分析する
4. 3を踏まえて原価率の改善を行う

原価管理は単に「かかった費用を計算する」だけでなく、そこから課題を抽出し、未来のコストを調整（コントロール）して利益を確保するための活動です。
  
  
より詳細なステップは、「[原価管理のおおまかな流れ](#content5)」で解説しています。

### 直接費・間接費とは？

直接費と間接費は、いずれも製品の製造や工事などにかかる費用のことです。**違いは、特定の製品や工事に結び付けられる費用かどうか**という点にあります。直接費・間接費それぞれについて、以下で詳しく解説します。

#### 直接費

**直接費とは、特定の製品の製造や工事のために発生したことが明確な費用のことです。**
  
  
たとえば、次のようなものが該当します。

  

| 項目 | 概要 |
| --- | --- |
| 材料費 | 製品の製造や工事の施工に直接使われる材料の費用 |
| 労務費（直接労務費） | 実際に製品を作る作業員や現場で作業する職人など、直接作業に従事するスタッフの人件費 |
| 外注費 | 加工や施工の一部を外部業者に依頼した際に支払う費用 |
| 運搬費（直接運搬費） | 特定の製品や工事に限定して発生する輸送費 |
| 機械使用料（特定使用分） | 特定の製品や工事だけに使われる機械・設備の使用費 |

  

間接費の項目でも説明しますが、「労務費」や「運搬費」などはその実態によって直接費か間接費かが変わってきます。

#### 間接費

**間接費は複数の製品の製造や工事に共通して発生するために、個別に割り当てるのが難しい費用を指します。**
  
  
そのため間接費においては、一定のルール（作業時間比、材料費比など）に基づいて費用を各製品・工事に振り分ける「配賦作業」が必要です。この配賦作業により、製品・工事の原価を正確に把握できるようになります。
  
  
間接費にはさまざまな費用が該当しますが、たとえば次のようなものが挙げられます。

  

| 項目 | 概要 |
| --- | --- |
| 光熱費 | 複数の製品や工事で共通して使用する電気・ガス・水道などの費用 |
| 減価償却費 | 機械設備や建設用重機、建物など、長期的に使用する資産の価値を少しずつ費用として計上するもの |
| 間接労務費 | 工場長、現場監督員、事務員など、直接製造や施工に携わらないスタッフの人件費 |
| 修繕費・消耗品費 | 工場設備や事務用品などの維持管理にかかる費用のうち、特定の製品・工事に使われていないもの |
| 通信費・事務費 | 電話代、インターネット代、コピー用紙など、全体の運営にかかる費用 |

### 業種ごとの原価項目の違い

管理する原価の項目は業種ごとに異なります。原価管理を行う際には業種ごとの違いについても理解しておきましょう。ここでは、製造業、IT業界、広告業界、建設業界に関する原価項目の例について説明します。

#### 製造業

製造業では、ひとつの製品を製造する際に発生する原価を管理します。
  
  
製造原価に含まれるものは、主に材料費・労務費・経費の3項目に大別されます。**それぞれ直接費と間接費に分けられるため、あわせて6つの項目で管理することになります。**

  

| 製造業の原価項目 | | |
| --- | --- | --- |
|  | 直接費 | 間接費 |
| 材料費 | 製造に必要な原材料などを購入した費用 | 塗料など複数の製造で使用する材料で、特定の製品にどれくらい使用したのかが曖昧なもの |
| 労務費 | 製造作業に関わった人件費など | 生産管理や総務など、直接、製品の製造に関わっていない従業員の人件費など |
| 経費 | 外注加工費など | 製品を製造する際に使う電力など、特定製品への使用を明確にするのが難しい経費 |

#### IT業界

IT業界ではプロジェクト形式で業務を遂行することが多いため、原価管理もプロジェクトごとに行うケースが一般的です。**原則として材料費が発生しないため、他業種に比べて労務費の割合が高くなる傾向があります。**

  

|  |  |  |
| --- | --- | --- |
| IT業界の原価項目 | | |
| 業務委託費 | 動作テストなどの一部の作業を外部業者に委託する際の費用 | |
|  | 直接費 | 間接費 |
| 労務費 | エンジニアの開発工数などの人件費 | 総務などプロジェクトに直接関わっていない従業員の人件費 |
| 経費 | システム利用料、IT資材の購入費など、プロジェクトに直接関わる経費 | 家賃や光熱費など、間接的に発生する経費 |

#### 広告業界

IT業界と同様に広告業界もプロジェクト形式で業務を遂行することが多く、原価管理もプロジェクトごとに行います。**広告業界はひとつのプロジェクトに関わる人が多く、原価の種類が多い点も特徴です。**

  

|  |  |  |
| --- | --- | --- |
| 広告業界の原価項目 | | |
| 媒体費用 | 広告枠の購入費用、広告運用費用など | |
| 業務委託費 | 制作外注費、カメラマンやタレント出演料など | |
|  | 直接費 | 間接費 |
| 労務費 | 広告制作ディレクションなどの人件費 | 総務などプロジェクトに直接関わっていない従業員の人件費 |
| 経費 | 撮影スタジオ代、機材費用、交通費、交際費など | 光熱費、事務所賃貸料など、プロジェクトに間接的に発生した経費 |

#### 建設業界

建設業界における原価管理は、「工事原価管理」ともよばれます。工事原価管理では、ほかの原価管理と同様に、工事にかかる外注費や材料費といった原価がどれだけ発生するかを適切に計算したうえで管理し、コストの改善を図ることが重視されます。
  
  
また建設業界では、**建設業許可を取得するうえで「完成工事原価報告書」という工事原価の内訳を明確に示す書類の提出が欠かせません。**その意味でも、適切な原価管理は事業を行ううえで不可欠な取り組みといえるでしょう。

## 原価計算、予算管理、利益管理との違い

原価管理と似た言葉に、原価計算や予算管理、利益管理などがあります。それぞれの用語と原価管理との違いについて理解しておきましょう。

### 原価計算との違い

原価計算とは、特定の製品の製造やプロジェクトが完了するまでに発生するコストを算出する作業を指します。一方、原価管理は原価計算で算出した結果から含め、企業の収益を確保するために最適な原価設定を検討したり、目標と実際の原価との差分を分析し改善を図ったりする全体の管理業務を指します。
  
  
そのため、**原価計算は「製品やサービスにかかった費用（実績）を正確に集計・記録する『過去』の活動」であり、原価管理は「その計算結果や予算（計画）を基に、コストをコントロールし、利益を確保する『未来』の活動」であるともいえます。**
  
  
原価管理には原価計算が必須の作業となることから、原価計算は原価管理に内包される業務のひとつといえるでしょう。
  
  
【関連記事】
  
[原価計算とは？計算方法や種類、基本知識を解説](http://freee.co.jp/kb/kb-hanbai-kanri/cost_accounting/)

### 予算管理との違い

予算管理は経営管理のひとつであり、売上高や利益などの企業の数値目標を管理することを指します。たとえば、「今年度は販売部門の経費を10％削減する」「研究開発に◯億円投資する」といった会社全体の計画を立て、その執行状況を管理するといった流れです。
  
  
**原価管理はあくまでも特定の製品やプロジェクトにかかるコストや利益率を管理するものなので、全体的な予算管理と比べると階層が階層がひとつ下の概念になります。**
  
  
ここまでにも例を挙げましたが、たとえばある製品にかかる材料費・人件費・外注費などを集計し、実行予算（目標原価）と実績を比較して、予定よりコストが増えそうなら材料の変更や作業手順の見直しをとおして赤字を防ぎます。
  
  
原価管理は、予算管理を構成するひとつの要素といえるでしょう。

### 利益管理との違い

利益管理とは、利益を上げることを念頭に目標を立て、計画的に収支を管理することです。売上アップやコスト削減の取り組みを行うため、**利益管理は原価（コスト）管理とセットの概念と認識しておきましょう。**

## 原価管理の目的とメリット

原価管理の目的とメリットは、下記の2点に集約されます。

### 原価を正確に把握・管理することで利益率向上につながる

**原価管理を行うことで自社の利益を適切に管理および把握し、収益力向上に結び付けることができます。**
  
  
原価を正確に把握できていなければ、「売上は立っているのに利益は出ない」といった危うい状況にもなりかねません。自社製品の原価を把握するプロセスにより適切な製品価格の設定や無駄なコストの削減ができ、利益確保に向けた事業活動の最適化が可能になります。

### 原価変動による影響を抑えるなどのリスク管理ができる

円相場や原油価格の高騰など、経済環境や社会情勢によって原価は大きく変動します。**原価管理はこのような外部環境の変化による経営への影響を最小限に留めるうえでも役立ちます。**
  
  
適切に原価管理を行っていれば原材料費が上がった場合でも、想定しうる損失や別のルートから仕入れた場合のコストをすぐに算出できれば、迅速に対策を講じられます。

## 原価管理の主な手法

原価管理の主な手法には、次の3つがあります。手法ごとのメリット・デメリットもあるため、企業の状況や目的に応じて適切な手法を選択することが重要です。

* [標準原価管理](#content4-1)
* [実際原価管理](#content4-2)
* [直接原価計算](#content4-3)

### 標準原価管理

**標準原価管理は、あらかじめ設定した「標準原価」と現実に発生した「実際原価」を比較し、差異の分析から効率化を図る手法です。**この標準原価管理における「標準原価」とは、過去の実績や理論的な分析に基づいて設定される効率的な製造原価のことです。
  
  
一般的に標準原価管理は管理がしやすく、業務改善や効率性向上に役立てやすいとされています。ただし、標準原価が実際のコストと乖離している場合、現状を正確に把握することが難しくなるというデメリットもあります。

### 実際原価管理

**実際原価管理は、製品を製造する際に「実際にかかった費用」を集計して原価を算出する手法です。**「全部原価計算」ともよばれます。
  
  
実際原価管理では、実際に発生した材料費、労務費、経費から原価計算を行います。財務会計で管理される数値から正確な情報を提供できるというメリットがありますが、予算超過や不要なコストを事前に探すことが難しいという欠点もあります。

### 直接原価計算

**直接原価計算とは、費用を「変動費」と「固定費」に分け、変動費のみを原価として扱い、固定費を期間費用として処理する手法のことです。**
  
  
この方法は売上から変動費を引いた「限界利益（貢献利益）」を重視するもので、損益分岐点の分析がしやすくなります。短期的な利益分析や製品別・工事別の採算を明確にできるというメリットがある一方、原則として財務会計には使用できないため、あくまで管理会計の手法のひとつという位置付けになります。

## 原価管理のおおまかな流れ

財務会計に使用できる「標準原価管理」および「実際原価管理」における原価管理の主な工程を解説します。なお、この工程は業種や製品、システムにかかわらず共通です。

### 1. 標準原価の設定

まずは、標準原価を設定します。標準原価は製造時に目標とする原価額のことです。製造業のケースでは、製造に不可欠な労働力や材料の消費量を、過去の実績や市場調査から統計的に算定します。
  
  
なお、実際にかかった費用やコストを「実際原価」といいます。**標準原価は事前調査による原価の目標値であるため、実際原価が必ず標準原価内に収まるわけではありません。**

### 2. 原価計算

次に、実際原価を求めるための原価計算を行います。製造業の原価計算では、製品を作るうえでかかった費用を材料費・加工費・減価償却費・労務費などの費目に分けて計算します。費目を分けることで、何にどれだけコストがかかったのかを把握できます。
  
  
また、固定費と変動費は分けて計上するようにしましょう。 固定費は、地代家賃など生産量にかかわらず必ずかかる費用のことです。一方、変動費は、材料費のように生産量によって増減する費用のことです。固定費は削減効果が大きく、変動費は製造個数や売上高によって増減するため、**分けておくとコスト削減や売上変動に対する原価の増減を計算しやすくなります。**
  
  
さらに、特定の製品を作る際に直接かかった費用（直接費）と、複数の種類の製品にまたがってかかる費用（間接費）も分けて原価を計算します。間接費は、地代家賃や光熱費、宣伝広告費、一般管理費などが当てはまり、一定の基準を設けて各製品に費用を配賦（はいふ）することで原価に計上できます。

### 3. 差異分析

次に、標準原価と実際原価の差異を分析します。費目別に標準原価と実際原価の差分を見て、どれほど乖離があるのかを確認します。標準原価よりも実際原価が低いならば、想定よりもコストを抑えられていることを意味しています。
  
  
原価計算で細分化した費目によって「どの費目でコストがかかっているのか」を可視化しているため、**標準原価と差異が生じた原因の解明や改善策の構築に役立ちます。**

### 4. 改善

最後に、差異分析で得られた結果に対して、改善策を検討し実行します。
  
  
たとえば、労務費が高い場合は「設備投資して作業を機械化する」「一部の作業を外部委託する」といった改善策で労務費を抑えることができます。
  
  
**変動費よりも固定費のほうがコスト削減効果が大きいため、固定費を優先的に見直すことがコスト改善につながります。**

## 原価管理における課題

原価管理は企業が利益を確保するために欠かせない取り組みですが、実務で課題が生じやすいものでもあります。ここでは、原価管理における主な課題について解説します。

### 経理担当者の負担が大きい

原価管理を行うためには、人件費や材料費、固定費などを理解したうえで正確な原価を算出する必要があり、同時に会計における経験や知識も必要です。さらに**作業が複雑なうえ、会社の収益や経営に直結する業務であるため担当者の負担が大きくなります。**
  
  
また、業界の慣習や企業ごとの決め事などをもって原価計上することも多く、作業が属人化しやすく引き継ぎも難しくなりがちです。

### Excel管理はヒューマンエラーや属人化を招く

原価管理ではExcelで行われることが多いですが、**計算ミスや入力ミスといったヒューマンエラーが発生しやすい点がデメリットです。**また、効率化を図って複雑な関数や計算式を入れると別の担当者が操作しづらく、属人化してしまうリスクも高まります。
  
  
ミスが起こらないようにフォーマットを揃えたり、無料で配布されているテンプレートを使ったりすることも有効ですが、最終的に手入力になる点は変わらないため、根本的な解決は難しいでしょう。

## 原価管理の課題解決に必要なもの

**原価管理を正確に行うには、日々のコストを正確かつタイムリーに把握する仕組みが欠かせません。**しかし、実際の現場では「Excelでの管理に限界を感じている」「データ更新に時間がかかり分析が追いつかない」といった課題も多いでしょう。
  
  
ここでは、原価管理が抱える典型的な問題とその解決に向けたポイントを紹介します。

### Excelで原価管理をするデメリット

導入のしやすさや使いやすさからExcelで原価管理を行う企業も少なくありませんが、Excelで原価管理を行うと次のような課題が発生しやすいといえます。

Excelによる原価管理のデメリット

* データ集計に時間がかかるため、リアルタイムな損益把握ができない
* 入力ミスや転記漏れが起こりやすく、数値の信頼性が低下する
* ファイルが属人化してしまい、管理プロセスがブラックボックス化する

これらの問題が積み重なると、データを分析・活用する機会がなくなってしまいます。単なる記録にとどまってしまうと、せっかくのデータを経営判断に活かせません。

### 原価管理の課題解決にはシステム導入の検討を

**原価管理における課題を解決するためには、原価管理システムの導入がおすすめです。**原価管理システムとは原価管理を自動化できるソフトウェアのことで、負担の大きい原価管理業務を効率的に行えるのが最大のメリットといえます。
  
  
原価管理システムでは、必要最低限の項目を入力すれば簡単に原価データを作成できるほか、差異分析や損益計算、間接費の配賦（はいふ）もシステムが自動で行ってくれます。
  
  
データがシステム内で一元管理されるため個別にExcelで管理する必要もなくなり、業務の効率化が図れます。また、リアルタイムでの損益把握や差異分析が可能になるため、迅速かつ精度の高い経営判断が可能です。
  
  
加えて、ほかの基幹ツールや会計システムと連携することで正確で漏れのない原価管理が実現できるという利点もあります。

## まとめ

原価管理を適切かつ正確に行えれば、事業活動からより安定的に利益を担保できるようになります。複雑になりがちな原価計算を効率化したいなら、原価管理システムを導入し、自動化や効率化を図るのがおすすめです。自社の商材・生産方式にあった原価管理システムを選ぶことで、さらなる利益向上につながるでしょう。

## よくある質問

### 原価管理とは？

原価管理とは、商品の製造やサービスの提供に必要な「原価」を管理することで、利益の改善につなげる手法です。原価管理は「コストマネジメント」とも呼ばれ、原価を分析・管理することにより利益の最大化を目指します。
  
  
詳しくは記事内の「[原価管理とは](#content1)」をご覧ください。

### 原価計算と原価管理の違いは？

原価計算とは、特定の製品製造やプロジェクトが完了するまでにかかるコストを算出する作業を指します。一方の原価管理は、原価計算で算出した結果から、企業の収益を確保するために最適な原価設定をしたり、目標と実際の原価との差分を分析し改善を図ったりする全体の管理を指します。
  
  
詳しくは記事内の「[原価計算との違い](#content2-1)」をご覧ください。

### 原価管理にはどんなやり方がある？

原価管理の主な手法には、「標準原価管理」「実際原価管理」「直接原価計算」の3つがあります。手法ごとにメリット・デメリットがあることから、企業の状況や目的に応じて適切な手法を選択しましょう。
  
  
詳しくは記事内の「[原価管理の主な手法](#content4)」をご覧ください。
