# 日本政策金融公庫「国民生活事業（旧・国民生活金融公庫）」での資金の借り方とは？

出典: https://www.freee.co.jp/kb/kb-finance/national-life-business-funding/

---

日本政策金融公庫の国民生活事業（旧：国民生活金融公庫）は、個人事業主やフリーランスなどの小規模事業者やこれから創業する企業を対象にした融資や、個人の教育資金の融資を行っています。

小規模事業者や創業したばかりの企業は、資金調達に悩みを抱えることが少なくありません。そんな時に、銀行よりも低金利で借入でき、さらに資金を借りやすい日本生活金融公庫は非常に心強い存在です。

本記事では、日本政策金融公庫の国民生活事業が提供する融資制度の魅力と、具体的な借り方のポイントについて解説していきます。

## 日本政策金融公庫の国民生活事業（旧・国民生活金融公庫）とは？

日本政策金融公庫とは、政府が100%出資する政府系の金融機関です。民間の金融機関の業務を補完することが目的で、銀行が融資を行いづらい小規模事業者や創業期の中小企業への融資・支援を行っています。国民の生活の向上も理念に掲げており、個人を対象に教育ローンを提供している点も特徴の一つです。

2008年10月1日に設立され、前身は、国民生活金融公庫、中小企業金融公庫、農林漁業金融公庫です。**旧・国民生活金融公庫の業務は国民生活事業が引継ぎ**ました。

|  |  |
| --- | --- |
| 国民生活事業 （旧・国民生活金融公庫） | 個人企業や小規模企業向けの小口資金を融資。融資額の平均は約700万円。短期の運転資金も取り扱いあり。 |
| 中小企業事業 （旧・中小企業金融公庫） | 中小企業向けに長期事業資金を融資。融資額の平均は約1億円。短期の運転資金の取り扱いはなし。 |
| 農林水産事業 （旧・農林漁業金融公庫） | 農林漁業や国産農林水産物を取り扱う事業者に長期事業資金を融資。 |

国民生活事業の融資先の約9割は、従業者9人以下の小規模事業者、約半数が個人企業です。平成30年度の融資実績は2兆1,684億円で、そのうち7.9％（年間約12万件）が教育貸付（国の教育ローン）です。

## 国民生活事業の融資とは

国民生活事業が提供する融資メニューは「セーフティネット貸付」や「創業企業・事業再生・事業継承の支援」、「ソーシャルビジネス、海外展開支援」など様々で、事業の成長ステージや状況に沿った支援を受けることができます。

| --- | --- | --- | --- |

国民生活事業のなかでも、特に人気が高いのが「**新創業融資制度**」です。新たに事業を始める人や事業を開始して間もない人でも、**無担保・無保証人**で利用することができます。借入申込みに必要な事業計画書や各種書類の準備が必要で、借入を成功させるためにも入念に準備する必要があります。書類の準備に不安がある場合は、[創業融資freee](https://www.freee.co.jp/funding/)の利用がおすすめです。3ステップで簡単に必要書類を作成できます。

## 国民生活事業からの資金の借り方

日本政策金融公庫から調達した資金は、主に「運転資金」「設備資金」として使われます。借入を申し込む際は、資金の使途を明確にし、目的に合った融資制度に申し込みましょう。

融資の申し込みをする際は、[事業資金相談ダイヤル](https://www.jfc.go.jp/n/inquiry/index.html)に電話をしてアポイントを取った後、[所定の借入申込書](https://www.jfc.go.jp/n/service/dl_kokumin.html)を持参もしくは郵送で提出します。Webサイトから申し込みをしたい場合は、[事業資金 お申込受付のページ](https://www.m.jfc.go.jp/sysped/ped010)からも手続き可能です。

申込の際は、借入申込書と一緒に以下の書類を提出する必要があります。

|  |  |
| --- | --- |
| 個人事業 | 最近2期分の申告決算書（申告している場合） |
| 法人 | ・最近2期分の確定申告書・決算書（勘定科目明細書を含む）  ・最近の試算表（決算後6ヵ月以上経過している場合または事業を始めたばかりで決算を終えていない場合） |
| 設備資金の申し込み | 見積書 |
| 初回の利用 | ・創業計画書  ・企業概要書  ・法人の履歴事項全部証明書または登記簿謄本 |

許認可が必要な事業を始める場合は、事前に取得をしておきましょう。法人として融資を受ける場合は、法人登記が完了している必要があります。

面談では、具体的に資金の使い道や事業計画について話をします。「融資をする価値がある」「返済能力がある」など、良い印象を持ってもらえるように、入念に準備をしましょう。店舗や工場がある場合は担当者が実際に訪問することもあります。審査を経て融資OKであれば資金が銀行口座に振り込まれ、その後、返済が始まります。なお、インターネット専業銀行への振り込みはできないため、必ず窓口がある銀行の事業用口座を開設しておきましょう。

### 融資の審査を有利にするために

小規模事業者や中小企業の支援を目的にしているとはいえ、100%資金が借りられるわけではありません。審査を少しでも有利にするために、以下の点に気をつけましょう。

* 公共料金や税金を延滞しない
* 自己資金をできるだけ準備する
* 説得力のある事業計画書を作成する

まず、クレジットカード料金や公共料金、税金の長期滞納など、個人の信用情報に問題がある場合は、審査に通りづらくなる可能性があります。また、審査を有利にしたいのであればできるだけ自己資金はあった方が良いです。事業を始める前に、自己資金を貯めておくことをおすすめします。

「事業計画に無理がある」など、面談で悪印象をもたれてしまった場合も審査は厳しくなります。事前に資料はしっかりと用意をして、身嗜みを整えて面談に臨みましょう。

## 資金調達の目的と手段をよく知ることが重要

資金調達が必要になるシーンは、積極的に事業を成長させたい時や、逆に事業が苦境に立たされた時など様々です。目的によって、いつまでに・いくら必要なのかも異なります。

日本政策金融公庫の国民生活事業が提供する融資は、まさにこういった小規模事業者のニーズに応えるものです。しかし融資のために多くの資料を準備する必要がありますし、申込から融資実行まではある程度の期間が必要です。このため、短期間でスピーディーに資金調達したい場合には向かないこともあるでしょう。審査に落ちてしまう可能性もあります。

資金調達には、日本政策金融公庫意外にも様々な種類があるため、それらの特性と自社の状況を鑑みた上で、適切な資金調達方法を選ぶことが重要です。その第一歩が、自社の資金繰りについて把握することです。
