## 所得税、復興特別所得税

所得税とはその名の通り、**所得にかかる税金**です。
  
  
所得とは、「収入」から「経費」を差し引いたものです。その年の全ての「所得」から、「所得控除」を差し引いた残りの「課税所得」に税率を適用して税額を計算します。
  
  
復興特別所得税とは、東日本大震災からの復興に必要な財源を確保するために新設された税金です。
  
  
平成25年（2013年）から令和19年（2037年）までの各年については、復興特別所得税を所得税と合わせて申告・納付します。
  
  
復興特別所得税は、基準所得税額（所得税額から差し引かれる金額を差し引いた後の所得税額）に2.1％の税率を乗じて計算されます。平成25年（2013年）から令和19年（2037年）までの間に発生した所得については、源泉所得税と合わせて復興特別所得税が徴収されます。

### 所得税の税率

所得税は、課税所得額により5～45％の課税率となる超過累進課税方式が採られています。

**所得税の速算表**

|  |  |  |
| --- | --- | --- |
| 課税される所得金額 | 税率 | 控除額 |
| 1,000円 から 1,949,000円まで | 5% | 0円 |
| 1,950,000円 から 3,299,000円まで | 10% | 97,500円 |
| 3,300,000円 から 6,949,000円まで | 20% | 427,500円 |
| 6,950,000円 から 8,999,000円まで | 23% | 636,000円 |
| 9,000,000円 から 17,999,000円まで | 33% | 1,536,000円 |
| 18,000,000円 から 39,999,000円まで | 40% | 2,796,000円 |
| 40,000,000円 以上 | 45% | 4,796,000円 |

参考：国税庁「[所得税の税率](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/2260.htm)」
  
  
所得税は、所得が増えるごとに税率が上がるよう設定されています。
  
  
個人事業主の方は、あまり税金を払いたくないけど、売上げを増やしたいと思っているのではないでしょうか。そのため、しっかりと経費を計上することで、所得を減らし、節税につなげることができます。
  
  
節税を考えているのであれば、上手に経費を計上していきましょう。

## 住民税

**住民税は、納税者の住所や事業所がある都道府県や市区町村に納める税金です**。
  
  
個人事業主の場合は、前年の所得に応じて課税される所得割と、所得金額に関係なく一定の税率で課税される均等割を元に税額が計算されます。確定申告をしていれば、自動的に納付書が送付られてきます。

### 所得割

所得割は、前年の所得金額に応じて課税されます。
  
  

  
（注１）分離課税となる退職所得については、「１１　退職金にかかる住民税」を参照してください。  
（注２）それぞれの所得金額は、損失の繰越控除をしたあとの金額です。
  
  
＜個人住民税所得割額の計算の流れ＞
  

参考・引用元：東京都主税局「[個人住民税の所得割](https://www.tax.metro.tokyo.lg.jp/kazei/kojin_ju.html#gaiyo_02)」

### 均等割

均等割は、定額で課税されます。個人都民税の税額は1,500円、個人区市町村民税の税額は3,500円です。
  
  
※平成26年度から令和5年度までの間、地方自治体の防災対策に充てるため、個人住民税の均等割額は都民税・区市町村民税それぞれ500円が加算されています。

住民税も、上手に経費を計上することで所得を減らすこができ、節税につながります。

## 消費税

**消費税は、消費一般に広く公平に課税する間接税です**。
  
  
消費税の納税義務があるかどうかは、次の3つのパターンのいずれかに該当するかどうかで判断されます。

1. 2年前の課税売上高が1,000万円を超えている。
2. 2年前の課税売上高が1,000万円以下だが、消費税課税事業者選択届出書を税務署へ提出している  
   ※消費税課税事業者選択届出書とは、消費税の免税事業者が消費税を納める事業者になるための届出のことです。
3. 1、2には該当せず、且つ1年前の1月1日から6月30日までの期間の課税売上高が1,000万円を超えている  
   ※なおこの期間の1,000万円の判定は、課税売上高に代えて給与等支払額の合計で判定することもできます。

上記に該当するためには、事業の規模がかなり大きくなければなりません。起業したばかりで収入がまだ少ない場合は、あまり気にする必要はありません。

## 健康保険

**健康保険とは、病気やケガによる休業、出産や死亡などを補償する公的医療保険制度です**。会社員は会社の健康保険組合に強制的に加入しますが、個人事業主の場合は自分で健康保険に加入しなければなりません。
  
  
個人事業主は一般的に国民健康保険に加入していると思われがちです。しかし、国民健康保険は前年の所得によって保険料が異なり、扶養も認められていません。これは安定した収入がない個人事業主や扶養家族がいる個人事業主にとっては悩みどころです。
  
  
しかし、実は個人事業主の方は、**国民健康保険以外の健康保険に加入する**という選択肢もあります。

### 健康保険組合の任意継続

まず、個人事業主になる前に会社勤めをしていた場合は、**会社員時代の健康保険組合の任意継続**することができます。ただし、会社員時代に会社と折半で支払っていた保険料を全額支払わなければなりません。

### 業界に特化した国民健康保険組合や団体・協会に加入する

もう一つの選択肢は、業界ごとに設けられている健康保険組合に加入することです。有名なところですと、クリエイター向けの「[文芸美術国民健康保険組合](http://www.bunbi.com/)」という組合があります。収入に関係なく保険料が一律なので、起業したばかりの個人事業主の方も安心です。
  
  
また、美容業界の方には「[東京美容国民健康保険組合](http://kokuho-tokyobiyo.or.jp/)」。広くフリーランスや個人事業主の方向けには「[プロフェッショナル＆パラレルキャリア・フリーランス協会](https://www.freelance-jp.org/)」などがあります。
  
  
これらの健康保険料は、社会保険料控除として所得から控除することができます。

## 国民年金

健康保険と同様に、個人事業主であれば国民年金にも自分で加入しなければなりません。会社員の場合、厚生年金にも加入していますが、個人事業主の場合、基本的には国民年金のみです。
  
  
そのため、「老後にもらえる年金額が心配」という人もいるでしょう。そこで、国民年金に上乗せして保険料を支払うことで老後の年金を上乗せする方法を３つ紹介します。

### ①[付加年金](https://www.nenkin.go.jp/service/jukyu/sonota-kyufu/1go-dokuji/20140625.html)

国民年金に上乗せで付加できるもの。通常の国民年金保険料に、月額400円上乗せして支払うことで、将来受給する年金額を増やせる制度のことです。
  
  
付加年金の加入には条件があります。

#### 加入できる人

* 国民年金第1号被保険者（自営業者・農業者・学生・アルバイト・無職の人で20歳以上60歳未満）
* 65歳以上の人を除く任意加入被保険者

#### 加入できない人

* サラリーマンや公務員などの第2号被保険者
* 第2号被保険者の配偶者である第3号被保険者
* 国民年金保険料の免除・猶予を受けている人
* 国民年金基金の加入者

付加年金の加入期間は決まっており、原則として20歳から60歳未満の月までの40年間です。付加年金は老齢基礎年金と合算されるため、老齢基礎年金と同時に支給されます。
  
  
ただし、国民年金基金との併用はできませんのでご注意ください。

### [②国民年金基金](https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/0000059350.html)

付加年金と同様に、国民年金に上乗せで付加できるものです。月々の掛け金は一定額で、口数を選ぶことができます。掛け金に応じて、将来受け取る年金額を増やすことができます。

  
ただし、付加年金との併用はできません。

### ③確定拠出年金

確定拠出年金とは、個人事業主が掛金を拠出し、自ら資産を運用し、運用の成果によって将来の年金受取額が決まる制度のことです。確定拠出年金には、企業が加入する「企業型確定拠出年金」と、個人が加入する「個人型確定拠出年金」の2種類があります。
  
  
運用がうまくいけば良いのですが、失敗すると元本割れのリスクがあるので注意が必要です。

## まとめ

個人事業主なら知っておきたい！納める税金と社会保険の種類について解説しました。個人事業主が加入できる税金や保険には様々な種類があり、混乱してしまった方もいらっしゃるのではないでしょうか。
  
  
しかし、税金や社会保険の知識をしっかりと身につけることで、正しい確定申告をすることができます。健康保険や国民年金を支払うことで、いざという時に役立ち、将来の自分を守ってくれますし、**確定申告**の際には**社会保険控除**として利用することもできます。
  
  
個人事業主が納めるべき税金や加入できる保険の種類について理解し、事業を成功させましょう。

### 4.確定申告書を自動作成！

[freee会計](https://www.freee.co.jp/kakuteishinkoku/)は取引内容や質問の回答をもとに確定申告書を自動で作成できます。自動作成した確定申告書に抜け漏れがないことを確認したら、税務署へ郵送もしくは電子申告などで提出して、納税をすれば確定申告は完了です。

e-Tax（電子申告）を検討されている方は[こちら](https://www.freee.co.jp/accounting/individual/e-tax-start/)をご覧ください。
