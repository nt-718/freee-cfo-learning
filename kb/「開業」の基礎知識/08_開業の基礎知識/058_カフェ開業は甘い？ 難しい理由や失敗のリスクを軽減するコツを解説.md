カフェを開業し、経営を続けていくことは想像以上に大変なことです。カフェの開業を甘く考えていると、事業運営で失敗してしまう恐れがあります。
  
  
**安定的な売上を確保するためには、コンセプトの確立や競合との差別化・メニュー開発など、開業前に経営戦略をしっかりと立てることが大切です。**
  
  
本記事では、カフェ開業を考えている人に向けて、開業前に考えておきたい戦略について紹介します。

## カフェ開業が難しいと言われる理由

**カフェを開業するには、開業前の物件の選定や開業届出書の作成、開業資金の調達など多くの準備が必要です。**開業後も幅広い業務に対応する必要があり、多忙な日々を過ごすことになるでしょう。
  
  
また、利益率が低い場合は、想定通りに収益を得られるとは限りません。以下で、カフェ開業の難しさについて詳しく解説します。

### やるべきことが多く仕事がハード

**カフェを開業する場合、準備期間はもちろんですが、運営時もやることが多く、食材の仕込み・在庫管理・接客・調理など多岐にわたります。**
  
  
一般的に、カフェにおける仕事内容と担当者は以下の通りです。

  

|  |  |
| --- | --- |
| 店長 | ・店舗全体の運営  ・売上管理  ・勤務シフト作成  ・新メニューの考案  ・スタッフの採用など |
| バリスタ | ・コーヒーを淹れる |
| キッチンスタッフ | ・フードやスイーツの調理・製造  ・食材・キッチンの管理など |
| ホールスタッフ | ・接客  ・会計  ・調理補助など |

  

スタッフを雇用せずに自分や家族だけで切り盛りする場合は、上表の仕事を全て行なわなければなりません。スタッフを雇用する場合も、採用活動や採用後の研修内容の設計や実施を行う必要があります。
  
  
**さらに、カフェの集客のためには、チラシ作成やSNSの更新なども必要です。**

### 利益率が低く採算が取れない

**自宅以外でカフェを開業する場合、テナントの家賃が発生します。**立地によっては毎月数十万円の家賃が発生し、固定費としてコストがかかり続けます。固定費は売上に関係なくかかるため、収益に大きく影響します。
  
  
**また、カフェは勉強や仕事をする客層や、ゆっくり会話を楽しむ客層などが多いため回転率が低く、高額な客単価は期待できません。**固定費がかかりやすいうえに大きな利益を期待しづらいため、利益率が低くなる傾向にあります。
  
  
事前に利益率を高めるための方法を検討し、収支のシミュレーションを行っておきましょう。

### 需要の把握が難しい

**カフェは客層によって需要が大きく変わるため、当初想定していたような運営ができな可能性があります。**
  
  
開業前は「ファミリー層に来てほしい」と考えていても、駐車場がなくなかなか集客に至らない・学校が近くて若い学生ばかり来るなど、ターゲットとする客層や立地を間違ってしまうとカフェの開業が失敗する原因になります。
  
  
事前に客層や立地を十分に検討しなければ、安定した集客は見込めません。

### 経営知識が不足している

**カフェを経営するには、資金繰り・事業環境・売上向上などについて考える必要があるため、経営知識が欠かせません。**
  
  
たとえば利益率を改善するには、売上と原価のバランスや適切な客単価、席あたりの回転率などを分析する必要があります。分析は大変ですが、店の課題を把握して適切に対策をすることが、経営改善につながります。
  
  
また、効率よく集客するためにも、マーケティングやSNSの運用方法について学ぶことが大切です。

## あなたはどっち？ カフェオーナーのタイプ

カフェを開業したいと考えている人は、大きく2つのタイプに分けることができます。
  
  
戦略を考えるうえで知っておきたいメリットとデメリットを確認しましょう。

### 想い重視型

**「こだわりのコーヒーを多くの人に味わってもらえるカフェにしたい」**
  
**「地域の人たちがリラックスできるカフェを作りたい」**
  
  
上記の想いでカフェ開業を目指す人は、「想い重視型」です。
  
  
**なかには、「少しくらい採算が合わなくても、自分が理想とするカフェを作りたい」という気持ちの人もいるかもしれません。**
  
  
「想い重視型」であるメリットは、自分が理想としているカフェ運営を実現できる点です。ある程度資金繰りに余裕がある人や、収益性よりも理想を重視している人に向いています。
  
  
しかし、想いを重視するあまり、採算が合わない経営を続けることはおすすめできません。**理想の物件でカフェ開業をしたとしても、家賃を滞納することがあれば、経営に大きく影響を及ぼします。**
  
  
また、収益性を無視するとカフェの運営を続けるのが難しくなります。

### 物件重視型

**「手持ちの不動産に空きが出た。カフェ経営で利益を得たい」**
  
**「子どもが独立して部屋が余っているから、カフェを開いてみたい」**
  
  
上記のように、カフェ自体にはこだわっていないけれど、物件を有効活用する方法としてカフェ開業を考えている人は「物件重視型」です。
  
  
保有している不動産の一部を有効活用できれば、固定費を抑えられます。また、開業する場所を探す手間を抑えられるため、開業までがスムーズです。
  
  
**しかし、物件を活用するためだけにカフェを開業した場合、大きな収益を得るのは難しい傾向にあります。**カフェで売上を出すには立地が重要で、開業する際には周辺地域との兼ね合いが大切になってくるためです。

## カフェオーナーのタイプ別の戦略とは？

想い重視型と物件重視型のカフェオーナーは、それぞれデメリットとなる点が異なるため、立てるべき戦略も異なります。
  
  
デメリットを解決し、経営がスムーズになる戦略を考えてみましょう。

### 共同経営（想い重視型・物件重視型）

たとえば、想い重視型オーナーと物件重視型オーナーの共同経営です。
  
  
**想い重視型の人は、家賃の負担が少なく（場合によってはゼロに）なります。一方で、物件重視型の人は場所を提供するだけで、カフェの実務を想い重視型の人に任せることができます。**
  
  
費用や収益の分配についての交渉は必要ですが、初期投資を抑え、互いのデメリットを補い合うことができます。
  
  
インターネットなどを使って情報を集め、カフェ開業を一緒にできる共同経営者を探してみましょう。

### 期間限定出店（想い重視型・物件重視型）

**カフェ経営はしたいけれど、不安を感じている人は、まずは期間限定出店で試してみるという方法もあります。**低コストで始められるうえに、うまくいかなかった場合も少ないリスクで済み、実践を通して学びも得られるでしょう。
  
  
想い重視型の人は、期間限定出店により自身の考えたメニューやコンセプトで安定経営を行えられるか調べることができます。また、物件重視型の人は、カフェ経営が自身に合っているのかを判断する機会にできます。
  
  
**最短1日からレンタルできるスペースもあるため、カフェ開業に不安を感じている人におすすめです。**

### 穴場物件（想い重視型）

カフェ開業をするなら、集客を望むのは当然のことです。**しかし、想い重視型の人で駅近など人気物件の契約を想定している人は、考え直す必要があるかもしれません。**人気物件は、集客を期待できるとともに、家賃が高くつくため、毎月の支出が高くなるためです。
  
  
想い重視で自分の好きなコーヒーを淹れたいと考えていたのに、固定費が圧迫するせいでお金儲けのことばかり考えなくてはいけない、ということになりかねません。
  
  
なるべく家賃を抑える方法を探しているのであれば、以下のような穴場物件を探してみるのもひとつです。

穴場物件の例

* 遊休施設（※）
* 空きスペースのある商業施設
* 空き家になっている古民家

  

※使用されていない施設のことで、廃校になった小学校などがカフェに生まれ変わった事例も多くあります。

比較的低コストで借りることができるため、初期投資を抑えることが可能です。
  
  
**また、インターネットで「カフェ 運営者 募集」といったキーワードで検索すると、出店者を公募しているサイトも出てきます。**
  
  
自分の理想とぴったりの物件を見つけるのはなかなか難しいですが、まずは低コストのカフェ経営からスタートし、自信をつけてから次の店舗をもつという点も大切です。

### 物件提供（物件重視型）

**物件重視型の人は、自身でのカフェ経営ではなく、カフェを経営したいと思っている人に場所を提供するのもおすすめです。**
  
  
人を雇うリスクやメニュー開発など、カフェの経営を成功させるには多くの知識が必要なため、長続きさせるのは大変です。カフェを開業する競合の中には、数百万円単位の借金をし、熱い想いを必死に実現させようとする人もいます。
  
  
**物件提供のみに照準を絞ることで、自身の動きや学びも効率的になります。**カフェ運営者募集のサイトやSNSを活用して告知をすれば、低コストで多くの人に知ってもらうことも可能です。

## 夢のカフェを 現実の「事業」とするために必要なこと

**自分のタイプを見極めて、どのようなカフェ経営者を目指すかを考えたら、具体的な出店準備を始めましょう。**
  
  
以下では、カフェ開業における手続きや、事業運営に必要なことを紹介していきます。

### 開業届出書（法人設立届出書）の提出

**カフェを開業すると、自分が事業主となるため、開業前に税務署へ開業届を提出（法人の場合は法務局へ法人設立届出）しましょう。**
  
  
なお、収入から経費を差し引いた所得から、一部を納税する義務が発生します。まずは、「個人事業の開業届出書」を所轄の税務署に提出し、開業年度から所得税などを納めましょう。
  
  
開業届について詳しく知りたい方は、「[開業届とは？個人事業主が知っておくべき基礎知識や提出するメリット・注意点について解説](https://www.freee.co.jp/kb/kb-kaigyou/basic-knowledge/)」をご覧ください。
  
  
また、株式会社などの法人を設立することで、個人事業よりも節税できる場合もあります。法人を設立するときには「法人設立届出書」を税務署に提出し、設立年度から法人税などの納税を行いましょう。
  
  
**また、個人・法人いずれの場合でも、確定申告は必須です。**帳簿をつけるのが簡易な「白色申告」もありますが、個人事業主の場合、条件を満たすことで最大65万円の控除を受けられる「[青色申告](https://www.freee.co.jp/kb/kb-blue-return/blue-white-difference/)」がおすすめです。
  
  
確定申告を税理士に依頼しない場合、自分で会計ソフトを活用することで、青色申告で求められる複式簿記や複雑な計算も負担をかけずに行うことができます。
  
  
必要経費は、カフェの開業後だけではなく、開業準備のために使った費用も経費として計上できます。領収書は忘れずに保管しておきましょう。

### 開業の準備や手続きを進める

**開業の準備として、開業するためのテナントを探したり、食品衛生責任者（調理師免許）の講習を受けて資格を取得したりする必要があります。**
  
  
また、防火管理者の講習を受けて資格を取得したり、保健所に営業許可申請を提出して衛生的に問題がないかの検査を受けたりしましょう。

### 開業資金の調達

**開業のために必要な資金は、開店直後の運転資金も含めて数百万円から1,000万円以上は用意しておくと安心です。**居抜き物件を転用したり、外装やインテリアをDIYしたり、開業費用を抑える方法もあります。
  
  
**実績がない事業の場合は、日本政策金融公庫に代表される公的金融機関に融資を申し込み資金を調達しましょう。**
  
  
ただし、実際に融資を受けるためには「事業計画書」など必要書類の作成が必要であり、内容次第では融資を断られることもあります。現実的な事業計画を立て、「成功する見込みのある事業であること」を示してください。
  
  
日本政策金融公庫以外にも、地元の自治体（都道府県・市町村）や、金融NPOによる融資制度もあります。一定の要件を満たすことで、返済不要の補助金・助成金を受けられる自治体もあるため、開業する地域の制度は念入りに調べましょう。
  
  
なお、カフェの開業にあたって必要となる具体的な費用の項目は以下の通りです。

  

|  |  |
| --- | --- |
| 物件取得費 | ・保証金  ・礼金  ・仲介手数料など |
| 内装工事費 | ・設計費を含む |
| 什器備品費 | ・テーブル  ・チェア  ・食器棚  ・冷蔵庫  ・調理器具  ・オーディオ設備  ・食器など |
| 広告宣伝費 | ・ホームページ制作費  ・広告チラシ制作費など |
| 商品仕入費 | ・食材  ・コーヒー豆など |

  

想い描く理想のカフェを作ろうと思うと、どうしても費用がかさみます。資金調達の方法にはいくつも種類があるので、それぞれ特徴を把握しておけば事業準備をスムーズに進められるでしょう。
  
  
事業運営に必要な資金調達の方法については、「[資金調達とは？ 企業の資金調達方法やメリット・デメリットを簡単に解説](https://www.freee.co.jp/kb/kb-launch/found-paperwork-articles-of-incorporation/)」で解説しています。

## カフェの開業後に失敗を避けるコツ

**カフェの開業後に失敗を避けるためには、事前に事業計画を練ることや徹底した市場調査が欠かせません。**
  
  
具体的に、カフェのオーナーとしてやるべきことを見ていきましょう。

### 事業計画をしっかりと練る

**カフェを開業する前に、「売上はどの程度見込めるのか」「集客しやすい立地なのか」「毎月のランニングコストはどの程度なのか」など、事業計画をしっかり練りましょう。**
  
  
また、内装の必要性や従業員を雇うのかどうか、原価計算なども含めてトータルで考えて、利益率をシミュレーションすることが大切です。
  
  
開業して間もない頃は赤字期間が発生する可能性を踏まえ、余裕を持った資金繰りを意識しましょう。
  
  
お金に関する計画だけではなく、提供するサービスにも意識を向ける必要があります。**見込んでいる客層に応じたドリンク・フードのメニューを用意したり、キャッシュレス決済の導入も考えてみてください。**

### 市場調査とロケーションの選定を綿密に行う

**自宅の一部をカフェにする場合、その地域にカフェの需要があるか考える必要があります。**テナントを借りて開業する場合は、どのような人が住んでいるのか、どのような人が通るのかリサーチしましょう。
  
  
たとえば、ビジネスマンが多い地域なら、朝食や昼食時の需要が見込まれます。そのため、フードのメニューを充実させると、安定したリピーターを獲得できる可能性が高くなります。
  
  
なお、出店する場所は、実際に自分で歩いて確認してから決めましょう。市場調査やロケーションの選定を怠ると、集客できず赤字が発生し続ける事態につながります。

### 市場調査とロケーションの選定を綿密に行う

**カフェの集客には、SNS運用が効果的です。**開店準備の段階からSNSをこまめに更新し、ターゲットになりそうな人と交流しましょう。
  
  
SNSを起点に口コミが広がれば、一気に人気店になる可能性もあります。開店準備の段階から見守ってくれたお客様が、常連になることもあるかもしれません。
  
  
また、Facebook広告など各種SNSを介した広告では、特定の年齢層・住所・趣味・嗜好の人をピンポイントに狙って、効果的に広告を出すことができます。1日数百円からでも出稿できるので、試してみるとよいでしょう。
  
  
立地によって商圏がある程度限定されるカフェの集客は、地方紙やフリーペーパーなどに広告を出すことも検討してみてください。チラシやショップカードを作って、近隣の店舗に置いてもらうのも有効でしょう。
  
  
また、ユニークなコンセプトやビジュアルに特徴があるカフェであれば、プレスリリースを出すことによって、テレビや雑誌、ネットメディアなどが取材に来て、宣伝してくれる可能性もあります。メディア向けの資料はしっかりと作り込み、取材対応も丁寧に行いましょう。

## まとめ

**カフェの開業は、やるべきことが多いうえに利益率が低いため、採算が取れない可能性があり経営が甘いとはいえません。**
  
  
カフェの経営を成功させるためには、計画的に開業の準備を進めたり、市場調査やロケーションの確認を行ったりすることが欠かせません。また、余裕を持った資金繰りを意識し、赤字期間が発生しても焦らず済むように準備しましょう。

## freee開業なら、税務署に行かずに開業届をかんたんに作成

個人事業を始める際には「**開業届**」を、青色申告をする際にはさらに「**青色申告承認申請書**」を提出する必要があります。 記入項目はそれほど多くはありませんが、どうやって記入したらいいのかわからないという方も多いと思います。
  
  
そこでおすすめなのが「[freee開業](https://k.secure.freee.co.jp/personal/users/register)」です。ステップに沿って簡単な質問に答えていくだけで、必要な届出をすぐに完成することができます。

#### freee開業で作成可能な5つの届出

**1. 個人事業の開業・廃業等届出書**
  
開業届のことです。
  
  
**2. 所得税の青色申告承認申請書**
  
青色申告承認申請書は**事業開始日から2ヶ月以内、もしくは1月1日から3月15日まで**に提出する必要があります。期限を過ぎた場合、青色申告できるのは翌年からになるため注意が必要です。
  
  
**3. 給与支払事務所等の開設・移転・廃止届出書**
  
家族や従業員に給与を支払うための申請書です。
  
  
**4. 源泉所得税の納期の特例の承認に関する申請書**
  
**原則毎月支払う源泉所得税を年2回にまとめて納付するための手続**です。毎月支払うのは手間ですので、ぜひ提出しましょう。
  
  
**5. 青色事業専従者給与に関する届出・変更届出書**
  
青色申告をする場合に、家族に支払う給与を経費にするための手続です。青色申告をして家族に給与を支払う場合は必ず提出しましょう。

## freee開業の使い方を徹底解説

[freee開業](https://www.freee.co.jp/lp/kaigyou/07/)を使った開業届の書き方は、準備→作成→提出の3ステップに沿って必要事項を記入していくだけです。

#### Step1：準備編

準備編では事業の基本情報を入力します。迷いやすい職業欄も多彩な選択肢のなかから選ぶだけ。

  

事業の開始年月日、想定月収、仕事をする場所を記入します。
  
想定月収を記入すると**青色申告、白色申告のどちらが、いくらお得かも自動で計算されます。**

#### Step2：作成編

次に、作成編です。

  

申請者の情報を入力します。
  
名前、住所、電話番号、生年月日を記入しましょう。

  

給与を支払う人がいる場合は、上記のように入力をします。
  
今回は準備編で「家族」を選択しましたので、妻を例に記入を行いました。

  

さらに、**見込み納税金額のシミュレーション**も可能。
  
※なお、売上の3割を経費とした場合の見込み額を表示しています。経費額やその他の控除によって実際の納税額は変化します。
  
  
今回は、青色申告65万円控除が一番おすすめの結果となりました。

#### Step3：提出編

最後のステップでは、開業に必要な書類をすべてプリントアウトし、税務署に提出します。

  

入力した住所をもとに、提出候補の地区がプルダウンで出てきます。地区を選ぶと、提出先の税務署が表示されますので、そちらに開業届けを提出しましょう。

  

届け出に関する説明とそれぞれの控えを含め、11枚のPDFが出来上がりました。印刷し、必要箇所に押印とマイナンバー（個人番号）の記載をしましょう。
  
  
郵送で提出したい方のために、**宛先も1ページ目に記載**されています。切り取って封筒に貼りつければ完了です。
  
  
いかがでしょう。
  
事業をスタートする際や、青色申告にしたい場合、切り替えたい場合など、届出の作成は意外と煩雑なものです。
  
  
しかし、[freee開業](https://k.secure.freee.co.jp/personal/users/register)を活用すれば、無料ですぐに届け出の作成が完了。
  
  
また、確定申告書の作成も[freee会計](https://secure.freee.co.jp/users/sign_up)を使えば、ステップに沿ってすぐに完了します。
  
[freee開業](https://k.secure.freee.co.jp/personal/users/register)と[freee会計](https://secure.freee.co.jp/users/sign_up)を使って、効率良く届出を作成しましょう。

### カフェを開業する考えは甘い？

**事業計画や資金計画、出店場所のリサーチが不足していればカフェを存続させることは難しく、簡単に開業できるほど甘くはないでしょう。**
  
  
カフェの開業は事業である以上、しっかりと事業を成功させる強い意志を持つこと、計画的に事業運営をすることが大切です。

**事業計画をしっかりと練って収支のシミュレーションを綿密に練ること、市場調査とロケーションの選定を行うことが挙げられます。**
  
  
また、効率よく集客するためにSNSを運用し、情報発信をすることも欠かせません。
