## 年末調整とは何か？

年末調整とは、1年間の所得税を正しく算出するための手続きです。
  
  
会社員などの給与所得者は、毎月の給与から概算の所得税が差し引かれています。これを「源泉徴収」といい、会社が納税者本人に代わって所得税を納める仕組みです。
  
  
しかし、実際の所得税の額は、扶養控除・配偶者控除・生命保険料控除などの各種控除を反映して計算する必要があります。そのため、**年末調整により、会社が従業員に代わって納めた所得税に過不足がないかを確認・調整します。**

## 年末調整が必要な人とは

**年末時点で、会社に勤めて給与を受け取っている人（会社員）は、基本的に年末調整の対象となります。**
  
  
一方、個人事業主は会社に属していないため、自分で所得・税額を算出して納める「確定申告」を行うことが一般的です。

### 年末調整と確定申告の違い

年末調整と確定申告の主な違いは、「誰が手続きを行うのか」という点です。

|  |  |
| --- | --- |
| 年末調整 | 会社が、従業員に代わって所得税の過不足を精算する |
| 確定申告 | 納税者本人が、1年間の所得と税額を計算して申告・納税する |

**個人事業主の場合、会社などに勤めて給与を受け取っていなければ、原則として年末調整の対象にはなりません。**
  
  
たとえば、個人事業とアルバイトを掛け持ちしていて、主たる勤務先であるアルバイト先に「扶養控除等申告書」を提出していれば、その勤務先で年末調整の対象になることがあります。
  
  
また、**従業員や青色事業専従者に給与を支払っている個人事業主は、年末調整を実施する立場になる場合もあります。**
  
  
働き方や立場によって、年末調整や確定申告の要否は異なります。申告漏れや納付忘れを防ぐためにも、まずは自分の収入区分や雇用形態を把握することが重要です。

## 個人事業主でも年末調整が必要なケース

年末調整は、原則として会社員などの給与所得者が対象となる手続きです。しかし、個人事業主であっても、働き方や事業の形態によっては年末調整が必要なケースがあります。
  
  
以下では、その代表的なケースを詳しく解説します。

### 個人事業主がバイトなどで働く場合

事業のほかに、パート・アルバイトなどで給与を受け取っている個人事業主は、年末調整の対象となるケースがあります。

個人事業主でも年末調整を受ける例

* 日中はフリーのWebデザイナーとして働き、夜は飲食店でアルバイトをしている
* ハンドメイド作家として事業を行いながら、週3日はパートタイムで勤務している
* 家事代行の事業を始めたばかりなので、収入を補うためにアルバイトをしている

原則として、**アルバイトで得た給与所得は年末調整の対象となり、個人事業による事業所得は確定申告によって所得と税額を確定します。**
  
  
個人事業の売上から経費を差し引いた金額が年間20万円以下であれば、確定申告を省略することも可能です。その場合、アルバイト・パート先で年末調整を受けることで、その年の所得税の手続きが完了します。ただし、住民税については別途、申告が必要です。
  
  
なお、青色申告で赤字を繰り越したい場合や、65万円控除を適用したい場合は、20万円以下でも確定申告を行うことをおすすめします。

### 個人事業主が従業員へ給与を支払っている場合

アルバイト・パートなどの従業員に給与を支払っている個人事業主は、年末調整を実施する立場になります。
  
  
**人を雇って給与を支払う場合、法人・個人を問わず源泉徴収を行う義務があります。加えて、年末調整の対象となる従業員については、年末調整を実施する必要があります。**
  
  
また、青色申告を行う個人事業主で、**配偶者や親族を「青色事業専従者」として雇用している場合も年末調整の対象**です。青色事業専従者とは、青色申告者と生計を一にする配偶者やそのほかの親族のうち、一定の要件を満たして給与の支払いを受けている人を指します。
  
  
なお、年末調整を怠って源泉徴収税額の納付が遅れた場合、個人事業主（雇用主）側に延滞税や不納付加算税などのペナルティが課される可能性があります。
  
  
こうしたトラブルを防ぐためにも、年末調整の時期と手続きを正確に把握し、計画的に対応することが重要です。

## 個人事業主の年末調整のやり方

従業員や青色事業専従者へ給与を支払っている個人事業主は、対象となる従業員について、雇用主として年末調整を行う必要があります。
  
  
年末調整では、まず従業員から必要書類を提出してもらい、正確な税額を算出します。その結果をもとに源泉徴収票の交付や源泉所得税の納付、給与支払報告書の提出など、所定の手続きを行います。
  
  
スムーズに手続きを進めるために、年末調整に必要な書類や手順をあらかじめ理解しておきましょう。

### 年末調整の必要書類

従業員それぞれの1年間に納めるべき所得税・復興特別所得税の金額を計算するには、以下の書類が必要です。

| 書類名 | 概要・目的 |
| --- | --- |
| ①扶養控除等（異動）申告書 | ・扶養親族の有無や異動状況を申告する書類  ・従業員の源泉徴収税額を適正に計算するため、原則として全員が提出する |
| ②基礎控除申告書 | ・本人の所得金額や他の控除との重複有無を申告する書類  ・基礎控除額を正しく適用するために必要 |
| ③配偶者控除等申告書 | ・配偶者の有無や所得金額などを申告する書類  ・配偶者控除・配偶者特別控除の適用可否を判断する |
| ④特定親族特別控除申告書 | ・19歳以上23歳未満の特定扶養親族に関する情報を申告する書類  ・特定扶養控除の適用対象を判断する |
| ⑤所得金額調整控除申告書 | ・所得金額が一定以上で扶養親族が多い場合や、障害者を扶養している場合などに提出する書類  ・追加控除の適用可否を判断する |
| ⑥保険料控除申告書 | ・生命保険料、地震保険料、社会保険料などの支払額を申告する書類  ・保険料控除額を算出するために必要 |
| ⑦住宅借入金等特別控除申告書 | ・住宅ローン控除の適用対象者が、年末調整で控除を受けるために提出する書類  ・2年目以降の適用時に使用し、税務署から本人へ送付される |

[出典：国税庁「各種申告書・記載例(扶養控除等申告書など)」](https://www.nta.go.jp/users/gensen/nencho/shinkokusyo/index.htm)

また、年の途中で雇用した従業員がいる場合は、前の職場から交付された「源泉徴収票」も提出してもらう必要があります。
  
  
上記①〜⑥の書類は、以下のいずれかの方法で書式を用意し、従業員に記入・提出してもらいます。

年末調整に関する書式の入手方法

* 国税庁の公式サイトからダウンロードする
* 管轄税務署の窓口で入手する
* 書類一式を郵送で受け取る

⑦の住宅ローンに関する申告書は、税務署から従業員本人の自宅に送付されるため、事業主側で書式を用意する必要はありません。
  
  
なお、**従業員がデータで取得・作成した控除証明書や申告書を、オンラインで受け取ることも可能です。**

### 年末調整の手続き方法

年末調整は、主に以下の4つの手順で進めます。

年末調整の4つのステップ

1. 従業員から年末調整に必要な書類を受理する
2. 正確な所得税額を計算し、過不足分を精算する
3. 過納額は還付し、不足額は徴収・納付する
4. 税務署などに源泉徴収票・法定調書合計表・給与支払報告書を提出する

あらかじめ全体の流れを把握しておくことで、手続きの漏れや遅れを防ぐことができます。年末調整を紙で行う場合は導入が容易ですが、作成・郵送の手間がかかります。一方、電子で行う場合は事前準備が必要なものの、提出や管理を効率化できます。
  
  
また、源泉徴収票や法定調書の提出方法は以下のとおりです。

源泉徴収票や法定調書の提出方法

* e-Tax
* 光ディスク等（CD・DVDなど）
* 書面

事業の規模や従業員の数に応じて、適切な提出方法を選択してください。
  
  

[出典：国税庁「年末調整のしかた」](https://www.nta.go.jp/publication/pamph/gensen/nencho2025/pdf/104.pdf)

## 個人事業主は年末調整をいつ行う？

年末調整を行う際に慌てて準備を始めると、書類の不備などが生じやすくなります。余裕をもって早めに取り組みましょう。
  
  
以下のように、**毎年10月から翌年1月にかけて、手続きを進めるのが一般的です。**

|  |  |
| --- | --- |
| 10～11月 | ・従業員へ各種申告書を配布し、期日までの提出を依頼する  ・各種申告書を受理後に、内容に誤り・不備がないかを確認する  ・提出された申告書の再確認や、不備がある場合の修正依頼を行う |
| 12月 | ・従業員ごとに「1年間に納めるべき所得税および復興特別所得税の額(年調年税額)」を計算する  ・従業員ごとに「1年間に源泉徴収した所得税および復興特別所得税の合計額」と年調年税額を比べて過不足額を精算する |
| 1月 | ・過納額がある場合、還付手続きを行う ・不足額がある場合、期限までに徴収・納付を行う  ・従業員へ「源泉徴収票」を交付する  ・徴収義務者の所轄税務署へ「源泉徴収票・法定調書合計表」を提出する  ・従業員の住所地の市区町村へ「給与支払報告書・総括表」を提出する |

[出典：国税庁「令和7年分年末調整についてのお知らせ」](https://www.nta.go.jp/users/gensen/nencho/pdf/02.pdf#page=2)

従業員から必要書類を受理する際、記入漏れや誤りがあると、所得税額を正確に計算できません。**必要書類の回収は余裕をもって開始し、内容の確認に十分な時間を確保しておきましょう。**
  
  

[出典：国税庁「令和7年分年末調整についてのお知らせ」](https://www.nta.go.jp/users/gensen/nencho/pdf/02.pdf#page=2)

## まとめ

年末調整とは、原則として給与所得者が対象となる手続きです。
  
  
しかし、個人事業主であっても、アルバイトやパートによる収入がある場合は、年末調整の対象となることがあります。年末調整が必要かどうかを確認するには、まず収入の区分や雇用形態を正確に把握することが重要です。
  
  
また、従業員や青色事業専従者へ給与を支払っている個人事業主は、年末調整を行う立場となります。その場合、10月から翌年1月末にかけて、計画的に手続きを進める必要があります。

## よくある質問

### 個人事業主は扶養控除申告書を提出する必要はある？

扶養控除申告書とは、会社員・アルバイト・パートなどの給与所得者に提出が義務付けられている書類です。会社などに勤めていない個人事業主であれば、原則として提出する必要はありません。
  
  
ただし、個人事業のほかにアルバイトを掛け持ちしている場合などは、提出が必要です。
  
  
詳しくは、記事内「[個人事業主の年末調整のやり方](#content4)」をご覧ください。

### 個人事業主が年末調整をしないとどうなる？

従業員や青色事業専従者へ給与を支払っている場合、個人事業主であっても年末調整を行う必要があります。年末調整が漏れて、納めるべき所得税の支払いが遅れる場合、延滞税などを徴収されます。
  
  
詳しくは、記事内「[個人事業主でも年末調整が必要なケース](#content3)」をご覧ください。
