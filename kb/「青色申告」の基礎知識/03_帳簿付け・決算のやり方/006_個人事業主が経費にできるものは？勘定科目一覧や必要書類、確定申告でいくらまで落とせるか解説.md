## 個人事業主の確定申告経費とは

**経費とは、事業を行ううえで発生した費用です。**事業所得・不動産所得・雑所得の計算においては、売上原価または販売費・一般管理費など業務上必要とされるものが経費として認められます。
  
  
国税庁では、経費について以下のように定義しています。
  
  
事業所得、不動産所得および雑所得の金額を計算するうえで、必要経費に算入できる金額は、次の金額です。

経費の定義

* 総収入金額に対応する売上原価その他その総収入金額を得るために直接要した費用の額
* その年に生じた販売費、一般管理費その他業務上の費用の額

**経費計上の一般的な判断基準は、正当な事業関連性があるかどうかです。事業に関わる費用であると客観的に証明できることが、経費計上できるかどうかの判断基準のひとつとなります。**
  
  
たとえば、飲食代（接待交際費）であれば「取引先または見込み客の誰と」「どのような必要性があって飲食したのか」を明確にできなければなりません。
  
  
場合によっては、当該出費を経費とする理由や必要性を、確実に記載し取りまとめておくことが必要です。
  
  
発行された領収書を保管する際に、「打ち合わせ」「営業」といった支出の理由を書き添えておくことで、経費計上がしやすくなります。

## 個人事業主の経費の上限はない

**個人事業主が経費として申告できる金額には、上限がありません。**
  
  
事業運営の規模や内容によって必要となる経費の金額は千差万別なので、事業に関わりのある出費であると認められれば経費計上は可能です。

### 業種によって妥当な経費の割合は変わる

**経費計上できる金額に上限はありませんが、売上に対して経費の割合が高すぎると税務署に不正を疑われ、税務調査の対象となる可能性があります。**
  
  
妥当な経費率は業種によって異なります。簡易課税のみなし仕入率が参考になるでしょう。簡易課税制度における業種ごとのみなし仕入率は、以下の通りです。

  

| 事業区分 | 該当する事業 | みなし仕入率 |
| --- | --- | --- |
| 第1種事業 | ・卸売業 | 90% |
| 第2種事業 | ・小売業（他の者から購入した商品をその性質、形状を変更しないで販売する事業で第1種事業以外のもの） ・農業 ・林業 ・漁業（飲食料品の譲渡に係る事業） | 80% |
| 第3種事業 | ・農業 ・林業 ・漁業（飲食料品の譲渡に係る事業を除く） ・鉱業 ・建設業 ・製造業（製造小売業を含む） ・電気業 ・ガス業 ・熱供給業および水道業 ※第1種事業、第2種事業に該当するものおよび加工賃その他これに類する料金を対価とする役務の提供を除く | 70% |
| 第4種事業 | ・第1種事業・第2種事業・第3種事業・第5種事業および第6種事業以外の事業をいい、具体的には、飲食店業など ※第3種事業から除かれる加工賃その他これに類する料金を対価とする役務の提供を行う事業も第4種事業となる | 60% |
| 第5種事業 | ・運輸通信業 ・金融・保険業 ・サービス業（飲食店業に該当する事業を除く） ※第1種事業から第3種事業までの事業に該当する事業を除く | 50% |
| 第6種事業 | 不動産業 | 40% |

[出典：国税庁「No.6509 簡易課税制度の事業区分」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shohi/6509.htm)
  
  

手がけている事業の種類に応じて、適切な経費率の目安を把握しておきましょう。

## 個人事業主が経費にできるもの一覧【勘定科目別】

**個人事業主が経費として計上できる主な項目を、勘定科目別に一覧で解説します。**事業との関連性を客観的に説明できることが、経費として認められるための基本的な考え方です。そのほか、消耗品費などは「使用可能期間が1年未満」かどうかも判断基準となります。

  

| 勘定科目 | 具体例 | 概要 |
| --- | --- | --- |
| 租税公課 | 個人事業税、固定資産税、自動車税、不動産取得税、登録免許税、印紙税。 | 国や地方自治体に納める税金（租税）と、商工会など公共団体へ納める会費（公課）のこと。 |
| 荷造運賃 | 宅配便の送料、梱包用の段ボールやガムテープ、緩衝材の購入費。 | 商品の発送にかかる運送料や、梱包に使用する資材の費用のこと。 |
| 水道光熱費 | 水道料金、電気料金、ガス料金、灯油代。 | 事業所の運営で発生する水道、電気、ガスなどのインフラ費用のこと。 |
| 旅費交通費 | 電車代、バス代、タクシー代、航空券代、出張時の宿泊費、ガソリン代、駐車場代。 | 事業の遂行に必要な移動や、出張に伴う宿泊にかかる費用のこと。 |
| 通信費 | 固定電話・携帯電話の料金、インターネット回線の使用料、切手代、はがき代、サーバー代。 | 事業で利用する電話やインターネット、郵便などにかかる費用のこと。 |
| 広告宣伝費 | Web広告費、チラシやパンフレットの作成費、看板制作費、新聞・雑誌への広告掲載料、名刺作成費。 | 不特定多数の人を対象とした、商品やサービスの宣伝にかかる費用のこと。 |
| 接待交際費 | 取引先との飲食代、贈答品代（お中元・お歳暮など）、慶弔見舞金（ご祝儀・香典など）。 | 事業に関係のある取引先などを接待するための費用のこと。 |
| 損害保険料 | 事務所の火災保険料、事業用自動車の任意保険料や自賠責保険料。 | 事業用資産にかける火災保険や自動車保険などの保険料のこと。 |
| 修繕費 | 店舗や機械、自動車の修理代、破損した備品の修理費用。 | 事業用資産の原状回復や、通常の維持管理のためにかかる費用のこと。 |
| 消耗品費 | 文房具、事務用品、作業用デスク、10万円未満のパソコンやカメラ。 | 使用可能期間が1年未満、または取得価額が10万円未満の物品の購入費用のこと。 |
| 減価償却費 | 建物、自動車、10万円以上のパソコンや機械装置などの購入費用。 | 取得価額が10万円以上の固定資産について、法定耐用年数に応じて分割して計上する費用のこと。 |
| 福利厚生費 | 従業員の健康診断費、社員旅行費、慶弔見舞金、忘年会・新年会の費用。 | 従業員の労働環境改善や慰安のために支出する費用のこと。 |
| 給料賃金 | 従業員への給料、残業代、賞与（ボーナス）。 | 従業員に支払う給与や賞与のこと。 |
| 外注工賃 （外注費） | ホームページのデザインや制作を外部に依頼した費用、専門家への業務委託費。 | 外部の法人や個人に業務を委託した際に支払う費用のこと。 |
| 支払利息 （利子割引料） | 運転資金の借入金の利息、分割払いの手数料。 | 事業用の借入金にかかる利息や、手形の割引料のこと。 |
| 地代家賃 | 事務所や店舗の家賃、月極駐車場代、レンタルオフィス代。 | 事業で使用する土地や建物の賃借料のこと。 |
| 貸倒損失 | 回収できなくなった売掛金、受取手形、貸付金。 | 取引先の倒産などにより、売掛金や貸付金が回収不能になった場合の損失額のこと。 |
| 雑費 | クリーニング代、ごみ処理費用、銀行の振込手数料、各種会費（商工会などを除く）。 | 他のどの勘定科目にも当てはまらない、少額で重要性の低い費用のこと。 |

  

**なお、勘定科目の種類や名称には明確な法律や決まりがないため、事業内容によって上述した勘定科目以外の項目を自由に追加することが可能です。**
  
  
たとえば、営業などで車を頻繁に使用する場合は、ガソリン代・軽油代・駐車場代・車検費用・修理代・自動車保険料（任意保険・自賠責保険）・自動車税など、事業用車両に関連する費用を独立した勘定科目「車両費」としてまとめることができます。
  
  
また、飲食店などで取引先や仕事仲間と打ち合わせをする機会が多い場合は「会議費」や「接待交際費」として計上する方法もあります。
  
  
決算書にある勘定科目だけで記帳すると特定の科目の金額が多くなり、具体的な用途がわからなくなるかもしれません。勘定科目を追加することで、特定の科目の金額が多くなることを防止できます。
  
  
それ以外に、取引内容を明確にして事業にかかる費用を管理しやすくなる点もメリットです。ただし、「記帳は可能な限り同じ勘定科目で行わなければならない」という決まりがあることは把握しておきましょう。

## 個人事業主が経費にできないもの一覧

**経費にできないものに明確な法的ルールはないため、常識の範囲内で判断しましょう。**以下では、経費として計上できない出費の具体例を一覧で解説します。

### 自身の生活や健康管理のための出費

**個人事業主本人の給与や年金、各種保険料などは経費として計上できません。**個人事業主には「福利厚生」という概念がなく、事業主自身の健康診断・人間ドックなどにかかった費用は、経費として計上できないことになっています。
  
  
**また、事業主本人が通うトレーニングジムやヨガなどのレッスン料金も原則計上できません。**ただし、事業主として従業員を雇っている場合、その従業員の給与や健康診断にかかった費用は経費計上できます。

### 個人として納める税金

**所得税や住民税、国民健康保険料、国民年金保険料などは経費にできません。**
  
  
これらは事業の利益から支払うものですが、経費として差し引くことは認められていません。**ただし、個人事業税や固定資産税（事業使用分）、自動車税（事業使用分）など、事業運営に直接関わる一部の税金は経費として計上できます（これらを「租税公課」と呼びます）。**   
  
なぜなら、所得税や住民税は、事業で得た利益（所得）が確定した後に個人として課される税金だからです。

### 私的な買い物や飲食費

**事業とは関係のない、個人的な趣味のための買い物や、家族や友人との食事代は経費になりません。**もし事業用のクレジットカードで支払ってしまった場合は、帳簿上で「事業主貸」として処理する必要があります。**事業との関連性を明確に説明できないものは、税務調査で指摘される可能性があるため注意が必要です。**なぜなら、売上を生み出すための直接的な関連性を客観的に説明することが困難だからです。

### 家族への給料

**生計を同一にする配偶者や親族に支払った給料は、原則として経費にできません。**これは、第三者への給与とは異なり、恣意的に金額を設定できてしまう可能性があるためです。
  
  
なお、白色申告の場合は、一定の条件を満たしていても家族に支払った給与を必要経費として計上することはできません。白色申告では「事業専従者控除」として、定められた金額を所得から差し引く制度のみが認められています。
  
  
**ただし、一定の要件を満たして「青色事業専従者給与に関する届出書」を提出すれば、青色申告者に限り、その給与を経費として計上することが特例で認められています。**
  
  
なぜなら、客観的な労働の対価というより、生計を共にする家族内での資金移動と見なされるためです。この例外的な制度が「青色事業専従者給与」です。

### 資産として減価償却できる物品

**購入金額が10万円以上のパソコンや機械、車両などは、購入した年に全額を経費にすることはできません。**
  
  
これらは固定資産として計上し、法律で定められた耐用年数に応じて分割して経費に計上する「減価償却」という手続きが必要になります。ただし、青色申告者の場合は「少額減価償却資産の特例」により、30万円未満の資産であれば購入した年に全額を経費にできる場合があります。
  
  
なぜなら、複数年にわたって事業に貢献する資産は、その価値の減少分（減価償却費）を、貢献する期間に応じて費用計上するのが会計の基本的な考え方だからです。

### 借入金の返済

**金融機関などから事業資金として借り入れたお金の返済額のうち、「元本」にあたる部分は経費にはなりません。**
  
  
これは、単に借りたお金を返しているだけであり、事業の売上を上げるために何かを使った対価ではないためです。経費として計上できるのは、返済額に含まれる「利息」の部分のみです。
  
  
**なぜなら、元本の返済は借りたお金を返す行為であり、費用ではないからです。**ただし、借入に伴う「利息」は資金調達のためのコストとして経費になります。

## 経費計上が不適切だった場合に生じるリスク

**経費計上は、税金の計算を適切に行うために必要不可欠なものです。**
  
  
売上に対する経費計上の割合が高すぎるなど、経費計上が不適切であると税務署に判断されると、脱税の疑いをかけられる可能性があります。
  
  
経費を多く計上できればそれだけ節税につながりますが、あくまでも「妥当」だと判断される範囲で計上してください。

## 確定申告の経費計上に必要な証拠書類

**事業を行ううえで発生した費用を経費として計上するためには、それらが経費に該当することを明確に裏付ける証拠書類が必要です。**
  
  
帳簿から明確に経費であることを証明できないものや、取引に必要な支出であることを証明できないものは、経費として申告することができません。そのため、日頃から領収書やレシートは忘れずに受け取り、保存しておきましょう。

### 証拠書類に必要な項目

**証拠書類として代表的なものは、取引先や物品を購入したお店が発行する領収書やレシートが挙げられます。**法律上、領収書やレシートに決まった形式はありませんが、一般的な証拠書類として、下記の項目が記載されている必要があります。

経費の証拠書類として必要な項目

* 支払った人の名前や会社名（宛名）
* 支払った金額
* 但し書き（具体的な支出内容）
* 支払いを受けた人の名前や会社名、所在地
* 支払った日付

**領収書やレシートを受け取る場合は、これらの項目が記載されているかどうか、確認する習慣を身に付けましょう。**

### 証拠書類の代わりとなるもの

**領収書を紛失した場合や、電車代・慶弔費のように領収書が発行されない場合は、支払を客観的に証明できる書類を証拠書類として使えることがあります。**

経費の証拠書類として認められる書類例

* 納品書
* 出金伝票
* クレジットカードの利用伝票
* ATMの振込明細書や通帳の記録
* 交通系ICカードの利用履歴
* インターネット通販の購入確認メールのプリントアウト
* パーティーや冠婚葬祭の案内状
* 祝儀袋や不祝儀袋の表書きのコピー

たとえば、クレジットカード利用時の証拠書類は利用伝票です。ATMで代金を振り込んだ際に発行された振込明細書も、請求書や納品書とセットにしておくことで証拠書類となります。
  
  
**また、交通系ICカードを利用している場合は利用履歴を印刷すれば、交通費として計上することが可能です。**クレジットカードや交通系ICカードは、あらかじめ仕事用とプライベート用を分けておくと、書類を整理する手間の削減につながります。
  
  
領収書やレシートは、日頃から少額でも忘れずに受け取る習慣を身につけることが重要です。領収書やレシート以外の証拠書類は、そもそもどのようなものが証拠書類として使えるのか、きちんと把握して保管しておきましょう。
  
  
そして、月別にファイリングするなどわかりやすく管理すると、必要なときにスムーズに見つけられます。

### 電子取引ではデータ保存が必要

**2021年から、「改正電子帳簿保存法」が新たに施行されました。**請求書や領収書などを電子データでやり取りした場合は、その電子データを保存しなければなりません。
  
  
たとえば、電子メールに添付された請求書や、通販サイトからダウンロードした領収書などが対象です。
  
  
2024年1月1日からは完全義務化されたため対応が必須になりました。
  
  
**一方、紙でやり取りしたものをデータ化する必要はなく、紙のまま保存して問題ありません。**

## 経費計上の幅が広がる「家事按分」とは

**個人事業主で自宅を仕事場としている場合、家賃や光熱費の一部は業務の必要経費として計上できます。**しかし、プライベートと業務で使用した費用を明確に分類するのは困難です。
  
  
そこで、経費全体のうち業務として使用している分を一定の比率で区分して経費計上できる「家事按分（かじあんぶん）」を活用します。
  
  
**家事按分に法的なルールはないため、税務署から説明を求められたときのために、使用時間や床面積など、第三者にも説明できる合理的かつ客観的な数値に基づいて、按分比率の根拠を示せるようにしておくことが大切です。**

### 家事按分できる費用とその計算例

家事按分が認められるのは、事業で使用していると合理的に説明できる場合のみであり、具体的には以下のような費用が該当します。

家事按分できる費用の例

* 家賃
* 水道光熱費
* インターネット使用料
* 携帯電話代
* 自動車のガソリン代
* 車検費用など

たとえば家賃を按分するには、「リビングの一部をパーティションで区切って仕事用に割り当てている」など、明確な事実が必要です。
  
  
もしくは「専用スペースを設けていないものの、自宅での時間のうち6時間は業務をしている」といった時間による按分も可能です。
  
  
家賃が月20万円で賃貸面積60㎡のマンションを例に、使用面積と業務時間それぞれで家事按分の計算方法を紹介します。

使用面積で家事按分した場合の計算例

例：仕事場として使用している面積が15㎡の場合
  
事業で使用した面積の割合を求める
  
15㎡ ÷ 60㎡ × 100 ＝ 25 ％
  
家賃のうち事業に使用している分を算出する
  
200,000（円） × 0.25 ＝ 50,000（円） <1ヶ月あたり>
  
  
業務時間で家事按分した場合の計算例
  
例：1ヶ月（30日）のうち、業務時間が1日9時間で10日間業務をする場合
  
業務用の按分割合を求める（業務時間 ÷ 1ヶ月の総時間）
  
90時間 ÷ 720時間 × 100 ＝ 12.5 ％
  
家賃のうち事業に使用した分を算出する
  
200,000（円） × 0.125 ＝ 25,000（円） <1ヶ月あたり>

なお、賃貸だけではなく、持ち家の場合にも家事按分が可能です。対象となるものは、建物部分の減価償却費・住宅ローンの利息・固定資産税・火災保険料などがあります。
  
  
住宅の減価償却費や住宅ローンについては、事業用の家事按分が2分の1を超えると住宅ローン控除の適用対象外になってしまうので注意しましょう。
  
  
**家事按分や経費計上の仕方と計算方法について詳しく知りたい方は、下記の記事も参考にしてください。**

## 経費の不正計上によるペナルティ

**税務署から帳簿の提出を求められた際、税額を過少申告していた場合や意図的に経費や所得を隠ぺいした場合は、ペナルティを課せられる場合があります。**

  

| ペナルティ | 要件 | ペナルティの内容 |
| --- | --- | --- |
| （調査後の） 無申告加算税 | 期限内に申告しなかった場合など | 15〜30%の加算税 |
| 重加算税 | 虚偽の申告をした場合 | 35〜40%の加算税 |
| 財産差押 | 滞納 | 売却、譲渡の禁止 |
| 刑事罰 | 悪質な脱税 | 懲役または罰金 |

  

ペナルティで受ける「加算税」は損金として算入できず、会計処理上の扱いは「租税公課」です。加算税には、過少申告加算税と重加算税の2つがあります。

### 過少申告加算税

**本来支払うべき税額より少ない額を申告した場合に、ペナルティとして加算されるのが「過少申告加算税」です。**期限内に確定申告をして税金を支払っていても、申告内容に問題があれば対象となります。
  
  
自主的に修正申告をすれば、過少申告加算税を課税されずに済むケースもあります。過少申告の場合にも延滞税が課されるので、申告内容に問題点を見つけた際にはできるだけ早く修正申告しましょう。
  
  
過少申告加算税の追徴課税額は、基本的に「増差額×10%」で算出されます。増差額とは、本来納めるはずだった税額から当初申告した税額を差し引いた金額です。
  
  
**ただし、新たに納める税金が、当初の申告納税額と50万円のいずれか大きい金額を超えている場合、超えている部分の税率は15％となります。**

### 重加算税

**架空の経費計上や領収書の偽造など、意図的に税金をごまかそうとしたと判断された場合にペナルティとして加算されるのが「重加算税」です。**
  
  
ほとんどの場合、重加算税が課されるのは税務調査で問題が判明したケースです。違反とみなされると修正申告や更正処分が行われますが、再調査の請求や不服申し立てもできます。
  
  
**しかし、請求や申し立てには期限があり、重加算税のほかに延滞税が加算されるため、指摘を受けた際には素早い対応が必要です。**
  
  
重加算税の税額は、増差額をもとに計算します。過少申告や不納付があった場合には、「過少申告加算税」または「不納付加算税」に代えて35％の税率で課税されます。無申告の場合には、「無申告加算税」に代えて40％の税率での課税です。
  
  
なお、2017年以降には、短期間のうちに隠蔽や仮装を繰り返した場合に、加算税額が10%増える制度が導入されています。この要件に該当すると、過少申告や不納付で45％、無申告で50％という重加算税が課税されることになります。

## よくある質問

### 個人事業主はどこまで経費で落とせますか？

**個人事業主の経費には法律上の上限額はなく、事業の売上を得るために直接必要な支出であれば、原則として経費として計上できます。**
  
  
例えば、事務所の家賃や水道光熱費、取引先との打ち合わせ費用（接待交際費）、移動のための交通費などが該当します。ただし、自宅兼事務所の家賃のようにプライベートと事業で兼用している費用は、事業で使用する割合だけを経費にする「家事按分」という計算が必要です。
  
  
一方で、事業主個人の生活費や健康管理費、所得税や住民税などは経費として認められません。
  
  
詳しくは「[個人事業主が経費にできるもの一覧【勘定科目別】](#content3)」や「[個人事業主が経費にできないもの一覧](#content4)」で解説しています。

### 経費で落とすとお金が戻ってくるの？

**「経費で落とす」とは、支払ったお金がそのまま戻ってくるという意味ではありません。**これは、「支払う税金が安くなる」という意味です。
  
  
**所得税は、「売上から経費を引いた利益」に対して課税されます。**
  
  
例えば、売上が300万円でも、経費が100万円かかっていれば、課税対象となる所得は200万円（300万円 - 100万円）として計算されます。もし経費を計上しなければ、300万円全体が課税対象として扱われてしまいます。
  
  
経費を正しく計上することで、課税対象となる所得が適切に圧縮され、結果として納める税金が適正な額になる（安くなる）のです。
  
  
詳しくは「[個人事業主の確定申告経費とは](#content1)」で解説しています。
