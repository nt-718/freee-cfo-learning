## 貸借対照表とは

**貸借対照表（たいしゃくたいしょうひょう）は、一定時点における事業の財政状態を「資産」「負債」「純資産」に分類して一覧表示する表で、バランスシート（B/S）ともよばれます。**
  
  
企業や個人事業主が保有する現金・在庫・借入金などの状況を把握し、経営の安定性や支払能力を評価するのに役立ちます。
  
  

[出典：国税庁「青色申告者のための貸借対照表作成の手引き」](https://www.nta.go.jp/taxes/shiraberu/shinkoku/tebiki/2018/pdf/025.pdf)
  
  
貸借対照表についての詳細は別記事「[貸借対照表とは？財務状況を分析するための見方やポイントを解説](https://www.freee.co.jp/kb/kb-accounting/balance-sheet/)」をあわせてご確認ください。

### 個人事業主に貸借対照表が必要な理由

**個人事業主が貸借対照表を作成する目的は、事業における資産と負債の現状を可視化し、経営判断に役立てるためです。**例えば、総資産のうち返済不要な自己資金の割合を示す「自己資本比率」などを確認することで、事業の安全性や支払い能力を客観的に分析できます。
  
  
加えて、青色申告で最大65万円の特別控除を適用するには、正規の簿記に基づき記帳し、確定申告書とともに貸借対照表と[損益計算書（P/L）](https://www.freee.co.jp/kb/kb-accounting/pl/)を提出する必要があります。
  
  
損益計算書（P/L）は、企業の一定期間における売上や費用、最終的な利益などの数値をまとめた財務諸表のひとつで、どの程度の利益を上げたかを把握するための資料です。

貸借対照表を必要とする主なケース

* 青色申告で65万円の控除を受けたい
* 資金繰りや将来計画を精緻に立てたい
* 金融機関への説明資料が求められる

財務管理の基礎として活用できるため、控除目的以外でも有用性は高いといえます。
  
  

[出典：国税庁「No.2072 青色申告特別控除」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/2072.htm)
  
  
青色申告特別控除についての詳細は別記事「[【2024年最新】青色申告特別控除で65万円・55万円・10万円の控除額を分ける要件を解説](https://www.freee.co.jp/kb/kb-blue-return/requirement/)」をあわせてご確認ください。

### 青色申告で貸借対照表が必要ないパターン

青色申告を選択していても、すべての個人事業主が貸借対照表を作成する必要があるわけではありません。**特に控除額10万円の特別控除を選んだ場合は、確定申告時に貸借対照表や損益計算書を提出する義務がありません。**
  
  
**また、現金主義を採用している場合も同様に不要です。**現金主義とは、現金や預金の出入りがあった時点で初めて収益や費用を計上する会計方式であり、取引の実態と帳簿上の数字にズレが生じやすい点が特徴です。
  
  
取引発生時点で費用・収益が認識される[発生主義](https://www.freee.co.jp/kb/kb-accounting/accounting-principles/#content2-1)とは異なり、現金主義では取引が完了していても入出金がなければ帳簿に反映されません。
  
  
現金主義や発生主義についての詳細は別記事「[発生主義とは？現金主義・実現主義との違いや適用場面をわかりやすく解説](https://www.freee.co.jp/kb/kb-accounting/accounting-principles/)」をあわせてご確認ください。

貸借対照表が不要なケース

* 10万円の青色申告特別控除を選んでいる
* 現金主義による記帳方法を選択している

ただし、節税効果や管理の観点から、必要のない場面でも作成を検討する価値はあります。なお、提出義務の有無は、控除額や記帳方法により異なることを覚えておきましょう。

## 貸借対照表作成のための事前準備

**貸借対照表を正しく作成するには、複式簿記に基づいた記帳が前提となります。**
  
  
[複式簿記](https://www.freee.co.jp/kb/kb-accounting/importance-of-double-entry-bookkeeping/)とは、取引の内容を「借方」と「貸方」に分けて記録する方式で、資産・負債・純資産の変動を正確に反映できます。一方、単式簿記は家計簿のように収支のみを記録する方法で、詳細な財務状況を把握するには不向きです。
  
  
複式簿記・単式簿記についての詳細は別記事「[青色申告で必要な複式簿記とは？単式簿記との違いや帳簿の書き方をわかりやすく解説！](https://www.freee.co.jp/kb/kb-accounting/importance-of-double-entry-bookkeeping/)」をあわせてご確認ください。
  
  
**青色申告で65万円の特別控除を受けるには、複式簿記による記帳と帳簿の保存が要件の1つであるため、日常的に複式簿記を用いた記録体制を整えておく必要があります。**
  
  
さらに、取引の証拠となる領収書や請求書などの証憑類も用途や日付で整理し、帳簿と突合できる状態を維持することも重要です。準備を怠ると、記帳ミスや控除漏れの原因にもなります。

| 準備項目 | 内容 |
| --- | --- |
| 記帳方法 | 複式簿記（借方・貸方に分けて記録） |
| 必要な書類 | 領収書、請求書、通帳明細など |
| 仕訳のポイント | 勘定科目の選定と金額の正確な入力 |
| 帳簿チェックの頻度 | 月次や四半期ごとの定期的な見直し |

貸借対照表の作成方法には、以下の種類があります。

| 作成方法 | 特徴と手順 | 向いている人 |
| --- | --- | --- |
| エクセルで手入力 | ・勘定科目をセルに割り当てて手作業で入力する  ・表形式の自由度が高い点が特徴 | 自分で細かく調整したい中上級者 |

**いずれも、取引内容をもとに金額を整理する工程が必要です。**作業にかかる手間や柔軟性、ミスの起きにくさには違いがあるため、自身のスキルや目的に応じた方法を選ぶことが重要になります。

## 貸借対照表の書き方・手順

貸借対照表の作成にはいくつかの段階があり、それぞれに異なる帳簿や書類の作成が必要です。

| 手順（工程） | 概要（目的・内容） |
| --- | --- |
| ①仕訳帳と総勘定元帳を作成・記載する | 日々の取引を勘定科目で仕訳し、科目別に記録する |
| ②試算表を作成・記載する | 帳簿の数字が正確に転記されているかを確認する |
| ③貸借対照表を作成・記載する | 決算整理仕訳を反映させ、財務状態を表にまとめる |
| ④青色申告決算書へ記載する | 完成した貸借対照表を法定書類に転記して申告準備を行う |

### ①仕訳帳と総勘定元帳を作成・記載する

貸借対照表を作る際はまず、取引を仕訳帳に記録する作業が求められます。**仕訳とは、各取引を勘定科目に基づいて借方（左側）と貸方（右側）に分類し、日付と金額を含めて記入する処理です。**
  
  
仕訳帳の内容を勘定科目ごとに整理したものが、総勘定元帳です。仕訳帳で記録されたデータを転記し、勘定科目別に集計することで帳簿の構造が整います。
  
  
**正確な仕訳と転記がなければ、以降の試算表や貸借対照表の整合性にも支障をきたします。**帳簿の信頼性を高めるためにも、取引ごとに丁寧な仕訳を心がけましょう。
  
  
仕訳帳と総勘定元帳についての詳細は別記事「[仕訳帳とは？総勘定元帳との違いや書き方の具体例を解説](https://www.freee.co.jp/kb/kb-accounting/journalbook/)」と「[総勘定元帳とは？仕訳帳との違いや書き方・項目をわかりやすく解説](https://www.freee.co.jp/kb/kb-accounting/general-ledger/)」をあわせてご確認ください。

### ②試算表を作成・記載する

次の工程として、仕訳帳と総勘定元帳の整合性を確認する目的で試算表を作成します。**試算表は、各勘定科目ごとに集計した金額を借方と貸方に分類し、帳簿が正確に記帳されているか検証する役割がある書類です。**
  
  
借方合計と貸方合計が一致していれば、基本的な記帳ミスはないと判断できます。特に複式簿記では、二重の記録による整合性チェックが可能となるため、試算表の作成によって数値の信頼性が向上します。
  
  
試算表についての詳細は別記事「[試算表とは？種類や見方、効率的な作り方などについて解説](https://www.freee.co.jp/kb/kb-accounting/what-is-trial-balance/)」をあわせてご確認ください。

### ③貸借対照表を作成・記載する

試算表による検証を終えたら、決算整理仕訳を加えて貸借対照表を作成します。**決算整理とは、減価償却費や棚卸資産など年度末に行う補正処理であり、実態に即した財務状況を反映させるための重要な工程です。**
  
  
試算表の数値に整理仕訳を反映し、資産・負債・純資産の項目ごとに金額を記載します。個人事業主特有の科目として、事業とプライベートのお金を区別する「事業主貸（事業用資金をプライベートで使った）」「事業主借（プライベート資金を事業で使った）」や、法人における資本金に相当する「元入金」の計算が必要です。
  
  
**あわせて損益計算書の作成も必要となるため、両者の整合性を確認しながら帳票を完成させる必要があります。**
  
  
最終的に税務申告にも利用するため、表の構成や数字の整合には十分注意しましょう。
  
  
損益計算書についての詳細は別記事「[損益計算書とは？項目別の見方やチェックポイントを解説](https://www.freee.co.jp/kb/kb-accounting/pl/)」をあわせてご確認ください。

### ④青色申告決算書へ記載する

貸借対照表が完成した後は、青色申告決算書への記載作業が必要です。**青色申告決算書は、確定申告時に税務署へ提出する書類の一部であり、貸借対照表と損益計算書の内容が反映されます。**
  
  
事業の財政状態を正しく申告するため、各項目の数値を正確に転記しなければなりません。**特に資産や負債、期末残高はそのまま課税額や控除額に影響するため、慎重に記載しましょう。**

## まとめ

**貸借対照表は、個人事業主が財務状況を把握し、青色申告特別控除を受けるうえでも重要な書類です。**資産・負債・純資産を可視化することで、資金繰りや経営判断の精度が向上します。
  
  
作成にあたっては、仕訳帳と総勘定元帳を整備し、試算表を作成することが前提です。試算表に決算整理仕訳を加えることで、正確な貸借対照表と損益計算書が完成します。**最終的には青色申告決算書への反映が求められます。**
  
  
確定申告ソフトやテンプレートを利用すれば、作業の手間を大きく削減可能です。制度の要件を正しく理解し、計画的に対応することが重視されます。

### 貸借対照表はどんな書類？

**貸借対照表は、個人事業主や法人の一定時点における財政状態を示す書類です。**資産・負債・純資産の3つの要素に分類し、経営の安全性や支払い能力の判断材料となります。
  
  
詳しくは記事内「[貸借対照表とは](#content1)」をご覧ください。

### 貸借対照表がないと青色申告特別控除は受けられない？

**青色申告特別控除には、控除額ごとに要件が異なります。**
  
  
65万円の控除を受けるためには、複式簿記に基づく記帳とともに、貸借対照表および損益計算書を青色申告決算書に添付し、e-Taxによる確定申告を行う必要があります。一方、10万円の控除を選択する場合や現金主義を採用している場合は、貸借対照表の提出は不要です。
  
  
詳しくは記事内「[青色申告で貸借対照表が必要ないパターン](#content1-2)」をご覧ください。

### 貸借対照表の数字が合わないときはどうすればいい？

**貸借対照表の資産合計と負債・純資産合計が一致しない場合、仕訳や転記のミスが考えられます。**
  
  
まずは仕訳帳や総勘定元帳、試算表に誤りがないかを見直し、決算整理仕訳の漏れや入力ミスを丁寧に確認してください。特に個人事業主の場合、事業用資金をプライベートで使った、あるいはその逆の記帳漏れが原因で現金残高が合わないケースがあります。
  
  
詳しくは記事内「[貸借対照表の書き方・手順](#content4)」をご覧ください。
