# 副業で青色申告できる条件とは？節税効果や注意点を解説

出典: https://www.freee.co.jp/kb/kb-blue-return/notes/

---

副業での年間所得が20万円を超える場合、確定申告が必要です。確定申告には青色申告と白色申告があり、それぞれメリット・デメリットがあります。青色申告は青色申告特別控除など節税効果が高い一方で、申請書の提出や複式簿記での記帳が求められるため、白色申告に比べて申告手続きが複雑です。
  
  
本記事では、副業で確定申告が必要かどうかの判断基準や、副業で青色申告するメリット、注意点について解説します。

## 副業で確定申告しなければならないのはいくらから？

会社員は通常、勤務先が従業員の年末調整を行うことで、その年の給与に関する所得税の計算や納税手続きが完了します。 ただし、**給与所得者が勤務先以外で副業をし収入を得ていた場合は、副業の年間所得額に応じて確定申告の要否が変わります。**
  
  
**副業でアルバイトをしているなど、勤務先から給与を受け取っている給与所得者は、副業で20万円を超える所得を得た場合に確定申告が必要です。**
  
  
申告義務があるにもかかわらず手続きを忘れると、本来納める税額とは別に加算税や延滞税などがかかることがあるので注意しましょう。

### 副業の所得が20万円以下でも確定申告をしたほうがよい例

**副業による年間所得が20万円以下であれば、確定申告の義務はありません。**しかし、以下のようなケースでは、副業による年間所得が20万円以下であっても確定申告したほうがよいといえます。

副業の所得が20万円以下でも確定申告したほうがよいケース

* 住宅ローン控除（最初の年分）や医療費控除などを受けたいとき
* 税金の還付を受けるとき

住宅ローンを利用してマイホームの新築・取得または増改築などを行った場合、一定の条件で住宅ローン控除を受けられます。**住宅ローン控除は、必要書類を勤務先に提出すれば年末調整で処理されますが、最初の年分だけは自分で確定申告を行わなければなりません。**
  
  
また、**医療費の金額が一定の基準を超えた場合に適用できる医療費控除も、年末調整では適用できないため、控除を受けたい場合は確定申告が必要です。**対象となる金額は、所得などの条件により異なります。
  
  
また、**副業による年間所得が20万円以下であっても、確定申告を行うことで税金の還付を受けられる可能性があります。**たとえば、副業先でも源泉徴収が行われているケースで、確定申告により過大な納税が判明した場合は納めすぎた税金が還付されます。
  
  

[出典：国税庁「No.1212 一般住宅の新築等をした場合（住宅借入金等特別控除）」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1212.htm)
  
[出典：「No.1120 医療費を支払ったとき（医療費控除）」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1120.htm)

## 青色申告とは

確定申告の方法には、白色申告と青色申告の2種類があり、それぞれ提出すべき書類の種類が異なります。
  
  
青色申告では、複式簿記によって取引内容を管理し、申告しなければなりません。白色申告に比べて、申告の手続きや帳簿の付け方は複雑ですが、青色申告特別控除を受けられるなどのメリットがあります。
  
  
ただし、副業の規模が小さく事務負担を抑えたい場合は、簡易な白色申告を選ぶことも一つの方法です。

  

|  | 青色申告 | 白色申告 |
| --- | --- | --- |
| 申告できる所得の種類 | 事業所得、不動産所得、山林所得のいずれかの所得 | それ以外 |
| 記帳義務 | あり | あり |
| 記帳方法 | 複式簿記 | 簡易簿記（単式簿記） |
| 事前申請 | 必要 | 不要 |
| 節税効果 | 大きい | 小さい |

  

青色申告は、事業所得、不動産所得、山林所得のいずれかでなければ行えません。副業で得られる収入は雑所得に該当する場合が多く、たとえば、原稿料・講演料・ネットオークションの収入などは、事業規模でない限り雑所得となる傾向があります。**雑所得では青色申告はできません。**
  
  
**事業所得であるかは、社会通念上で事業と言える規模で取引を行っているかどうかで判断されます。**
  
  
所得にかかる取引を記録した帳簿書類が保存されていなければ、概ね雑所得と判断される可能性が高いでしょう。また、帳簿書類が保存されていても、収入金額が少ない場合は、事業と認められるかどうかは個別に判断されることになります。
  
  
なお、**青色申告を行うには、期日までに税務署に「所得税の青色申告承認申請書」を提出しなければなりません。**提出時期は、青色申告書による申告をしようとする年の3月15日までです。また、その年の1月16日以後に開業した場合は、事業開始等の日から2ヶ月以内です。

[出典：「A1-8 所得税の青色申告承認申請手続」](https://www.nta.go.jp/taxes/tetsuzuki/shinsei/annai/shinkoku/annai/09.htm)

## 副業で青色申告するメリット

副業が事業所得として申告が認められる事業規模になれば、青色申告の対象となります。**青色申告は、特別控除を受けられたり、損益通算ができたりといった節税効果が高いというメリットがあります。**

副業で青色申告をするメリット

* 青色申告特別控除を受けられる
* 純損失を繰り越せる
* 仕事で必要な電気代などを経費にできる
* 少額減価償却資産の特例を活用できる
* 貸倒引当金を計上できる

### 青色申告特別控除を受けられる

青色申告は、要件を満たすことで青色申告特別控除が適用されます。青色申告特別控除は10万円、55万円、65万円の3つの区分に分けられ、それぞれの要件は以下のとおりです。

  

| 適用条件 | 青色申告特別控除額 | | |
| --- | --- | --- | --- |
| 65万円 | 55万円 | 10万円 |
| 複式簿記 | ◯ | ◯ | ×  （簡易な記帳） |
| 貸借対照表と損益計算書の提出 | ◯ | ◯ | △  （損益計算書のみ） |
| 期限内の申告 | ◯ | ◯ | − |
| e-Taxでの申告  または電子帳簿保存 | ◯ | − | − |

  

65万円の青色申告特別控除を受けるには、複式簿記での記帳や期日以内に確定申告書を提出するなどの55万円控除を受けるための要件をすべて満たした上で、**e-Taxによる電子申告を行う、または要件を満たす優良な電子帳簿保存を行うことのいずれかが必要です。**

### 純損失を繰り越せる

事業で赤字が出た場合、まずはその年の給与所得などと相殺（損益通算）して所得税を減らすことが可能です。それでも相殺しきれない赤字は、翌年以降3年間繰り越して計上できます。
  
  
たとえば、2023年に10万円の赤字となった場合を考えてみましょう。損失申告を行った場合、2024年の課税対象額が20万円になったとしても、10万円を差し引くことができ、2024年の課税対象額は10万円になります。

### 業務に必要な電気代などを経費にできる

副業のために自宅を事務所や作業場にしている場合、業務で支出した費用を経費として計上できます。経費にできる費用の例は以下のとおりです。

自宅を事務所や作業場にしている場合の経費にできる費用

* 電気代
* インターネット料金
* プリンター
* 交通費　など

課税所得額は、収入から経費を差し引いて求めます。よって、業務で使用した費用を正しく経費計上することで収入から差し引ける経費額が増え、課税所得額が減り、節税につながります。
  
  
ただし、たとえば自宅を副業の事務所にしているなど、プライベートと事業で兼用しているのであれば、家事按分によって事業で使用した分のみを算出・計上する必要があります。

### 少額減価償却資産の特例を活用できる

**少額減価償却資産の特例とは、30万円未満の減価償却資産を取得した際、その費用を一括で経費にできる制度です。**減価償却資産とは、1年以上の使用によって価値が減少する固定資産です。たとえばパソコンやソフトウェア、事務机、キャビネットなどが該当します。
  
  
少額減価償却資産の特例を活用し、購入した年にまとめて経費計上することで、減価償却資産を取得した年の所得を圧縮して節税につなげられます。
  
  

[出典：国税庁「No.5408 中小企業者等の少額減価償却資産の取得価額の損金算入の特例」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/hojin/5408.htm)

### 貸倒引当金を経費計上できる

**貸倒引当金とは、取引先の倒産などにより売掛金などの回収不能リスクを見積もり、事前に計上しておく資金です。**
  
  
貸倒引当金は、不正な利益操作につながらないよう合理的な範囲でのみ計上が認められます。青色申告であれば、売掛金の5.5%を一括評価による貸倒引当金として経費計上できます。
  
  
貸倒引当金は、白色申告でも経費として計上できます。ただし、青色申告と白色申告では評価方法が異なり、白色申告の評価方法（個別評価）では節税効果は望めません。

## 副業で青色申告するときの注意点

青色申告は、最大65万円の青色申告特別控除が受けられることは大きなメリットですが、以下の点に注意し申告をしなければなりません。

副業で青色申告するときの注意点

* 申告できる所得の種類が限られる
* 受けられる控除額ごとに要件が異なる
* 開業届および青色申告承認申請書の提出が必要
* 帳簿書類を保管する必要がある

### 申告できる所得の種類が限られる

所得は税制上10種類に分類されますが、青色申告を利用できる所得は、以下のいずれかに限定されます。

青色申告を利用できる所得

* 事業所得
* 不動産所得
* 山林所得

**副業として得た所得が低かったり一時的な収入であったりすると、青色申告対象外の雑所得として判断されることがあります。**まずは自身の副業収入が事業所得、不動産所得、山林所得のいずれかに該当するかどうかを確認しましょう。
  
  
なお、副業が事業所得と判断されるかは収入規模や継続性など総合的に見られますが、日々の記帳を行い、請求書・領収書・帳簿などを保存して収支を継続的に管理していると、事業としての実態を説明しやすくなり、事業所得として認められる可能性を高められます。

### 受けられる控除額ごとに要件が異なる

青色申告では、以下の要件を満たすことで55万円の青色申告特別控除を受けられます。

55万円の青色申告特別控除の要件

* 不動産所得または事業所得を生ずべき事業を営んでいること
* これらの所得にかかる取引を正規の簿記の原則（一般的には複式簿記）により記帳していること
* 正規の簿記の原則に基づいて作成した貸借対照表および損益計算書を確定申告書に添付し、期限内申告を行うこと

  

[出典：国税庁「No.2072 青色申告特別控除」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/2072.htm)

複式簿記とは、正規の簿記の原則に従い、ひとつの取引に関して貸方・借方の複数の科目で帳簿を付ける方法です。
  
  
さらに、以下の条件も満たす場合は控除額に10万円が加算され、合計65万円の控除を受けることができます。

65万円の青色申告特別控除の要件

* 上記「55万円の青色申告特別控除」の要件に該当していること
* 次のいずれかに該当していること  
  + a.その年分の事業にかかる仕訳帳および総勘定元帳について、電子帳簿保存を行っていること※
  + b.その年分の所得税の確定申告書・貸借対照表および損益計算書等の提出を、確定申告書の提出期限までにe-Tax（国税電子申告・納税システム）を使用して行うこと

  

※2022年分以後の申告であれば、その年分の事業における仕訳帳および総勘定元帳について、優良な電子帳簿の要件を満たした電子データによる備付けおよび保存を行い、一定の事項を記載した届出書を提出していること
  
  

[出典：国税庁「No.2072 青色申告特別控除」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/2072.htm)

ただし、過去に電子帳簿保存の要件を満たして青色申告特別控除（65万円）の適用を受けていた人の場合は例外です。引き続き当該要件を満たしていれば、一定の事項を記載した届出書の提出は必要ありません。
  
  

[出典：国税庁「No.2072 青色申告特別控除」](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/2072.htm)

### 青色申告承認申請書の提出が必要

青色申告するためには、「所得税の青色申告承認申請書」を事前に税務署に提出しなければなりません。
  
  
申請書は、青色申告をしたい年の3月15日までに提出が必要です。ただし、1月16日以降に事業を行った場合は、事業開始から2ヶ月以内の提出で青色申告ができます。
  
  
**なお、開業届は開業から1ヶ月以内の提出が必要で、青色申告承認申請書の期限と近いため、開業時のタイミングで一緒に提出することで、申請漏れを防ぐことができます。**

### 帳簿書類の保管が必須

青色申告を利用するためには、日々の取引を記帳するだけでなく、帳簿書類の保管も必要です。帳簿の種類によって保管期間は異なります。

  

|  |  |
| --- | --- |
| 保存期間7年 | 帳簿（仕訳帳、総勘定元帳、現金出納帳、売掛帳、買掛帳、経費帳、固定資産台帳など） |
| 決算関係書類（損益計算書、貸借対照表、棚卸表など） |
| 現金預金取引等関係書類（領収証、小切手控、預金通帳、借用証など）※前々年分の事業所得及び不動産所得の金額が300万円以下の場合は5年 |
| 保存期間5年 | その他の書類（取引に関して作成し、または受領した上記以外の請求書、見積書、契約書、納品書、送り状など） |

#### 電子帳簿保存法への対応も求められる

また、**青色申告で最大65万円の青色申告特別控除を受けるには、電子帳簿保存法で定められている「優良な電子帳簿の保存」にも対応しなければなりません。**
  
  
電子帳簿保存法とは、国税に関係する帳簿や書類をデータとして保存する際の扱い方を定めた法律です。電子帳簿保存法では、以下3つの電子データの保存区分があります。

電子データの保存区分

* 電子帳簿等保存
* スキャナ保存
* 電子取引データ保存

たとえば会計ソフトウェアで作成した帳簿は電子データのまま保存が可能です。電子帳簿保存およびスキャナ保存は、電子データでの保存を希望する事業者が任意で行います。
  
  
しかし、電子取引でデータをやり取りした場合は注意が必要です。申告所得税に関して帳簿･書類の保存義務が課されている場合、取引のやり取りが電子データであれば、その電子データを必ず保存しなければなりません。
  
  
電子帳簿保存を行い65万円の青色申告特別控除を受ける場合、作成した仕訳帳および総勘定元帳が以下の「優良な電子帳簿の保存条件」を満たしている必要があります。

優良な電子帳簿保存の要件

* 「モニター・説明書等を備え付ける」などの電子帳簿を保存するための要件を満たしている
* 訂正等の履歴が残る
* 帳簿間で相互関連性がある（ほかの帳簿類の金額との関連を照合できる）
* 日付・金額・相手方による検索機能がある

  

[出典：国税庁「e-Tax による申告又は優良な電子帳簿の保存により65 万円の青色申告特別控除を適用しましょう！」](https://www.nta.go.jp/publication/pamph/pdf/0021010-076.pdf)

## まとめ

給与所得のある会社員は、副業での所得が20万円を超えたら確定申告が必要です。青色申告は帳簿付けの手間がかかりますが、白色申告と比較して、控除や損失の繰り越しなど複数のメリットがあります。
  
  
ただし、青色申告をするには、所得が事業所得、不動産所得、山林所得のいずれかであり、副業が事業的規模であると認められなければなりません。
  
  
青色申告は複式簿記での記帳や事前の書類提出が必要であり、白色申告よりも手間がかかる方法ですが、会計ソフトを活用するとスムーズです。副業の青色申告に必要な書類や手続きを把握し、確定申告に向けて準備を進めましょう。

## よくある質問

### 副業で確定申告をしたほうがいい場合とは？

会社員が副業でアルバイトをしているなど、勤務先受け取る給与以外の副業で20万円を超える年間所得を得た場合に確定申告が必要です。
  
  
本業の給与で年末調整が行われ、副業の所得が20万円以下の場合、確定申告を必ず行う必要はありません。ただし、年末調整では適用できない控除を受けたい場合は、確定申告したほうがよいでしょう。
  
  
詳しくは記事内、「[副業で確定申告しなければならないのはいくらから？](#content1)」をご覧ください。

### 副業で青色申告する際の注意点は？

青色申告は事業所得、不動産所得、山林所得のみが対象であり、給与所得や株で得た所得（雑所得）などは青色申告の対象となりません。また、最大65万円の青色申告特別控除を受けるには、帳簿の作成や提出などの要件を満たした上で、e-Taxまたは電子帳簿保存法にも対応する必要があります。
  
  
詳しくは記事内、「[副業で青色申告するときの注意点](#content4)」をご覧ください。
