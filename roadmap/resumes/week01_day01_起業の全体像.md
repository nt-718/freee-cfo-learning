# Day 1 レジュメ｜起業の全体像を掴む

> **Week 1 — 起業と法人形態の選択** | Day 1–2: 起業の全体像を掴む

---

## 今日のゴール

「起業とは何か」を正確に定義し、**失敗パターンを先に学ぶ**ことで、設計段階でリスクを排除できる視点を身につける。

---

## 1. 起業とは何か

**起業** = 自ら新しい事業を立ち上げること（広義では個人・法人を問わない）

| 用語 | 意味 |
|---|---|
| 起業 | 新しい事業を立ち上げること |
| 開業 | 現在携わっている事業を新たに始めること |
| 創業 | 事業を始めたそのとき（過去形で使用） |

**ポイント**: 起業のハードルは年々下がっている（手続き簡素化、補助金整備、情報収集のしやすさ）

---

## 2. 起業の5ステップ（全体フロー）

```
Step 1: 起業する目的を定める
    ↓
Step 2: どんな事業で起業するか決める
    ↓
Step 3: 起業形態を決める（株式会社・合同会社・個人事業主…）
    ↓
Step 4: 必要な資金を用意する
    ↓
Step 5: 会社設立手続きを行う
    ↓
    事業開始後の届出・PR活動
```

### Step 1: 目的を定める

- なぜ起業したいのか、ビジョンを立てる
- 「こういう会社をつくりたい」という具体的なイメージが **意思決定の軸** になる
- 将来の経営理念につながる目的を考える

### Step 2: 事業を決める

ビジネスモデルの4要素：

| 要素 | 問い | 例 |
|---|---|---|
| **WHO** | 誰が顧客か | 30〜40代・経営者・男性 |
| **WHAT** | どんな価値を提供するか | 業務効率化が図れる |
| **HOW** | どのように提供するか | サブスクリプション型で販売 |
| **WHY** | なぜ取り組むのか | 自身の経験・ストーリー |

**事業計画書の主な項目**: 起業メンバープロフィール / ビジョン・理念 / 市場・競合環境 / 自社の強み / 販売戦略 / 財務計画

### Step 3: 起業形態を決める

主な形態（詳細はDay 3–4で学習）：

| 形態 | 特徴 | 向いているケース |
|---|---|---|
| 株式会社 | 信用力高い・上場可 | 外部資金調達・大規模事業 |
| 合同会社 | 設立費用安い・手続き簡単 | 小規模・自己資金のみ |
| 個人事業主 | 最も手続きが簡単・無料 | まず始める・副業 |
| NPO法人 | 非営利・寄付を受けやすい | 社会貢献活動 |
| 一般社団法人 | 非営利・法人格あり | 業界団体・教育・研究 |

### Step 4: 資金を用意する

**2022年度 新規開業実態調査（日本政策金融公庫）**より：
- 開業費用の中央値：**550万円**
- 自己資金の平均：**271万円**
- 金融機関からの借入平均：**882万円**

主な調達方法：
1. 自己資金（余剰資金のみ充てること）
2. 共同出資（経営権についての取り決めが必要）
3. 株式の発行（株主への利益還元義務あり）
4. 金融機関からの融資（返済計画を必ず立てる）
5. 補助金・助成金（後払いが基本→一時的な自己負担が必要）
6. クラウドファンディング

### Step 5: 会社設立手続き（株式会社の場合）

```
① 会社の基本事項を決める（商号・事業目的・資本金・役員構成 etc.）
    ↓
② 会社用の印鑑を購入する
    ↓
③ 定款を作成する（絶対的記載事項：目的・商号・所在地・資本金・発起人）
    ↓
④ 定款の認証を受ける（公証役場｜認証手数料3〜5万円）
    ↓
⑤ 資本金払込を行う（発起人の個人口座へ）
    ↓
⑥ 登記申請を行う（法務局｜登録免許税15万円〜）
    ↓
登記完了（申請から約10日）
```

**設立にかかる費用目安：**

| 形態 | 費用 |
|---|---|
| 個人事業主 | 0円 |
| 合同会社 | 約10万円＋資本金 |
| 株式会社 | 約25万円＋資本金 |

---

## 3. 起業後にやるべきこと

登記完了後、速やかに行う手続き：

| 届出先 | 内容 | 期限 |
|---|---|---|
| 税務署 | 法人設立届出書・青色申告承認申請書 など | 設立後2ヶ月以内 |
| 年金事務所 | 健康保険・厚生年金保険新規適用届 | 設立後**5日以内** |
| 労働基準監督署 | 労働保険成立届（従業員を雇った場合） | 雇用翌日から**10日以内** |
| ハローワーク | 雇用保険関係の届出（従業員を雇った場合） | 雇用翌日から**10日以内** |
| 金融機関 | 法人口座の開設 | 速やかに |

## 4. 起業が失敗する6つの理由

> CFO視点：「なぜ失敗するか」を先に学ぶことで、設計段階でリスクを排除できる

| 失敗理由 | CFO的な対策 |
|---|---|
| **業績不振** | 採算性の見込める事業選択・事業計画の定期見直し |
| **資金不足** | 3〜6ヶ月分の運転資金を確保・資金繰り表の作成 |
| **集客・営業力の不足** | 起業前からのPR戦略立案 |
| **人材採用・育成の不足** | 人員計画を事業計画に組み込む |
| **利益の使い道を誤る** | 利益配分ルールを事前に決める |
| **経営層の不仲** | 共同創業者との役割・報酬・持ち分を書面で決める |

---

## 5. 失敗する人の5つの特徴と対策

| 特徴 | 対策 |
|---|---|
| 資金計画が不十分 | 初期費用＋当面の運転資金を正確に把握する |
| 起業自体が目的 | 「なぜ起業するか」を言語化する |
| 経営者としての自覚がない | CFO思考（財務・意思決定・先見性）を意識する |
| 社会の変化に対応できない | 事業計画を柔軟に修正する仕組みを作る |
| 人の意見に耳を傾けない | 顧問・メンター・共同創業者を活用する |

---

## 6. 起業を失敗させないための6つのポイント

1. **副業からスタートする** — リスクを抑えながら問題点を洗い出す
2. **小規模・低リスクで始める** — 失敗時のリカバリーを容易にする
3. **綿密な事業計画を立てる** — 問題発生時に改善の基準ができる
4. **資金管理を確実に行う** — 資金繰り表でキャッシュフローを管理
5. **苦手分野を補う仲間を選ぶ** — 自分の弱みを補完するチームを組む
6. **変化を恐れない** — ケースバイケースで柔軟に判断する

---

## CFO目線の Key Insight

> **起業のStep 3「形態選択」こそが、最初の財務意思決定である。**
>
> 多くの起業家は「とりあえず合同会社で安くはじめよう」と判断するが、CFOとして考えるべきは **将来の資金調達計画・株主構成・IPO可能性** だ。形態変更（合同会社→株式会社）は可能だが、手続きとコストがかかる。スタート時点での選択が後のオプションを左右する。
