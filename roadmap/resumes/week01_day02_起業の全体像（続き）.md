## 1. 起業に成功するための6つの準備項目

> 「起業前にやるべきことは何か」を具体的に整理する

| # | 項目 | CFO的ポイント |
|---|---|---|
| 1 | **実現可能なビジネスモデルの構築** | 単なる理想ではなく採算性・継続性を検証。最小規模でサンプルテストを実施 |
| 2 | **核となる人材の確保** | 自分の弱みを補完する人材を早期に確保。役割・責任を明確にして共有 |
| 3 | **事業計画の策定** | 月次・年次・中期（3〜5年）の計画を立てる。キャッシュフロー計画を必ず含める |
| 4 | **起業に必要な資金の調達** | 必要資金を項目別に積み上げ。運転資金分も含めた調達手段を確保 |
| 5 | **顧客選定と広告宣伝方針の決定** | 起業前からターゲット層・PR手段を決める。費用対効果を考慮してSNS戦略を立案 |
| 6 | **法的手続き** | 会社法・税法の基本を自分で理解する。専門家任せでなく経営者自身が内容を把握 |

**特に重要**: 事業計画は「作って終わり」ではなく、**月次・四半期で実績と比較し、柔軟に修正**する仕組みが必要。

## 2. 経営者とは何か

### 定義と役割

**経営者** = 企業の経営方針や経営計画を立案し、意思決定を行って最終的な責任を負う人

経営者の主な役割：

| 役割 | 内容 |
|---|---|
| **経営方針の決定** | 市場動向・自社の強みを踏まえ、中長期的な目標を設定・浸透させる |
| **事業の推進・実行管理** | KPIをモニタリングし、計画とのずれを是正する |
| **資金繰り・調達・投資判断** | キャッシュフローを把握し、健全な財務基盤を維持する |
| **人材の採用・育成・評価** | 人員配置の方向性を示し、安心して働ける環境を整える |

### 4大経営資源

```
ヒト  → 組織・人事・リーダーシップ（最も重要な資源）
モノ  → 製品・設備など物的資源の有効活用
カネ  → 財務・資金調達・投資判断（CFOの核心領域）
情報  → 顧客データ・ノウハウ・市場動向の収集と活用
```

### 経営者に求められる7つの資質

（日本能率協会「トップマネジメント意識調査2022」より）

| 資質 | 説明 |
|---|---|
| **本質を見抜く力** | 情報過多の時代に論理的思考・洞察力で核心を捉える |
| **変化への柔軟性** | 環境変化に合わせてビジネスモデルを転換できる |
| **イノベーションの気概** | 既存の枠を超えた行動を継続するリーダーシップ |
| **決断力・実行力** | 全情報が揃う前にリスクとリターンを判断して動く |
| **リーダーシップ** | 組織を動かし、万が一の際は自ら責任を取る覚悟 |
| **ビジョン・信念** | 判断の軸がぶれない明確な方向性を持つ |
| **忍耐強さ** | 成果が出ない時期を冷静に乗り越える継続力 |

### 経営者になる4つのルート

1. **起業する** — 自ら事業を立ち上げる（最もリスクが高いが自由度も高い）
2. **フランチャイズ契約で独立** — 既存ノウハウを活用してリスクを下げる
3. **組織で昇格する** — 中小企業では評価次第で経営者になれる
4. **事業承継する** — 後継者不足の企業を引き継ぐ（近年増加傾向）

## 3. スモールビジネスとは

### 定義

**スモールビジネス** = 小規模で運営される事業（一般的に従業員5人以下）

個人事業主・フリーランス・小規模法人が含まれる。

### ベンチャー・スタートアップとの比較

| 概念 | 定義 | 成長志向 |
|---|---|---|
| スモールビジネス | 小規模で運営される事業 | 必ずしも前提としない |
| ベンチャー | 革新的な技術・商品を提供する新興企業 | 中程度 |
| スタートアップ | 新市場を開拓し急成長を目指す企業 | 非常に高い |

### メリット・デメリット

**メリット:**
1. 比較的少ない資金で起業できる（初期費用・固定費を抑えられる）
2. 仕事のやりがいを感じやすい（幅広い業務経験、自身の成長が事業に直結）
3. 働き方を自分で設計できる（時間・場所の自由度が高い）

**デメリット:**
1. **社会的信用度が低い** — 融資額の制限・取引断られるリスク
2. **労働集約型になりやすい** — 属人化により一定額以上の拡大が困難
3. **収入が不安定** — 案件の継続性がなく、キャパシティも限られる

### 成功のための5つのポイント

1. **市場ニーズと競合を把握する** — 需要の確認と差別化を明確に
2. **利益率の高いビジネスを選ぶ** — 月単位・中長期の収益性で判断
3. **専門分野・得意分野を活かす** — 未経験参入はリスクが高い
4. **継続的に収益が生まれる仕組みを作る** — サブスク・保守契約など
5. **誰がやっても一定成果が出せる形を目指す** — 再現性を高めてスケールへ

### 向いている業種例

| 業種 | 特徴 |
|---|---|
| ネットショップ（EC） | 初期費用不要のサービスあり・拡張性高い |
| ホームページ制作 | PC＋スキルがあればすぐ開始可能 |
| プログラミング | 需要高く・自社サービス展開も可能 |
| アフィリエイト | 副業からスタートするのが現実的 |
| コンサルティング | 専門知識を収益化・副業から人脈構築が有効 |
| キッチンカー | 店舗費用なし・移動で需要の高い場所へ |
| 家事代行 | 初期費用不要・隙間時間を活用 |

## CFO目線の Key Insight

> **スモールビジネスを「永遠の小規模」で終わらせないための設計**
>
> スモールビジネスの罠は「自分が動かないと収益が止まる」構造にある。CFO思考では、**ビジネスモデルの再現性と収益の継続性**（ストック型収益）を設計段階で組み込むことが重要だ。単発の案件収益（フロー型）だけでは、体調不良や繁閑差で即キャッシュフローが危機に陥る。「誰でもできる仕組み」と「継続課金の仕組み」の両輪を最初から意識して設計する。
