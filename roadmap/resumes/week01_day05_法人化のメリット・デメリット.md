## 今日のゴール

個人事業主が法人化（法人成り）することのメリット7つ・デメリット5つを深く理解し、自分の事業に当てはめた場合の損得を判断できるようになる。

## 1. 法人化（法人成り）とは

**法人化** = 個人事業主として行っていた事業を、会社設立によって法人に移行すること

「法人成り」とも呼ばれる（個人が形を変えて法人へ「成る」ことから）。

### 個人事業主と法人：13項目比較

| 項目 | 個人事業主 | 法人 |
|---|---|---|
| 1. 事業開始の手続き | 開業届（税務署） | 法人登記（法務局） |
| 2. 費用 | **0円** | 株式会社：約22.2万円〜、合同会社：約10万円〜 |
| 3. 税金 | 所得税（累進課税：5〜45%） | 法人税（800万円以下：15%、超過：23.2%） |
| 4. 社会保険負担 | なし（従業員5人未満） | **あり（必須）** |
| 5. 維持費（税金以外） | なし | 社会保険料の企業負担あり |
| 6. 経費の範囲 | 事業費用のみ | **広い**（役員報酬・社宅・生命保険等） |
| 7. 社会的信用 | 低い | **高い** |
| 8. 資金調達 | 小規模のみ | 大規模調達が可能 |
| 9. 廃業手続き | 廃業届のみ | 解散登記・公告等（約8万円〜） |
| 10. 事業承継のしやすさ | しにくい | **しやすい** |
| 11. 赤字繰越 | **3年**（青色申告） | **10年** |
| 12. 責任範囲 | **無限責任** | **有限責任** |
| 13. 会計・経理 | 個人の確定申告 | 法人決算（税理士が必要になることが多い） |

## 2. 法人化の7つのメリット

### メリット① 税制上有利になるケースが多い

法人化で活用できる節税手段：

| 節税手段 | 内容 |
|---|---|
| **役員報酬を損金計上** | 自分への給与を経費化し、法人所得を圧縮。受け取る個人には給与所得控除も適用 |
| **退職金を損金計上** | 役員退職金を法人の経費として処理（個人事業主本人への退職金は不可） |
| **欠損金10年繰越** | 赤字を最大10年間繰り越して将来の利益と相殺（個人は3年） |
| **消費税2年免除** | 法人設立から最大2年間、消費税納税が免除される（資本金1,000万円未満の場合） |
| **生命保険料の経費計上** | 法人名義の生命保険料の一部を経費計上（個人は12万円上限の所得控除のみ） |

### メリット② 賠償範囲が有限責任になる

個人事業主は無限責任（仕入先への未払い・借金・税金滞納はすべて個人の責任）。
法人化すると**出資額の範囲内に限定される有限責任**になる（個人保証付き借入を除く）。

### メリット③ 決算月を任意に設定できる

個人事業主は12月末固定。法人は**自由に決算月を設定**できるため、繁忙期を避けた設定が可能。

### メリット④ 社会的信用が高まる

- 登記情報が公開され、法人の存在が証明される
- 大企業との取引・金融機関からの融資に有利
- BtoB取引で信頼を得やすい

### メリット⑤ 経費の範囲が広がる

法人化で経費にできるもの（個人事業主では不可）：
- 役員報酬（自分・家族への給与）
- 社宅家賃
- 出張日当
- 法人契約の生命保険料
- 退職金

### メリット⑥ 事業が継続しやすくなる

- 代表者が死亡しても法人は存続（口座凍結なし）
- 法人株式の相続は個人事業の承継より容易
- 相続対策・支配権維持に有利

### メリット⑦ 優秀な人材を確保しやすくなる

- 社会保険（厚生年金・健康保険）への加入が必須→従業員にとってメリット
- 独自の福利厚生制度の整備が可能
- 個人事業主より正社員を採用しやすい

## 3. 法人化の5つのデメリット

| # | デメリット | 詳細 |
|---|---|---|
| 1 | **設立費用がかかる** | 株式会社：約22.2万円〜、合同会社：約10万円〜 |
| 2 | **赤字でも税金がかかる** | 法人住民税の均等割（最低7万円）は赤字でも必ず発生 |
| 3 | **社会保険への加入が必須** | 社長1人でも全員加入義務。保険料の企業負担（給与の約15%）が発生 |
| 4 | **会計・事務手続きが増える** | 決算書作成・申告が複雑化。税理士依頼が現実的（月2〜5万円/月） |
| 5 | **交際費の全額損金計上が制限される** | 資本金1億円以下：飲食費の50%または年800万円上限のみ損金算入可能（2027年3月末まで特例） |

---

## 4. 法人化の最適なタイミング

| タイミング | 理由 |
|---|---|
| **所得が800万円を超えたとき** | 所得税（累進課税）が23%になる水準で、法人税15%（800万円以下）より高くなる |
| **売上が1,000万円を超えたとき** | 2年後に消費税課税事業者となる前に法人化すると、免税期間を最大2年延長できる |
| **事業拡大・大幅な資金調達を検討するとき** | 法人でなければできない契約・株式による資金調達が必要になるタイミング |

> **注意**: 所得700万円を超えたタイミングで税理士へ相談することを推奨

**所得税（個人）**:
| 課税所得 | 税率 |
|---|---|
| 695万円〜900万円 | 23% |
| 900万円〜1,800万円 | 33% |
| 1,800万円〜4,000万円 | 40% |
| 4,000万円超 | 45% |

**法人税（法人）**:
| 所得 | 税率 |
|---|---|
| 800万円以下 | **15%** |
| 800万円超 | **23.2%** |

## CFO目線の Key Insight

> **法人化の意思決定は「今の税額最小化」ではなく「5年後のキャッシュフロー最大化」で考える**
>
> 単純な税率比較だけで法人化を判断するのは危険だ。法人化すると社会保険料（企業負担分）・税理士費用・均等割・事務コストが増加するため、「名目上の節税額」と「実際に増えるコスト」を正確に比較しないと、むしろ手取りが減るケースもある。CFO思考では、役員報酬の設定・退職金の積み立て・消費税免税期間の活用・将来の資金調達計画を統合した「長期キャッシュフロー最適化」の観点から法人化を検討する。
