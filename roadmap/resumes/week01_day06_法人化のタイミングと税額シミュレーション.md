# Day 6 レジュメ｜法人化のタイミングと税額シミュレーション

## 今日のゴール

法人化のタイミングを3つの観点（利益・売上・節税）から理解し、具体的な税額シミュレーションで「いくらから法人化が得か」を数字で把握する。さらに法人化後の後悔事例から、事前に防ぐべきリスクを学ぶ。

---

## 1. 法人化するタイミング：3つの観点

### 観点① 利益の観点（所得800万円が目安）

個人事業主の所得税（累進課税）と法人税（比例課税）の税率比較：

| 所得金額 | 個人：所得税率 | 法人：法人税率 |
|---|---|---|
| 330万〜695万円 | 20% | 15%（800万以下） |
| 695万〜900万円 | 23% | 15%（800万以下） |
| 900万〜1,800万円 | 33% | 23.2%（800万超） |
| 1,800万〜4,000万円 | 40% | 23.2% |
| 4,000万円超 | 45% | 23.2% |

→ **所得800万円**を超えたあたりから法人税の方が有利になり始める（ただし役員報酬・社会保険等の条件による）

**注意**: 役員報酬を自分に支払うと、法人側は損金算入できるが、個人側に所得税・社会保険料が発生。「法人の税額」だけでなく「個人の税額も含めたトータル」で比較が必要。

### 観点② 売上の観点（売上1,000万円が目安）

**なぜ売上1,000万円が目安か**:
- 年間売上が1,000万円を超えると、**その2年後**から消費税課税事業者となる
- 1,000万円超えのタイミングで法人化すると、基準がリセットされる

**消費税免税期間の延長例**:
```
2023年：個人事業主として売上1,000万円超
  ↓  （このタイミングで法人化）
2024年：新法人で売上基準がリセット
2025年：免税継続（本来は2025年から課税事業者になるはずだった）
  ↓  最短で2026年からの課税となり、最大2年間延長
```

**注意**: 資本金1,000万円以上で設立した場合は初年度から課税事業者となるため注意。

### 観点③ 節税の観点（役員報酬・退職金・欠損金繰越）

法人化することで活用できる節税メカニズム：

1. **役員報酬** → 給与所得控除が適用される（個人の実質税負担が下がる）
2. **退職金** → 法人の損金＋受け取る側に退職所得控除（二重の節税効果）
3. **欠損金の10年繰越** → 3年しか繰越できない個人事業より有利
4. **生命保険の経費計上** → 法人名義で加入することで保険料を経費化
5. **決算月の自由設定** → 消費税判定期間を有利にコントロールできる

## 2. 税額シミュレーション（具体的な数値）

> 前提条件：所得控除120万円、中小法人で設立、事業利益を全額役員報酬として支払う

### ケース①：事業利益500万円の場合

**個人事業主のままの場合**:
| 税金 | 計算 | 金額 |
|---|---|---|
| 所得税 | (500−120)×20%−42.75 | 約33万円 |
| 復興特別所得税 | 33×2.1% | 約0.7万円 |
| 住民税 | (500−120)×10% | 約38万円 |
| 個人事業税 | (500−290)×5% | 約10.5万円 |
| **合計** | | **約83万円** |

**法人化した場合**（利益を全額役員報酬として支払い）:
| 税金 | 計算 | 金額 |
|---|---|---|
| 法人税 | 役員報酬で課税所得ゼロ | 0円 |
| 法人住民税（均等割） | | 約7万円 |
| 所得税（役員個人） | (500−154−120)×10%−9.75 | 約13万円 |
| 復興特別所得税 | 12.85×2.1% | 約0.3万円 |
| 住民税（役員個人） | (500−154−120)×10% | 約23万円 |
| **合計** | | **約43万円** |

**節税効果: 約40万円**

---

### ケース②：事業利益1,000万円の場合

**個人事業主のままの場合**:
| 税金 | 計算 | 金額 |
|---|---|---|
| 所得税 | (1,000−120)×23%−63.6 | 約139万円 |
| 復興特別所得税 | 138.8×2.1% | 約3万円 |
| 住民税 | (1,000−120)×10% | 約88万円 |
| 個人事業税 | (1,000−290)×5% | 約35.5万円 |
| **合計** | | **約266万円** |

**法人化した場合**（利益を全額役員報酬として支払い）:
| 税金 | 計算 | 金額 |
|---|---|---|
| 法人住民税（均等割） | | 約7万円 |
| 所得税（役員個人） | (1,000−220−120)×20%−42.75 | 約89万円 |
| 復興特別所得税 | 89.25×2.1% | 約1.9万円 |
| 住民税（役員個人） | (1,000−220−120)×10% | 約66万円 |
| **合計** | | **約164万円** |

**節税効果: 約102万円**

> ※給与所得控除：500万円の場合 500×20%+54=154万円 / 1,000万円の場合 1,000×10%+120=220万円

## 3. 法人化で後悔する6つの理由と対策

| 後悔の理由 | 具体的な内容 | CFO的な対策 |
|---|---|---|
| **コストが想定より高かった** | 設立費用（株式会社：約22万円〜）＋維持費を軽視 | 設立前に3年間の収支シミュレーションを作成する |
| **個人事業主にない固定費が発生** | 法人住民税均等割（最低7万円）・社会保険料企業負担・税理士費用 | 年間固定コストを「利益がゼロでも発生するコスト」として先に試算する |
| **節税効果が期待以下だった** | 所得水準が低い段階での法人化・社会保険料でむしろ負担増 | 所得700万円を超えてから税理士に相談し、シミュレーションしてから判断 |
| **お金を自由に使えなくなった** | 法人の利益は法人のもの。役員報酬として受け取る必要がある | 役員報酬額を事業年度開始3ヶ月以内に確定する。変更は年1回のみ |
| **経営方針を自分だけで決められなくなった** | 出資者・共同創業者との意見対立 | 創業時に株式比率・議決権・拒否権を明確に設計する |
| **事務負担が増えた** | 決算申告・株主総会議事録・役員変更登記等が発生 | 会計ソフト導入＋税理士起用をコストではなく投資として計画する |

### 後悔しないための3つの対策

1. **法人化前後のコストを正確に把握する**（設立費用＋均等割＋社会保険料＋税理士費用）
2. **課税所得を正確に把握し、シミュレーションを行う**（個人vs法人のトータル比較）
3. **会計ソフトの導入・専門家への相談を事前に検討する**

### 法人化せずに後悔するタイミング（逆のリスク）

- 売上が安定し、累進課税で税負担が急増したとき
- 個人事業主の信用不足で大企業との契約・融資審査に失敗したとき
- 消費税免税期間の活用タイミングを逃したとき

## 4. 法人所得が残る場合の税率計算方法

役員報酬では吸収しきれない法人所得が発生した場合の税率体系：

| 税種 | 所得区分 | 税率 |
|---|---|---|
| **法人税** | 800万円以下 | 15% |
| **法人税** | 800万円超 | 23.4%（2023年〜23.2%） |
| **法人事業税** | 400万円以下 | 5% |
| **法人事業税** | 400万〜800万円 | 7.3% |
| **法人事業税** | 800万円超 | 9.6% |
| **法人住民税（均等割）** | 資本金1,000万円以下・従業員50人以下 | **7万円** |
| **法人住民税（法人税割）** | | 法人税額×17.3% |

## CFO目線の Key Insight

> **法人化の意思決定は「設立時の手取り計算」ではなく「3年分の資金計画」で判断せよ**
>
> 法人化を判断するには、「今年の節税額」だけでなく、設立コスト・社会保険料の増加・税理士費用・均等割を含めた**3年間のトータルキャッシュフロー比較**が必要だ。また、設立から2年間の消費税免税期間を最大化するには、課税売上が1,000万円を超えた「翌年の早いタイミング」での設立が有利だ。税理士への相談は「決める前」ではなく、「所得700万円を超えた時点」で行うのが理想。
