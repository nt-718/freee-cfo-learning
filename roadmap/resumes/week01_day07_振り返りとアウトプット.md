## 2. CFO視点：Week 1 の核心3原則

### 原則① 「形態選択は資本政策の第一歩」

起業形態（株式会社か合同会社か、法人化するかどうか）は、**将来の資金調達計画と経営権構造**から逆算して選ぶ。「今の費用が安いから」という理由だけで選ぶと、後の組織変更コストと機会損失が発生する。

### 原則② 「節税額だけでなくトータルキャッシュで判断」

法人化した場合のメリットを「所得税率 vs 法人税率」だけで比較するのは不十分。設立費用・社会保険料の企業負担・税理士費用・法人住民税均等割を含めた**実際の手取りの変化**で判断する。

### 原則③ 「タイミングは事前に設計する」

法人化のベストタイミングは「売上1,000万円超え直後」「所得700万円超えから税理士と相談開始」。後手に回ると消費税免税期間を逃したり、累進課税で高い税率を何年も払い続けることになる。

## 3. アウトプット課題

以下の質問に対して、自分の言葉で答えをまとめてみよう（メモ・口頭・どちらでも可）。

### 課題① 基礎確認
> **「個人事業主として年商800万円の事業を持つ人が法人化を検討している。メリット・デメリットを整理し、あなたならどうアドバイスするか？」**

考えるポイント：
- 所得（手取り）はいくらか（年商≠所得）
- 消費税は今後2年以内に課税事業者になるか
- BtoB事業か、BtoC事業か
- 将来の資金調達・事業拡大の計画はあるか

### 課題② 比較分析
> **「株式会社と合同会社、あなたが今の事業を法人化するとしたらどちらを選ぶか、理由を3つ挙げて説明してみよう。」**

考えるポイント：
- 設立費用・ランニングコストの違い
- 将来的にVCからの出資・IPOを考えるか
- 取引先がBtoB（法人）か、BtoC（個人）か

### 課題③ 数値計算
> **「事業利益700万円の個人事業主が法人化した場合（全額役員報酬として支払い、所得控除120万円）、税額はどう変わるか。概算で計算してみよう。」**

ヒント：
- 個人の場合：所得税・住民税・個人事業税を計算
- 法人の場合：均等割＋役員個人の所得税・住民税を計算
- 給与所得控除：700×10%+120=190万円

## 4. Week 1 の重要数字まとめ

| 数字 | 意味 |
|---|---|
| **800万円**（所得） | 法人化検討の目安所得（法人税15% vs 所得税23%の分岐点） |
| **1,000万円**（売上） | 消費税課税事業者となる基準売上（2年後） |
| **22.2万円〜** | 株式会社の最低設立費用 |
| **10万円〜** | 合同会社の最低設立費用 |
| **7万円** | 法人住民税均等割の最低額（赤字でも必ず発生） |
| **40万円** | 利益500万円時の法人化による節税効果（概算） |
| **102万円** | 利益1,000万円時の法人化による節税効果（概算） |
| **10年** | 法人の欠損金繰越年数（個人は3年） |
| **2年** | 法人設立後の消費税免除期間（最大） |
