# Week 6: 財務三表を読む

> **Phase 2 — 会計・経理の実務力** | B/S、P/L、C/F — CFOの「共通言語」をマスターする

---

## 今週のゴール

貸借対照表（B/S）、損益計算書（P/L）、キャッシュフロー計算書（C/F）の **構造・読み方・三表のつながり** を理解し、経営状態を数字から判断できるようになる。

---

## 学習マップ

```
財務三表
  ├── 決算書の全体像
  ├── 貸借対照表（B/S）

  ├── キャッシュフロー計算書（C/F）
  │     ├── 営業・投資・財務の3区分
  │     └── 黒字倒産の仕組み
  └── 三表のつながり
```

## Day 1: 決算書の全体像

### 読むべき記事（`kb/「会計」の基礎知識/03_決算書/`）

> このカテゴリの全記事を概観し、決算書の種類と役割を把握する

### 押さえるべきポイント

- **決算書 = 財務諸表**: 主に B/S、P/L、C/F、株主資本等変動計算書、注記表で構成
- **誰のために作るか**: ① 経営者自身（意思決定） ② 株主（投資判断） ③ 金融機関（融資審査） ④ 税務署（申告）
- **CFOの役割**: 決算書を「作る」だけでなく、経営陣に「説明する」、金融機関に「プレゼンする」力が求められる

## Day 2–3: 貸借対照表（B/S）

### 読むべき記事（`kb/「会計」の基礎知識/04_貸借対照表/`）

> このカテゴリの全記事を精読

### 押さえるべきポイント

- **B/Sの基本構造**:

```
┌─────────────────┬─────────────────┐
│    資産の部      │   負債の部      │
│  (持っているもの) │  (借りているもの) │
│                 │                 │
│  流動資産        │  流動負債        │
│  固定資産        │  固定負債        │
│                 ├─────────────────┤
│                 │  純資産の部      │
│                 │ (自己資金)       │
├─────────────────┼─────────────────┤
│  資産合計 = 負債合計 + 純資産合計     │
└─────────────────┴─────────────────┘
```

- **CFOが見るべき指標**:

| 指標 | 計算式 | 目安 | 意味 |
|---|---|---|---|
| 自己資本比率 | 純資産÷総資産 | 40%以上 | 財務の安全性 |
| 流動比率 | 流動資産÷流動負債 | 200%以上 | 短期の支払能力 |
| 当座比率 | 当座資産÷流動負債 | 100%以上 | より厳密な支払能力 |

- **流動と固定の区分基準**: 1年基準（ワンイヤールール）— 1年以内に現金化/支払期限が来るものが「流動」

## Day 4–5: 損益計算書（P/L）

### 読むべき記事（`kb/「会計」の基礎知識/05_損益計算書/`）

> このカテゴリの全記事を精読

### 押さえるべきポイント

- **P/Lの5段階利益**:

```
  売上高
- 売上原価
──────────
= 売上総利益（粗利）        ← 商品力の指標
- 販売費及び一般管理費
──────────
= 営業利益                 ← 本業の稼ぐ力
+ 営業外収益 - 営業外費用
──────────
= 経常利益                 ← 通常の経営力
+ 特別利益 - 特別損失
──────────
= 税引前当期純利益
- 法人税等
──────────
= 当期純利益               ← 最終的な儲け
```

- **CFOが注目すべき利益率**:

| 指標 | 意味 | SaaS企業の目安 |
|---|---|---|
| 売上総利益率 | 付加価値の高さ | 70%以上 |
| 営業利益率 | 本業の効率性 | 20%以上 |
| 経常利益率 | 総合的な収益力 | 15%以上 |

- **費用の「変動費 vs 固定費」分解**: P/Lの費用を変動費と固定費に分けると、損益分岐点売上高を計算できる。これはCFOが必ず把握すべき数字

## Day 6: キャッシュフロー計算書（C/F）

### 読むべき記事（`kb/「会計」の基礎知識/06_キャッシュフロー計算書/`）

> このカテゴリの全記事を精読

### 押さえるべきポイント

- **C/Fの3区分**:

| 区分 | 内容 | 健全な状態 |
|---|---|---|
| 営業CF | 本業での現金の動き | **プラス**（稼いでいる） |
| 投資CF | 設備投資・資産売却 | マイナス（成長投資中） |
| 財務CF | 借入・返済・配当 | 状況による |

- **黒字倒産の仕組み**: P/Lが黒字でも、売掛金の回収が遅れて現金が不足すると倒産する。**利益≠キャッシュ**。だからC/Fが必要

- **フリーキャッシュフロー（FCF）** = 営業CF + 投資CF。これが会社が自由に使えるお金。FCFがマイナスの状態が続くと資金調達が必要になる

## Day 7: 三表のつながりと振り返り

### 三表の連動

```
P/L の当期純利益 → B/S の利益剰余金に加算
B/S の現金 → C/F の期末残高と一致
C/F の営業CF → P/L の利益 ± 非現金項目の調整
```

- [ ] C/Fの3区分で、それぞれプラス・マイナスが意味することを説明できるか？
- [ ] 「黒字倒産」がなぜ起きるか説明できるか？
- [ ] 三表がどのようにつながっているか図示できるか？

### アウトプット課題

1. **三表つながり図**: B/S・P/L・C/Fのつながりを1枚の図にまとめる
2. **上場企業の決算書分析**: 任意の上場企業の決算書を取得し、主要指標を計算して経営状態を読み解く

---

## 💡 CFO目線の Key Insight

> **CFOは「P/L脳」から「B/S・C/F脳」に進化しなければならない。**
> 多くの経営者は売上と利益（P/L）だけを見る。しかし、会社を潰すのは赤字ではなく資金ショート（C/F）であり、会社の体力を見るのはB/S。三表をセットで読み、「利益は出ているがキャッシュが減っている」「資産は増えているが借入依存度が上がっている」といった複合的な判断ができるのがCFOの価値。
