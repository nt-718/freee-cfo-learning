### 押さえるべきポイント

- **法人に関わる主な税金**:

| 税目 | 概要 | 納付時期 |
|---|---|---|
| 法人税 | 利益に対して課税 | 事業年度終了後2ヶ月以内 |
| 法人住民税 | 均等割＋法人税割 | 同上 |
| 法人事業税 | 所得に対して課税 | 同上 |
| 消費税 | 取引に対して課税 | 事業年度終了後2ヶ月以内 |
| 源泉所得税 | 給与等から天引き | 毎月 or 半年に1回 |
| 固定資産税 | 資産保有に対して課税 | 年4回 |

- **CFOの税務カレンダー管理**: 各税目の申告・納付期限をカレンダーに登録し、キャッシュフロー計画に反映させる

### 押さえるべきポイント

- **法人税の実効税率**:

| 区分 | 800万円以下 | 800万円超 |
|---|---|---|
| 法人税率 | 15% | 23.2% |
| 実効税率（法人住民税・事業税含む） | 約22% | 約34% |

- **損金算入のCFO重要ポイント**:
  - 交際費: 中小法人は年800万円まで全額損金
  - 役員報酬: 定期同額給与でなければ損金不算入（Week 4で学習済み）
  - 寄附金: 限度額あり

- **繰越欠損金**: 赤字（欠損金）は最大10年間繰り越し、将来の黒字と相殺可能。スタートアップの初期赤字は将来の法人税を減らす「資産」になる

### 押さえるべきポイント

- **消費税のキーコンセプト**:

| 概念 | 説明 |
|---|---|
| 課税事業者 | 基準期間の売上1,000万円超で課税事業者 |
| 免税事業者 | 基準期間の売上1,000万円以下 |
| 課税取引 | 国内における対価を得て行う資産の譲渡等 |
| 非課税取引 | 土地の譲渡、社会保険医療など |
| 不課税取引 | 寄附金、配当金など |

- **簡易課税 vs 本則課税**:

| | 簡易課税 | 本則課税 |
|---|---|---|
| 対象 | 基準期間の売上5,000万円以下 | 全事業者 |
| 計算方法 | みなし仕入率で計算 | 実際の仕入税額を控除 |
| メリット | 事務負担が軽い | 実態に即した計算 |
| デメリット | 実際の仕入れが多いと損 | 帳簿管理が複雑 |

- **CFOの判断**: 創業2年間は資本金1,000万円未満なら免税事業者になれる（ただし適格請求書発行事業者に登録すると課税事業者に）。この2年間の消費税メリットは大きい

## 💡 CFO目線の Key Insight

> **税金は「払うもの」ではなく「最適化するもの」。**
> 合法的な節税とは、税法が用意した制度（繰越欠損金、少額資産の特例、簡易課税等）を最大限活用すること。CFOに求められるのは「脱税の知恵」ではなく「制度を網羅的に知り、自社に適用できる判断力」。税理士と対等に議論できるだけの知識基盤をこの週で作る。
