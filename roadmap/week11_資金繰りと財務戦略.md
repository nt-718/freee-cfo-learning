### 押さえるべきポイント

- **資金繰り表の基本構造**:

| 項目 | 4月 | 5月 | 6月 | … |
|---|---|---|---|---|
| **前月繰越残高** | 100 | 80 | 120 | |
| 営業収入 | 300 | 350 | 320 | |
| 営業支出 | △280 | △270 | △290 | |
| **営業収支** | 20 | 80 | 30 | |
| 財務収入（借入等） | 0 | 100 | 0 | |
| 財務支出（返済等） | △40 | △40 | △40 | |
| **当月収支** | △20 | 140 | △10 | |
| **月末残高** | 80 | 120 | 110 | |

- **キャッシュ・コンバージョン・サイクル（CCC）**:
  - CCC = 売掛金回転日数 + 棚卸資産回転日数 − 買掛金回転日数
  - **CCCが短いほどキャッシュ効率が良い**
  - 改善方法: 入金サイトを短縮する、支払サイトを延長する、在庫を圧縮する

- **CFOの鉄則**: 月末残高が2ヶ月分の固定費を下回ったら「要注意」、1ヶ月分を下回ったら「危険」

## Day 4–5: 予実管理

### 読むべき記事（`kb/「会計」の基礎知識/13_予実管理/`）

> このカテゴリの全記事を精読

### 押さえるべきポイント

- **予算の3レベル**:

| レベル | 内容 | 作成頻度 |
|---|---|---|
| 年間予算 | 年度の売上・費用・利益計画 | 年1回 |
| 四半期予算 | 3ヶ月単位の詳細計画 | 年4回 |
| 月次予算 | 月単位のキャッシュ計画 | 月次ローリング |

- **差異分析のフレームワーク**:
  - **好ましい差異**: 売上が予算を上回った、費用が予算を下回った
  - **好ましくない差異**: 売上が未達、費用が超過
  - **差異の原因分析**: 数量差異なのか、価格差異なのかを分解する

- **CFOの予実管理**: 毎月の経営会議で「予算 vs 実績 vs 前年同月」の3方向比較を報告。差異が5%以上ある項目には原因と対策を添える

### 読むべき記事（`kb/「請求書」の基礎知識/09_債権・債務/`）

> 全記事を概観

### 押さえるべきポイント

- **売掛金管理のKPI**:
  - DSO（Days Sales Outstanding）= 売掛金÷1日平均売上
  - 目安: 30日以内が健全、60日超は改善が必要
  - 定期的に「得意先別 × 期日別」の一覧を確認し、長期滞留を早期発見

- **買掛金管理のバランス**: 支払いを遅らせすぎると取引先との関係が悪化する。支払条件は取引先ごとに交渉し、自社のCCCが最適化されるよう設計

## 💡 CFO目線の Key Insight

> **「利益」ではなく「現金」で会社は回っている。**
> B/Sの利益剰余金がプラスでも、銀行口座の残高がゼロなら給料は払えない。資金繰り表はCFOの「命綱」。毎週更新し、最低3ヶ月先までの資金残高を常に把握する。資金ショートの兆候が見えたら、借入やファクタリングなどの手を「事前に」打つのがCFOの仕事。
