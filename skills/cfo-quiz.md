# CFO Quiz Skill（理解度クイズ生成）

ユーザーが指定したWeekまたはトピックについて、`quiz-master` エージェントを使って理解度クイズを生成します。

## 実行手順

1. **対象を特定する**
   - Week番号が指定されている場合: `roadmap/week{NN}_*.md` を読み込んで学習項目を確認する
   - トピックが指定されている場合: そのまま `quiz-master` に渡す
   - 難易度が指定されていない場合: 初級・中級を混在させた標準セット（5問）を生成する

2. **`quiz-master` エージェントを呼び出す**
   - 対象トピック、対応するKBパス、要求された難易度・問題数を渡す
   - `quiz-master` はKBの実際の内容に基づいて問題を生成する

3. **出力後の案内**
   - ユーザーが回答したら採点・解説を行う
   - 「もっと難しい問題」「別のトピック」などの続けやすい提案を添える

## 呼び出し例

```
/cfo-quiz                        → 直近のWeek（または今週の学習項目）でクイズ生成
/cfo-quiz Week 5                 → Week 5（簿記と勘定科目）の5問
/cfo-quiz 消費税 難易度：上級    → 消費税テーマの上級問題
/cfo-quiz 仕訳 10問              → 仕訳問題を10問生成
```
