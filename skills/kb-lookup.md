# KB Lookup Skill（KB検索・用語解説）

ユーザーが知りたい用語・概念・手続きを `kb-navigator` エージェントを使ってfreee KBから検索し、CFO学習に最適化された解説を返します。

## 実行手順

1. **検索キーワードを特定する**
   - ユーザーの質問から核となる用語・トピックを抽出する
   - 複数の関連キーワードがある場合は並行して探索する

2. **`kb-navigator` エージェントを呼び出す**
   - 特定したキーワードを渡し、関連KB記事を優先度付きで取得する

3. **解説を構成する**
   以下の構成で回答を作成する：
   ```
   ## {用語・トピック名}

   **一言定義**（15字以内）
   {簡潔な定義}

   **CFOとして知っておくべきこと**
   {実務上の意味・判断への影響・注意点を箇条書きで3〜5点}

   **📚 詳しく学べるKB記事**
   {kb-navigatorが返した優先度付きリスト}

   **🔗 関連トピック**
   {次に調べると理解が深まる隣接概念}
   ```

4. **深掘り案内**
   - 「さらに詳しく知りたい場合は `/kb-deep-dive {トピック}` を使用」と案内する

## 呼び出し例

```
/kb-lookup 定期同額給与          → 役員報酬の税務要件を解説
/kb-lookup インボイス制度        → 登録・仕訳・影響を解説
/kb-lookup 減価償却              → 計算方法と会計処理を解説
/kb-lookup 資金繰り表            → 作り方・見方を解説
```
