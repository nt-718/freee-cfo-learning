# Scenario Skill（シナリオ練習）

`scenario-coach` エージェントを呼び出し、学習者が指定したテーマ・フェーズについてのリアルなCFOシナリオを提示します。

## 実行手順

1. **対象テーマの特定**
   - Weekまたはトピックが指定されている場合: そのテーマでシナリオを生成
   - 指定がない場合: 最近学習したWeekから推定して提案する

2. **`scenario-coach` エージェントを呼び出す**
   - テーマ、フェーズ（Phase 1〜4）、難易度を渡す
   - `scenario-coach` がシナリオを生成・対話進行する

3. **対話セッションの進行**
   - ユーザーの回答を受けて `scenario-coach` がコーチングする
   - フィードバック後、関連KBを提示して知識補強につなげる

4. **セッション終了時**
   - 「このシナリオで使った知識を `/cfo-quiz` で確認しますか？」と案内する

## シナリオ難易度の目安

| 難易度 | 対象 | 特徴 |
|---|---|---|
| 入門 | Phase 1〜2を学習中 | 1つの判断、シンプルな状況 |
| 標準 | Phase 3〜4を学習中 | 複数の選択肢、トレードオフあり |
| 実戦 | 全Phase完了後 | 複合的な状況、リスクの優先順位付け |

## 呼び出し例

```
/scenario                       → テーマを提案してもらう
/scenario 役員報酬の決め方      → 役員報酬設計シナリオ
/scenario 資金繰り 実戦         → 実戦難易度の資金繰りシナリオ
/scenario Phase 2               → Phase 2（会計・経理）からランダム生成
```
