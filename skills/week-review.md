# Week Review Skill（週次振り返り）

指定されたWeekの学習内容を総括し、「理解できたか」を自己確認するための振り返りセッションをサポートします。ログファイルへの記録も支援します。

## 実行手順

1. **対象Weekの確認**
   - `roadmap/week{NN}_*.md` を読み込んで学習項目・到達チェックを取得する
   - 既存の学習ログ（`roadmap/log/`）があれば参照する

2. **到達チェックの実施**
   `roadmap/week{NN}_*.md` の「到達チェック」項目を質問として提示し、ユーザーに自己採点してもらう。

3. **重要ポイント整理**
   `kb-navigator` を呼び出してWeekの中心トピックを確認し、以下を出力する：
   ```
   ## Week {N} 振り返りサマリー

   ### 今週のテーマ
   {Phase・週タイトル・CFO視点の意義}

   ### 核心3点（このWeekで最も重要だったこと）
   1. ...
   2. ...
   3. ...

   ### よく混乱するポイント（注意事項）
   - ...

   ### 📚 読み返すべきKB記事
   {見直し推奨の記事リスト}
   ```

4. **理解度クイズの提案**
   振り返り後に `/cfo-quiz Week {N}` を自動実行するか確認する。

5. **ログテンプレートの記入支援**
   `roadmap/log/_template.md` のフォーマットに従い、今日のログ（`YYYY-MM-DD.md`）の下書きを生成する。ユーザーが確認した上で保存する。

## 呼び出し例

```
/week-review         → 直近のWeekを振り返る
/week-review Week 3  → Week 3（資本金と資金調達）を振り返る
/week-review 全体    → Phase全体（4週分）をまとめて振り返る
```
